# Komunitní filamenty

Komunitní filamenty jsou sdílená databáze profilů filamentů, přispěná uživateli Bambu Dashboard z celého světa. Najděte nejlepší nastavení pro váš filament a sdílejte vlastní zkušenosti.

Přejděte na: **https://localhost:3443/#community**

## Procházení komunitních profilů

### Vyhledávání

1. Přejděte na **Komunitní filamenty**
2. Použijte vyhledávací pole pro hledání podle:
   - Materiálu (PLA, PETG, ABS, PA, TPU atd.)
   - Dodavatele (PolyMaker, Bambu Lab, Prusament, Sunlu atd.)
   - Názvu barvy nebo hex kódu barvy
   - Vlastnosti (průhledný, matný, hedvábný, třpytivý atd.)
3. Filtrování podle **Hodnocení** (pouze nejlépe hodnocené)
4. Filtrování podle **Modelu tiskárny** pro profily optimalizované pro vaši tiskárnu

### Karty profilů

Každá karta profilu zobrazuje:

| Pole | Příklad |
|---|---|
| Název filamentu | PolyTerra PLA – Matte Forest Green |
| Materiál | PLA |
| Dodavatel | PolyMaker |
| Přispěvatel | uživatelské jméno |
| Hodnocení | ⭐⭐⭐⭐½ (4,6 / 5) |
| Počet testů | 23 |
| Doporučené teploty | Tryska: 215 °C / Podložka: 55 °C |
| Doporučená rychlost | Standardní |

Kliknutím na kartu zobrazíte kompletní profil se všemi parametry a komentáři uživatelů.

## Import profilu

1. Otevřete profil z komunity
2. Klikněte na **Importovat do skladu**
3. Vyberte, zda chcete:
   - **Propojit s existující cívkou** — profil se propojí s registrovanou cívkou
   - **Vytvořit novou cívku** — vytvoří nový záznam cívky s daty profilu
4. Upravte teplotu nebo jiná nastavení podle potřeby
5. Klikněte na **Importovat**

Profil je nyní dostupný ve vašem skladu filamentů.

## Přispívání vlastními profily

### Export a sdílení

1. Přejděte na **Sklad filamentů**
2. Najděte cívku, kterou chcete sdílet
3. Klikněte na **Sdílet s komunitou**
4. Vyplňte:
   - **Název dodavatele** — přesný název dodavatele
   - **Název produktu** — přesný název produktu z obalu
   - **Název barvy** — oficiální název barvy a hex kód barvy
   - **Komentář** — vaše zkušenosti, tipy a triky
   - **Model tiskárny** — na které tiskárně jste testovali
5. Klikněte na **Odeslat**

Profil je posuzován a obvykle se zveřejní do 24 hodin.

### Pokyny pro příspěvky

:::warning Požadavky na kvalitu
- Uvádějte pouze reálná, otestovaná nastavení — ne spekulativní hodnoty
- Používejte přesný název dodavatele a produktu (zkontrolujte obal)
- Jeden profil pro konkrétní produkt a model tiskárny
- Vyhněte se duplicitním profilům — zkontrolujte, zda profil již neexistuje
:::

## Hodnocení profilů

Pomozte komunitě hodnocením profilů, které jste testovali:

1. Otevřete komunitní profil
2. Klikněte na **Ohodnotit**
3. Zadejte počet hvězd (1–5) a krátký komentář
4. Klikněte na **Odeslat**

Profily s vysokým hodnocením se zobrazují nahoře ve výsledcích vyhledávání.

## Offline použití

Komunitní databáze se automaticky synchronizuje lokálně (aktualizuje se denně). Všechny profily jsou dostupné bez internetového připojení po první synchronizaci.

Stáhněte databázi ručně: **Komunitní filamenty → Nastavení → Synchronizovat nyní**
