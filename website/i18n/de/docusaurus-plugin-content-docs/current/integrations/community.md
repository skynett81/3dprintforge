# Community-Filamente

Community-Filamente sind eine gemeinsam genutzte Datenbank mit Filamentprofilen, die von Bambu Dashboard-Benutzern aus der ganzen Welt beigetragen wurden. Finden Sie die besten Einstellungen für Ihr Filament und teilen Sie Ihre eigenen Erfahrungen.

Navigieren Sie zu: **https://localhost:3443/#community**

## Community-Profile durchsuchen

### Suche

1. Gehen Sie zu **Community-Filamente**
2. Verwenden Sie das Suchfeld, um nach folgenden Kriterien zu suchen:
   - Material (PLA, PETG, ABS, PA, TPU usw.)
   - Hersteller (PolyMaker, Bambu Lab, Prusament, Sunlu usw.)
   - Farbname oder Farb-Hex
   - Eigenschaft (transparent, matt, Seide, Glitter usw.)
3. Nach **Bewertung** filtern (nur am höchsten bewertete)
4. Nach **Druckermodell** filtern, um Profile anzuzeigen, die für Ihren Drucker optimiert sind

### Profilkarten

Jede Profilkarte zeigt:

| Feld | Beispiel |
|---|---|
| Filamentname | PolyTerra PLA – Matte Forest Green |
| Material | PLA |
| Hersteller | PolyMaker |
| Beitragsleistender | Benutzername |
| Bewertung | ⭐⭐⭐⭐½ (4,6 / 5) |
| Anzahl Tests | 23 |
| Empfohlene Temperaturen | Düse: 215°C / Bett: 55°C |
| Empfohlene Geschwindigkeit | Standard |

Klicken Sie auf die Karte, um das vollständige Profil mit allen Parametern und Benutzerkommentaren zu sehen.

## Ein Profil importieren

1. Ein Profil aus der Community öffnen
2. Klicken Sie auf **In Lager importieren**
3. Wählen Sie, ob Sie:
   - **Mit vorhandener Spule verknüpfen** — Profil wird mit einer registrierten Spule verknüpft
   - **Neue Spule erstellen** — Erstellt einen neuen Spuleneintrag mit Profildaten
4. Temperatur oder andere Einstellungen nach Bedarf anpassen
5. Klicken Sie auf **Importieren**

Das Profil ist nun in Ihrem Filamentlager verfügbar.

## Eigene Profile beitragen

### Export und Teilen

1. Gehen Sie zu **Filamentlager**
2. Finden Sie die Spule, die Sie teilen möchten
3. Klicken Sie auf **Mit Community teilen**
4. Ausfüllen:
   - **Herstellername** — genauer Herstellername
   - **Produktname** — exakter Produktname von der Verpackung
   - **Farbname** — offizieller Farbname und Farb-Hex
   - **Kommentar** — Ihre Erfahrungen, Tipps und Tricks
   - **Druckermodell** — auf welchem Drucker Sie getestet haben
5. Klicken Sie auf **Einreichen**

Das Profil wird überprüft und in der Regel innerhalb von 24 Stunden veröffentlicht.

### Beitragsrichtlinien

:::warning Qualitätsanforderungen
- Geben Sie nur echte, getestete Einstellungen an — keine spekulativen Werte
- Verwenden Sie genaue Hersteller- und Produktnamen (Verpackung prüfen)
- Ein Profil pro spezifischem Produkt und Druckermodell
- Doppelte Profile vermeiden — prüfen Sie, ob das Profil bereits vorhanden ist
:::

## Profile bewerten

Helfen Sie der Community, indem Sie Profile bewerten, die Sie getestet haben:

1. Ein Community-Profil öffnen
2. Klicken Sie auf **Bewertung abgeben**
3. Anzahl Sterne (1–5) und einen kurzen Kommentar eingeben
4. Klicken Sie auf **Senden**

Profile mit hoher Bewertung erscheinen ganz oben in den Suchergebnissen.

## Offline-Nutzung

Die Community-Datenbank wird automatisch lokal synchronisiert (täglich aktualisiert). Alle Profile sind nach der ersten Synchronisierung ohne Internetverbindung verfügbar.

Datenbank manuell herunterladen: **Community-Filamente → Einstellungen → Jetzt synchronisieren**
