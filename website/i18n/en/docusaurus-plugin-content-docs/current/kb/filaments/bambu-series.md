# Bambu Lab filaments

Bambu Lab produces a wide range of filaments specially optimized for their printers. All Bambu Lab filaments come with an **RFID tag** that is automatically detected by the printer and sets the correct settings.

## RFID and AMS

All Bambu Lab filaments have an **RFID chip** built into the spool. This provides:

- **Automatic recognition** — the printer reads material type, color, and settings
- **Remaining amount** — estimated filament left on the spool
- **Correct settings** — temperature, speed, and cooling are set automatically
- **AMS compatibility** — seamless material switching in AMS

:::tip Third-party filaments in AMS
AMS also works with third-party filaments, but you must set the settings manually in Bambu Studio. RFID auto-detection is exclusive to Bambu Lab filaments.
:::

## Quality control and color consistency

Bambu Lab maintains strict quality control on their filaments:

- **Diameter tolerance** — ±0.02 mm (industry-leading)
- **Color consistency** — batch control ensures matching color between spools
- **Spool quality** — even winding without knots or overlap
- **Vacuum sealed** — each spool is delivered vacuum-packed with desiccant
- **Temperature profile testing** — each batch is tested for optimal temperature

:::tip Color number for consistency
Bambu Lab uses color numbers (e.g., "Bambu PLA Matte Charcoal") with batch control. If you need identical color across multiple spools for a large project, order from the same batch or contact support for batch matching.
:::
