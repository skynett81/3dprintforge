# Textured PEI Plate

La Textured PEI Plate (no confundir con la Engineering Plate) tiene una textura más rugosa y es especialmente adecuada para TPU y otros materiales flexibles, además de una variedad de materiales comunes.

## Características

- **Superficie:** Textura PEI rugosa — da una superficie inferior notablemente texturizada
- **Adherencia:** Excelente agarre mecánico, especialmente para TPU
- **Remoción:** Muy sencilla después de enfriar — dobla la placa y las piezas se sueltan
- **Temperatura máxima de cama:** 100 °C

## Mejor adecuada para

- **TPU** — esta es la mejor placa para materiales flexibles
- PLA y PLA+ — buena adherencia, buena superficie texturizada
- PETG — con una pequeña cantidad de bastón adhesivo
- PLA-CF — sin bastón adhesivo

## Compatibilidad

| Material | Idoneidad | Bastón adhesivo |
|-----------|---------|---------|
| TPU | ★★★★★ | No |
| PLA | ★★★★★ | No |
| PLA+ | ★★★★★ | No |
| PETG | ★★★★ | Poco (recomendado) |
| PLA-CF | ★★★★ | No |
| ABS | ★★★ | Sí |
| PA12 | ★★★ | Sí |

## TPU en Textured PEI

El TPU se adhiere perfectamente a la Textured PEI sin bastón adhesivo:
1. Establece la temperatura de cama a 30–45 °C
2. Calibra el desplazamiento Z con cuidado — un poco más alto que para PLA
3. Deja que la cama se enfríe a menos de 30 °C después de imprimir
4. Dobla la placa suavemente — el TPU se suelta sin esfuerzo

:::tip Remoción en frío
El TPU se suelta de la Textured PEI perfectamente al enfriar. Si tienes problemas para retirar la pieza, espera hasta que la placa esté completamente fría (20–25 °C) y dobla con más fuerza.
:::

## Mantenimiento

La Textured PEI es robusta y fácil de mantener:

```bash
# Entre impresiones:
- Limpia con IPA y papel absorbente sin pelusa

# Mensual (o si hay mala adherencia):
- Lava con agua caliente y detergente suave
- Enjuaga y seca bien
- Finaliza con IPA
```

## La superficie en la pieza terminada

La textura rugosa da una superficie inferior con un patrón distintivo en todas las piezas. Esto es:
- Deseable para prototipos y piezas funcionales (mejor agarre)
- No deseado para impresiones decorativas/visuales (usa Cool Plate para superficie lisa)
