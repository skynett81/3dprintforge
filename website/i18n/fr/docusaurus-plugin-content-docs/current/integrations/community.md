# Filaments communautaires

Les filaments communautaires constituent une base de données partagée de profils de filaments contribués par les utilisateurs de 3DPrintForge du monde entier. Trouvez les meilleurs réglages pour votre filament et partagez vos propres expériences.

Accédez à : **https://localhost:3443/#community**

## Parcourir les profils communautaires

### Recherche

1. Accédez à **Filaments communautaires**
2. Utilisez le champ de recherche pour trouver par :
   - Matériau (PLA, PETG, ABS, PA, TPU, etc.)
   - Fournisseur (PolyMaker, Bambu Lab, Prusament, Sunlu, etc.)
   - Nom de couleur ou code hexadécimal
   - Propriété (transparent, mat, soie, pailleté, etc.)
3. Filtrez par **Note** (uniquement les mieux notés)
4. Filtrez par **Modèle d'imprimante** pour voir les profils optimisés pour votre imprimante

### Fiches de profil

Chaque fiche de profil affiche :

| Champ | Exemple |
|---|---|
| Nom du filament | PolyTerra PLA – Matte Forest Green |
| Matériau | PLA |
| Fournisseur | PolyMaker |
| Contributeur | nom d'utilisateur |
| Note | ⭐⭐⭐⭐½ (4.6 / 5) |
| Nombre de tests | 23 |
| Températures recommandées | Buse : 215 °C / Plateau : 55 °C |
| Vitesse recommandée | Standard |

Cliquez sur la fiche pour voir le profil complet avec tous les paramètres et les commentaires des utilisateurs.

## Importer un profil

1. Ouvrez un profil communautaire
2. Cliquez sur **Importer dans le stock**
3. Choisissez si vous souhaitez :
   - **Lier à une bobine existante** — le profil est lié à une bobine enregistrée
   - **Créer une nouvelle bobine** — crée un nouvel enregistrement avec les données du profil
4. Ajustez la température ou d'autres réglages si nécessaire
5. Cliquez sur **Importer**

Le profil est maintenant disponible dans votre stock de filaments.

## Contribuer vos propres profils

### Export et partage

1. Accédez au **Stock de filaments**
2. Trouvez la bobine que vous souhaitez partager
3. Cliquez sur **Partager avec la communauté**
4. Renseignez :
   - **Nom du fournisseur** — nom exact du fournisseur
   - **Nom du produit** — nom exact du produit tel qu'indiqué sur l'emballage
   - **Nom de la couleur** — nom officiel de la couleur et code hexadécimal
   - **Commentaire** — vos expériences, conseils et astuces
   - **Modèle d'imprimante** — l'imprimante sur laquelle vous avez testé
5. Cliquez sur **Soumettre**

Le profil est en cours d'examen et sera généralement publié dans les 24 heures.

### Directives de contribution

:::warning Exigences de qualité
- Ne fournissez que des réglages réels et testés — pas de valeurs spéculatives
- Utilisez le nom exact du fournisseur et du produit (vérifiez l'emballage)
- Un profil par produit spécifique et modèle d'imprimante
- Évitez les doublons — vérifiez si le profil existe déjà
:::

## Évaluer les profils

Aidez la communauté en évaluant les profils que vous avez testés :

1. Ouvrez un profil communautaire
2. Cliquez sur **Donner une évaluation**
3. Indiquez le nombre d'étoiles (1–5) et un bref commentaire
4. Cliquez sur **Envoyer**

Les profils avec une note élevée apparaissent en tête des résultats de recherche.

## Utilisation hors ligne

La base communautaire se synchronise automatiquement en local (mise à jour quotidienne). Tous les profils sont disponibles sans connexion Internet après la première synchronisation.

Téléchargez manuellement la base : **Filaments communautaires → Paramètres → Synchroniser maintenant**
