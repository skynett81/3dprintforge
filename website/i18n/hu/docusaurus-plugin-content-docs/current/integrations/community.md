# Közösségi filamentek

A közösségi filamentek egy megosztott adatbázis, amelyet a világ minden tájáról érkező 3DPrintForge felhasználók töltöttek fel filamentprofilokkal. Keresd meg a filamentedhez legjobb beállításokat, és oszd meg saját tapasztalataidat.

Navigálj ide: **https://localhost:3443/#community**

## Böngészés a közösségi profilokban

### Keresés

1. Navigálj a **Közösségi filamentek** oldalra
2. Használd a keresőmezőt a kereséshez:
   - Anyag (PLA, PETG, ABS, PA, TPU stb.)
   - Szállító (PolyMaker, Bambu Lab, Prusament, Sunlu stb.)
   - Színnév vagy szín-hex
   - Tulajdonság (átlátszó, matt, selyem, csillogó stb.)
3. Szűrj **Értékelés** szerint (csak a legjobban értékeltek)
4. Szűrj **Nyomtatómodell** szerint, hogy a nyomtatódhoz optimalizált profilokat láss

### Profilkártyák

Minden profilkártya a következőket mutatja:

| Mező | Példa |
|---|---|
| Filament neve | PolyTerra PLA – Matte Forest Green |
| Anyag | PLA |
| Szállító | PolyMaker |
| Közreműködő | felhasználónév |
| Értékelés | ⭐⭐⭐⭐½ (4,6 / 5) |
| Tesztek száma | 23 |
| Ajánlott hőmérsékletek | Fúvóka: 215°C / Ágy: 55°C |
| Ajánlott sebesség | Normál |

Kattints a kártyára a teljes profil megtekintéséhez az összes paraméterrel és felhasználói megjegyzésekkel.

## Profil importálása

1. Nyiss meg egy közösségi profilt
2. Kattints az **Importálás a raktárba** gombra
3. Válaszd ki:
   - **Csatolás meglévő orsóhoz** — a profil egy regisztrált orsóhoz kapcsolódik
   - **Új orsó létrehozása** — új orsóbejegyzést hoz létre a profiladatokkal
4. Módosítsd a hőmérsékletet vagy más beállításokat szükség esetén
5. Kattints az **Importálás** gombra

A profil most elérhető a filamentraktáradban.

## Saját profilok hozzájárulása

### Exportálás és megosztás

1. Navigálj a **Filamentraktárba**
2. Keresd meg a megosztani kívánt orsót
3. Kattints a **Megosztás a közösséggel** gombra
4. Töltsd ki:
   - **Szállító neve** — pontos szállítónév
   - **Terméknév** — pontos terméknév a csomagolásról
   - **Színnév** — hivatalos színnév és szín-hex
   - **Megjegyzés** — tapasztalataid, tippek és trükkök
   - **Nyomtatómodell** — melyik nyomtatón tesztelted
5. Kattints a **Beküldés** gombra

A profil elbírálás alatt van, és általában 24 órán belül közzétételre kerül.

### Hozzájárulási irányelvek

:::warning Minőségi követelmények
- Csak valódi, tesztelt beállításokat adj meg — ne spekulatív értékeket
- Használd a pontos szállítói és terméknevet (ellenőrizd a csomagoláson)
- Egy profil egy konkrét termékhez és nyomtatómodellhez
- Kerüld a duplikált profilokat — ellenőrizd, hogy a profil már létezik-e
:::

## Profilok értékelése

Segítsd a közösséget a tesztelt profilok értékelésével:

1. Nyiss meg egy közösségi profilt
2. Kattints az **Értékelés adása** gombra
3. Add meg a csillagok számát (1–5) és egy rövid megjegyzést
4. Kattints a **Küldés** gombra

A magas értékelésű profilok a keresési eredmények tetején jelennek meg.

## Offline használat

A közösségi adatbázis automatikusan szinkronizálódik helyben (naponta frissül). Az összes profil elérhető internet-kapcsolat nélkül az első szinkronizálás után.

Adatbázis manuális letöltése: **Közösségi filamentek → Beállítások → Szinkronizálás most**
