# コスト計算機

コスト計算機では、プリンターに送信する前に印刷の総コストを見積もることができます。フィラメント消費量、電力料金、機械消耗に基づいています。

アクセス先：**https://localhost:3443/#cost-estimator**

## ファイルのアップロード

1. **コスト計算機**に移動します
2. ファイルをアップロードフィールドにドラッグ＆ドロップするか、**ファイルを選択**をクリックします
3. サポートされている形式：`.3mf`、`.gcode`、`.bgcode`
4. **分析**をクリックします

:::info 分析
システムはG-codeを分析してフィラメント消費量、推定プリント時間、素材プロファイルを抽出します。通常2〜10秒かかります。
:::

## フィラメントの計算

分析後、以下が表示されます。

| フィールド | 値（例） |
|---|---|
| 推定フィラメント | 47.3 g |
| 素材（ファイルから） | PLA |
| グラム単価 | 0.025円（フィラメント在庫から） |
| **フィラメントコスト** | **1.18円** |

ドロップダウンリストで素材を変更して異なるフィラメントタイプやメーカーのコストを比較します。

:::tip 素材の上書き
G-codeに素材情報が含まれていない場合は、リストから手動で素材を選択してください。価格はフィラメント在庫から自動的に取得されます。
:::

## 電力の計算

電力コストは以下に基づいて計算されます。

- **推定プリント時間** — G-code分析から
- **プリンターの消費電力** — プリンターモデルごとに設定（W）
- **電力料金** — 固定料金（円/kWh）またはTibber/Nordpoolからのライブ料金

| フィールド | 値（例） |
|---|---|
| 推定プリント時間 | 3時間22分 |
| プリンター消費電力 | 350 W（X1C） |
| 推定消費量 | 1.17 kWh |
| 電力料金 | 1.85円/kWh |
| **電力コスト** | **2.16円** |

希望する開始時刻に基づいて計画された時間別料金を使用するために、TibberまたはNordpoolインテグレーションを有効にしてください。

## 機械消耗

消耗コストは以下に基づいて推定されます。

- プリント時間 × プリンターモデルあたりの時間コスト
- 研磨素材（CF、GFなど）の追加消耗

| フィールド | 値（例） |
|---|---|
| プリント時間 | 3時間22分 |
| 時間コスト（消耗） | 0.80円/時間 |
| **消耗コスト** | **2.69円** |

時間コストはコンポーネント価格と予想寿命から計算されます（[消耗度予測](../overvaaking/wearprediction)を参照）。

## 合計

| コスト項目 | 金額 |
|---|---|
| フィラメント | 1.18円 |
| 電力 | 2.16円 |
| 機械消耗 | 2.69円 |
| **合計** | **6.03円** |
| + マークアップ（30%） | 1.81円 |
| **販売価格** | **7.84円** |

顧客への推奨販売価格を計算するために、パーセントフィールドでマークアップを調整します。

## 見積もりの保存

**見積もりを保存**をクリックして分析をプロジェクトにリンクします。

1. 既存のプロジェクトを選択するか新しいものを作成します
2. 見積もりが保存され、請求書の基礎として使用できます
3. 実際のコスト（プリント後）が自動的に見積もりと比較されます

## バッチ計算

複数のファイルを同時にアップロードして、完全なセットの総コストを計算します。

1. **バッチモード**をクリックします
2. すべての`.3mf`/`.gcode`ファイルをアップロードします
3. システムが個別および合計コストを計算します
4. PDFまたはCSVとして概要をエクスポートします
