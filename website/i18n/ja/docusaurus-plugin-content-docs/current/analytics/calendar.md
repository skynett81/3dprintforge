# アクティビティカレンダー

アクティビティカレンダーは、1年間のプリンターアクティビティを視覚的に概観します。GitHubのコントリビューション概要から着想を得ています。

アクセス先：**https://localhost:3443/#calendar**

## ヒートマップ概要

カレンダーには365日（52週）が色付きのタイルのグリッドとして表示されます。

- **灰色** — この日にプリントなし
- **薄緑** — 1〜2プリント
- **緑** — 3〜5プリント
- **濃緑** — 6〜10プリント
- **深緑** — 11プリント以上

タイルは垂直方向に曜日（月〜日）、左（1月）から右（12月）に横方向に週が並んでいます。

:::tip 色分け
カレンダー上部のセレクターでヒートマップの指標を**プリント数**から**時間**または**フィラメントグラム数**に切り替えることができます。
:::

## 年の選択

**< 年 >**をクリックして年を移動します。

- プリントアクティビティが記録されているすべての年が利用可能です
- デフォルトで現在の年が表示されます
- 将来は灰色です（データなし）

## 日ごとの詳細表示

タイルをクリックしてその日の詳細を確認します。

- **日付**と曜日
- **プリント数** — 成功したものと失敗したもの
- **総フィラメント使用量**（グラム）
- **総プリント時間**
- **プリントのリスト** — クリックして履歴で開く

## 月次概要

ヒートマップの下に月次概要が表示されます。
- 月ごとの総プリント数（棒グラフ）
- 月内の最もアクティブな日がハイライト
- 前年の同じ月との比較（%）

## プリンターフィルター

上部のドロップダウンリストからプリンターを選択して1つのプリンターのアクティビティのみ表示するか、**すべて**を選択して集計表示にします。

複数プリンター表示では、表示セレクターで**積み上げ**をクリックすると色が積み上げられます。

## ストリークと記録

カレンダーの下に以下が表示されます。

| 統計 | 説明 |
|---|---|
| **最長ストリーク** | 少なくとも1プリントがある連続日数の最高記録 |
| **現在のストリーク** | アクティブな日の現在の連続数 |
| **最もアクティブな日** | 総プリント数が最も多い日 |
| **最もアクティブな週** | プリント数が最も多い週 |
| **最もアクティブな月** | プリント数が最も多い月 |

## エクスポート

**エクスポート**をクリックしてカレンダーデータをダウンロードします。

- **PNG** — ヒートマップの画像（共有用）
- **CSV** — 1日1行の生データ（日付、数、グラム、時間）

PNGエクスポートはプリンター名と年をサブタイトルとしてソーシャルメディアでの共有用に最適化されています。
