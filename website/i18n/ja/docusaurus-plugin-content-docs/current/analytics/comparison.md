# プリント比較

プリント比較では、2つのプリントを並べて分析できます。設定、素材、プリンター、または同じモデルのバージョンを比較するのに便利です。

アクセス先：**https://localhost:3443/#comparison**

## 比較するプリントの選択

1. **プリント比較**に移動します
2. **プリントAを選択**をクリックして履歴を検索します
3. **プリントBを選択**をクリックして履歴を検索します
4. **比較**をクリックして比較表示を読み込みます

:::tip より速いアクセス
**履歴**から、プリントを右クリックして**プリントAとして設定**または**と比較...**を選択して比較モードに直接ジャンプできます。
:::

## 指標の比較

指標は2列（AとB）で表示され、どちらが優れているかがハイライトされます。

| 指標 | 説明 |
|---|---|
| 成功 | 完了 / キャンセル / 失敗 |
| 時間 | 総プリント時間 |
| フィラメント消費 | 色ごとの総グラム数 |
| フィラメント効率 | 総消費量に対するモデルの割合 |
| 最高ノズル温度 | 記録された最高ノズル温度 |
| 最高ベッド温度 | 記録された最高ベッド温度 |
| 速度設定 | 静音 / 標準 / スポーツ / ターボ |
| AMSカラーチェンジ | 色の切り替え回数 |
| HMSエラー | プリント中のエラー |
| プリンター | 使用したプリンター |

最も優れた値のセルが緑の背景で表示されます。

## 温度グラフ

2つの温度グラフが並んで（または重ねて）表示されます。

- **別々の表示** — グラフAが左、グラフBが右
- **重ねた表示** — 異なる色で同じグラフに両方表示

重ねた表示を使用して温度の安定性と加熱速度を直接比較します。

## ギャラリー画像

両方のプリントにマイルストーンスクリーンショットがある場合、グリッドに表示されます。

| プリントA | プリントB |
|---|---|
| 25%の画像A | 25%の画像B |
| 50%の画像A | 50%の画像B |
| 75%の画像A | 75%の画像B |
| 100%の画像A | 100%の画像B |

スライドアニメーション付きのフルスクリーンプレビューで画像を開くにはクリックしてください。

## タイムラプス比較

両方のプリントにタイムラプスがある場合、動画が並べて表示されます。

- 同期再生 — 両方が同時に開始・一時停止
- 独立した再生 — 各動画を別々に制御

## 設定の違い

システムは自動的に印刷設定の違いをハイライトします（G-codeメタデータから取得）。

- 異なるレイヤー厚さ
- 異なるインフィルパターンまたはパーセント
- 異なるサポート設定
- 異なる速度プロファイル

違いは設定テーブルにオレンジのハイライトで表示されます。

## 比較の保存

1. **比較を保存**をクリックします
2. 比較に名前を付けます（例：「PLA vs PETG - Benchy」）
3. 比較が保存され、**履歴 → 比較**から利用できます

## エクスポート

1. **エクスポート**をクリックします
2. **PDF**を選択してすべての指標と画像を含むレポートにします
3. 素材選択の文書化のためにレポートをプロジェクトにリンクできます
