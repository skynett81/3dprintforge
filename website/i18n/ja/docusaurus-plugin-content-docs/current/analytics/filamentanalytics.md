# フィラメント分析

フィラメント分析では、フィラメントの使用状況を完全に把握できます — 何を使っているか、コストはいくらか、何を節約できるかがわかります。

アクセス先：**https://localhost:3443/#filament-analytics**

## 消費概要

上部に選択した期間のサマリーが表示されます。

- **総消費量** — すべての素材のグラム数とメートル数
- **推定コスト** — 登録されたスプール単価に基づく
- **最も使用した素材** — 種類とメーカー
- **再利用率** — 実際のモデルに対するサポート/パージの割合

### 素材別消費量

円グラフと表で素材間の分布を表示します。

| 列 | 説明 |
|---|---|
| 素材 | PLA、PETG、ABS、PAなど |
| メーカー | Bambu Lab、PolyMaker、Prusamentなど |
| 使用グラム数 | 総重量 |
| メートル | 推定長さ |
| コスト | グラム数 × グラム単価 |
| プリント数 | この素材を使用したプリント数 |

行をクリックして個別スプールレベルに掘り下げます。

## 消費率

消費率は単位時間あたりの平均フィラメント消費量を表示します。

- **グラム/時間** — アクティブプリント中
- **グラム/週** — プリンター停止時間を含む
- **グラム/プリント** — プリントあたりの平均

これらは将来の必要量の予測計算に使用されます。

:::tip 購入計画
消費率を使用してスプール在庫を計画します。推定在庫が14日以内に切れる場合、システムが自動的に警告します（設定可能）。
:::

## コスト予測

過去の消費率に基づいて以下が計算されます。

- **次の30日間の推定消費量**（素材別グラム数）
- **次の30日間の推定コスト**
- **推奨在庫量**（30 / 60 / 90日間の運用に十分な量）

少なくとも1年分のデータがある場合、予測は季節変動を考慮します。

## 廃棄と効率

完全なドキュメントは[廃棄物追跡](./waste)を参照してください。フィラメント分析では概要が表示されます。

- **AMSパージ** — グラム数と総消費量に対する割合
- **サポート材** — グラム数と割合
- **実際のモデル材** — 残りの割合（効率%）
- **廃棄コストの推定** — 廃棄にかかるコスト

## スプールログ

すべてのスプール（アクティブおよび空）がログに記録されます。

| フィールド | 説明 |
|---|---|
| スプール名 | 素材名とカラー |
| 初期重量 | 開始時に登録された重量 |
| 残重量 | 計算された残量 |
| 使用済み | 総使用グラム数 |
| 最終使用日 | 最後のプリントの日付 |
| ステータス | アクティブ / 空 / 保管中 |

## 価格登録

正確なコスト分析のためにスプールごとの価格を登録します。

1. **フィラメント在庫**に移動します
2. スプールをクリックして**編集**を選択します
3. **購入価格**と**購入時の重量**を入力します
4. システムがグラム単価を自動的に計算します

価格が登録されていないスプールは**標準グラム単価**（**設定 → フィラメント → 標準価格**で設定）を使用します。

## エクスポート

1. **フィラメントデータをエクスポート**をクリックします
2. 期間と形式（CSV / PDF）を選択します
3. CSVにはプリントごとのグラム数、コスト、素材を含む1行が含まれます
