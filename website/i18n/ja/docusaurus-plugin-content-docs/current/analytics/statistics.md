# 統計

統計ページでは、プリンターのアクティビティについて、任意の期間にわたる主要指標、トレンド、フィラメント消費量の完全な概要が得られます。

アクセス先：**https://localhost:3443/#statistics**

## 主要指標

ページ上部に4つのKPIカードが表示されます。

| 主要指標 | 説明 |
|---|---|
| **成功率** | 総プリント数に対する成功したプリントの割合 |
| **総フィラメント** | 選択した期間に使用したグラム数 |
| **総プリント時間** | 累積プリント時間 |
| **平均プリント時間** | プリントごとの中央値時間 |

各主要指標には前の期間からの変化が（↑上昇 / ↓下降）パーセンテージ偏差として表示されます。

## 成功率

成功率はプリンターごとおよび合計で計算されます。

- **成功** — 中断なしにプリント完了
- **キャンセル** — ユーザーによる手動停止
- **失敗** — Print Guard、HMSエラー、またはハードウェア障害によって停止

成功率のグラフをクリックして、失敗したプリントとその原因を確認します。

:::tip 成功率の改善
[エラーパターン分析](../monitoring/erroranalysis)を使用して失敗したプリントの原因を特定して修正してください。
:::

## トレンド

トレンド表示では経時変化を折れ線グラフで表示します。

1. **期間**を選択：過去7 / 30 / 90 / 365日
2. **グループ化**を選択：日 / 週 / 月
3. **指標**を選択：プリント数 / 時間 / グラム / 成功率
4. **比較**をクリックして2つの指標を重ねる

グラフはズーム（スクロール）とパン（クリック＆ドラッグ）をサポートしています。

## フィラメント消費量

フィラメント消費量は以下として表示されます。

- **棒グラフ** — 日/週/月ごとの消費量
- **円グラフ** — 素材間の分布（PLA、PETG、ABSなど）
- **テーブル** — 素材ごとの総グラム、メートル、コストの詳細リスト

### プリンターごとの消費量

上部の複数選択フィルターを使用して：
- 1つのプリンターのみ表示
- 2つのプリンターを並べて比較
- すべてのプリンターの集計合計を確認

## アクティビティカレンダー

統計ページで簡略版のGitHubスタイルのヒートマップを直接確認できます（簡略表示）。または詳細表示のために完全な[アクティビティカレンダー](./calendar)に移動します。

## エクスポート

1. **統計をエクスポート**をクリックします
2. 日付範囲と含めたい指標を選択します
3. 形式を選択：**CSV**（生データ）、**PDF**（レポート）、または**JSON**
4. ファイルがダウンロードされます

CSVエクスポートはExcelおよびGoogle Sheetsと互換性があり、さらなる分析に使用できます。

## 前の期間との比較

**前の期間を表示**を有効にして、対応する前の期間にグラフを重ねます。

- 直近30日 vs. その前の30日
- 今月 vs. 先月
- 今年 vs. 昨年

これにより、以前より多くまたは少なくプリントしているか、そして成功率が改善されているかどうかを簡単に確認できます。
