# 廃棄物追跡

廃棄物追跡では、プリント中に無駄になるフィラメントの量（AMSパージ、素材交換時のフラッシング、サポート材）と、そのコストを完全に把握できます。

アクセス先：**https://localhost:3443/#waste**

## 廃棄カテゴリー

Bambu Dashboardは3種類の廃棄を区別します。

| カテゴリー | 発生源 | 典型的な量 |
|---|---|---|
| **AMSパージ** | マルチカラープリント中のAMSでの色変更 | 色変更1回あたり5〜30 g |
| **素材交換フラッシング** | 異なる素材間の切り替え時のクリーニング | 交換1回あたり10〜50 g |
| **サポート材** | プリント後に取り除くサポート構造 | 変動 |

## AMSパージ追跡

AMSパージデータはMQTTテレメトリとGコード分析から直接取得されます。

- **色変更ごとのグラム数** — Gコードパージブロックから計算
- **色変更回数** — プリントログからカウント
- **総パージ消費量** — 選択した期間の合計

:::tip パージを減らす
Bambu Studioには色の組み合わせごとのパージ量設定があります。色の差が小さいペア（例：白 → ライトグレー）のパージ量を減らしてフィラメントを節約できます。
:::

## 効率計算

効率は次のように計算されます。

```
効率% = (モデル材 / 総消費量) × 100

総消費量 = モデル材 + パージ + サポート材
```

**例：**
- モデル：45 g
- パージ：12 g
- サポート：8 g
- 合計：65 g
- **効率：69%**

効率は時系列のトレンドグラフとして表示され、改善状況を確認できます。

## 廃棄コスト

登録されたフィラメント価格に基づいて以下が計算されます。

| 項目 | 計算方法 |
|---|---|
| パージコスト | パージグラム数 × 色ごとのグラム単価 |
| サポートコスト | サポートグラム数 × グラム単価 |
| **総廃棄コスト** | 上記の合計 |
| **成功プリントあたりのコスト** | 廃棄コスト / プリント数 |

## プリンターおよび素材別廃棄

表示を以下でフィルタリングします。

- **プリンター** — どのプリンターが最も廃棄を生成しているかを確認
- **素材** — フィラメントタイプ別の廃棄量を確認
- **期間** — 日、週、月、年

テーブル表示では廃棄量の多い順にランク付けされたリストが表示され、推定コストも含まれます。

## 最適化ヒント

システムは廃棄を減らすための自動提案を生成します。

- **カラー順序の反転** — 色A→BよりB→Aのパージが少ない場合、順序の変更を提案
- **色変更レイヤーの統合** — 同じ色のレイヤーをグループ化して交換回数を最小化
- **サポート構造の最適化** — 向きを変えることによるサポート削減量を推定

:::info 精度
パージ計算はGコードから推定されます。実際の廃棄量はプリンターの動作により10〜20%異なる場合があります。
:::

## エクスポートとレポート

1. **廃棄データをエクスポート**をクリックします
2. 期間と形式（CSV / PDF）を選択します
3. 廃棄データはコスト項目としてプロジェクトレポートや請求書に含めることができます

総合的な消費概要については[フィラメント分析](./filamentanalytics)も参照してください。
