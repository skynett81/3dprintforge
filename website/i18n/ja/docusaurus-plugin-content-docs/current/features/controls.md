# プリンター制御

制御パネルでは、ダッシュボードから直接プリンターを完全に手動制御できます。

## 温度管理

### ノズル
- 目標温度を0〜350°Cの範囲で設定
- **設定**をクリックしてコマンドを送信
- リアルタイム読み取り値をアニメーションリングゲージで表示

### ヒートベッド
- 目標温度を0〜120°Cの範囲で設定
- プリント後の自動オフ（設定可能）

### チャンバー
- チャンバー温度をリアルタイムで確認
- **X1E、H2S、H2D、H2C**：M141経由のアクティブなチャンバー加熱制御（制御可能な目標温度）
- **X1C**：パッシブエンクロージャー — チャンバー温度は表示されますが直接制御はできません
- **P1S**：パッシブエンクロージャー — 温度は表示されますが、アクティブなチャンバー加熱制御はありません
- **P1P、A1、A1 mini、chamberHeatのないHシリーズ**：チャンバーセンサーなし

:::warning 最高温度
ノズルとベッドの推奨温度を超えないでください。硬化鋼ノズル（HFタイプ）の最高温度：300°C。真鍮製：最高260°C。プリンターのマニュアルを参照してください。
:::

## 速度プロファイル

速度コントロールには4つのプリセットプロファイルがあります。

| プロファイル | 速度 | 用途 |
|--------|----------|-------------|
| 静音 | 50% | 騒音低減、夜間印刷 |
| 標準 | 100% | 通常使用 |
| スポーツ | 124% | 高速プリント |
| ターボ | 166% | 最高速度（品質低下あり） |

スライダーで50〜200%のカスタムパーセントを設定できます。

## ファン制御

ファン速度を手動で制御します。

| ファン | 説明 | 範囲 |
|-------|-------------|--------|
| パーツ冷却ファン | 印刷物を冷却 | 0〜100% |
| 補助ファン | チャンバー循環 | 0〜100% |
| チャンバーファン | アクティブなチャンバー冷却 | 0〜100% |

:::tip 推奨設定
- **PLA/PETG：** パーツ冷却100%、補助30%
- **ABS/ASA：** パーツ冷却0〜20%、チャンバーファンオフ
- **TPU：** パーツ冷却50%、低速
:::

## G-codeコンソール

プリンターに直接G-codeコマンドを送信します。

```gcode
; 例：ヘッド位置を移動
G28 ; すべての軸をホーム
G1 X150 Y150 Z10 F3000 ; 中央に移動
M104 S220 ; ノズル温度を設定
M140 S60  ; ベッド温度を設定
```

:::danger G-codeに注意
G-codeが誤るとプリンターが損傷する可能性があります。理解しているコマンドのみ送信してください。プリント中に`M600`（フィラメント交換）を使用しないでください。
:::

## フィラメント操作

制御パネルから以下の操作ができます。

- **フィラメントをロード** — ノズルを加熱してフィラメントを引き込む
- **フィラメントをアンロード** — 加熱してフィラメントを引き出す
- **ノズルをクリーニング** — クリーニングサイクルを実行

## マクロ

G-codeコマンドのシーケンスをマクロとして保存・実行します。

1. **新しいマクロ**をクリックします
2. マクロに名前を付けます
3. G-codeシーケンスを入力します
4. 保存してワンクリックで実行します

ベッドキャリブレーション用のマクロ例：
```gcode
G28
M84
M500
```

## プリント制御

アクティブなプリント中に以下の操作ができます。

- **一時停止** — 現在のレイヤー後にプリントを一時停止
- **再開** — 一時停止したプリントを続行
- **停止** — プリントを中断（元に戻せません）
- **緊急停止** — すべてのモーターを即座に停止
