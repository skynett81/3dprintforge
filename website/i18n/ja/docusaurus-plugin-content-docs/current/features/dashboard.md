# メインパネル

メインパネルは3DPrintForgeの中央コントロールセンターです。選択したプリンターのリアルタイム状態を表示し、必要に応じて監視、制御、表示をカスタマイズできます。

アクセス先：**https://localhost:3443/**

## リアルタイム概要

プリンターがアクティブな場合、すべての値がMQTT経由で継続的に更新されます。

- **ノズル温度** — 目標温度付きのアニメーションSVGリングゲージ
- **ベッド温度** — ビルドプレート用の同様のリングゲージ
- **進捗率** — 残り時間付きの大きな進捗インジケーター
- **レイヤーカウンター** — 現在のレイヤー / 総レイヤー数
- **速度** — 静音 / 標準 / スポーツ / ターボのスライダー付き

:::tip リアルタイム更新
すべての値は、ページを再読み込みすることなくMQTT経由でプリンターから直接更新されます。遅延は通常1秒未満です。
:::

## 3Dモデル表示

プリンターが`.3mf`ファイルをモデルと共に送信する場合、インタラクティブな3Dプレビューが表示されます。

1. プリント開始時にモデルが自動的に読み込まれます
2. マウスでドラッグしてモデルを回転させます
3. スクロールでズームイン/アウトします
4. **リセット**をクリックしてデフォルト表示に戻ります

:::info サポートについて
3D表示には、プリンターがモデルデータを送信する必要があります。すべてのプリントジョブにこれが含まれるわけではありません。
:::

## AMS状態

AMSパネルには、すべてのマウントされたAMSユニットのスロットとフィラメントが表示されます。

- **スロットの色** — Bambuメタデータからのビジュアルカラー表示
- **フィラメント名** — 素材とブランド
- **アクティブスロット** — 印刷中にパルスアニメーションでハイライト
- **エラー** — AMSエラー時（詰まり、空、湿気）に赤いインジケーター

スロットをクリックして完全なフィラメント情報を確認し、フィラメント在庫にリンクできます。

## カメラフィード

ライブカメラ映像はffmpeg（RTSPS → MPEG1）で変換されます。

1. ダッシュボードを開くとカメラが自動的に起動します
2. カメラ画像をクリックしてフルスクリーンで開きます
3. **スナップショット**ボタンをクリックして静止画を撮影します
4. **カメラを非表示**をクリックしてスペースを確保します

:::warning パフォーマンス
カメラストリームは約2〜5 Mbit/sを使用します。低速なネットワーク接続ではカメラを無効にしてください。
:::

## 温度スパークライン

AMSパネルの下に過去30分間のミニグラフ（スパークライン）が表示されます。

- 時系列のノズル温度
- 時系列のベッド温度
- チャンバー温度（利用可能な場合）

スパークラインをクリックして完全なテレメトリーグラフビューを開きます。

## ウィジェットのカスタマイズ

ダッシュボードはドラッグ＆ドロップグリッドレイアウトを使用しています。

1. **レイアウトをカスタマイズ**（右上の鉛筆アイコン）をクリックします
2. ウィジェットを希望の位置にドラッグします
3. 角をドラッグしてサイズを変更します
4. **レイアウトをロック**をクリックして位置を固定します
5. **保存**をクリックしてレイアウトを保持します

利用可能なウィジェット：
| ウィジェット | 説明 |
|---|---|
| カメラ | ライブカメラ映像 |
| AMS | スプールとフィラメント状態 |
| 温度 | ノズルとベッドのリングゲージ |
| 進捗 | 進捗インジケーターと時間推定 |
| テレメトリー | ファン、気圧、速度 |
| 3Dモデル | インタラクティブモデル表示 |
| スパークライン | ミニ温度グラフ |

:::tip 保存について
レイアウトはブラウザのlocalStorageにユーザーごとに保存されます。異なるユーザーは異なるレイアウトを持てます。
:::
