# フィラメント在庫

フィラメント在庫では、AMSとプリント履歴に統合されたすべてのフィラメントスプールの完全な概要を確認できます。

## 概要

在庫には登録されたすべてのスプールが以下の情報と共に表示されます。

- **色** — ビジュアルカラーカード
- **素材** — PLA、PETG、ABS、TPU、PAなど
- **メーカー** — Bambu Lab、Polymaker、eSUNなど
- **重量** — 残量（グラム）（推定または計量）
- **AMSスロット** — スプールが入っているスロット
- **状態** — アクティブ、空、乾燥中、保管中

## スプールの追加

1. **+ 新しいスプール**をクリックします
2. 素材、色、メーカー、重量を入力します
3. 利用可能な場合はNFCタグをスキャン、またはI手動で入力します
4. 保存します

:::tip Bambu Labスプール
Bambu Labの公式スプールは、Bambu Cloudインテグレーションから自動的にインポートできます。[Bambu Cloud](../getting-started/bambu-cloud)を参照してください。
:::

## AMS同期

ダッシュボードがプリンターに接続されると、AMSの状態が自動的に同期されます。

- スロットにAMSから正しい色と素材が表示されます
- 各プリント後に消費量が更新されます
- 空のスプールに自動的にマークが付きます

ローカルスプールをAMSスロットにリンクするには：
1. **フィラメント → AMS**に移動します
2. リンクしたいスロットをクリックします
3. 在庫からスプールを選択します

## 乾燥

乾燥サイクルを記録して湿気の露出を追跡します。

| フィールド | 説明 |
|------|-------------|
| 乾燥日 | スプールを乾燥させた日付 |
| 温度 | 乾燥温度（°C） |
| 時間 | 時間数 |
| 方法 | オーブン、乾燥ボックス、フィラメント乾燥機 |

:::info 推奨乾燥温度
素材別の乾燥時間と温度については、[ナレッジベース](../kb/intro)を参照してください。
:::

## カラーカード

カラーカード表示では、スプールを色で視覚的に整理します。素早く正しい色を見つけるのに便利です。素材、メーカー、または状態でフィルタリングできます。

## NFCタグ

3DPrintForgeはNFCタグによるスプールの迅速な識別をサポートしています。

1. 在庫のスプールにNFCタグIDを記入します
2. スマートフォンでタグをスキャンします
3. ダッシュボードでスプールが直接開きます

## インポートとエクスポート

### エクスポート
在庫全体をCSVとしてエクスポート：**フィラメント → エクスポート → CSV**

### インポート
CSVからスプールをインポート：**フィラメント → インポート → ファイルを選択**

CSVフォーマット：
```
名前,素材,色_hex,メーカー,重量_グラム,nfc_id
PLA ホワイト,PLA,#FFFFFF,Bambu Lab,1000,
PETG ブラック,PETG,#000000,Polymaker,850,ABC123
```

## 統計

**フィラメント → 統計**では以下を確認できます。

- 素材ごとの総消費量（過去30/90/365日）
- プリンターごとの消費量
- スプールごとの推定残り寿命
- 最も使用された色とメーカー
