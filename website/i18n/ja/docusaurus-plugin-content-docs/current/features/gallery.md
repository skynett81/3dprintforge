# ギャラリー

ギャラリーには各プリント中に自動撮影されたスクリーンショットが集められています。画像は固定マイルストーンで撮影され、プリントの進行状況を視覚的にログとして記録します。

アクセス先：**https://localhost:3443/#gallery**

## マイルストーンスクリーンショット

3DPrintForgeは以下のマイルストーンでカメラからスクリーンショットを自動的に撮影します。

| マイルストーン | タイミング |
|---|---|
| **25%** | プリントの4分の1 |
| **50%** | 中間点 |
| **75%** | プリントの4分の3 |
| **100%** | プリント完了 |

スクリーンショットは対応するプリント履歴エントリーに保存され、ギャラリーに表示されます。

:::info 要件
マイルストーンスクリーンショットには、カメラが接続されてアクティブである必要があります。カメラが無効になっている場合は画像が生成されません。
:::

## スクリーンショット機能の有効化

1. **設定 → ギャラリー**に移動します
2. **自動マイルストーンスクリーンショット**をオンにします
3. 有効にするマイルストーンを選択します（デフォルトで4つすべてがオン）
4. **画像品質**を選択：低（640×360）/ 中（1280×720）/ 高（1920×1080）
5. **保存**をクリックします

## 画像表示

ギャラリーはプリントごとに整理されています。

1. 上部の**フィルター**を使用してプリンター、日付、ファイル名を選択します
2. プリント行をクリックして展開し、4枚の画像をすべて確認します
3. 画像をクリックしてプレビューを開きます

### プレビュー

プレビューには以下が表示されます。
- フルサイズの画像
- マイルストーンとタイムスタンプ
- プリント名とプリンター
- **←** / **→** で同じプリントの画像間を移動

## フルスクリーン表示

プレビューで**フルスクリーン**をクリック（または`F`を押す）して画面全体に表示します。矢印キーで画像間を移動します。

## 画像のダウンロード

- **単一画像**：プレビューで**ダウンロード**をクリック
- **プリントの全画像**：プリント行で**すべてダウンロード**をクリック — `.zip`ファイルが取得できます
- **複数選択**：チェックボックスにチェックを入れて**選択したものをダウンロード**をクリック

## 画像の削除

:::warning ストレージ容量
ギャラリー画像は時間とともにかなりのスペースを占める可能性があります。古い画像の自動削除を設定してください。
:::

### 手動削除

1. 1つまたは複数の画像を選択（チェック）します
2. **選択したものを削除**をクリックします
3. ダイアログで確認します

### 自動クリーンアップ

1. **設定 → ギャラリー → 自動クリーンアップ**に移動します
2. **X日より古い画像を削除**を有効にします
3. 日数を設定します（例：90日）
4. クリーンアップは毎晩午前3時に自動実行されます

## プリント履歴へのリンク

各画像はプリント履歴のエントリーにリンクされています。

- ギャラリーでプリントの**履歴で表示**をクリックして履歴エントリーにジャンプします
- 履歴では100%画像が存在する場合にサムネイルが表示されます

## 共有

期限付きリンクでギャラリー画像を共有します。

1. プレビューで画像を開きます
2. **共有**をクリックします
3. 有効期限を選択してリンクをコピーします
