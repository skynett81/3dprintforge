# プリント履歴

プリント履歴には、ダッシュボードで実行されたすべてのプリントの完全なログが提供されます。統計、フィラメント消費量、モデルソースへのリンクが含まれます。

## 履歴テーブル

テーブルにはすべてのプリントが以下の情報と共に表示されます。

| カラム | 説明 |
|---------|-------------|
| 日時 | 開始時刻 |
| モデル名 | ファイル名またはMakerWorldのタイトル |
| プリンター | 使用したプリンター |
| 時間 | 総プリント時間 |
| フィラメント | 素材と使用グラム数 |
| レイヤー | レイヤー数と重量（g） |
| 状態 | 完了、中断、失敗 |
| 画像 | サムネイル（クラウドインテグレーション時） |

## 検索とフィルタリング

検索フィールドとフィルターを使用してプリントを見つけます。

- モデル名のフリーテキスト検索
- プリンター、素材、状態、日付でフィルタリング
- すべてのカラムでソート

## モデルソースリンク

プリントがMakerWorldから開始された場合、モデルページへの直接リンクが表示されます。モデル名をクリックしてMakerWorldを新しいタブで開きます。

:::info Bambu Cloud
モデルリンクとサムネイルにはBambu Cloudインテグレーションが必要です。[Bambu Cloud](../getting-started/bambu-cloud)を参照してください。
:::

## フィラメント追跡

各プリントに以下が記録されます。

- **素材** — PLA、PETG、ABSなど
- **使用グラム数** — 推定消費量
- **スプール** — 使用したスプール（在庫に登録されている場合）
- **色** — 色のhexコード

これにより、時間の経過に伴うフィラメント消費量の正確な把握と、購入計画に役立てることができます。

## 統計

**履歴 → 統計**では集計データを確認できます。

- **総プリント数** — と成功率
- **総プリント時間** — 時間と日数
- **フィラメント消費量** — 素材ごとのグラムとメートル
- **1日あたりのプリント数** — スクロールグラフ
- **最も使用された素材** — 円グラフ
- **プリント時間の分布** — ヒストグラム

統計は期間（7日、30日、90日、1年、すべて）でフィルタリングできます。

## エクスポート

### CSVエクスポート
履歴全体またはフィルタリングされた結果をエクスポート：
**履歴 → エクスポート → CSVをダウンロード**

CSVファイルにはすべてのカラムが含まれており、Excel、LibreOffice Calc、または他のツールへのインポートに使用できます。

### 自動バックアップ
履歴はSQLiteデータベースの一部であり、更新時に自動的にバックアップされます。手動バックアップは**設定 → バックアップ**から行えます。

## 編集

プリントログエントリーを後から編集できます。

- モデル名を修正
- メモを追加
- フィラメント消費量を修正
- 誤って記録されたプリントを削除

行を右クリックして**編集**を選択するか、鉛筆アイコンをクリックしてください。
