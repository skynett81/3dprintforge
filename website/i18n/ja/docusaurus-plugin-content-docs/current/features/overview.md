# 機能概要

Bambu Dashboardは、Bambu Labプリンターを監視・制御するために必要なすべてを一か所に集めています。

## ダッシュボード

メインダッシュボードには、アクティブなプリンターのリアルタイム状態が表示されます。

- **温度** — ノズルとベッドのアニメーションSVGリングゲージ
- **進捗** — 完了予定時刻付きの進捗率
- **カメラ** — ライブカメラ映像（RTSPS → ffmpeg経由でMPEG1）
- **AMSパネル** — フィラメント色付きすべてのAMSスロットのビジュアル表示
- **速度コントロール** — 速度調整スライダー（静音、標準、スポーツ、ターボ）
- **統計パネル** — Grafanaスタイルのスクロールグラフパネル
- **テレメトリー** — ファン、温度、気圧のライブ値

パネルはドラッグ＆ドロップでレイアウトをカスタマイズできます。ロックボタンでレイアウトを固定できます。

## フィラメント在庫

詳細は[フィラメント](./filament)を参照してください。

- 名前、色、重量、メーカーですべてのスプールを追跡
- AMS同期 — AMSに入っているスプールを確認
- 乾燥ログと乾燥計画
- カラーカードとNFCタグサポート
- インポート/エクスポート（CSV）

## プリント履歴

詳細は[履歴](./history)を参照してください。

- すべてのプリントの完全なログ
- プリントごとのフィラメント追跡
- MakerWorldモデルへのリンク
- 統計とCSVエクスポート

## スケジューラー

詳細は[スケジューラー](./scheduler)を参照してください。

- プリントのカレンダー表示
- 優先度付きプリントキュー
- マルチプリンターディスパッチ

## プリンター制御

詳細は[制御](./controls)を参照してください。

- 温度制御（ノズル、ベッド、チャンバー）
- 速度プロファイル制御
- ファン制御
- G-codeコンソール
- フィラメントのロード/アンロード

## アラート

Bambu Dashboardは7つのアラートチャンネルをサポートしています。

| チャンネル | イベント |
|-------|----------|
| Telegram | プリント完了、エラー、一時停止 |
| Discord | プリント完了、エラー、一時停止 |
| メール | プリント完了、エラー |
| ntfy | すべてのイベント |
| Pushover | すべてのイベント |
| SMS（Twilio） | 重大なエラー |
| Webhook | カスタムペイロード |

**設定 → アラート**から設定してください。

## Print Guard

Print Guardはカメラ（xcam）とセンサーを通じてアクティブなプリントを監視します。

- スパゲッティエラー発生時の自動一時停止
- 設定可能な感度レベル
- 検出されたイベントのログ

## メンテナンス

メンテナンスセクションで追跡できる内容：

- コンポーネントごとの次回推奨サービス（ノズル、プレート、AMS）
- プリント履歴に基づく消耗度追跡
- メンテナンスタスクの手動記録

## マルチプリンター

複数プリンターのサポートにより：

- 1つのダッシュボードから複数のプリンターを管理
- プリンター選択機能でプリンター間を切り替え
- すべてのプリンターの状態を同時に確認
- プリントキューでプリントジョブを分配

## OBSオーバーレイ

専用の`obs.html`ページにより、3Dプリントのライブストリーミング中にOBS Studioインテグレーション用のクリーンなオーバーレイを提供します。

## アップデート

GitHub Releasesによる組み込みの自動更新。**設定 → 更新**からダッシュボード上で通知と更新ができます。
