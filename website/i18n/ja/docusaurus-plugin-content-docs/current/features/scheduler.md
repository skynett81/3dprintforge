# スケジューラー

スケジューラーを使用すると、カレンダー表示とインテリジェントな印刷キューでプリントジョブを整理・自動化できます。

## カレンダー表示

カレンダー表示では、計画済みおよび完了したすべてのプリントの概要が表示されます。

- **月、週、日表示** — 詳細レベルを選択
- **色分け** — プリンターと状態ごとに異なる色
- **イベントをクリック** — プリントの詳細を確認

完了したプリントはプリント履歴に基づいて自動的に表示されます。

## 印刷キュー

印刷キューを使用すると、順番にプリンターに送信されるジョブを並べることができます。

### キューへのジョブの追加

1. **+ ジョブを追加**をクリックします
2. ファイルを選択します（プリンターSDカード、ローカルアップロード、またはFTPから）
3. 優先度を設定します（高、通常、低）
4. 対象プリンターを選択します（または「自動」）
5. **追加**をクリックします

### キュー管理

| アクション | 説明 |
|----------|-------------|
| ドラッグ＆ドロップ | 順序を変更 |
| キューを一時停止 | 一時的に送信を停止 |
| スキップ | 待機せずに次のジョブを送信 |
| 削除 | キューからジョブを削除 |

:::tip マルチプリンターディスパッチ
複数のプリンターがある場合、キューは空いているプリンターにジョブを自動的に分配できます。**スケジューラー → 設定**で**自動ディスパッチ**を有効にしてください。
:::

## 計画済みプリント

特定の時刻に開始するプリントを設定します。

1. **+ プリントを計画**をクリックします
2. ファイルとプリンターを選択します
3. 開始時刻を設定します
4. アラートを設定します（任意）
5. 保存します

:::warning プリンターは空き状態である必要があります
計画済みのプリントは、指定された時刻にプリンターがスタンバイモードの場合のみ開始されます。プリンターが使用中の場合、次に利用可能な時刻まで延期されます（設定可能）。
:::

## 負荷分散

自動負荷分散により、プリンター間でジョブをインテリジェントに分配します。

- **ラウンドロビン** — すべてのプリンター間で均等に分配
- **最も空いているもの** — 推定完了時刻が最も短いプリンターに送信
- **手動** — ジョブごとに自分でプリンターを選択

**スケジューラー → 負荷分散**から設定します。

## アラート

スケジューラーはアラートチャンネルと統合されています。

- ジョブ開始時のアラート
- ジョブ完了時のアラート
- エラーまたは遅延時のアラート

アラートチャンネルの設定については[機能概要](./overview#varsler)を参照してください。
