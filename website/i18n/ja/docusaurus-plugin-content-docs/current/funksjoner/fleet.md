# フリート概要

フリート概要では、接続されているすべてのプリンターの一覧を1ページでコンパクトに確認できます。ワークショップ、教室、または複数のプリンターをお持ちの方に最適です。

アクセス先：**https://localhost:3443/#fleet**

## マルチプリンターグリッド

登録されたすべてのプリンターがレスポンシブグリッドに表示されます。

- **カードサイズ** — 小（コンパクト）、中（標準）、大（詳細）
- **列数** — 画面幅に応じて自動調整、または手動設定
- **更新** — 各カードがMQTT経由で独立して更新

各プリンターカードの表示内容：
| フィールド | 説明 |
|---|---|
| プリンター名 | モデルアイコン付きの設定名 |
| 状態 | アイドル / 印刷中 / 一時停止 / エラー / オフライン |
| 進捗 | 残り時間付きの進捗バー |
| 温度 | ノズルとベッド（コンパクト） |
| アクティブフィラメント | AMSからの色と素材 |
| カメラサムネイル | 30秒ごとに更新される静止画 |

## プリンターごとの状態インジケーター

状態の色により、離れた場所からでも状態を簡単に確認できます。

- **緑のパルス** — アクティブに印刷中
- **青** — アイドル状態で準備完了
- **黄** — 一時停止中（手動またはPrint Guardによる）
- **赤** — エラーを検出
- **灰色** — オフラインまたは利用不可

:::tip キオスクモード
壁掛けスクリーンでフリート概要をキオスクモードで使用できます。設定については[キオスクモード](../system/kiosk)を参照してください。
:::

## ソート

**ソート**をクリックして並び順を選択します。

1. **名前** — アルファベット順A〜Z
2. **状態** — アクティブなプリンターを先頭に
3. **進捗** — 最も進んでいるものを先頭に
4. **最終アクティブ** — 最近使用したものを先頭に
5. **モデル** — プリンターモデルでグループ化

ソート設定は次回の訪問まで記憶されます。

## フィルタリング

上部のフィルターフィールドを使用して表示を絞り込みます。

- プリンター名または名前の一部を入力します
- ドロップダウンから**状態**を選択（全て / 印刷中 / アイドル / エラー）
- **モデル**を選択して1種類のプリンターのみ表示（X1C、P1S、A1など）
- **フィルターをリセット**をクリックして全て表示

:::info 検索
検索はページを再読み込みすることなくリアルタイムでフィルタリングされます。
:::

## フリート概要からのアクション

カードを右クリック（または3点アイコンをクリック）してクイックアクション：

- **ダッシュボードを開く** — プリンターのメインパネルに直接移動
- **プリントを一時停止** — プリンターを一時停止
- **プリントを停止** — 進行中のプリントを中断（確認が必要）
- **カメラを表示** — ポップアップでカメラ映像を開く
- **設定に移動** — プリンター設定を開く

:::danger プリントの停止
プリントの停止は元に戻せません。表示されるダイアログボックスで必ず確認してください。
:::

## 集計統計

フリート概要の上部に要約行が表示されます。

- プリンターの総数
- アクティブなプリントの数
- 今日の総フィラメント消費量
- 最も長い進行中のプリントの推定完了時刻
