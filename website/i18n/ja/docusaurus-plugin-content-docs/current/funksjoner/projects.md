# プロジェクト

プロジェクトを使用すると、関連するプリントをグループ化し、素材コストを追跡し、顧客に請求書を発行し、作業の概要を共有できます。

アクセス先：**https://localhost:3443/#projects**

## プロジェクトの作成

1. **新しいプロジェクト**（+アイコン）をクリックします
2. 以下を入力します：
   - **プロジェクト名** — 説明的な名前（最大100文字）
   - **顧客** — オプションの顧客アカウント（[Eコマース](../integrasjoner/ecommerce)を参照）
   - **説明** — 短いテキスト説明
   - **色** — 視覚的な識別のための色を選択
   - **タグ** — カンマ区切りのキーワード
3. **プロジェクトを作成**をクリックします

## プリントをプロジェクトにリンク

### プリント中

1. プリントが進行中にダッシュボードを開きます
2. サイドパネルの**プロジェクトにリンク**をクリックします
3. 既存のプロジェクトを選択するか新しいものを作成します
4. プリントが完了するとプロジェクトに自動的にリンクされます

### 履歴から

1. **履歴**に移動します
2. 対象のプリントを見つけます
3. プリントをクリック → **プロジェクトにリンク**
4. ドロップダウンリストからプロジェクトを選択します

### 一括リンク

1. 履歴でチェックボックスを使用して複数のプリントを選択します
2. **アクション → プロジェクトにリンク**をクリックします
3. プロジェクトを選択 — 選択したすべてのプリントがリンクされます

## コスト概要

各プロジェクトは以下に基づいて総コストを計算します。

| コストタイプ | ソース |
|---|---|
| フィラメント消費 | グラム × 素材ごとのグラム単価 |
| 電力 | kWh × 電力単価（Tibber/Nordpool設定時） |
| 機械消耗 | [消耗度予測](../overvaaking/wearprediction)から計算 |
| 手動コスト | 手動で追加する自由入力項目 |

コスト概要はプリントごとおよび合計で表と円グラフで表示されます。

:::tip 時間別料金
正確な電力コストのためにTibberまたはNordpoolインテグレーションを有効にしてください。[電力料金](../integrasjoner/energy)を参照してください。
:::

## 請求書発行

1. プロジェクトを開いて**請求書を生成**をクリックします
2. 以下を入力します：
   - **請求書日**と**支払期日**
   - **消費税率**（0%、15%、25%）
   - **マークアップ**（%）
   - **顧客へのメモ**
3. PDF形式で請求書をプレビューします
4. **PDFをダウンロード**または**顧客に送信**（メール経由）をクリックします

請求書はプロジェクトに保存され、送信されるまで再度開いて編集できます。

:::info 顧客データ
顧客データ（名前、住所、法人番号）はプロジェクトにリンクした顧客アカウントから取得されます。顧客の管理については[Eコマース](../integrasjoner/ecommerce)を参照してください。
:::

## プロジェクトの状態

| 状態 | 説明 |
|---|---|
| アクティブ | プロジェクトが進行中 |
| 完了 | すべてのプリントが準備完了、請求書送信済み |
| アーカイブ | デフォルト表示では非表示だが検索可能 |
| 保留中 | 一時的に停止 |

プロジェクト上部の状態インジケーターをクリックして状態を変更します。

## プロジェクトの共有

顧客にプロジェクト概要を表示するための共有リンクを生成します。

1. プロジェクトメニューで**プロジェクトを共有**をクリックします
2. 表示する内容を選択します：
   - ✅ プリントと画像
   - ✅ 総フィラメント消費量
   - ❌ コストと価格（デフォルトでは非表示）
3. リンクの有効期限を設定します
4. リンクをコピーして共有します

顧客はログインなしで読み取り専用ページを見ることができます。
