# タイムラプス

Bambu Dashboardは印刷中に自動的に写真を撮影し、タイムラプス動画に組み合わせることができます。動画はローカルに保存され、ダッシュボードで直接再生できます。

アクセス先：**https://localhost:3443/#timelapse**

## 有効化

1. **設定 → タイムラプス**に移動します
2. **タイムラプス録画を有効にする**をオンにします
3. **録画モード**を選択します：
   - **レイヤーごと** — 1レイヤーにつき1枚（高品質に推奨）
   - **時間ベース** — N秒ごとに1枚（例：30秒ごと）
4. タイムラプスを有効にするプリンターを選択します
5. **保存**をクリックします

:::tip 画像間隔
「レイヤーごと」は動きが一定のため最も滑らかなアニメーションが得られます。「時間ベース」はストレージ容量を節約します。
:::

## 録画設定

| 設定 | デフォルト値 | 説明 |
|---|---|---|
| 解像度 | 1280×720 | 画像サイズ（640×480 / 1280×720 / 1920×1080） |
| 画像品質 | 85% | JPEG圧縮品質 |
| 動画FPS | 30 | 完成動画のフレーム/秒 |
| 動画形式 | MP4（H.264） | 出力形式 |
| 画像を回転 | オフ | 取り付け向きに応じて90°/180°/270°回転 |

:::warning ストレージ容量
1080pで500枚のタイムラプスは結合前に約200〜400 MBを使用します。完成したMP4動画は通常20〜80 MBです。
:::

## ストレージ

タイムラプスの写真と動画は、プロジェクトフォルダ内の`data/timelapse/`に保存されます。構造はプリンターとプリントごとに整理されます。

```
data/timelapse/
├── <printer-id>/                     ← ユニークなプリンターID
│   ├── 2026-03-22_modelname/         ← プリントセッション（日付_モデル名）
│   │   ├── frame_0001.jpg
│   │   ├── frame_0002.jpg
│   │   ├── frame_0003.jpg
│   │   └── ...                       ← 生の画像（結合後に削除）
│   ├── 2026-03-22_modelname.mp4      ← 完成したタイムラプス動画
│   ├── 2026-03-20_3dbenchy.mp4
│   └── 2026-03-15_telefonstativ.mp4
├── <printer-id-2>/                   ← 複数プリンターの場合
│   └── ...
```

:::tip 外部ストレージ
システムディスクのスペースを節約するために、タイムラプスフォルダを外部ディスクにシンボリックリンクできます：
```bash
# 例：/mnt/storageにマウントされた外部ディスクに移動
mv data/timelapse /mnt/storage/timelapse

# シンボリックリンクを作成
ln -s /mnt/storage/timelapse data/timelapse
```
ダッシュボードは自動的にシンボリックリンクに従います。任意のディスクまたはネットワーク共有を使用できます。
:::

## 自動結合

プリントが完了すると、ffmpegを使用して画像が自動的に動画に結合されます。

1. Bambu DashboardがMQTTから「print complete」イベントを受信します
2. 収集された画像でffmpegが呼び出されます
3. 動画がストレージフォルダに保存されます
4. タイムラプスページが新しい動画で更新されます

**タイムラプス → 処理中**タブで進捗を確認できます。

## 再生

1. **https://localhost:3443/#timelapse**に移動します
2. ドロップダウンリストからプリンターを選択します
3. リストの動画をクリックして再生します
4. 再生コントロールを使用します：
   - ▶ / ⏸ — 再生 / 一時停止
   - ⏪ / ⏩ — 巻き戻し / 早送り
   - 速度ボタン：0.5× / 1× / 2× / 4×
5. **フルスクリーン**をクリックしてフルスクリーンで開きます
6. **ダウンロード**をクリックしてMP4ファイルをダウンロードします

## タイムラプスの削除

1. リストで動画を選択します
2. **削除**（ゴミ箱アイコン）をクリックします
3. ダイアログボックスで確認します

:::danger 永久削除
削除されたタイムラプス動画と生の画像は復元できません。保存したい場合は先に動画をダウンロードしてください。
:::

## タイムラプスの共有

タイムラプス動画を期限付きリンクで共有できます。

1. 動画を選択して**共有**をクリックします
2. 有効期限を設定します（1時間 / 24時間 / 7日間 / 有効期限なし）
3. 生成されたリンクをコピーして共有します
4. 受信者はログインなしで動画を視聴できます
