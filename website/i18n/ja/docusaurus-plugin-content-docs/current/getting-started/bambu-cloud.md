# Bambu Cloudインテグレーション

3DPrintForgeは**Bambu Lab Cloud**に接続して、モデル画像、プリント履歴、フィラメントデータを取得することができます。クラウド接続なしでも問題なく動作しますが、クラウドインテグレーションにより追加のメリットが得られます。

## クラウドインテグレーションのメリット

| 機能 | クラウドなし | クラウドあり |
|---------|-----------|----------|
| リアルタイムプリンター状態 | はい | はい |
| プリント履歴（ローカル） | はい | はい |
| MakerWorldのモデル画像 | いいえ | はい |
| BambuのフィラメントプロファイルI | いいえ | はい |
| プリント履歴の同期 | いいえ | はい |
| クラウドからのAMSフィラメント | いいえ | はい |

## Bambu Cloudへの接続

1. **設定 → Bambu Cloud**に移動します
2. Bambu LabのメールアドレスとパスワードをI入力します
3. **ログイン**をクリックします
4. 同期するデータを選択します

:::warning プライバシーについて
ユーザー名とパスワードは平文では保存されません。ダッシュボードはBambu Labs APIを使用してOAuthトークンを取得し、ローカルに保存します。データがお客様のサーバーから外部に出ることはありません。
:::

## 同期

### モデル画像

クラウドが接続されると、モデル画像が自動的に**MakerWorld**から取得され、以下の場所に表示されます。
- プリント履歴
- ダッシュボード（アクティブなプリント中）
- 3Dモデルビューア

### プリント履歴

クラウド同期はBambu Labアプリからプリント履歴をインポートします。タイムスタンプとシリアル番号に基づいて重複が自動的に除去されます。

### フィラメントプロファイル

Bambu Labの公式フィラメントプロファイルが同期され、フィラメント在庫に表示されます。これらを独自プロファイルの出発点として使用できます。

## クラウドなしで利用できる機能

すべてのコア機能はクラウド接続なしで動作します。

- LAN経由でプリンターに直接MQTT接続
- リアルタイム状態、温度、カメラ
- ローカルプリント履歴と統計
- フィラメント在庫（手動管理）
- アラートとスケジューラー

:::tip LANのみモード
インターネット接続なしでダッシュボードを使用したいですか？隔離されたネットワーク内でも完璧に動作します — IPでプリンターに接続し、クラウドインテグレーションをオフにしておくだけです。
:::

## トラブルシューティング

**ログインが失敗する場合：**
- Bambu Labアプリのメールアドレスとパスワードが正しいか確認してください
- アカウントに二要素認証が設定されているか確認してください（現在未対応）
- ログアウトして再度ログインしてみてください

**同期が停止する場合：**
- トークンが期限切れの可能性があります — 設定でログアウトして再度ログインしてください
- サーバーからのインターネット接続を確認してください
