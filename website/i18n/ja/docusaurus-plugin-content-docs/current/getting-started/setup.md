# 初回セットアップ

ダッシュボードを初めて起動すると、セットアップウィザードが自動的に開きます。

## セットアップウィザード

ウィザードは`https://あなたのサーバー:3443/setup`で利用できます。以下の手順をガイドします。

1. 管理者ユーザーを作成する
2. プリンターを追加する
3. 接続をテストする
4. アラートを設定する（任意）

## プリンターの追加

プリンターに接続するには3つの情報が必要です。

| フィールド | 説明 | 例 |
|------|-------------|---------|
| IPアドレス | プリンターのローカルIP | `192.168.1.100` |
| シリアル番号 | 15文字、プリンター底部に記載 | `01P09C123456789` |
| アクセスコード | 8文字、プリンターのネットワーク設定で確認 | `12345678` |

### プリンターでアクセスコードを確認する

**X1C / P1S / P1P：**
1. 画面で**設定**に移動します
2. **WLAN**または**LAN**を選択します
3. **アクセスコード**を確認します

**A1 / A1 Mini：**
1. 画面をタップして**設定**を選択します
2. **WLAN**に移動します
3. **アクセスコード**を確認します

:::tip 固定IPアドレス
ルーターでプリンターに固定IPアドレスを設定してください（DHCP予約）。そうすることで、プリンターが新しいIPを取得するたびにダッシュボードを更新する必要がなくなります。
:::

## AMS設定

プリンターが接続されると、AMSの状態が自動的に更新されます。以下のことができます。

- 各スロットに名前と色を設定する
- スプールをフィラメント在庫にリンクする
- スプールごとのフィラメント消費量を確認する

手動設定は**設定 → プリンター → AMS**から行えます。

## HTTPS証明書 {#https-sertifikater}

### 自己署名証明書（デフォルト）

ダッシュボードは起動時に自己署名証明書を自動生成します。ブラウザで信頼するには：

- **Chrome/Edge：** 「詳細設定」→「サイトへ移動（安全でない）」をクリック
- **Firefox：** 「詳細情報」→「危険性を承知で続行」をクリック

### 独自の証明書

証明書ファイルをフォルダに配置し、`config.json`で設定します。

```json
{
  "ssl": {
    "cert": "/path/to/cert.pem",
    "key": "/path/to/key.pem"
  }
}
```

:::info Let's Encrypt
ドメイン名を使用していますか？Let's EncryptとCertbotで無料の証明書を生成し、`cert`と`key`を`/etc/letsencrypt/live/your-domain/`のファイルに指定してください。
:::

## 環境変数

すべての設定は環境変数で上書きできます。

| 変数 | デフォルト | 説明 |
|----------|---------|-------------|
| `PORT` | `3000` | HTTPポート |
| `HTTPS_PORT` | `3443` | HTTPSポート |
| `NODE_ENV` | `production` | 環境 |
| `AUTH_SECRET` | （自動） | JWTシークレット |

## 複数プリンターのセットアップ

**設定 → プリンター → プリンターを追加**から複数のプリンターを追加できます。ダッシュボード上部のプリンター選択機能でプリンター間を切り替えられます。
