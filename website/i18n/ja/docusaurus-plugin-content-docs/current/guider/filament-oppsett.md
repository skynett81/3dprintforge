# フィラメント在庫の設定

Bambu Dashboardのフィラメント在庫では、すべてのスプールの完全な概要が確認できます — 残量、使用量、現在AMSに入っているスプールなど。

## AMSからの自動作成

AMSを搭載したプリンターを接続すると、ダッシュボードがBambuスプールのRFIDチップから自動的に情報を読み取ります：

- フィラメントタイプ（PLA、PETG、ABS、TPUなど）
- 色（16進数カラーコード付き）
- ブランド（Bambu Lab）
- スプール重量と残量

**これらのスプールは在庫に自動的に作成されます** — 何もする必要はありません。**フィラメント → 在庫**で確認できます。

:::info RFIDがあるのはBambuスプールのみです
サードパーティのスプール（eSUN、Polymaker、チップなしのBambuリフィルなど）は自動的に認識されません。これらは手動で入力する必要があります。
:::

## スプールを手動で追加する

RFIDのないスプール、またはAMSに入っていないスプールの場合：

1. **フィラメント → 在庫**に移動します
2. 右上の**+ 新しいスプール**をクリックします
3. フィールドに入力します：

| フィールド | 例 | 必須 |
|-----------|-----|------|
| ブランド | eSUN、Polymaker、Bambu | はい |
| タイプ | PLA、PETG、ABS、TPU | はい |
| 色 | #FF5500 またはカラーホイールから選択 | はい |
| 開始重量 | 1000 g | 推奨 |
| 残量 | 850 g | 推奨 |
| 直径 | 1.75 mm | はい |
| メモ | 「2025年1月購入、良好」 | 任意 |

4. **保存**をクリックします

## 色とブランドの設定

在庫でスプールをクリックすることでいつでも編集できます：

- **色** — カラーホイールから選択するか、16進数値を入力してください。色はAMS概要の視覚的なマーカーとして使用されます
- **ブランド** — 統計とフィルタリングに表示されます。**フィラメント → ブランド**でカスタムブランドを作成できます
- **温度プロファイル** — フィラメントメーカーが推奨するノズルとプレートの温度を入力してください。誤った温度を選択した場合にダッシュボードが警告できます

## AMS同期について

ダッシュボードはAMSのステータスをリアルタイムで同期します：

```
AMSスロット1 → スプール：Bambu PLA ホワイト  [███████░░░] 72% 残り
AMSスロット2 → スプール：eSUN PETG グレー    [████░░░░░░] 41% 残り
AMSスロット3 → （空）
AMSスロット4 → スプール：Bambu PLA レッド    [██████████] 98% 残り
```

同期は以下のタイミングで更新されます：
- **印刷中** — 消費量がリアルタイムで差し引かれます
- **印刷終了時** — 最終消費量が履歴に記録されます
- **手動** — スプールの同期アイコンをクリックしてAMSから最新データを取得します

:::tip AMS推定値の修正
RFIDからのAMS推定値は初回使用後に100%正確とは限りません。スプールを計量して手動で重量を更新すると、より正確になります。
:::

## 消費量と残量の確認

### スプール別
在庫のスプールをクリックして以下を確認します：
- 総使用量（グラム、全印刷）
- 推定残量
- このスプールを使用した全印刷のリスト

### 集計統計
**分析 → フィラメント分析**では以下が確認できます：
- 時間経過によるフィラメントタイプ別消費量
- 最もよく使用するブランド
- 1kgあたりの購入価格に基づく推定コスト

### 低残量アラート
スプールが残り少なくなったときのアラートを設定します：

1. **フィラメント → 設定**に移動します
2. **残量低下時に通知**を有効にします
3. しきい値を設定します（例：残り100g）
4. 通知チャンネルを選択します（Telegram、Discord、メール）

## ヒント：精度のためにスプールを計量する

AMSと印刷統計からの推定値は完全に正確ではありません。最も正確な方法はスプール自体を計量することです：

**手順：**

1. タラ重量（空のスプール）を確認します — 通常200〜250g、メーカーのウェブサイトやスプールの底を確認してください
2. キッチンスケールでフィラメントを付けた状態でスプールを計量します
3. タラ重量を引きます
4. スプールプロファイルの**残量**を更新します

**例：**
```
計量重量：    743 g
タラ（空）：- 230 g
フィラメント残量： 513 g
```

:::tip スプールラベルジェネレーター
**ツール → ラベル**でスプール用のQRコード付きラベルを印刷できます。スマートフォンでコードをスキャンするとスプールプロファイルをすぐに開けます。
:::
