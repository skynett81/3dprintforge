# はじめての印刷

このガイドでは、プリンターの接続から印刷完了まで、Bambu Dashboardをコントロールセンターとして使用する全プロセスをご案内します。

## ステップ1 — プリンターが接続されていることを確認する

ダッシュボードを開くと、サイドバーの上部またはオーバービューパネルにプリンターのステータスカードが表示されます。

**緑のステータス**はプリンターがオンラインで準備完了であることを意味します。

| ステータス | 色 | 意味 |
|-----------|-----|------|
| オンライン | 緑 | 印刷準備完了 |
| アイドル | グレー | 接続済み、未稼働 |
| 印刷中 | 青 | 印刷進行中 |
| エラー | 赤 | 対応が必要 |

プリンターが赤のステータスを示している場合：
1. プリンターの電源が入っていることを確認してください
2. ダッシュボードと同じネットワークに接続されていることを確認してください
3. **設定 → プリンター**に移動し、IPアドレスとアクセスコードを確認してください

:::tip 応答速度向上のためにLANモードを使用する
LANモードはクラウドモードより低遅延です。プリンターとダッシュボードが同じネットワーク上にある場合は、プリンター設定で有効にしてください。
:::

## ステップ2 — モデルをアップロードする

Bambu Dashboardは直接印刷を開始しません — それはBambu StudioまたはMakerWorldの役割です。印刷が始まるとダッシュボードが引き継ぎます。

**Bambu Studio経由：**
1. PCでBambu Studioを開きます
2. `.stl`または`.3mf`ファイルをインポートまたは開きます
3. モデルをスライスします（フィラメント、サポート、インフィルなどを選択）
4. 右上の**印刷**をクリックします

**MakerWorld経由：**
1. [makerworld.com](https://makerworld.com)でモデルを見つけます
2. ウェブサイトから直接**印刷**をクリックします
3. Bambu Studioがモデルを読み込んで自動的に開きます

## ステップ3 — 印刷を開始する

Bambu Studioで送信方法を選択します：

| 方法 | 要件 | メリット |
|------|------|----------|
| **クラウド** | Bambuアカウント + インターネット | どこからでも使用可能 |
| **LAN** | 同じネットワーク | 高速、クラウド不要 |
| **SDカード** | 物理的なアクセス | ネットワーク不要 |

**送信**をクリックすると、プリンターがジョブを受信して自動的にウォームアップフェーズを開始します。

:::info 印刷がダッシュボードに表示されます
Bambu Studioがジョブを送信してから数秒以内に、アクティブな印刷が**アクティブな印刷**の下にダッシュボードに表示されます。
:::

## ステップ4 — ダッシュボードで監視する

印刷が進行中の場合、ダッシュボードは完全な概要を提供します：

### 進捗状況
- 完了率と推定残り時間がプリンターカードに表示されます
- カードをクリックするとレイヤー情報付きの詳細ビューが表示されます

### 温度
詳細パネルにリアルタイムの温度が表示されます：
- **ノズル** — 現在の温度と目標温度
- **ビルドプレート** — 現在の温度と目標温度
- **チャンバー** — プリンター内部の温度（ABS/ASAに重要）

### カメラ
プリンターカードのカメラアイコンをクリックすると、ダッシュボードでライブフィードを見ることができます。他の作業をしながらカメラを別のウィンドウで開いたままにすることができます。

:::warning 最初のレイヤーを確認する
最初の3〜5レイヤーは重要です。ここでの密着不良は後で印刷失敗につながります。カメラを見て、フィラメントが整然と均一に積層されていることを確認してください。
:::

### Print Guard
Bambu DashboardにはAI駆動の**Print Guard**があり、スパゲッティエラーを自動的に検出して印刷を一時停止することができます。**監視 → Print Guard**で有効にしてください。

## ステップ5 — 印刷完了後

印刷が完了すると、ダッシュボードに完了メッセージが表示されます（[通知](./varsler-oppsett)を設定している場合は通知も送信されます）。

### 履歴を確認する
サイドバーの**履歴**に移動して完了した印刷を確認します：
- 総印刷時間
- フィラメント消費量（使用グラム数、推定コスト）
- 印刷中のエラーまたはHMSイベント
- 完了時のカメラ画像（有効にしている場合）

### メモを記録する
履歴の印刷をクリックしてメモを追加します — 例：「もう少しブリムが必要だった」または「完璧な結果」。同じモデルを再度印刷する際に役立ちます。

### フィラメント消費量を確認する
**フィラメント**の下で、使用量に基づいてスプール重量が更新されていることを確認できます。ダッシュボードが自動的に差し引きます。

## 初心者向けのヒント

:::tip 最初の印刷を放置しない
最初の10〜15分は注意して見ていてください。印刷がしっかりと密着していることを確認したら、残りはダッシュボードに監視を任せることができます。
:::

- **空のスプールを計量する** — 正確な残量計算のためにスプールの開始重量を入力してください（[フィラメント在庫](./filament-oppsett)を参照）
- **Telegram通知を設定する** — 待たずに印刷完了の通知を受け取ります（[通知](./varsler-oppsett)を参照）
- **ビルドプレートを確認する** — きれいなプレート = より良い密着性。印刷間にIPA（イソプロパノール）で拭いてください
- **適切なプレートを使用する** — フィラメントに合うものについては[正しいビルドプレートの選択](./velge-rett-plate)を参照してください
