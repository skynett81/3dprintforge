# 3D印刷のOBSストリーミング

Bambu Dashboardには、プリンターのステータス、進捗状況、温度、カメラフィードを直接ストリームに表示するOBSオーバーレイが内蔵されています。

## 前提条件

- OBS Studioのインストール（[obsproject.com](https://obsproject.com)）
- Bambu Dashboardが実行中でプリンターに接続済み
- （任意）ライブフィード用のBambuカメラが有効

## ステップ1 — OBS Browser Source

OBSにはシーンに直接ウェブページを表示する**Browser Source**が内蔵されています。

**OBSにオーバーレイを追加する：**

1. OBS Studioを開きます
2. **ソース**（Sources）の下で**+**をクリックします
3. **ブラウザ**（Browser）を選択します
4. ソースに名前をつけます、例：「Bambu Overlay」
5. 入力します：

| 設定 | 値 |
|------|-----|
| URL | `http://localhost:3000/obs/overlay` |
| 幅 | `1920` |
| 高さ | `1080` |
| FPS | `30` |
| カスタムCSS | 下を参照 |

6. **OBSでオーディオを制御**をチェックします
7. **OK**をクリックします

:::info サーバーに合わせてURLを変更する
OBSとは別のマシンでダッシュボードを実行していますか？`localhost`をサーバーのIPアドレスに置き換えてください、例：`http://192.168.1.50:3000/obs/overlay`
:::

## ステップ2 — 透明な背景

オーバーレイが画像に溶け込むには背景が透明である必要があります：

**OBS Browser Sourceの設定で：**
- **背景を削除**（Shutdown source when not visible / Remove background）をチェックします

**透明性を強制するカスタムCSS：**
```css
body {
  background-color: rgba(0, 0, 0, 0) !important;
  margin: 0;
  overflow: hidden;
}
```

これをBrowser Sourceの設定の**カスタムCSS**フィールドに貼り付けます。

これでオーバーレイはウィジェット自体のみを表示します — 白または黒の背景なし。

## ステップ3 — オーバーレイのカスタマイズ

Bambu Dashboardでオーバーレイが何を表示するかを設定できます：

1. **機能 → OBSオーバーレイ**に移動します
2. 設定します：

| 設定 | オプション |
|------|-----------|
| 位置 | 左上、右上、左下、右下 |
| サイズ | 小、中、大 |
| テーマ | ダーク、ライト、透明 |
| アクセントカラー | ストリームのスタイルに合う色を選択 |
| 要素 | 表示するものを選択（下を参照） |

**利用可能なオーバーレイ要素：**

- プリンター名とステータス（オンライン/印刷中/エラー）
- 進捗バーと残り時間の割合
- フィラメントと色
- ノズル温度とプレート温度
- 使用フィラメント（グラム）
- AMS概要（コンパクト）
- Print Guardステータス

3. **プレビュー**をクリックしてOBSに切り替えずに結果を確認します
4. **保存**をクリックします

:::tip プリンターごとのURL
複数のプリンターがありますか？別々のオーバーレイURLを使用します：
```
/obs/overlay?printer=1
/obs/overlay?printer=2
```
:::

## OBSのカメラフィード（別のソース）

Bambuカメラは OBSに別のソースとして追加できます — オーバーレイとは独立して：

**代替1：ダッシュボードのカメラプロキシ経由**

1. **システム → カメラ**に移動します
2. **RTSPまたはMJPEGストリーミングのURL**をコピーします
3. OBSで：**+**をクリック → **メディアソース**（Media Source）
4. URLを貼り付けます
5. **ループ**をチェックし、ローカルファイルを無効にします

**代替2：カメラビュー付きBrowser Source**

1. OBSで：**Browser Source**を追加します
2. URL：`http://localhost:3000/obs/camera?printer=1`
3. 幅/高さ：カメラの解像度に合わせます（1080pまたは720p）

これでカメラフィードをシーンに自由に配置し、その上にオーバーレイを重ねることができます。

## 良いストリームのためのヒント

### ストリームシーンのセットアップ

3D印刷ストリーミングの典型的なシーン：

```
┌─────────────────────────────────────────┐
│                                         │
│      [プリンターからのカメラフィード]   │
│                                         │
│  ┌──────────────────┐                  │
│  │ Bambu Overlay    │  ← 左下          │
│  │ 印刷: Logo.3mf   │                  │
│  │ ████████░░ 82%   │                  │
│  │ 1時間24分残り    │                  │
│  │ 220°C / 60°C     │                  │
│  └──────────────────┘                  │
└─────────────────────────────────────────┘
```

### 推奨設定

| パラメーター | 推奨値 |
|-----------|--------|
| オーバーレイサイズ | 中（主張しすぎない） |
| 更新頻度 | 30 FPS（OBSに合わせる） |
| オーバーレイ位置 | 左下（顔/チャットを避ける） |
| カラーテーマ | 青いアクセントのダーク |

### シーンとシーン切り替え

カスタムOBSシーンを作成します：

- **「印刷中」** — カメラビュー + オーバーレイ
- **「一時停止 / 待機中」** — 静止画 + オーバーレイ
- **「完了」** — 結果画像 + 「完了」と表示するオーバーレイ

OBSのキーボードショートカットやScene Collectionでシーンを切り替えます。

### カメラ画像の安定化

Bambuカメラが時々フリーズすることがあります。ダッシュボードの**システム → カメラ**で：
- **自動再接続**を有効にします — ダッシュボードが自動的に再接続します
- **再接続間隔**を10秒に設定します

### 音声

3Dプリンターは音を出します — 特にAMSと冷却。検討してください：
- マイクをプリンターから離して配置します
- OBSのマイクにノイズフィルターを追加します（Noise Suppression）
- または背景音楽/チャット音声を使用します

:::tip 自動シーン切り替え
OBSにはタイトルに基づくシーン切り替えの内蔵サポートがあります。プラグイン（例：obs-websocket）とBambu Dashboard APIを組み合わせて、印刷が開始・停止したときに自動的にシーンを切り替えます。
:::
