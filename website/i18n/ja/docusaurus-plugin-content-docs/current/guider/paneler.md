# ダッシュボードのナビゲーション

このガイドでは、ダッシュボードの構成と効率的なナビゲーション方法を簡単に説明します。

## サイドパネル

左側のサイドパネルはナビゲーションセンターです。セクションで整理されています：

```
┌────────────────────┐
│ 🖨  プリンターステータス │  ← プリンターごとに1行
├────────────────────┤
│ 概要               │
│ フリート           │
│ アクティブな印刷   │
├────────────────────┤
│ フィラメント       │
│ 履歴               │
│ プロジェクト       │
│ キュー             │
│ スケジューラー     │
├────────────────────┤
│ 監視               │
│  └ Print Guard     │
│  └ エラー          │
│  └ 診断            │
│  └ メンテナンス    │
├────────────────────┤
│ 分析               │
│ ツール             │
│ 統合               │
│ システム           │
├────────────────────┤
│ ⚙ 設定            │
└────────────────────┘
```

**サイドパネルを非表示にする**には、左上のハンバーガーアイコン（☰）をクリックします。小さい画面やキオスクモードに便利です。

## メインパネル

サイドパネルの項目をクリックすると、右のメインパネルにコンテンツが表示されます。レイアウトは以下のように変わります：

| パネル | レイアウト |
|--------|-----------|
| 概要 | すべてのプリンターのカードグリッド |
| アクティブな印刷 | 大きな詳細カード + 温度グラフ |
| 履歴 | フィルタリング可能なテーブル |
| フィラメント | スプールのカードビュー |
| 分析 | グラフとダイアグラム |

## ステータスをクリックして詳細を確認する

概要パネルのプリンターカードはクリック可能です：

**シングルクリック** → そのプリンターの詳細パネルを開きます：
- リアルタイム温度
- アクティブな印刷（進行中の場合）
- すべてのスロットのAMSステータス
- 最新のエラーとイベント
- クイックボタン：一時停止、停止、ライトのオン/オフ

**カメラアイコンをクリック** → ライブカメラビューを開きます

**⚙アイコンをクリック** → プリンター設定

## キーボードショートカット — コマンドパレット

コマンドパレットでナビゲートせずにすべての機能に素早くアクセスできます：

| ショートカット | アクション |
|-------------|-----------|
| `Ctrl + K` (Linux/Windows) | コマンドパレットを開く |
| `Cmd + K` (macOS) | コマンドパレットを開く |
| `Esc` | パレットを閉じる |

コマンドパレットでできること：
- ページと機能を検索する
- 印刷を直接開始する
- アクティブな印刷を一時停止/再開する
- テーマを切り替える（ライト/ダーク）
- 任意のページに移動する

**例：** `Ctrl+K`を押して「一時停止」と入力 → 「すべてのアクティブな印刷を一時停止」を選択

## ウィジェットのカスタマイズ

概要パネルは自分で選んだウィジェットでカスタマイズできます：

**ダッシュボードの編集方法：**
1. 概要パネル右上の**レイアウトを編集**（鉛筆アイコン）をクリックします
2. ウィジェットを希望の位置にドラッグします
3. ウィジェットの角をクリックしてドラッグしてサイズを変更します
4. **+ ウィジェットを追加**をクリックして新しいものを追加します：

利用可能なウィジェット：

| ウィジェット | 表示内容 |
|-----------|---------|
| プリンターステータス | すべてのプリンターのカード |
| アクティブな印刷（大） | 進行中の印刷の詳細ビュー |
| AMS概要 | すべてのスロットとフィラメントレベル |
| 温度グラフ | リアルタイムグラフ |
| 電力価格 | 次の24時間の価格グラフ |
| フィラメントメーター | 過去30日間の総消費量 |
| 履歴ショートカット | 最新5件の印刷 |
| カメラフィード | ライブカメラ画像 |

5. **レイアウトを保存**をクリックします

:::tip 複数のレイアウトを保存する
異なる目的のために異なるレイアウトを持てます — 日常使いのコンパクトなもの、大画面に表示する大きなもの。レイアウトセレクターで切り替えます。
:::

## テーマ — ライトとダークの切り替え

**素早い切り替え：**
- ナビゲーション右上の太陽/月アイコンをクリック
- または：`Ctrl+K` → 「テーマ」と入力

**永続的な設定：**
1. **システム → テーマ**に移動します
2. 以下から選択します：
   - **ライト** — 白い背景
   - **ダーク** — 暗い背景（夜間推奨）
   - **自動** — デバイスのシステム設定に従う
3. アクセントカラーを選択します（青、緑、紫など）
4. **保存**をクリックします

## キーボードナビゲーション

マウスなしで効率的にナビゲートするには：

| ショートカット | アクション |
|-------------|-----------|
| `Tab` | 次のインタラクティブ要素 |
| `Shift+Tab` | 前の要素 |
| `Enter` / `Space` | ボタン/リンクを有効化 |
| `Esc` | モーダル/ドロップダウンを閉じる |
| `Ctrl+K` | コマンドパレット |
| `Alt+1` – `Alt+9` | 最初の9ページに直接移動 |

## PWA — アプリとしてインストール

Bambu Dashboardはプログレッシブウェブアプリ（PWA）としてインストールし、ブラウザメニューなしのスタンドアロンアプリとして実行できます：

1. Chrome、Edge、またはSafariでダッシュボードに移動します
2. アドレスバーの**アプリをインストール**アイコンをクリックします
3. インストールを確認します

詳細については[PWAドキュメント](../system/pwa)を参照してください。

## キオスクモード

キオスクモードはすべてのナビゲーションを非表示にしてダッシュボードだけを表示します — 印刷ワークショップの専用画面に最適です：

1. **システム → キオスク**に移動します
2. **キオスクモード**を有効にします
3. 表示するウィジェットを選択します
4. 更新間隔を設定します

完全な設定については[キオスクドキュメント](../system/kiosk)を参照してください。
