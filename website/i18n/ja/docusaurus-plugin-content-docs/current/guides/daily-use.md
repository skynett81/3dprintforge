# 日常的な使い方

このガイドでは、毎日の生活の中で3DPrintForgeを効率的に使う方法を説明します — 一日の始まりから終わりまで。

## 朝のルーティン

ダッシュボードを開いて、以下の点を素早く確認します：

### 1. プリンターのステータスを確認する
オーバービューパネルにすべてのプリンターのステータスが表示されます。以下を確認してください：
- **赤いアイコン** — 対応が必要なエラー
- **保留中のメッセージ** — 夜間のHMS警告
- **未完了の印刷** — 夜間印刷があった場合、完了していますか？

### 2. AMSレベルを確認する
**フィラメント**に移動するか、ダッシュボードのAMSウィジェットを確認します：
- 100g以下のスプールはありますか？交換または新規注文
- 今日の印刷に適切なフィラメントが適切なスロットに入っていますか？

### 3. アラートとイベントを確認する
**通知ログ**（ベルアイコン）では以下を確認できます：
- 夜間に発生したイベント
- 自動的に記録されたエラー
- アラームをトリガーしたHMSコード

## 印刷を開始する

### ファイルから（Bambu Studio）
1. Bambu Studioを開きます
2. モデルを読み込んでスライスします
3. プリンターに送信 — ダッシュボードが自動的に更新されます

### キューから
事前に印刷を計画している場合：
1. **キュー**に移動します
2. **次を開始**をクリックするか、ジョブをトップにドラッグします
3. **プリンターに送信**で確認します

キュー管理の詳細については[印刷キューのドキュメント](../features/queue)を参照してください。

### スケジュール印刷（スケジューラー）
特定の時間に印刷を開始するには：
1. **スケジューラー**に移動します
2. **+ 新しいジョブ**をクリックします
3. ファイル、プリンター、時間を選択します
4. **電力価格最適化**を有効にして最も安い時間帯を自動選択します

詳細については[スケジューラー](../features/scheduler)を参照してください。

## アクティブな印刷を監視する

### カメラビュー
プリンターカードのカメラアイコンをクリックします。以下ができます：
- ダッシュボードでライブフィードを見る
- バックグラウンド監視用に別のタブで開く
- 手動でスクリーンショットを撮る

### 進捗情報
アクティブな印刷カードには以下が表示されます：
- 完了率
- 推定残り時間
- 現在のレイヤー / 総レイヤー数
- アクティブなフィラメントと色

### 温度
リアルタイムの温度グラフが詳細パネルに表示されます：
- ノズル温度 — ±2°C以内で安定しているべきです
- プレート温度 — 良い密着性に重要
- チャンバー温度 — 徐々に上昇、ABS/ASAに特に関連

### Print Guard
**Print Guard**が有効な場合、ダッシュボードはスパゲッティや体積的な偏差を自動的に監視します。何かが検出された場合：
1. 印刷が一時停止されます
2. 通知を受け取ります
3. カメラ画像が後の確認のために保存されます

## 印刷後 — チェックルーティン

### 品質を確認する
1. カメラを開いてプレートにある間に結果を確認します
2. **履歴 → 最後の印刷**に移動して統計を確認します
3. メモを記録する：何がうまくいったか、何を改善できるか

### アーカイブ
履歴の印刷は自動的にアーカイブされません — そのままです。整理したい場合：
- 印刷をクリック → **アーカイブ**してアーカイブに移動
- **プロジェクト**を使用して関連する印刷をグループ化

### フィラメント重量を更新する
精度のためにスプールを計量する場合（推奨）：
1. スプールを計量します
2. **フィラメント → [スプール]**に移動します
3. **残量**を更新します

## メンテナンスリマインダー

ダッシュボードはメンテナンス間隔を自動的に追跡します。**メンテナンス**では以下が確認できます：

| タスク | 間隔 | ステータス |
|--------|------|-----------|
| ノズルクリーニング | 50時間ごと | 自動確認 |
| ロッドへの注油 | 200時間ごと | ダッシュボードで追跡 |
| プレートキャリブレーション | プレート交換後 | 手動リマインダー |
| AMSクリーニング | 月次 | カレンダー通知 |

**監視 → メンテナンス → 通知**でメンテナンスアラートを有効にします。

:::tip 週次メンテナンスデーを設定する
週に一度の固定メンテナンスデー（例：日曜日の夜）は不必要なダウンタイムを防ぎます。ダッシュボードのリマインダー機能を使用してください。
:::

## 電力価格 — 印刷に最適な時間

電力価格統合（Nordpool / Home Assistant）を接続している場合：

1. **分析 → 電力価格**に移動します
2. 次の24時間の価格グラフを確認します
3. 最安値の時間帯が緑で表示されます

**電力価格最適化**を有効にした**スケジューラー**を使用します — ダッシュボードが利用可能な最安値の時間帯に自動的にジョブを開始します。

:::info 典型的に最も安い時間帯
ノルウェーでは通常夜間（01:00〜06:00）が最安値です。前夜にキューに入れた8時間の印刷で電力コストを30〜50%節約できます。
:::
