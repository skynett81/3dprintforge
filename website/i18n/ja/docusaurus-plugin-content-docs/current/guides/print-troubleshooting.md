# 印刷失敗のトラブルシューティング

何かがうまくいかなかったですか？心配しないでください — ほとんどの印刷問題には簡単な解決策があります。3DPrintForgeが原因を素早く見つけるのに役立ちます。

## ステップ1 — HMSエラーコードを確認する

HMS（Handling, Monitoring, Sensing）はBambu Labsのエラーシステムです。すべてのエラーがダッシュボードに自動的に記録されます。

1. **監視 → エラー**に移動します
2. 失敗した印刷を見つけます
3. エラーコードをクリックして詳細な説明と提案された解決策を確認します

一般的なHMSコード：

| コード | 説明 | 素早い解決策 |
|-------|------|------------|
| 0700 1xxx | AMSエラー（詰まり、モーター問題） | AMSのフィラメントパスを確認 |
| 0300 0xxx | 押出エラー（過少/過剰押出） | ノズル清掃、フィラメント確認 |
| 0500 xxxx | キャリブレーションエラー | 再キャリブレーションを実行 |
| 1200 xxxx | 温度偏差 | ケーブル接続を確認 |
| 0C00 xxxx | カメラエラー | プリンターを再起動 |

:::tip 履歴のエラーコード
**履歴 → [印刷] → HMSログ**で、印刷中に発生したすべてのエラーコードを確認できます — 印刷が「完了」した場合でも。
:::

## 一般的な問題と解決策

### 密着不良（最初のレイヤーが付かない）

**症状：** 印刷物がプレートから剥がれる、丸まる、最初のレイヤーがない

**原因と解決策：**

| 原因 | 解決策 |
|------|--------|
| 汚れたプレート | IPA（イソプロピルアルコール）で拭く |
| 誤ったプレート温度 | 5°C上げる |
| Z-オフセットエラー | オートベッドレベリングを再実行 |
| グルースティック不足（PETG/ABS） | 薄いグルースティック層を塗布 |
| 最初のレイヤーの速度が速すぎる | 最初のレイヤーを20〜30mm/sに下げる |

**素早いチェックリスト：**
1. プレートはきれいですか？（IPA + 無毛のペーパータオル）
2. フィラメントタイプに正しいプレートを使用していますか？（[正しいプレートの選択](./choosing-plate)を参照）
3. 最後のプレート交換後にZキャリブレーションを行いましたか？

---

### 反り（角が持ち上がる）

**症状：** 特に大きくて平らなモデルで、角がプレートから上向きに曲がる

**原因と解決策：**

| 原因 | 解決策 |
|------|--------|
| 温度差 | プリンターの前面カバーを閉じる |
| ブリム不足 | Bambu Studioでブリムを有効にする（3〜5mm） |
| プレートが冷たすぎる | プレート温度を5〜10°C上げる |
| 収縮率の高いフィラメント（ABS） | Engineering Plate + チャンバー >40°C |

**ABSとASAは特に影響を受けやすいです。** 常に以下を確認してください：
- 前面カバーを閉じる
- 換気を最小限に
- Engineering Plate + グルースティック
- チャンバー温度 40°C以上

---

### ストリンギング（部品間の糸）

**症状：** モデルの独立した部品の間に細いプラスチックの糸

**原因と解決策：**

| 原因 | 解決策 |
|------|--------|
| 湿ったフィラメント | フィラメントを6〜8時間乾燥（60〜70°C） |
| ノズル温度が高すぎる | 5°C下げる |
| リトラクション不足 | Bambu Studioでリトラクション長を増やす |
| 移動速度が低すぎる | 移動速度を200+mm/sに増やす |

**湿気テスト：** パチパチ音を聞くか、押出でバブルを探す — 湿ったフィラメントを示します。Bambu AMSには湿度計が内蔵されています；**AMSステータス**で湿度を確認してください。

:::tip フィラメント乾燥機
ナイロンやTPUを使用する場合はフィラメント乾燥機（例：Bambu Filament Dryer）に投資してください — これらは12時間未満で湿気を吸収します。
:::

---

### スパゲッティ（印刷物が塊に崩れる）

**症状：** フィラメントが空中に浮いている糸、印刷物が認識できない

**原因と解決策：**

| 原因 | 解決策 |
|------|--------|
| 最初の密着不良 → 剥がれ → 崩壊 | 上の密着セクションを参照 |
| 速度が高すぎる | 速度を20〜30%下げる |
| 誤ったサポート設定 | Bambu Studioでサポートを有効にする |
| オーバーハングが急すぎる | モデルを分割するか45°回転させる |

**Print Guardを使用してスパゲッティを自動停止する** — 次のセクションを参照してください。

---

### 過少押出（薄くて弱いレイヤー）

**症状：** レイヤーが固くない、壁に穴、弱いモデル

**原因と解決策：**

| 原因 | 解決策 |
|------|--------|
| ノズルが部分的に詰まっている | コールドプル実行（メンテナンスを参照） |
| フィラメントが湿りすぎ | フィラメントを乾燥させる |
| 温度が低すぎる | ノズル温度を5〜10°C上げる |
| 速度が高すぎる | 20〜30%下げる |
| PTFEチューブが損傷 | PTFEチューブを検査して交換 |

## 自動保護のためにPrint Guardを使用する

Print Guardは画像認識でカメラ画像を監視し、スパゲッティが検出された場合に自動的に印刷を停止します。

**Print Guardの有効化：**
1. **監視 → Print Guard**に移動します
2. **自動検出**を有効にします
3. アクションを選択：**一時停止**（推奨）または**キャンセル**
4. 感度を設定します（**中**から始める）

**Print Guardが介入した場合：**
1. 検出されたものの画像付きの通知を受け取ります
2. 印刷が一時停止されます
3. 選択できます：**続行**（誤検出の場合）または**印刷をキャンセル**

:::info 誤検出
Print Guardは細い柱が多いモデルに反応することがあります。複雑なモデルでは感度を下げるか一時的に無効にしてください。
:::

## ダッシュボードの診断ツール

### 温度ログ
**履歴 → [印刷] → 温度**では、印刷全体の温度曲線を確認できます。以下を探してください：
- 急激な温度降下（ノズルまたはプレートの問題）
- 不規則な温度（キャリブレーションの必要性）

### フィラメント統計
消費されたフィラメントが推定値と一致するか確認してください。大きな差異は過少押出またはフィラメント切れを示す可能性があります。

## サポートへの連絡が必要な場合

以下の場合はBambu Labsサポートに連絡してください：
- 提案された解決策をすべて試した後もHMSコードが繰り返される
- プリンターに機械的な損傷が見える（曲がったガイド、壊れたギア）
- 温度値が不可能な値（例：ノズルが-40°Cを読む）
- ファームウェアアップデートで問題が解決しない

**サポートに用意すると便利なもの：**
- ダッシュボードのエラーログからのHMSエラーコード
- エラーのカメラ画像
- 使用されたフィラメントと設定（履歴からエクスポート可能）
- プリンターモデルとファームウェアバージョン（**設定 → プリンター → 情報**に表示）
