# 製品価格設定 — 販売価格の計算

このガイドでは、コスト計算機を使用して販売する3Dプリントの適切な販売価格を見つける方法を説明します。

## コスト概要

3Dプリントのコストは以下の要素で構成されます：

| 要素 | 説明 | 例 |
|-----------|-------------|---------|
| **フィラメント** | 重量とスプール価格に基づく材料費 | 100g × 0.25 kr/g = 25 kr |
| **廃棄物** | 材料の無駄（パージ、失敗プリント、サポート） | 10%追加 = 2.50 kr |
| **電気** | 印刷中の電力消費 | 3.5時間 × 150W × 1.50 kr/kWh = 0.79 kr |
| **摩耗** | ノズル + 機械の耐用年数にわたる価値 | 3.5時間 × 0.15 kr/時 = 0.53 kr |
| **労働** | セットアップ、後処理、梱包のための時間 | 10分 × 200 kr/時 = 33.33 kr |
| **マークアップ** | 利益率 | 20% = 12.43 kr |

**総生産コスト** = すべての要素の合計

## 設定の構成

### 基本設定

**Filament → ⚙ 設定**に移動して入力します：

1. **電気料金（kr/kWh）** — あなたの電気料金。電気代の請求書を確認するか、Nordpool統合を使用してください
2. **プリンター電力（W）** — Bambu Labプリンターの場合、通常150W
3. **機械コスト（kr）** — プリンターに支払った金額
4. **機械寿命（時間）** — 予想される耐用年数（3000-8000時間）
5. **労働コスト（kr/時間）** — あなたの時給
6. **セットアップ時間（分）** — フィラメント交換、プレート確認、梱包の平均時間
7. **マークアップ（%）** — 希望する利益率
8. **ノズルコスト（kr/時間）** — ノズルの摩耗（HS01 ≈ 0.05 kr/時）
9. **廃棄係数** — 材料の無駄（1.1 = 10%追加、1.15 = 15%）

:::tip Bambu Labの一般的な値
| 設定 | ホビイスト | セミプロ | プロフェッショナル |
|---|---|---|---|
| 電気料金 | 1.50 kr/kWh | 1.50 kr/kWh | 1.00 kr/kWh |
| プリンター電力 | 150W | 150W | 150W |
| 機械コスト | 5 000 kr | 12 000 kr | 25 000 kr |
| 機械寿命 | 3 000時間 | 5 000時間 | 8 000時間 |
| 労働コスト | 0 kr/時 | 150 kr/時 | 250 kr/時 |
| セットアップ時間 | 5分 | 10分 | 15分 |
| マークアップ | 0% | 30% | 50% |
| 廃棄係数 | 1.05 | 1.10 | 1.15 |
:::

## コストの計算

1. **コスト計算機**（`https://localhost:3443/#costestimator`）に移動します
2. `.3mf`または`.gcode`ファイルを**ドラッグ＆ドロップ**します
3. システムが自動的に読み取ります：フィラメント重量、推定時間、色
4. **スプールをリンク** — 在庫からどのスプールを使用するか選択します
5. **コストを計算**をクリックします

### 結果表示：

- **フィラメント** — 色ごとの材料費
- **廃棄/スクラップ** — 廃棄係数に基づく
- **電気** — 利用可能な場合、Nordpoolからのライブスポット価格を使用
- **摩耗** — ノズル + 機械価値
- **労働** — 時給 + セットアップ時間
- **生産コスト** — 上記すべての合計
- **マークアップ** — あなたの利益率
- **総コスト** — 最低限請求すべき金額
- **推奨販売価格** — 2×、2.5×、3×マージン

## 価格設定戦略

### 2×マージン（推奨最低値）
生産コスト + 不測の出費をカバーします。友人/家族や単純な形状に使用してください。

### 2.5×マージン（標準）
価格と価値の良いバランス。ほとんどの製品に適しています。

### 3×マージン（プレミアム）
複雑なモデル、マルチカラー、高品質、またはニッチ市場向け。

:::warning 隠れたコストを忘れないでください
- 失敗プリント（すべてのプリントの5-15%が失敗します）
- 使い切れないフィラメント（最後の50gは難しいことが多い）
- カスタマーサービスに費やす時間
- 梱包と配送
- プリンターのメンテナンス
:::

## 例：電話ホルダーの価格設定

| パラメータ | 値 |
|-----------|-------|
| フィラメント重量 | 45g PLA |
| 印刷時間 | 2時間 |
| スポット価格 | 1.20 kr/kWh |

**計算：**
- フィラメント：45g × 0.25 kr/g = 11.25 kr
- 廃棄（10%）：1.13 kr
- 電気：2時間 × 0.15kW × 1.20 = 0.36 kr
- 摩耗：2時間 × 0.15 = 0.30 kr
- 労働：（2時間 + 10分）× 200 kr/時 = 433 kr（ホビーの場合は0）
- **生産コスト（ホビー）**：約13 kr
- **販売価格 2.5×**：約33 kr

## 見積もりを保存

**見積もりを保存**をクリックして計算をアーカイブします。保存された見積もりはコスト計算機の**保存済み**タブにあります。

## Eコマース

[Eコマースモジュール](../integrations/ecommerce)を使用している場合、コスト見積もりを注文に直接リンクして自動価格計算を行うことができます。
