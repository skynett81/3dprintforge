# 電力料金

電力料金統合では、TibberまたはNordpoolからリアルタイムの電力料金を取得して、プリントごとの正確な電力コスト計算と、プリントに適した時間帯や避けるべき時間帯のアラートを提供します。

アクセス先：**https://localhost:3443/#settings** → **統合 → 電力料金**

## Tibber統合

Tibberはスポット価格のオープンAPIを持つ電力プロバイダーです。

### 設定

1. [developer.tibber.com](https://developer.tibber.com)にログインします
2. **Personal Access Token**を生成します
3. Bambu Dashboardで：**Tibber APIトークン**にトークンを貼り付けます
4. **ホーム**を選択します（複数のホームがある場合に価格を取得する場所）
5. **接続テスト**をクリックします
6. **保存**をクリックします

### Tibberから取得できるデータ

- **現在のスポット価格** — 税込みの即時価格（円/kWh）
- **次の24時間の価格** — Tibberは約13:00から翌日の価格を提供
- **価格履歴** — 過去30日分
- **プリントあたりのコスト** — 実際のプリント時間 × 時間別料金で計算

## Nordpool統合

Nordpoolは北欧の生スポット価格を提供するエネルギー取引所です。

### 設定

1. **統合 → Nordpool**に移動します
2. **価格エリア**を選択：NO1（オスロ）/ NO2（クリスティアンサン）/ NO3（トロンハイム）/ NO4（トロムソ）/ NO5（ベルゲン）
3. **通貨**を選択：NOK / EUR
4. **税金と手数料**を設定：
   - **消費税を含める**にチェック（25%）
   - **送電網料金**（円/kWh）を入力 — 送電会社の請求書を確認
   - **消費税（電力税、円/kWh）**を入力
5. **保存**をクリックします

:::info 送電網料金
送電網料金は送電会社と料金プランによって異なります。正しい料率については最新の電力請求書を確認してください。
:::

## 時間別料金

時間別料金は今後24〜48時間の棒グラフとして表示されます。

- **緑** — 安い時間帯（平均以下）
- **黄** — 平均価格
- **赤** — 高い時間帯（平均以上）
- **グレー** — 価格予測のない時間帯

時間にホバーすると正確な価格（円/kWh）が表示されます。

## 価格履歴

**電力料金 → 履歴**に移動して以下を確認します。

- 過去30日間の日次平均価格
- 1日あたりの最高・最安値の時間帯
- 1日あたりのプリントの総電力コスト

## 価格アラート

電力料金に基づく自動アラートを設定します。

1. **電力料金 → 価格アラート**に移動します
2. **新しいアラート**をクリックします
3. アラートタイプを選択します：
   - **価格が上限以下** — 電力料金がX円/kWh未満になったときに通知
   - **価格が上限以上** — 価格がX円/kWhを超えたときに通知
   - **今日の最安値時間帯** — 今日の最安値時間帯が始まったときに通知
4. 通知チャンネルを選択します
5. **保存**をクリックします

:::tip スマートプランニング
価格アラートとプリントキューを組み合わせます：電力料金が低いときにキュージョブを自動送信する自動化を設定します（WebhookまたはHome Assistant統合が必要）。
:::

## コスト計算機での電力料金

電力料金統合を有効にすると、[コスト計算機](../analyse/costestimator)で正確な電力コストが得られます。現在のTibber/Nordpool価格を使用するために、固定料金の代わりに**ライブ料金**を選択します。
