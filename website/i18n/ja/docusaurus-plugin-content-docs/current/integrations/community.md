# コミュニティフィラメント

コミュニティフィラメントは、世界中のBambu Dashboardユーザーが投稿したフィラメントプロファイルの共有データベースです。お使いのフィラメントに最適な設定を見つけて、ご自身の経験を共有しましょう。

アクセス先：**https://localhost:3443/#community**

## コミュニティプロファイルを閲覧する

### 検索

1. **コミュニティフィラメント**に移動します
2. 以下で検索するために検索フィールドを使用します：
   - 素材（PLA、PETG、ABS、PA、TPUなど）
   - メーカー（PolyMaker、Bambu Lab、Prusament、Sunluなど）
   - カラー名またはカラーHEX
   - 特性（透明、マット、シルク、グリッターなど）
3. **評価**でフィルタリング（最高評価のみ）
4. **プリンターモデル**でフィルタリングして自分のプリンターに最適化されたプロファイルを表示

### プロファイルカード

各プロファイルカードには以下が表示されます。

| フィールド | 例 |
|---|---|
| フィラメント名 | PolyTerra PLA – Matte Forest Green |
| 素材 | PLA |
| メーカー | PolyMaker |
| 投稿者 | ユーザー名 |
| 評価 | ⭐⭐⭐⭐½ (4.6 / 5) |
| テスト数 | 23 |
| 推奨温度 | ノズル：215°C / ベッド：55°C |
| 推奨速度 | 標準 |

カードをクリックして、すべてのパラメーターとユーザーコメントを含む完全なプロファイルを表示します。

## プロファイルをインポートする

1. コミュニティからプロファイルを開きます
2. **在庫にインポート**をクリックします
3. 以下を選択します：
   - **既存のスプールにリンク** — プロファイルが登録済みスプールにリンクされます
   - **新しいスプールを作成** — プロファイルデータで新しいスプールエントリーを作成
4. 必要に応じて温度やその他の設定を調整します
5. **インポート**をクリックします

プロファイルがフィラメント在庫に追加されます。

## 自分のプロファイルを投稿する

### エクスポートと共有

1. **フィラメント在庫**に移動します
2. 共有するスプールを見つけます
3. **コミュニティと共有**をクリックします
4. 以下を入力します：
   - **メーカー名** — 正確なメーカー名
   - **製品名** — パッケージから正確な製品名
   - **カラー名** — 公式カラー名とカラーHEX
   - **コメント** — あなたの経験、ヒントとトリック
   - **プリンターモデル** — テストしたプリンター
5. **送信**をクリックします

プロファイルは審査中で、通常24時間以内に公開されます。

### 投稿ガイドライン

:::warning 品質要件
- 実際にテストした設定のみを提供してください — 推測値は不可
- 正確なメーカー名と製品名を使用してください（パッケージを確認）
- 特定の製品とプリンターモデルごとに1つのプロファイル
- 重複プロファイルを避けてください — プロファイルが既に存在するか確認
:::

## プロファイルを評価する

テストしたプロファイルを評価してコミュニティを助けましょう。

1. コミュニティプロファイルを開きます
2. **評価を付ける**をクリックします
3. 星の数（1〜5）と短いコメントを入力します
4. **送信**をクリックします

評価の高いプロファイルが検索結果の上位に表示されます。

## オフライン使用

コミュニティデータベースはローカルに自動同期されます（毎日更新）。最初の同期後はすべてのプロファイルがインターネット接続なしで利用可能です。

データベースを手動でダウンロード：**コミュニティフィラメント → 設定 → 今すぐ同期**
