# クールプレート（スムースPEI）

クールプレートはプリントに光沢のある滑らかな底面を提供するスムースPEIプレートです。ほとんどのBambu Labプリンターに標準プレートとして同梱されています。

## 特性

- **表面：** プリントに滑らかで光沢のある底面
- **素材：** 弾性鋼板上のPEI（ポリエーテルイミド）コーティング
- **最大ベッド温度：** 90°C

## 最適な素材

- PLA（すべてのバリアント）— グルースティック不要
- PLA+ — グルースティック不要
- PLAシルク — グルースティック不要
- PLA-CF — グルースティック不要

## 注意が必要な使用

:::danger PETG — グルースティックは必ず使用すること
PETGはスムースPEIに**非常に強く**定着します。グルースティックなしでは、プリントを取り外す際にPEIコーティングがプレートから剥がれるリスクがあります。グルースティックをバリアーとして使用するか、エンジニアリングプレートを選択してください。
:::

## メンテナンス

### プリントごと
- プリントを取り外す前にプレートを室温（40°C以下）まで冷却させる
- PLAは25〜30°Cで自然にプレートから外れます
- 頑固なプリントはプレートをそっと曲げて外す

### 週次クリーニング
イソプロパノール（IPA、70〜99%）でプレートを拭きます：
```
1. クリーンなほこりの出ない紙にIPAを付ける（プレートに直接スプレーしない）
2. 一方向に表面を拭く
3. 次のプリントの前に完全に乾燥させる（30秒）
```

### 定着不良の場合
1. お湯と中性洗剤で洗う
2. 十分にすすぐ
3. ほこりの出ない紙で拭く
4. 最後にIPAで拭く

:::warning アセトンを使用しないこと
アセトンはPEIコーティングを溶かしてプレートを永久に損傷させます。IPAまたは水/石けんのみを使用してください。
:::

## プレートの交換

プレートが消耗したサイン：
- PEIコーティングに目に見える損傷がある（傷、へこみ、剥がれ）
- 徹底的なクリーニング後もプリントが定着しない
- 表面が不均一で良いファーストレイヤーが得られない

クールプレートは通常の使用と適切なメンテナンスで一般的に200〜500プリント持続します。
