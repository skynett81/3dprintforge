# エンジニアリングプレート（テクスチャードPEI）

エンジニアリングプレートはPETG、ABS、ASA、PAなどの技術的な素材向けに設計されたテクスチャードPEIプレートです。細かいテクスチャーの底面を提供し、Bambu Labのラインナップで最も汎用性の高いプレートです。

## 特性

- **表面：** テクスチャードPEI — プリントにパターン/マットな底面を提供
- **定着：** スムースPEIより優れた機械的定着
- **最大ベッド温度：** 110°C
- **取り外し：** 40〜50°Cに冷却するとプリントが簡単に外れます

## 最適な素材

- PETGとPETG-CF（グルースティックなし、または安全のために少量）
- ABSとASA（グルースティックあり）
- PA6 / PA12 / PA-CF（グルースティックあり — 必須）
- TPU（グルースティックなし）
- PLA（良好な定着、テクスチャーの底面になります）

## 適さない素材

- 滑らかな底面が必要なプリント（クールプレートを使用）
- 高温でのPC（高温プレートを使用）

## メンテナンス

### クリーニング
```bash
# 週次（またはグルースティック使用後）：
1. お湯と中性洗剤で洗う
2. 十分にすすぐ
3. 乾燥させるか、ほこりの出ない紙で拭く
4. 最後にIPAで拭く

# グルースティックなしのプリント間：
1. IPAとほこりの出ない紙で拭く
```

### グルースティック後
グルースティックを使用した後は必ずプレートを洗ってください。残ったグルースティックが次のプリントの不均一な定着の原因になることがあります。

:::tip テクスチャーの向き
エンジニアリングプレートにはテクスチャーの向きがあります。テクスチャーのくぼみにプリントが引っかかることを避けるために、常にクリップを正しい向きにプレートを取り付けてください。
:::

## プレートの交換

エンジニアリングプレートが消耗したサイン：
- 中央ゾーン（最も使用される部分）のテクスチャーが摩耗して滑らかになっている
- クリーニングとグルースティックの後もプリントが定着しない
- プレートに深い傷や変形がある

適切なメンテナンスで通常の寿命：300〜700プリント。
