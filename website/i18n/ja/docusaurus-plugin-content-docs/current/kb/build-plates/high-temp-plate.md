# 高温プレート

高温プレートは高いベッド温度が必要な素材向けに設計されています。最大120°Cに対応し、ポリカーボネート（PC）とPAの高温バリアント向けに必要です。

## 特性

- **表面：** 高温コーティング付きスムース（滑らか）
- **最大ベッド温度：** 120°C
- **プリントの底面：** 光沢/滑らか
- **定着：** 高温で良好な定着 — 冷却時に外れる

## 最適な素材

| 素材 | ベッド温度 | グルースティック |
|-----------|---------------|---------|
| ABS | 90〜110°C | 必要 |
| ASA | 90〜110°C | 必要 |
| PC | 100〜120°C | 必要（必須） |
| PA6 | 80〜90°C | 必要 |
| PA12 | 70〜85°C | 必要 |
| PA-CF | 90〜100°C | 必要 |

## 適さない素材

- PLA — プレートが高温すぎて定着しすぎる
- PETG — グルースティックなしではプレート損傷のリスク
- TPU — エンジニアリングプレートの方が適している

## グルースティック

グルースティックは強く推奨（PCには必須）です：
1. ファーストレイヤーの定着を改善する
2. プレートのコーティングを保護する
3. 冷却後のプリントの取り外しを容易にする

薄く均一に塗布してください。PCとPAにはABSよりも少し多く使用します。

:::warning PCには特別な注意が必要
ポリカーボネート（PC）はノズル250〜280°C、ベッド100〜120°Cでプリントします。エンクロージャー（チャンバー）と高温プレートなしでは成功率が非常に低くなります。PCプリントにはX1CとP1Sのみを推奨します。
:::

## メンテナンス

```bash
# グルースティック付きPC/ABS後：
1. プレートを50°Cまで冷却させる
2. プリントをそっと取り外す（プレートを曲げる）
3. お湯と洗剤で洗う
4. 十分に乾燥させる — 熱いプレートに水が付くとコーティングが損傷することがある

# プリント間：
1. IPAで拭く
2. コーティングの摩耗を確認する
```

:::danger 熱衝撃を避けること
冷たいプレートを熱いベッドに置いたり、熱いプレートに冷たいIPAをスプレーしたりしないでください。温度差はコーティングを損傷させたり鋼板の変形を引き起こすことがあります。
:::

## 寿命

高温プレートは高い熱負荷のためエンジニアリングプレートより短命です：
- **通常の使用（ABS/ASA）：** 200〜400プリント
- **集中的な使用（PC）：** 100〜200プリント

コーティングに目に見える摩耗が見られる場合、または定着問題が継続する場合は交換してください。
