# ビルドプレート概要

Bambu Labはそれぞれ異なる素材と用途に最適化された4種類のビルドプレートを提供しています。失敗したプリントとプレートの損傷を避けるために正しいプレートを選んでください。

## プレート概要

### クールプレート（スムースPEI）
滑らかなPEIプレートで、プリントに光沢のある均一な底面を提供します。PLAとPLAバリアントに最適です。**グルースティックなしでPETGを使用しないでください** — PETGが強く定着してコーティングが剥がれることがあります。

[詳細を読む →](./cool-plate)

### エンジニアリングプレート（テクスチャードPEI）
技術的な素材に対してより良い定着を持つテクスチャードPEIです。プリントに細かいパターンの底面を提供します。最も汎用性の高いプレートです。

[詳細を読む →](./engineering-plate)

### 高温プレート
高温度要件の素材向け：ABS、ASA、PC、PA。最大120°Cのベッド温度に対応します。

[詳細を読む →](./high-temp-plate)

### テクスチャードPEIプレート
粗いテクスチャーを提供します。TPUや機械的な定着が必要な素材に優秀です。冷却後にプリントが簡単に外れます。

[詳細を読む →](./textured-pei)

## グルースティックの使用

グルースティックには2つの目的があります：
1. **バリアー**（スムースPEI上のPETG）— 強すぎる定着とプレートの損傷を防ぐ
2. **定着の改善**（すべてのプレートでABS、PA）— 素材とプレートの接触を改善

**推奨グルースティック：** Bambu Lab純正品、または標準のプリットスティック。薄く均一に塗布して30秒乾燥させてください。

:::tip グルースティック後のクリーニング
グルースティックを使用した後はぬるま湯と中性洗剤でプレートを洗ってください。次のプリントの前に完全に乾燥させてください。
:::
