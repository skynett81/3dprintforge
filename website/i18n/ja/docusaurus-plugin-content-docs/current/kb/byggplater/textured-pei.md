# テクスチャードPEIプレート

テクスチャードPEIプレート（エンジニアリングプレートと混同しないでください）はより粗いテクスチャーを持ち、TPUやその他の柔軟な素材、また一般的な様々な素材に特に適しています。

## 特性

- **表面：** 粗いPEIテクスチャー — 際立ったテクスチャーの底面を提供
- **定着：** 優秀な機械的グリップ、特にTPUに
- **取り外し：** 冷却後に非常に簡単 — プレートを曲げるとプリントが外れる
- **最大ベッド温度：** 100°C

## 最適な素材

- **TPU** — これが柔軟な素材に最適なプレートです
- PLAとPLA+ — 良好な定着、細かいテクスチャーの底面
- PETG — 少量のグルースティックを使用
- PLA-CF — グルースティックなし

## 互換性

| 素材 | 適合度 | グルースティック |
|-----------|---------|---------|
| TPU | ★★★★★ | 不要 |
| PLA | ★★★★★ | 不要 |
| PLA+ | ★★★★★ | 不要 |
| PETG | ★★★★ | 少量（推奨） |
| PLA-CF | ★★★★ | 不要 |
| ABS | ★★★ | 必要 |
| PA12 | ★★★ | 必要 |

## テクスチャードPEI上のTPU

TPUはテクスチャードPEIにグルースティックなしで完璧に定着します：
1. ベッド温度を30〜45°Cに設定する
2. Zオフセットを慎重に調整 — PLAより少し高め
3. プリント後にベッドが30°C未満まで冷却させる
4. プレートをそっと曲げる — TPUが力なく外れます

:::tip 冷却後の取り外し
TPUはテクスチャードPEIから冷却時に優秀に外れます。プリントの取り外しに問題がある場合は、プレートが完全に冷えるまで（20〜25°C）待ってから強く曲げてください。
:::

## メンテナンス

テクスチャードPEIは丈夫でメンテナンスが簡単です：

```bash
# プリント間：
- IPAとほこりの出ない紙で拭く

# 月次（または定着不良時）：
- お湯と中性洗剤で洗う
- すすいで十分に乾燥させる
- 最後にIPAで処理する
```

## 完成プリントの表面

粗いテクスチャーはすべてのプリントの裏面に独特のパターンを提供します。これは：
- プロトタイプや機能部品に望ましい（より良いグリップ）
- 装飾/ビジュアルプリントには望ましくない（滑らかな表面にはクールプレートを使用）
