# 定着不良

定着不良は3Dプリントで最も一般的な問題の1つです。ファーストレイヤーが定着しないか、プリントが途中で外れます。

## 症状

- ファーストレイヤーが定着しない — プリントが動いたり持ち上がったりする
- エッジとコーナーが持ち上がる（反り）
- プリントがジョブの途中で外れる
- 穴または緩い糸のある不均一なファーストレイヤー

## チェックリスト — この順序で試してください

### 1. プレートをクリーニングする
定着不良の最も一般的な原因はプレートの油脂または汚れです。

```
1. IPA（イソプロピルアルコール）でプレートを拭く
2. 素手でプリント表面を触らない
3. 問題が続く場合：水と中性洗剤で洗う
```

### 2. Zオフセットを調整する

Zオフセットはファーストレイヤーでのノズルとプレート間の高さです。高すぎる = 糸が浮く。低すぎる = ノズルがプレートをこする。

**正しいZオフセット：**
- ファーストレイヤーは軽く透明に見えるはず
- 糸はプレートに少し「押し付け」られるはず
- 糸は互いに軽く溶け込むはず

プリント中に**コントロール → ライブZ調整**でZオフセットを調整します。

:::tip プリント中にライブ調整
Bambu Dashboardはアクティブなプリント中にZオフセット調整ボタンを表示します。ファーストレイヤーを見ながら±0.02 mmのステップで調整します。
:::

### 3. ベッド温度を確認する

| 素材 | 低すぎる温度 | 推奨 |
|-----------|-------------|---------|
| PLA | 30°C未満 | 35〜45°C |
| PETG | 60°C未満 | 70〜85°C |
| ABS | 80°C未満 | 90〜110°C |
| TPU | 25°C未満 | 30〜45°C |

ベッド温度を5°Cずつ上げてみてください。

### 4. グルースティックを使用する

グルースティックはほとんどのプレートでほとんどの素材の定着を改善します：
- 薄く均一に塗布する
- 開始前に30秒乾燥させる
- 特に重要：ABS、PA、PC、PETG（スムースPEI上）

### 5. ファーストレイヤー速度を下げる

ファーストレイヤーでの低速はフィラメントとプレートの接触を改善します：
- 標準：ファーストレイヤーで50 mm/s
- 試す：30〜40 mm/s
- Bambu Studio：**品質 → ファーストレイヤー速度**の下

### 6. プレートの状態を確認する

摩耗したプレートは完璧な設定でも定着が悪くなります。次の場合はプレートを交換します：
- PEIコーティングに目に見える損傷がある
- クリーニングが効果なし

### 7. ブリムを使用する

反りの傾向がある素材に（ABS、PA、大きな平らなオブジェクト）：
- スライサーにブリムを追加：幅5〜10 mm
- 接触面積を増やしてエッジを保持する

## 特別なケース

### 大きな平らなオブジェクト
大きな平らなオブジェクトは最も外れやすいです。対策：
- ブリム8〜10 mm
- ベッド温度を上げる
- チャンバーを閉じる（ABS/PA）
- パーツ冷却を下げる

### 光沢のある表面
時間とともに多すぎるグルースティックでプレートが光沢になることがあります。水でよく洗い直してください。

### フィラメント交換後
異なる素材は異なる設定が必要です。新しい素材のベッド温度とプレートが設定されていることを確認してください。
