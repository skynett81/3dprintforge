# 反り

反りはプリント中またはプリント後に、プリントの角やエッジがプレートから浮き上がる現象です。素材の熱収縮によって引き起こされます。

## 反りとは？

プラスチックが冷却されると収縮します。上部の層は下部の層より温かい — これがエッジを上に引っ張り、プリントを曲げる応力を生み出します。温度差が大きいほど、反りも大きくなります。

## 最も影響を受けやすい素材

| 素材 | 反りリスク | チャンバー必要性 |
|-----------|-------------|-----------------|
| PLA | 低 | 不要 |
| PETG | 低〜中 | 不要 |
| ABS | 高 | 必要 |
| ASA | 高 | 必要 |
| PA/ナイロン | 非常に高い | 必要 |
| PC | 非常に高い | 必要 |
| TPU | 低 | 不要 |

## 解決策

### 1. チャンバー（密閉空間）を使用する

ABS、ASA、PA、PCに最も重要な対策：
- チャンバー温度40〜55°Cが最良の結果をもたらす
- X1CとP1S：チャンバーファンを「密閉」モードに設定
- A1/P1P：カバーを使用して熱を保持する

### 2. ブリムを使用する

ブリムはプリントをプレートに固定する幅広い単一層の追加エッジです：

```
Bambu Studio：
1. スライサーでプリントを選択
2. サポート → ブリムに移動
3. 幅を5〜10 mmに設定（反りが大きいほど広く）
4. タイプ：外部ブリムのみ（推奨）
```

:::tip ブリム幅ガイド
- PLA（ほとんど不要）：3〜5 mm
- PETG：4〜6 mm
- ABS/ASA：6〜10 mm
- PA/ナイロン：8〜15 mm
:::

### 3. ベッド温度を上げる

高いベッド温度は層間の温度差を減らします：
- ABS：105〜110°Cを試す
- PA：85〜95°C
- PETG：80〜85°C

### 4. パーツ冷却を下げる

反りの傾向がある素材には — パーツ冷却を下げるか無効にします：
- ABS/ASA：0〜20% パーツ冷却
- PA：0〜30% パーツ冷却

### 5. 気流と冷たい空気を避ける

プリンターを以下から遠ざけてください：
- 窓と外ドア
- エアコンとファン
- 部屋の気流

P1PとA1の場合：重要なプリント中は段ボールで開口部を覆う。

### 6. ドラフトシールド

ドラフトシールドはオブジェクトの周りに熱を保持する薄い壁です：

```
Bambu Studio：
1. サポート → ドラフトシールドに移動
2. 有効にして距離を設定（3〜5 mm）
```

特に高くて細いオブジェクトに有効です。

### 7. モデル設計上の対策

独自モデルを設計する際：
- 大きな平らな底面を避ける（角にチャンファー/丸みを追加）
- 大きな平らな部品を小さなセクションに分割する
- スライサーまたはCADで「マウスイヤー」（角の小さな円）を使用する

## 冷却後の反り

プリントが問題なく見えても、プレートから取り外した後に反りが発生することがあります：
- プレートとプリントが**完全に冷却**する（40°C未満）まで常に待ってから取り外す
- ABSの場合：ゆっくり冷却するために密閉チャンバー内で冷却させる
- 温かいプリントを冷たい表面に置かない
