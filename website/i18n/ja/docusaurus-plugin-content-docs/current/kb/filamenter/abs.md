# ABS

ABS（アクリロニトリル・ブタジエン・スチレン）は優れた熱安定性と耐衝撃性を持つ熱可塑性プラスチックです。エンクロージャーが必要でPLA/PETGより難しいですが、耐久性のある機能的なパーツが作れます。

## 設定

| パラメーター | 値 |
|-----------|-------|
| ノズル温度 | 240〜260°C |
| ベッド温度 | 90〜110°C |
| チャンバー温度 | 45〜55°C（X1C/P1S） |
| パーツ冷却 | 0〜20% |
| 補助ファン | 0% |
| 速度 | 80〜100% |
| 乾燥 | 推奨（70°Cで4〜6時間） |

## 推奨ビルドプレート

| プレート | 適合度 | グルースティック？ |
|-------|---------|----------|
| エンジニアリングプレート（テクスチャードPEI） | 優秀 | 必要（推奨） |
| 高温プレート | 優秀 | 必要 |
| クールプレート（スムースPEI） | 避ける | — |
| テクスチャードPEI | 良好 | 必要 |

:::tip ABSにグルースティックを使用する
ABSをプリントする際はエンジニアリングプレートに必ずグルースティックを使用してください。定着が改善され、プレートを傷めずにプリントを外しやすくなります。
:::

## エンクロージャー（チャンバー）

ABSは反りを防ぐために閉じたチャンバーが**必要**です：

- **X1CとP1S：** アクティブな温度制御付き内蔵チャンバー — ABSに最適
- **P1P：** 部分的にオープン — より良い結果のためにトップカバー（トップ上のカバー）を追加
- **A1 / A1 Mini：** オープンCoreXY — カスタムエンクロージャーなしではABSに**非推奨**

プリント中はチャンバーを閉じたままにしてください。プリントを確認するために開けないでください — 冷却まで待てば、解放時の反りも防げます。

## 反り

ABSは反り（コーナーが持ち上がる）が非常に起きやすいです：

- **ベッド温度を上げる** — 105〜110°Cを試す
- **ブリムを使用する** — Bambu Studioで5〜10 mmのブリム
- **気流を避ける** — プリンター周囲の気流をすべて遮断
- **パーツ冷却を0%に下げる** — 冷却が反りの原因になります

:::warning 蒸気
ABSはプリント中にスチレン蒸気を放出します。部屋の換気を十分に行うか、HEPAまたは活性炭フィルターを使用してください。Bambu P1Sには内蔵フィルターがあります。
:::

## 後処理

ABSはPETGやPLAより研磨、塗装、接着が容易です。アセトンで蒸気磨きして滑らかな表面にすることもできますが、アセトンの取り扱いには十分注意してください。

## 保管

プリント前に**70°Cで4〜6時間**乾燥させてください。密閉ボックスで保管 — ABSは吸湿してポップ音と弱い層の原因になります。
