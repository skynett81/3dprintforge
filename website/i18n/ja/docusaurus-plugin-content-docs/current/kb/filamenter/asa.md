# ASA

ASA（Acrylonitrile Styrene Acrylate）は、屋外使用向けに特別に開発されたABSのUV耐性バリアントです。ABSの強度と剛性に加え、UV放射、経年劣化、気象の影響に対する大幅に優れた耐性を備えています。

## 設定

| パラメータ | 値 |
|-----------|-------|
| ノズル温度 | 240–260 °C |
| ベッド温度 | 90–110 °C |
| チャンバー温度 | 40–50 °C（推奨） |
| パーツ冷却 | 30–50% |
| 速度 | 80–100% |
| 乾燥が必要 | はい |

## 推奨ビルドプレート

| プレート | 適合性 | のり? |
|-------|---------|----------|
| Engineering Plate | 優秀 | いいえ |
| High Temp Plate | 良好 | はい |
| Textured PEI | 許容可 | はい |
| Cool Plate (Smooth PEI) | 非推奨 | — |

:::tip Engineering PlateがASAに最適
Engineering PlateはのりなしでもASAに最も信頼性の高い定着を提供します。高いベッド温度に耐え、パーツが永久的に固着することなく良好な接着力を発揮します。
:::

## プリンターの要件

ASAは最良の結果を得るために**密閉チャンバー（エンクロージャー）**が必要です。エンクロージャーがないと以下の問題が発生します：

- **反り** — 角がビルドプレートから持ち上がる
- **層間剥離** — 層間の接着不良
- **表面のひび割れ** — プリントに沿った目に見えるひび割れ

| プリンター | ASAに適していますか？ | 備考 |
|---------|---------------|---------|
| X1C | 優秀 | 完全密閉、アクティブ加熱 |
| X1E | 優秀 | 完全密閉、アクティブ加熱 |
| P1S | 良好 | 密閉、パッシブ加熱 |
| P1P | アクセサリーで可能 | エンクロージャーアクセサリーが必要 |
| A1 | 非推奨 | オープンフレーム |
| A1 Mini | 非推奨 | オープンフレーム |

## ASA vs ABS — 比較

| 特性 | ASA | ABS |
|----------|-----|-----|
| UV耐性 | 優秀 | 劣る |
| 屋外使用 | はい | いいえ（黄変・脆化） |
| 反り | 中程度 | 高い |
| 表面 | マット、均一 | マット、均一 |
| 耐薬品性 | 良好 | 良好 |
| 価格 | やや高い | 低い |
| プリント中の臭い | 中程度 | 強い |
| 衝撃強度 | 良好 | 良好 |
| 耐熱温度 | ~95–105 °C | ~95–105 °C |

:::warning 換気
ASAはプリント中に刺激性のガスを発生します。換気の良い部屋またはエアフィルターシステムのある場所でプリントしてください。換気のない部屋で長時間滞在しながらASAをプリントしないでください。
:::

## 乾燥

ASAは**中程度の吸湿性**を持ち、時間の経過とともに空気中の湿気を吸収します。

| パラメータ | 値 |
|-----------|-------|
| 乾燥温度 | 65 °C |
| 乾燥時間 | 4–6時間 |
| 吸湿レベル | 中程度 |
| 湿気の兆候 | パチパチ音、気泡、表面品質の低下 |

- 開封後はシリカゲル入りの密封袋で保管
- 乾燥剤入りAMSは短期保管に十分
- 長期保管：真空バッグまたはフィラメント乾燥ボックスを使用

## 用途

ASAは**屋外**で使用するすべてのものに最適な素材です：

- **自動車部品** — ミラーハウジング、ダッシュボードディテール、バルブキャップ
- **ガーデンツール** — クリップ、クランプ、ガーデン家具パーツ
- **屋外サイン** — サイン、文字、ロゴ
- **ドローン部品** — ランディングギア、カメラマウント
- **ソーラーパネルマウント** — ブラケットとアングル
- **郵便受け部品** — 機構と装飾

## ASAプリント成功のヒント

### ブリムと定着

- **大きなパーツ**や接触面積の小さいパーツにはブリムを使用
- 5–8 mmのブリムで効果的に反りを防止
- 小さなパーツはブリムなしで試せますが、バックアップとして準備しておく

### すきま風を避ける

- プリント中は部屋の**すべてのドアと窓を閉める**
- すきま風と冷気はASAの最大の敵
- プリント中にチャンバードアを開けない

### 温度の安定性

- プリント開始前にチャンバーを**10–15分**予熱する
- 安定したチャンバー温度でより均一な結果が得られる
- プリンターを窓や換気口の近くに置かない

### 冷却

- ASAは**制限されたパーツ冷却**が必要 — 30–50%が一般的
- オーバーハングやブリッジでは60–70%まで上げられるが、若干の層間剥離が予想される
- 機械部品：冷却を下げてディテールよりも層間接着を優先

:::tip 初めてのASA？
設定を校正するために小さなテストパーツ（例：30 mmのキューブ）から始めてください。ASAはABSに非常に似た挙動を示しますが、反りの傾向がやや低くなります。ABSの経験があれば、ASAはアップグレードに感じるでしょう。
:::

## 収縮

ASAはPLAやPETGよりも収縮が大きいですが、一般的にABSよりもやや小さいです：

| 素材 | 収縮率 |
|-----------|----------|
| PLA | ~0.3–0.5% |
| PETG | ~0.3–0.6% |
| ASA | ~0.5–0.7% |
| ABS | ~0.7–0.8% |

精密な公差が必要なパーツ：スライサーで0.5–0.7%の補正を行うか、テストパーツで最初に確認してください。

## 後処理

- **アセトン蒸気仕上げ** — ASAはABSと同様にアセトン蒸気で表面を滑らかにできる
- **研磨** — 200–400番のサンドペーパーでよく研磨できる
- **接着** — 瞬間接着剤またはアセトン接着が優秀
- **塗装** — 軽い研磨後に塗料がよく乗る

:::danger アセトンの取り扱い
アセトンは可燃性で有毒ガスを発生します。必ず換気の良い部屋で使用し、裸火を避け、保護具（手袋とゴーグル）を着用してください。
:::
