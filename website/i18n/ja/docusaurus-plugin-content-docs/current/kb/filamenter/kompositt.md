# コンポジット素材（CF/GF）

コンポジットフィラメントには、PLA、PETG、PA、ABSなどのベースプラスチックに混合された短いカーボンファイバー（CF）またはガラスファイバー（GF）が含まれています。剛性の向上、重量の軽減、優れた寸法安定性を提供します。

## 利用可能なタイプ

| フィラメント | ベース | 剛性 | 軽量化 | 難易度 |
|----------|-------|---------|--------------|------------------|
| PLA-CF | PLA | 高 | 中程度 | 容易 |
| PETG-CF | PETG | 高 | 中程度 | 普通 |
| PA6-CF | ナイロン6 | 非常に高い | 良好 | 難しい |
| PA12-CF | ナイロン12 | 非常に高い | 良好 | 普通 |
| ABS-CF | ABS | 高 | 中程度 | 普通 |
| PLA-GF | PLA | 高 | 中程度 | 容易 |

## 硬化スチールノズルが必須

:::danger CF/GFに真鍮ノズルを絶対に使用しないこと
カーボンファイバーとガラスファイバーは非常に研磨性が高いです。標準の真鍮ノズルを数時間から数日で摩耗させます。すべてのCFおよびGF素材には必ず**硬化スチールノズル**（Hardened Steel）または**HS01ノズル**を使用してください。

- Bambu Lab 硬化スチールノズル（0.4 mm）
- Bambu Lab HS01ノズル（特殊コーティング、より長い寿命）
:::

## 設定（PA-CFの例）

| パラメーター | 値 |
|-----------|-------|
| ノズル温度 | 270〜290°C |
| ベッド温度 | 80〜100°C |
| パーツ冷却 | 0〜20% |
| 速度 | 80% |
| 乾燥 | 80°C / 12時間 |

PLA-CFの場合：ノズル220〜230°C、ベッド35〜50°C — PA-CFよりはるかに容易。

## ビルドプレート

| プレート | 適合度 | グルースティック？ |
|-------|---------|----------|
| エンジニアリングプレート（テクスチャードPEI） | 優秀 | 必要（PAベースの場合） |
| 高温プレート | 良好 | 必要 |
| クールプレート | 避ける（CFで傷つく） | — |
| テクスチャードPEI | 良好 | 必要 |

:::warning プレートに傷がつく可能性
CF素材は取り外し時に滑らかなプレートに傷をつけることがあります。常にエンジニアリングプレートまたはテクスチャードPEIを使用してください。プリントを引っ張らず — プレートをそっと曲げて外してください。
:::

## 表面処理

CFフィラメントはマットなカーボンのような表面を提供し、塗装は不要です。表面はやや多孔質で、より滑らかな仕上げのためにエポキシを含浸させることができます。

## 消耗とノズルの寿命

| ノズルタイプ | CFでの寿命 | コスト |
|----------|---------------|---------|
| 真鍮（標準） | 数時間〜数日 | 低 |
| 硬化スチール | 200〜500時間 | 中程度 |
| HS01（Bambu） | 500〜1000時間 | 高 |

目に見える消耗（拡大したノズル穴、薄い壁、寸法精度の低下）でノズルを交換してください。

## 乾燥

PAおよびPETGのCFバリアントはベースと同様に乾燥が必要です：
- **PLA-CF：** 乾燥推奨、ただし必須ではない
- **PETG-CF：** 65°C / 6〜8時間
- **PA-CF：** 80°C / 12時間 — 必須
