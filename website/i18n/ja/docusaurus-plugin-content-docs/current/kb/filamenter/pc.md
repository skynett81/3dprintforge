# PC（ポリカーボネート）

ポリカーボネートは、FDMプリントで使用できる最も強力な熱可塑性プラスチックの一つです。極めて高い衝撃強度、110–130 °Cまでの耐熱性、そして天然の透明性を兼ね備えています。PCはプリントが難しい素材ですが、射出成形部品に近い品質の結果が得られます。

## 設定

| パラメータ | 純PC | PC-ABSブレンド | PC-CF |
|-----------|--------|-------------|-------|
| ノズル温度 | 260–280 °C | 250–270 °C | 270–290 °C |
| ベッド温度 | 100–120 °C | 90–110 °C | 100–120 °C |
| チャンバー温度 | 50–60 °C（必須） | 45–55 °C | 50–60 °C |
| パーツ冷却 | 0–20% | 20–30% | 0–20% |
| 速度 | 60–80% | 70–90% | 50–70% |
| 乾燥が必要 | はい（重要） | はい | はい（重要） |

## 推奨ビルドプレート

| プレート | 適合性 | のり? |
|-------|---------|----------|
| High Temp Plate | 優秀（必須） | いいえ |
| Engineering Plate | 許容可 | はい |
| Textured PEI | 非推奨 | — |
| Cool Plate (Smooth PEI) | 使用不可 | — |

:::danger High Temp Plateが必須
PCは100–120 °Cのベッド温度が必要です。Cool PlateとTextured PEIはこの温度に耐えられず損傷します。純PCには**常に**High Temp Plateを使用してください。
:::

## プリンターと機器の要件

### エンクロージャー（必須）

PCは50–60 °Cの安定した温度を持つ**完全密閉チャンバー**が必要です。これがないと、深刻な反り、層間剥離、デラミネーションが発生します。

### 硬化ノズル（強く推奨）

純PCは研磨性はありませんが、PC-CFおよびPC-GFには**硬化スチールノズル**（例：Bambu Lab HS01）が必要です。純PCでも高温のため硬化ノズルが推奨されます。

### プリンター互換性

| プリンター | PCに適していますか？ | 備考 |
|---------|--------------|---------|
| X1C | 優秀 | 完全密閉、HS01対応 |
| X1E | 優秀 | エンジニアリング素材向け設計 |
| P1S | 制限あり | 密閉だがアクティブチャンバー加熱なし |
| P1P | 非推奨 | エンクロージャーなし |
| A1 / A1 Mini | 使用不可 | オープンフレーム、温度不足 |

:::warning X1CとX1Eのみ推奨
PCは安定した結果のためにアクティブチャンバー加熱が必要です。P1Sは小さなパーツでは許容可能な結果が得られますが、大きなパーツでは反りと層間剥離が予想されます。
:::

## 乾燥

PCは**非常に吸湿性が高く**、湿気を急速に吸収します。湿ったPCはプリント結果を壊滅的にします。

| パラメータ | 値 |
|-----------|-------|
| 乾燥温度 | 70–80 °C |
| 乾燥時間 | 6–8時間 |
| 吸湿レベル | 高 |
| 最大推奨湿度 | < 0.02% |

- PCはプリント前に**常に**乾燥させる — 新しいスプールでも湿気を吸収している可能性あり
- 可能であれば乾燥ボックスから直接プリント
- AMSはPC保管には**不十分** — 湿度が高すぎる
- アクティブ加熱付き専用フィラメントドライヤーを使用

:::danger 湿気がPCプリントを破壊する
湿ったPCの兆候：激しいパチパチ音、表面の気泡、非常に悪い層間接着、ストリンギング。湿ったPCは設定では補正できません — まず**必ず**乾燥させてください。
:::

## PCブレンド

### PC-ABS

ポリカーボネートとABSのブレンドで、両素材の強度を組み合わせます：

- **純PCより印刷が容易** — 低温で反りが少ない
- **衝撃強度**はABSとPCの中間
- **産業で人気** — 自動車インテリアや電子機器ハウジングに使用
- ノズル250–270 °C、ベッド90–110 °Cでプリント

### PC-CF（カーボンファイバー）

最大の剛性と強度のためのカーボンファイバー強化PC：

- **極めて剛性が高い** — 構造部品に最適
- **軽量** — カーボンファイバーが重量を軽減
- **硬化ノズルが必要** — 真鍮は数時間で摩耗
- ノズル270–290 °C、ベッド100–120 °Cでプリント
- 純PCより高価だが、アルミニウムに近い機械的特性

### PC-GF（グラスファイバー）

グラスファイバー強化PC：

- **PC-CFより安価**で良好な剛性
- PC-CFより**白い表面**
- **硬化ノズルが必要** — グラスファイバーは非常に研磨性が高い
- PC-CFよりやや剛性は低いが、衝撃強度は優れる

## 用途

PCは**最大の強度および/または耐熱性**が必要な場所に使用されます：

- **機械部品** — ギア、固定具、荷重下のカップリング
- **光学部品** — レンズ、導光体、透明カバー（クリアPC）
- **耐熱部品** — エンジンルーム、発熱体付近
- **電子機器ハウジング** — 良好な衝撃強度を持つ保護ケース
- **工具とジグ** — 精密な組立工具

## PC印刷成功のヒント

### 第一層

- 第一層の速度を**30–40%**に下げる
- 最初の3–5層はベッド温度を標準より5 °C上げる
- **ブリムはほとんどのPC部品で必須** — 8–10 mmを使用

### チャンバー温度

- プリント開始前にチャンバーが**50 °C+**に達するまで待つ
- **プリント中にチャンバードアを開けない** — 温度低下で即座に反りが発生
- プリント後：チャンバー内で**ゆっくり**冷却（1–2時間）

### 冷却

- 最良の層間接着のために**最小限のパーツ冷却**（0–20%）を使用
- ブリッジとオーバーハングでは一時的に30–40%に増加
- PCでは見た目より層間強度を優先

### 設計上の考慮事項

- **鋭い角を避ける** — 最小1 mmの半径で丸める
- **均一な壁厚** — 不均一な厚さは内部応力を生む
- **大きな平面**は難しい — 分割するかリブを追加

:::tip PCが初めて？PC-ABSから始めましょう
PCのプリント経験がない場合は、PC-ABSブレンドから始めてください。純PCよりはるかに寛容で、極端な要件なしにPC素材の経験を積めます。PC-ABSをマスターしたら、純PCに進みましょう。
:::

## 後処理

- **研磨** — PCはよく研磨できるが、クリアPCはウェットサンディングを使用
- **ポリッシング** — クリアPCはほぼ光学品質まで磨ける
- **接着** — ジクロロメタン接着で目に見えない接合部（保護具使用！）
- **塗装** — 良好な密着にはプライマーが必要
- **アニーリング** — 120 °Cで1–2時間処理すると内部応力が軽減

:::warning ジクロロメタン接着
ジクロロメタンは有毒で、排気装置、耐薬品手袋、保護メガネが必要です。必ず換気の良い部屋または排気フードで作業してください。
:::
