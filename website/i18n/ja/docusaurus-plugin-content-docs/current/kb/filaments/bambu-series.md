# Bambu Labフィラメント

Bambu Labは、自社プリンター向けに特別に最適化された幅広いフィラメントを製造しています。すべてのBambu Labフィラメントには、プリンターが自動検出して正しい設定を適用する**RFIDタグ**が付属しています。

## RFIDとAMS

すべてのBambu Labフィラメントにはスプールに**RFIDチップ**が内蔵されています。これにより：

- **自動認識** — プリンターが材料の種類、色、設定を読み取る
- **残量表示** — スプールの推定残量
- **正確な設定** — 温度、速度、冷却が自動設定される
- **AMS互換性** — AMS内でのシームレスな材料切替

:::tip サードパーティフィラメントのAMS使用
AMSはサードパーティフィラメントでも動作しますが、Bambu Studioで手動で設定を行う必要があります。RFID自動検出はBambu Labフィラメント専用です。
:::

## 品質管理とカラー一貫性

Bambu Labはフィラメントに対して厳格な品質管理を実施しています：

- **直径公差** — ±0.02 mm（業界トップレベル）
- **カラー一貫性** — バッチ管理によりスプール間で同一の色を確保
- **スプール品質** — 結び目や重なりのない均一な巻き
- **真空密封** — 各スプールは乾燥剤付きで真空パック
- **温度プロファイルテスト** — 各バッチは最適温度でテスト済み

:::tip カラー一貫性のための色番号
Bambu Labはバッチ管理付きの色番号（例：「Bambu PLA Matte Charcoal」）を使用しています。大きなプロジェクトで複数スプールにわたり同一色が必要な場合は、同じバッチから注文するか、バッチマッチングについてサポートに問い合わせてください。
:::
