**凡例：**
- **優秀** — 最適に機能、推奨の組み合わせ
- **良好** — 良く機能、許容可能な代替
- **許容可** — 機能するが理想的ではない — 追加対策が必要
- **不良** — 変更を加えれば機能する可能性があるが非推奨
- **非推奨** — 結果が悪いかプレート損傷のリスクあり

:::tip PETGとCool Plate
PETGはCool Plate（Smooth PEI）に**強く付着しすぎ**、部品取り外し時にPEIコーティングを剥がす可能性があります。常にのりを分離膜として使用するか、Engineering Plateを選択してください。
:::

:::warning PCとプレート選択
PCはベッド温度が高い（100–120 °C）ためHigh Temp Plateが必要です。他のプレートはこの温度で永久変形する可能性があります。
:::

**凡例：**
- **可** — 完全対応、推奨
- **可*** — 硬化スチールノズル（HS01または同等品）が必要
- **条件付**** — 制限付きで機能する可能性あり、公式には非推奨
- **不可** — 非対応（エンクロージャーなし、温度不足等）

:::danger エンクロージャー要件
エンクロージャーが必要な材料（ABS、ASA、PA、PC）：
- **A1とA1 Mini**はオープンフレーム — 非対応
- **P1P**はオープンフレーム — エンクロージャーアクセサリーが必要
- **P1S**はエンクロージャーあり、ただしアクティブチャンバー加熱なし
- **X1CとX1E**はアクティブ加熱付きフルエンクロージャー — 高要求材料に推奨
:::

:::danger カーボンファイバーとグラスファイバーには硬化ノズルが必要
**-CF**（カーボンファイバー）または**-GF**（グラスファイバー）を含むすべての材料には**硬化スチールノズル**が必要です。真鍮はこれらの材料で数時間から数日で摩耗します。Bambu Lab HS01が推奨されます。

硬化ノズルが必要な材料：
- PLA-CF
- PETG-CF
- PA-CF / PA-GF
- PC-CF / PC-GF
:::

:::tip 一般材料での真鍮vs硬化スチール
真鍮ノズルは一般材料（PLA、PETG、ABS）に対して**より良い熱伝導性**を提供し、より均一な押出が可能です。硬化スチールも問題なく機能しますが、5–10 °C高い温度が必要な場合があります。日常使用には真鍮を使用し、CF/GF材料には硬化スチールに交換してください。
:::

## 材料切替のヒント

AMS内またはマニュアルで材料を切り替える際は、汚染を避けるために適切なパージが重要です。

### 推奨パージ量

| 切替 | パージ量 | 備考 |
|-----------|-------------|---------|
| PLA → PLA (別色) | 100–150 mm³ | 標準カラーチェンジ |
| PLA → PETG | 200–300 mm³ | 温度上昇、異なるフロー |
| PETG → PLA | 200–300 mm³ | 温度下降 |
| ABS → PLA | 300–400 mm³ | 大きな温度差 |
| PLA → ABS | 300–400 mm³ | 大きな温度差 |
| PA → PLA | 400–500 mm³ | ナイロンがホットエンドに残る |
| PC → PLA | 400–500 mm³ | PCは徹底的なパージが必要 |
| 暗い → 明るい色 | 200–300 mm³ | 暗い顔料は除去しにくい |
| 明るい → 暗い色 | 100–150 mm³ | 容易な移行 |

### 材料切替時の温度変更

| 移行 | 推奨 |
|----------|-----------|
| 低温 → 高温（例：PLA → ABS） | 新しい材料の温度まで加熱し、十分にパージ |
| 高温 → 低温（例：ABS → PLA） | 高温でまずパージし、その後温度を下げる |
| 同様の温度（例：PLA → PLA） | 標準パージ |
| 大きな差（例：PLA → PC） | PETGでの中間ステップが役立つ場合あり |

:::warning ナイロンとPCは残留物を残す
PA（ナイロン）とPCは特にパージが困難です。これらの材料使用後：
1. **PETG**または**ABS**で高温（260–280 °C）でパージ
2. 少なくとも**500 mm³**のパージ材料を流す
3. 押出を目視で確認 — 変色なく完全にクリーンであること
:::
