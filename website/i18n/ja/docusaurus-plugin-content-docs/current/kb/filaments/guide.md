## 素材別ヒント

### スタンダード (PLA, PETG)
- PLAは最も簡単な素材 — プロトタイプや装飾に最適
- PETGはより高い強度と耐熱性を提供しますが、糸引きが多くなることがあります
- どちらもエンクロージャーなし、標準の真鍮ノズルで使用可能

### エンジニアリング (ABS, ASA, PC)
- 反りを防ぐために密閉型プリンターが必要
- ASAはUV耐性 — 屋外パーツに使用
- PCは最高の強度と耐熱性を提供しますが、最も印刷が難しい

### コンポジット (CF/GF)
- **必ず**硬化スチールノズルを使用 — カーボンファイバーは真鍮ノズルを急速に摩耗させます
- CF系は剛性が高く軽量で、マット仕上げの部品が得られます
- GF系は安価ですが、表面が粗くなります

### フレキシブル (TPU)
- 最良の結果を得るために低速（50 mm/s以下）で印刷
- 詰まりを防ぐためにリトラクションを減らす
- ダイレクトドライブエクストルーダーはボーデン方式より優れています

### サポート (PVA, HIPS)
- PVAは水に溶ける — PLAのサポートに最適
- HIPSはリモネンに溶ける — ABSと一緒に使用
- どちらも湿度に非常に敏感 — 乾燥した状態で保管

:::warning ナイロン素材
PA、PA-CF、PA-GFは**極めて**吸湿性が高いです。数分で空気中の水分を吸収します。使用前に必ず12時間以上乾燥させ、可能であればドライボックスから直接印刷してください。
:::
