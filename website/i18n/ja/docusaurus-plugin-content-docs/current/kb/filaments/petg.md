# PETG

PETG（ポリエチレンテレフタレートグリコール）は機能的なパーツに人気の素材です。PLAより強く熱安定性が高く、軽度の化学薬品への耐性があります。

## 設定

| パラメーター | 値 |
|-----------|-------|
| ノズル温度 | 230〜250°C |
| ベッド温度 | 70〜85°C |
| パーツ冷却 | 30〜60% |
| 速度 | 標準 |
| 乾燥 | 推奨（65°Cで6〜8時間） |

## 推奨ビルドプレート

| プレート | 適合度 | グルースティック？ |
|-------|---------|----------|
| エンジニアリングプレート（テクスチャードPEI） | 優秀 | 不要/必要* |
| テクスチャードPEI | 良好 | 必要** |
| クールプレート（スムースPEI） | 警告を参照 | 警告を参照 |
| 高温プレート | 良好 | 必要 |

:::danger 重要：PETGでスムースPEIにグルースティックを使用すること
PETGはスムースPEI（クールプレート）に**非常に強く**定着します。グルースティックなしでは、プリントを取り外す際に**プレートのコーティングを剥がす**リスクがあります。PETGをプリントする際はスムースPEIに必ず薄くグルースティックを塗布してください — これがバリアーとして機能します。

**代替方法：** エンジニアリングプレートまたはテクスチャードPEIを使用 — プレートを傷めずに良好な定着が得られます。
:::

## 成功プリントのヒント

- **パーツ冷却を下げる** — 冷却が多すぎると層分離と脆いプリントの原因になります
- **ノズル温度を調整する** — ストリングが出る場合は5〜10°C下げる；層融合が悪い場合は上げる
- **ファーストレイヤーのベッド温度** — 定着向上のために80〜85°C、ファーストレイヤー後は70°Cに下げる
- **速度を下げる** — PETGはPLAより難しい、80%の速度から始める

:::warning ストリング
PETGはストリングが発生しやすいです。リトラクト距離を増やし（ダイレクトドライブで0.8〜1.5 mmを試す）、リトラクト速度を上げ、ノズル温度を5°Cずつ下げてみてください。
:::

## 乾燥

PETGはPLAより速く吸湿します。湿ったPETGは：
- プリント中に泡立ちと音がする
- 多孔質な表面で弱い層になる
- ストリングが増加する

特にスプールが長時間開封されていた場合は、プリント前に**65°Cで6〜8時間**乾燥させてください。

## 保管

常にシリカゲル入りの密閉袋または乾燥ボックスで保管してください。PETGは湿度の高い環境では数日以上開封したまま放置しないでください。
