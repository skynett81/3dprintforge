# PVAとサポート材

サポート材は、一時的な支持なしではプリントできないオーバーハング、ブリッジ、内部空洞を持つ複雑な形状をプリントするために使用されます。プリント後、サポート材は機械的に、または溶剤で溶解して除去されます。

## 概要

| 材料 | 溶剤 | 対応材料 | 溶解時間 | 難易度 |
|-----------|-----------|-------------|----------------|-------------------|
| PVA | 水 | PLA, PETG | 12–24時間 | 難しい |
| HIPS | d-リモネン | ABS, ASA | 12–24時間 | 中程度 |
| PVB | イソプロパノール (IPA) | PLA, PETG | 6–12時間 | 中程度 |
| BVOH | 水 | PLA, PETG, PA | 4–8時間 | 難しい |

---

## PVA（ポリビニルアルコール）

PVAは水溶性のサポート材で、複雑なサポート構造を持つPLAベースのプリントに最も広く使用されています。

### 設定

| パラメータ | 値 |
|-----------|-------|
| ノズル温度 | 190–210 °C |
| ベッド温度 | 45–60 °C |
| パーツ冷却 | 100% |
| 速度 | 60–80% |
| リトラクション | 増加（6–8 mm） |

### 推奨ビルドプレート

| プレート | 適合性 | のり? |
|-------|---------|----------|
| Cool Plate (Smooth PEI) | 優秀 | いいえ |
| Textured PEI | 良好 | いいえ |
| Engineering Plate | 良好 | いいえ |
| High Temp Plate | 避ける | — |

### 互換性

PVAは**同様の温度**でプリントする材料と最も相性が良いです：

| 本体材料 | 互換性 | 備考 |
|---------------|---------------|---------|
| PLA | 優秀 | 理想的な組み合わせ |
| PETG | 良好 | ベッド温度がPVAにやや高い場合がある |
| ABS/ASA | 不良 | チャンバー温度が高すぎる — PVAが劣化 |
| PA（ナイロン） | 不良 | 温度が高すぎる |

### 溶解

- 完成したプリントを**ぬるま湯**（約40 °C）に入れる
- PVAは厚さに応じて**12–24時間**で溶解
- 定期的に水をかき混ぜてプロセスを加速
- 6–8時間ごとに水を交換するとより速く溶解
- 超音波洗浄器を使用すると大幅に速くなる（2–6時間）

:::danger PVAは極めて吸湿性が高い
PVAは空気中の湿気を**非常に急速に**吸収します — 数時間の露出でもプリント結果を損なう可能性があります。湿気を吸収したPVAは以下を引き起こします：

- 激しい気泡とパチパチ音
- 本体材料への接着不良
- ストリンギングとべたつく表面
- ノズルの詰まり

**使用直前に必ずPVAを乾燥**させ、乾燥環境（乾燥ボックス）からプリントしてください。
:::

### PVAの乾燥

| パラメータ | 値 |
|-----------|-------|
| 乾燥温度 | 45–55 °C |
| 乾燥時間 | 6–10時間 |
| 吸湿レベル | 極めて高い |
| 保管方法 | 乾燥剤入り密閉ボックス、常時 |

## HIPS（ハイインパクトポリスチレン）

HIPSはd-リモネン（柑橘系溶剤）に溶解するサポート材です。ABSとASAに最適なサポート材です。

### 設定

| パラメータ | 値 |
|-----------|-------|
| ノズル温度 | 220–240 °C |
| ベッド温度 | 90–100 °C |
| チャンバー温度 | 40–50 °C（推奨） |
| パーツ冷却 | 20–40% |
| 速度 | 70–90% |

### 互換性

| 本体材料 | 互換性 | 備考 |
|---------------|---------------|---------|
| ABS | 優秀 | 理想的な組み合わせ — 同様の温度 |
| ASA | 優秀 | 非常に良好な接着 |
| PLA | 不良 | 温度差が大きすぎる |
| PETG | 不良 | 熱特性が異なる |

### d-リモネンでの溶解

- プリントを**d-リモネン**（柑橘系溶剤）に入れる
- 溶解時間：室温で**12–24時間**
- 35–40 °Cへの加温でプロセスが加速
- d-リモネンは2–3回再利用可能
- 溶解後、水で洗い流して乾燥

### PVAに対する利点

- **湿気に対する感度がはるかに低い** — 保管と取り扱いが容易
- **サポート材としてより強固** — 崩壊せずにより多くの負荷に耐える
- **ABS/ASAとのより良い熱的互換性**
- **プリントが容易** — 詰まりや問題が少ない

:::warning d-リモネンは溶剤です
手袋を着用し、換気の良い部屋で作業してください。d-リモネンは皮膚や粘膜を刺激する可能性があります。子供の手の届かない場所に保管してください。
:::

## PVB（ポリビニルブチラール）

PVBはイソプロパノール（IPA）に溶解するユニークなサポート材で、IPA蒸気による表面の平滑化にも使用できます。

### 設定

| パラメータ | 値 |
|-----------|-------|
| ノズル温度 | 200–220 °C |
| ベッド温度 | 55–75 °C |
| パーツ冷却 | 80–100% |
| 速度 | 70–80% |

### 互換性

| 本体材料 | 互換性 | 備考 |
|---------------|---------------|---------|
| PLA | 良好 | 許容可能な接着 |
| PETG | 中程度 | ベッド温度が変動する場合がある |
| ABS/ASA | 不良 | 温度が高すぎる |

### IPA蒸気による表面平滑化

PVBのユニークな特性として、IPA蒸気で表面を平滑化できます：

1. パーツを密閉容器に入れる
2. IPA浸漬した布を底に置く（パーツに直接触れないように）
3. 蒸気を**30–60分**作用させる
4. 取り出して24時間乾燥させる
5. 結果は滑らかなセミグロス表面

:::tip PVBを表面仕上げとして
PVBは主にサポート材ですが、PLA部品の最外層としてプリントし、IPA蒸気で平滑化できる表面を得ることもできます。これにより、アセトン平滑化されたABSに似た仕上がりが得られます。
:::

## サポート材の比較

| 特性 | PVA | HIPS | PVB | BVOH |
|----------|-----|------|-----|------|
| 溶剤 | 水 | d-リモネン | IPA | 水 |
| 溶解時間 | 12–24時間 | 12–24時間 | 6–12時間 | 4–8時間 |
| 湿気感度 | 極めて高い | 低い | 中程度 | 極めて高い |
| 難易度 | 難しい | 中程度 | 中程度 | 難しい |
| 価格 | 高い | 中程度 | 高い | 非常に高い |
| 最適な組み合わせ | PLA, PETG | ABS, ASA | PLA | PLA, PETG, PA |
| 入手性 | 良好 | 良好 | 限定的 | 限定的 |
| AMS互換 | はい（乾燥剤付き） | はい | はい | 問題あり |

## デュアルエクストルージョンとマルチカラーのヒント

### 一般的なガイドライン

- **パージ量** — サポート材は材料切替時に十分なパージが必要（最低150–200 mm³）
- **インターフェース層** — サポートと本体部品の間に2–3のインターフェース層を使用して清潔な表面を得る
- **距離** — 溶解後の簡単な除去のためにサポート距離を0.1–0.15 mmに設定
- **サポートパターン** — PVA/BVOHにはトライアングルパターン、HIPSにはグリッドを使用

### AMSセットアップ

- サポート材を**乾燥剤付きAMSスロット**に配置
- PVA：ボーデン接続付き外部乾燥ボックスを検討
- Bambu Studioで正しい材料プロファイルを設定
- 複雑なパーツの前にシンプルなオーバーハングモデルでテスト

### 一般的な問題と解決策

| 問題 | 原因 | 解決策 |
|---------|-------|---------|
| サポートが付着しない | 距離が大きすぎる | インターフェース距離を0.05 mmに減少 |
| サポートが強く付着しすぎる | 距離が小さすぎる | インターフェース距離を0.2 mmに増加 |
| サポート材に気泡 | 湿気 | フィラメントを十分に乾燥 |
| 材料間のストリンギング | リトラクション不足 | リトラクションを1–2 mm増加 |
| サポート面の表面品質不良 | インターフェース層不足 | 3–4インターフェース層に増加 |

:::tip シンプルに始めましょう
サポート材での最初のプリント：PLA + PVA、明確なオーバーハング（45°以上）のシンプルなモデル、Bambu Studioの標準設定を使用してください。経験を積むにつれて最適化していきましょう。
:::
