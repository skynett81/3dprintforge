# TPU

TPU（熱可塑性ポリウレタン）は弾性が必要なカバー、ガスケット、ホイール、その他のパーツに使用される柔軟な素材です。

## 設定

| パラメーター | 値 |
|-----------|-------|
| ノズル温度 | 220〜240°C |
| ベッド温度 | 30〜45°C |
| パーツ冷却 | 50〜80% |
| 速度 | 30〜50%（重要） |
| リトラクト | 最小限または無効 |
| 乾燥 | 推奨（60°Cで6〜8時間） |

:::danger 低速は重要
TPUはゆっくりプリントする必要があります。速度が高すぎると素材がエクストルーダー内で圧縮されて詰まりが発生します。30%の速度から始めて慎重に増やしてください。
:::

## 推奨ビルドプレート

| プレート | 適合度 | グルースティック？ |
|-------|---------|----------|
| テクスチャードPEI | 優秀 | 不要 |
| クールプレート（スムースPEI） | 良好 | 不要 |
| エンジニアリングプレート | 良好 | 不要 |

## リトラクト設定

TPUは弾性があり、積極的なリトラクトには反応が悪いです：

- **ダイレクトドライブ（X1C/P1S/A1）：** リトラクト0.5〜1.0 mm、25 mm/s
- **ボーデン（TPUでは避ける）：** 非常に難しく、推奨しない

非常に柔らかいTPU（ショアA 85以下）：リトラクトを完全に無効にして、温度と速度制御に頼ってください。

## ヒント

- **フィラメントを乾燥させる** — 湿ったTPUは非常にプリントが難しい
- **ダイレクトエクストルーダーを使用** — Bambu Lab P1S/X1C/A1はすべてダイレクトドライブ
- **高温を避ける** — 250°C以上ではTPUが劣化して変色したプリントになる
- **傾向** — TPUはストリングが発生しやすい；温度を5°C下げるか冷却を増やす

:::tip ショア硬度
TPUにはさまざまなショア硬度があります（A85、A95、A98）。ショアAが低いほど柔らかく、プリントが難しくなります。Bambu LabのTPUはショアA 95 — 良い出発点です。
:::

## 保管

TPUは非常に吸湿性が高い（水分を引き寄せる）。湿ったTPUは：
- 泡立ちと音がする
- 弱く脆いプリントになる（柔軟な素材にとって逆説的）
- ストリングが発生する

プリント前に**TPUを必ず60°Cで6〜8時間**乾燥させてください。シリカゲル入りの密閉ボックスで保管してください。
