# ナレッジベース

ナレッジベースにはBambu Labプリンターで最良の結果を得るための実践ガイドが含まれています。P1S、X1C、A1での経験に基づいた具体的な数値とヒントが見つかります。

## 何が見つかりますか？

### フィラメント
各素材タイプの詳細ガイド — 温度設定、プレートの選択、乾燥時間、成功プリントのヒント。

- [PLA](./filaments/pla) — 最もプリントしやすく、初心者に推奨
- [PETG](./filaments/petg) — PLAより強く、優れた耐薬品性
- [ABS](./filaments/abs) — 熱安定性が高く、エンクロージャーが必要
- [TPU](./filaments/tpu) — 柔軟素材、低速が必要
- [PA/ナイロン](./filaments/nylon) — 非常に耐久性が高く、プリントが難しい
- [コンポジット（CF/GF）](./filaments/composite) — 高剛性、ノズルが消耗

### ビルドプレート
各素材に適したプレートを選び、メンテナンス方法を学びます。

- [概要と互換性マトリックス](./build-plates/overview)
- [クールプレート（スムースPEI）](./build-plates/cool-plate)
- [エンジニアリングプレート（テクスチャードPEI）](./build-plates/engineering-plate)
- [高温プレート](./build-plates/high-temp-plate)
- [テクスチャードPEIプレート](./build-plates/textured-pei)

### メンテナンス
定期的なメンテナンスでプリンターを良好な状態に保ちます。

- [ノズルメンテナンス](./maintenance/nozzle) — クリーニング、コールドプル、交換
- [プレートメンテナンス](./maintenance/plate) — IPAクリーニングと損傷防止
- [AMSメンテナンス](./maintenance/ams) — PTFEチューブとフィラメントパス
- [潤滑](./maintenance/lubrication) — ロッド、ベアリング、インターバル

### トラブルシューティング
一般的な問題と実践的な解決策。

- [定着不良](./troubleshooting/adhesion) — ファーストレイヤーが剥がれる
- [反り](./troubleshooting/warping) — コーナーが持ち上がる
- [ストリング](./troubleshooting/stringing) — オブジェクト間の細い糸

## 誰のためのものですか？

このナレッジベースは次のような方向けです：
- Bambu Labで3Dプリントを始めたばかりの方
- 新しい素材に切り替えて初期設定が必要な方
- 特定の問題を解決したい方
- メンテナンスと機器の長寿命化についてより理解したい方

:::tip ダッシュボードに内蔵
ナレッジベースはダッシュボードの**ナレッジ**タブからも直接利用できます — プリント中にブラウザを切り替える必要がありません。
:::
