# ナレッジベース

ナレッジベースにはBambu Labプリンターで最良の結果を得るための実践ガイドが含まれています。P1S、X1C、A1での経験に基づいた具体的な数値とヒントが見つかります。

## 何が見つかりますか？

### フィラメント
各素材タイプの詳細ガイド — 温度設定、プレートの選択、乾燥時間、成功プリントのヒント。

- [PLA](./filamenter/pla) — 最もプリントしやすく、初心者に推奨
- [PETG](./filamenter/petg) — PLAより強く、優れた耐薬品性
- [ABS](./filamenter/abs) — 熱安定性が高く、エンクロージャーが必要
- [TPU](./filamenter/tpu) — 柔軟素材、低速が必要
- [PA/ナイロン](./filamenter/nylon) — 非常に耐久性が高く、プリントが難しい
- [コンポジット（CF/GF）](./filamenter/kompositt) — 高剛性、ノズルが消耗

### ビルドプレート
各素材に適したプレートを選び、メンテナンス方法を学びます。

- [概要と互換性マトリックス](./byggplater/oversikt)
- [クールプレート（スムースPEI）](./byggplater/cool-plate)
- [エンジニアリングプレート（テクスチャードPEI）](./byggplater/engineering-plate)
- [高温プレート](./byggplater/high-temp-plate)
- [テクスチャードPEIプレート](./byggplater/textured-pei)

### メンテナンス
定期的なメンテナンスでプリンターを良好な状態に保ちます。

- [ノズルメンテナンス](./vedlikehold/dyse) — クリーニング、コールドプル、交換
- [プレートメンテナンス](./vedlikehold/plate) — IPAクリーニングと損傷防止
- [AMSメンテナンス](./vedlikehold/ams) — PTFEチューブとフィラメントパス
- [潤滑](./vedlikehold/smoring) — ロッド、ベアリング、インターバル

### トラブルシューティング
一般的な問題と実践的な解決策。

- [定着不良](./feilsoking/heft) — ファーストレイヤーが剥がれる
- [反り](./feilsoking/warping) — コーナーが持ち上がる
- [ストリング](./feilsoking/stringing) — オブジェクト間の細い糸

## 誰のためのものですか？

このナレッジベースは次のような方向けです：
- Bambu Labで3Dプリントを始めたばかりの方
- 新しい素材に切り替えて初期設定が必要な方
- 特定の問題を解決したい方
- メンテナンスと機器の長寿命化についてより理解したい方

:::tip ダッシュボードに内蔵
ナレッジベースはダッシュボードの**ナレッジ**タブからも直接利用できます — プリント中にブラウザを切り替える必要がありません。
:::
