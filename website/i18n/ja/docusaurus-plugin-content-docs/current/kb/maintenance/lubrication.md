# 潤滑

可動部品の適切な潤滑は摩耗を減らし、騒音レベルを下げ、精確な動作を確保します。Bambu Labプリンターは定期的な潤滑が必要なリニア運動システムを使用しています。

## 潤滑タイプ

| コンポーネント | 潤滑タイプ | 製品 |
|-----------|-------------|---------|
| リニアロッド（XY） | 軽いマシンオイルまたはPTFEスプレー | 3-in-1、Super Lube |
| Z軸スクリュースピンドル | 厚いグリース | Super Lube グリース |
| リニアベアリング | 軽いリチウムグリース | Bambu Lab グリース |
| ケーブルチェーンジョイント | なし（乾燥） | — |

## リニアロッド

### XおよびY軸
ロッドはリニアベアリングを通って滑るポリッシュスチールロッドです：

```
間隔：200〜300時間ごと、またはきしみ音がする時
量：非常に少量 — ロッドポイントごとに1滴で十分
方法：
1. プリンターを電源オフにする
2. キャリッジを手動で端に移動させる
3. ロッドの真ん中に軽いオイルを1滴付ける
4. キャリッジをゆっくり前後に10回動かす
5. ほこりの出ない紙で余分なオイルを拭き取る
```

:::warning 過剰な潤滑はしないこと
オイルが多すぎると埃を引き寄せて研磨ペーストを作ります。最小量を使用して常に余分を拭き取ってください。
:::

### Z軸（垂直）
Z軸はオイルではなくグリースが必要なスクリュースピンドル（リードスクリュー）を使用します：

```
間隔：200時間ごと
方法：
1. プリンターを電源オフにする
2. スクリュースピンドルに沿って薄いグリースの層を塗布する
3. Z軸を手動で（またはメンテナンスメニューから）上下に動かす
4. グリースが自動的に分配される
```

## リニアベアリング

Bambu Lab P1SとX1CはY軸にリニアベアリング（MGN12）を使用します：

```
間隔：300〜500時間ごと
方法：
1. 針または爪楊枝で注入口から少しグリースを取り除く
2. 注射器と細いカニューレを使って新しいグリースを注入する
3. 軸を前後に動かしてグリースを分配する
```

Bambu Labはシステムに合わせて調整された公式グリース（Bambu Lubricant）を販売しています。

## モデル別の潤滑メンテナンス

### X1C / P1S
- Y軸：リニアベアリング — BambuのグリースBambu
- X軸：カーボンロッド — 軽いオイル
- Z軸：デュアルスクリュースピンドル — Bambuグリース

### A1 / A1 Mini
- 全軸：スチールロッド — 軽いオイル
- Z軸：シングルスクリュースピンドル — Bambuグリース

## 潤滑が必要なサイン

- 動作時の**きしみまたはこすれる音**
- 縦の壁に見える**振動パターン**（VFA）
- 他の原因のない**不正確な寸法**
- 運動システムからの**音程が高い音**

## 潤滑間隔

| 作業 | 間隔 |
|-----------|---------|
| XYロッドにオイル塗布 | 200〜300時間ごと |
| Zスピンドルにグリース塗布 | 200時間ごと |
| リニアベアリングにグリース（X1C/P1S） | 300〜500時間ごと |
| フルメンテナンスサイクル | 半年ごと（または500時間） |

間隔を自動的に追跡するためにダッシュボードのメンテナンスモジュールを使用してください。
