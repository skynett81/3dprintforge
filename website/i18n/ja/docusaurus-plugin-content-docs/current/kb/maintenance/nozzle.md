# ノズルメンテナンス

ノズルはプリンターの最も重要なコンポーネントの1つです。適切なメンテナンスで寿命が延び、良好なプリント結果が得られます。

## ノズルタイプ

| ノズルタイプ | 対応素材 | 推定寿命 | 最大温度 |
|----------|-----------|-------------------|----------|
| 真鍮（標準） | PLA、PETG、ABS、TPU | 200〜500時間 | 300°C |
| 硬化スチール | CF/GFを含むすべて | 300〜600時間 | 300°C |
| HS01（Bambu） | CF/GFを含むすべて | 500〜1000時間 | 300°C |

:::danger CF/GFに真鍮ノズルを絶対に使用しないこと
カーボンファイバーとガラスファイバー充填フィラメントは真鍮ノズルを数時間で摩耗させます。CF/GF素材をプリントする前に硬化スチールに交換してください。
:::

## クリーニング

### 簡易クリーニング（スプール交換時）
1. ノズルを200〜220°Cに加熱する
2. 手動でフィラメントを押し通して清潔になるまで
3. フィラメントを素早く引き出す（「コールドプル」— 下記を参照）

### IPAクリーニング
頑固な残留物に対して：
1. ノズルを200°Cに加熱する
2. ノズルの先端に1〜2滴のIPAを垂らす（注意！）
3. 蒸気で残留物を溶かす
4. 新鮮なフィラメントを通す

:::warning 熱いノズルでのIPAには注意
IPAは83°Cで沸騰し、熱いノズルで激しく蒸発します。少量を使用し、吸入を避けてください。
:::

## コールドプル

コールドプルはノズルから汚染物とカーボン残留物を除去する最も効果的な方法です。

**ステップごとの手順：**
1. ノズルを200〜220°Cに加熱する
2. ナイロンフィラメント（またはノズル内のもの）を手動で押し込む
3. ナイロンをノズル内で1〜2分浸す
4. 温度を80〜90°Cに下げる（ナイロン用）
5. ノズルが目標まで冷却するまで待つ
6. フィラメントを1回の動作で素早くしっかりと引き出す
7. 先端を確認：ノズルの内側の形状になっているはずで、清潔で残留物がない
8. フィラメントが清潔で白く引き出されるまで3〜5回繰り返す

:::tip コールドプル用ナイロン
ナイロンは汚染物にしっかり付着するため、コールドプルに最良の結果を得られます。白いナイロンで引き出しが清潔かどうかを確認しやすくなります。
:::

## ノズル交換

### ノズルを交換すべきサイン
- でこぼこの底面と寸法精度の低下
- クリーニング後も続くエクスルード問題
- ノズルの穴の目に見える摩耗または変形
- ノズルが推定寿命を超えた

### 手順（P1S/X1C）
1. ノズルを200°Cに加熱する
2. エクストルーダーモーターを解放する（フィラメントを解放）
3. スパナを使ってノズルを緩める（反時計回り）
4. ノズルが温かい間に交換する — **ツールが付いた状態でノズルを冷却させないこと**
5. 適切なトルクで締める（締めすぎないこと）
6. 交換後にキャリブレーションを実行する

:::warning 常に温かい状態で交換すること
冷たいノズルの締め付けトルクは加熱中に部品を壊す可能性があります。ノズルが温かい間（200°C）に常に交換して締め付けてください。
:::

## メンテナンス間隔

| 作業 | 間隔 |
|-----------|---------|
| クリーニング（コールドプル） | 50時間ごと、または素材交換時 |
| 目視確認 | 週次 |
| ノズル交換（真鍮） | 200〜500時間 |
| ノズル交換（硬化スチール） | 300〜600時間 |
