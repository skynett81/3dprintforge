# ストリング

ストリング（または「oozing」）は、ノズルがエクストルードせずに移動する際に、オブジェクトの別々の部分の間に形成される細いプラスチックの糸です。プリントに「クモの巣」のような外観を与えます。

## ストリングの原因

1. **ノズル温度が高すぎる** — 熱いプラスチックは液状になり垂れる
2. **リトラクト設定が不適切** — フィラメントが十分に素早く引き戻されない
3. **フィラメントが湿っている** — 湿気が蒸気と余分な流れを引き起こす
4. **速度が低すぎる** — ノズルがトランジット位置に長くとどまる

## 診断

**フィラメントが湿っている？** プリント中にパチパチ・ポップ音が聞こえますか？その場合フィラメントが湿っています — 他の設定を調整する前にまず乾燥させてください。

**温度が高すぎる？** 「一時停止」の瞬間にノズルから垂れが見えますか？温度を5〜10°C下げてください。

## 解決策

### 1. フィラメントを乾燥させる

湿ったフィラメントは、設定で解消できないストリングの最も一般的な原因です：

| 素材 | 乾燥温度 | 時間 |
|-----------|----------------|-----|
| PLA | 45〜55°C | 4〜6時間 |
| PETG | 60〜65°C | 6〜8時間 |
| TPU | 55〜60°C | 6〜8時間 |
| PA | 75〜85°C | 8〜12時間 |

### 2. ノズル温度を下げる

5°Cずつ下げることから始めてください：
- PLA：210〜215°Cを試す（220°Cから下げる）
- PETG：235〜240°Cを試す（245°Cから下げる）

:::warning 温度が低すぎると層融合が不良になります
温度は慎重に下げてください。温度が低すぎると層融合不良、強度不足のプリント、エクストルード問題が発生します。
:::

### 3. リトラクト設定を調整する

リトラクトは「トラベル」移動中に垂れを防ぐためにフィラメントをノズル内に引き戻します：

```
Bambu Studio → フィラメント → リトラクト：
- リトラクト距離：0.4〜1.0 mm（ダイレクトドライブ）
- リトラクト速度：30〜45 mm/s
```

:::tip Bambu Labプリンターはダイレクトドライブを使用
すべてのBambu Labプリンター（X1C、P1S、A1）はダイレクトドライブエクストルーダーを使用しています。ダイレクトドライブはBowdenシステムより**短い**リトラクト距離が必要です（一般的に0.5〜1.5 mm対3〜7 mm）。
:::

### 4. トラベル速度を上げる

ポイント間の素早い移動により、ノズルが垂れる時間が短くなります：
- 「トラベル速度」を200〜300 mm/sに上げる
- Bambu Labプリンターはこれを十分に処理できます

### 5. 「外周を横断しない」を有効にする

ストリングが目立つ開放エリアをノズルが避けるようにするスライサー設定：
```
Bambu Studio → 品質 → 外周を横断しない
```

### 6. 速度を下げる（TPU向け）

TPUの場合、解決策は他の素材と逆です：
- プリント速度を20〜35 mm/sに下げる
- TPUは弾性があり、速度が高すぎると圧縮される — これが「後流れ」を引き起こす

## 調整後

標準的なストリングテストモデル（例：MakerWorldの「torture tower」）でテストしてください。一度に1つの変数を調整して変化を観察します。

:::note 完璧は稀にしか達成できません
ほとんどの素材でいくらかのストリングは正常です。完全に排除しようとせず、許容できるレベルまで減らすことに集中してください。PETGは常にPLAよりわずかに多くストリングが出ます。
:::
