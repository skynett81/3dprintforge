# AMSメンテナンス

AMS（Automatic Material System）は信頼性高く機能するために定期的なメンテナンスが必要な精密システムです。最も一般的な問題は汚れたフィラメントパスとハウジング内の湿度です。

## PTFEチューブ

PTFEチューブはAMSからプリンターへのフィラメントを搬送します。最初に摩耗する部品の1つです。

### 検査
PTFEチューブを以下について確認します：
- **折れまたは曲がり** — フィラメントの流れを妨げる
- **コネクション部の摩耗** — 入口周りの白い粉
- **形状の変形** — 特にCF素材を使用した場合

### PTFEチューブの交換
1. AMSからフィラメントを解放する（アンロードサイクルを実行）
2. コネクション部のチューブの周りの青いロックリングを押す
3. チューブを引き出す（しっかりと掴む必要がある）
4. 新しいチューブを正しい長さに切る（元のものより短くしない）
5. 止まるまで押し込んでロックする

:::tip AMS Lite vs. AMS
AMS Lite（A1/A1 Mini）はフルAMS（P1S/X1C）よりシンプルなPTFE構成を持っています。チューブが短く交換が容易です。
:::

## フィラメントパス

### フィラメントパスのクリーニング
フィラメントはフィラメントパスに、特にCF素材は埃と残留物を残します：

1. すべてのスロットのアンロードを実行する
2. 圧縮空気またはソフトブラシで粉塵を吹き飛ばす
3. クリーニング用のナイロンまたはPTFEクリーニングフィラメントをパスに通す

### センサー
AMSはフィラメントの位置とフィラメント切れを検出するためにセンサーを使用します。センサーウィンドウを清潔に保ちます：
- クリーンなブラシでセンサーレンズを優しく拭く
- センサーに直接IPAを塗布しない

## 湿度

AMSはフィラメントを湿度から保護しません。吸湿性の高い素材（PA、PETG、TPU）には以下を推奨します：

### 乾燥AMSの代替手段
- **密閉ボックス：** スプールをシリカゲル入りの密閉容器に入れる
- **Bambu Dry Box：** 公式ドライボックスアクセサリー
- **外部フィーダー：** デリケートな素材にはAMSの外でフィラメントフィーダーを使用

### 湿度インジケーター
AMSハウジングに湿度インジケーターカード（湿度計）を入れてください。相対湿度が30%を超えたらシリカゲルバッグを交換します。

## ドライブホイールとクランプ機構

### 検査
AMSのドライブホイール（エクストルーダーホイール）を以下について確認します：
- 歯の間のフィラメント残留物
- 歯セットの摩耗
- 手動引き出し時の不均一な摩擦

### クリーニング
1. 歯ブラシまたはブラシを使用してドライブホイールの歯の間の残留物を取り除く
2. 圧縮空気で吹く
3. オイルと潤滑剤を避ける — グリップレベルは乾燥作動向けに調整されています

## メンテナンス間隔

| 作業 | 間隔 |
|-----------|---------|
| PTFEチューブの目視検査 | 月次 |
| フィラメントパスのクリーニング | 100時間ごと |
| センサーの確認 | 月次 |
| シリカゲルの交換（乾燥セットアップ） | 必要に応じて（30%+RHで） |
| PTFEチューブの交換 | 目に見える摩耗時 |
