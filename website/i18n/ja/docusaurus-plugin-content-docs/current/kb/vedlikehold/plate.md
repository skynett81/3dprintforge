# プレートメンテナンス

ビルドプレートは良好な定着と成功したプリントに不可欠です。適切なクリーニングと取り扱いで寿命を大幅に延ばせます。

## クリーニング

### プリント間（IPA）
IPA（イソプロピルアルコール）は油脂残留物と指紋を取り除く標準的な方法です：

```
1. プレートが40°C未満になるまで待つ
2. ほこりの出ない紙にIPAを付ける（直接スプレーしない）
3. 丸い動きで表面を拭く
4. 30〜60秒乾燥させる
5. その後表面を触らない（指からの油脂残留物）
```

:::tip 指紋を避けること
指からの油脂残留物は定着不良の最も一般的な原因です。常に端でプレートを扱うか、使い捨て手袋を使用してください。
:::

### ディープクリーニング（水と石けん）
月次またはグルースティック使用後に実施：

1. ぬるま湯（30〜50°C）と中性洗剤
2. 柔らかい布またはスポンジで優しくこする（研磨面を使わない）
3. 流水で十分にすすぐ
4. ほこりの出ない紙で拭く
5. 完全に乾燥させるか、最終仕上げにIPAで拭く

### グルースティック後
グルースティックの残留物はプリント後に**必ず**洗い落とすこと：
1. ぬるま湯にプレートを2〜3分浸す
2. グルースティックは水で簡単に溶ける
3. 上記のようにすすいで乾燥させる

## 損傷防止

### プリントの正しい取り外し方
- **冷却させる** — ほとんどの素材は30〜40°Cで自然に外れます
- **プレートを曲げる** — 弾性鋼板は容易に曲がります；端を押して外す
- **尖ったものを使わない** — ナイフとスクレーパーはPEIコーティングを傷付けて損傷させます

:::danger 金属スクレーパーを使用しないこと
金属スクレーパーはPEIコーティングを傷付けてプレートの寿命を大幅に短くします。曲げ技術か必要であればプラスチックスクレーパーを使用してください。
:::

### 温度
- **冷たいプレートを熱いベッドに置かない** — 熱衝撃に注意
- **熱いプレートに冷たいIPAをスプレーしない** — 同じ理由

## プレートを交換すべき時期

プレートを交換する場合：
- PEIコーティングに目に見える損傷がある（亀裂、剥がれ、深い傷）
- 徹底的なクリーニングとディープクリーニングの後もプリントが定着しない
- 表面が変形または不均一
- 中央部分が滑らかで摩耗している（エンジニアリングプレート）

### 一般的な寿命

| プレート | 通常の使用 | 集中的な使用 |
|-------|------------|--------------|
| クールプレート | 200〜500プリント | 100〜200プリント |
| エンジニアリングプレート | 300〜700プリント | 200〜400プリント |
| 高温プレート | 200〜400プリント | 100〜200プリント |
| テクスチャードPEI | 300〜600プリント | 200〜300プリント |

## 保管

未使用のプレートを保管する際は：
- オリジナルの保護材または平らなフォルダーに
- 直射日光を避ける（UVがPEIを劣化させる）
- 埃の多い環境を避ける — 埃がテクスチャーに入り込んで定着を低下させる
