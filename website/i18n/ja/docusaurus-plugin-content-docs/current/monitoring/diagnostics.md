# 診断

診断ページでは、プリンターの健康状態、パフォーマンス、経時的な状態についての詳細な概要が得られます。

アクセス先：**https://localhost:3443/#diagnostics**

## 健康スコア

各プリンターは以下に基づいて0〜100の**健康スコア**を計算します。

| 要因 | 重み | 説明 |
|---|---|---|
| 成功率（30日） | 30% | 過去30日間の成功したプリントの割合 |
| コンポーネント消耗度 | 25% | 重要部品の平均消耗度 |
| HMSエラー（30日） | 20% | エラーの数と深刻度 |
| キャリブレーション状態 | 15% | 最後のキャリブレーションからの経過時間 |
| 温度安定性 | 10% | 印刷中の目標温度からの偏差 |

**スコアの解釈：**
- 🟢 80〜100 — 優れた状態
- 🟡 60〜79 — 良好だが、調査が必要なことも
- 🟠 40〜59 — パフォーマンス低下、メンテナンス推奨
- 🔴 0〜39 — 重大、メンテナンスが必要

:::tip 履歴
健康グラフをクリックしてスコアの経時変化を確認します。大きな低下は特定のイベントを示している場合があります。
:::

## テレメトリーグラフ

テレメトリーページには、すべてのセンサー値のインタラクティブなグラフが表示されます。

### 利用可能なデータセット

- **ノズル温度** — 実際の温度と目標温度
- **ベッド温度** — 実際の温度と目標温度
- **チャンバー温度** — 機械内部の周囲温度
- **エクストルーダーモーター** — 消費電流と温度
- **ファン速度** — ツールヘッド、チャンバー、AMS
- **気圧**（X1C） — AMS用チャンバー圧力
- **加速度** — 振動データ（ADXL345）

### グラフのナビゲーション

1. **時間範囲**を選択：直近1時間 / 24時間 / 7日 / 30日 / カスタム
2. ドロップダウンから**プリンター**を選択
3. 表示する**データセット**を選択（複数選択可）
4. スクロールしてタイムラインをズームイン
5. クリック＆ドラッグでパン
6. ダブルクリックでズームをリセット

### テレメトリーデータのエクスポート

1. グラフで**エクスポート**をクリックします
2. 形式を選択：**CSV**、**JSON**、または**PNG**（画像）
3. 選択した時間範囲とデータセットがエクスポートされます

## ベッドメッシュ

ベッドメッシュの可視化では、ビルドプレートの平面性キャリブレーションが表示されます。

1. **診断 → ベッドメッシュ**に移動します
2. プリンターを選択します
3. 最新のメッシュが3Dサーフェスとヒートマップで表示されます：
   - **青** — 中心より低い（凹型）
   - **緑** — ほぼ平坦
   - **赤** — 中心より高い（凸型）
4. ポイントにカーソルを合わせてmmの正確な偏差を確認します

### UIからのベッドメッシュスキャン

1. **今すぐスキャン**をクリックします（プリンターがアイドル状態である必要があります）
2. ダイアログで確認 — プリンターが自動的にキャリブレーションを開始します
3. スキャンが完了するまで待ちます（約3〜5分）
4. 新しいメッシュが自動的に表示されます

:::warning 先に温めてください
正確なキャリブレーションのため、ベッドメッシュは暖かいベッド（PLAの場合50〜60°C）でスキャンする必要があります。
:::

## コンポーネント消耗度

詳細なドキュメントについては[消耗度予測](./wearprediction)を参照してください。

診断ページには圧縮された概要が表示されます。
- コンポーネントごとのパーセントスコア
- 次の推奨メンテナンス
- **詳細**をクリックして完全な消耗度分析を表示
