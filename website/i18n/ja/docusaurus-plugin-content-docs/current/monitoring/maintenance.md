# メンテナンス

メンテナンスモジュールは、Bambu Labプリンターのすべてのメンテナンスを計画・追跡するのに役立ちます。ノズル交換からレールの注油まで。

アクセス先：**https://localhost:3443/#maintenance**

## メンテナンス計画

Bambu Dashboardには、すべてのBambu Labプリンターモデル用の事前設定されたメンテナンス間隔が付属しています。

| タスク | 間隔（標準） | モデル |
|---|---|---|
| ノズルクリーニング | 200時間ごと | すべて |
| ノズル交換（真鍮） | 500時間ごと | すべて |
| ノズル交換（硬化鋼） | 2000時間ごと | すべて |
| X軸の注油 | 300時間ごと | X1C、P1S |
| Z軸の注油 | 300時間ごと | すべて |
| AMSギアクリーニング | 200時間ごと | AMS |
| チャンバークリーニング | 500時間ごと | X1C |
| PTFEチューブ交換 | 必要に応じて / 1000時間 | すべて |
| キャリブレーション（フル） | 月1回 | すべて |

すべての間隔はプリンターごとにカスタマイズできます。

## ノズル交換ログ

1. **メンテナンス → ノズル**に移動します
2. **ノズル交換をログ**をクリックします
3. 以下を入力します：
   - **日付** — 自動的に今日に設定
   - **ノズル素材** — 真鍮 / 硬化鋼 / 銅 / ルビーチップ
   - **ノズル径** — 0.2 / 0.4 / 0.6 / 0.8 mm
   - **ブランド/モデル** — オプション
   - **価格** — コストログ用
   - **交換時の時間** — プリント時間カウンターから自動取得
4. **保存**をクリックします

ログはすべてのノズル履歴を日付順に表示します。

:::tip 事前リマインダー
次の推奨交換の十分前に通知を受け取るために**X時間前に通知**（例：50時間）を設定してください。
:::

## メンテナンスタスクの作成

1. **新しいタスク**（+アイコン）をクリックします
2. 以下を入力します：
   - **タスク名** — 例：「Y軸の注油」
   - **プリンター** — 対象プリンターを選択
   - **間隔タイプ** — 時間 / 日数 / プリント数
   - **間隔** — 例：300時間
   - **最後の実施** — 最後に実施した日時を入力（過去日付の設定）
3. **作成**をクリックします

## 間隔とリマインダー

アクティブなタスクには以下が表示されます。
- **緑** — 次のメンテナンスまで間隔の50%以上残っている
- **黄** — 次のメンテナンスまで間隔の50%未満残っている
- **オレンジ** — 次のメンテナンスまで間隔の20%未満残っている
- **赤** — メンテナンスが期限切れ

### リマインダーの設定

1. タスクをクリック → **編集**
2. **リマインダー**を有効にします
3. **通知する条件**を設定します（例：期限の10%前）
4. アラートチャンネルを選択します（[アラート](../features/notifications)を参照）

## 完了としてマーク

1. リストでタスクを見つけます
2. **完了**（チェックマークアイコン）をクリックします
3. 今日の日付/時間から間隔がリセットされます
4. ログエントリーが自動的に作成されます

## コストログ

すべてのメンテナンスタスクに関連コストを設定できます。

- **部品** — ノズル、PTFEチューブ、潤滑剤
- **時間** — 使用時間 × 時間単価
- **外部サービス** — 有料修理

コストはプリンターごとに合計され、統計概要に表示されます。

## メンテナンス履歴

**メンテナンス → 履歴**で以下を確認できます。
- 実施されたすべてのメンテナンスタスク
- 日付、時間、コスト
- 実施者（マルチユーザーシステムの場合）
- コメントとメモ

会計目的でCSVに履歴をエクスポートできます。
