# エラーログ

エラーログはプリンターからのすべてのエラーとHMSアラート（健康、メンテナンス、安全）を収集します。Bambu DashboardにはBambu Labプリンター用の269以上のHMSコードのデータベースが内蔵されています。

アクセス先：**https://localhost:3443/#errors**

## HMSコード

Bambu LabプリンターはMQTT経由で問題が発生するとHMSコードを送信します。Bambu Dashboardはこれらを自動的に読みやすいエラーメッセージに変換します。

| コード | 例 | カテゴリ |
|---|---|---|
| `0700 0100 0001 0001` | Nozzle heatbreak clogged | ノズル/エクストルーダー |
| `0700 0200 0002 0001` | AMS filament stuck | AMS |
| `0700 0300 0003 0001` | Bed leveling failed | ビルドプレート |
| `0700 0500 0001 0001` | MC disconnect | 電子部品 |

完全なリストは、X1C、X1C Combo、X1E、P1S、P1S Combo、P1P、P2S、P2S Combo、A1、A1 Combo、A1 mini、H2S、H2D、H2Cの269以上の既知のコードをカバーしています。

## 深刻度

エラーは4つのレベルに分類されます。

| レベル | 色 | 説明 |
|---|---|---|
| **重大** | 赤 | 即座の対応が必要 — プリントが停止 |
| **高** | オレンジ | 速やかに対応すべき — プリントは継続可能 |
| **中** | 黄 | 調査すべき — 即座の危険はなし |
| **情報** | 青 | 情報メッセージ、対応不要 |

## 検索とフィルタリング

エラーログ上部のツールバーを使用します。

1. **フリーテキスト検索** — エラーメッセージ、HMSコード、プリンターの説明を検索
2. **プリンター** — 1つのプリンターのエラーのみ表示
3. **カテゴリ** — AMS / ノズル / プレート / 電子部品 / キャリブレーション / その他
4. **深刻度** — すべて / 重大 / 高 / 中 / 情報
5. **日付** — 日付範囲でフィルタリング
6. **未確認** — 確認されていないエラーのみ表示

**フィルターをリセット**をクリックしてすべてのエラーを表示します。

## Wikiリンク

各HMSコードにはBambu Lab Wikiへのリンクが表示されます。

- 完全なエラーの説明
- 考えられる原因
- ステップごとのトラブルシューティングガイド
- Bambu Labの公式推奨事項

エラーエントリーの**Wikiを開く**をクリックして関連するWikiページを新しいタブで開きます。

:::tip ローカルコピー
Bambu DashboardはオフラインでI使用するためにWikiのコンテンツをローカルにキャッシュします。コンテンツは毎週自動的に更新されます。
:::

## エラーの確認

確認はエラーを削除せずに対応済みとしてマークします。

1. リストでエラーをクリックします
2. **確認**（チェックマークアイコン）をクリックします
3. 対応内容についてオプションのメモを入力します
4. エラーにチェックマークが付き「確認済み」リストに移動します

### 一括確認

1. チェックボックスで複数のエラーを選択します
2. **選択したものを確認**をクリックします
3. 選択したすべてのエラーが同時に確認されます

## 統計

エラーログの上部に以下が表示されます。

- 過去30日間のエラー総数
- 未確認のエラー数
- 最も頻繁に発生するHMSコード
- 最もエラーの多いプリンター

## エクスポート

1. **エクスポート**（ダウンロードアイコン）をクリックします
2. 形式を選択：**CSV**または**JSON**
3. エクスポートにフィルターが適用されます — 先に希望のフィルターを設定してください
4. ファイルが自動的にダウンロードされます

## 新しいエラーのアラート

新しいHMSエラーのアラートを有効にします。

1. **設定 → アラート**に移動します
2. **新しいHMSエラー**にチェックを入れます
3. アラートの最小深刻度を選択します（推奨：**高**以上）
4. アラートチャンネルを選択します

チャンネル設定については[アラート](../funksjoner/notifications)を参照してください。
