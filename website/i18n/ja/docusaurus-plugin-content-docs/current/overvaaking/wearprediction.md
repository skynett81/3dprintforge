# 消耗度予測

消耗度予測は、実際の使用量、フィラメントタイプ、プリンターの動作に基づいて重要なコンポーネントの予想寿命を計算します。これにより、事後的ではなく予防的にメンテナンスを計画できます。

アクセス先：**https://localhost:3443/#wear**

## 監視するコンポーネント

Bambu Dashboardはプリンターごとに8つのコンポーネントの消耗度を追跡します。

| コンポーネント | 主な消耗要因 | 典型的な寿命 |
|---|---|---|
| **ノズル（真鍮）** | フィラメントタイプ + 時間 | 300〜800時間 |
| **ノズル（硬化鋼）** | 時間 + 研磨素材 | 1500〜3000時間 |
| **PTFEチューブ** | 時間 + 高温 | 500〜1500時間 |
| **エクストルーダーギア** | 時間 + 研磨素材 | 1000〜2000時間 |
| **X軸シャフト（CNC）** | プリント数 + 速度 | 2000〜5000時間 |
| **ビルドプレート表面** | プリント数 + 温度 | 200〜500プリント |
| **AMSギア** | フィラメント交換回数 | 5000〜15000回 |
| **チャンバーファン** | 稼働時間 | 3000〜8000時間 |

## 消耗度の計算

消耗度は総合パーセンテージ（0〜100%消耗）として計算されます。

```
消耗度% = (実際の使用量 / 予想寿命) × 100
         × 素材乗数
         × 速度乗数
```

**素材乗数：**
- PLA、PETG：1.0×（通常の消耗）
- ABS、ASA：1.1×（やや侵食性が高い）
- PA、PC：1.2×（PTFEとノズルに負担）
- CF/GFコンポジット：2.0〜3.0×（非常に研磨性が高い）

:::warning カーボンファイバー
カーボンファイバー強化フィラメント（CF-PLA、CF-PAなど）は真鍮ノズルを急速に摩耗させます。硬化鋼ノズルを使用し、2〜3倍速い摩耗を見込んでください。
:::

## 寿命の計算

各コンポーネントに以下が表示されます。

- **現在の消耗度** — 使用済みパーセンテージ
- **推定残り寿命** — 時間またはプリント数
- **推定終了日** — 過去30日間の平均使用量に基づく
- **信頼区間** — 予測の不確実性マージン

コンポーネントをクリックして、時間経過に伴う消耗度蓄積の詳細グラフを確認します。

## アラート

コンポーネントごとに自動アラートを設定します。

1. **消耗度 → 設定**に移動します
2. 各コンポーネントの**アラート閾値**を設定します（推奨：75%と90%）
3. アラートチャンネルを選択します（[アラート](../funksjoner/notifications)を参照）

**アラートメッセージの例：**
> ⚠️ 私のX1CのノズルI（真鍮）が78%消耗しています。推定寿命：約45時間。推奨：ノズル交換を計画してください。

## メンテナンスコスト

消耗度モジュールはコストログと統合されています。

- **コンポーネントごとのコスト** — 予備部品の価格
- **総交換コスト** — 閾値に近いすべてのコンポーネントの合計
- **今後6ヶ月のプロジェクション** — 将来の推定メンテナンスコスト

**消耗度 → 価格**でコンポーネント価格を入力します。

1. **価格を設定**をクリックします
2. 各コンポーネントの単価を入力します
3. 価格はコストプロジェクションに使用され、プリンターモデルごとに異なる場合があります

## 消耗度カウンターのリセット

メンテナンス後、対象コンポーネントのカウンターをリセットします。

1. **消耗度 → [コンポーネント名]**に移動します
2. **交換済みとしてマーク**をクリックします
3. 以下を入力します：
   - 交換日
   - コスト（オプション）
   - メモ（オプション）
4. 消耗度カウンターがリセットされ再計算されます

リセットはメンテナンス履歴に表示されます。

:::tip キャリブレーション
消耗度予測を実際の経験データと比較し、**消耗度 → 寿命を設定**で寿命パラメーターを調整して、実際の使用状況に合わせた計算を行ってください。
:::
