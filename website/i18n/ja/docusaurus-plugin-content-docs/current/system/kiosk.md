# キオスクモード

キオスクモードは、キーボード、マウス操作、ブラウザUIなしにプリンターステータスを継続的に表示する壁掛けディスプレイ、テレビ、または専用モニター向けに設計されています。

アクセス先：**https://localhost:3443/#settings** → **システム → キオスク**

## キオスクモードとは

キオスクモードでは：
- ナビゲーションメニューが非表示
- インタラクティブなコントロールが表示されない
- ダッシュボードが自動更新
- プリンター間で画面がローテーション（設定した場合）
- 非活動タイムアウトが無効

## URLでキオスクモードを有効にする

設定を変更せずにキオスクモードを有効にするには、URLに`?kiosk=true`を追加します：

```
https://localhost:3443/?kiosk=true
https://localhost:3443/#fleet?kiosk=true
```

パラメーターを削除するか`?kiosk=false`を追加してキオスクモードを無効にします。

## キオスク設定

1. **設定 → システム → キオスク**に移動します
2. 設定します：

| 設定 | デフォルト値 | 説明 |
|---|---|---|
| デフォルトビュー | フリート概要 | 表示するページ |
| ローテーション間隔 | 30秒 | ローテーションでのプリンターごとの時間 |
| ローテーションモード | アクティブのみ | アクティブなプリンター間のみローテーション |
| テーマ | ダーク | ディスプレイに推奨 |
| フォントサイズ | 大 | 遠くからでも読みやすい |
| 時計表示 | オフ | 隅に時計を表示 |

## キオスク用フリートビュー

フリート概要はキオスク用に最適化されています：

```
https://localhost:3443/#fleet?kiosk=true&cols=3&size=large
```

フリートビューのパラメーター：
- `cols=N` — 列数（1〜6）
- `size=small|medium|large` — カードサイズ

## 単一プリンターローテーション

プリンター間でローテーション（一度に1台）するには：

```
https://localhost:3443/?kiosk=true&rotate=true&interval=20
```

- `rotate=true` — ローテーションを有効化
- `interval=N` — プリンターごとの秒数

## Raspberry Pi / NUCでのセットアップ

専用のキオスクハードウェア向け：

### キオスクモードのChromium（Linux）

```bash
chromium-browser \
  --kiosk \
  --noerrdialogs \
  --disable-infobars \
  --no-first-run \
  --app="https://localhost:3443/?kiosk=true"
```

autostart（`~/.config/autostart/bambu-kiosk.desktop`）にコマンドを追加します。

### 自動ログインと起動

1. OSで自動ログインを設定します
2. Chromiumのautostart項目を作成します
3. スクリーンセーバーと省電力を無効にします：
   ```bash
   xset s off
   xset -dpms
   xset s noblank
   ```

:::tip 専用ユーザーアカウント
キオスクデバイス用に**ゲスト**ロールの専用3DPrintForgeユーザーアカウントを作成します。これにより、デバイスは読み取りアクセスのみを持ち、誰かが画面にアクセスしても設定を変更できません。
:::

## ハブ設定

ハブモードはすべてのプリンターと主要統計の概要ページを表示します — 大型テレビ向けに設計：

```
https://localhost:3443/#hub?kiosk=true
```

ハブビューには以下が含まれます：
- ステータス付きプリンターグリッド
- 集計KPI（アクティブプリント、総進捗）
- 時計と日付
- 最新のHMSアラート
