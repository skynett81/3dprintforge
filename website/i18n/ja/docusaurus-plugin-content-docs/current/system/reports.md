# レポート

Bambu Dashboardは統計とアクティビティサマリーを含む自動メールレポートを送信できます — 週次、月次、または両方。

アクセス先：**https://localhost:3443/#settings** → **システム → レポート**

## 前提条件

レポートにはメール通知の設定が必要です。レポートを有効にする前に**設定 → 通知 → メール**でSMTPを設定してください。[通知](../funksjoner/notifications)を参照してください。

## 自動レポートを有効にする

1. **設定 → レポート**に移動します
2. **週次レポート**および/または**月次レポート**を有効にします
3. **送信時刻**を選択します：
   - 週次：曜日と時刻
   - 月次：月の日（例：第1月曜日 / 最終金曜日）
4. **受信者メール**を入力します（複数の場合はカンマ区切り）
5. **保存**をクリックします

フォーマットを確認するためにテストレポートを送信：**今すぐテストレポートを送信**をクリックします。

## 週次レポートの内容

週次レポートは過去7日間をカバーします。

### サマリー
- 総プリント数
- 成功 / 失敗 / キャンセルの数
- 成功率と前週からの変化
- 最もアクティブなプリンター

### アクティビティ
- 1日あたりのプリント数（ミニグラフ）
- 総プリント時間
- 総フィラメント消費量（グラム数とコスト）

### フィラメント
- 素材・メーカー別消費量
- スプールごとの推定残量（20%未満のスプールが強調表示）

### メンテナンス
- 今週完了したメンテナンスタスク
- 期限切れのメンテナンスタスク（赤い警告）
- 来週期限のタスク

### HMSエラー
- 今週のプリンターごとのHMSエラー数
- 未確認のエラー（注意が必要）

## 月次レポートの内容

月次レポートは過去30日間をカバーし、週次レポートのすべてに加えて以下が含まれます。

### トレンド
- 前月との比較（%）
- アクティビティマップ（月のヒートマップ縮小版）
- 月次成功率の推移

### コスト
- 総フィラメントコスト
- 総電力コスト（電力測定が設定されている場合）
- 総消耗コスト
- 総メンテナンスコスト

### 消耗とヘルス
- プリンターごとのヘルススコア（前月からの変化付き）
- 交換時期が近づいているコンポーネント

### 統計ハイライト
- 最長の成功プリント
- 最も使用されたフィラメントタイプ
- 最もアクティブなプリンター

## レポートのカスタマイズ

1. **設定 → レポート → カスタマイズ**に移動します
2. 含めたいセクションにチェック/チェック解除します
3. **プリンターフィルター**を選択：すべてのプリンターまたは選択したもの
4. **ロゴ表示**を選択：ヘッダーにBambu Dashboardロゴを表示するかオフにする
5. **保存**をクリックします

## レポートアーカイブ

送信されたすべてのレポートが保存され、再度開くことができます。

1. **設定 → レポート → アーカイブ**に移動します
2. リストのレポートを選択します（日付順に並び替え）
3. **開く**をクリックしてHTMLバージョンを表示します
4. **PDFをダウンロード**をクリックしてレポートをダウンロードします

レポートは**90日後**に自動削除されます（設定可能）。
