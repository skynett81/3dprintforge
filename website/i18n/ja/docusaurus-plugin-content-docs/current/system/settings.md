# 設定

Bambu Dashboardのすべての設定は、明確なカテゴリーで1つのページにまとめられています。各カテゴリーの内容の概要です。

アクセス先：**https://localhost:3443/#settings**

## プリンター

登録済みプリンターを管理します：

| 設定 | 説明 |
|---|---|
| プリンターを追加 | シリアル番号とアクセスキーで新しいプリンターを登録 |
| プリンター名 | カスタム表示名 |
| プリンターモデル | X1C / X1C Combo / X1E / P1S / P1S Combo / P1P / P2S / P2S Combo / A1 / A1 Combo / A1 mini / H2S / H2D / H2C |
| MQTT接続 | Bambu Cloud MQTTまたはローカルMQTT |
| アクセスキー | Bambu LabアプリのLANアクセスコード |
| IPアドレス | ローカル（LAN）モード用 |
| カメラ設定 | 有効化/無効化、解像度 |

最初のプリンターのステップごとのセットアップは[はじめに](../kom-i-gang/oppsett)を参照してください。

## 通知

完全なドキュメントは[通知](../funksjoner/notifications)を参照してください。

クイック概要：
- 通知チャンネルの有効化/無効化（Telegram、Discord、メールなど）
- チャンネルごとのイベントフィルター
- サイレント時間（通知なしの時間帯）
- チャンネルごとのテストボタン

## テーマ

完全なドキュメントは[テーマ](./themes)を参照してください。

- ライト / ダーク / 自動モード
- 6つのカラーパレット
- カスタムアクセントカラー
- 角丸とコンパクト度

## OBSオーバーレイ

OBSオーバーレイの設定：

| 設定 | 説明 |
|---|---|
| デフォルトテーマ | dark / light / minimal |
| デフォルト位置 | オーバーレイの角 |
| デフォルトスケール | スケーリング（0.5〜2.0） |
| QRコードを表示 | オーバーレイにダッシュボードへのQRコードを表示 |

完全なURL構文とセットアップについては[OBSオーバーレイ](../funksjoner/obs-overlay)を参照してください。

## エネルギーと電力

| 設定 | 説明 |
|---|---|
| Tibber APIトークン | Tibberスポット価格へのアクセス |
| Nordpool価格エリア | 価格地域を選択 |
| 送電網料金（円/kWh） | 送電網料金レート |
| プリンター電力（W） | プリンターモデルごとの電力消費量を設定 |

## Home Assistant

| 設定 | 説明 |
|---|---|
| MQTTブローカー | IP、ポート、ユーザー名、パスワード |
| Discoveryプレフィックス | デフォルト：`homeassistant` |
| Discoveryを有効化 | HAにデバイスを公開 |

## Webhook

グローバルWebhook設定：

| 設定 | 説明 |
|---|---|
| Webhook URL | イベントの受信URL |
| シークレットキー | HMAC-SHA256署名 |
| イベントフィルター | 送信するイベント |
| リトライ回数 | 失敗時の試行回数（デフォルト：3） |
| タイムアウト | リクエストが諦めるまでの秒数（デフォルト：10） |

## キュー設定

| 設定 | 説明 |
|---|---|
| 自動ディスパッチ | 有効化/無効化 |
| ディスパッチ戦略 | 最初に空いている / 最も使用回数が少ない / ラウンドロビン |
| 確認を要求 | 送信前の手動承認 |
| スタガードスタート | キュー内のプリンター間の遅延 |

## セキュリティ

| 設定 | 説明 |
|---|---|
| セッション期間 | 自動ログアウトまでの時間/日数 |
| 2FAを強制 | すべてのユーザーに2FAを要求 |
| IPホワイトリスト | 特定のIPアドレスへのアクセスを制限 |
| HTTPS証明書 | カスタム証明書をアップロード |

## システム

| 設定 | 説明 |
|---|---|
| サーバーポート | デフォルト：3443 |
| ログフォーマット | JSON / テキスト |
| ログレベル | Error / Warn / Info / Debug |
| データベースクリーンアップ | 古い履歴の自動削除 |
| アップデート | 新バージョンを確認 |
