# テーマ

Bambu Dashboardには、お好みや用途に合わせて外観をカスタマイズできる柔軟なテーマシステムがあります。

アクセス先：**https://localhost:3443/#settings** → **テーマ**

## カラーモード

3つのモードから選択します：

| モード | 説明 |
|---|---|
| **ライト** | 明るい背景、暗いテキスト — 明るい部屋に適している |
| **ダーク** | 暗い背景、明るいテキスト — モニタリングに推奨の標準設定 |
| **自動** | OSの設定に従う（OS ダーク/ライト） |

テーマ設定の上部でモードを変更するか、ナビゲーションバーのショートカット（月/太陽アイコン）を使用します。

## カラーパレット

6つのプリセットカラーパレットが利用可能です：

| パレット | プライマリカラー | スタイル |
|---|---|---|
| **Bambu** | 緑（#00C853） | 標準、Bambu Lab teasing |
| **ブルーナイト** | 青（#2196F3） | 落ち着いてプロフェッショナル |
| **サンセット** | オレンジ（#FF6D00） | 温かくエネルギッシュ |
| **パープル** | 紫（#9C27B0） | クリエイティブで個性的 |
| **レッド** | 赤（#F44336） | ハイコントラスト、目立つ |
| **モノクローム** | グレー（#607D8B） | ニュートラルでミニマリスト |

パレットをクリックしてプレビューし、すぐに有効化します。

## カスタムアクセントカラー

独自の色をアクセントカラーとして使用します：

1. パレットセレクターの下の**カスタムカラー**をクリックします
2. カラーピッカーを使用するか、HEXコードを入力します（例：`#FF5722`）
3. プレビューがリアルタイムで更新されます
4. **適用**をクリックして有効化します

:::tip コントラスト
アクセントカラーが背景に対して十分なコントラストを持つことを確認してください。色が読みやすさの問題を引き起こす可能性がある場合（WCAG AA標準）、システムが警告します。
:::

## 角丸

ボタン、カード、要素の角丸を調整します：

| 設定 | 説明 |
|---|---|
| **シャープ** | 角丸なし（長方形スタイル） |
| **スモール** | 微妙な角丸（4 px） |
| **ミディアム** | 標準的な角丸（8 px） |
| **ラージ** | 明確な角丸（16 px） |
| **ピル** | 最大角丸（50 px） |

スライダーをドラッグして0〜50 pxの間で手動調整します。

## コンパクト度

インターフェースの密度をカスタマイズします：

| 設定 | 説明 |
|---|---|
| **スペーシャス** | 要素間のスペースが広い |
| **スタンダード** | バランスの取れたデフォルト設定 |
| **コンパクト** | より密な配置 — 画面に多くの情報を表示 |

コンパクトモードは1080p未満のディスプレイやキオスク表示に推奨されます。

## タイポグラフィ

フォントを選択します：

- **システム** — OSのデフォルトフォントを使用（読み込みが速い）
- **Inter** — クリアでモダン（デフォルトの選択）
- **JetBrains Mono** — モノスペース、データ値に適している
- **Nunito** — よりソフトで丸みのあるスタイル

## アニメーション

アニメーションをオフにするか調整します：

- **フル** — すべてのトランジションとアニメーションがアクティブ（デフォルト）
- **リデュースド** — 必要なアニメーションのみ（OS設定を尊重）
- **オフ** — 最高のパフォーマンスのためアニメーションなし

:::tip キオスクモード
キオスク表示には、遠くからの最適なパフォーマンスと読みやすさのために**コンパクト** + **ダーク** + **アニメーションリデュースド**を有効にします。[キオスクモード](./kiosk)を参照してください。
:::

## テーマ設定のエクスポートとインポート

テーマを他のユーザーと共有します：

1. **テーマをエクスポート**をクリック — `.json`ファイルがダウンロードされます
2. 他のBambu Dashboardユーザーとファイルを共有します
3. インポートは**テーマをインポート** → ファイルを選択で行います
