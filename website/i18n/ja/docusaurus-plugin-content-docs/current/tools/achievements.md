# 実績

実績（achievements）はプリントの旅におけるマイルストーンと特別な瞬間を報酬として与えるゲーミフィケーション要素です。実績を集めて次のアンロックに向けた進捗を確認しましょう。

アクセス先：**https://localhost:3443/#achievements**

## レア度等級

実績は4段階のレア度に分類されます。

| 等級 | 色 | 説明 |
|---|---|---|
| **コモン** | グレー | 達成しやすいシンプルなマイルストーン |
| **アンコモン** | 緑 | ある程度の努力や時間が必要 |
| **レア** | 青 | 長期にわたる継続的な努力が必要 |
| **レジェンダリー** | 金 | 並外れた偉業 |

## 実績の例

### プリントマイルストーン（コモン / アンコモン）
| 実績 | 条件 |
|---|---|
| 初プリント | 最初のプリントを完了する |
| 丸一日 | 合計24時間以上プリントする |
| 高成功率 | 10回連続で成功プリント |
| フィラメントコレクター | 10種類の異なるフィラメントを登録する |
| マルチカラー | マルチカラープリントを完了する |

### ボリューム実績（アンコモン / レア）
| 実績 | 条件 |
|---|---|
| 1キログラム | 合計1 kgのフィラメントを使用する |
| 10キログラム | 合計10 kgのフィラメントを使用する |
| 100プリント | 成功プリント100回 |
| 500時間 | 累積プリント時間500時間 |
| 夜勤 | 20時間以上のプリントを完了する |

### メンテナンスとケア（アンコモン / レア）
| 実績 | 条件 |
|---|---|
| 責任感のあるユーザー | メンテナンスタスクを記録する |
| プリンターケアラー | メンテナンスタスク10回を記録する |
| 無駄ゼロ | 素材効率90%超のプリントを作成する |
| マスターノズル | ノズルを5回交換する（ドキュメント化） |

### レジェンダリー実績
| 実績 | 条件 |
|---|---|
| 不屈 | 成功プリント1000回 |
| フィラメントタイタン | 総フィラメント消費量50 kg |
| エラーフリー週間 | 7日間で一度も失敗プリントなし |
| プリントライブラリアン | ファイルライブラリに100種類の異なるモデル |

## 実績を見る

実績ページには以下が表示されます。

- **アンロック済み** — 達成した実績（日付付き）
- **もうすぐ** — 達成に近い実績（プログレスバー）
- **ロック中** — まだ達成していない全実績

**等級**、**カテゴリー**、**ステータス**（アンロック済み / 進行中 / ロック中）でフィルタリングします。

## プログレスバー

カウント付きの実績にはプログレスバーが表示されます。

```
1キログラム — 1 kgのフィラメント
[████████░░] 847 g / 1000 g (84.7%)
```

## 通知

新しい実績を達成すると自動的に通知されます。
- 実績名とグラフィック付きの**ブラウザポップアップ**
- オプション：Telegram / Discord経由の通知（**設定 → 通知 → 実績**で設定）

## マルチユーザーサポート

複数ユーザーのシステムでは、各ユーザーが自分の実績プロフィールを持ちます。**ランキング（リーダーボード）**には以下でのランキングが表示されます。

- アンロックした実績の総数
- 総プリント数
- 総プリント時間

:::tip プライベートモード
**設定 → 実績 → リーダーボードから非表示**でリーダーボードをオフにしてプロフィールをプライベートに保てます。
:::
