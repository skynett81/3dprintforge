# ベッドメッシュ

ベッドメッシュツールは、ビルドプレートの平坦性を視覚的に表示します — 良好な定着と均一なファーストレイヤーに不可欠です。

アクセス先：**https://localhost:3443/#bedmesh**

## ベッドメッシュとは

Bambu Labのプリンターはプローブを使用してビルドプレートの表面をスキャンし、高さの偏差のマップ（メッシュ）を作成します。プリンターのファームウェアがプリント中に偏差を自動的に補正します。3DPrintForgeはこのマップを視覚化します。

## 可視化

### 3Dサーフェス

ベッドメッシュマップはインタラクティブな3Dサーフェスとして表示されます。

- マウスで表示を回転
- スクロールしてズームイン/アウト
- **トップビュー**をクリックして真上からの視点に
- **サイドビュー**をクリックしてプロフィールを確認

カラースケールは平均高さからの偏差を示します。
- **青** — 中心より低い（凹面）
- **緑** — ほぼ平坦（偏差0.1 mm未満）
- **黄** — 中程度の偏差（0.1〜0.2 mm）
- **赤** — 高い偏差（0.2 mm超）

### ヒートマップ

**ヒートマップ**をクリックしてメッシュマップの平坦な2D表示に切り替えます — ほとんどの人にとって読みやすい表示です。

ヒートマップには以下が表示されます。
- 各測定点の正確な偏差値（mm）
- マークされた問題点（偏差0.3 mm超）
- 測定のサイズ（行数 × 列数）

## UIからベッドメッシュをスキャン

:::warning 要件
スキャンにはプリンターがアイドル状態で、ベッド温度が安定していることが必要です。スキャン前にベッドを希望温度まで**事前に**加熱してください。
:::

1. **ベッドメッシュ**に移動します
2. ドロップダウンリストからプリンターを選択します
3. **今すぐスキャン**をクリックします
4. スキャン用のベッド温度を選択します：
   - **コールド**（室温）— 速いが精度は低い
   - **ウォーム**（PLA 50〜60°C、PETG 70〜90°C）— 推奨
5. ダイアログで確認 — プリンターが自動的にプローブシーケンスを開始します
6. スキャンが完了するまで待ちます（メッシュサイズによって3〜8分）
7. 新しいメッシュマップが自動的に表示されます

## キャリブレーションガイド

スキャン後、システムが具体的な推奨事項を提供します。

| 結果 | 推奨事項 |
|---|---|
| 偏差全体0.1 mm未満 | 優秀 — 対応不要 |
| 偏差0.1〜0.2 mm | 良好 — ファームウェアが補正を処理 |
| コーナーで偏差0.2 mm超 | ベッドスプリングを手動調整（可能な場合） |
| 偏差0.3 mm超 | ベッドが損傷または取り付けミスの可能性 |
| センターがコーナーより高い | 熱膨張 — 温かいベッドでは正常 |

:::tip 履歴比較
**前回と比較**をクリックしてメッシュマップが時間とともに変化したかどうかを確認 — プレートが徐々に反っているかを検出するのに便利です。
:::

## メッシュ履歴

すべてのメッシュスキャンはタイムスタンプ付きで保存されます。

1. ベッドメッシュのサイドパネルで**履歴**をクリックします
2. 2つのスキャンを選択して比較します（差分マップが表示されます）
3. 不要になった古いスキャンを削除します

## エクスポート

メッシュデータを以下としてエクスポートします。
- **PNG** — ヒートマップの画像（ドキュメント用）
- **CSV** — 各点のX、Y、高さ偏差を含む生データ
