# ラベル

ラベルツールはフィラメントスプール用のプロフェッショナルなラベルを生成します — QRコード、サーマルプリンター向けスプールラベル、視覚的識別用カラーカード。

アクセス先：**https://localhost:3443/#labels**

## QRコード

ダッシュボードのフィラメント情報にリンクするQRコードを生成します。

1. **ラベル → QRコード**に移動します
2. QRコードを生成するスプールを選択します
3. QRコードが自動生成されてプレビューに表示されます
4. **PNGをダウンロード**または**印刷**をクリックします

QRコードにはダッシュボードのフィラメントプロファイルへのURLが含まれます。スマートフォンでスキャンしてスプール情報を素早く確認できます。

### バッチ生成

1. **すべて選択**をクリックするか個別のスプールにチェックを入れます
2. **すべてのQRコードを生成**をクリックします
3. スプールごとに1つのPNGを含むZIPとしてダウンロードするか、すべてを一度に印刷します

## スプールラベル

完全なスプール情報を含むサーマルプリンター用プロフェッショナルラベル：

### ラベル内容（標準）

- スプールカラー（塗りつぶしカラーブロック）
- 素材名（大きな文字）
- メーカー
- カラーHEXコード
- 温度推奨値（ノズルとベッド）
- QRコード
- バーコード（オプション）

### サーマルプリンター用ZPL

Zebra、Brother、DymoプリンターのZPLコード（Zebra Programming Language）を生成します。

1. **ラベル → サーマル印刷**に移動します
2. ラベルサイズを選択：**25×54 mm** / **36×89 mm** / **62×100 mm**
3. スプールを選択します
4. **ZPLを生成**をクリックします
5. ZPLコードをプリンターに送信します：
   - **直接印刷**（USB接続）
   - **ZPLをコピー**してターミナルコマンドで送信
   - **.zplファイルをダウンロード**

:::tip プリンター設定
自動印刷のために、**設定 → ラベルプリンター**でIPアドレスとポート（標準：RAW TCPの場合9100）を使用してプリンターステーションを設定します。
:::

### PDFラベル

通常のプリンターの場合、適切なサイズのPDFを生成します。

1. テンプレートからラベルサイズを選択します
2. **PDFを生成**をクリックします
3. シール紙（Averyまたは同等品）に印刷します

## カラーカード

カラーカードはすべてのスプールを視覚的に表示するコンパクトなグリッドです。

1. **ラベル → カラーカード**に移動します
2. 含めるスプールを選択します（すべてのアクティブ、または手動選択）
3. カードフォーマットを選択：**A4**（4×8）、**A3**（6×10）、**レター**
4. **PDFを生成**をクリックします

各フィールドには以下が表示されます。
- 実際の色のカラーブロック
- 素材名とカラーHEX
- 素材番号（クイックリファレンス用）

ラミネートしてプリンターステーションの近くに貼るのに最適です。

## 共有カラーパレット

選択した色を共有パレットとしてエクスポートします。

1. **ラベル → カラーパレット**に移動します
2. パレットに含めるスプールを選択します
3. **パレットを共有**をクリックします
4. リンクをコピー — 他の人がそのパレットを自分のダッシュボードにインポートできます
5. パレットはHEXコード付きで表示され、**Adobe Swatch**（`.ase`）または**Procreate**（`.swatches`）にエクスポートできます
