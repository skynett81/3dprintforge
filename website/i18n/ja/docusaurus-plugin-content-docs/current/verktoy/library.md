# ファイルライブラリ

ファイルライブラリはすべての3DモデルとGコードファイルを保存・管理する中心的な場所です — 自動Gコード分析とMakerWorldおよびPrintablesとの統合機能付き。

アクセス先：**https://localhost:3443/#library**

## モデルのアップロード

### 単一アップロード

1. **ファイルライブラリ**に移動します
2. **アップロード**をクリックするか、ファイルをアップロードエリアにドラッグします
3. サポートされている形式：`.3mf`、`.gcode`、`.bgcode`、`.stl`、`.obj`
4. アップロード後にファイルが自動的に分析されます

:::info 保存フォルダ
ファイルは**設定 → ファイルライブラリ → 保存フォルダ**で設定したフォルダに保存されます。デフォルト：`./data/library/`
:::

### バッチアップロード

フォルダ全体をドラッグ＆ドロップして、サポートされているすべてのファイルを一度にアップロードします。ファイルはバックグラウンドで処理され、すべての準備ができたら通知されます。

## Gコード分析

アップロード後、`.gcode`および`.bgcode`ファイルが自動的に分析されます。

| 指標 | 説明 |
|---|---|
| 推定プリント時間 | Gコードコマンドから計算された時間 |
| フィラメント消費量 | 素材/色ごとのグラム数とメートル数 |
| レイヤーカウント | 総レイヤー数 |
| レイヤー厚さ | 記録されたレイヤー厚さ |
| 素材 | 検出された素材（PLA、PETGなど） |
| インフィル率 | メタデータで利用可能な場合 |
| サポート材 | 推定サポート重量 |
| プリンターモデル | メタデータからのターゲットプリンター |

分析データはファイルカードに表示され、[コスト計算機](../analyse/costestimator)でも使用されます。

## ファイルカードとメタデータ

各ファイルカードには以下が表示されます。
- **ファイル名**と形式
- **アップロード日**
- **サムネイル**（`.3mf`から、または生成）
- **分析済みプリント時間**とフィラメント消費量
- **タグ**とカテゴリー
- **関連プリント** — プリント回数

カードをクリックして、完全なメタデータと履歴を含む詳細ビューを開きます。

## 整理

### タグ

簡単な検索のためにタグを追加します。
1. ファイルをクリックして**メタデータを編集**を選択します
2. タグを入力します（カンマ区切り）：`benchy, test, PLA, キャリブレーション`
3. タグフィルターでライブラリを検索します

### カテゴリー

ファイルをカテゴリーに整理します。
- サイドバーで**新しいカテゴリー**をクリックします
- ファイルをカテゴリーにドラッグします
- カテゴリーはネスト可能です（サブカテゴリーをサポート）

## MakerWorldへのリンク

1. **設定 → 統合 → MakerWorld**に移動します
2. Bambu Labアカウントでログインします
3. ライブラリに戻り：ファイルをクリックして**MakerWorldにリンク**を選択します
4. MakerWorldでモデルを検索して正しい結果を選択します
5. メタデータ（デザイナー、ライセンス、評価）がMakerWorldからインポートされます

リンクにはファイルカードにデザイナー名と元のURLが表示されます。

## Printablesへのリンク

1. **設定 → 統合 → Printables**に移動します
2. PrintablesのAPIキーを貼り付けます
3. MakerWorldと同じ方法でファイルをPrintablesモデルにリンクします

## プリンターへの送信

ファイルライブラリから直接プリンターに送信できます。

1. ファイルをクリックして**プリンターに送信**を選択します
2. ターゲットプリンターを選択します
3. AMSスロットを選択します（マルチカラープリントの場合）
4. **プリント開始**または**キューに追加**をクリックします

:::warning 直接送信
直接送信はBambu Studioでの確認なしにすぐにプリントを開始します。プリンターの準備ができていることを確認してください。
:::
