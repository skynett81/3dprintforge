# プリントプロファイル

プリントプロファイルは複数のプリントやプリンターにわたって再利用できるプリント設定の保存済みセットです。さまざまな目的のプロファイルを定義することで時間を節約し、一貫した品質を確保できます。

アクセス先：**https://localhost:3443/#profiles**

## プロファイルを作成する

1. **ツール → プリントプロファイル**に移動します
2. **新しいプロファイル**（+アイコン）をクリックします
3. 以下を入力します：
   - **プロファイル名** — わかりやすい名前（例：「PLA - 高速生産」）
   - **素材** — リストから選択（PLA / PETG / ABS / PA / PC / TPU など）
   - **プリンターモデル** — X1C / X1C Combo / X1E / P1S / P1S Combo / P1P / P2S / P2S Combo / A1 / A1 Combo / A1 mini / H2S / H2D / H2C / すべて
   - **説明** — オプションのテキスト

4. 設定を入力します（以下のセクションを参照）
5. **プロファイルを保存**をクリックします

## プロファイルの設定

### 温度
| フィールド | 例 |
|---|---|
| ノズル温度 | 220°C |
| ベッド温度 | 60°C |
| チャンバー温度（X1C） | 35°C |

### 速度
| フィールド | 例 |
|---|---|
| 速度設定 | 標準 |
| 最高速度（mm/s） | 200 |
| 加速度 | 5000 mm/s² |

### 品質
| フィールド | 例 |
|---|---|
| レイヤー厚さ | 0.2 mm |
| インフィル率 | 15% |
| インフィルパターン | グリッド |
| サポート材 | 自動 |

### AMSとカラー
| フィールド | 説明 |
|---|---|
| パージ量 | 色変更時のフラッシング量 |
| 優先スロット | 優先するAMSスロット |

### 詳細設定
| フィールド | 説明 |
|---|---|
| 乾燥モード | 湿った素材用のAMS乾燥を有効化 |
| 冷却時間 | レイヤー間の冷却のための一時停止 |
| ファン速度 | 冷却ファン速度（パーセント） |

## プロファイルを編集する

1. リストのプロファイルをクリックします
2. **編集**（鉛筆アイコン）をクリックします
3. 変更を加えます
4. **保存**（上書き）または**名前を付けて保存**（コピーを作成）をクリックします

:::tip バージョン管理
変更を実験する際に動作するプロファイルを保持するために「名前を付けて保存」を使用します。
:::

## プロファイルを使用する

### ファイルライブラリから

1. ライブラリでファイルを選択します
2. **プリンターに送信**をクリックします
3. ドロップダウンリストから**プロファイル**を選択します
4. プロファイルの設定が使用されます

### プリントキューから

1. 新しいキュージョブを作成します
2. 設定の下で**プロファイル**を選択します
3. プロファイルがキュージョブに関連付けられます

## プロファイルのインポートとエクスポート

### エクスポート
1. 1つまたは複数のプロファイルを選択します
2. **エクスポート**をクリックします
3. 形式を選択：**JSON**（他のダッシュボードへのインポート用）または**PDF**（印刷/ドキュメント用）

### インポート
1. **プロファイルをインポート**をクリックします
2. 別のBambu Dashboardからエクスポートされた`.json`ファイルを選択します
3. 同じ名前の既存プロファイルを上書きするか、両方を保持するかを選択できます

## プロファイルを共有する

コミュニティフィラメントモジュール（[コミュニティフィラメント](../integrasjoner/community)を参照）またはJSON直接エクスポートを通じて他のユーザーとプロファイルを共有します。

## デフォルトプロファイル

素材ごとにデフォルトプロファイルを設定します。

1. プロファイルを選択します
2. **[素材]のデフォルトとして設定**をクリックします
3. その素材のファイルを送信する際にデフォルトプロファイルが自動的に選択されます
