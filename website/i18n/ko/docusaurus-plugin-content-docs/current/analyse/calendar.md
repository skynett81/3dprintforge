# 활동 캘린더

활동 캘린더는 GitHub의 기여 현황에서 영감을 받아, 연간 프린터 활동을 시각적으로 보여줍니다.

이동: **https://localhost:3443/#calendar**

## 히트맵 개요

캘린더는 365일(52주)을 색상이 있는 셀 격자로 표시합니다:

- **회색** — 해당 날 프린트 없음
- **연두색** — 1–2회 프린트
- **초록색** — 3–5회 프린트
- **진한 초록색** — 6–10회 프린트
- **짙은 초록색** — 11회 이상 프린트

셀은 요일 순으로 세로 배열(월~일)되며, 왼쪽(1월)에서 오른쪽(12월)으로 가로 배열됩니다.

:::tip 색상 코딩
히트맵 지표를 **프린트 횟수**에서 **시간** 또는 **필라멘트 그램**으로 캘린더 위의 선택기를 통해 변경할 수 있습니다.
:::

## 연도 선택기

**< 연도 >**를 클릭하여 연도 간 이동합니다:

- 프린트 활동이 기록된 모든 연도를 이용할 수 있습니다
- 현재 연도가 기본으로 표시됩니다
- 미래는 회색으로 표시됩니다(데이터 없음)

## 일별 상세 보기

셀을 클릭하면 해당 날의 상세 정보를 확인할 수 있습니다:

- **날짜** 및 요일
- **프린트 횟수** — 성공 및 실패
- **총 사용 필라멘트** (그램)
- **총 프린트 시간**
- **프린트 목록** — 클릭하여 기록에서 열기

## 월별 개요

히트맵 아래에 다음 내용이 포함된 월별 개요가 표시됩니다:
- 막대 차트로 월별 총 프린트 횟수
- 해당 월의 최고 활동일 강조 표시
- 전년도 같은 달과의 비교(%)

## 프린터 필터

상단 드롭다운에서 프린터를 선택하여 특정 프린터의 활동만 표시하거나, **전체**를 선택하여 통합 보기를 사용합니다.

다중 프린터 보기에서는 표시 선택기의 **누적**을 클릭하면 색상이 겹쳐 표시됩니다.

## 스트리크 및 기록

캘린더 아래에 다음 통계가 표시됩니다:

| 통계 | 설명 |
|---|---|
| **최장 스트리크** | 최소 1회 프린트가 있는 연속 일수 최고치 |
| **현재 스트리크** | 현재 진행 중인 활동 일수 연속 |
| **가장 활동적인 날** | 총 프린트 횟수가 가장 많은 날 |
| **가장 활동적인 주** | 프린트 횟수가 가장 많은 주 |
| **가장 활동적인 달** | 프린트 횟수가 가장 많은 달 |

## 내보내기

**내보내기**를 클릭하여 캘린더 데이터를 다운로드합니다:

- **PNG** — 히트맵 이미지(공유용)
- **CSV** — 날짜, 횟수, 그램, 시간이 포함된 일별 행의 원시 데이터

PNG 내보내기는 프린터 이름과 연도를 부제목으로 하여 소셜 미디어 공유에 최적화되어 있습니다.
