# 프린트 비교

프린트 비교를 통해 두 프린트를 나란히 분석할 수 있습니다 — 설정, 재료, 프린터 또는 동일 모델의 버전을 비교하는 데 유용합니다.

이동: **https://localhost:3443/#comparison**

## 비교할 프린트 선택

1. **프린트 비교**로 이동합니다
2. **프린트 A 선택**을 클릭하고 기록에서 검색합니다
3. **프린트 B 선택**을 클릭하고 기록에서 검색합니다
4. **비교**를 클릭하여 비교 보기를 로드합니다

:::tip 빠른 접근
**기록**에서 프린트를 우클릭하고 **프린트 A로 설정** 또는 **다음과 비교...**를 선택하여 비교 모드로 바로 이동할 수 있습니다.
:::

## 지표 비교

지표는 두 열(A와 B)로 표시되며, 어느 쪽이 더 좋은지 표시됩니다:

| 지표 | 설명 |
|---|---|
| 성공 여부 | 완료 / 중단 / 실패 |
| 소요 시간 | 총 프린트 시간 |
| 필라멘트 소비량 | 총 그램 및 색상별 그램 |
| 필라멘트 효율 | 총 소비량 대비 모델 비율(%) |
| 최고 노즐 온도 | 기록된 최고 노즐 온도 |
| 최고 베드 온도 | 기록된 최고 베드 온도 |
| 속도 설정 | 조용 / 표준 / 스포츠 / 터보 |
| AMS 교체 횟수 | 색상 변경 횟수 |
| HMS 오류 | 프린트 중 발생한 오류 |
| 프린터 | 사용된 프린터 |

더 나은 값을 가진 셀은 녹색 배경으로 표시됩니다.

## 온도 그래프

두 온도 그래프가 나란히(또는 겹쳐서) 표시됩니다:

- **별도 보기** — 왼쪽에 그래프 A, 오른쪽에 그래프 B
- **겹쳐 보기** — 서로 다른 색상으로 같은 그래프에 둘 다 표시

겹쳐 보기를 사용하면 온도 안정성과 가열 속도를 직접 비교할 수 있습니다.

## 갤러리 이미지

두 프린트 모두 마일스톤 스크린샷이 있는 경우, 그리드 형태로 표시됩니다:

| 프린트 A | 프린트 B |
|---|---|
| 25% 이미지 A | 25% 이미지 B |
| 50% 이미지 A | 50% 이미지 B |
| 75% 이미지 A | 75% 이미지 B |
| 100% 이미지 A | 100% 이미지 B |

이미지를 클릭하면 슬라이드 애니메이션과 함께 전체 화면 미리보기가 열립니다.

## 타임랩스 비교

두 프린트 모두 타임랩스가 있는 경우, 동영상이 나란히 표시됩니다:

- 동기화 재생 — 둘 다 동시에 시작하고 일시 정지
- 독립 재생 — 각 동영상을 별도로 제어

## 설정 차이점

시스템은 프린트 설정의 차이점(G-코드 메타데이터에서 가져옴)을 자동으로 강조 표시합니다:

- 다른 레이어 두께
- 다른 인필 패턴 또는 비율
- 다른 지지대 설정
- 다른 속도 프로파일

차이점은 설정 표에서 주황색으로 강조 표시됩니다.

## 비교 저장

1. **비교 저장**을 클릭합니다
2. 비교에 이름을 지정합니다(예: «PLA vs PETG - Benchy»)
3. 비교가 저장되며 **기록 → 비교**를 통해 접근할 수 있습니다

## 내보내기

1. **내보내기**를 클릭합니다
2. 모든 지표와 이미지가 포함된 보고서를 위해 **PDF**를 선택합니다
3. 보고서를 재료 선택 문서화를 위한 프로젝트에 연결할 수 있습니다
