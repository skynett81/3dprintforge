# 필라멘트 분석

필라멘트 분석을 통해 필라멘트 소비에 대한 완전한 통찰력을 얻을 수 있습니다 — 무엇을 사용하는지, 얼마나 드는지, 그리고 절약할 수 있는 것은 무엇인지를 확인합니다.

이동: **https://localhost:3443/#filament-analytics**

## 소비 개요

상단에는 선택한 기간에 대한 요약이 표시됩니다:

- **총 소비량** — 모든 재료의 그램 및 미터
- **예상 비용** — 스풀당 등록된 가격 기준
- **가장 많이 사용된 재료** — 유형 및 공급업체
- **재활용률** — 지지대/퍼지 대비 실제 모델의 필라멘트 비율

### 재료별 소비량

파이 차트와 표에 재료 간 분포가 표시됩니다:

| 열 | 설명 |
|---|---|
| 재료 | PLA, PETG, ABS, PA 등 |
| 공급업체 | Bambu Lab, PolyMaker, Prusament 등 |
| 사용 그램 | 총 무게 |
| 미터 | 예상 길이 |
| 비용 | 그램 × 그램당 가격 |
| 프린트 횟수 | 해당 재료를 사용한 프린트 수 |

행을 클릭하면 개별 스풀 수준까지 드릴다운할 수 있습니다.

## 소비율

소비율은 시간 단위당 평균 필라멘트 소비량을 보여줍니다:

- **시간당 그램** — 활성 프린팅 중
- **주당 그램** — 프린터 중단 시간 포함
- **프린트당 그램** — 출력물당 평균

이는 향후 필요량 예측을 계산하는 데 사용됩니다.

:::tip 구매 계획
소비율을 사용하여 스풀 재고를 계획합니다. 예상 재고가 14일 이내에 소진될 경우 시스템이 자동으로 경고를 보냅니다(구성 가능).
:::

## 비용 예측

과거 소비율을 기반으로 다음이 계산됩니다:

- **다음 30일 예상 소비량** (재료별 그램)
- **다음 30일 예상 비용**
- **권장 재고량** (30 / 60 / 90일 운영을 위한 충분한 양)

최소 1년치 데이터가 있는 경우 예측에 계절 변동이 반영됩니다.

## 낭비 및 효율성

전체 문서는 [낭비 추적](./waste)을 참조하세요. 필라멘트 분석은 요약을 표시합니다:

- **AMS 퍼지** — 총 소비량 대비 그램 및 비율
- **지지재** — 그램 및 비율
- **실제 모델 재료** — 나머지 비율(효율성 %)
- **낭비 예상 비용** — 낭비로 인해 발생하는 비용

## 스풀 기록

모든 스풀(활성 및 빈 것)이 기록됩니다:

| 항목 | 설명 |
|---|---|
| 스풀 이름 | 재료 이름 및 색상 |
| 초기 무게 | 시작 시 등록된 무게 |
| 남은 무게 | 계산된 잔량 |
| 사용량 | 총 사용 그램 |
| 마지막 사용 | 마지막 프린트 날짜 |
| 상태 | 활성 / 비어 있음 / 보관 중 |

## 가격 등록

정확한 비용 분석을 위해 스풀당 가격을 등록합니다:

1. **필라멘트 재고**로 이동합니다
2. 스풀을 클릭 → **편집**
3. **구매 가격**과 **구매 시 무게**를 입력합니다
4. 시스템이 그램당 가격을 자동으로 계산합니다

가격이 등록되지 않은 스풀은 **기본 그램당 가격**을 사용합니다(**설정 → 필라멘트 → 기본 가격**에서 설정).

## 내보내기

1. **필라멘트 데이터 내보내기**를 클릭합니다
2. 기간 및 형식(CSV / PDF)을 선택합니다
3. CSV에는 프린트당 그램, 비용 및 재료가 포함된 행이 포함됩니다
