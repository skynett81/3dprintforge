# 통계

통계 페이지는 선택 가능한 기간 동안의 핵심 지표, 트렌드 및 필라멘트 소비량을 포함한 프린터 활동에 대한 완전한 개요를 제공합니다.

이동: **https://localhost:3443/#statistics**

## 핵심 지표

페이지 상단에 네 개의 KPI 카드가 표시됩니다:

| 핵심 지표 | 설명 |
|---|---|
| **성공률** | 총 프린트 횟수 대비 성공한 프린트 비율 |
| **총 필라멘트** | 선택한 기간에 사용된 그램 |
| **총 프린트 시간** | 누적 프린트 시간 |
| **평균 프린트 시간** | 프린트당 중간 소요 시간 |

각 핵심 지표는 이전 기간 대비 변화(↑ 증가 / ↓ 감소)를 백분율로 표시합니다.

## 성공률

성공률은 프린터별 및 전체로 계산됩니다:

- **성공** — 중단 없이 완료된 프린트
- **중단** — 사용자가 수동으로 중단
- **실패** — Print Guard, HMS 오류 또는 하드웨어 결함으로 중단

성공률 차트를 클릭하면 어떤 프린트가 실패했는지와 그 이유를 확인할 수 있습니다.

:::tip 성공률 향상
[오류 패턴 분석](../overvaaking/erroranalysis)을 사용하여 실패한 프린트의 원인을 파악하고 수정합니다.
:::

## 트렌드

트렌드 보기는 시간에 따른 변화를 선형 차트로 표시합니다:

1. **기간** 선택: 최근 7 / 30 / 90 / 365일
2. **그룹화** 선택: 일 / 주 / 월
3. **지표** 선택: 프린트 횟수 / 시간 / 그램 / 성공률
4. **비교**를 클릭하여 두 지표를 겹쳐 표시

그래프는 확대(스크롤)와 이동(클릭 및 드래그)을 지원합니다.

## 필라멘트 소비량

필라멘트 소비량이 다음으로 표시됩니다:

- **막대 차트** — 일/주/월별 소비량
- **파이 차트** — 재료 간 분포(PLA, PETG, ABS 등)
- **표** — 재료별 총 그램, 미터 및 비용이 포함된 상세 목록

### 프린터별 소비량

상단의 다중 선택 필터를 사용하여:
- 한 프린터만 표시
- 두 프린터를 나란히 비교
- 모든 프린터의 집계 합계 보기

## 활동 캘린더

통계 페이지에서 간소화된 보기로 GitHub 스타일 히트맵을 직접 확인하거나, 더 자세한 보기를 위해 전체 [활동 캘린더](./calendar)로 이동합니다.

## 내보내기

1. **통계 내보내기**를 클릭합니다
2. 날짜 범위와 포함할 지표를 선택합니다
3. 형식 선택: **CSV**(원시 데이터), **PDF**(보고서) 또는 **JSON**
4. 파일이 다운로드됩니다

CSV 내보내기는 추가 분석을 위해 Excel 및 Google 스프레드시트와 호환됩니다.

## 이전 기간과의 비교

**이전 기간 표시**를 활성화하면 해당하는 이전 기간과 그래프를 겹쳐 볼 수 있습니다:

- 최근 30일 vs. 이전 30일
- 현재 월 vs. 지난 달
- 현재 연도 vs. 작년

이를 통해 이전보다 더 많이 또는 적게 프린트하고 있는지, 그리고 성공률이 향상되고 있는지 쉽게 확인할 수 있습니다.
