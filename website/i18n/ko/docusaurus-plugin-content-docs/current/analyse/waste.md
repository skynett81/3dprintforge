# 낭비 추적

낭비 추적은 프린팅 중에 얼마나 많은 필라멘트가 낭비되는지 — AMS 퍼지, 재료 교체 시 세척, 지지재 — 그리고 그 비용에 대한 완전한 통찰력을 제공합니다.

이동: **https://localhost:3443/#waste**

## 낭비 카테고리

Bambu Dashboard는 세 가지 유형의 낭비를 구분합니다:

| 카테고리 | 출처 | 일반적인 양 |
|---|---|---|
| **AMS 퍼지** | 멀티컬러 프린트 중 AMS의 색상 변경 | 교체당 5–30g |
| **재료 교체 세척** | 서로 다른 재료 간 교체 시 정리 | 교체당 10–50g |
| **지지재** | 프린트 후 제거하는 지지 구조물 | 다양함 |

## AMS 퍼지 추적

AMS 퍼지 데이터는 MQTT 텔레메트리 및 G-코드 분석에서 직접 가져옵니다:

- **색상 변경당 그램** — G-코드 퍼지 블록에서 계산
- **색상 변경 횟수** — 프린트 기록에서 집계
- **총 퍼지 소비량** — 선택한 기간의 합계

:::tip 퍼지 줄이기
Bambu Studio에는 색상 조합당 퍼지 볼륨 설정이 있습니다. 낮은 색상 차이(예: 흰색 → 밝은 회색)를 가진 색상 쌍의 퍼지 볼륨을 줄이면 필라멘트를 절약할 수 있습니다.
:::

## 효율성 계산

효율성은 다음과 같이 계산됩니다:

```
효율성 % = (모델 재료 / 총 소비량) × 100

총 소비량 = 모델 재료 + 퍼지 + 지지재
```

**예시:**
- 모델: 45g
- 퍼지: 12g
- 지지대: 8g
- 합계: 65g
- **효율성: 69%**

효율성은 시간에 따른 개선 여부를 확인하기 위해 트렌드 차트로 표시됩니다.

## 낭비 비용

등록된 필라멘트 가격을 기반으로 다음이 계산됩니다:

| 항목 | 계산 |
|---|---|
| 퍼지 비용 | 퍼지 그램 × 색상별 그램당 가격 |
| 지지대 비용 | 지지대 그램 × 그램당 가격 |
| **총 낭비 비용** | 위의 합계 |
| **성공한 프린트당 비용** | 낭비 비용 / 프린트 횟수 |

## 프린터 및 재료별 낭비

다음으로 보기를 필터링합니다:

- **프린터** — 어떤 프린터가 가장 많은 낭비를 발생시키는지 확인
- **재료** — 필라멘트 유형별 낭비 확인
- **기간** — 일, 주, 월, 연도

표 보기에는 예상 비용을 포함하여 낭비가 가장 많은 항목부터 순위가 매겨진 목록이 표시됩니다.

## 최적화 팁

시스템은 낭비를 줄이기 위한 자동 제안을 생성합니다:

- **색상 순서 반전** — 색상 A→B가 B→A보다 더 많이 퍼지하는 경우 시스템이 순서 변경을 제안합니다
- **색상 변경 레이어 통합** — 같은 색상의 레이어를 그룹화하여 변경을 최소화합니다
- **지지 구조 최적화** — 방향 변경으로 지지대 감소 추정

:::info 정확도
퍼지 계산은 G-코드 기반 추정치입니다. 실제 낭비는 프린터 동작으로 인해 10–20% 차이가 날 수 있습니다.
:::

## 내보내기 및 보고

1. **낭비 데이터 내보내기**를 클릭합니다
2. 기간 및 형식(CSV / PDF)을 선택합니다
3. 낭비 데이터를 프로젝트 보고서 및 인보이스에 비용 항목으로 포함할 수 있습니다

[필라멘트 분석](./filamentanalytics)도 참조하여 전체 소비 개요를 확인하세요.
