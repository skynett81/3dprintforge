# 비용 계산기

비용 계산기를 사용하면 프린터로 전송하기 전에 필라멘트 소비량, 전력 요금 및 기계 마모를 기반으로 프린트 총 비용을 추정할 수 있습니다.

이동: **https://localhost:3443/#cost-estimator**

## 파일 업로드

1. **비용 계산기**로 이동합니다
2. 업로드 필드에 파일을 드래그 앤 드롭하거나 **파일 선택**을 클릭합니다
3. 지원 형식: `.3mf`, `.gcode`, `.bgcode`
4. **분석**을 클릭합니다

:::info 분석
시스템은 G-코드를 분석하여 필라멘트 소비량, 예상 프린트 시간 및 재료 프로파일을 추출합니다. 일반적으로 2–10초가 소요됩니다.
:::

## 필라멘트 계산

분석 후 다음이 표시됩니다:

| 항목 | 값(예시) |
|---|---|
| 예상 필라멘트 | 47.3 g |
| 재료(파일에서) | PLA |
| 그램당 가격 | 0.025원(필라멘트 재고에서) |
| **필라멘트 비용** | **1.18원** |

드롭다운에서 재료를 변경하여 다양한 필라멘트 유형이나 공급업체의 비용을 비교합니다.

:::tip 재료 재정의
G-코드에 재료 정보가 없는 경우, 목록에서 재료를 수동으로 선택합니다. 가격은 필라멘트 재고에서 자동으로 가져옵니다.
:::

## 전력 계산

전력 비용은 다음을 기반으로 계산됩니다:

- **예상 프린트 시간** — G-코드 분석에서
- **프린터 소비 전력** — 프린터 모델별로 구성(W)
- **전기 요금** — 고정 요금(원/kWh) 또는 Tibber/Nordpool의 실시간 요금

| 항목 | 값(예시) |
|---|---|
| 예상 프린트 시간 | 3시간 22분 |
| 프린터 소비 전력 | 350 W (X1C) |
| 예상 전력 소비 | 1.17 kWh |
| 전기 요금 | 1.85원/kWh |
| **전력 비용** | **2.16원** |

원하는 시작 시간을 기반으로 예정된 시간별 요금을 사용하려면 Tibber 또는 Nordpool 통합을 활성화하세요.

## 기계 마모

마모 비용은 다음을 기반으로 추정됩니다:

- 프린트 시간 × 프린터 모델별 시간당 비용
- 마모성 재료(CF, GF 등)에 대한 추가 마모

| 항목 | 값(예시) |
|---|---|
| 프린트 시간 | 3시간 22분 |
| 시간당 비용(마모) | 0.80원/시간 |
| **마모 비용** | **2.69원** |

시간당 비용은 부품 가격과 예상 수명에서 계산됩니다([마모 예측](../monitoring/wearprediction) 참조).

## 총계

| 비용 항목 | 금액 |
|---|---|
| 필라멘트 | 1.18원 |
| 전력 | 2.16원 |
| 기계 마모 | 2.69원 |
| **합계** | **6.03원** |
| + 마진(30%) | 1.81원 |
| **판매 가격** | **7.84원** |

고객에게 권장 판매 가격을 계산하기 위해 비율 필드에서 마진을 조정합니다.

## 견적 저장

**견적 저장**을 클릭하여 분석을 프로젝트에 연결합니다:

1. 기존 프로젝트를 선택하거나 새로 생성합니다
2. 견적이 저장되며 인보이스 기준으로 사용할 수 있습니다
3. 실제 비용(프린트 후)이 자동으로 견적과 비교됩니다

## 배치 계산

여러 파일을 동시에 업로드하여 완전한 세트의 총 비용을 계산합니다:

1. **배치 모드**를 클릭합니다
2. 모든 `.3mf`/`.gcode` 파일을 업로드합니다
3. 시스템이 개별 및 합산 비용을 계산합니다
4. 요약을 PDF 또는 CSV로 내보냅니다
