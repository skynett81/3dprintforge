# 프린터 제어

제어 패널을 통해 대시보드에서 프린터를 완전히 수동으로 제어할 수 있습니다.

## 온도 제어

### 노즐
- 0–350°C 사이에서 목표 온도 설정
- **설정**을 클릭하여 명령 전송
- 애니메이션 링 게이지로 실시간 수치 표시

### 베드
- 0–120°C 사이에서 목표 온도 설정
- 프린트 후 자동 냉각(구성 가능)

### 챔버
- 챔버 온도 확인(실시간 수치)
- **X1E, H2S, H2D, H2C**: M141을 통한 능동 챔버 가열 제어(제어 가능한 목표 온도)
- **X1C**: 수동 인클로저 — 챔버 온도 표시, 직접 제어 불가
- **P1S**: 수동 인클로저 — 온도 표시, 능동 챔버 가열 없음
- **P1P, A1, A1 mini 및 chamberHeat 없는 H 시리즈**: 챔버 센서 없음

:::warning 최고 온도
노즐 및 베드에 권장 온도를 초과하지 마세요. 강화 강철 노즐(HF 유형): 최대 300°C. 황동: 최대 260°C. 프린터 설명서를 참조하세요.
:::

## 속도 프로필

속도 제어는 네 가지 사전 설정된 프로필을 제공합니다:

| 프로필 | 속도 | 용도 |
|--------|----------|-------------|
| 조용 | 50% | 소음 감소, 야간 프린팅 |
| 표준 | 100% | 일반 사용 |
| 스포츠 | 124% | 더 빠른 프린팅 |
| 터보 | 166% | 최대 속도(품질 저하) |

슬라이더를 사용하면 50–200% 사이에서 사용자 지정 비율을 설정할 수 있습니다.

## 팬 제어

팬 속도를 수동으로 제어합니다:

| 팬 | 설명 | 범위 |
|-------|-------------|--------|
| 부품 냉각 팬 | 프린트 객체 냉각 | 0–100% |
| 보조 팬 | 챔버 순환 | 0–100% |
| 챔버 팬 | 능동 챔버 냉각 | 0–100% |

:::tip 권장 설정
- **PLA/PETG:** 부품 냉각 100%, 보조 팬 30%
- **ABS/ASA:** 부품 냉각 0–20%, 챔버 팬 끔
- **TPU:** 부품 냉각 50%, 낮은 속도
:::

## G-코드 콘솔

프린터에 G-코드 명령을 직접 전송합니다:

```gcode
; 예시: 헤드 위치 이동
G28 ; 모든 축 홈
G1 X150 Y150 Z10 F3000 ; 중앙으로 이동
M104 S220 ; 노즐 온도 설정
M140 S60  ; 베드 온도 설정
```

:::danger G-코드 주의
잘못된 G-코드는 프린터를 손상시킬 수 있습니다. 이해하는 명령만 전송하세요. 프린트 중간에 `M600`(필라멘트 교체)을 사용하지 마세요.
:::

## 필라멘트 작업

제어 패널에서 다음을 수행할 수 있습니다:

- **필라멘트 로드** — 노즐을 가열하고 필라멘트를 공급
- **필라멘트 언로드** — 가열 후 필라멘트 제거
- **노즐 청소** — 청소 사이클 실행

## 매크로

G-코드 명령 시퀀스를 매크로로 저장하고 실행합니다:

1. **새 매크로**를 클릭합니다
2. 매크로에 이름을 지정합니다
3. G-코드 시퀀스를 작성합니다
4. 저장하고 한 번의 클릭으로 실행합니다

베드 보정 매크로 예시:
```gcode
G28
M84
M500
```

## 프린트 제어

활성 프린트 중 다음을 수행할 수 있습니다:

- **일시 정지** — 현재 레이어 후 프린트 일시 정지
- **재개** — 일시 정지된 프린트 계속
- **중지** — 프린트 중단(되돌릴 수 없음)
- **비상 정지** — 모든 모터 즉시 정지
