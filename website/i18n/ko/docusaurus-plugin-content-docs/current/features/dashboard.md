# 메인 패널

메인 패널은 3DPrintForge의 중앙 제어 센터입니다. 선택한 프린터의 실시간 상태를 표시하고 필요에 따라 보기를 모니터링, 제어 및 사용자 지정할 수 있습니다.

이동: **https://localhost:3443/**

## 실시간 개요

프린터가 활성화되면 모든 값이 MQTT를 통해 지속적으로 업데이트됩니다:

- **노즐 온도** — 목표 온도가 있는 애니메이션 SVG 링 게이지
- **베드 온도** — 베드에 해당하는 링 게이지
- **진행률** — 남은 시간이 있는 큰 비율 표시기
- **레이어 카운터** — 현재 레이어 / 총 레이어 수
- **속도** — 슬라이더가 있는 조용 / 표준 / 스포츠 / 터보

:::tip 실시간 업데이트
모든 값은 페이지를 새로 고침하지 않고 MQTT를 통해 프린터에서 직접 업데이트됩니다. 지연은 일반적으로 1초 미만입니다.
:::

## 3D 모델 보기

프린터가 모델이 포함된 `.3mf` 파일을 전송하는 경우 대화형 3D 미리보기가 표시됩니다:

1. 프린트가 시작될 때 모델이 자동으로 로드됩니다
2. 마우스로 드래그하여 모델 회전
3. 스크롤하여 확대/축소
4. **재설정**을 클릭하여 기본 보기로 돌아가기

:::info 지원
3D 보기는 프린터가 모델 데이터를 전송해야 합니다. 모든 프린트 작업이 이를 포함하지는 않습니다.
:::

## AMS 상태

AMS 패널은 슬롯 및 필라멘트가 있는 모든 장착된 AMS 장치를 표시합니다:

- **슬롯 색상** — Bambu 메타데이터의 시각적 색상 표현
- **필라멘트 이름** — 재료 및 브랜드
- **활성 슬롯** — 프린팅 중 펄스 애니메이션으로 표시
- **오류** — AMS 오류 시 빨간색 표시기(막힘, 빈 것, 습기)

슬롯을 클릭하면 전체 필라멘트 정보를 확인하고 필라멘트 재고에 연결할 수 있습니다.

## 카메라 피드

라이브 카메라 보기는 ffmpeg(RTSPS → MPEG1)을 통해 변환됩니다:

1. 대시보드를 열면 카메라가 자동으로 시작됩니다
2. 카메라 화면을 클릭하면 전체 화면으로 열립니다
3. **스냅샷** 버튼을 사용하여 정지 이미지 촬영
4. **카메라 숨기기**를 클릭하여 공간 확보

:::warning 성능
카메라 스트림은 약 2–5Mbit/s를 사용합니다. 느린 네트워크 연결에서는 카메라를 비활성화하세요.
:::

## 온도 스파크라인

AMS 패널 아래에 최근 30분에 대한 미니 그래프(스파크라인)가 표시됩니다:

- 시간에 따른 노즐 온도
- 시간에 따른 베드 온도
- 챔버 온도(사용 가능한 경우)

스파크라인을 클릭하면 전체 텔레메트리 그래프 보기가 열립니다.

## 위젯 사용자 지정

대시보드는 드래그 앤 드롭 그리드 레이아웃을 사용합니다:

1. **레이아웃 사용자 지정**(오른쪽 상단 연필 아이콘)을 클릭합니다
2. 위젯을 원하는 위치로 드래그합니다
3. 모서리를 드래그하여 크기 변경
4. **레이아웃 잠금**을 클릭하여 위치 고정
5. **저장**을 클릭하여 레이아웃 유지

사용 가능한 위젯:
| 위젯 | 설명 |
|---|---|
| 카메라 | 라이브 카메라 보기 |
| AMS | 스풀 및 필라멘트 상태 |
| 온도 | 노즐 및 베드용 링 게이지 |
| 진행률 | 비율 표시기 및 시간 추정 |
| 텔레메트리 | 팬, 압력, 속도 |
| 3D 모델 | 대화형 모델 보기 |
| 스파크라인 | 미니 온도 그래프 |

:::tip 저장
레이아웃은 브라우저(localStorage)에서 사용자별로 저장됩니다. 다른 사용자는 다른 레이아웃을 가질 수 있습니다.
:::
