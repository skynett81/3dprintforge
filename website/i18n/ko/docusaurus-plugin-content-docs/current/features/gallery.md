# 갤러리

갤러리는 각 프린트 중에 자동으로 촬영된 스크린샷을 수집합니다. 이미지는 고정된 마일스톤에서 촬영되며 프린트 진행의 시각적 기록을 제공합니다.

이동: **https://localhost:3443/#gallery**

## 마일스톤 스크린샷

Bambu Dashboard는 다음 마일스톤에서 카메라의 스크린샷을 자동으로 촬영합니다:

| 마일스톤 | 시점 |
|---|---|
| **25%** | 프린트의 사분의 일 지점 |
| **50%** | 절반 지점 |
| **75%** | 사분의 삼 지점 |
| **100%** | 프린트 완료 |

스크린샷은 해당 프린트 기록 항목에 연결되어 갤러리에 표시됩니다.

:::info 요구 사항
마일스톤 스크린샷은 카메라가 연결되어 있고 활성화되어 있어야 합니다. 비활성화된 카메라는 이미지를 생성하지 않습니다.
:::

## 스크린샷 기능 활성화

1. **설정 → 갤러리**로 이동합니다
2. **자동 마일스톤 스크린샷**을 켭니다
3. 활성화할 마일스톤을 선택합니다(기본적으로 4개 모두 켜져 있음)
4. **이미지 품질** 선택: 낮음(640×360) / 중간(1280×720) / 높음(1920×1080)
5. **저장**을 클릭합니다

## 이미지 보기

갤러리는 프린트별로 구성됩니다:

1. 상단의 **필터**를 사용하여 프린터, 날짜 또는 파일 이름을 선택합니다
2. 프린트 행을 클릭하여 확장하고 네 이미지를 모두 봅니다
3. 이미지를 클릭하면 미리보기가 열립니다

### 미리보기

미리보기에 표시됩니다:
- 전체 크기 이미지
- 마일스톤 및 타임스탬프
- 프린트 이름 및 프린터
- **←** / **→**으로 같은 프린트의 이미지 간 탐색

## 전체 화면 보기

미리보기에서 **전체 화면**(또는 `F` 키 누르기)을 클릭하면 전체 화면으로 표시됩니다. 방향키를 사용하여 이미지 간 탐색합니다.

## 이미지 다운로드

- **단일 이미지**: 미리보기에서 **다운로드** 클릭
- **프린트의 모든 이미지**: 프린트 행에서 **모두 다운로드** 클릭 — `.zip` 파일을 받습니다
- **여러 선택**: 체크박스를 선택하고 **선택 항목 다운로드** 클릭

## 이미지 삭제

:::warning 저장 공간
갤러리 이미지는 시간이 지남에 따라 상당한 공간을 차지할 수 있습니다. 오래된 이미지의 자동 삭제를 설정하세요.
:::

### 수동 삭제

1. 하나 이상의 이미지를 선택합니다(체크박스)
2. **선택 항목 삭제**를 클릭합니다
3. 대화 상자에서 확인합니다

### 자동 정리

1. **설정 → 갤러리 → 자동 정리**로 이동합니다
2. **다음보다 오래된 이미지 삭제** 활성화
3. 일 수를 설정합니다(예: 90일)
4. 정리는 매일 밤 03:00에 자동으로 실행됩니다

## 프린트 기록과의 연결

각 이미지는 기록의 프린트 항목과 연결됩니다:

- 갤러리의 프린트에서 **기록에서 보기**를 클릭하면 기록 항목으로 이동합니다
- 기록에서는 100% 이미지의 썸네일이 있으면 표시됩니다

## 공유

시간 제한이 있는 링크로 갤러리 이미지를 공유합니다:

1. 미리보기에서 이미지 열기
2. **공유** 클릭
3. 만료 시간을 선택하고 링크를 복사합니다
