# 기능 개요

3DPrintForge는 Bambu Lab 프린터를 모니터링하고 제어하는 데 필요한 모든 것을 한 곳에 모았습니다.

## 대시보드

메인 대시보드는 활성 프린터의 실시간 상태를 표시합니다:

- **온도** — 노즐 및 베드용 애니메이션 SVG 링 게이지
- **진행률** — 예상 완료 시간이 있는 진행률
- **카메라** — 라이브 카메라 보기(RTSPS → ffmpeg을 통한 MPEG1)
- **AMS 패널** — 필라멘트 색상이 있는 모든 AMS 슬롯의 시각적 표현
- **속도 제어** — 속도 조정 슬라이더(조용, 표준, 스포츠, 터보)
- **통계 패널** — 롤링 그래프가 있는 Grafana 스타일 패널
- **텔레메트리** — 팬, 온도, 압력의 라이브 값

패널을 드래그 앤 드롭하여 레이아웃을 사용자 지정할 수 있습니다. 잠금 버튼을 사용하여 레이아웃을 잠급니다.

## 필라멘트 재고

전체 문서는 [필라멘트](./filament)를 참조하세요.

- 이름, 색상, 무게 및 공급업체와 함께 모든 스풀 추적
- AMS 동기화 — AMS에 어떤 스풀이 있는지 확인
- 건조 로그 및 건조 계획
- 색상 카드 및 NFC 태그 지원
- 가져오기/내보내기(CSV)

## 프린트 기록

전체 문서는 [기록](./history)을 참조하세요.

- 모든 프린트의 완전한 기록
- 프린트당 필라멘트 추적
- MakerWorld 모델 링크
- 통계 및 CSV 내보내기

## 스케줄러

전체 문서는 [스케줄러](./scheduler)를 참조하세요.

- 프린트의 캘린더 보기
- 우선순위가 있는 프린트 대기열
- 다중 프린터 디스패치

## 프린터 제어

전체 문서는 [제어](./controls)를 참조하세요.

- 온도 제어(노즐, 베드, 챔버)
- 속도 프로필 제어
- 팬 제어
- G-코드 콘솔
- 필라멘트 로드/언로드

## 알림

3DPrintForge는 7개의 알림 채널을 지원합니다:

| 채널 | 이벤트 |
|-------|----------|
| Telegram | 프린트 완료, 오류, 일시 정지 |
| Discord | 프린트 완료, 오류, 일시 정지 |
| 이메일 | 프린트 완료, 오류 |
| ntfy | 모든 이벤트 |
| Pushover | 모든 이벤트 |
| SMS (Twilio) | 중요 오류 |
| 웹훅 | 사용자 지정 페이로드 |

**설정 → 알림**에서 구성합니다.

## Print Guard

Print Guard는 카메라(xcam) 및 센서를 통해 활성 프린트를 모니터링합니다:

- 스파게티 오류 시 자동 일시 정지
- 구성 가능한 감도 수준
- 감지된 이벤트 로그

## 유지 관리

유지 관리 섹션은 다음을 추적합니다:

- 구성 요소별 다음 권장 서비스(노즐, 플레이트, AMS)
- 프린트 기록 기반 마모 추적
- 유지 관리 작업 수동 등록

## 다중 프린터

다중 프린터 지원으로 다음이 가능합니다:

- 하나의 대시보드에서 여러 프린터 관리
- 프린터 선택기로 프린터 간 전환
- 모든 프린터 상태를 동시에 개요 보기
- 프린트 대기열로 프린트 작업 배분

## OBS 오버레이

전용 `obs.html` 페이지는 프린트 라이브 스트리밍 중 OBS Studio 통합을 위한 깔끔한 오버레이를 제공합니다.

## 업데이트

GitHub Releases를 통한 내장 자동 업데이트. **설정 → 업데이트**에서 대시보드를 통해 직접 알림 및 업데이트를 확인합니다.
