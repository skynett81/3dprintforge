# 스케줄러

스케줄러를 사용하면 캘린더 보기와 지능형 출력 대기열로 프린트 작업을 구성하고 자동화할 수 있습니다.

## 캘린더 보기

캘린더 보기는 계획된 프린트와 완료된 프린트의 개요를 제공합니다:

- **월별, 주별 및 일별 보기** — 세부 수준 선택
- **색상 코딩** — 프린터 및 상태별로 다른 색상
- **이벤트 클릭** — 프린트에 대한 세부 정보 보기

완료된 프린트는 프린트 기록을 기반으로 자동으로 표시됩니다.

## 출력 대기열

출력 대기열을 사용하면 프린터로 순서대로 전송되는 작업을 줄 세울 수 있습니다:

### 대기열에 작업 추가

1. **+ 작업 추가** 클릭
2. 파일 선택(프린터 SD, 로컬 업로드 또는 FTP에서)
3. 우선순위 지정(높음, 보통, 낮음)
4. 대상 프린터 선택(또는 "자동")
5. **추가** 클릭

### 대기열 관리

| 작업 | 설명 |
|----------|-------------|
| 드래그 앤 드롭 | 순서 재정렬 |
| 대기열 일시 정지 | 디스패치를 임시로 중단 |
| 건너뛰기 | 대기하지 않고 다음 작업 전송 |
| 삭제 | 대기열에서 작업 제거 |

:::tip 다중 프린터 디스패치
여러 프린터를 사용하면 대기열이 사용 가능한 프린터에 자동으로 작업을 배분합니다. **스케줄러 → 설정**에서 **자동 디스패치**를 활성화합니다.
:::

## 예약된 프린트

특정 시간에 시작할 프린트를 설정합니다:

1. **+ 프린트 예약** 클릭
2. 파일 및 프린터 선택
3. 시작 시간 지정
4. 알림 구성(선택 사항)
5. 저장

:::warning 프린터가 사용 가능해야 함
예약된 프린트는 지정된 시간에 프린터가 대기 모드인 경우에만 시작됩니다. 프린터가 사용 중인 경우 다음 사용 가능한 시간으로 시작이 연기됩니다(구성 가능).
:::

## 부하 분산

자동 부하 분산으로 프린터 간에 지능적으로 작업이 배분됩니다:

- **라운드 로빈** — 모든 프린터 간에 균등한 배분
- **가장 적게 사용됨** — 예상 완료 시간이 가장 짧은 프린터에 전송
- **수동** — 각 작업에 대해 직접 프린터 선택

**스케줄러 → 부하 분산**에서 구성합니다.

## 알림

스케줄러는 알림 채널과 통합됩니다:

- 작업 시작 시 알림
- 작업 완료 시 알림
- 오류 또는 지연 시 알림

알림 채널 구성은 [기능 개요](./overview#알림)를 참조하세요.
