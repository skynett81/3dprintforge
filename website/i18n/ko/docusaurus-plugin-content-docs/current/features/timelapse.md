# 타임랩스

3DPrintForge는 프린팅 중 자동으로 사진을 촬영하여 타임랩스 영상으로 합성합니다. 영상은 로컬에 저장되며 대시보드에서 직접 재생할 수 있습니다.

이동: **https://localhost:3443/#timelapse**

## 활성화

1. **설정 → 타임랩스**로 이동합니다
2. **타임랩스 녹화 활성화** 켜기
3. **녹화 모드** 선택:
   - **레이어별** — 레이어마다 한 장 촬영(고품질 권장)
   - **시간 기반** — N초마다 한 장 촬영(예: 30초마다)
4. 타임랩스를 활성화할 프린터 선택
5. **저장** 클릭

:::tip 촬영 간격
「레이어별」은 움직임이 일정하여 가장 부드러운 애니메이션을 제공합니다. 「시간 기반」은 저장 공간을 덜 사용합니다.
:::

## 녹화 설정

| 설정 | 기본값 | 설명 |
|---|---|---|
| 해상도 | 1280×720 | 이미지 크기 (640×480 / 1280×720 / 1920×1080) |
| 이미지 품질 | 85% | JPEG 압축 품질 |
| 영상 FPS | 30 | 완성 영상의 초당 프레임 수 |
| 영상 형식 | MP4 (H.264) | 출력 형식 |
| 이미지 회전 | 끄기 | 마운트 방향에 따라 90°/180°/270° 회전 |

:::warning 저장 공간
1080p 500장 타임랩스는 합성 전 약 200–400 MB를 사용합니다. 완성된 MP4 영상은 보통 20–80 MB입니다.
:::

## 저장소

타임랩스 이미지와 영상은 프로젝트 폴더의 `data/timelapse/` 아래에 저장됩니다. 구조는 프린터와 프린트별로 정리됩니다:

```
data/timelapse/
├── <printer-id>/                     ← 고유 프린터 ID
│   ├── 2026-03-22_modellnavn/        ← 프린트 세션 (날짜_모델명)
│   │   ├── frame_0001.jpg
│   │   ├── frame_0002.jpg
│   │   ├── frame_0003.jpg
│   │   └── ...                       ← 원본 이미지 (합성 후 삭제)
│   ├── 2026-03-22_modellnavn.mp4     ← 완성된 타임랩스 영상
│   ├── 2026-03-20_3dbenchy.mp4
│   └── 2026-03-15_telefonstativ.mp4
├── <printer-id-2>/                   ← 복수 프린터의 경우
│   └── ...
```

:::tip 외부 저장소
시스템 디스크 공간을 절약하려면 타임랩스 폴더를 외부 디스크로 심볼릭 링크할 수 있습니다:
```bash
# 예시: /mnt/storage에 마운트된 외부 디스크로 이동
mv data/timelapse /mnt/storage/timelapse

# 심볼릭 링크 생성
ln -s /mnt/storage/timelapse data/timelapse
```
대시보드는 심볼릭 링크를 자동으로 따라갑니다. 어떤 디스크나 네트워크 공유도 사용할 수 있습니다.
:::

## 자동 합성

프린트가 완료되면 이미지가 자동으로 ffmpeg를 통해 영상으로 합성됩니다:

1. 3DPrintForge가 MQTT에서 「print complete」 이벤트를 수신
2. 수집된 이미지로 ffmpeg 실행
3. 영상이 저장 폴더에 저장
4. 타임랩스 페이지에 새 영상 표시

**타임랩스 → 처리 중** 탭에서 진행 상황을 확인할 수 있습니다.

## 재생

1. **https://localhost:3443/#timelapse**로 이동합니다
2. 드롭다운에서 프린터 선택
3. 목록에서 영상을 클릭하여 재생
4. 재생 컨트롤 사용:
   - ▶ / ⏸ — 재생 / 일시 정지
   - ⏪ / ⏩ — 되감기 / 빨리 감기
   - 속도 버튼: 0.5× / 1× / 2× / 4×
5. **전체 화면** 클릭으로 전체 화면 모드
6. **다운로드** 클릭으로 MP4 파일 다운로드

## 타임랩스 삭제

1. 목록에서 영상 선택
2. **삭제** (휴지통 아이콘) 클릭
3. 대화 상자에서 확인

:::danger 영구 삭제
삭제된 타임랩스 영상과 원본 이미지는 복구할 수 없습니다. 보관하려면 먼저 영상을 다운로드하십시오.
:::

## 타임랩스 공유

타임랩스 영상은 시간 제한 링크를 통해 공유할 수 있습니다:

1. 영상을 선택하고 **공유** 클릭
2. 만료 시간 설정 (1시간 / 24시간 / 7일 / 만료 없음)
3. 생성된 링크를 복사하여 공유
4. 수신자는 로그인 없이 영상을 볼 수 있습니다
