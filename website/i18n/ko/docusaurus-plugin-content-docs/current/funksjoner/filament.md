# 필라멘트 재고

필라멘트 재고는 AMS 및 프린트 기록과 통합되어 모든 필라멘트 스풀에 대한 완전한 개요를 제공합니다.

## 개요

재고에는 등록된 모든 스풀이 다음과 함께 표시됩니다:

- **색상** — 시각적 색상 카드
- **재료** — PLA, PETG, ABS, TPU, PA 등
- **공급업체** — Bambu Lab, Polymaker, eSUN 등
- **무게** — 남은 그램(예상 또는 측정)
- **AMS 슬롯** — 스풀이 장착된 슬롯
- **상태** — 활성, 빈 것, 건조 중, 보관 중

## 스풀 추가

1. **+ 새 스풀**을 클릭합니다
2. 재료, 색상, 공급업체 및 무게를 입력합니다
3. NFC 태그를 스캔하거나(사용 가능한 경우) 수동으로 입력합니다
4. 저장합니다

:::tip Bambu Lab 스풀
Bambu Lab의 공식 스풀은 Bambu Cloud 통합을 통해 자동으로 가져올 수 있습니다. [Bambu Cloud](../kom-i-gang/bambu-cloud)를 참조하세요.
:::

## AMS 동기화

대시보드가 프린터에 연결되면 AMS 상태가 자동으로 동기화됩니다:

- 슬롯이 AMS의 올바른 색상 및 재료로 표시됩니다
- 각 프린트 후 소비량이 업데이트됩니다
- 빈 스풀이 자동으로 표시됩니다

로컬 스풀을 AMS 슬롯에 연결하려면:
1. **필라멘트 → AMS**로 이동합니다
2. 연결하려는 슬롯을 클릭합니다
3. 재고에서 스풀을 선택합니다

## 건조

습기 노출을 추적하기 위해 건조 사이클을 등록합니다:

| 항목 | 설명 |
|------|-------------|
| 건조 날짜 | 스풀을 건조한 날짜 |
| 온도 | 건조 온도(°C) |
| 시간 | 시간 수 |
| 방법 | 오븐, 건조 박스, 필라멘트 건조기 |

:::info 권장 건조 온도
재료별 건조 시간 및 온도는 [지식 베이스](../kb/intro)를 참조하세요.
:::

## 색상 카드

색상 카드 보기는 색상별로 스풀을 시각적으로 구성합니다. 올바른 색상을 빠르게 찾는 데 유용합니다. 재료, 공급업체 또는 상태별로 필터링합니다.

## NFC 태그

Bambu Dashboard는 스풀을 빠르게 식별하기 위한 NFC 태그를 지원합니다:

1. 재고의 스풀에 NFC 태그 ID를 기록합니다
2. 휴대폰으로 태그를 스캔합니다
3. 스풀이 대시보드에서 직접 열립니다

## 가져오기 및 내보내기

### 내보내기
전체 재고를 CSV로 내보냅니다: **필라멘트 → 내보내기 → CSV**

### 가져오기
CSV에서 스풀 가져오기: **필라멘트 → 가져오기 → 파일 선택**

CSV 형식:
```
name,material,color_hex,vendor,weight_grams,nfc_id
PLA White,PLA,#FFFFFF,Bambu Lab,1000,
PETG Black,PETG,#000000,Polymaker,850,ABC123
```

## 통계

**필라멘트 → 통계**에서 다음을 찾을 수 있습니다:

- 재료별 총 소비량(최근 30/90/365일)
- 프린터별 소비량
- 스풀당 예상 남은 수명
- 가장 많이 사용된 색상 및 공급업체
