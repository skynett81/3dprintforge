# 플리트 개요

플리트 개요는 한 페이지에서 연결된 모든 프린터의 간략한 개요를 제공합니다. 프린터가 두 대 이상인 작업장, 교실 또는 모든 환경에 완벽합니다.

이동: **https://localhost:3443/#fleet**

## 다중 프린터 그리드

등록된 모든 프린터가 반응형 그리드에 표시됩니다:

- **카드 크기** — 소형(간략), 중형(표준), 대형(상세)
- **열 수** — 화면 너비에 따라 자동 조정 또는 수동 설정
- **업데이트** — 각 카드가 MQTT를 통해 독립적으로 업데이트됨

각 프린터 카드에 표시됩니다:
| 항목 | 설명 |
|---|---|
| 프린터 이름 | 모델 아이콘이 있는 설정된 이름 |
| 상태 | 대기 / 프린팅 / 일시 정지 / 오류 / 연결 끊김 |
| 진행률 | 남은 시간이 있는 진행률 막대 |
| 온도 | 노즐 및 베드(간략) |
| 활성 필라멘트 | AMS의 색상 및 재료 |
| 카메라 썸네일 | 30초마다 업데이트되는 정지 이미지 |

## 프린터별 상태 표시기

상태 색상을 통해 원거리에서 상태를 쉽게 확인할 수 있습니다:

- **녹색 펄스** — 활성 프린팅 중
- **파란색** — 대기 중 및 준비됨
- **노란색** — 일시 정지됨(수동 또는 Print Guard에 의해)
- **빨간색** — 오류 감지됨
- **회색** — 연결 끊김 또는 사용 불가

:::tip 키오스크 모드
벽걸이 화면에서 플리트 개요를 키오스크 모드로 사용합니다. 설정은 [키오스크 모드](../system/kiosk)를 참조하세요.
:::

## 정렬

**정렬**을 클릭하여 순서를 선택합니다:

1. **이름** — 알파벳 순
2. **상태** — 활성 프린터가 맨 위
3. **진행률** — 가장 완료된 것이 맨 위
4. **마지막 활동** — 가장 최근에 사용된 것이 맨 위
5. **모델** — 프린터 모델별로 그룹화

정렬은 다음 방문까지 기억됩니다.

## 필터링

상단 필터 필드를 사용하여 보기를 제한합니다:

- 프린터 이름 또는 이름의 일부를 입력합니다
- 드롭다운에서 **상태** 선택(전체 / 프린팅 / 대기 / 오류)
- 한 프린터 유형만 표시하기 위해 **모델** 선택(X1C, P1S, A1 등)
- **필터 초기화**를 클릭하여 모두 표시

:::info 검색
검색은 페이지를 새로 고침하지 않고 실시간으로 필터링됩니다.
:::

## 플리트 개요에서의 작업

카드를 우클릭(또는 세 점 클릭)하면 빠른 작업이 표시됩니다:

- **대시보드 열기** — 프린터의 메인 패널로 직접 이동
- **프린트 일시 정지** — 프린터 일시 정지
- **프린트 중지** — 진행 중인 프린트 중단(확인 필요)
- **카메라 보기** — 팝업에서 카메라 보기 열기
- **설정으로 이동** — 프린터 설정 열기

:::danger 프린트 중지
프린트를 중지하는 것은 되돌릴 수 없습니다. 표시되는 대화 상자에서 항상 확인하세요.
:::

## 집계 통계

플리트 개요 상단에 요약 행이 표시됩니다:

- 총 프린터 수
- 활성 프린트 수
- 오늘의 총 필라멘트 소비량
- 가장 긴 진행 중인 프린트의 예상 완료 시간
