# 프린트 기록

프린트 기록은 대시보드로 수행한 모든 프린트에 대한 통계, 필라멘트 소비량 및 모델 소스 링크를 포함한 완전한 기록을 제공합니다.

## 기록 표

표에는 다음이 포함된 모든 프린트가 표시됩니다:

| 열 | 설명 |
|---------|-------------|
| 날짜/시간 | 시작 시간 |
| 모델 이름 | 파일 이름 또는 MakerWorld 제목 |
| 프린터 | 사용된 프린터 |
| 소요 시간 | 총 프린트 시간 |
| 필라멘트 | 재료 및 사용된 그램 |
| 레이어 | 레이어 수 및 무게(g) |
| 상태 | 완료, 중단, 실패 |
| 이미지 | 썸네일(클라우드 통합 시) |

## 검색 및 필터링

검색 필드와 필터를 사용하여 프린트를 찾습니다:

- 모델 이름으로 전체 텍스트 검색
- 프린터, 재료, 상태, 날짜로 필터링
- 모든 열에서 정렬

## 모델 소스 링크

프린트가 MakerWorld에서 시작된 경우 모델 페이지로 직접 링크가 표시됩니다. 모델 이름을 클릭하면 새 탭에서 MakerWorld가 열립니다.

:::info Bambu Cloud
모델 링크 및 썸네일은 Bambu Cloud 통합이 필요합니다. [Bambu Cloud](../kom-i-gang/bambu-cloud)를 참조하세요.
:::

## 필라멘트 추적

각 프린트에 대해 다음이 기록됩니다:

- **재료** — PLA, PETG, ABS 등
- **사용된 그램** — 예상 소비량
- **스풀** — 사용된 스풀(재고에 등록된 경우)
- **색상** — 색상 헥스 코드

이를 통해 시간에 따른 필라멘트 소비량을 정확하게 파악하고 구매를 계획할 수 있습니다.

## 통계

**기록 → 통계**에서 집계 데이터를 찾을 수 있습니다:

- **총 프린트 횟수** — 및 성공률
- **총 프린트 시간** — 시간 및 일
- **필라멘트 소비량** — 재료별 그램 및 km
- **일별 프린트** — 롤링 그래프
- **가장 많이 사용된 재료** — 파이 차트
- **프린트 길이 분포** — 히스토그램

통계는 기간(7일, 30일, 90일, 1년, 전체)으로 필터링할 수 있습니다.

## 내보내기

### CSV 내보내기
전체 기록 또는 필터링된 결과를 내보냅니다:
**기록 → 내보내기 → CSV 다운로드**

CSV 파일에는 모든 열이 포함되어 있으며 Excel, LibreOffice Calc에서 열거나 다른 도구로 가져올 수 있습니다.

### 자동 백업
기록은 업데이트 시 자동으로 백업되는 SQLite 데이터베이스의 일부입니다. **설정 → 백업**에서 수동 백업을 수행합니다.

## 편집

이후에 프린트 기록 항목을 편집할 수 있습니다:

- 모델 이름 수정
- 메모 추가
- 필라멘트 소비량 수정
- 잘못 등록된 프린트 삭제

행을 우클릭하고 **편집**을 선택하거나 연필 아이콘을 클릭합니다.
