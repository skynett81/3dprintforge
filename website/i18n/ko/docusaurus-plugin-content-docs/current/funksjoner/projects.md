# 프로젝트

프로젝트를 사용하면 관련 프린트를 그룹화하고, 재료 비용을 추적하고, 고객에게 청구하고, 작업 개요를 공유할 수 있습니다.

이동: **https://localhost:3443/#projects**

## 프로젝트 만들기

1. **새 프로젝트**(+ 아이콘) 클릭
2. 다음을 입력합니다:
   - **프로젝트 이름** — 설명적인 이름(최대 100자)
   - **고객** — 선택적 고객 계정([전자상거래](../integrasjoner/ecommerce) 참조)
   - **설명** — 간단한 텍스트 설명
   - **색상** — 시각적 식별을 위한 색상 선택
   - **태그** — 쉼표로 구분된 키워드
3. **프로젝트 만들기** 클릭

## 프로젝트에 프린트 연결

### 프린트 중

1. 프린트가 진행되는 동안 대시보드 열기
2. 사이드 패널에서 **프로젝트에 연결** 클릭
3. 기존 프로젝트를 선택하거나 새로 만들기
4. 프린트가 완료되면 자동으로 프로젝트에 연결됩니다

### 기록에서

1. **기록**으로 이동합니다
2. 해당 프린트 찾기
3. 프린트 클릭 → **프로젝트에 연결**
4. 드롭다운에서 프로젝트 선택

### 일괄 연결

1. 기록에서 체크박스로 여러 프린트 선택
2. **작업 → 프로젝트에 연결** 클릭
3. 프로젝트 선택 — 선택한 모든 프린트가 연결됩니다

## 비용 개요

각 프로젝트는 다음을 기반으로 총 비용을 계산합니다:

| 비용 유형 | 출처 |
|---|---|
| 필라멘트 소비량 | 그램 × 재료별 그램당 가격 |
| 전력 | kWh × 전기 요금(구성된 경우 Tibber/Nordpool에서) |
| 기계 마모 | [마모 예측](../overvaaking/wearprediction)에서 계산 |
| 수동 비용 | 수동으로 추가하는 자유 텍스트 항목 |

비용 개요는 프린트당 및 총합으로 표와 파이 차트로 표시됩니다.

:::tip 시간별 요금
정확한 프린트당 전기 비용을 위해 Tibber 또는 Nordpool 통합을 활성화합니다. [전기 요금](../integrasjoner/energy)을 참조하세요.
:::

## 청구

1. 프로젝트를 열고 **인보이스 생성** 클릭
2. 다음을 입력합니다:
   - **인보이스 날짜** 및 **만기 날짜**
   - **부가가치세율**(0%, 15%, 25%)
   - **마진**(%)
   - **고객에게 메모**
3. PDF 형식으로 인보이스 미리 보기
4. **PDF 다운로드** 또는 **고객에게 전송** 클릭(이메일을 통해)

인보이스는 프로젝트 아래에 저장되며 전송되기 전까지 다시 열고 편집할 수 있습니다.

:::info 고객 데이터
고객 데이터(이름, 주소, 사업자 번호)는 프로젝트에 연결한 고객 계정에서 가져옵니다. 고객 관리는 [전자상거래](../integrasjoner/ecommerce)를 참조하세요.
:::

## 프로젝트 상태

| 상태 | 설명 |
|---|---|
| 활성 | 프로젝트가 진행 중 |
| 완료 | 모든 프린트가 준비되고 인보이스가 전송됨 |
| 보관됨 | 기본 보기에서 숨겨지지만 검색 가능 |
| 보류 중 | 임시로 중단됨 |

프로젝트 상단의 상태 표시기를 클릭하여 상태를 변경합니다.

## 프로젝트 공유

고객에게 프로젝트 개요를 보여주는 공유 링크를 생성합니다:

1. 프로젝트 메뉴에서 **프로젝트 공유** 클릭
2. 표시할 내용 선택:
   - ✅ 프린트 및 이미지
   - ✅ 총 필라멘트 소비량
   - ❌ 비용 및 가격(기본적으로 숨김)
3. 링크 만료 시간 설정
4. 링크 복사 및 공유

고객은 로그인 없이 읽기 전용 페이지를 볼 수 있습니다.
