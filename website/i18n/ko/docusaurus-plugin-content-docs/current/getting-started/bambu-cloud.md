# Bambu Cloud 통합

3DPrintForge는 **Bambu Lab Cloud**에 연결하여 모델 이미지, 프린트 기록 및 필라멘트 데이터를 가져올 수 있습니다. 클라우드 연결 없이도 대시보드는 완전히 작동하지만, 클라우드 통합은 추가적인 이점을 제공합니다.

## 클라우드 통합의 이점

| 기능 | 클라우드 없이 | 클라우드 사용 |
|---------|-----------|----------|
| 라이브 프린터 상태 | 예 | 예 |
| 프린트 기록 (로컬) | 예 | 예 |
| MakerWorld의 모델 이미지 | 아니오 | 예 |
| Bambu의 필라멘트 프로필 | 아니오 | 예 |
| 프린트 기록 동기화 | 아니오 | 예 |
| 클라우드의 AMS 필라멘트 | 아니오 | 예 |

## Bambu Cloud 연결

1. **설정 → Bambu Cloud**로 이동합니다
2. Bambu Lab 이메일과 비밀번호 입력
3. **로그인** 클릭
4. 동기화할 데이터 선택

:::warning 개인정보
사용자 이름과 비밀번호는 평문으로 저장되지 않습니다. 대시보드는 Bambu Labs API를 사용하여 로컬에 저장되는 OAuth 토큰을 가져옵니다. 사용자의 데이터는 서버를 벗어나지 않습니다.
:::

## 동기화

### 모델 이미지

클라우드가 연결되면 모델 이미지가 **MakerWorld**에서 자동으로 가져와져 다음에 표시됩니다:
- 프린트 기록
- 대시보드 (활성 프린트 중)
- 3D 모델 뷰어

### 프린트 기록

클라우드 동기화는 Bambu Lab 앱에서 프린트 기록을 가져옵니다. 타임스탬프와 일련번호를 기반으로 중복 항목이 자동으로 필터링됩니다.

### 필라멘트 프로필

Bambu Labs의 공식 필라멘트 프로필이 동기화되어 필라멘트 재고에 표시됩니다. 이를 자신만의 프로필의 시작점으로 사용할 수 있습니다.

## 클라우드 없이 작동하는 기능

클라우드 연결 없이도 모든 핵심 기능이 작동합니다:

- LAN을 통해 프린터에 직접 MQTT 연결
- 라이브 상태, 온도, 카메라
- 로컬 프린트 기록 및 통계
- 필라멘트 재고 (수동 관리)
- 알림 및 스케줄러

:::tip LAN 전용 모드
인터넷 연결 없이 대시보드를 사용하고 싶으신가요? 격리된 네트워크에서도 완벽하게 작동합니다 — IP로 프린터에 연결하고 클라우드 통합은 꺼두면 됩니다.
:::

## 문제 해결

**로그인 실패:**
- Bambu Lab 앱의 이메일과 비밀번호가 올바른지 확인
- 계정이 이중 인증을 사용하는지 확인 (아직 지원되지 않음)
- 로그아웃 후 다시 로그인 시도

**동기화 중지:**
- 토큰이 만료되었을 수 있음 — 설정에서 로그아웃 후 다시 로그인
- 서버에서 인터넷 연결 확인
