# 일상적인 사용

이 가이드는 일상 생활에서 Bambu Dashboard를 효율적으로 사용하는 방법을 다룹니다 — 하루를 시작할 때부터 불을 끌 때까지.

## 아침 루틴

대시보드를 열고 다음 항목들을 빠르게 확인합니다:

### 1. 프린터 상태 확인
개요 패널에 모든 프린터의 상태가 표시됩니다. 다음을 확인하세요:
- **빨간 아이콘** — 주의가 필요한 오류
- **대기 중인 메시지** — 밤새 발생한 HMS 경고
- **미완료 출력** — 야간 출력이 있었다면 완료되었나요?

### 2. AMS 수준 확인
**필라멘트**로 이동하거나 대시보드의 AMS 위젯을 확인합니다:
- 100g 미만인 스풀이 있나요? 교체하거나 새것을 주문하세요
- 오늘 출력에 올바른 필라멘트가 올바른 슬롯에 있나요?

### 3. 알림 및 이벤트 확인
**알림 로그** (벨 아이콘)에서 다음을 확인합니다:
- 밤새 발생한 이벤트
- 자동으로 기록된 오류
- 알람을 트리거한 HMS 코드

## 출력 시작하기

### 파일에서 (Bambu Studio)
1. Bambu Studio를 엽니다
2. 모델을 로드하고 슬라이스합니다
3. 프린터로 전송 — 대시보드가 자동으로 업데이트됩니다

### 큐에서
미리 출력을 계획한 경우:
1. **큐**로 이동합니다
2. **다음 시작**을 클릭하거나 작업을 맨 위로 드래그합니다
3. **프린터로 전송**으로 확인합니다

큐 관리에 대한 전체 정보는 [출력 큐 문서](../funksjoner/queue)를 참조하세요.

### 예약 출력 (스케줄러)
특정 시간에 출력을 시작하려면:
1. **스케줄러**로 이동합니다
2. **+ 새 작업**을 클릭합니다
3. 파일, 프린터, 시간을 선택합니다
4. **전력 가격 최적화**를 활성화하여 가장 저렴한 시간대를 자동 선택합니다

자세한 내용은 [스케줄러](../funksjoner/scheduler)를 참조하세요.

## 활성 출력 모니터링

### 카메라 뷰
프린터 카드의 카메라 아이콘을 클릭합니다. 다음을 할 수 있습니다:
- 대시보드에서 라이브 피드 보기
- 백그라운드 모니터링을 위해 별도의 탭에서 열기
- 수동으로 스크린샷 찍기

### 진행 정보
활성 출력 카드에 표시됩니다:
- 완료 비율
- 예상 남은 시간
- 현재 레이어 / 총 레이어 수
- 활성 필라멘트와 색상

### 온도
실시간 온도 곡선이 상세 패널에 표시됩니다:
- 노즐 온도 — ±2°C 이내로 안정적이어야 합니다
- 플레이트 온도 — 좋은 접착력에 중요
- 챔버 온도 — 점진적으로 상승, ABS/ASA에 특히 관련

### Print Guard
**Print Guard**가 활성화된 경우 대시보드가 스파게티와 체적 편차를 자동으로 모니터링합니다. 무언가 감지되면:
1. 출력이 일시 중지됩니다
2. 알림을 받습니다
3. 나중에 확인할 수 있도록 카메라 이미지가 저장됩니다

## 출력 후 — 체크 루틴

### 품질 확인
1. 카메라를 열고 플레이트에 있는 동안 결과를 확인합니다
2. **기록 → 최근 출력**으로 이동하여 통계를 확인합니다
3. 메모 기록: 잘된 것, 개선할 수 있는 것

### 보관
기록의 출력은 자동으로 보관되지 않습니다 — 그대로 남아 있습니다. 정리하려면:
- 출력 클릭 → **보관**으로 아카이브로 이동
- **프로젝트**를 사용하여 관련 출력을 그룹화

### 필라멘트 무게 업데이트
정확도를 위해 스풀을 계량하는 경우 (권장):
1. 스풀을 계량합니다
2. **필라멘트 → [스풀]**로 이동합니다
3. **남은 무게**를 업데이트합니다

## 유지 관리 알림

대시보드가 유지 관리 간격을 자동으로 추적합니다. **유지 관리** 아래에서 확인:

| 작업 | 간격 | 상태 |
|------|------|------|
| 노즐 청소 | 50시간마다 | 자동 확인 |
| 로드 윤활 | 200시간마다 | 대시보드에서 추적 |
| 플레이트 캘리브레이션 | 플레이트 교체 후 | 수동 알림 |
| AMS 청소 | 월별 | 캘린더 알림 |

**모니터링 → 유지 관리 → 알림**에서 유지 관리 알림을 활성화합니다.

:::tip 주간 유지 관리 날 설정하기
주간 고정 유지 관리 날 (예: 일요일 저녁)은 불필요한 중단을 방지합니다. 대시보드의 알림 기능을 사용하세요.
:::

## 전력 가격 — 출력하기 가장 좋은 시간

전력 가격 통합 (Nordpool / Home Assistant)을 연결한 경우:

1. **분석 → 전력 가격**으로 이동합니다
2. 다음 24시간의 가격 그래프를 확인합니다
3. 가장 저렴한 시간대가 녹색으로 표시됩니다

**전력 가격 최적화**가 활성화된 **스케줄러**를 사용하세요 — 대시보드가 가장 저렴한 사용 가능한 시간 창에서 자동으로 작업을 시작합니다.

:::info 일반적으로 가장 저렴한 시간대
노르웨이에서는 일반적으로 야간 (01:00〜06:00)이 가장 저렴합니다. 전날 저녁에 8시간 출력을 큐에 넣으면 전기 비용을 30〜50% 절약할 수 있습니다.
:::
