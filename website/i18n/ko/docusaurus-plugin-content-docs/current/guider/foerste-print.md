# 첫 번째 출력

이 가이드는 연결된 프린터부터 완성된 출력물까지 Bambu Dashboard를 제어 센터로 사용하는 전체 과정을 안내합니다.

## 1단계 — 프린터가 연결되었는지 확인하기

대시보드를 열면 사이드바 상단이나 개요 패널에서 프린터의 상태 카드를 볼 수 있습니다.

**녹색 상태**는 프린터가 온라인 상태이며 준비되었음을 의미합니다.

| 상태 | 색상 | 의미 |
|------|------|------|
| 온라인 | 녹색 | 출력 준비 완료 |
| 유휴 | 회색 | 연결됨, 비활성 |
| 출력 중 | 파란색 | 출력 진행 중 |
| 오류 | 빨간색 | 주의 필요 |

프린터가 빨간색 상태를 표시하는 경우:
1. 프린터 전원이 켜져 있는지 확인합니다
2. 대시보드와 같은 네트워크에 연결되어 있는지 확인합니다
3. **설정 → 프린터**로 이동하여 IP 주소와 액세스 코드를 확인합니다

:::tip 빠른 응답을 위해 LAN 모드 사용하기
LAN 모드는 클라우드 모드보다 낮은 지연 시간을 제공합니다. 프린터와 대시보드가 같은 네트워크에 있는 경우 프린터 설정에서 활성화하세요.
:::

## 2단계 — 모델 업로드하기

Bambu Dashboard는 직접 출력을 시작하지 않습니다 — 그것은 Bambu Studio 또는 MakerWorld의 역할입니다. 출력이 시작되면 대시보드가 인계받습니다.

**Bambu Studio를 통해:**
1. PC에서 Bambu Studio를 엽니다
2. `.stl` 또는 `.3mf` 파일을 가져오거나 엽니다
3. 모델을 슬라이스합니다 (필라멘트, 서포트, 인필 등 선택)
4. 오른쪽 상단의 **출력**을 클릭합니다

**MakerWorld를 통해:**
1. [makerworld.com](https://makerworld.com)에서 모델을 찾습니다
2. 웹사이트에서 직접 **출력**을 클릭합니다
3. Bambu Studio가 모델을 로드하여 자동으로 열립니다

## 3단계 — 출력 시작하기

Bambu Studio에서 전송 방법을 선택합니다:

| 방법 | 요구 사항 | 장점 |
|------|-----------|------|
| **클라우드** | Bambu 계정 + 인터넷 | 어디서든 작동 |
| **LAN** | 같은 네트워크 | 더 빠름, 클라우드 불필요 |
| **SD 카드** | 물리적 액세스 | 네트워크 불필요 |

**전송**을 클릭하면 프린터가 작업을 수신하고 자동으로 예열 단계를 시작합니다.

:::info 출력이 대시보드에 표시됩니다
Bambu Studio가 작업을 전송한 후 몇 초 이내에 활성 출력이 대시보드의 **활성 출력** 아래에 표시됩니다.
:::

## 4단계 — 대시보드에서 모니터링하기

출력이 진행 중일 때 대시보드는 전체 개요를 제공합니다:

### 진행 상황
- 완료 비율과 예상 남은 시간이 프린터 카드에 표시됩니다
- 카드를 클릭하면 레이어 정보가 있는 상세 뷰가 표시됩니다

### 온도
상세 패널에 실시간 온도가 표시됩니다:
- **노즐** — 현재 온도와 목표 온도
- **빌드 플레이트** — 현재 온도와 목표 온도
- **챔버** — 프린터 내부 온도 (ABS/ASA에 중요)

### 카메라
프린터 카드의 카메라 아이콘을 클릭하면 대시보드에서 라이브 피드를 볼 수 있습니다. 다른 작업을 하면서 카메라를 별도의 창에서 열어 둘 수 있습니다.

:::warning 첫 번째 레이어를 확인하세요
처음 3〜5개 레이어가 중요합니다. 지금 접착력이 좋지 않으면 나중에 출력이 실패합니다. 카메라를 보면서 필라멘트가 깔끔하고 균일하게 쌓이는지 확인하세요.
:::

### Print Guard
Bambu Dashboard에는 AI 기반 **Print Guard**가 있어 스파게티 오류를 자동으로 감지하고 출력을 일시 중지할 수 있습니다. **모니터링 → Print Guard**에서 활성화하세요.

## 5단계 — 출력 완료 후

출력이 완료되면 대시보드에 완료 메시지가 표시됩니다 ([알림](./varsler-oppsett)을 설정한 경우 알림도 전송됩니다).

### 기록 확인하기
사이드바의 **기록**으로 이동하여 완료된 출력을 확인합니다:
- 총 출력 시간
- 필라멘트 소비량 (사용된 그램, 예상 비용)
- 출력 중 오류 또는 HMS 이벤트
- 완료 시 카메라 이미지 (활성화된 경우)

### 메모 기록하기
기록에서 출력을 클릭하고 메모를 추가합니다 — 예: "브림이 조금 더 필요했음" 또는 "완벽한 결과". 같은 모델을 다시 출력할 때 유용합니다.

### 필라멘트 소비량 확인하기
**필라멘트** 아래에서 스풀 무게가 사용량에 따라 업데이트된 것을 볼 수 있습니다. 대시보드가 자동으로 차감합니다.

## 초보자를 위한 팁

:::tip 첫 번째 출력을 방치하지 마세요
처음 10〜15분은 지켜보세요. 출력이 잘 부착되는 것을 확인한 후에는 대시보드가 나머지를 모니터링하도록 할 수 있습니다.
:::

- **빈 스풀 무게 측정** — 정확한 잔량 계산을 위해 스풀의 시작 무게를 입력하세요 ([필라멘트 재고](./filament-oppsett) 참조)
- **Telegram 알림 설정** — 기다리지 않고 출력 완료 시 알림을 받으세요 ([알림](./varsler-oppsett) 참조)
- **빌드 플레이트 확인** — 깨끗한 플레이트 = 더 좋은 접착력. 출력 사이에 IPA(이소프로판올)로 닦으세요
- **올바른 플레이트 사용** — 필라멘트에 맞는 것에 대해서는 [올바른 빌드 플레이트 선택](./velge-rett-plate)을 참조하세요
