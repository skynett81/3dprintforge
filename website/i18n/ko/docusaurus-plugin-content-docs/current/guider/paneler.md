# 대시보드 탐색

이 가이드는 대시보드의 구성 방식과 효율적인 탐색 방법을 빠르게 소개합니다.

## 사이드 패널

왼쪽의 사이드 패널은 탐색 센터입니다. 섹션으로 구성되어 있습니다:

```
┌────────────────────┐
│ 🖨  프린터 상태    │  ← 프린터당 한 줄
├────────────────────┤
│ 개요               │
│ 플리트             │
│ 활성 출력          │
├────────────────────┤
│ 필라멘트           │
│ 기록               │
│ 프로젝트           │
│ 큐                 │
│ 스케줄러           │
├────────────────────┤
│ 모니터링           │
│  └ Print Guard     │
│  └ 오류            │
│  └ 진단            │
│  └ 유지 관리       │
├────────────────────┤
│ 분석               │
│ 도구               │
│ 통합               │
│ 시스템             │
├────────────────────┤
│ ⚙ 설정            │
└────────────────────┘
```

왼쪽 상단의 햄버거 아이콘 (☰)을 클릭하여 **사이드 패널을 숨길 수 있습니다**. 작은 화면이나 키오스크 모드에 유용합니다.

## 메인 패널

사이드 패널의 항목을 클릭하면 오른쪽 메인 패널에 콘텐츠가 표시됩니다. 레이아웃은 다양합니다:

| 패널 | 레이아웃 |
|------|---------|
| 개요 | 모든 프린터의 카드 그리드 |
| 활성 출력 | 큰 상세 카드 + 온도 곡선 |
| 기록 | 필터링 가능한 테이블 |
| 필라멘트 | 스풀의 카드 뷰 |
| 분석 | 그래프와 차트 |

## 프린터 상태를 클릭하여 세부 정보 보기

개요 패널의 프린터 카드는 클릭 가능합니다:

**단일 클릭** → 해당 프린터의 상세 패널 열기:
- 실시간 온도
- 활성 출력 (진행 중인 경우)
- 모든 슬롯의 AMS 상태
- 최근 오류 및 이벤트
- 빠른 버튼: 일시 중지, 중지, 조명 켜기/끄기

**카메라 아이콘 클릭** → 라이브 카메라 뷰 열기

**⚙ 아이콘 클릭** → 프린터 설정

## 키보드 단축키 — 명령 팔레트

명령 팔레트는 탐색 없이 모든 기능에 빠르게 접근할 수 있습니다:

| 단축키 | 동작 |
|--------|------|
| `Ctrl + K` (Linux/Windows) | 명령 팔레트 열기 |
| `Cmd + K` (macOS) | 명령 팔레트 열기 |
| `Esc` | 팔레트 닫기 |

명령 팔레트에서 할 수 있는 것:
- 페이지와 기능 검색
- 직접 출력 시작
- 활성 출력 일시 중지/재개
- 테마 전환 (밝음/어둠)
- 모든 페이지로 이동

**예시:** `Ctrl+K`를 누르고 "일시 중지"를 입력 → "모든 활성 출력 일시 중지" 선택

## 위젯 사용자 정의

개요 패널은 원하는 위젯으로 사용자 정의할 수 있습니다:

**대시보드 편집 방법:**
1. 개요 패널 오른쪽 상단의 **레이아웃 편집** (연필 아이콘)을 클릭합니다
2. 위젯을 원하는 위치로 드래그합니다
3. 위젯의 모서리를 클릭하고 드래그하여 크기를 변경합니다
4. **+ 위젯 추가**를 클릭하여 새것을 추가합니다:

사용 가능한 위젯:

| 위젯 | 표시 내용 |
|------|---------|
| 프린터 상태 | 모든 프린터의 카드 |
| 활성 출력 (큰) | 진행 중인 출력의 상세 뷰 |
| AMS 개요 | 모든 슬롯과 필라멘트 수준 |
| 온도 곡선 | 실시간 그래프 |
| 전력 가격 | 다음 24시간 가격 그래프 |
| 필라멘트 계량기 | 최근 30일 총 소비량 |
| 기록 단축키 | 최근 5개 출력 |
| 카메라 피드 | 라이브 카메라 이미지 |

5. **레이아웃 저장**을 클릭합니다

:::tip 여러 레이아웃 저장하기
다양한 목적을 위한 다양한 레이아웃을 가질 수 있습니다 — 일상 사용을 위한 컴팩트한 것, 대형 화면에 걸기 위한 큰 것. 레이아웃 선택기로 전환합니다.
:::

## 테마 — 밝음과 어둠 사이 전환

**빠른 전환:**
- 탐색 오른쪽 상단의 태양/달 아이콘 클릭
- 또는: `Ctrl+K` → "테마" 입력

**영구 설정:**
1. **시스템 → 테마**로 이동합니다
2. 다음 중 선택합니다:
   - **밝음** — 흰색 배경
   - **어둠** — 어두운 배경 (야간 권장)
   - **자동** — 기기의 시스템 설정을 따름
3. 강조 색상을 선택합니다 (파란색, 녹색, 보라색 등)
4. **저장**을 클릭합니다

## 키보드 탐색

마우스 없이 효율적으로 탐색하기:

| 단축키 | 동작 |
|--------|------|
| `Tab` | 다음 인터랙티브 요소 |
| `Shift+Tab` | 이전 요소 |
| `Enter` / `Space` | 버튼/링크 활성화 |
| `Esc` | 모달/드롭다운 닫기 |
| `Ctrl+K` | 명령 팔레트 |
| `Alt+1` – `Alt+9` | 처음 9개 페이지로 직접 이동 |

## PWA — 앱으로 설치

Bambu Dashboard는 프로그레시브 웹 앱 (PWA)으로 설치하여 브라우저 메뉴 없이 독립적인 앱으로 실행할 수 있습니다:

1. Chrome, Edge 또는 Safari에서 대시보드로 이동합니다
2. 주소창의 **앱 설치** 아이콘을 클릭합니다
3. 설치를 확인합니다

자세한 내용은 [PWA 문서](../system/pwa)를 참조하세요.

## 키오스크 모드

키오스크 모드는 모든 탐색을 숨기고 대시보드만 표시합니다 — 출력 작업실의 전용 화면에 완벽합니다:

1. **시스템 → 키오스크**로 이동합니다
2. **키오스크 모드**를 활성화합니다
3. 표시할 위젯을 선택합니다
4. 업데이트 간격을 설정합니다

완전한 설정은 [키오스크 문서](../system/kiosk)를 참조하세요.
