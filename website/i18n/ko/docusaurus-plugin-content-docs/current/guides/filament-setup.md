# 필라멘트 재고 설정

3DPrintForge의 필라멘트 재고는 모든 스풀에 대한 완전한 개요를 제공합니다 — 남은 양, 사용한 양, 현재 AMS에 있는 스풀 등.

## AMS에서 자동 생성

AMS가 있는 프린터를 연결하면 대시보드가 Bambu 스풀의 RFID 칩에서 자동으로 정보를 읽어옵니다:

- 필라멘트 유형 (PLA, PETG, ABS, TPU 등)
- 색상 (16진수 색상 코드 포함)
- 브랜드 (Bambu Lab)
- 스풀 무게와 남은 양

**이 스풀들은 재고에 자동으로 생성됩니다** — 아무것도 할 필요가 없습니다. **필라멘트 → 재고**에서 확인하세요.

:::info RFID가 있는 것은 Bambu 스풀뿐입니다
서드파티 스풀 (예: eSUN, Polymaker, 칩 없는 Bambu 리필)은 자동으로 인식되지 않습니다. 이것들은 수동으로 입력해야 합니다.
:::

## 스풀 수동 추가

RFID가 없는 스풀이나 AMS에 없는 스풀의 경우:

1. **필라멘트 → 재고**로 이동합니다
2. 오른쪽 상단의 **+ 새 스풀**을 클릭합니다
3. 필드를 채웁니다:

| 필드 | 예시 | 필수 |
|------|------|------|
| 브랜드 | eSUN, Polymaker, Bambu | 예 |
| 유형 | PLA, PETG, ABS, TPU | 예 |
| 색상 | #FF5500 또는 색상 휠에서 선택 | 예 |
| 시작 무게 | 1000 g | 권장 |
| 남은 양 | 850 g | 권장 |
| 지름 | 1.75 mm | 예 |
| 메모 | "2025년 1월 구매, 잘 작동함" | 선택 |

4. **저장**을 클릭합니다

## 색상 및 브랜드 설정

재고에서 스풀을 클릭하여 언제든지 편집할 수 있습니다:

- **색상** — 색상 휠에서 선택하거나 16진수 값을 입력하세요. 색상은 AMS 개요에서 시각적 마커로 사용됩니다
- **브랜드** — 통계와 필터링에 표시됩니다. **필라멘트 → 브랜드**에서 사용자 정의 브랜드를 만들 수 있습니다
- **온도 프로파일** — 필라멘트 제조사의 권장 노즐 및 플레이트 온도를 입력하세요. 잘못된 온도를 선택하면 대시보드가 경고할 수 있습니다

## AMS 동기화 이해하기

대시보드는 AMS 상태를 실시간으로 동기화합니다:

```
AMS 슬롯 1 → 스풀: Bambu PLA 화이트  [███████░░░] 72% 남음
AMS 슬롯 2 → 스풀: eSUN PETG 그레이  [████░░░░░░] 41% 남음
AMS 슬롯 3 → (비어 있음)
AMS 슬롯 4 → 스풀: Bambu PLA 레드    [██████████] 98% 남음
```

동기화는 다음 시점에 업데이트됩니다:
- **출력 중** — 소비량이 실시간으로 차감됩니다
- **출력 종료 시** — 최종 소비량이 기록에 기록됩니다
- **수동** — 스풀의 동기화 아이콘을 클릭하여 AMS에서 최신 데이터를 가져옵니다

:::tip AMS 추정값 수정
RFID의 AMS 추정값은 처음 사용 후 항상 100% 정확하지 않습니다. 스풀을 계량하고 무게를 수동으로 업데이트하면 더 정확해집니다.
:::

## 소비량 및 남은 양 확인

### 스풀별
재고의 스풀을 클릭하여 확인합니다:
- 총 사용량 (그램, 모든 출력)
- 추정 남은 양
- 이 스풀을 사용한 모든 출력 목록

### 집계 통계
**분석 → 필라멘트 분석**에서 확인합니다:
- 시간에 따른 필라멘트 유형별 소비량
- 가장 많이 사용하는 브랜드
- kg당 구매 가격에 기반한 예상 비용

### 잔량 부족 경보
스풀이 소진될 때 경보를 설정합니다:

1. **필라멘트 → 설정**으로 이동합니다
2. **잔량 부족 시 알림**을 활성화합니다
3. 임계값을 설정합니다 (예: 100g 남음)
4. 알림 채널을 선택합니다 (Telegram, Discord, 이메일)

## 팁: 정확도를 위해 스풀 계량하기

AMS와 출력 통계의 추정값은 완전히 정확하지 않습니다. 가장 정확한 방법은 스풀 자체를 계량하는 것입니다:

**방법:**

1. 타라 중량(빈 스풀)을 찾습니다 — 보통 200〜250g, 제조사 웹사이트나 스풀 하단을 확인하세요
2. 주방 저울로 필라멘트가 있는 스풀을 계량합니다
3. 타라 중량을 뺍니다
4. 스풀 프로파일의 **남은 양**을 업데이트합니다

**예시:**
```
측정 중량:     743 g
타라 (빈):   - 230 g
남은 필라멘트: 513 g
```

:::tip 스풀 라벨 생성기
**도구 → 라벨**에서 스풀용 QR 코드가 있는 라벨을 인쇄할 수 있습니다. 스마트폰으로 코드를 스캔하면 스풀 프로파일을 빠르게 열 수 있습니다.
:::
