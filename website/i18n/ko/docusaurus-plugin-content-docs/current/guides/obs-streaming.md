# OBS로 3D 출력 스트리밍

3DPrintForge에는 스트림에서 직접 프린터 상태, 진행 상황, 온도, 카메라 피드를 표시하는 OBS 오버레이가 내장되어 있습니다.

## 사전 요구 사항

- OBS Studio 설치 ([obsproject.com](https://obsproject.com))
- 3DPrintForge 실행 중이며 프린터에 연결됨
- (선택 사항) 라이브 피드를 위한 Bambu 카메라 활성화

## 1단계 — OBS 브라우저 소스

OBS에는 장면에 직접 웹 페이지를 표시하는 **브라우저 소스**가 내장되어 있습니다.

**OBS에 오버레이 추가:**

1. OBS Studio를 엽니다
2. **소스** (Sources) 아래에서 **+**를 클릭합니다
3. **브라우저** (Browser)를 선택합니다
4. 소스에 이름을 붙입니다, 예: "Bambu Overlay"
5. 다음을 입력합니다:

| 설정 | 값 |
|------|-----|
| URL | `http://localhost:3000/obs/overlay` |
| 너비 | `1920` |
| 높이 | `1080` |
| FPS | `30` |
| 사용자 정의 CSS | 아래 참조 |

6. **OBS를 통해 오디오 제어**를 체크합니다
7. **확인**을 클릭합니다

:::info 서버에 맞게 URL 조정
OBS와 다른 컴퓨터에서 대시보드를 실행하고 있나요? `localhost`를 서버의 IP 주소로 대체하세요, 예: `http://192.168.1.50:3000/obs/overlay`
:::

## 2단계 — 투명 배경

오버레이가 화면에 자연스럽게 녹아들려면 배경이 투명해야 합니다:

**OBS 브라우저 소스 설정에서:**
- **배경 제거** (Shutdown source when not visible / Remove background)를 체크합니다

**투명성을 강제하는 사용자 정의 CSS:**
```css
body {
  background-color: rgba(0, 0, 0, 0) !important;
  margin: 0;
  overflow: hidden;
}
```

브라우저 소스 설정의 **사용자 정의 CSS** 필드에 이것을 붙여넣습니다.

이제 오버레이는 위젯 자체만 표시합니다 — 흰색이나 검정 배경 없이.

## 3단계 — 오버레이 사용자 정의

3DPrintForge에서 오버레이가 무엇을 표시할지 구성할 수 있습니다:

1. **기능 → OBS 오버레이**로 이동합니다
2. 구성합니다:

| 설정 | 옵션 |
|------|------|
| 위치 | 왼쪽 상단, 오른쪽 상단, 왼쪽 하단, 오른쪽 하단 |
| 크기 | 작음, 중간, 큼 |
| 테마 | 어둠, 밝음, 투명 |
| 강조 색상 | 스트림 스타일에 맞는 색상 선택 |
| 요소 | 표시할 내용 선택 (아래 참조) |

**사용 가능한 오버레이 요소:**

- 프린터 이름과 상태 (온라인/출력 중/오류)
- 완료 비율과 남은 시간이 있는 진행 바
- 필라멘트와 색상
- 노즐 온도와 플레이트 온도
- 사용된 필라멘트 (그램)
- AMS 개요 (컴팩트)
- Print Guard 상태

3. **미리보기**를 클릭하여 OBS로 전환하지 않고 결과를 확인합니다
4. **저장**을 클릭합니다

:::tip 프린터별 URL
여러 프린터가 있나요? 별도의 오버레이 URL을 사용합니다:
```
/obs/overlay?printer=1
/obs/overlay?printer=2
```
:::

## OBS의 카메라 피드 (별도 소스)

Bambu 카메라는 OBS에 별도의 소스로 추가할 수 있습니다 — 오버레이와 독립적으로:

**대안 1: 대시보드의 카메라 프록시를 통해**

1. **시스템 → 카메라**로 이동합니다
2. **RTSP 또는 MJPEG 스트리밍의 URL**을 복사합니다
3. OBS에서: **+**를 클릭 → **미디어 소스** (Media Source)
4. URL을 붙여넣습니다
5. **루프**를 체크하고 로컬 파일을 비활성화합니다

**대안 2: 카메라 뷰가 있는 브라우저 소스**

1. OBS에서: **브라우저 소스** 추가
2. URL: `http://localhost:3000/obs/camera?printer=1`
3. 너비/높이: 카메라 해상도에 맞춤 (1080p 또는 720p)

이제 카메라 피드를 장면에 자유롭게 배치하고 그 위에 오버레이를 겹칠 수 있습니다.

## 좋은 스트림을 위한 팁

### 스트림 장면 설정

3D 출력 스트리밍을 위한 일반적인 장면:

```
┌─────────────────────────────────────────┐
│                                         │
│      [프린터에서의 카메라 피드]         │
│                                         │
│  ┌──────────────────┐                  │
│  │ Bambu Overlay    │  ← 왼쪽 하단     │
│  │ 출력: Logo.3mf   │                  │
│  │ ████████░░ 82%   │                  │
│  │ 1시간 24분 남음  │                  │
│  │ 220°C / 60°C     │                  │
│  └──────────────────┘                  │
└─────────────────────────────────────────┘
```

### 권장 설정

| 매개변수 | 권장값 |
|---------|--------|
| 오버레이 크기 | 중간 (너무 지배적이지 않게) |
| 업데이트 빈도 | 30 FPS (OBS에 맞춤) |
| 오버레이 위치 | 왼쪽 하단 (얼굴/채팅 피하기) |
| 색상 테마 | 파란색 강조의 어두운 테마 |

### 장면 및 장면 전환

사용자 정의 OBS 장면 만들기:

- **"출력 진행 중"** — 카메라 뷰 + 오버레이
- **"일시 중지 / 대기 중"** — 정적 이미지 + 오버레이
- **"완료"** — 결과 이미지 + "완료"를 표시하는 오버레이

OBS의 키보드 단축키 또는 장면 컬렉션으로 장면을 전환합니다.

### 카메라 이미지 안정화

Bambu 카메라가 때때로 멈출 수 있습니다. 대시보드의 **시스템 → 카메라**에서:
- **자동 재연결** 활성화 — 대시보드가 자동으로 다시 연결됩니다
- **재연결 간격**을 10초로 설정합니다

### 오디오

3D 프린터는 소리를 냅니다 — 특히 AMS와 냉각. 고려 사항:
- 마이크를 프린터에서 멀리 배치합니다
- OBS에서 마이크에 노이즈 필터를 추가합니다 (Noise Suppression)
- 또는 배경 음악/채팅 오디오를 사용합니다

:::tip 자동 장면 전환
OBS에는 타이틀 기반 장면 전환에 대한 내장 지원이 있습니다. 플러그인 (예: obs-websocket)과 3DPrintForge API를 결합하여 출력이 시작되고 멈출 때 자동으로 장면을 전환합니다.
:::
