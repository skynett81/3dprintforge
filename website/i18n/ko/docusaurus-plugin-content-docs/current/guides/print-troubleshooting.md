# 출력 실패 문제 해결

뭔가 잘못됐나요? 걱정하지 마세요 — 대부분의 출력 문제는 간단한 해결책이 있습니다. Bambu Dashboard가 원인을 빠르게 찾는 데 도움을 줍니다.

## 1단계 — HMS 오류 코드 확인하기

HMS (Handling, Monitoring, Sensing)는 Bambu Labs의 오류 시스템입니다. 모든 오류가 대시보드에 자동으로 기록됩니다.

1. **모니터링 → 오류**로 이동합니다
2. 실패한 출력을 찾습니다
3. 오류 코드를 클릭하여 자세한 설명과 제안된 해결책을 확인합니다

일반적인 HMS 코드:

| 코드 | 설명 | 빠른 해결책 |
|------|------|-----------|
| 0700 1xxx | AMS 오류 (걸림, 모터 문제) | AMS의 필라멘트 경로 확인 |
| 0300 0xxx | 압출 오류 (과소/과다 압출) | 노즐 청소, 필라멘트 확인 |
| 0500 xxxx | 캘리브레이션 오류 | 재캘리브레이션 실행 |
| 1200 xxxx | 온도 편차 | 케이블 연결 확인 |
| 0C00 xxxx | 카메라 오류 | 프린터 재시작 |

:::tip 기록의 오류 코드
**기록 → [출력] → HMS 로그**에서 출력 중에 발생한 모든 오류 코드를 확인할 수 있습니다 — 출력이 "완료"된 경우에도.
:::

## 일반적인 문제와 해결책

### 접착력 불량 (첫 번째 레이어가 붙지 않음)

**증상:** 출력물이 플레이트에서 분리됨, 말림, 첫 번째 레이어 없음

**원인과 해결책:**

| 원인 | 해결책 |
|------|--------|
| 더러운 플레이트 | IPA 알코올로 닦기 |
| 잘못된 플레이트 온도 | 5°C 올리기 |
| Z-오프셋 오류 | 오토 베드 레벨링 재실행 |
| 접착제 스틱 부족 (PETG/ABS) | 얇은 접착제 스틱 층 바르기 |
| 첫 번째 레이어 속도 너무 빠름 | 첫 번째 레이어를 20〜30 mm/s로 낮추기 |

**빠른 체크리스트:**
1. 플레이트가 깨끗한가요? (IPA + 보풀 없는 종이타월)
2. 필라멘트 유형에 올바른 플레이트를 사용하고 있나요? ([올바른 플레이트 선택](./choosing-plate) 참조)
3. 마지막 플레이트 교체 후 Z 캘리브레이션을 했나요?

---

### 워핑 (모서리가 들림)

**증상:** 모서리가 플레이트에서 위로 휘어짐, 특히 크고 평평한 모델에서

**원인과 해결책:**

| 원인 | 해결책 |
|------|--------|
| 온도 차이 | 프린터 전면 커버 닫기 |
| 브림 부족 | Bambu Studio에서 브림 활성화 (3〜5 mm) |
| 플레이트가 너무 차가움 | 플레이트 온도 5〜10°C 올리기 |
| 수축률 높은 필라멘트 (ABS) | Engineering Plate + 챔버 >40°C |

**ABS와 ASA는 특히 취약합니다.** 항상 다음을 확인하세요:
- 전면 커버 닫기
- 환기 최소화
- Engineering Plate + 접착제 스틱
- 챔버 온도 40°C 이상

---

### 스트링잉 (부품 사이의 실)

**증상:** 모델의 독립된 부품 사이에 가는 플라스틱 실

**원인과 해결책:**

| 원인 | 해결책 |
|------|--------|
| 습기 있는 필라멘트 | 필라멘트 6〜8시간 건조 (60〜70°C) |
| 노즐 온도 너무 높음 | 5°C 낮추기 |
| 리트랙션 부족 | Bambu Studio에서 리트랙션 길이 늘리기 |
| 이동 속도 너무 낮음 | 이동 속도를 200+ mm/s로 높이기 |

**습기 테스트:** 딱딱 소리가 나는지 듣거나 압출 시 기포를 확인하세요 — 습기 있는 필라멘트를 나타냅니다. Bambu AMS에는 습도 측정기가 내장되어 있습니다; **AMS 상태**에서 습도를 확인하세요.

:::tip 필라멘트 건조기
나일론이나 TPU를 사용한다면 필라멘트 건조기 (예: Bambu Filament Dryer)에 투자하세요 — 이것들은 12시간 미만으로 습기를 흡수합니다.
:::

---

### 스파게티 (출력물이 덩어리로 붕괴)

**증상:** 필라멘트가 공중에서 느슨하게 매달림, 출력물을 알아볼 수 없음

**원인과 해결책:**

| 원인 | 해결책 |
|------|--------|
| 초기 접착력 불량 → 분리 → 붕괴 | 위의 접착력 섹션 참조 |
| 속도 너무 빠름 | 속도를 20〜30% 낮추기 |
| 잘못된 서포트 설정 | Bambu Studio에서 서포트 활성화 |
| 오버행이 너무 가파름 | 모델을 분할하거나 45° 회전 |

**스파게티를 자동으로 멈추려면 Print Guard 사용** — 다음 섹션을 참조하세요.

---

### 과소 압출 (얇고 약한 레이어)

**증상:** 레이어가 단단하지 않음, 벽에 구멍, 약한 모델

**원인과 해결책:**

| 원인 | 해결책 |
|------|--------|
| 노즐 부분 막힘 | 콜드 풀 실행 (유지 관리 참조) |
| 필라멘트 너무 습함 | 필라멘트 건조 |
| 온도 너무 낮음 | 노즐 온도 5〜10°C 높이기 |
| 속도 너무 빠름 | 20〜30% 낮추기 |
| PTFE 튜브 손상 | PTFE 튜브 검사 및 교체 |

## 자동 보호를 위한 Print Guard 사용

Print Guard는 이미지 인식으로 카메라 이미지를 모니터링하고 스파게티가 감지되면 자동으로 출력을 멈춥니다.

**Print Guard 활성화:**
1. **모니터링 → Print Guard**로 이동합니다
2. **자동 감지**를 활성화합니다
3. 동작을 선택합니다: **일시 중지** (권장) 또는 **취소**
4. 감도를 설정합니다 (**중간**으로 시작)

**Print Guard가 개입했을 때:**
1. 감지된 내용의 카메라 이미지가 담긴 알림을 받습니다
2. 출력이 일시 중지됩니다
3. 선택할 수 있습니다: **계속** (오탐지인 경우) 또는 **출력 취소**

:::info 오탐지
Print Guard는 가는 기둥이 많은 모델에 반응할 수 있습니다. 복잡한 모델에서는 감도를 낮추거나 일시적으로 비활성화하세요.
:::

## 대시보드의 진단 도구

### 온도 로그
**기록 → [출력] → 온도**에서 전체 출력 동안의 온도 곡선을 확인할 수 있습니다. 다음을 확인하세요:
- 갑작스러운 온도 강하 (노즐 또는 플레이트 문제)
- 불규칙한 온도 (캘리브레이션 필요)

### 필라멘트 통계
소비된 필라멘트가 추정값과 일치하는지 확인하세요. 큰 차이는 과소 압출 또는 필라멘트 끊김을 나타낼 수 있습니다.

## 지원팀에 연락해야 할 때

다음 경우에는 Bambu Labs 지원팀에 연락하세요:
- 모든 제안된 해결책을 따른 후에도 HMS 코드가 반복됨
- 프린터에 기계적 손상이 보임 (휘어진 가이드, 부러진 기어)
- 온도 값이 불가능함 (예: 노즐이 -40°C를 읽음)
- 펌웨어 업데이트로 문제가 해결되지 않음

**지원팀에 준비하면 유용한 것:**
- 대시보드 오류 로그의 HMS 오류 코드
- 오류의 카메라 이미지
- 사용된 필라멘트와 설정 (기록에서 내보낼 수 있음)
- 프린터 모델과 펌웨어 버전 (**설정 → 프린터 → 정보**에 표시됨)
