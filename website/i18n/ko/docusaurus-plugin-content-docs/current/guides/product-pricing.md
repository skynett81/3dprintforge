# 제품 가격 책정 — 판매 가격 계산

이 가이드는 비용 계산기를 사용하여 판매하는 3D 프린트의 적절한 판매 가격을 찾는 방법을 설명합니다.

## 비용 개요

3D 프린트의 비용은 다음 요소로 구성됩니다:

| 요소 | 설명 | 예시 |
|-----------|-------------|---------|
| **필라멘트** | 무게와 스풀 가격에 기반한 재료비 | 100g × 0.25 kr/g = 25 kr |
| **폐기물** | 재료 낭비 (퍼지, 실패 프린트, 서포트) | 10% 추가 = 2.50 kr |
| **전기** | 인쇄 중 전력 소비 | 3.5시간 × 150W × 1.50 kr/kWh = 0.79 kr |
| **마모** | 노즐 + 수명 기간 동안의 기계 가치 | 3.5시간 × 0.15 kr/시간 = 0.53 kr |
| **인건비** | 설정, 후처리, 포장에 드는 시간 | 10분 × 200 kr/시간 = 33.33 kr |
| **마크업** | 이익 마진 | 20% = 12.43 kr |

**총 생산 비용** = 모든 요소의 합계

## 설정 구성

### 기본 설정

**Filament → ⚙ 설정**으로 이동하여 입력합니다:

1. **전기 요금 (kr/kWh)** — 귀하의 전기 요금. 전기 요금 고지서를 확인하거나 Nordpool 통합을 사용하세요
2. **프린터 전력 (W)** — Bambu Lab 프린터의 경우 일반적으로 150W
3. **기계 비용 (kr)** — 프린터에 지불한 금액
4. **기계 수명 (시간)** — 예상 수명 (3000-8000시간)
5. **인건비 (kr/시간)** — 귀하의 시간당 요금
6. **설정 시간 (분)** — 필라멘트 교체, 플레이트 확인, 포장의 평균 시간
7. **마크업 (%)** — 원하는 이익 마진
8. **노즐 비용 (kr/시간)** — 노즐 마모 (HS01 ≈ 0.05 kr/시간)
9. **폐기물 계수** — 재료 낭비 (1.1 = 10% 추가, 1.15 = 15%)

:::tip Bambu Lab의 일반적인 값
| 설정 | 취미가 | 세미프로 | 전문가 |
|---|---|---|---|
| 전기 요금 | 1.50 kr/kWh | 1.50 kr/kWh | 1.00 kr/kWh |
| 프린터 전력 | 150W | 150W | 150W |
| 기계 비용 | 5 000 kr | 12 000 kr | 25 000 kr |
| 기계 수명 | 3 000시간 | 5 000시간 | 8 000시간 |
| 인건비 | 0 kr/시간 | 150 kr/시간 | 250 kr/시간 |
| 설정 시간 | 5분 | 10분 | 15분 |
| 마크업 | 0% | 30% | 50% |
| 폐기물 계수 | 1.05 | 1.10 | 1.15 |
:::

## 비용 계산

1. **비용 계산기** (`https://localhost:3443/#costestimator`)로 이동합니다
2. `.3mf` 또는 `.gcode` 파일을 **드래그 앤 드롭**합니다
3. 시스템이 자동으로 읽습니다: 필라멘트 무게, 예상 시간, 색상
4. **스풀 연결** — 재고에서 어떤 스풀을 사용할지 선택합니다
5. **비용 계산**을 클릭합니다

### 결과 표시:

- **필라멘트** — 색상별 재료비
- **폐기물/스크랩** — 폐기물 계수에 기반
- **전기** — 가능한 경우 Nordpool의 실시간 스팟 가격 사용
- **마모** — 노즐 + 기계 가치
- **인건비** — 시간당 요금 + 설정 시간
- **생산 비용** — 위 모든 항목의 합계
- **마크업** — 귀하의 이익 마진
- **총 비용** — 최소한 청구해야 할 금액
- **추천 판매 가격** — 2×, 2.5×, 3× 마진

## 가격 전략

### 2× 마진 (권장 최소값)
생산 비용 + 예상치 못한 비용을 커버합니다. 친구/가족 및 단순한 형상에 사용하세요.

### 2.5× 마진 (표준)
가격과 가치 사이의 좋은 균형. 대부분의 제품에 적합합니다.

### 3× 마진 (프리미엄)
복잡한 모델, 멀티컬러, 고품질 또는 틈새 시장용.

:::warning 숨겨진 비용을 잊지 마세요
- 실패 프린트 (모든 프린트의 5-15%가 실패합니다)
- 다 사용하지 못하는 필라멘트 (마지막 50g은 종종 어렵습니다)
- 고객 서비스에 소요되는 시간
- 포장 및 배송
- 프린터 유지보수
:::

## 예시: 폰 홀더 가격 책정

| 파라미터 | 값 |
|-----------|-------|
| 필라멘트 무게 | 45g PLA |
| 인쇄 시간 | 2시간 |
| 스팟 가격 | 1.20 kr/kWh |

**계산:**
- 필라멘트: 45g × 0.25 kr/g = 11.25 kr
- 폐기물 (10%): 1.13 kr
- 전기: 2시간 × 0.15kW × 1.20 = 0.36 kr
- 마모: 2시간 × 0.15 = 0.30 kr
- 인건비: (2시간 + 10분) × 200 kr/시간 = 433 kr (취미의 경우 0)
- **생산 비용 (취미)**: ~13 kr
- **판매 가격 2.5×**: ~33 kr

## 견적 저장

**견적 저장**을 클릭하여 계산을 보관합니다. 저장된 견적은 비용 계산기의 **저장됨** 탭에서 찾을 수 있습니다.

## 전자상거래

[전자상거래 모듈](../integrations/ecommerce)을 사용하는 경우, 비용 견적을 주문에 직접 연결하여 자동 가격 계산을 할 수 있습니다.
