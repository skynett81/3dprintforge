# 전기 요금

전기 요금 통합은 Tibber 또는 Nordpool에서 라이브 전기 요금을 가져와 프린트별 정확한 전력 비용 계산과 프린팅에 좋거나 나쁜 시간대에 대한 알림을 제공합니다.

이동: **https://localhost:3443/#settings** → **통합 → 전기 요금**

## Tibber 통합

Tibber는 스팟 가격을 위한 개방형 API를 제공하는 노르웨이 전력 공급업체입니다.

### 설정

1. [developer.tibber.com](https://developer.tibber.com)에 로그인합니다
2. **Personal Access Token** 생성
3. Bambu Dashboard에서: **Tibber API 토큰** 아래에 토큰 붙여넣기
4. **홈** 선택 (여러 홈이 있는 경우 요금을 가져올 홈)
5. **연결 테스트** 클릭
6. **저장** 클릭

### Tibber에서 사용 가능한 데이터

- **현재 스팟 가격** — 세금 포함 즉시 가격 (원/kWh)
- **다음 24시간 가격** — Tibber는 약 오후 1시부터 다음 날 가격 제공
- **요금 기록** — 최대 30일 이전까지
- **프린트별 비용** — 실제 프린팅 시간 × 시간별 요금으로 계산

## Nordpool 통합

Nordpool은 북유럽 지역의 원자재 스팟 가격을 제공하는 에너지 거래소입니다.

### 설정

1. **통합 → Nordpool**로 이동합니다
2. **가격 구역** 선택: NO1 (Oslo) / NO2 (Kristiansand) / NO3 (Trondheim) / NO4 (Tromsø) / NO5 (Bergen)
3. **통화** 선택: NOK / EUR
4. **세금 및 부과금** 선택:
   - **부가세 포함** (25%) 체크
   - **배전망 요금** (원/kWh) 입력 — 배전 회사 청구서 참조
   - **전력 소비세** (원/kWh) 입력
5. **저장** 클릭

:::info 배전망 요금
배전망 요금은 배전 회사 및 요금제에 따라 다릅니다. 정확한 요율은 최근 전기 청구서를 확인하세요.
:::

## 시간별 요금

시간별 요금은 향후 24–48시간에 대한 막대 그래프로 표시됩니다:

- **초록** — 저렴한 시간 (평균 이하)
- **노랑** — 평균 가격
- **빨강** — 비싼 시간 (평균 이상)
- **회색** — 요금 예측이 없는 시간

시간 위에 마우스를 올리면 정확한 가격 (원/kWh)을 볼 수 있습니다.

## 요금 기록

**전기 요금 → 기록**으로 이동하면:

- 최근 30일간 일별 평균 요금
- 일별 가장 비싸고 저렴한 시간
- 일별 프린트의 총 전기 비용

## 요금 알림

전기 요금을 기반으로 자동 알림을 설정합니다:

1. **전기 요금 → 요금 알림**으로 이동합니다
2. **새 알림** 클릭
3. 알림 유형 선택:
   - **가격이 한도 이하** — 전기 요금이 X원/kWh 이하로 떨어질 때 알림
   - **가격이 한도 이상** — 가격이 X원/kWh 이상으로 오를 때 알림
   - **오늘 가장 저렴한 시간** — 오늘 가장 저렴한 시간이 시작될 때 알림
4. 알림 채널 선택
5. **저장** 클릭

:::tip 스마트 계획
요금 알림과 출력 대기열을 결합하세요: 전기 요금이 낮을 때 자동으로 대기열 작업을 보내는 자동화를 설정합니다 (웹훅 통합 또는 Home Assistant 필요).
:::

## 비용 계산기의 전기 요금

전기 요금 통합이 활성화되면 [비용 계산기](../analyse/costestimator)에서 정확한 전력 비용을 제공합니다. 고정 가격 대신 **라이브 가격**을 선택하여 현재 Tibber/Nordpool 가격을 사용합니다.
