# 전력 측정

에너지 측정 기능이 있는 스마트 플러그를 프린터에 연결하여 추정치가 아닌 프린트별 실제 전력 소비를 기록합니다.

이동: **https://localhost:3443/#settings** → **통합 → 전력 측정**

## 지원 장치

| 장치 | 프로토콜 | 권장 사항 |
|---|---|---|
| **Shelly Plug S / Plus Plug S** | HTTP REST / MQTT | 권장 — 간단한 설정 |
| **Shelly 1PM / 2PM** | HTTP REST / MQTT | 고정 설치용 |
| **Shelly Gen2 / Gen3** | HTTP REST / MQTT | 확장된 API의 최신 모델 |
| **Tasmota 장치** | MQTT | 커스텀 설정에 유연 |

:::tip 권장 장치
펌웨어 1.0+ Shelly Plug S Plus는 테스트되고 권장됩니다. 클라우드 의존성 없이 Wi-Fi, MQTT 및 HTTP REST를 지원합니다.
:::

## Shelly 설정

### 사전 요구사항

- Shelly 플러그가 Bambu Dashboard와 동일한 네트워크에 연결
- Shelly가 고정 IP 또는 DHCP 예약으로 구성

### 구성

1. **설정 → 전력 측정**으로 이동합니다
2. **전력 측정기 추가** 클릭
3. **유형** 선택: Shelly
4. 다음을 입력합니다:
   - **IP 주소**: 예: `192.168.1.150`
   - **채널**: 0 (단일 콘센트 플러그의 경우)
   - **인증**: 구성된 경우 사용자 이름 및 비밀번호
5. **연결 테스트** 클릭
6. **프린터**에 플러그 연결: 드롭다운에서 선택
7. **저장** 클릭

### 폴링 간격

기본 폴링 간격은 10초입니다. 더 정확한 측정을 위해 5로 줄이거나, 네트워크 부하를 줄이려면 30으로 늘립니다.

## Tasmota 설정

1. MQTT로 Tasmota 장치 구성 (Tasmota 문서 참조)
2. Bambu Dashboard에서: **유형** 선택: Tasmota
3. 장치의 MQTT 토픽 입력: 예: `tasmota/power-plug-1`
4. 프린터에 연결하고 **저장** 클릭

Bambu Dashboard가 전력 측정을 위해 `{topic}/SENSOR`를 자동으로 구독합니다.

## 측정 항목

전력 측정이 활성화되면 프린트별로 다음이 기록됩니다:

| 메트릭 | 설명 |
|---|---|
| **즉시 전력** | 프린팅 중 와트 (라이브) |
| **총 에너지 소비** | 전체 프린트의 kWh |
| **평균 전력** | kWh / 프린팅 시간 |
| **에너지 비용** | kWh × 전기 요금 (Tibber/Nordpool에서) |

데이터는 프린트 기록에 저장되며 분석에 사용할 수 있습니다.

## 라이브 보기

즉시 전력 소비가 다음에 표시됩니다:

- **대시보드** — 추가 위젯으로 (위젯 설정에서 활성화)
- **플리트 개요** — 프린터 카드의 작은 표시기로

## 추정치와 비교

프린트 후 비교가 표시됩니다:

| | 추정 | 실제 |
|---|---|---|
| 에너지 소비 | 1.17 kWh | 1.09 kWh |
| 전력 비용 | 2.16원 | 2.02원 |
| 편차 | — | -6.8% |

일관된 편차는 [비용 계산기](../analyse/costestimator)에서 추정치를 보정하는 데 사용할 수 있습니다.

## 프린트 후 자동 전원 끄기

프린트 완료 후 Shelly/Tasmota가 자동으로 프린터를 끌 수 있습니다:

1. **전력 측정 → [프린터] → 자동 끄기**로 이동합니다
2. **프린트 완료 X분 후 끄기** 활성화
3. 지연 시간 설정 (예: 10분)

:::danger 냉각
전원을 끄기 전에 프린터를 최소 5–10분 동안 냉각시키십시오. 핫엔드의 열 크리프를 방지하기 위해 노즐이 50°C 이하로 냉각되어야 합니다.
:::
