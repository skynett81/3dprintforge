# 커뮤니티 필라멘트

커뮤니티 필라멘트는 전 세계 Bambu Dashboard 사용자들이 기여한 필라멘트 프로필의 공유 데이터베이스입니다. 내 필라멘트에 최적의 설정을 찾고, 직접 경험을 공유하세요.

이동: **https://localhost:3443/#community**

## 커뮤니티 프로필 탐색

### 검색

1. **커뮤니티 필라멘트**로 이동합니다
2. 검색 필드를 사용하여 다음으로 찾기:
   - 재료 (PLA, PETG, ABS, PA, TPU 등)
   - 공급업체 (PolyMaker, Bambu Lab, Prusament, Sunlu 등)
   - 색상 이름 또는 색상 HEX
   - 특성 (투명, 매트, 실크, 글리터 등)
3. **평점**으로 필터링 (최고 평점만)
4. **프린터 모델**로 필터링하여 해당 프린터에 최적화된 프로필 확인

### 프로필 카드

각 프로필 카드에 표시되는 정보:

| 항목 | 예시 |
|---|---|
| 필라멘트 이름 | PolyTerra PLA – Matte Forest Green |
| 재료 | PLA |
| 공급업체 | PolyMaker |
| 기여자 | 사용자명 |
| 평점 | ⭐⭐⭐⭐½ (4.6 / 5) |
| 테스트 수 | 23 |
| 권장 온도 | 노즐: 215°C / 베드: 55°C |
| 권장 속도 | 표준 |

카드를 클릭하면 모든 파라미터와 사용자 댓글이 포함된 전체 프로필을 볼 수 있습니다.

## 프로필 가져오기

1. 커뮤니티에서 프로필 열기
2. **재고로 가져오기** 클릭
3. 선택:
   - **기존 스풀에 연결** — 등록된 스풀에 프로필 연결
   - **새 스풀 만들기** — 프로필 데이터로 새 추적 항목 생성
4. 필요에 따라 온도 또는 기타 설정 조정
5. **가져오기** 클릭

이제 필라멘트 재고에서 프로필을 사용할 수 있습니다.

## 자신의 프로필 기여

### 내보내기 및 공유

1. **필라멘트 재고**로 이동합니다
2. 공유할 스풀 찾기
3. **커뮤니티와 공유** 클릭
4. 다음을 입력합니다:
   - **공급업체 이름** — 정확한 공급업체 이름
   - **제품 이름** — 포장에 표기된 정확한 제품명
   - **색상 이름** — 공식 색상 이름과 색상 HEX
   - **댓글** — 경험, 팁 및 요령
   - **프린터 모델** — 테스트한 프린터
5. **제출** 클릭

프로필이 검토 중이며 보통 24시간 내에 게시됩니다.

### 기여 지침

:::warning 품질 요구사항
- 실제로 테스트한 설정만 제공 — 추측 값 제외
- 정확한 공급업체 및 제품명 사용 (포장 확인)
- 특정 제품 및 프린터 모델당 하나의 프로필
- 중복 프로필 방지 — 프로필이 이미 있는지 확인
:::

## 프로필 평가

테스트한 프로필을 평가하여 커뮤니티를 도와주세요:

1. 커뮤니티 프로필 열기
2. **평가하기** 클릭
3. 별점 (1–5) 및 짧은 댓글 입력
4. **전송** 클릭

높은 평점의 프로필이 검색 결과 상위에 표시됩니다.

## 오프라인 사용

커뮤니티 데이터베이스는 자동으로 로컬에 동기화됩니다 (매일 업데이트). 첫 번째 동기화 후 인터넷 연결 없이도 모든 프로필을 사용할 수 있습니다.

수동 데이터베이스 다운로드: **커뮤니티 필라멘트 → 설정 → 지금 동기화**
