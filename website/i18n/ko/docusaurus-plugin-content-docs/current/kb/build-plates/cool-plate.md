# Cool Plate (Smooth PEI)

Cool Plate는 프린트에 광택 있고 매끄러운 하단면을 제공하는 매끄러운 PEI 플레이트입니다. 대부분의 Bambu Lab 프린터에 표준 플레이트로 포함되어 있습니다.

## 특성

- **표면:** 프린트에 매끄럽고 광택 있는 하단면
- **재료:** 스프링 강철 플레이트에 PEI (Polyetherimide) 코팅
- **최대 베드 온도:** 90°C

## 가장 적합한 재료

- PLA (모든 변종) — 접착제 없이
- PLA+ — 접착제 없이
- PLA Silk — 접착제 없이
- PLA-CF — 접착제 없이

## 주의하여 사용

:::danger PETG — 항상 접착제 사용
PETG는 Smooth PEI에 **매우 강하게** 접착됩니다. 접착제 없이는 프린트를 제거할 때 플레이트에서 PEI 코팅이 벗겨질 위험이 있습니다. Smooth PEI에는 항상 접착제를 장벽으로 사용하거나 Engineering Plate를 선택하세요.
:::

## 유지보수

### 각 프린트 후
- 프린트를 제거하기 전에 플레이트를 실온 (40°C 이하)으로 냉각시킵니다
- PLA는 25–30°C에서 자연스럽게 플레이트에서 분리됩니다
- 플레이트를 조심스럽게 구부려 고집스러운 프린트를 떼어냅니다

### 주간 청소
이소프로판올 (IPA, 70–99%)로 플레이트를 닦습니다:
```
1. 깨끗하고 보풀 없는 페이퍼 타월에 IPA를 적십니다 (플레이트에 직접 스프레이하지 마세요)
2. 한 방향으로 표면을 닦습니다
3. 다음 프린트 전에 완전히 건조시킵니다 (30초)
```

### 접착 불량 시
1. 따뜻한 물과 순한 세제로 씻습니다
2. 충분히 헹굽니다
3. 보풀 없는 페이퍼로 건조시킵니다
4. 마지막으로 IPA로 닦습니다

:::warning 아세톤 사용 금지
아세톤은 PEI 코팅을 녹여 플레이트를 영구적으로 손상시킵니다. IPA 또는 물/세제만 사용하세요.
:::

## 플레이트 교체

다음의 경우 플레이트를 교체합니다:
- PEI 코팅이 눈에 띄게 손상됨 (긁힘, 흉터, 벗겨짐)
- 철저한 청소 후에도 프린트가 더 이상 접착되지 않음
- 표면이 불균일하여 첫 번째 레이어가 불량

Cool Plate는 정상적인 사용과 적절한 유지보수로 보통 200–500회 프린트를 견딥니다.
