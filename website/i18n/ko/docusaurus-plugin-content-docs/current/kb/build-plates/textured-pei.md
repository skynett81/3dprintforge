# Textured PEI Plate

Textured PEI Plate (Engineering Plate와 혼동하지 마세요)는 더 거친 텍스처를 가지며 TPU 및 기타 유연한 재료에 특히 적합합니다. 또한 다양한 일반 재료에도 사용할 수 있습니다.

## 특성

- **표면:** 거친 PEI 텍스처 — 프린트에 뚜렷한 텍스처드 하단면 제공
- **접착력:** 탁월한 기계적 그립, 특히 TPU
- **제거:** 냉각 후 매우 쉬움 — 플레이트를 구부리면 프린트가 분리됨
- **최대 베드 온도:** 100°C

## 가장 적합한 재료

- **TPU** — 이것이 유연한 재료를 위한 최고의 플레이트
- PLA 및 PLA+ — 좋은 접착력, 아름다운 텍스처드 하단면
- PETG — 소량의 접착제 사용
- PLA-CF — 접착제 없이

## 호환성

| 재료 | 적합도 | 접착제 |
|-----------|---------|---------|
| TPU | ★★★★★ | 아니오 |
| PLA | ★★★★★ | 아니오 |
| PLA+ | ★★★★★ | 아니오 |
| PETG | ★★★★ | 소량 (권장) |
| PLA-CF | ★★★★ | 아니오 |
| ABS | ★★★ | 예 |
| PA12 | ★★★ | 예 |

## Textured PEI에서 TPU

TPU가 접착제 없이 Textured PEI에 완벽하게 접착됩니다:
1. 베드 온도 30–45°C 설정
2. Z 오프셋을 세심하게 보정 — PLA보다 약간 더 높게
3. 프린트 후 베드를 30°C 이하로 냉각시킵니다
4. 플레이트를 조심스럽게 구부립니다 — TPU가 힘들이지 않고 분리됩니다

:::tip 냉각 후 제거
TPU는 냉각 시 Textured PEI에서 완벽하게 분리됩니다. 프린트를 제거하는 데 문제가 있으면 플레이트가 완전히 냉각될 때까지 (20–25°C) 기다렸다가 더 강하게 구부리세요.
:::

## 유지보수

Textured PEI는 견고하고 유지보수가 간단합니다:

```bash
# 프린트 사이:
- IPA와 보풀 없는 페이퍼 타월로 닦습니다

# 월간 (또는 접착 불량 시):
- 따뜻한 물과 순한 세제로 씻습니다
- 헹구고 철저히 건조시킵니다
- 마지막으로 IPA로 처리합니다
```

## 완성된 프린트의 표면

거친 텍스처는 모든 프린트의 하단면에 뚜렷한 패턴을 남깁니다. 이것은:
- 프로토타입 및 기능성 부품에 원하는 결과 (더 나은 그립)
- 장식적/시각적 프린트에는 바람직하지 않음 (매끄러운 표면은 Cool Plate 사용)
