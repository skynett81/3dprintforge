# Engineering Plate (Textured PEI)

Engineering Plate는 PETG, ABS, ASA 및 PA와 같은 기술 재료를 위해 설계된 텍스처드 PEI 플레이트입니다. 아름다운 텍스처드 하단면을 제공하며 Bambu Lab 라인업에서 가장 다재다능한 플레이트입니다.

## 특성

- **표면:** 텍스처드 PEI — 프린트에 패턴/매트 하단면 제공
- **접착력:** Smooth PEI보다 더 강한 기계적 접착력
- **최대 베드 온도:** 110°C
- **제거:** 40–50°C로 냉각 후 프린트가 쉽게 분리됨

## 가장 적합한 재료

- PETG 및 PETG-CF (접착제 없이, 또는 추가 안전을 위해 사용)
- ABS 및 ASA (접착제 사용)
- PA6 / PA12 / PA-CF (접착제 필수)
- TPU (접착제 없이)
- PLA (잘 작동하며 텍스처드 하단면 제공)

## 적합하지 않은 재료

- 매끄러운 하단면이 필요한 프린트 (Cool Plate 사용)
- 고온의 PC (High Temp Plate 사용)

## 유지보수

### 청소
```bash
# 주간 (또는 접착제 사용 후):
1. 따뜻한 물과 순한 세제로 씻습니다
2. 충분히 헹굽니다
3. 건조시키거나 보풀 없는 페이퍼로 닦습니다
4. 마지막으로 IPA로 닦습니다

# 접착제 없이 프린트 사이:
1. IPA와 보풀 없는 페이퍼로 닦습니다
```

### 접착제 사용 후
접착제 사용 후에는 항상 플레이트를 씻으세요. 남은 접착제 잔여물이 다음 프린트에서 불균일한 접착력을 줄 수 있습니다.

:::tip 텍스처 방향
Engineering Plate는 텍스처에 방향이 있습니다. 텍스처의 홈에 프린트가 끼는 것을 방지하기 위해 항상 클립이 올바른 방향으로 플레이트를 장착하세요.
:::

## 플레이트 교체

다음의 경우 Engineering Plate를 교체합니다:
- 중앙 부분의 텍스처가 닳아서 매끄러워짐 (가장 많이 사용되는 부분)
- 청소와 접착제 후에도 프린트가 접착되지 않음
- 플레이트에 깊은 긁힘이나 변형이 있음

정상 수명: 적절한 유지보수로 300–700회 프린트.
