# High Temp Plate

High Temp Plate는 높은 베드 온도가 필요한 재료를 위해 설계되었습니다. 최대 120°C를 견디며 폴리카보네이트 (PC)와 고온 변종 PA에 필수적입니다.

## 특성

- **표면:** 고온 코팅의 Smooth (매끄러움)
- **최대 베드 온도:** 120°C
- **프린트 하단면:** 광택/매끄러움
- **접착력:** 고온에서 양호 — 냉각 시 분리됨

## 가장 적합한 재료

| 재료 | 베드 온도 | 접착제 |
|-----------|---------------|---------|
| ABS | 90–110°C | 예 |
| ASA | 90–110°C | 예 |
| PC | 100–120°C | 예 (필수) |
| PA6 | 80–90°C | 예 |
| PA12 | 70–85°C | 예 |
| PA-CF | 90–100°C | 예 |

## 적합하지 않은 재료

- PLA — 플레이트가 너무 뜨거워서 PLA가 과도하게 접착됨
- PETG — 접착제 없이 플레이트 손상 위험
- TPU — Engineering Plate가 더 적합

## 접착제

접착제는 강력히 권장되며 (PC의 경우 필수):
1. 첫 번째 레이어의 접착력 향상
2. 플레이트 코팅 보호
3. 냉각 후 프린트 제거 용이

얇고 균일하게 한 층 바릅니다. PC 및 PA의 경우: ABS보다 약간 더 많이 사용합니다.

:::warning PC는 특별한 주의가 필요
폴리카보네이트 (PC)는 250–280°C 노즐과 100–120°C 베드에서 프린팅됩니다. 밀폐된 챔버와 High Temp Plate 없이는 성공률이 매우 낮습니다. X1C와 P1S만 PC 프린팅에 권장됩니다.
:::

## 유지보수

```bash
# PC/ABS 접착제 사용 후:
1. 플레이트를 50°C로 냉각시킵니다
2. 프린트를 조심스럽게 제거합니다 (플레이트 구부리기)
3. 따뜻한 물과 세제로 씻습니다
4. 철저히 건조시킵니다 — 뜨거운 플레이트의 물은 코팅을 손상시킬 수 있음

# 프린트 사이:
1. IPA로 닦습니다
2. 코팅 마모 여부 확인
```

:::danger 열 충격 방지
차가운 플레이트를 뜨거운 베드에 놓거나 뜨거운 플레이트에 차가운 IPA를 스프레이하지 마세요. 온도 차이가 코팅을 손상시키거나 강철 플레이트를 변형시킬 수 있습니다.
:::

## 수명

High Temp Plate는 더 높은 열 부하로 인해 Engineering Plate보다 수명이 짧습니다:
- **일반 사용 (ABS/ASA):** 200–400회 프린트
- **집중 사용 (PC):** 100–200회 프린트

코팅에 눈에 띄는 마모가 있거나 지속적인 접착 문제가 발생할 때 교체합니다.
