# 호환성 매트릭스

이 페이지는 어떤 재료가 어떤 빌드 플레이트, 프린터 및 노즐 유형과 호환되는지에 대한 완전한 개요를 제공합니다. 새로운 재료로 프린트를 계획할 때 참고로 사용하세요.

| 재료 | Cool Plate | Engineering Plate | High Temp Plate | Textured PEI | 풀칠 |

| PVB | 양호 | 양호 | 비권장 | 양호 | 불필요 |

**범례:**
- **우수** — 최적 작동, 권장 조합
- **양호** — 잘 작동, 허용 가능한 대안
- **허용 가능** — 작동하지만 이상적이지 않음 — 추가 조치 필요
- **불량** — 수정하면 작동 가능하지만 비권장
- **비권장** — 결과가 좋지 않거나 플레이트 손상 위험

:::tip PETG와 Cool Plate
PETG는 Cool Plate(Smooth PEI)에 **너무 강하게 부착**되어 부품 제거 시 PEI 코팅을 벗길 수 있습니다. 항상 풀칠을 분리막으로 사용하거나 Engineering Plate를 선택하세요.
:::

:::warning PC와 플레이트 선택
PC는 높은 베드 온도(100–120 °C) 때문에 High Temp Plate가 필요합니다. 다른 플레이트는 이 온도에서 영구적으로 변형될 수 있습니다.
:::

:::danger 탄소섬유와 유리섬유는 경화 노즐 필요
**-CF**(탄소섬유) 또는 **-GF**(유리섬유)가 포함된 모든 재료는 **경화 스틸 노즐**이 필요합니다. 황동은 이러한 재료로 수 시간에서 수 일 만에 마모됩니다. Bambu Lab HS01이 권장됩니다.

경화 노즐이 필요한 재료:
- PLA-CF
- PETG-CF
- PA-CF / PA-GF
- PC-CF / PC-GF
:::

:::tip 일반 재료에서 황동 vs 경화 스틸
황동 노즐은 일반 재료(PLA, PETG, ABS)에 **더 나은 열전도성**을 제공하여 더 균일한 압출이 가능합니다. 경화 스틸도 잘 작동하지만 5–10 °C 높은 온도가 필요할 수 있습니다. 일상 사용에는 황동을, CF/GF 재료에는 경화 스틸로 교체하세요.
:::

## 재료 교체 팁

AMS 내에서 또는 수동으로 재료를 교체할 때, 오염을 방지하기 위해 적절한 퍼지가 중요합니다.

### 권장 퍼지량

| 교체 | 퍼지량 | 비고 |
|-----------|-------------|---------|
| PLA → PLA (다른 색) | 100–150 mm³ | 표준 색상 전환 |
| PLA → PETG | 200–300 mm³ | 온도 상승, 다른 유동 |
| PETG → PLA | 200–300 mm³ | 온도 하강 |
| ABS → PLA | 300–400 mm³ | 큰 온도 차이 |
| PLA → ABS | 300–400 mm³ | 큰 온도 차이 |
| PA → PLA | 400–500 mm³ | 나일론이 핫엔드에 잔류 |
| PC → PLA | 400–500 mm³ | PC는 철저한 퍼지 필요 |
| 어두운 → 밝은 색 | 200–300 mm³ | 어두운 안료는 제거 어려움 |
| 밝은 → 어두운 색 | 100–150 mm³ | 쉬운 전환 |

### 재료 교체 시 온도 변경

| 전환 | 권장사항 |
|----------|-----------|
| 저온 → 고온 (예: PLA → ABS) | 새 재료 온도로 가열, 철저히 퍼지 |
| 고온 → 저온 (예: ABS → PLA) | 고온에서 먼저 퍼지, 그 후 온도 낮추기 |
| 유사한 온도 (예: PLA → PLA) | 표준 퍼지 |
| 큰 차이 (예: PLA → PC) | PETG로 중간 단계가 도움될 수 있음 |

:::warning 나일론과 PC는 잔류물을 남김
PA(나일론)와 PC는 특히 퍼지가 어렵습니다. 이 재료 사용 후:
1. **PETG** 또는 **ABS**로 고온(260–280 °C)에서 퍼지
2. 최소 **500 mm³** 퍼지 재료 사용
3. 압출을 시각적으로 확인 — 변색 없이 완전히 깨끗해야 함
:::
