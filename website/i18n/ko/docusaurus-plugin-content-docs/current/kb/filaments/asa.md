# ASA

ASA(Acrylonitrile Styrene Acrylate)는 야외 사용을 위해 특별히 개발된 ABS의 UV 내성 변형입니다. ABS의 강도와 강성에 UV 방사, 노화 및 기상 영향에 대한 크게 향상된 내성을 결합합니다.

## 설정

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 240–260 °C |
| 베드 온도 | 90–110 °C |
| 챔버 온도 | 40–50 °C (권장) |
| 부품 냉각 | 30–50% |
| 속도 | 80–100% |
| 건조 필요 | 예 |

## 권장 빌드 플레이트

| 플레이트 | 적합성 | 풀칠? |
|-------|---------|----------|
| Engineering Plate | 우수 | 아니오 |
| High Temp Plate | 양호 | 예 |
| Textured PEI | 허용 가능 | 예 |
| Cool Plate (Smooth PEI) | 비권장 | — |

:::tip Engineering Plate가 ASA에 최적
Engineering Plate는 풀칠 없이도 ASA에 가장 안정적인 접착력을 제공합니다. 높은 베드 온도를 견디며 부품이 영구적으로 붙지 않으면서 좋은 접착력을 발휘합니다.
:::

## 프린터 요구사항

ASA는 최상의 결과를 위해 **밀폐 챔버(인클로저)**가 필요합니다. 인클로저 없이는 다음과 같은 문제가 발생합니다:

- **뒤틀림** — 모서리가 빌드 플레이트에서 들어올림
- **층간 분리** — 층 사이 접착 불량
- **표면 균열** — 프린트를 따라 보이는 균열

| 프린터 | ASA 적합 여부 | 비고 |
|---------|---------------|---------|
| X1C | 우수 | 완전 밀폐, 능동 가열 |
| X1E | 우수 | 완전 밀폐, 능동 가열 |
| P1S | 양호 | 밀폐, 수동 가열 |
| P1P | 액세서리 필요 | 인클로저 액세서리 필요 |
| A1 | 비권장 | 오픈 프레임 |
| A1 Mini | 비권장 | 오픈 프레임 |

## ASA vs ABS — 비교

| 특성 | ASA | ABS |
|----------|-----|-----|
| UV 내성 | 우수 | 불량 |
| 야외 사용 | 예 | 아니오 (황변 및 취화) |
| 뒤틀림 | 보통 | 높음 |
| 표면 | 매트, 균일 | 매트, 균일 |
| 내약품성 | 양호 | 양호 |
| 가격 | 다소 높음 | 낮음 |
| 프린트 중 냄새 | 보통 | 강함 |
| 충격 강도 | 양호 | 양호 |
| 내열 온도 | ~95–105 °C | ~95–105 °C |

:::warning 환기
ASA는 프린팅 중 자극적인 가스를 배출합니다. 환기가 잘 되는 방이나 공기 필터 시스템이 있는 곳에서 프린트하세요. 환기 없이 장시간 머무는 방에서 ASA를 프린트하지 마세요.
:::

## 건조

ASA는 **중간 수준의 흡습성**을 가지며 시간이 지남에 따라 공기 중 수분을 흡수합니다.

| 매개변수 | 값 |
|-----------|-------|
| 건조 온도 | 65 °C |
| 건조 시간 | 4–6시간 |
| 흡습 수준 | 보통 |
| 습기 징후 | 탁탁 소리, 기포, 표면 품질 저하 |

- 개봉 후 실리카겔이 포함된 밀봉 봉투에 보관
- 건조제가 있는 AMS는 단기 보관에 충분
- 장기 보관: 진공 봉투 또는 필라멘트 건조 박스 사용

## 용도

ASA는 **야외**에서 사용되는 모든 것에 선호되는 소재입니다:

- **자동차 부품** — 미러 하우징, 대시보드 디테일, 밸브 캡
- **정원 도구** — 클립, 클램프, 정원 가구 부품
- **야외 표지판** — 사인, 글자, 로고
- **드론 부품** — 랜딩 기어, 카메라 마운트
- **태양광 패널 마운트** — 브래킷과 앵글
- **우편함 부품** — 메커니즘과 장식

## 성공적인 ASA 프린팅 팁

### 브림과 접착

- **큰 부품**이나 접촉 면적이 작은 부품에는 브림 사용
- 5–8 mm 브림으로 효과적으로 뒤틀림 방지
- 작은 부품은 브림 없이 시도 가능하나, 백업으로 준비해 둘 것

### 외풍 방지

- 프린팅 중 방의 **모든 문과 창문을 닫으세요**
- 외풍과 찬 공기는 ASA의 최대 적
- 프린팅 중 챔버 도어를 열지 마세요

### 온도 안정성

- 프린팅 시작 전 챔버를 **10–15분** 예열
- 안정적인 챔버 온도로 더 균일한 결과 달성
- 프린터를 창문이나 환기구 근처에 배치하지 마세요

### 냉각

- ASA는 **제한된 부품 냉각**이 필요 — 30–50%가 일반적
- 오버행과 브릿지에서는 60–70%까지 올릴 수 있으나, 약간의 층간 분리 예상
- 기계 부품: 디테일보다 층간 접착을 우선하여 냉각 낮추기

:::tip ASA 처음이신가요?
설정을 교정하기 위해 작은 테스트 부품(예: 30 mm 큐브)으로 시작하세요. ASA는 ABS와 매우 유사하게 동작하지만, 뒤틀림 경향이 약간 낮습니다. ABS 경험이 있다면 ASA가 업그레이드처럼 느껴질 것입니다.
:::

## 수축

ASA는 PLA와 PETG보다 수축이 크지만, 일반적으로 ABS보다는 약간 작습니다:

| 소재 | 수축률 |
|-----------|----------|
| PLA | ~0.3–0.5% |
| PETG | ~0.3–0.6% |
| ASA | ~0.5–0.7% |
| ABS | ~0.7–0.8% |

정밀한 공차가 필요한 부품: 슬라이서에서 0.5–0.7% 보정하거나, 테스트 부품으로 먼저 확인하세요.

## 후처리

- **아세톤 증기 처리** — ASA는 ABS와 마찬가지로 아세톤 증기로 표면을 매끄럽게 할 수 있음
- **샌딩** — 200–400방 사포로 잘 연마됨
- **접착** — 순간접착제 또는 아세톤 접착이 우수
- **도색** — 가벼운 샌딩 후 페인트가 잘 부착됨

:::danger 아세톤 취급
아세톤은 가연성이며 유독 가스를 발생시킵니다. 항상 환기가 잘 되는 방에서 사용하고, 화기를 피하며, 보호 장비(장갑 및 보안경)를 착용하세요.
:::
