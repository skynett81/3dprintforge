# Bambu Lab 필라멘트

Bambu Lab은 자사 프린터에 특별히 최적화된 다양한 필라멘트를 생산합니다. 모든 Bambu Lab 필라멘트에는 프린터가 자동으로 감지하여 올바른 설정을 적용하는 **RFID 태그**가 포함되어 있습니다.

## RFID와 AMS

모든 Bambu Lab 필라멘트에는 스풀에 **RFID 칩**이 내장되어 있습니다. 이를 통해:

- **자동 인식** — 프린터가 재료 유형, 색상 및 설정을 읽음
- **잔량 표시** — 스풀에 남은 추정 필라멘트
- **정확한 설정** — 온도, 속도 및 냉각이 자동 설정
- **AMS 호환성** — AMS 내에서 원활한 재료 전환

:::tip AMS에서 서드파티 필라멘트
AMS는 서드파티 필라멘트에서도 작동하지만, Bambu Studio에서 설정을 수동으로 지정해야 합니다. RFID 자동 감지는 Bambu Lab 필라멘트 전용입니다.
:::

## PLA 시리즈

Bambu Lab의 PLA 시리즈는 가장 포괄적이며, 기본 제품부터 특수 효과까지 모든 것을 다룹니다.

### PLA Basic

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 220 °C |
| 베드 온도 | 35–45 °C |
| 냉각 | 100% |
| RFID | 예 |
| AMS 호환 | 예 |
| 가격 | 예산형 |

일상 프린트를 위한 표준 필라멘트. 다양한 색상으로 제공.

### PLA Matte

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 220 °C |
| 베드 온도 | 35–45 °C |
| 냉각 | 100% |
| 표면 | 매트, 광택 없음 |

표준 PLA보다 적층 라인을 더 잘 숨기는 균일한 매트 표면 제공. 미관 프린트에 인기 있는 선택.

### PLA Silk

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 230 °C |
| 베드 온도 | 45–55 °C |
| 냉각 | 80% |
| 표면 | 광택, 메탈릭 느낌 |

메탈릭 효과의 광택 있는 실크 같은 표면 제공. 표준 PLA보다 약간 낮은 냉각과 속도 필요.

### PLA Sparkle

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 220–230 °C |
| 베드 온도 | 35–45 °C |
| 냉각 | 100% |
| 표면 | 글리터 입자 |

반짝이는 효과를 주는 글리터 입자 포함. 표준 PLA 설정으로 프린트.

### PLA Marble

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 220 °C |
| 베드 온도 | 35–45 °C |
| 냉각 | 100% |
| 표면 | 대리석 패턴 |

프린트 전체에 색상 변화가 있는 독특한 대리석 효과 제공. 각 프린트가 약간 다릅니다.

### PLA Tough (PLA-S)

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 220–230 °C |
| 베드 온도 | 35–55 °C |
| 냉각 | 100% |
| 강도 | 표준 PLA 대비 20–30% 강화 |

충격 강도가 향상된 강화 PLA. 표준 PLA보다 더 많은 강도가 필요한 기계 부품에 적합.

### PLA Galaxy

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 220–230 °C |
| 베드 온도 | 35–45 °C |
| 냉각 | 100% |
| 표면 | 글리터 + 색상 그라데이션 |

글리터 효과와 색상 그라데이션을 결합한 독특한 시각 효과. 표준 PLA 설정으로 프린트.

## PETG 시리즈

### PETG Basic

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 240–250 °C |
| 베드 온도 | 70–80 °C |
| 냉각 | 50–70% |
| RFID | 예 |
| AMS 호환 | 예 |

양호한 강도와 유연성을 가진 표준 PETG. 다양한 색상으로 제공.

### PETG HF (High Flow)

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 240–260 °C |
| 베드 온도 | 70–80 °C |
| 냉각 | 50–70% |
| 속도 | 최대 300 mm/s |

품질을 희생하지 않고 더 빠른 압출을 위해 배합된 고속 PETG. 대형 부품과 배치 생산에 이상적.

### PETG-CF

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 250–270 °C |
| 베드 온도 | 70–80 °C |
| 냉각 | 40–60% |
| 노즐 | 경화 스틸 필수 |

강성과 치수 안정성이 향상된 탄소섬유 강화 PETG. 경화 노즐(HS01 또는 동급) 필요.

:::warning CF 변형에는 경화 노즐
모든 탄소섬유 강화 필라멘트(PLA-CF, PETG-CF, PA-CF, PC-CF)에는 경화 스틸 노즐이 필요합니다. 황동은 CF 재료로 수 시간에서 수 일 만에 마모됩니다.
:::

## TPU 95A

| 매개변수 | 값 |
|-----------|-------|
| 노즐 온도 | 220–240 °C |
| 베드 온도 | 35–50 °C |
| 냉각 | 50–80% |
| 속도 | 50–70% (감속) |
| 쇼어 경도 | 95A |
| AMS 호환 | 제한적 (직접 공급 권장) |

고무와 같은 부품을 위한 유연한 필라멘트. AMS는 TPU 95A를 처리할 수 있지만, 더 부드러운 변형에서는 직접 공급이 더 나은 결과를 제공합니다.

:::tip AMS에서의 TPU
Bambu Lab의 TPU 95A는 AMS에서 작동하도록 특별히 배합되었습니다. 더 부드러운 TPU(85A 이하)는 익스트루더에 직접 공급해야 합니다.
:::

## 품질 관리 및 색상 일관성

Bambu Lab은 필라멘트에 대해 엄격한 품질 관리를 실시합니다:

- **직경 공차** — ±0.02 mm (업계 최고 수준)
- **색상 일관성** — 배치 관리로 스풀 간 동일한 색상 보장
- **스풀 품질** — 매듭이나 겹침 없는 균일한 감기
- **진공 밀봉** — 각 스풀은 건조제와 함께 진공 포장
- **온도 프로파일 테스트** — 각 배치는 최적 온도로 테스트

:::tip 일관성을 위한 색상 번호
Bambu Lab은 배치 관리가 포함된 색상 번호(예: "Bambu PLA Matte Charcoal")를 사용합니다. 대형 프로젝트에서 여러 스풀에 걸쳐 동일한 색상이 필요한 경우, 같은 배치에서 주문하거나 배치 매칭에 대해 지원팀에 문의하세요.
:::
