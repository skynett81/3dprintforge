## 소재별 팁

### 표준 (PLA, PETG)
- PLA는 가장 쉬운 소재 — 프로토타입과 장식에 완벽
- PETG는 더 나은 강도와 내열성을 제공하지만 실 끌림이 더 발생할 수 있음
- 둘 다 인클로저 없이 표준 황동 노즐로 사용 가능

### 엔지니어링 (ABS, ASA, PC)
- 워핑을 방지하기 위해 밀폐형 프린터 필요
- ASA는 UV 내성 — 야외 부품에 사용
- PC는 최고의 강도와 내열성을 제공하지만 출력이 가장 어려움

### 복합소재 (CF/GF)
- **반드시** 경화 스틸 노즐을 사용 — 탄소 섬유가 황동 노즐을 빠르게 마모시킴
- CF 변형은 더 단단하고 가벼운 부품을 매트 표면으로 생산
- GF 변형은 더 저렴하지만 표면이 더 거칠음

### 유연성 (TPU)
- 최상의 결과를 위해 천천히 출력 (50 mm/s 이하)
- 막힘을 방지하기 위해 리트랙션 감소
- 다이렉트 드라이브 익스트루더가 보든 방식보다 훨씬 효과적

### 서포트 (PVA, HIPS)
- PVA는 물에 녹음 — PLA 서포트에 완벽
- HIPS는 리모넨에 녹음 — ABS와 함께 사용
- 둘 다 습기에 매우 민감 — 건조하게 보관

:::warning 나일론 소재
PA, PA-CF, PA-GF는 **극도로** 흡습성이 높습니다. 몇 분 만에 공기 중의 수분을 흡수합니다. 사용 전에 반드시 12시간 이상 건조하고, 가능하면 드라이 박스에서 직접 출력하세요.
:::
