# PETG

PETG (Polyethylene Terephthalate Glycol)는 기능성 부품에 인기 있는 재료입니다. PLA보다 강하고 열 안정적이며, 가벼운 화학 물질에도 잘 견딥니다.

## 설정

| 파라미터 | 값 |
|-----------|-------|
| 노즐 온도 | 230–250°C |
| 베드 온도 | 70–85°C |
| 파트 쿨링 | 30–60% |
| 속도 | 표준 |
| 건조 | 권장 (65°C에서 6–8시간) |

## 권장 빌드 플레이트

| 플레이트 | 적합도 | 접착제? |
|-------|---------|----------|
| Engineering Plate (Textured PEI) | 우수 | 아니오/예* |
| Textured PEI | 양호 | 예** |
| Cool Plate (Smooth PEI) | 경고 참조 | 경고 참조 |
| High Temp Plate | 양호 | 예 |

:::danger 중요: PETG와 Smooth PEI에서의 접착제
PETG는 Smooth PEI (Cool Plate)에 **매우 강하게** 접착됩니다. 접착제 없이는 프린트를 제거할 때 **플레이트에서 코팅이 벗겨질** 위험이 있습니다. Smooth PEI에서 PETG를 프린팅할 때는 항상 얇은 접착제 층을 장벽으로 사용하세요.

**대안:** Engineering Plate 또는 Textured PEI 사용 — 플레이트를 손상시키지 않고 우수한 접착력을 제공합니다.
:::

## 성공적인 프린팅을 위한 팁

- **파트 쿨링 줄이기** — 너무 많은 쿨링은 레이어 분리와 취성 프린트를 유발합니다
- **노즐 온도 높이기** — 스트링 시 5–10°C 낮추기; 불량한 레이어 융합 시 높이기
- **첫 번째 레이어 베드 온도** — 우수한 접착을 위해 80–85°C, 첫 번째 레이어 후 70°C로 낮추기
- **속도 낮추기** — PETG는 PLA보다 더 까다로우므로 80% 속도로 시작하기

:::warning 스트링
PETG는 스트링이 발생하기 쉽습니다. 리트랙트 거리 증가 (다이렉트 드라이브에서 0.8–1.5 mm 시도), 리트랙트 속도 증가 및 노즐 온도를 5°C씩 낮추세요.
:::

## 건조

PETG는 PLA보다 더 빠르게 수분을 흡수합니다. 습기 있는 PETG는 다음을 유발합니다:
- 프린팅 중 기포와 쉿 소리
- 다공성 표면의 약한 레이어
- 스트링 증가

스풀이 오랫동안 열려 있었다면 **65°C에서 6–8시간** 건조하세요.

## 보관

항상 실리카겔이 든 밀봉된 봉투 또는 건조 박스에 보관하세요. PETG는 습한 환경에서 며칠 이상 열어두어서는 안 됩니다.
