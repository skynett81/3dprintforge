# AMS 유지보수

AMS (Automatic Material System)는 안정적으로 작동하기 위해 정기적인 유지보수가 필요한 정밀 시스템입니다. 가장 흔한 문제는 더러운 필라멘트 경로와 하우징 내 습기입니다.

## PTFE 튜브

PTFE 튜브는 필라멘트를 AMS에서 프린터로 운반합니다. 가장 먼저 마모되는 부품 중 하나입니다.

### 검사
PTFE 튜브를 다음 항목으로 확인하세요:
- **꺾임 또는 굽힘** — 필라멘트 흐름을 방해합니다
- **연결 부위의 마모** — 입구 주변의 흰색 분말
- **형상 변형** — 특히 CF 재료 사용 시

### PTFE 튜브 교체
1. AMS에서 필라멘트 해제 (언로드 사이클 실행)
2. 연결 부위의 파란색 잠금 링을 튜브 주위로 누르기
3. 튜브 당겨 빼기 (단단히 잡아야 함)
4. 새 튜브를 올바른 길이로 자르기 (원본보다 짧게 자르지 마세요)
5. 멈출 때까지 밀어 넣고 잠그기

:::tip AMS Lite vs. AMS
AMS Lite (A1/A1 Mini)는 풀 AMS (P1S/X1C)보다 더 간단한 PTFE 구성을 가지고 있습니다. 튜브가 더 짧고 교체하기 더 쉽습니다.
:::

## 필라멘트 경로

### 필라멘트 경로 청소
필라멘트는 특히 CF 재료의 경우 필라멘트 경로에 먼지와 잔여물을 남깁니다:

1. 모든 슬롯의 필라멘트 언로드
2. 압축 공기 또는 부드러운 브러시로 느슨한 먼지 불어내기
3. 깨끗한 나일론 또는 PTFE 청소용 필라멘트를 경로를 통해 통과시키기

### 센서
AMS는 필라멘트 위치와 필라멘트 단선을 감지하는 센서를 사용합니다. 센서 창을 청결하게 유지하세요:
- 깨끗한 브러시로 센서 렌즈를 부드럽게 닦기
- 센서에 IPA를 직접 사용하지 마세요

## 습기

AMS는 필라멘트를 습기로부터 보호하지 않습니다. 흡습성 재료 (PA, PETG, TPU)에는 다음을 권장합니다:

### 건조 AMS 대안
- **밀봉된 박스:** 실리카겔이 든 밀봉된 박스에 스풀 배치
- **Bambu Dry Box:** 공식 건조 박스 액세서리
- **외부 피더:** 민감한 재료의 경우 AMS 외부의 필라멘트 피더 사용

### 습기 표시기
AMS 하우징에 습기 표시 카드 (하이그로미터)를 배치하세요. 상대 습도 30% 이상에서 실리카겔 팩을 교체하세요.

## 구동 휠 및 클램프 메커니즘

### 검사
AMS의 구동 휠 (익스트루더 휠)을 다음 항목으로 확인하세요:
- 기어 사이의 필라멘트 잔여물
- 기어 세트의 마모
- 수동 당김 시 불균일한 마찰

### 청소
1. 칫솔 또는 브러시로 구동 휠 기어 사이의 잔여물 제거
2. 압축 공기로 불어내기
3. 오일과 윤활제 피하기 — 당김 수준은 건조 작동에 맞게 보정되어 있습니다

## 유지보수 간격

| 활동 | 간격 |
|-----------|---------|
| PTFE 튜브 시각적 검사 | 매월 |
| 필라멘트 경로 청소 | 매 100시간 |
| 센서 확인 | 매월 |
| 실리카겔 교체 (건조 설정) | 필요 시 (30%+ RH에서) |
| PTFE 튜브 교체 | 눈에 띄는 마모 시 |
