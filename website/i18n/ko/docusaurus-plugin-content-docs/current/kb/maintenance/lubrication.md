# 윤활

움직이는 부품의 올바른 윤활은 마모를 줄이고, 소음을 낮추며, 정밀한 움직임을 보장합니다. Bambu Lab 프린터는 주기적인 윤활이 필요한 선형 모션 시스템을 사용합니다.

## 윤활 유형

| 구성 요소 | 윤활 유형 | 제품 |
|-----------|-------------|---------|
| 선형 로드 (XY) | 가벼운 기계 오일 또는 PTFE 스프레이 | 3-in-1, Super Lube |
| Z축 리드 스크류 | 두꺼운 그리스 | Super Lube 그리스 |
| 선형 레일 | 가벼운 리튬 그리스 | Bambu Lab 그리스 |
| 케이블 체인 조인트 | 없음 (건조) | — |

## 선형 로드

### X 및 Y축
로드는 선형 베어링을 통해 미끄러지는 경화강 샤프트입니다:

```
간격: 200–300시간마다 또는 삐걱거리는 소리 시
양: 매우 소량 — 로드 포인트당 한 방울로 충분
방법:
1. 프린터 전원 끄기
2. 캐리지를 끝까지 수동으로 이동
3. 로드 중간에 가벼운 오일 1방울 바르기
4. 캐리지를 천천히 앞뒤로 10회 이동
5. 보풀 없는 종이로 과도한 오일 닦기
```

:::warning 과도한 윤활 금지
오일이 너무 많으면 먼지를 끌어당겨 연마성 페이스트가 만들어집니다. 최소량만 사용하고 항상 과잉분을 닦아내세요.
:::

### Z축 (수직)
Z축은 그리스가 필요한 리드 스크류를 사용합니다 (오일 아님):

```
간격: 200시간마다
방법:
1. 프린터 전원 끄기
2. 리드 스크류를 따라 얇은 그리스 층 바르기
3. Z축을 수동으로 (또는 유지보수 메뉴를 통해) 위아래로 이동
4. 그리스가 자동으로 분산됩니다
```

## 선형 레일

Bambu Lab P1S 및 X1C는 Y축에 선형 레일 (MGN12)을 사용합니다:

```
간격: 300–500시간마다
방법:
1. 바늘이나 이쑤시개로 주입 구멍에서 약간의 그리스 제거
2. 주사기와 얇은 캐뉼라로 새 그리스 주입
3. 그리스를 분산시키기 위해 축을 앞뒤로 이동
```

Bambu Lab은 시스템에 맞게 보정된 공식 윤활제 (Bambu Lubricant)를 판매합니다.

## 모델별 윤활 유지보수

### X1C / P1S
- Y축: 선형 레일 — Bambu 그리스
- X축: 탄소 로드 — 가벼운 오일
- Z축: 이중 리드 스크류 — Bambu 그리스

### A1 / A1 Mini
- 모든 축: 강철 로드 — 가벼운 오일
- Z축: 단일 리드 스크류 — Bambu 그리스

## 윤활이 필요한 징후

- **움직임 시 삐걱거리거나 긁히는 소리**
- **수직 벽에 보이는 진동 패턴** (VFA)
- **다른 원인 없이 부정확한 치수**
- **모션 시스템에서 증가한 소음**

## 윤활 간격

| 활동 | 간격 |
|-----------|---------|
| XY 로드 오일 | 200–300시간마다 |
| Z 스핀들 그리스 | 200시간마다 |
| 선형 레일 그리스 (X1C/P1S) | 300–500시간마다 |
| 전체 유지보수 사이클 | 반년 (또는 500시간) |

간격을 자동으로 추적하려면 대시보드의 유지보수 모듈을 사용하세요.
