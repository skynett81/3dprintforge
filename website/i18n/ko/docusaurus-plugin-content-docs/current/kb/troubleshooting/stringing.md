# 스트링

스트링 (또는 "우징")은 노즐이 압출 없이 이동하는 동안 물체의 분리된 부분 사이에 형성되는 얇은 플라스틱 실입니다. 프린트에 "거미줄" 같은 외관을 만들어냅니다.

## 스트링의 원인

1. **노즐 온도가 너무 높음** — 뜨거운 플라스틱이 액체 상태로 흘러내림
2. **불량한 리트랙트 설정** — 필라멘트가 충분히 빠르게 당겨지지 않음
3. **습기 있는 필라멘트** — 수분이 증기와 추가적인 흐름을 유발함
4. **속도가 너무 낮음** — 노즐이 이동 위치에 오래 머무름

## 진단

**습기 있는 필라멘트?** 프린팅 중 딸깍거리거나 터지는 소리가 들리시나요? 그렇다면 필라멘트에 습기가 있는 것입니다 — 다른 설정을 조정하기 전에 먼저 건조하세요.

**온도가 너무 높음?** "일시 정지" 순간에 노즐에서 떨어지는 것이 보이나요? 온도를 5–10°C 낮추세요.

## 해결책

### 1. 필라멘트 건조

습기 있는 필라멘트는 조정으로 해결되지 않는 스트링의 가장 흔한 원인입니다:

| 재료 | 건조 온도 | 시간 |
|-----------|----------------|-----|
| PLA | 45–55°C | 4–6시간 |
| PETG | 60–65°C | 6–8시간 |
| TPU | 55–60°C | 6–8시간 |
| PA | 75–85°C | 8–12시간 |

### 2. 노즐 온도 낮추기

5°C씩 낮추기 시작하세요:
- PLA: 210–215°C 시도 (220°C에서 낮춤)
- PETG: 235–240°C 시도 (245°C에서 낮춤)

:::warning 온도가 너무 낮으면 레이어 융합이 불량해짐
온도를 조심스럽게 낮추세요. 너무 낮은 온도는 불량한 레이어 융합, 약한 프린트 및 압출 문제를 유발합니다.
:::

### 3. 리트랙트 설정 조정

리트랙트는 이동 중에 떨어짐을 방지하기 위해 필라멘트를 노즐 안으로 당겨줍니다:

```
Bambu Studio → 필라멘트 → 리트랙트:
- 리트랙트 거리: 0.4–1.0 mm (다이렉트 드라이브)
- 리트랙트 속도: 30–45 mm/s
```

:::tip Bambu Lab 프린터는 다이렉트 드라이브를 사용합니다
모든 Bambu Lab 프린터 (X1C, P1S, A1)는 다이렉트 드라이브 익스트루더를 사용합니다. 다이렉트 드라이브는 보덴 시스템보다 **더 짧은** 리트랙트 거리가 필요합니다 (일반적으로 0.5–1.5 mm vs. 3–7 mm).
:::

### 4. 이동 속도 높이기

포인트 사이의 빠른 이동은 노즐이 떨어질 시간을 줄입니다:
- "Travel speed"를 200–300 mm/s로 높이기
- Bambu Lab 프린터는 이를 잘 처리합니다

### 5. "Avoid Crossing Perimeters" 활성화

스트링이 보이게 될 열린 영역을 노즐이 가로지르지 않도록 하는 슬라이서 설정:
```
Bambu Studio → 품질 → Avoid crossing perimeters
```

### 6. 속도 낮추기 (TPU의 경우)

TPU의 경우 해결책은 다른 재료와 반대입니다:
- 프린트 속도를 20–35 mm/s로 낮추기
- TPU는 탄성이 있어 너무 높은 속도에서 압축되어 "후 흐름"이 발생합니다

## 조정 후

표준 스트링 테스트 모델 (예: MakerWorld의 "torture tower")로 테스트하세요. 한 번에 하나의 변수만 조정하고 변화를 관찰하세요.

:::note 완벽한 제거는 드뭅니다
대부분의 재료에서 약간의 스트링은 정상입니다. 완전히 제거하기보다는 허용 수준으로 줄이는 데 집중하세요. PETG는 항상 PLA보다 약간 더 많은 스트링이 발생합니다.
:::
