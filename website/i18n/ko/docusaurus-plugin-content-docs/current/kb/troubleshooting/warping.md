# 워핑

워핑은 프린팅 중 또는 후에 프린트의 모서리나 가장자리가 플레이트에서 들뜨는 현상입니다. 재료의 열 수축으로 인해 발생합니다.

## 워핑이란?

플라스틱이 냉각되면 수축합니다. 상부 레이어가 하부 레이어보다 뜨거우며 — 이로 인해 모서리를 위로 당기고 프린트를 구부리는 장력이 생깁니다. 온도 차이가 클수록 워핑이 심해집니다.

## 워핑에 취약한 재료

| 재료 | 워핑 위험 | 인클로저 필요 |
|-----------|-------------|-----------------|
| PLA | 낮음 | 아니오 |
| PETG | 낮음–보통 | 아니오 |
| ABS | 높음 | 예 |
| ASA | 높음 | 예 |
| PA/Nylon | 매우 높음 | 예 |
| PC | 매우 높음 | 예 |
| TPU | 낮음 | 아니오 |

## 해결책

### 1. 인클로저 (챔버) 사용

ABS, ASA, PA 및 PC에 가장 중요한 대응책:
- 최상의 결과를 위해 챔버 온도를 40–55°C로 유지
- X1C 및 P1S: "닫힘" 모드에서 챔버 팬 활성화
- A1/P1P: 열을 유지하기 위해 덮개 캡 사용

### 2. 브림 사용

브림은 플레이트에 프린트를 고정하는 넓은 단일 레이어 테두리입니다:

```
Bambu Studio:
1. 슬라이서에서 프린트 선택
2. Support → Brim으로 이동
3. 너비를 5–10 mm로 설정 (워핑이 심할수록 더 넓게)
4. Type: Outer Brim Only (권장)
```

:::tip 브림 너비 가이드
- PLA (거의 필요 없음): 3–5 mm
- PETG: 4–6 mm
- ABS/ASA: 6–10 mm
- PA/Nylon: 8–15 mm
:::

### 3. 베드 온도 높이기

베드 온도를 높이면 레이어 간 온도 차이가 줄어듭니다:
- ABS: 105–110°C 시도
- PA: 85–95°C
- PETG: 80–85°C

### 4. 파트 쿨링 감소

워핑 경향이 있는 재료에는 파트 쿨링을 낮추거나 비활성화하세요:
- ABS/ASA: 0–20% 파트 쿨링
- PA: 0–30% 파트 쿨링

### 5. 외풍과 찬 공기 피하기

프린터를 다음으로부터 멀리하세요:
- 창문과 외부 문
- 에어컨과 선풍기
- 방의 외풍

P1P 및 A1: 중요한 프린트 중 골판지로 개구부를 막으세요.

### 6. 드래프트 쉴드

드래프트 쉴드는 열을 유지하는 물체 주변의 얇은 벽입니다:

```
Bambu Studio:
1. Support → Draft Shield로 이동
2. 활성화하고 거리 설정 (3–5 mm)
```

높고 날씬한 물체에 특히 유용합니다.

### 7. 모델 디자인 조치

자체 모델 설계 시:
- 큰 평평한 바닥 피하기 (모서리에 챔퍼/라운딩 추가)
- 큰 평평한 부분을 작은 섹션으로 분할
- 슬라이서 또는 CAD에서 모서리에 "마우스 이어" — 작은 원 — 사용

## 냉각 후 워핑

가끔 프린트가 잘 보이다가 플레이트에서 제거한 후 워핑이 발생합니다:
- 플레이트와 프린트가 **완전히 냉각될** (40°C 이하) 때까지 항상 기다리고 나서 제거하세요
- ABS: 더 느린 냉각을 위해 닫힌 챔버 내에서 냉각하세요
- 뜨거운 프린트를 차가운 표면에 놓지 마세요
