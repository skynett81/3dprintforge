# 노즐 유지보수

노즐은 프린터에서 가장 중요한 구성 요소 중 하나입니다. 적절한 유지보수는 수명을 연장하고 우수한 프린트 결과를 보장합니다.

## 노즐 유형

| 노즐 유형 | 재료 | 예상 수명 | 최대 온도 |
|----------|-----------|-------------------|----------|
| 황동 (표준) | PLA, PETG, ABS, TPU | 200–500시간 | 300°C |
| 경화강 | CF/GF 포함 모든 재료 | 300–600시간 | 300°C |
| HS01 (Bambu) | CF/GF 포함 모든 재료 | 500–1000시간 | 300°C |

:::danger CF/GF에 황동 노즐 절대 사용 금지
탄소 섬유 및 유리 섬유 충전 필라멘트는 황동 노즐을 몇 시간 내에 마모시킵니다. CF/GF 재료를 프린팅하기 전에 경화강으로 교체하세요.
:::

## 청소

### 간단한 청소 (스풀 교체 사이)
1. 노즐을 200–220°C로 가열
2. 깨끗해질 때까지 필라멘트를 수동으로 통과시키기
3. 필라멘트를 빠르게 당겨 빼기 ("콜드 풀" — 아래 참조)

### IPA 청소
고착된 잔여물:
1. 노즐을 200°C로 가열
2. 노즐 끝에 IPA 1–2 방울 떨어뜨리기 (조심스럽게!)
3. 증기가 잔여물을 녹이도록 두기
4. 새 필라멘트를 통과시키기

:::warning 뜨거운 노즐에 IPA 주의
IPA는 83°C에서 끓으며 뜨거운 노즐에서 강하게 증발합니다. 소량을 사용하고 증기 흡입을 피하세요.
:::

## 콜드 풀 (Cold Pull)

콜드 풀은 노즐에서 오염물과 탄화 잔여물을 제거하는 가장 효과적인 방법입니다.

**단계별:**
1. 노즐을 200–220°C로 가열
2. 나일론 필라멘트 (또는 노즐에 있는 것)를 수동으로 밀어 넣기
3. 나일론이 노즐에 1–2분간 포화되도록 두기
4. 온도를 80–90°C로 낮추기 (나일론의 경우)
5. 목표 온도로 냉각될 때까지 기다리기
6. 한 번의 동작으로 빠르고 확실하게 필라멘트 당겨 빼기
7. 끝 부분 확인: 노즐 내부 형태를 가져야 함 — 깨끗하고 잔여물 없음
8. 필라멘트가 깨끗하고 하얗게 나올 때까지 3–5회 반복

:::tip 콜드 풀용 나일론
나일론은 오염물에 잘 달라붙기 때문에 콜드 풀에 최상의 결과를 줍니다. 흰색 나일론은 풀이 깨끗한지 확인하기 쉽습니다.
:::

## 노즐 교체

### 노즐 교체가 필요한 징후
- 울퉁불퉁한 하부 표면 및 불량한 치수 정확도
- 청소 후에도 지속적인 압출 문제
- 노즐 구멍의 눈에 띄는 마모 또는 변형
- 노즐이 예상 수명을 지남

### 절차 (P1S/X1C)
1. 노즐을 200°C로 가열
2. 익스트루더 모터 브레이크 해제 (필라멘트 해제)
3. 렌치를 사용하여 노즐 풀기 (반시계 방향)
4. 뜨거운 상태에서 노즐 교체 — **공구를 댄 채로 노즐이 냉각되도록 두지 마세요**
5. 원하는 값으로 조이기 (과도하게 조이지 마세요)
6. 교체 후 보정 실행

:::warning 항상 뜨거울 때 교체하세요
차가운 노즐의 조임 토크가 가열 시 부품을 파손시킬 수 있습니다. 항상 노즐이 뜨거울 때 (200°C) 교체하고 조이세요.
:::

## 유지보수 간격

| 활동 | 간격 |
|-----------|---------|
| 청소 (콜드 풀) | 50시간 후 또는 재료 교체 시 |
| 시각적 검사 | 매주 |
| 노즐 교체 (황동) | 200–500시간 |
| 노즐 교체 (경화강) | 300–600시간 |
