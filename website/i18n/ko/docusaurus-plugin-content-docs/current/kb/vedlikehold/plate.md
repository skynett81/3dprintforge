# 플레이트 유지보수

빌드 플레이트는 우수한 접착력과 성공적인 프린트에 필수적입니다. 올바른 청소와 취급으로 수명을 크게 연장할 수 있습니다.

## 청소

### 프린트 사이 (IPA)
IPA (이소프로필알코올)는 기름기와 지문을 제거하는 표준 방법입니다:

```
1. 플레이트가 40°C 이하로 냉각될 때까지 기다리기
2. 보풀 없는 페이퍼 타월에 IPA 적시기 (직접 스프레이하지 마세요)
3. 원형 동작으로 표면 닦기
4. 30–60초 건조
5. 그 후 표면을 만지지 마세요 (손가락의 기름기 잔여물)
```

:::tip 지문 방지
손가락의 기름기 잔여물은 불량 접착의 가장 흔한 원인입니다. 항상 플레이트의 가장자리를 잡거나 일회용 장갑을 사용하세요.
:::

### 딥 클리닝 (물과 비누)
매월 또는 접착제 사용 후:

1. 따뜻한 물 (30–50°C)과 순한 세제
2. 부드러운 천이나 스펀지로 부드럽게 문지르기 (스크럽 면 사용 금지)
3. 흐르는 물에 충분히 헹구기
4. 보풀 없는 페이퍼 타월로 닦기
5. 완전히 건조되도록 두거나 마지막 마감으로 IPA로 닦기

### 접착제 사용 후
접착제 잔여물은 프린팅 후 **반드시** 세척해야 합니다:
1. 플레이트를 미지근한 물에 2–3분간 담그기
2. 접착제가 물에 쉽게 녹습니다
3. 위에 설명된 대로 헹구고 건조하기

## 손상 방지

### 프린트 올바르게 제거하기
- **냉각 기다리기** — 대부분의 재료는 30–40°C에서 자연스럽게 분리됩니다
- **플레이트 구부리기** — 스프링 강철 플레이트가 쉽게 구부러집니다; 가장자리를 눌러 느슨하게 하기
- **날카로운 물체 절대 사용 금지** — 칼과 스크래퍼는 PEI 코팅을 긁고 손상시킵니다

:::danger 금속 스크래퍼 사용 금지
금속 스크래퍼는 PEI 코팅을 긁어 플레이트 수명을 크게 줄입니다. 구부리기 기법을 사용하거나 필요한 경우 플라스틱 스크래퍼를 사용하세요.
:::

### 온도
- **차가운 플레이트를 뜨거운 베드에 놓지 마세요** — 열 충격에 주의하세요
- **뜨거운 플레이트에 차가운 IPA를 스프레이하지 마세요** — 같은 이유

## 플레이트를 교체해야 할 때는?

다음의 경우 플레이트를 교체하세요:
- PEI 코팅이 눈에 띄게 손상됨 (균열, 박리, 깊은 긁힘)
- 철저한 청소 및 딥 클리닝 후에도 프린트가 더 이상 접착되지 않음
- 표면이 변형되거나 불균일함
- 중간 섹션이 매끄럽고 닳아서 평평해짐 (Engineering Plate)

### 일반적인 수명

| 플레이트 | 일반 사용 | 집중 사용 |
|-------|------------|--------------|
| Cool Plate | 200–500회 프린트 | 100–200회 프린트 |
| Engineering Plate | 300–700회 프린트 | 200–400회 프린트 |
| High Temp Plate | 200–400회 프린트 | 100–200회 프린트 |
| Textured PEI | 300–600회 프린트 | 200–300회 프린트 |

## 보관

미사용 플레이트 보관:
- 원래 보호재 또는 평평한 폴더에
- 직사광선 피하기 (UV가 PEI를 열화시킴)
- 먼지가 많은 환경 피하기 — 먼지가 텍스처에 쌓여 접착력을 감소시킵니다
