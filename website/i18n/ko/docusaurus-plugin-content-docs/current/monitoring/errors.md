# 오류 로그

오류 로그는 프린터의 모든 오류 및 HMS (Health, Maintenance, Safety) 경고를 수집합니다. 3DPrintForge는 Bambu Lab 프린터를 위한 269개 이상의 HMS 코드로 이루어진 내장 데이터베이스를 보유하고 있습니다.

이동: **https://localhost:3443/#errors**

## HMS 코드

Bambu Lab 프린터는 문제가 발생하면 MQTT를 통해 HMS 코드를 전송합니다. 3DPrintForge가 자동으로 이를 읽을 수 있는 오류 메시지로 변환합니다:

| 코드 | 예시 | 분류 |
|---|---|---|
| `0700 0100 0001 0001` | Nozzle heatbreak clogged | 노즐/익스트루더 |
| `0700 0200 0002 0001` | AMS filament stuck | AMS |
| `0700 0300 0003 0001` | Bed leveling failed | 빌드 플레이트 |
| `0700 0500 0001 0001` | MC disconnect | 전자기기 |

전체 목록은 X1C, X1C Combo, X1E, P1S, P1S Combo, P1P, P2S, P2S Combo, A1, A1 Combo, A1 mini, H2S, H2D 및 H2C의 알려진 코드 269개 이상을 모두 포함합니다.

## 심각도

오류는 네 가지 수준으로 분류됩니다:

| 수준 | 색상 | 설명 |
|---|---|---|
| **위험** | 빨간색 | 즉각적인 조치 필요 — 프린트 중단 |
| **높음** | 주황색 | 빠른 처리 필요 — 프린트는 계속될 수 있음 |
| **중간** | 노란색 | 검사 필요 — 즉각적인 위험 없음 |
| **정보** | 파란색 | 정보 메시지, 조치 불필요 |

## 검색 및 필터링

오류 로그 상단의 툴바를 사용합니다:

1. **자유 텍스트 검색** — 오류 메시지, HMS 코드 또는 프린터 설명으로 검색
2. **프린터** — 특정 프린터의 오류만 표시
3. **분류** — AMS / 노즐 / 플레이트 / 전자기기 / 캘리브레이션 / 기타
4. **심각도** — 전체 / 위험 / 높음 / 중간 / 정보
5. **날짜** — 날짜 범위로 필터링
6. **미확인** — 확인되지 않은 오류만 표시

**필터 초기화**를 클릭하면 모든 오류를 볼 수 있습니다.

## Wiki 링크

각 HMS 코드에 대해 다음 정보가 포함된 Bambu Lab Wiki 링크가 표시됩니다:

- 전체 오류 설명
- 가능한 원인
- 단계별 문제 해결 가이드
- 공식 Bambu Lab 권장 사항

오류 항목에서 **Wiki 열기**를 클릭하면 새 탭에서 관련 Wiki 페이지가 열립니다.

:::tip 로컬 복사
3DPrintForge는 오프라인 사용을 위해 Wiki 콘텐츠를 로컬에 캐싱합니다. 콘텐츠는 매주 자동으로 업데이트됩니다.
:::

## 오류 확인

확인은 오류를 삭제하지 않고 처리됨으로 표시합니다:

1. 목록에서 오류 클릭
2. **확인** (체크 아이콘) 클릭
3. 수행한 조치에 대한 선택적 메모 입력
4. 오류가 체크 표시로 표시되고 「확인됨」 목록으로 이동

### 대량 확인

1. 체크박스로 여러 오류 선택
2. **선택한 항목 확인** 클릭
3. 선택한 모든 오류가 동시에 확인됩니다

## 통계

오류 로그 상단에 표시되는 정보:

- 최근 30일간 총 오류 수
- 미확인 오류 수
- 가장 자주 발생하는 HMS 코드
- 오류가 가장 많은 프린터

## 내보내기

1. **내보내기** (다운로드 아이콘) 클릭
2. 형식 선택: **CSV** 또는 **JSON**
3. 현재 필터가 내보내기에 적용됨 — 원하는 필터를 먼저 설정
4. 파일이 자동으로 다운로드됩니다

## 새 오류 알림

새 HMS 오류에 대한 알림을 활성화합니다:

1. **설정 → 알림**으로 이동합니다
2. **새 HMS 오류** 체크
3. 알림을 받을 최소 심각도 선택 (권장: **높음** 이상)
4. 알림 채널 선택

채널 설정은 [알림](../features/notifications)을 참조하세요.
