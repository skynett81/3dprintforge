# 유지보수

유지보수 모듈은 Bambu Lab 프린터의 모든 유지보수를 계획하고 추적하는 데 도움을 줍니다 — 노즐 교체부터 레일 윤활까지.

이동: **https://localhost:3443/#maintenance**

## 유지보수 계획

3DPrintForge는 모든 Bambu Lab 프린터 모델에 대해 미리 구성된 유지보수 간격을 제공합니다:

| 작업 | 간격 (기본값) | 모델 |
|---|---|---|
| 노즐 청소 | 200시간마다 | 전체 |
| 노즐 교체 (황동) | 500시간마다 | 전체 |
| 노즐 교체 (강화 강철) | 2000시간마다 | 전체 |
| X축 윤활 | 300시간마다 | X1C, P1S |
| Z축 윤활 | 300시간마다 | 전체 |
| AMS 기어 청소 | 200시간마다 | AMS |
| 챔버 청소 | 500시간마다 | X1C |
| PTFE 튜브 교체 | 필요 시 / 1000시간 | 전체 |
| 캘리브레이션 (전체) | 월별 | 전체 |

모든 간격은 프린터별로 조정할 수 있습니다.

## 노즐 교체 로그

1. **유지보수 → 노즐**로 이동합니다
2. **노즐 교체 기록** 클릭
3. 다음을 입력합니다:
   - **날짜** — 자동으로 오늘로 설정
   - **노즐 재료** — 황동 / 강화 강철 / 구리 / 루비 팁
   - **노즐 직경** — 0.2 / 0.4 / 0.6 / 0.8 mm
   - **브랜드/모델** — 선택 사항
   - **가격** — 비용 로그용
   - **교체 시 시간** — 프린트 타이머에서 자동으로 가져옴
4. **저장** 클릭

로그에는 날짜순으로 정렬된 모든 노즐 기록이 표시됩니다.

:::tip 사전 알림
**X시간 전 알림** 설정 (예: 50시간)으로 다음 권장 교체 시기 전에 충분한 시간을 두고 알림을 받습니다.
:::

## 유지보수 작업 만들기

1. **새 작업** (+ 아이콘) 클릭
2. 다음을 입력합니다:
   - **작업 이름** — 예: 「Y축 윤활」
   - **프린터** — 해당 프린터 선택
   - **간격 유형** — 시간 / 일 / 프린트 수
   - **간격** — 예: 300시간
   - **마지막 수행** — 마지막으로 수행한 시기 입력 (날짜를 과거로 설정)
3. **만들기** 클릭

## 간격 및 알림

활성 작업에 대해 표시되는 정보:
- **초록색** — 다음 유지보수까지 간격의 50% 이상 남음
- **노란색** — 다음 유지보수까지 50% 미만 남음
- **주황색** — 다음 유지보수까지 20% 미만 남음
- **빨간색** — 유지보수 기한 초과

### 알림 구성

1. 작업 클릭 → **편집**
2. **알림** 활성화
3. 기한까지 예를 들어 10% 남았을 때 **알림** 설정
4. 알림 채널 선택 ([알림](../features/notifications) 참조)

## 완료로 표시

1. 목록에서 작업 찾기
2. **완료** (체크 아이콘) 클릭
3. 오늘 날짜/시간부터 간격이 재설정됩니다
4. 로그 항목이 자동으로 생성됩니다

## 비용 로그

모든 유지보수 작업에 비용을 연결할 수 있습니다:

- **부품** — 노즐, PTFE 튜브, 윤활제
- **시간** — 소요 시간 × 시간당 요금
- **외부 서비스** — 유료 수리

비용은 프린터별로 합산되어 통계 개요에 표시됩니다.

## 유지보수 기록

**유지보수 → 기록**으로 이동하면:
- 수행된 모든 유지보수 작업
- 날짜, 시간 및 비용
- 수행자 (다중 사용자 시스템의 경우)
- 댓글 및 메모

회계 목적으로 기록을 CSV로 내보낼 수 있습니다.
