# 마모 예측

마모 예측은 실제 사용량, 필라멘트 유형 및 프린터 동작을 기반으로 주요 구성요소의 예상 수명을 계산합니다 — 사후 대응이 아닌 사전 예방적으로 유지보수를 계획할 수 있습니다.

이동: **https://localhost:3443/#wear**

## 모니터링 대상 구성요소

Bambu Dashboard는 프린터당 8개 구성요소의 마모를 추적합니다:

| 구성요소 | 주요 마모 요인 | 일반적인 수명 |
|---|---|---|
| **노즐 (황동)** | 필라멘트 유형 + 시간 | 300–800시간 |
| **노즐 (강화 강철)** | 시간 + 마모성 재료 | 1500–3000시간 |
| **PTFE 튜브** | 시간 + 고온 | 500–1500시간 |
| **익스트루더 기어** | 시간 + 마모성 재료 | 1000–2000시간 |
| **X축 레일 (CNC)** | 프린트 수 + 속도 | 2000–5000시간 |
| **빌드 플레이트 표면** | 프린트 수 + 온도 | 200–500회 |
| **AMS 기어** | 필라멘트 교체 수 | 5000–15000회 |
| **챔버 팬** | 작동 시간 | 3000–8000시간 |

## 마모 계산

마모는 누적 백분율 (0–100% 마모)로 계산됩니다:

```
마모 % = (실제 사용량 / 예상 수명) × 100
        × 재료 배수
        × 속도 배수
```

**재료 배수:**
- PLA, PETG: 1.0× (일반 마모)
- ABS, ASA: 1.1× (약간 더 공격적)
- PA, PC: 1.2× (PTFE 및 노즐에 더 강함)
- CF/GF 복합 재료: 2.0–3.0× (매우 마모성)

:::warning 탄소 섬유
탄소 섬유 강화 필라멘트 (CF-PLA, CF-PA 등)는 황동 노즐을 극도로 빠르게 마모시킵니다. 강화 강철 노즐을 사용하고 2–3배 더 빠른 마모를 예상하세요.
:::

## 수명 계산

각 구성요소에 대해 표시되는 정보:

- **현재 마모도** — 사용 백분율
- **예상 남은 수명** — 시간 또는 프린트 수
- **예상 만료 날짜** — 최근 30일 평균 사용량 기반
- **신뢰 구간** — 예측의 불확실성 범위

구성요소를 클릭하면 시간 경과에 따른 마모 누적의 상세 그래프를 볼 수 있습니다.

## 알림

구성요소별 자동 알림을 구성합니다:

1. **마모 → 설정**으로 이동합니다
2. 각 구성요소에 대해 **알림 임계값** 설정 (권장: 75% 및 90%)
3. 알림 채널 선택 ([알림](../features/notifications) 참조)

**알림 메시지 예시:**
> ⚠️ My X1C의 노즐 (황동)이 78% 마모되었습니다. 예상 수명: 약 45시간. 권장 사항: 노즐 교체를 계획하세요.

## 유지보수 비용

마모 모듈이 비용 로그와 통합됩니다:

- **구성요소당 비용** — 예비 부품 가격
- **총 교체 비용** — 한계에 가까운 모든 구성요소의 합계
- **향후 6개월 예측** — 예상 유지보수 비용

**마모 → 가격**에서 구성요소 가격을 입력합니다:

1. **가격 설정** 클릭
2. 각 구성요소의 단위당 가격 입력
3. 가격이 비용 예측에 사용되며 프린터 모델별로 다를 수 있습니다

## 마모 카운터 초기화

유지보수 후 해당 구성요소의 카운터를 초기화합니다:

1. **마모 → [구성요소 이름]**으로 이동합니다
2. **교체됨으로 표시** 클릭
3. 다음을 입력합니다:
   - 교체 날짜
   - 비용 (선택 사항)
   - 메모 (선택 사항)
4. 마모 카운터가 초기화되고 재계산됩니다

초기화 내역은 유지보수 기록에 표시됩니다.

:::tip 보정
마모 예측을 실제 경험 데이터와 비교하고 **마모 → 수명 구성**에서 수명 파라미터를 조정하여 실제 사용에 맞게 계산을 최적화하세요.
:::
