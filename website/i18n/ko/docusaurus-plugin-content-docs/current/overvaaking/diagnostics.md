# 진단

진단 페이지는 프린터의 상태, 성능 및 시간 경과에 따른 상태에 대한 심층적인 개요를 제공합니다.

이동: **https://localhost:3443/#diagnostics**

## 상태 점수

각 프린터는 다음을 기반으로 0–100의 **상태 점수**를 계산합니다:

| 요소 | 가중치 | 설명 |
|---|---|---|
| 성공률 (30일) | 30% | 최근 30일간 성공한 프린트 비율 |
| 구성요소 마모 | 25% | 중요 부품의 평균 마모도 |
| HMS 오류 (30일) | 20% | 오류 수 및 심각도 |
| 캘리브레이션 상태 | 15% | 마지막 캘리브레이션 이후 경과 시간 |
| 온도 안정성 | 10% | 프린팅 중 목표 온도로부터의 편차 |

**점수 해석:**
- 🟢 80–100 — 우수한 상태
- 🟡 60–79 — 양호하지만 일부 검사 필요
- 🟠 40–59 — 성능 저하, 유지보수 권장
- 🔴 0–39 — 위험, 즉시 유지보수 필요

:::tip 기록
상태 그래프를 클릭하면 시간 경과에 따른 점수 변화를 볼 수 있습니다. 큰 하락은 특정 이벤트를 나타낼 수 있습니다.
:::

## 텔레메트리 그래프

텔레메트리 페이지는 모든 센서 값에 대한 대화형 그래프를 표시합니다:

### 사용 가능한 데이터셋

- **노즐 온도** — 실제 vs 목표
- **베드 온도** — 실제 vs 목표
- **챔버 온도** — 기기 내부 주변 온도
- **익스트루더 모터** — 전류 소비 및 온도
- **팬 속도** — 툴헤드, 챔버, AMS
- **압력** (X1C) — AMS 챔버 압력
- **가속도** — 진동 데이터 (ADXL345)

### 그래프 탐색

1. **기간** 선택: 최근 1시간 / 24시간 / 7일 / 30일 / 사용자 정의
2. 드롭다운에서 **프린터** 선택
3. 표시할 **데이터셋** 선택 (다중 선택 지원)
4. 스크롤로 타임라인 확대
5. 클릭 및 드래그로 이동
6. 더블 클릭으로 줌 초기화

### 텔레메트리 데이터 내보내기

1. 그래프에서 **내보내기** 클릭
2. 형식 선택: **CSV**, **JSON** 또는 **PNG** (이미지)
3. 선택한 기간 및 데이터셋이 내보내집니다

## 베드 메시

베드 메시 시각화는 빌드 플레이트의 평탄도 캘리브레이션을 표시합니다:

1. **진단 → 베드 메시**로 이동합니다
2. 프린터 선택
3. 최신 메시가 3D 표면과 히트맵으로 표시됩니다:
   - **파란색** — 중심보다 낮음 (오목)
   - **초록색** — 대략적으로 평탄
   - **빨간색** — 중심보다 높음 (볼록)
4. 포인트 위에 마우스를 올리면 mm 단위의 정확한 편차를 볼 수 있습니다

### UI에서 베드 메시 스캔

1. **지금 스캔** 클릭 (프린터가 대기 상태여야 함)
2. 대화 상자에서 확인 — 프린터가 자동으로 캘리브레이션 시작
3. 스캔 완료까지 대기 (약 3–5분)
4. 새 메시가 자동으로 표시됩니다

:::warning 먼저 예열
정확한 캘리브레이션을 위해 베드 메시는 베드를 예열한 상태 (PLA의 경우 50–60°C)에서 스캔해야 합니다.
:::

## 구성요소 마모

자세한 문서는 [마모 예측](./wearprediction)을 참조하세요.

진단 페이지에는 요약 보기가 표시됩니다:
- 구성요소별 백분율 점수
- 다음 권장 유지보수
- **세부 정보** 클릭으로 전체 마모 분석 확인
