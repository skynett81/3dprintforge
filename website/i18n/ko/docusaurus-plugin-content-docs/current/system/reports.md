# 보고서

3DPrintForge는 통계 및 활동 요약이 포함된 자동 이메일 보고서를 보낼 수 있습니다 — 주간, 월간 또는 둘 다.

이동: **https://localhost:3443/#settings** → **시스템 → 보고서**

## 사전 요구사항

보고서를 사용하려면 이메일 알림이 구성되어 있어야 합니다. 보고서를 활성화하기 전에 **설정 → 알림 → 이메일**에서 SMTP를 설정하세요. [알림](../features/notifications)을 참조하세요.

## 자동 보고서 활성화

1. **설정 → 보고서**로 이동합니다
2. **주간 보고서** 및/또는 **월간 보고서** 활성화
3. **전송 시간** 선택:
   - 주간: 요일 및 시간
   - 월간: 월의 날짜 (예: 1일 / 마지막 금요일)
4. **수신자 이메일** 입력 (여러 명인 경우 쉼표로 구분)
5. **저장** 클릭

형식을 확인하려면 테스트 보고서를 보냅니다: **지금 테스트 보고서 보내기** 클릭.

## 주간 보고서 내용

주간 보고서는 최근 7일을 다룹니다:

### 요약
- 총 프린트 수
- 성공 / 실패 / 취소된 수
- 성공률 및 전주 대비 변화
- 가장 활성화된 프린터

### 활동
- 일별 프린트 수 (미니 그래프)
- 총 프린트 시간
- 총 필라멘트 소비 (그램 및 비용)

### 필라멘트
- 재료 및 공급업체별 소비
- 스풀별 예상 잔량 (20% 미만 스풀 강조 표시)

### 유지보수
- 이번 주에 수행된 유지보수 작업
- 기한이 지난 유지보수 작업 (빨간 경고)
- 다음 주에 기한이 되는 작업

### HMS 오류
- 프린터별 이번 주 HMS 오류 수
- 미확인 오류 (주의 필요)

## 월간 보고서 내용

월간 보고서는 최근 30일을 다루며 주간 보고서의 모든 내용에 추가로:

### 트렌드
- 전월 대비 비교 (%)
- 활동 지도 (월간 히트맵 미니어처)
- 월간 성공률 추이

### 비용
- 총 필라멘트 비용
- 총 전력 비용 (전력 측정이 구성된 경우)
- 총 마모 비용
- 전체 유지보수 비용

### 마모 및 상태
- 프린터별 상태 점수 (전월 대비 변화 포함)
- 교체 시기에 가까운 구성요소

### 통계 하이라이트
- 가장 긴 성공적인 프린트
- 가장 많이 사용된 필라멘트 유형
- 가장 활발한 프린터

## 보고서 사용자 정의

1. **설정 → 보고서 → 사용자 정의**로 이동합니다
2. 포함할 섹션을 체크/해제합니다
3. **프린터 필터** 선택: 모든 프린터 또는 선택된 것만
4. **로고 표시** 선택: 헤더에 3DPrintForge 로고 표시 또는 끄기
5. **저장** 클릭

## 보고서 아카이브

보내진 모든 보고서가 저장되어 다시 열 수 있습니다:

1. **설정 → 보고서 → 아카이브**로 이동합니다
2. 목록에서 보고서 선택 (날짜순 정렬)
3. **열기** 클릭으로 HTML 버전 보기
4. **PDF 다운로드** 클릭으로 보고서 다운로드

보고서는 **90일** 후 자동으로 삭제됩니다 (구성 가능).
