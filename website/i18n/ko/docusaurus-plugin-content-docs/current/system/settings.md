# 설정

Bambu Dashboard의 모든 설정은 명확한 분류가 있는 한 페이지에 모여 있습니다. 각 분류에 있는 내용의 개요입니다.

이동: **https://localhost:3443/#settings**

## 프린터

등록된 프린터 관리:

| 설정 | 설명 |
|---|---|
| 프린터 추가 | 일련번호와 액세스 코드로 새 프린터 등록 |
| 프린터 이름 | 사용자 정의 표시 이름 |
| 프린터 모델 | X1C / X1C Combo / X1E / P1S / P1S Combo / P1P / P2S / P2S Combo / A1 / A1 Combo / A1 mini / H2S / H2D / H2C |
| MQTT 연결 | Bambu Cloud MQTT 또는 로컬 MQTT |
| 액세스 코드 | Bambu Lab 앱의 LAN 액세스 코드 |
| IP 주소 | 로컬 (LAN) 모드용 |
| 카메라 설정 | 활성화/비활성화, 해상도 |

첫 번째 프린터 단계별 설정은 [시작하기](../kom-i-gang/oppsett)를 참조하세요.

## 알림

전체 문서는 [알림](../funksjoner/notifications)을 참조하세요.

빠른 개요:
- 알림 채널 활성화/비활성화 (Telegram, Discord, 이메일 등)
- 채널별 이벤트 필터
- 방해 금지 시간 (알림 없는 시간대)
- 채널별 테스트 버튼

## 테마

전체 문서는 [테마](./themes)를 참조하세요.

- 밝게 / 어둡게 / 자동 모드
- 6가지 색상 팔레트
- 사용자 정의 강조 색상
- 모서리 둥글기 및 밀도

## OBS 오버레이

OBS 오버레이 구성:

| 설정 | 설명 |
|---|---|
| 기본 테마 | dark / light / minimal |
| 기본 위치 | 오버레이 모서리 |
| 기본 배율 | 크기 조정 (0.5–2.0) |
| QR 코드 표시 | 오버레이에 대시보드 QR 코드 표시 |

전체 URL 구문 및 설정은 [OBS 오버레이](../funksjoner/obs-overlay)를 참조하세요.

## 에너지 및 전력

| 설정 | 설명 |
|---|---|
| Tibber API 토큰 | Tibber 스팟 가격 접근 |
| Nordpool 가격 구역 | 가격 지역 선택 |
| 배전망 요금 (원/kWh) | 배전망 요금 |
| 프린터 전력 (W) | 프린터 모델별 전력 소비 구성 |

## Home Assistant

| 설정 | 설명 |
|---|---|
| MQTT 브로커 | IP, 포트, 사용자 이름, 비밀번호 |
| Discovery 접두사 | 기본값: `homeassistant` |
| Discovery 활성화 | HA에 장치 게시 |

## 웹훅

전역 웹훅 설정:

| 설정 | 설명 |
|---|---|
| 웹훅 URL | 이벤트 수신자 URL |
| 비밀 키 | HMAC-SHA256 서명 |
| 이벤트 필터 | 전송할 이벤트 |
| 재시도 횟수 | 실패 시 시도 횟수 (기본값: 3) |
| 타임아웃 | 요청 포기 전 초 (기본값: 10) |

## 대기열 설정

| 설정 | 설명 |
|---|---|
| 자동 디스패치 | 활성화/비활성화 |
| 디스패치 전략 | 첫 번째 사용 가능 / 최소 사용 / 라운드 로빈 |
| 확인 필요 | 전송 전 수동 승인 |
| 분산 시작 | 대기열의 프린터 간 지연 |

## 보안

| 설정 | 설명 |
|---|---|
| 세션 기간 | 자동 로그아웃 전 시간/일 |
| 2FA 강제 | 모든 사용자에게 2FA 요구 |
| IP 화이트리스트 | 특정 IP 주소로 접근 제한 |
| HTTPS 인증서 | 사용자 정의 인증서 업로드 |

## 시스템

| 설정 | 설명 |
|---|---|
| 서버 포트 | 기본값: 3443 |
| 로그 형식 | JSON / 텍스트 |
| 로그 수준 | Error / Warn / Info / Debug |
| 데이터베이스 정리 | 오래된 기록 자동 삭제 |
| 업데이트 | 새 버전 확인 |
