# 테마

Bambu Dashboard는 취향과 사용 환경에 맞게 외관을 사용자 정의할 수 있는 유연한 테마 시스템을 제공합니다.

이동: **https://localhost:3443/#settings** → **테마**

## 색상 모드

세 가지 모드 중에서 선택합니다:

| 모드 | 설명 |
|---|---|
| **밝게** | 밝은 배경, 어두운 텍스트 — 밝은 환경에 적합 |
| **어둡게** | 어두운 배경, 밝은 텍스트 — 기본값으로 모니터링에 권장 |
| **자동** | 운영 체제 설정 따름 (OS 어둡게/밝게) |

테마 설정 상단에서 모드를 변경하거나 탐색 바의 빠른 전환 버튼 (달/태양 아이콘)을 사용합니다.

## 색상 팔레트

6가지 미리 설정된 색상 팔레트를 사용할 수 있습니다:

| 팔레트 | 기본 색상 | 스타일 |
|---|---|---|
| **Bambu** | 초록 (#00C853) | 기본값, Bambu Lab에서 영감 |
| **파란 밤** | 파란 (#2196F3) | 차분하고 전문적 |
| **일몰** | 주황 (#FF6D00) | 따뜻하고 활기찬 |
| **보라** | 보라 (#9C27B0) | 창의적이고 독특한 |
| **빨강** | 빨간 (#F44336) | 높은 대비, 눈에 띄는 |
| **모노크롬** | 회색 (#607D8B) | 중립적이고 미니멀리스트 |

팔레트를 클릭하면 즉시 미리보기 및 활성화됩니다.

## 사용자 정의 강조 색상

완전히 자신만의 색상을 강조 색상으로 사용합니다:

1. 팔레트 선택기 아래에서 **사용자 정의 색상** 클릭
2. 색상 선택기를 사용하거나 HEX 코드 입력 (예: `#FF5722`)
3. 미리보기가 실시간으로 업데이트됩니다
4. **적용** 클릭으로 활성화

:::tip 대비
강조 색상이 배경과 충분한 대비를 갖는지 확인하세요. 색상이 가독성 문제를 일으킬 수 있는 경우 시스템이 경고합니다 (WCAG AA 표준).
:::

## 모서리 둥글기

버튼, 카드 및 요소의 둥글기를 조정합니다:

| 설정 | 설명 |
|---|---|
| **날카롭게** | 둥글기 없음 (직사각형 스타일) |
| **작게** | 미묘한 둥글기 (4px) |
| **중간** | 표준 둥글기 (8px) |
| **크게** | 뚜렷한 둥글기 (16px) |
| **알약** | 최대 둥글기 (50px) |

슬라이더를 드래그하여 0–50px 사이에서 수동으로 조정합니다.

## 밀도

인터페이스의 밀도를 조정합니다:

| 설정 | 설명 |
|---|---|
| **넉넉하게** | 요소 간 더 많은 공간 |
| **표준** | 균형 잡힌 기본 설정 |
| **컴팩트** | 더 촘촘한 배치 — 화면에 더 많은 정보 |

컴팩트 모드는 1080p 미만의 화면이나 키오스크 보기에 권장됩니다.

## 타이포그래피

글꼴 선택:

- **시스템** — 운영 체제의 기본 글꼴 사용 (빠른 로딩)
- **Inter** — 명확하고 현대적 (기본 선택)
- **JetBrains Mono** — 모노스페이스, 데이터 값에 적합
- **Nunito** — 부드럽고 더 둥근 스타일

## 애니메이션

애니메이션을 끄거나 조정합니다:

- **전체** — 모든 전환 및 애니메이션 활성화 (기본값)
- **축소** — 필수 애니메이션만 (OS 설정 존중)
- **끄기** — 최대 성능을 위해 애니메이션 없음

:::tip 키오스크 모드
키오스크 보기에는 최적의 성능과 원거리 가독성을 위해 **컴팩트** + **어둡게** + **애니메이션 축소**를 활성화하세요. [키오스크 모드](./kiosk)를 참조하세요.
:::

## 테마 설정 내보내기 및 가져오기

테마를 다른 사람과 공유합니다:

1. **테마 내보내기** 클릭 — `.json` 파일 다운로드
2. 파일을 다른 Bambu Dashboard 사용자와 공유
3. **테마 가져오기** → 파일 선택으로 가져옵니다
