# 프린트 프로필

프린트 프로필은 프린트와 프린터 전반에 걸쳐 재사용할 수 있는 저장된 프린트 설정 세트입니다. 다양한 목적을 위한 프로필을 정의하여 시간을 절약하고 일관된 품질을 보장하세요.

이동: **https://localhost:3443/#profiles**

## 프로필 만들기

1. **도구 → 프린트 프로필**로 이동합니다
2. **새 프로필** (+ 아이콘) 클릭
3. 다음을 입력합니다:
   - **프로필 이름** — 설명적인 이름, 예: 「PLA - 빠른 생산」
   - **재료** — 목록에서 선택 (PLA / PETG / ABS / PA / PC / TPU 등)
   - **프린터 모델** — X1C / X1C Combo / X1E / P1S / P1S Combo / P1P / P2S / P2S Combo / A1 / A1 Combo / A1 mini / H2S / H2D / H2C / 전체
   - **설명** — 선택적 텍스트

4. 설정 입력 (아래 섹션 참조)
5. **프로필 저장** 클릭

## 프로필의 설정

### 온도
| 항목 | 예시 |
|---|---|
| 노즐 온도 | 220°C |
| 베드 온도 | 60°C |
| 챔버 온도 (X1C) | 35°C |

### 속도
| 항목 | 예시 |
|---|---|
| 속도 설정 | 표준 |
| 최대 속도 (mm/s) | 200 |
| 가속도 | 5000 mm/s² |

### 품질
| 항목 | 예시 |
|---|---|
| 레이어 두께 | 0.2 mm |
| 인필 비율 | 15% |
| 인필 패턴 | Grid |
| 지지 재료 | 자동 |

### AMS 및 색상
| 항목 | 설명 |
|---|---|
| 퍼지 볼륨 | 색상 교체 시 세척 양 |
| 선호 슬롯 | 선호할 AMS 슬롯 |

### 고급
| 항목 | 설명 |
|---|---|
| 건조 모드 | 습한 재료를 위한 AMS 건조 활성화 |
| 냉각 시간 | 냉각을 위한 레이어 간 일시 정지 |
| 팬 속도 | 냉각 팬 속도 (백분율) |

## 프로필 편집

1. 목록에서 프로필 클릭
2. **편집** (연필 아이콘) 클릭
3. 변경 사항 적용
4. **저장** (덮어쓰기) 또는 **새로 저장** (복사본 생성) 클릭

:::tip 버전 관리
변경 사항을 실험하면서 작동하는 프로필을 유지하려면 「새로 저장」을 사용하세요.
:::

## 프로필 사용

### 파일 라이브러리에서

1. 라이브러리에서 파일 선택
2. **프린터로 전송** 클릭
3. 드롭다운에서 **프로필** 선택
4. 프로필의 설정이 사용됩니다

### 출력 대기열에서

1. 새 대기열 작업 만들기
2. 설정에서 **프로필** 선택
3. 프로필이 대기열 작업에 연결됩니다

## 프로필 가져오기 및 내보내기

### 내보내기
1. 하나 이상의 프로필 선택
2. **내보내기** 클릭
3. 형식 선택: **JSON** (다른 대시보드로 가져오기용) 또는 **PDF** (인쇄/문서화용)

### 가져오기
1. **프로필 가져오기** 클릭
2. 다른 3DPrintForge에서 내보낸 `.json` 파일 선택
3. 같은 이름의 기존 프로필은 덮어쓰거나 둘 다 유지할 수 있습니다

## 프로필 공유

커뮤니티 필라멘트 모듈을 통해 다른 사람과 프로필을 공유합니다 ([커뮤니티 필라멘트](../integrations/community) 참조) 또는 직접 JSON 내보내기를 통해 공유합니다.

## 기본 프로필

재료별 기본 프로필을 설정합니다:

1. 프로필 선택
2. **[재료]의 기본값으로 설정** 클릭
3. 해당 재료의 파일을 전송할 때 기본 프로필이 자동으로 선택됩니다
