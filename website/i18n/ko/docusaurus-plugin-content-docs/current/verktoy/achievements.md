# 업적

업적(achievements)은 프린팅 여정에서 마일스톤과 특별한 순간을 보상하는 게임화 요소입니다. 업적을 수집하고 다음 잠금 해제를 향한 진행 상황을 확인하세요.

이동: **https://localhost:3443/#achievements**

## 희귀도 등급

업적은 네 가지 희귀도 등급으로 분류됩니다:

| 등급 | 색상 | 설명 |
|---|---|---|
| **일반** | 회색 | 간단한 마일스톤, 달성하기 쉬움 |
| **비범** | 초록 | 어느 정도의 노력이나 시간 필요 |
| **희귀** | 파란 | 시간이 지남에 따라 전념된 노력 필요 |
| **전설** | 금색 | 탁월한 업적 |

## 업적 예시

### 프린팅 마일스톤 (일반 / 비범)
| 업적 | 요건 |
|---|---|
| 첫 번째 프린트 | 첫 번째 프린트 완료 |
| 꼬박 하루 | 총 24시간 이상 프린팅 |
| 높은 성공률 | 10개 연속 성공 프린트 |
| 필라멘트 수집가 | 10가지 다른 필라멘트 유형 등록 |
| 멀티컬러 | 멀티컬러 프린트 완료 |

### 볼륨 업적 (비범 / 희귀)
| 업적 | 요건 |
|---|---|
| 1킬로그램 | 총 1kg 필라멘트 사용 |
| 10kg | 총 10kg 필라멘트 사용 |
| 100개 프린트 | 100개 성공 프린트 |
| 500시간 | 500시간 누적 프린트 시간 |
| 야간 교대 | 20시간 이상 지속되는 프린트 완료 |

### 유지보수 및 관리 (비범 / 희귀)
| 업적 | 요건 |
|---|---|
| 성실한 사람 | 유지보수 작업 기록 |
| 프린터 관리자 | 유지보수 작업 10개 기록 |
| 낭비 없음 | 재료 효율 90% 이상의 프린트 제작 |
| 마스터 노즐리스트 | 5번 노즐 교체 (문서화됨) |

### 전설 업적
| 업적 | 요건 |
|---|---|
| 지칠 줄 모르는 | 1000개 성공 프린트 |
| 필라멘트 타이탄 | 총 50kg 필라멘트 소비 |
| 무결점 주 | 실패한 프린트 없이 7일 |
| 프린트 사서 | 파일 라이브러리에 100가지 다른 모델 |

## 업적 보기

업적 페이지에 표시되는 내용:

- **잠금 해제됨** — 달성한 업적 (날짜 포함)
- **거의 달성** — 거의 달성 직전인 업적 (진행률 표시줄)
- **잠김** — 아직 달성하지 못한 모든 업적

**등급**, **분류** 또는 **상태** (잠금 해제됨 / 진행 중 / 잠김)로 필터링합니다.

## 진행률 표시줄

카운터가 있는 업적의 경우 진행률 표시줄이 표시됩니다:

```
1킬로그램 — 1kg 필라멘트
[████████░░] 847g / 1000g (84.7%)
```

## 알림

새 업적 달성 시 자동으로 알림을 받습니다:
- **브라우저 팝업** 업적 이름과 그래픽 포함
- 선택 사항: Telegram / Discord를 통한 알림 (**설정 → 알림 → 업적**에서 구성)

## 다중 사용자 지원

여러 사용자가 있는 시스템에서는 각 사용자가 자신의 업적 프로필을 갖습니다. **리더보드**는 다음을 기준으로 순위를 표시합니다:

- 총 잠금 해제된 업적 수
- 총 프린트 수
- 총 프린트 시간

:::tip 비공개 모드
**설정 → 업적 → 리더보드에서 숨기기**에서 리더보드를 끄면 프로필을 비공개로 유지할 수 있습니다.
:::
