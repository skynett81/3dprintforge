# 베드 메시

베드 메시 도구는 좋은 접착력과 균일한 첫 번째 레이어에 필수적인 빌드 플레이트의 평탄도에 대한 시각적 표현을 제공합니다.

이동: **https://localhost:3443/#bedmesh**

## 베드 메시란?

Bambu Lab 프린터는 프로브로 빌드 플레이트 표면을 스캔하여 높이 편차의 맵 (메시)을 만듭니다. 프린터 펌웨어가 프린팅 중 자동으로 편차를 보정합니다. Bambu Dashboard는 이 맵을 시각화합니다.

## 시각화

### 3D 표면

베드 메시 맵이 대화형 3D 표면으로 표시됩니다:

- 마우스를 사용하여 보기 회전
- 스크롤로 확대/축소
- **위에서 보기** 클릭으로 조감 시점
- **측면 보기** 클릭으로 프로파일 확인

색상 스케일은 평균 높이로부터의 편차를 보여줍니다:
- **파란색** — 중심보다 낮음 (오목)
- **초록색** — 대략적으로 평탄 (< 0.1mm 편차)
- **노란색** — 보통 편차 (0.1–0.2mm)
- **빨간색** — 높은 편차 (> 0.2mm)

### 히트맵

**히트맵** 클릭으로 메시 맵의 평면 2D 보기 — 대부분의 사람에게 더 읽기 쉽습니다.

히트맵에 표시되는 내용:
- 각 측정 포인트의 정확한 편차 값 (mm)
- 표시된 문제 포인트 (편차 > 0.3mm)
- 측정의 치수 (행 × 열 수)

## UI에서 베드 메시 스캔

:::warning 요구사항
스캔은 프린터가 대기 상태이고 베드 온도가 안정화된 상태여야 합니다. 스캔 전에 베드를 원하는 온도로 예열하세요.
:::

1. **베드 메시**로 이동합니다
2. 드롭다운에서 프린터 선택
3. **지금 스캔** 클릭
4. 스캔 베드 온도 선택:
   - **냉각** (실온) — 빠르지만 덜 정확
   - **예열** (PLA 50–60°C, PETG 70–90°C) — 권장
5. 대화 상자에서 확인 — 프린터가 자동으로 프로브 시퀀스 시작
6. 스캔 완료까지 대기 (메시 크기에 따라 3–8분)
7. 새 메시 맵이 자동으로 표시됩니다

## 캘리브레이션 가이드

스캔 후 시스템이 구체적인 권장 사항을 제공합니다:

| 발견 사항 | 권장 사항 |
|---|---|
| 모든 곳에서 편차 < 0.1mm | 우수 — 조치 불필요 |
| 편차 0.1–0.2mm | 양호 — 펌웨어가 보정 처리 |
| 모서리에서 편차 > 0.2mm | 수동으로 베드 스프링 조정 (가능한 경우) |
| 편차 > 0.3mm | 베드가 손상되거나 잘못 설치되었을 수 있음 |
| 중앙이 모서리보다 높음 | 열 팽창 — 예열된 베드에서 정상 |

:::tip 과거 비교
**이전과 비교** 클릭으로 시간 경과에 따라 메시 맵이 변했는지 확인 — 플레이트가 점진적으로 구부러지는 것을 감지하는 데 유용합니다.
:::

## 메시 기록

모든 메시 스캔이 타임스탬프와 함께 저장됩니다:

1. 베드 메시 사이드 패널에서 **기록** 클릭
2. 두 스캔을 선택하여 비교 (차이 맵 표시)
3. 더 이상 필요하지 않은 이전 스캔 삭제

## 내보내기

메시 데이터를 다음으로 내보냅니다:
- **PNG** — 히트맵 이미지 (문서화용)
- **CSV** — 포인트당 X, Y 및 높이 편차가 있는 원시 데이터
