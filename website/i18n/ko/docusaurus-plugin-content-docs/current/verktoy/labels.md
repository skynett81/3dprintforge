# 라벨

라벨 도구는 필라멘트 스풀을 위한 전문적인 라벨을 생성합니다 — QR 코드, 열전사 프린터용 스풀 라벨 및 시각적 식별을 위한 컬러 카드.

이동: **https://localhost:3443/#labels**

## QR 코드

대시보드의 필라멘트 정보로 연결되는 QR 코드를 생성합니다:

1. **라벨 → QR 코드**로 이동합니다
2. QR 코드를 생성할 스풀 선택
3. QR 코드가 자동으로 생성되어 미리보기에 표시됩니다
4. **PNG 다운로드** 또는 **인쇄** 클릭

QR 코드에는 대시보드의 필라멘트 프로필 URL이 포함됩니다. 모바일로 스캔하면 스풀 정보를 빠르게 확인할 수 있습니다.

### 일괄 생성

1. **모두 선택** 클릭하거나 개별 스풀 체크
2. **모든 QR 코드 생성** 클릭
3. 스풀당 하나의 PNG가 있는 ZIP으로 다운로드하거나 한 번에 모두 인쇄

## 스풀 라벨

전체 스풀 정보가 포함된 열전사 프린터용 전문 라벨:

### 라벨 내용 (기본값)

- 스풀 색상 (채워진 색상 블록)
- 재료 이름 (큰 글자)
- 공급업체
- 색상 HEX 코드
- 온도 권장 사항 (노즐 및 베드)
- QR 코드
- 바코드 (선택 사항)

### 열전사 프린터용 ZPL

Zebra, Brother 및 Dymo 프린터용 ZPL 코드 (Zebra Programming Language) 생성:

1. **라벨 → 열전사 인쇄**로 이동합니다
2. 라벨 크기 선택: **25×54mm** / **36×89mm** / **62×100mm**
3. 스풀 선택
4. **ZPL 생성** 클릭
5. 다음을 통해 프린터로 ZPL 코드 전송:
   - **직접 인쇄** (USB 연결)
   - **ZPL 복사** 후 터미널 명령으로 전송
   - **.zpl 파일 다운로드**

:::tip 프린터 설정
자동 인쇄를 위해 **설정 → 라벨 프린터**에서 IP 주소와 포트 (기본값: RAW TCP의 경우 9100)와 함께 인쇄 스테이션을 구성하세요.
:::

### PDF 라벨

일반 프린터용, 올바른 치수로 PDF를 생성합니다:

1. 템플릿에서 라벨 크기 선택
2. **PDF 생성** 클릭
3. 접착식 용지 (Avery 또는 유사품)에 인쇄

## 컬러 카드

컬러 카드는 모든 스풀을 시각적으로 표시하는 컴팩트한 그리드입니다:

1. **라벨 → 컬러 카드**로 이동합니다
2. 포함할 스풀 선택 (모든 활성 스풀 또는 수동으로 선택)
3. 카드 형식 선택: **A4** (4×8), **A3** (6×10), **Letter**
4. **PDF 생성** 클릭

각 필드에 표시되는 내용:
- 실제 색상이 있는 색상 블록
- 재료 이름 및 색상 HEX
- 재료 번호 (빠른 참조용)

프린터 스테이션 근처에 코팅하여 걸어두기에 이상적입니다.

## 공유 색상 팔레트

선택한 색상을 공유 팔레트로 내보냅니다:

1. **라벨 → 색상 팔레트**로 이동합니다
2. 팔레트에 포함할 스풀 선택
3. **팔레트 공유** 클릭
4. 링크 복사 — 다른 사람이 자신의 대시보드로 팔레트를 가져올 수 있습니다
5. 팔레트가 HEX 코드와 함께 표시되며 **Adobe Swatch** (`.ase`) 또는 **Procreate** (`.swatches`)로 내보낼 수 있습니다
