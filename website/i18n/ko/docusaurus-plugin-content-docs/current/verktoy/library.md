# 파일 라이브러리

파일 라이브러리는 모든 3D 모델과 G코드 파일을 저장하고 관리하는 중앙 장소입니다 — 자동 G코드 분석 및 MakerWorld와 Printables 통합.

이동: **https://localhost:3443/#library**

## 모델 업로드

### 단일 업로드

1. **파일 라이브러리**로 이동합니다
2. **업로드** 클릭하거나 업로드 영역에 파일 드래그
3. 지원 형식: `.3mf`, `.gcode`, `.bgcode`, `.stl`, `.obj`
4. 업로드 후 파일이 자동으로 분석됩니다

:::info 저장 폴더
파일은 **설정 → 파일 라이브러리 → 저장 폴더**에서 구성된 폴더에 저장됩니다. 기본값: `./data/library/`
:::

### 일괄 업로드

전체 폴더를 드래그 앤 드롭하여 지원되는 모든 파일을 한 번에 업로드합니다. 파일이 백그라운드에서 처리되고 모두 준비되면 알림이 표시됩니다.

## G코드 분석

업로드 후 `.gcode` 및 `.bgcode` 파일이 자동으로 분석됩니다:

| 메트릭 | 설명 |
|---|---|
| 예상 프린트 시간 | G코드 명령에서 계산된 시간 |
| 필라멘트 소비 | 재료/색상별 그램 및 미터 |
| 레이어 수 | 총 레이어 수 |
| 레이어 두께 | 기록된 레이어 두께 |
| 재료 | 감지된 재료 (PLA, PETG 등) |
| 인필 비율 | 메타데이터에서 사용 가능한 경우 |
| 지지 재료 | 예상 지지 무게 |
| 프린터 모델 | 메타데이터의 대상 프린터 |

분석 데이터가 파일 카드에 표시되며 [비용 계산기](../analyse/costestimator)에서 사용됩니다.

## 파일 카드 및 메타데이터

각 파일 카드에 표시되는 내용:
- **파일 이름** 및 형식
- **업로드 날짜**
- **썸네일** (`.3mf`에서 또는 생성됨)
- **분석된 프린트 시간** 및 필라멘트 소비
- **태그** 및 분류
- **연결된 프린트** — 프린트한 횟수

카드를 클릭하면 전체 메타데이터와 기록이 있는 상세 보기가 열립니다.

## 정리

### 태그

쉬운 검색을 위해 태그를 추가합니다:
1. 파일 클릭 → **메타데이터 편집**
2. 태그 입력 (쉼표로 구분): `benchy, 테스트, PLA, 캘리브레이션`
3. 태그 필터로 라이브러리 검색

### 분류

분류로 파일을 정리합니다:
- 사이드 바에서 **새 분류** 클릭
- 분류로 파일 드래그
- 분류는 중첩 가능 (하위 분류 지원)

## MakerWorld 연동

1. **설정 → 통합 → MakerWorld**로 이동합니다
2. Bambu Lab 계정으로 로그인
3. 라이브러리로 돌아가서: 파일 클릭 → **MakerWorld에 연결**
4. MakerWorld에서 모델을 검색하고 올바른 항목 선택
5. 메타데이터 (디자이너, 라이선스, 평점)가 MakerWorld에서 가져와집니다

연결에는 디자이너 이름과 원본 URL이 파일 카드에 표시됩니다.

## Printables 연동

1. **설정 → 통합 → Printables**로 이동합니다
2. Printables API 키 붙여넣기
3. MakerWorld와 동일한 방식으로 파일을 Printables 모델에 연결

## 프린터로 전송

파일 라이브러리에서 직접 프린터로 전송할 수 있습니다:

1. 파일 클릭 → **프린터로 전송**
2. 대상 프린터 선택
3. AMS 슬롯 선택 (멀티컬러 프린트의 경우)
4. **프린트 시작** 또는 **대기열에 추가** 클릭

:::warning 직접 전송
직접 전송은 Bambu Studio에서 확인 없이 즉시 프린트를 시작합니다. 프린터가 준비되어 있는지 확인하세요.
:::
