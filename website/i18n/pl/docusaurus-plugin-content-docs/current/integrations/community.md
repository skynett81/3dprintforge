# Filamenty społeczności

Filamenty społeczności to wspólna baza danych profili filamentów wnoszona przez użytkowników 3DPrintForge z całego świata. Znajdź najlepsze ustawienia dla swojego filamentu i podziel się własnym doświadczeniem.

Przejdź do: **https://localhost:3443/#community**

## Przeglądanie profili społeczności

### Wyszukiwanie

1. Przejdź do **Filamenty społeczności**
2. Użyj pola wyszukiwania, aby znaleźć według:
   - Materiału (PLA, PETG, ABS, PA, TPU itp.)
   - Dostawcy (PolyMaker, Bambu Lab, Prusament, Sunlu itp.)
   - Nazwy koloru lub kodu hex koloru
   - Właściwości (przezroczysty, matowy, jedwabisty, brokatowy itp.)
3. Filtruj według **Oceny** (tylko najwyżej oceniane)
4. Filtruj według **Modelu drukarki**, aby zobaczyć profile zoptymalizowane dla twojej drukarki

### Karty profili

Każda karta profilu pokazuje:

| Pole | Przykład |
|---|---|
| Nazwa filamentu | PolyTerra PLA – Matte Forest Green |
| Materiał | PLA |
| Dostawca | PolyMaker |
| Autor | nazwa użytkownika |
| Ocena | ⭐⭐⭐⭐½ (4,6 / 5) |
| Liczba testów | 23 |
| Zalecane temperatury | Dysza: 215°C / Stół: 55°C |
| Zalecana prędkość | Standardowy |

Kliknij kartę, aby zobaczyć pełny profil ze wszystkimi parametrami i komentarzami użytkowników.

## Importowanie profilu

1. Otwórz profil ze społeczności
2. Kliknij **Importuj do magazynu**
3. Wybierz, czy chcesz:
   - **Połączyć z istniejącą szpulą** — profil jest połączony z zarejestrowaną szpulą
   - **Utwórz nową szpulę** — tworzy nowy wpis szpuli z danymi profilu
4. W razie potrzeby dostosuj temperaturę lub inne ustawienia
5. Kliknij **Importuj**

Profil jest teraz dostępny w twoim magazynie filamentów.

## Wnoszenie własnych profili

### Eksport i udostępnianie

1. Przejdź do **Magazyn filamentów**
2. Znajdź szpulę, którą chcesz udostępnić
3. Kliknij **Udostępnij społeczności**
4. Wypełnij:
   - **Nazwa dostawcy** — dokładna nazwa dostawcy
   - **Nazwa produktu** — dokładna nazwa produktu z opakowania
   - **Nazwa koloru** — oficjalna nazwa koloru i kod hex
   - **Komentarz** — twoje doświadczenia, wskazówki
   - **Model drukarki** — na której drukarce testowałeś
5. Kliknij **Prześlij**

Profil jest pod recenzją i zwykle jest publikowany w ciągu 24 godzin.

### Wytyczne dotyczące wkładów

:::warning Wymagania jakościowe
- Podawaj tylko rzeczywiste, przetestowane ustawienia — nie spekulatywne wartości
- Używaj dokładnej nazwy dostawcy i produktu (sprawdź opakowanie)
- Jeden profil na konkretny produkt i model drukarki
- Unikaj zduplikowanych profili — sprawdź, czy profil już istnieje
:::

## Ocenianie profili

Pomóż społeczności oceniając profile, które przetestowałeś:

1. Otwórz profil społeczności
2. Kliknij **Oceń**
3. Podaj liczbę gwiazdek (1–5) i krótki komentarz
4. Kliknij **Wyślij**

Profile z wysoką oceną pojawiają się na szczycie wyników wyszukiwania.

## Użytek offline

Baza społeczności jest automatycznie synchronizowana lokalnie (aktualizowana codziennie). Wszystkie profile są dostępne bez połączenia z internetem po pierwszej synchronizacji.

Pobierz bazę ręcznie: **Filamenty społeczności → Ustawienia → Synchronizuj teraz**
