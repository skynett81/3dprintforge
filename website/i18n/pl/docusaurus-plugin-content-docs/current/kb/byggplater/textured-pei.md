# Textured PEI Plate

Textured PEI Plate (nie mylić z Engineering Plate) ma grubszą teksturę i jest szczególnie odpowiednia dla TPU i innych elastycznych materiałów, a także dla różnych popularnych materiałów.

## Właściwości

- **Powierzchnia:** Gruboziarnista tekstura PEI — daje wyraźną teksturowaną powierzchnię spodnią
- **Przyczepność:** Doskonały mechaniczny chwyt, szczególnie dla TPU
- **Zdejmowanie:** Bardzo łatwe po ostudzeniu — wyginaj płytę a wydruki odpadają
- **Maks. temperatura stołu:** 100 °C

## Najlepsza dla

- **TPU** — to jest najlepsza płyta dla elastycznych materiałów
- PLA i PLA+ — dobra przyczepność, ładna teksturowana powierzchnia spodnia
- PETG — z małą ilością kleju
- PLA-CF — bez kleju

## Kompatybilność

| Materiał | Przydatność | Klej |
|-----------|---------|---------|
| TPU | ★★★★★ | Nie |
| PLA | ★★★★★ | Nie |
| PLA+ | ★★★★★ | Nie |
| PETG | ★★★★ | Trochę (zalecany) |
| PLA-CF | ★★★★ | Nie |
| ABS | ★★★ | Tak |
| PA12 | ★★★ | Tak |

## TPU na Textured PEI

TPU przylega doskonale do Textured PEI bez kleju:
1. Ustaw temperaturę stołu 30–45 °C
2. Dokładnie skalibruj przesunięcie Z — nieco wyżej niż dla PLA
3. Pozwól stołowi ostygnąć do poniżej 30 °C po wydruku
4. Delikatnie wyginaj płytę — TPU odpada bez wysiłku

:::tip Zimne zdejmowanie
TPU doskonale odpada od Textured PEI po ostudzeniu. Jeśli masz problemy z usunięciem wydruku, poczekaj, aż płyta będzie całkowicie zimna (20–25 °C) i wyginaj silniej.
:::

## Konserwacja

Textured PEI jest wytrzymała i łatwa w konserwacji:

```bash
# Między wydrukami:
- Wytrzyj IPA i bezpyłowym papierem

# Miesięcznie (lub przy słabej przyczepności):
- Myj ciepłą wodą z łagodnym detergentem
- Spłucz i dokładnie osusz
- Na koniec wytrzyj IPA
```

## Powierzchnia gotowego wydruku

Gruba tekstura daje wyraźny wzorzysty spód na wszystkich wydrukach. Jest to:
- Pożądane dla prototypów i części funkcjonalnych (lepszy chwyt)
- Niepożądane dla wydruków dekoracyjnych/wizualnych (użyj Cool Plate dla gładkiej powierzchni)
