# Community-filamenter

Community-filamenter är en delad databas med filamentprofiler som bidragits av 3DPrintForge-användare från hela världen. Hitta de bästa inställningarna för ditt filament och dela dina egna erfarenheter.

Gå till: **https://localhost:3443/#community**

## Bläddra bland communityprofilerna

### Sök

1. Gå till **Community-filamenter**
2. Använd sökfältet för att söka efter:
   - Material (PLA, PETG, ABS, PA, TPU, osv.)
   - Leverantör (PolyMaker, Bambu Lab, Prusament, Sunlu, osv.)
   - Färgnamn eller färg-hex
   - Egenskap (transparent, matt, silke, glitter, osv.)
3. Filtrera på **Betyg** (enbart högst rankade)
4. Filtrera på **Skrivarmodell** för att se profiler optimerade för din skrivare

### Profilkort

Varje profilkort visar:

| Fält | Exempel |
|---|---|
| Filamentnamn | PolyTerra PLA – Matte Forest Green |
| Material | PLA |
| Leverantör | PolyMaker |
| Bidragsgivare | användarnamn |
| Betyg | ⭐⭐⭐⭐½ (4.6 / 5) |
| Antal tester | 23 |
| Rekommenderade temperaturer | Munstycke: 215°C / Bädd: 55°C |
| Rekommenderad hastighet | Standard |

Klicka på kortet för att se fullständig profil med alla parametrar och användarkommentarer.

## Importera en profil

1. Öppna en profil från communityn
2. Klicka **Importera till lager**
3. Välj om du vill:
   - **Koppla till befintlig spole** — profilen kopplas till en registrerad spole
   - **Skapa ny spole** — skapar en ny spolelpost med profildata
4. Justera temperatur eller andra inställningar efter behov
5. Klicka **Importera**

Profilen är nu tillgänglig i ditt filamentlager.

## Bidra med egna profiler

### Export och delning

1. Gå till **Filamentlager**
2. Hitta spolen du vill dela
3. Klicka **Dela med communityn**
4. Fyll i:
   - **Leverantörsnamn** — exakt leverantörsnamn
   - **Produktnamn** — exakt produktnamn från förpackningen
   - **Färgnamn** — officiellt färgnamn och färg-hex
   - **Kommentar** — dina erfarenheter, tips och tricks
   - **Skrivarmodell** — vilken skrivare du testade på
5. Klicka **Skicka in**

Profilen granskas och publiceras vanligtvis inom 24 timmar.

### Riktlinjer för bidrag

:::warning Kvalitetskrav
- Uppge endast verkliga, testade inställningar — inte spekulativa värden
- Använd exakt leverantörs- och produktnamn (kontrollera förpackningen)
- En profil per specifikt produkt och skrivarmodell
- Undvik dubblettprofiler — kontrollera om profilen redan finns
:::

## Betygsätta profiler

Hjälp communityn genom att betygsätta profiler du har testat:

1. Öppna en communityprofil
2. Klicka **Ge betyg**
3. Ange antal stjärnor (1–5) och en kort kommentar
4. Klicka **Skicka**

Profiler med högt betyg visas överst i sökresultaten.

## Offlineanvändning

Communitybasen synkroniseras lokalt automatiskt (uppdateras dagligen). Alla profiler är tillgängliga utan internetanslutning efter första synkronisering.

Ladda ner basen manuellt: **Community-filamenter → Inställningar → Synkronisera nu**
