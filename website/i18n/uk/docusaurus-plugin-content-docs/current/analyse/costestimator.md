# Калькулятор витрат

Калькулятор витрат дозволяє оцінити загальну вартість друку перед відправкою на принтер — на основі витрати філаменту, ціни електроенергії та зносу обладнання.

Перейдіть до: **https://localhost:3443/#cost-estimator**

## Завантаження файлу

1. Перейдіть до **Калькулятор витрат**
2. Перетягніть файл у поле завантаження або натисніть **Вибрати файл**
3. Підтримувані формати: `.3mf`, `.gcode`, `.bgcode`
4. Натисніть **Аналізувати**

:::info Аналіз
Система аналізує G-код для вилучення витрати філаменту, орієнтовного часу друку та профілю матеріалу. Зазвичай це займає 2–10 секунд.
:::

## Розрахунок філаменту

Після аналізу відображається:

| Поле | Значення (приклад) |
|---|---|
| Орієнтовний філамент | 47.3 г |
| Матеріал (з файлу) | PLA |
| Ціна за грам | 0.025 грн (зі сховища філаменту) |
| **Вартість філаменту** | **1.18 грн** |

Змініть матеріал у списку для порівняння вартості різних типів філаменту або постачальників.

:::tip Заміна матеріалу
Якщо G-код не містить інформації про матеріал, виберіть матеріал вручну зі списку. Ціна автоматично береться зі сховища філаменту.
:::

## Розрахунок електроенергії

Вартість електроенергії розраховується на основі:

- **Орієнтовний час друку** — з аналізу G-коду
- **Потужність принтера** — налаштована за моделлю принтера (Вт)
- **Ціна електроенергії** — фіксована ціна (грн/кВт·год) або в реальному часі від Tibber/Nordpool

| Поле | Значення (приклад) |
|---|---|
| Орієнтовний час друку | 3 год 22 хв |
| Потужність принтера | 350 Вт (X1C) |
| Орієнтовне споживання | 1.17 кВт·год |
| Ціна електроенергії | 1.85 грн/кВт·год |
| **Вартість електроенергії** | **2.16 грн** |

Активуйте інтеграцію Tibber або Nordpool для використання запланованих погодинних цін на основі бажаного часу початку.

## Знос обладнання

Вартість зносу оцінюється на основі:

- Час друку × вартість за годину для моделі принтера
- Додатковий знос для абразивних матеріалів (CF, GF тощо)

| Поле | Значення (приклад) |
|---|---|
| Час друку | 3 год 22 хв |
| Вартість за годину (знос) | 0.80 грн/год |
| **Вартість зносу** | **2.69 грн** |

Вартість за годину розраховується з цін компонентів та очікуваного строку служби (дивіться [Прогноз зносу](../overvaaking/wearprediction)).

## Загальна сума

| Стаття витрат | Сума |
|---|---|
| Філамент | 1.18 грн |
| Електроенергія | 2.16 грн |
| Знос обладнання | 2.69 грн |
| **Разом** | **6.03 грн** |
| + Надбавка (30 %) | 1.81 грн |
| **Ціна продажу** | **7.84 грн** |

Відрегулюйте надбавку у відсотковому полі для розрахунку рекомендованої ціни продажу клієнту.

## Збереження оцінки

Натисніть **Зберегти оцінку** для прив'язки аналізу до проекту:

1. Виберіть існуючий проект або створіть новий
2. Оцінка зберігається та може бути використана як основа для рахунку-фактури
3. Фактична вартість (після друку) автоматично порівнюється з оцінкою

## Пакетний розрахунок

Завантажте кілька файлів одночасно для розрахунку загальної вартості повного набору:

1. Натисніть **Пакетний режим**
2. Завантажте всі файли `.3mf`/`.gcode`
3. Система розраховує індивідуально та загальну вартість
4. Експортуйте зведення у PDF або CSV
