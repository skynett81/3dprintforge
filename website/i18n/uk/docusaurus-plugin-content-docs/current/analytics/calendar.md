# Календар активності

Календар активності показує візуальний огляд активності вашого принтера протягом усього року — натхненний оглядом внесків GitHub.

Перейдіть до: **https://localhost:3443/#calendar**

## Огляд теплової карти

Календар показує 365 днів (52 тижні) у вигляді сітки кольорових клітин:

- **Сірий** — друки в цей день відсутні
- **Світло-зелений** — 1–2 друки
- **Зелений** — 3–5 друків
- **Темно-зелений** — 6–10 друків
- **Насичений зелений** — 11+ друків

Клітини організовані за днями тижня вертикально (Пн–Нд) та тижнями горизонтально зліва (січень) до правого (грудень).

:::tip Кодування кольором
Ви можете змінити показник теплової карти з **Кількість друків** на **Години** або **Грами філаменту** за допомогою вибору над календарем.
:::

## Вибір року

Натисніть **< Рік >** для навігації між роками:

- Доступні всі роки з зареєстрованою активністю друку
- Поточний рік відображається за замовчуванням
- Майбутні дні сірі (немає даних)

## Детальний вигляд за день

Натисніть на клітину для перегляду деталей відповідного дня:

- **Дата** та день тижня
- **Кількість друків** — успішних та невдалих
- **Загальний використаний філамент** (грами)
- **Загальний час друку**
- **Список друків** — натисніть для відкриття в історії

## Місячний огляд

Під тепловою картою відображається місячний огляд з:
- Загальна кількість друків на місяць у вигляді стовпчастої діаграми
- Найкращий день місяця виділено
- Порівняння з тим самим місяцем минулого року (%)

## Фільтр принтерів

Виберіть принтер зі списку вгорі для перегляду активності лише для одного принтера, або виберіть **Всі** для агрегованого перегляду.

Вигляд кількох принтерів показує кольори в стосах при натисканні **Стос** у виборі вигляду.

## Серії та рекорди

Під календарем відображається:

| Статистика | Опис |
|-----------|------|
| **Найдовша серія** | Найбільша кількість послідовних днів з принаймні одним друком |
| **Поточна серія** | Поточна послідовність активних днів |
| **Найактивніший день** | День з найбільшою кількістю друків загалом |
| **Найактивніший тиждень** | Тиждень з найбільшою кількістю друків |
| **Найактивніший місяць** | Місяць з найбільшою кількістю друків |

## Експорт

Натисніть **Експортувати** для завантаження даних календаря:

- **PNG** — зображення теплової карти (для поширення)
- **CSV** — вихідні дані з одним рядком на день (дата, кількість, грами, години)

PNG-експорт оптимізований для поширення у соціальних мережах із назвою принтера та роком як підзаголовком.
