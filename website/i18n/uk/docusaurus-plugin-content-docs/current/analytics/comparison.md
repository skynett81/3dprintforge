# Порівняння друків

Порівняння друків дозволяє аналізувати два друки поруч — зручно для порівняння налаштувань, матеріалів, принтерів або версій однієї й тієї ж моделі.

Перейдіть до: **https://localhost:3443/#comparison**

## Вибір друків для порівняння

1. Перейдіть до **Порівняння друків**
2. Натисніть **Вибрати друк A** та знайдіть потрібний у історії
3. Натисніть **Вибрати друк B** та знайдіть потрібний у історії
4. Натисніть **Порівняти** для завантаження режиму порівняння

:::tip Швидший доступ
З **Історії** можна натиснути правою кнопкою миші на друк та вибрати **Встановити як друк A** або **Порівняти з...** для переходу одразу в режим порівняння.
:::

## Порівняння показників

Показники відображаються у двох колонках (A та B) з виділенням кращого значення:

| Показник | Опис |
|---|---|
| Успіх | Завершено / Скасовано / Не вдалось |
| Тривалість | Загальний час друку |
| Витрата філаменту | Грами загалом та за кольором |
| Ефективність філаменту | % матеріалу моделі від загального |
| Макс. температура сопла | Найвища зареєстрована температура сопла |
| Макс. температура столу | Найвища зареєстрована температура столу |
| Параметр швидкості | Тихий / Стандартний / Спорт / Турбо |
| Замін AMS | Кількість змін кольору |
| HMS-помилки | Можливі помилки під час друку |
| Принтер | Який принтер використовувався |

Клітини з найкращим значенням відображаються з зеленим фоном.

## Графіки температури

Два графіки температури відображаються поруч (або накладено):

- **Окремий вигляд** — графік A ліворуч, графік B праворуч
- **Накладений вигляд** — обидва в одному графіку різними кольорами

Використовуйте накладений вигляд для порівняння стабільності температури та швидкості нагріву.

## Зображення галереї

Якщо обидва друки мають milestone-скріншоти, вони відображаються у сітці:

| Друк A | Друк B |
|---|---|
| Зображення A 25% | Зображення B 25% |
| Зображення A 50% | Зображення B 50% |
| Зображення A 75% | Зображення B 75% |
| Зображення A 100% | Зображення B 100% |

Натисніть на зображення для відкриття повноекранного перегляду зі слайд-анімацією.

## Порівняння таймлапсу

Якщо обидва друки мають таймлапс, відео відображаються поруч:

- Синхронізоване відтворення — обидва починають та зупиняються одночасно
- Незалежне відтворення — керуйте кожним відео окремо

## Відмінності налаштувань

Система автоматично виділяє відмінності в налаштуваннях друку (отримані з метаданих G-коду):

- Різна висота шару
- Різні шаблони або відсоток заповнення
- Різні налаштування підтримок
- Різні профілі швидкості

Відмінності відображаються з помаранчевим виділенням у таблиці налаштувань.

## Збереження порівняння

1. Натисніть **Зберегти порівняння**
2. Введіть назву порівняння (наприклад, «PLA vs PETG - Benchy»)
3. Порівняння зберігається та доступне через **Історія → Порівняння**

## Експорт

1. Натисніть **Експортувати**
2. Виберіть **PDF** для звіту з усіма показниками та зображеннями
3. Звіт можна прив'язати до проекту для документування вибору матеріалів
