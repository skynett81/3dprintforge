# Аналіз філаменту

Аналіз філаменту дає вам повне уявлення про витрату філаменту — що ви використовуєте, скільки це коштує та де можна заощадити.

Перейдіть до: **https://localhost:3443/#filament-analytics**

## Огляд споживання

Вгорі відображається зведення за вибраний период:

- **Загальне споживання** — грами та метри для всіх матеріалів
- **Орієнтовна вартість** — на основі зареєстрованої ціни за котушку
- **Найбільш використовуваний матеріал** — тип і постачальник
- **Відсоток повторного використання** — частка філаменту в фактичній моделі проти підтримок/очищення

### Споживання за матеріалом

Кругова діаграма та таблиця показують розподіл між матеріалами:

| Колонка | Опис |
|---|---|
| Матеріал | PLA, PETG, ABS, PA тощо |
| Постачальник | Bambu Lab, PolyMaker, Prusament тощо |
| Використано (г) | Загальна вага |
| Метри | Орієнтовна довжина |
| Вартість | Грами × ціна за грам |
| Друки | Кількість друків з цим матеріалом |

Натисніть на рядок для детального перегляду на рівні окремих котушок.

## Норми споживання

Норма споживання показує середнє споживання філаменту за одиницю часу:

- **Грамів за годину** — під час активного друку
- **Грамів за тиждень** — включаючи простої принтера
- **Грамів за друк** — середнє на один друк

Вони використовуються для розрахунку прогнозів майбутніх потреб.

:::tip Планування закупівель
Використовуйте норму споживання для планування запасу котушок. Система автоматично попереджає, коли орієнтовний запас закінчиться протягом 14 днів (налаштовується).
:::

## Прогноз витрат

На основі історичної норми споживання розраховується:

- **Орієнтовне споживання наступні 30 днів** (грами за матеріалом)
- **Орієнтовна вартість наступні 30 днів**
- **Рекомендований запас** (достатньо для 30 / 60 / 90 днів роботи)

Прогноз враховує сезонні коливання, якщо є дані принаймні за один рік.

## Відходи та ефективність

Дивіться [Відстеження відходів](./waste) для повної документації. Аналіз філаменту показує зведення:

- **AMS-очищення** — грами та частка від загального споживання
- **Матеріал підтримок** — грами та частка
- **Фактичний матеріал моделі** — решта частки (% ефективності)
- **Орієнтовна вартість відходів** — скільки коштують відходи

## Журнал котушок

Всі котушки (активні та порожні) зафіксовані:

| Поле | Опис |
|---|---|
| Назва котушки | Назва матеріалу та колір |
| Початкова вага | Зареєстрована вага на початку |
| Залишкова вага | Розрахована залишкова |
| Використано | Грами використано загалом |
| Останнє використання | Дата останнього друку |
| Статус | Активна / Порожня / Збережена |

## Реєстрація цін

Для точного аналізу витрат реєструйте ціни за котушку:

1. Перейдіть до **Сховище філаменту**
2. Натисніть на котушку → **Редагувати**
3. Заповніть **Ціна покупки** та **Вага при покупці**
4. Система автоматично розраховує ціну за грам

Котушки без зареєстрованої ціни використовують **стандартну ціну за грам** (встановлюється в **Налаштування → Філамент → Стандартна ціна**).

## Експорт

1. Натисніть **Експортувати дані філаменту**
2. Виберіть период та формат (CSV / PDF)
3. CSV включає один рядок на друк з грамами, вартістю та матеріалом
