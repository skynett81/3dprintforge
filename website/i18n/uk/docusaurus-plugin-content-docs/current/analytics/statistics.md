# Статистика

Сторінка статистики надає повний огляд активності принтерів з ключовими показниками, трендами та витратою філаменту за будь-який вибраний часовий период.

Перейдіть до: **https://localhost:3443/#statistics**

## Ключові показники

Вгорі сторінки відображаються чотири KPI-картки:

| Показник | Опис |
|---------|------|
| **Відсоток успіху** | Частка успішних друків від загальної кількості |
| **Загальний філамент** | Грами використані за вибраний период |
| **Загальний час друку** | Накопичений час друку |
| **Середній час друку** | Медіанна тривалість на друк |

Кожен показник відображає зміну від попереднього периоду (↑ вгору / ↓ вниз) як відсоткове відхилення.

## Відсоток успіху

Відсоток успіху розраховується за принтером і загалом:

- **Успішний** — друк завершено без переривань
- **Скасовано** — зупинено вручну користувачем
- **Не вдалось** — зупинено Print Guard, HMS-помилкою або апаратним збоєм

Натисніть на діаграму відсотка успіху для перегляду, які друки не вдалися та причини.

:::tip Покращення відсотка успіху
Використовуйте [Аналіз закономірностей помилок](../monitoring/erroranalysis) для виявлення та усунення причин невдалих друків.
:::

## Тренди

Вигляд трендів показує розвиток з часом у вигляді лінійних графіків:

1. Виберіть **Часовий период**: Останні 7 / 30 / 90 / 365 днів
2. Виберіть **Групування**: День / Тиждень / Місяць
3. Виберіть **Показник**: Кількість друків / Години / Грами / Відсоток успіху
4. Натисніть **Порівняти** для накладання двох показників

Графік підтримує масштабування (прокрутка) та панорамування (натиснути та перетягнути).

## Витрата філаменту

Витрата філаменту відображається у вигляді:

- **Стовпчаста діаграма** — витрата за день/тиждень/місяць
- **Кругова діаграма** — розподіл між матеріалами (PLA, PETG, ABS тощо)
- **Таблиця** — детальний список з загальними грамами, метрами та вартістю за матеріалом

### Витрата за принтером

Використовуйте фільтр множинного вибору вгорі для:
- Відображення лише одного принтера
- Порівняння двох принтерів поруч
- Перегляду агрегованого підсумку для всіх принтерів

## Календар активності

Перегляньте компактну теплову карту в стилі GitHub безпосередньо на сторінці статистики (спрощений вигляд), або перейдіть до повного [Календаря активності](./calendar) для більш детального перегляду.

## Експорт

1. Натисніть **Експортувати статистику**
2. Виберіть діапазон дат та показники, які хочете включити
3. Виберіть формат: **CSV** (вихідні дані), **PDF** (звіт) або **JSON**
4. Файл завантажується

CSV-експорт сумісний з Excel та Google Sheets для подальшого аналізу.

## Порівняння з попереднім периодом

Активуйте **Показати попередній период** для накладання графіків з відповідним попереднім периодом:

- Останні 30 днів проти попередніх 30 днів
- Поточний місяць проти попереднього місяця
- Поточний рік проти минулого року

Це дозволяє легко побачити, чи ви друкуєте більше або менше, ніж раніше, і чи покращується відсоток успіху.
