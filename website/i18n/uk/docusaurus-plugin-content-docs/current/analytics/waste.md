# Відстеження відходів

Відстеження відходів дає вам повне уявлення про те, скільки філаменту витрачається під час друку — AMS-очищення, змивання при замінах матеріалів та матеріали підтримок — і скільки це вам коштує.

Перейдіть до: **https://localhost:3443/#waste**

## Категорії відходів

3DPrintForge розрізняє три типи відходів:

| Категорія | Джерело | Типова частка |
|---|---|---|
| **AMS-очищення** | Зміна кольору в AMS під час багатокольорового друку | 5–30 г за заміну |
| **Змивання при заміні матеріалу** | Очищення при переході між різними матеріалами | 10–50 г за заміну |
| **Матеріал підтримок** | Підтримуючі структури, що видаляються після друку | Варіюється |

## Відстеження AMS-очищення

Дані AMS-очищення отримуються безпосередньо з MQTT-телеметрії та аналізу G-коду:

- **Грамів за зміну кольору** — розраховано з блоку очищення G-коду
- **Кількість змін кольору** — підраховано з журналу друку
- **Загальне споживання на очищення** — сума за вибраний период

:::tip Зменшення очищення
Bambu Studio має налаштування для об'єму очищення за кожною комбінацією кольорів. Зменшіть об'єм очищення для пар кольорів з невеликою різницею (наприклад, білий → світло-сірий) для економії філаменту.
:::

## Розрахунок ефективності

Ефективність розраховується як:

```
Ефективність % = (матеріал моделі / загальне споживання) × 100

Загальне споживання = матеріал моделі + очищення + матеріал підтримок
```

**Приклад:**
- Модель: 45 г
- Очищення: 12 г
- Підтримки: 8 г
- Разом: 65 г
- **Ефективність: 69 %**

Ефективність відображається у вигляді трендового графіка з часом, щоб ви могли бачити покращення.

## Вартість відходів

На основі зареєстрованих цін на філамент розраховується:

| Стаття | Розрахунок |
|---|---|
| Вартість очищення | Грами очищення × ціна/грам за кольором |
| Вартість підтримок | Грами підтримок × ціна/грам |
| **Загальна вартість відходів** | Сума наведеного |
| **Вартість на успішний друк** | Вартість відходів / кількість друків |

## Відходи за принтером та матеріалом

Фільтруйте вигляд за:

- **Принтер** — дивіться, який принтер генерує найбільше відходів
- **Матеріал** — дивіться відходи за типом філаменту
- **Период** — день, тиждень, місяць, рік

Таблиця відображає ранжований список з найвищими відходами вгорі, включаючи орієнтовну вартість.

## Поради з оптимізації

Система автоматично генерує пропозиції для зменшення відходів:

- **Зміна порядку кольорів** — якщо перехід A→B очищує більше, ніж B→A, система пропонує змінити порядок
- **Об'єднання шарів зміни кольору** — групує шари з однаковим кольором для мінімізації замін
- **Оптимізація підтримуючих структур** — оцінює зменшення підтримок при зміні орієнтації

:::info Точність
Розрахунки очищення є приблизними з G-коду. Фактичні відходи можуть відрізнятися на 10–20 % через поведінку принтера.
:::

## Експорт та звітність

1. Натисніть **Експортувати дані відходів**
2. Виберіть период та формат (CSV / PDF)
3. Дані відходів можна включити до проектних звітів та рахунків-фактур як статтю витрат

Дивіться також [Аналіз філаменту](./filamentanalytics) для загального огляду споживання.
