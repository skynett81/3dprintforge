# Керування принтером

Панель керування надає повний ручний контроль над принтером безпосередньо з дашборду.

## Керування температурою

### Сопло
- Встановіть цільову температуру від 0 до 350 °C
- Натисніть **Встановити** для надсилання команди
- Показання в реальному часі відображаються з анімованим кільцевим вимірювачем

### Нагрівальний стіл
- Встановіть цільову температуру від 0 до 120 °C
- Автоматичне вимкнення після друку (налаштовується)

### Камера
- Перегляд температури камери (показання в реальному часі)
- **X1E, H2S, H2D, H2C**: Активне керування нагрівом камери через M141 (регульована цільова температура)
- **X1C**: Пасивне корпусування — температура камери відображається, але не може контролюватися безпосередньо
- **P1S**: Пасивне корпусування — відображає температуру, без активного керування нагрівом камери
- **P1P, A1, A1 mini та серія H без chamberHeat**: Немає датчика камери

:::warning Максимальні температури
Не перевищуйте рекомендовані температури для сопла та столу. Для загартованого сталевого сопла (тип HF): макс. 300 °C. Для латунного: макс. 260 °C. Дивіться посібник принтера.
:::

## Профілі швидкості

Контроль швидкості надає чотири попередньо встановлені профілі:

| Профіль | Швидкість | Застосування |
|---------|----------|--------------|
| Тихо | 50% | Зменшення шуму, нічний друк |
| Стандарт | 100% | Нормальне використання |
| Спорт | 124% | Швидший друк |
| Турбо | 166% | Максимальна швидкість (зниження якості) |

Повзунок дозволяє встановити власний відсоток від 50 до 200%.

## Керування вентиляторами

Керуйте швидкістю вентиляторів вручну:

| Вентилятор | Опис | Діапазон |
|-----------|------|---------|
| Part cooling fan | Охолодження об'єкта друку | 0–100% |
| Auxiliary fan | Циркуляція в камері | 0–100% |
| Chamber fan | Активне охолодження камери | 0–100% |

:::tip Рекомендовані налаштування
- **PLA/PETG:** Охолодження деталі 100%, aux 30%
- **ABS/ASA:** Охолодження деталі 0–20%, вентилятор камери вимкнено
- **TPU:** Охолодження деталі 50%, низька швидкість
:::

## Консоль G-code

Надсилайте команди G-code безпосередньо до принтера:

```gcode
; Приклад: Переміщення голівки
G28 ; Припаркувати всі осі
G1 X150 Y150 Z10 F3000 ; Перемістити до центру
M104 S220 ; Встановити температуру сопла
M140 S60  ; Встановити температуру столу
```

:::danger Будьте обережні з G-code
Неправильний G-code може пошкодити принтер. Надсилайте лише команди, які ви розумієте. Уникайте `M600` (заміна філаменту) під час друку.
:::

## Операції з філаментом

З панелі керування ви можете:

- **Завантажити філамент** — нагріти сопло та подати філамент
- **Вивантажити філамент** — нагріти та витягнути філамент
- **Очистити сопло** — запустити цикл очищення

## Макроси

Збережіть і запустіть послідовності команд G-code як макроси:

1. Натисніть **Новий макрос**
2. Дайте макросу назву
3. Введіть послідовність G-code
4. Збережіть і запустіть одним натисканням

Приклад макросу для калібрування столу:
```gcode
G28
M84
M500
```

## Керування друком

Під час активного друку ви можете:

- **Пауза** — призупинити друк після поточного шару
- **Продовжити** — продовжити призупинений друк
- **Стоп** — скасувати друк (незворотно)
- **Аварійна зупинка** — негайна зупинка всіх двигунів
