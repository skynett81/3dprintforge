# Головна панель

Головна панель є центральним центром керування Bambu Dashboard. Вона показує статус вибраного принтера в реальному часі та дозволяє відстежувати, керувати та налаштовувати вигляд на свій розсуд.

Перейдіть до: **https://localhost:3443/**

## Огляд у реальному часі

Коли принтер активний, всі значення постійно оновлюються через MQTT:

- **Температура сопла** — анімований SVG-кільцевий вимірювач з цільовою температурою
- **Температура столу** — аналогічний кільцевий вимірювач для платформи друку
- **Відсоток виконання** — великий індикатор відсотка з залишковим часом
- **Лічильник шарів** — поточний шар / загальна кількість шарів
- **Швидкість** — Тихо / Стандарт / Спорт / Турбо з повзунком

:::tip Оновлення в реальному часі
Усі значення оновлюються безпосередньо з принтера через MQTT без перезавантаження сторінки. Затримка зазвичай менше 1 секунди.
:::

## Відображення 3D-моделі

Якщо принтер надсилає файл `.3mf` з моделлю, відображається інтерактивний 3D-попередній перегляд:

1. Модель завантажується автоматично при початку друку
2. Обертайте модель, перетягуючи мишею
3. Прокручуйте для масштабування
4. Натисніть **Скинути** для повернення до стандартного вигляду

:::info Підтримка
3D-відображення потребує надсилання даних моделі принтером. Не всі завдання друку включають це.
:::

## Статус AMS

Панель AMS показує всі встановлені пристрої AMS зі слотами та філаментом:

- **Колір слота** — візуальне представлення кольору з метаданих Bambu
- **Назва філаменту** — матеріал та бренд
- **Активний слот** — позначений анімацією пульсації під час друку
- **Помилки** — червоний індикатор при помилках AMS (блокування, порожньо, вологість)

Натисніть на слот для перегляду повної інформації про філамент і прив'язки до сховища філаменту.

## Відеопотік камери

Живий відеопотік камери конвертується через ffmpeg (RTSPS → MPEG1):

1. Камера запускається автоматично при відкритті дашборду
2. Натисніть на зображення камери для відкриття на повний екран
3. Використовуйте кнопку **Знімок** для збереження фото
4. Натисніть **Приховати камеру** для звільнення місця

:::warning Продуктивність
Відеопотік камери використовує приблизно 2–5 Мбіт/с. Вимкніть камеру при повільному мережевому підключенні.
:::

## Міні-графіки температури

Під панеллю AMS відображаються міні-графіки (sparklines) за останні 30 хвилин:

- Температура сопла в часі
- Температура столу в часі
- Температура камери (де доступно)

Натисніть на міні-графік для відкриття повного графіка телеметрії.

## Налаштування віджетів

Дашборд використовує сітку з перетягуванням:

1. Натисніть **Налаштувати макет** (іконка олівця вгорі праворуч)
2. Перетягніть віджети на потрібну позицію
3. Змініть розмір, перетягуючи за кут
4. Натисніть **Заблокувати макет** для фіксації розміщення
5. Натисніть **Зберегти** для збереження макета

Доступні віджети:
| Віджет | Опис |
|--------|------|
| Камера | Живий відеопотік |
| AMS | Статус слотів та філаменту |
| Температура | Кільцеві вимірювачі для сопла та столу |
| Прогрес | Індикатор відсотка та оцінка часу |
| Телеметрія | Вентилятори, тиск, швидкість |
| 3D-модель | Інтерактивне відображення моделі |
| Міні-графіки | Мініатюрні графіки температури |

:::tip Збереження
Макет зберігається для кожного користувача в браузері (localStorage). Різні користувачі можуть мати різні макети.
:::
