# Сховище філаменту

Сховище філаменту надає повний огляд усіх котушок філаменту, інтегрований з AMS та історією друку.

## Огляд

Сховище відображає всі зареєстровані котушки з:

- **Колір** — візуальна кольорова карта
- **Матеріал** — PLA, PETG, ABS, TPU, PA тощо
- **Постачальник** — Bambu Lab, Polymaker, eSUN тощо
- **Вага** — залишок у грамах (розрахунковий або зважений)
- **Слот AMS** — в якому слоті знаходиться котушка
- **Статус** — активна, порожня, сушиться, на зберіганні

## Додавання котушок

1. Натисніть **+ Нова котушка**
2. Заповніть матеріал, колір, постачальника та вагу
3. Відскануйте NFC-тег, якщо доступний, або введіть вручну
4. Збережіть

:::tip Котушки Bambu Lab
Офіційні котушки Bambu Lab можна імпортувати автоматично через інтеграцію з Bambu Cloud. Дивіться [Bambu Cloud](../getting-started/bambu-cloud).
:::

## Синхронізація AMS

Коли дашборд підключений до принтера, статус AMS синхронізується автоматично:

- Слоти відображаються з правильним кольором та матеріалом з AMS
- Витрата оновлюється після кожного друку
- Порожні котушки позначаються автоматично

Щоб прив'язати локальну котушку до слота AMS:
1. Перейдіть до **Філамент → AMS**
2. Натисніть на слот, який хочете прив'язати
3. Виберіть котушку зі сховища

## Сушіння

Реєструйте цикли сушіння для відстеження впливу вологості:

| Поле | Опис |
|------|------|
| Дата сушіння | Коли котушку висушили |
| Температура | Температура сушіння (°C) |
| Тривалість | Кількість годин |
| Метод | Духовка, скринька для сушіння, осушувач філаменту |

:::info Рекомендовані температури сушіння
Дивіться [Базу знань](../kb/intro) для специфічних для матеріалу часів та температур сушіння.
:::

## Кольорова карта

Кольорова карта організовує котушки візуально за кольором. Зручно для швидкого пошуку потрібного кольору. Фільтруйте за матеріалом, постачальником або статусом.

## NFC-теги

Bambu Dashboard підтримує NFC-теги для швидкої ідентифікації котушок:

1. Запишіть ID NFC-тега до котушки у сховищі
2. Відскануйте тег телефоном
3. Котушка відкривається безпосередньо в дашборді

## Імпорт та експорт

### Експорт
Експортуйте все сховище як CSV: **Філамент → Експорт → CSV**

### Імпорт
Імпортуйте котушки з CSV: **Філамент → Імпорт → Обрати файл**

Формат CSV:
```
назва,матеріал,колір_hex,постачальник,вага_г,nfc_id
PLA Білий,PLA,#FFFFFF,Bambu Lab,1000,
PETG Чорний,PETG,#000000,Polymaker,850,ABC123
```

## Статистика

У розділі **Філамент → Статистика** ви знайдете:

- Загальна витрата за матеріалом (за останні 30/90/365 днів)
- Витрата за принтером
- Розрахунковий залишок строку служби для кожної котушки
- Найбільш використовувані кольори та постачальники
