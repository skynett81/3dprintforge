# Огляд парку принтерів

Огляд парку принтерів надає компактний огляд усіх підключених принтерів на одній сторінці. Ідеально для майстерень, навчальних аудиторій або всіх, хто має більше одного принтера.

Перейдіть до: **https://localhost:3443/#fleet**

## Сітка кількох принтерів

Усі зареєстровані принтери відображаються у адаптивній сітці:

- **Розмір картки** — Малий (компактний), Середній (стандартний), Великий (детальний)
- **Кількість стовпців** — Автоматично підлаштовується під ширину екрана або встановлюється вручну
- **Оновлення** — Кожна картка оновлюється незалежно через MQTT

Кожна картка принтера показує:
| Поле | Опис |
|------|------|
| Назва принтера | Налаштована назва з іконкою моделі |
| Статус | Вільний / Друкує / Пауза / Помилка / Відключений |
| Прогрес | Панель відсотка з залишковим часом |
| Температура | Сопло та стіл (компактно) |
| Активний філамент | Колір та матеріал з AMS |
| Мініатюра камери | Фото, що оновлюється кожні 30 секунд |

## Індикатор статусу для кожного принтера

Кольори статусу дозволяють легко бачити стан здалеку:

- **Зелений пульс** — Активний друк
- **Синій** — Вільний та готовий
- **Жовтий** — Призупинений (вручну або Print Guard)
- **Червоний** — Виявлено помилку
- **Сірий** — Відключений або недоступний

:::tip Режим кіоску
Використовуйте огляд парку в режимі кіоску на настінному екрані. Дивіться [Режим кіоску](../system/kiosk) для налаштування.
:::

## Сортування

Натисніть **Сортувати** для вибору порядку:

1. **Назва** — Алфавітно А–Я
2. **Статус** — Активні принтери зверху
3. **Прогрес** — Найбільш готові зверху
4. **Остання активність** — Нещодавно використані зверху
5. **Модель** — Згруповано за моделлю принтера

Сортування запам'ятовується до наступного візиту.

## Фільтрація

Використовуйте поле фільтра вгорі для обмеження вигляду:

- Введіть назву принтера або частину назви
- Виберіть **Статус** зі списку, що розкривається (Всі / Друкує / Вільний / Помилка)
- Виберіть **Модель** для відображення лише одного типу принтера (X1C, P1S, A1 тощо)
- Натисніть **Скинути фільтр** для відображення всіх

:::info Пошук
Пошук фільтрує в реальному часі без перезавантаження сторінки.
:::

## Дії з огляду парку

Клацніть правою кнопкою миші на картці (або натисніть три крапки) для швидких дій:

- **Відкрити дашборд** — Перейти безпосередньо до головної панелі принтера
- **Призупинити друк** — Поставити принтер на паузу
- **Зупинити друк** — Скасувати поточний друк (потребує підтвердження)
- **Переглянути камеру** — Відкрити вигляд камери у спливаючому вікні
- **Перейти до налаштувань** — Відкрити налаштування принтера

:::danger Зупинити друк
Зупинка друку незворотна. Завжди підтверджуйте у діалоговому вікні, що з'являється.
:::

## Агрегована статистика

Вгорі огляду парку відображається рядок підсумків:

- Загальна кількість принтерів
- Кількість активних друків
- Загальна витрата філаменту сьогодні
- Розрахунковий час завершення для найдовшого поточного друку
