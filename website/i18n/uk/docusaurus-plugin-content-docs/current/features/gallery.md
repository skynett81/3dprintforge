# Галерея

Галерея збирає автоматичні знімки екрана, зроблені під час кожного друку. Знімки робляться у фіксованих контрольних точках і надають візуальний журнал розвитку друку.

Перейдіть до: **https://localhost:3443/#gallery**

## Знімки контрольних точок

Bambu Dashboard автоматично робить знімок з камери у таких контрольних точках:

| Контрольна точка | Момент |
|-----------------|--------|
| **25 %** | Чверть шляху через друк |
| **50 %** | На половині |
| **75 %** | Три чверті шляху |
| **100 %** | Друк завершено |

Знімки зберігаються та прив'язуються до відповідного запису в історії друку та відображаються в галереї.

:::info Вимоги
Знімки контрольних точок потребують підключеної та активної камери. Вимкнені камери не генерують зображень.
:::

## Активація функції знімків

1. Перейдіть до **Налаштування → Галерея**
2. Увімкніть **Автоматичні знімки контрольних точок**
3. Виберіть, які контрольні точки активувати (всі чотири включені за замовчуванням)
4. Виберіть **Якість зображення**: Низька (640×360) / Середня (1280×720) / Висока (1920×1080)
5. Натисніть **Зберегти**

## Перегляд зображень

Галерея організована за друками:

1. Використовуйте **фільтр** вгорі для вибору принтера, дати або назви файлу
2. Натисніть на рядок друку для розгортання та перегляду всіх чотирьох зображень
3. Натисніть на зображення для відкриття попереднього перегляду

### Попередній перегляд

Попередній перегляд показує:
- Зображення у повному розмірі
- Контрольна точка та мітка часу
- Назва друку та принтер
- **←** / **→** для навігації між зображеннями одного друку

## Повноекранний перегляд

Натисніть **Повний екран** (або натисніть `F`) у попередньому перегляді для заповнення всього екрана. Використовуйте клавіші зі стрілками для навігації між зображеннями.

## Завантаження зображень

- **Одне зображення**: Натисніть **Завантажити** у попередньому перегляді
- **Всі зображення для друку**: Натисніть **Завантажити все** на рядку друку — отримаєте файл `.zip`
- **Вибрати кілька**: Позначте прапорці та натисніть **Завантажити вибрані**

## Видалення зображень

:::warning Місце на диску
Зображення галереї можуть займати значне місце з часом. Налаштуйте автоматичне видалення старих зображень.
:::

### Ручне видалення

1. Виберіть одне або кілька зображень (позначте прапорці)
2. Натисніть **Видалити вибрані**
3. Підтвердіть у діалозі

### Автоматичне очищення

1. Перейдіть до **Налаштування → Галерея → Автоматичне очищення**
2. Активуйте **Видаляти зображення старші ніж**
3. Встановіть кількість днів (наприклад, 90 днів)
4. Очищення запускається автоматично щоночі о 03:00

## Зв'язок з історією друку

Кожне зображення прив'язане до запису друку в історії:

- Натисніть **Переглянути в історії** для друку в галереї, щоб перейти до запису в історії
- В історії відображається мініатюра зображення 100%, якщо воно є

## Поширення

Поширте зображення галереї через посилання з часовим обмеженням:

1. Відкрийте зображення у попередньому перегляді
2. Натисніть **Поширити**
3. Виберіть час закінчення дії та скопіюйте посилання
