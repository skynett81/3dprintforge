# Історія друку

Історія друку надає повний журнал усіх друків, виконаних за допомогою дашборду, включаючи статистику, витрату філаменту та посилання на джерела моделей.

## Таблиця історії

Таблиця відображає всі друки з:

| Стовпець | Опис |
|---------|------|
| Дата/час | Час початку |
| Назва моделі | Ім'я файлу або заголовок MakerWorld |
| Принтер | Який принтер використовувався |
| Тривалість | Загальний час друку |
| Філамент | Матеріал та використані грами |
| Пластини | Кількість шарів та вага (г) |
| Статус | Завершено, скасовано, помилка |
| Зображення | Мініатюра (при хмарній інтеграції) |

## Пошук та фільтрація

Використовуйте поле пошуку та фільтри для пошуку друків:

- Повнотекстовий пошук за назвою моделі
- Фільтрація за принтером, матеріалом, статусом, датою
- Сортування за всіма стовпцями

## Посилання на джерело моделі

Якщо друк був запущений з MakerWorld, відображається посилання безпосередньо на сторінку моделі. Натисніть на назву моделі для відкриття MakerWorld у новій вкладці.

:::info Bambu Cloud
Посилання на моделі та мініатюри потребують інтеграції з Bambu Cloud. Дивіться [Bambu Cloud](../getting-started/bambu-cloud).
:::

## Відстеження філаменту

Для кожного друку реєструється:

- **Матеріал** — PLA, PETG, ABS тощо
- **Використано грам** — розрахункова витрата
- **Котушка** — яка котушка використовувалася (якщо зареєстрована у сховищі)
- **Колір** — hex-код кольору

Це надає точну картину витрати філаменту з часом і допомагає планувати закупівлі.

## Статистика

У розділі **Історія → Статистика** ви знайдете агреговані дані:

- **Загальна кількість друків** — та відсоток успіху
- **Загальний час друку** — години та дні
- **Витрата філаменту** — грами та км за матеріалом
- **Друків на день** — прокручуваний графік
- **Найбільш використовувані матеріали** — кругова діаграма
- **Розподіл тривалості друку** — гістограма

Статистику можна фільтрувати за часовим періодом (7д, 30д, 90д, 1рік, все).

## Експорт

### Експорт CSV
Експортуйте повну історію або відфільтровані результати:
**Історія → Експорт → Завантажити CSV**

CSV-файли містять усі стовпці та можуть бути відкриті в Excel, LibreOffice Calc або імпортовані в інші інструменти.

### Автоматичне резервне копіювання
Історія є частиною бази даних SQLite, яка автоматично резервується при оновленнях. Ручне резервне копіювання в розділі **Налаштування → Резервне копіювання**.

## Редагування

Ви можете редагувати записи журналу друку після завершення:

- Виправляти назву моделі
- Додавати нотатки
- Виправляти витрату філаменту
- Видаляти помилково зареєстровані друки

Клацніть правою кнопкою миші на рядок і виберіть **Редагувати** або натисніть іконку олівця.
