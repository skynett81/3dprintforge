# OBS-оверлей

OBS-оверлей дозволяє відображати статус принтера в реальному часі безпосередньо в OBS Studio — ідеально для прямих трансляцій або запису 3D-друку.

## URL оверлею

Оверлей доступний як веб-сторінка з прозорим фоном:

```
https://localhost:3443/obs-overlay?printer=PRINTER_ID
```

Замініть `PRINTER_ID` на ID вашого принтера (знаходиться в **Налаштування → Принтери**).

### Доступні параметри

| Параметр | Стандартне значення | Опис |
|---------|---------------------|------|
| `printer` | перший принтер | ID принтера для відображення |
| `theme` | `dark` | `dark`, `light` або `minimal` |
| `scale` | `1.0` | Масштаб (0.5–2.0) |
| `position` | `bottom-left` | `top-left`, `top-right`, `bottom-left`, `bottom-right` |
| `opacity` | `0.9` | Прозорість (0.0–1.0) |
| `fields` | все | Список через кому: `progress,temp,ams,time` |
| `color` | `#00d4ff` | Акцентний колір (hex) |

**Приклад з параметрами:**
```
https://localhost:3443/obs-overlay?printer=abc123&theme=minimal&scale=1.2&position=top-right&fields=progress,time
```

## Налаштування в OBS Studio

### Крок 1: Додайте джерело браузера

1. Відкрийте OBS Studio
2. Натисніть **+** в розділі **Джерела**
3. Виберіть **Браузер** (Browser Source)
4. Дайте джерелу назву, наприклад `Bambu Overlay`
5. Натисніть **OK**

### Крок 2: Налаштуйте джерело браузера

Заповніть наступне в діалозі налаштувань:

| Поле | Значення |
|------|---------|
| URL | `https://localhost:3443/obs-overlay?printer=ВАШ_ID` |
| Ширина | `400` |
| Висота | `200` |
| FPS | `30` |
| Власний CSS | *(залиште порожнім)* |

Позначте:
- ✅ **Вимкнути джерело, коли воно не видиме**
- ✅ **Оновити браузер, коли сцена активується**

:::warning HTTPS і localhost
OBS може попередити про самопідписаний сертифікат. Спочатку перейдіть до `https://localhost:3443` у Chrome/Firefox та прийміть сертифікат. OBS використовуватиме те ж підтвердження.
:::

### Крок 3: Прозорий фон

Оверлей побудований з `background: transparent`. Щоб це працювало в OBS:

1. **Не** позначайте **Власний колір фону** у джерелі браузера
2. Переконайтесь, що оверлей не обгорнутий у непрозорий елемент
3. За бажанням встановіть **Режим змішування** на **Нормальний** для джерела в OBS

:::tip Альтернатива: Chroma key
Якщо прозорість не працює, використовуйте фільтр → **Chroma Key** із зеленим фоном:
Додайте `&bg=green` до URL і встановіть фільтр chroma key на джерелі в OBS.
:::

## Що відображається в оверлеї

Стандартний оверлей містить:

- **Панель прогресу** з відсотковим значенням
- **Залишковий час** (розрахунковий)
- **Температура сопла** та **температура столу**
- **Активний слот AMS** з кольором філаменту та назвою
- **Модель принтера** та назва (можна вимкнути)

## Мінімальний режим для трансляцій

Для дискретного оверлею під час трансляцій:

```
https://localhost:3443/obs-overlay?theme=minimal&fields=progress,time&scale=0.8&opacity=0.7
```

Відображає лише невелику панель прогресу із залишковим часом у куті.
