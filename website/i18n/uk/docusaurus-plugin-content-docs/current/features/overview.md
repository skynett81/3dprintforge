# Огляд функцій

Bambu Dashboard збирає все необхідне для моніторингу та керування принтерами Bambu Lab в одному місці.

## Дашборд

Головний дашборд показує статус активного принтера в реальному часі:

- **Температура** — анімовані SVG-кільцеві вимірювачі для сопла та столу
- **Прогрес** — відсоток виконання з розрахунковим часом завершення
- **Камера** — живий відеопотік (RTSPS → MPEG1 через ffmpeg)
- **Панель AMS** — візуальне відображення всіх слотів AMS з кольором філаменту
- **Контроль швидкості** — повзунок для регулювання швидкості (Тихо, Стандарт, Спорт, Турбо)
- **Статистичні панелі** — панелі в стилі Grafana з графіками прокрутки
- **Телеметрія** — значення вентиляторів, температур, тиску в реальному часі

Панелі можна перетягувати для налаштування макета. Використовуйте кнопку блокування для фіксації макета.

## Сховище філаменту

Дивіться [Філамент](./filament) для повної документації.

- Відстежуйте всі котушки з назвою, кольором, вагою та постачальником
- Синхронізація AMS — перегляд котушок, встановлених в AMS
- Журнал та план сушіння
- Кольорова карта та підтримка NFC-тегів
- Імпорт/експорт (CSV)

## Історія друку

Дивіться [Історія](./history) для повної документації.

- Повний журнал усіх друків
- Відстеження філаменту на кожен друк
- Посилання на моделі MakerWorld
- Статистика та експорт у CSV

## Планувальник

Дивіться [Планувальник](./scheduler) для повної документації.

- Календарний вигляд друків
- Черга друку з пріоритизацією
- Розподіл на кілька принтерів

## Керування принтером

Дивіться [Керування](./controls) для повної документації.

- Контроль температури (сопло, стіл, камера)
- Контроль профілів швидкості
- Керування вентиляторами
- Консоль G-code
- Завантаження/вивантаження філаменту

## Сповіщення

Bambu Dashboard підтримує 7 каналів сповіщень:

| Канал | Події |
|-------|-------|
| Telegram | Друк завершено, помилка, пауза |
| Discord | Друк завершено, помилка, пауза |
| Електронна пошта | Друк завершено, помилка |
| ntfy | Усі події |
| Pushover | Усі події |
| SMS (Twilio) | Критичні помилки |
| Webhook | Налаштоване корисне навантаження |

Налаштування в розділі **Налаштування → Сповіщення**.

## Print Guard

Print Guard відстежує активний друк через камеру (xcam) та датчики:

- Автоматична пауза при помилці спагеті
- Налаштовуваний рівень чутливості
- Журнал виявлених подій

## Технічне обслуговування

Розділ технічного обслуговування відстежує:

- Наступне рекомендоване обслуговування за компонентом (сопло, пластини, AMS)
- Відстеження зносу на основі історії друку
- Ручна реєстрація завдань обслуговування

## Кілька принтерів

З підтримкою кількох принтерів ви можете:

- Керувати кількома принтерами з одного дашборду
- Перемикатися між принтерами за допомогою вибору принтера
- Одночасно переглядати статус усіх принтерів
- Розподіляти завдання друку за допомогою черги друку

## OBS-оверлей

Спеціальна сторінка `obs.html` забезпечує чистий оверлей для інтеграції з OBS Studio під час прямих трансляцій друку.

## Оновлення

Вбудоване автоматичне оновлення через GitHub Releases. Сповіщення та оновлення безпосередньо з дашборду в розділі **Налаштування → Оновлення**.
