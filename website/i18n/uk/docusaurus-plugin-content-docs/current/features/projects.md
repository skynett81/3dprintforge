# Проекти

Проекти дозволяють групувати пов'язані друки, відстежувати витрати на матеріали, виставляти рахунки клієнтам та ділитися оглядом вашої роботи.

Перейдіть до: **https://localhost:3443/#projects**

## Створення проекту

1. Натисніть **Новий проект** (іконка +)
2. Заповніть:
   - **Назва проекту** — описова назва (макс. 100 символів)
   - **Клієнт** — необов'язковий обліковий запис клієнта (дивіться [E-commerce](../integrations/ecommerce))
   - **Опис** — короткий текстовий опис
   - **Колір** — виберіть колір для візуальної ідентифікації
   - **Теги** — ключові слова, розділені комами
3. Натисніть **Створити проект**

## Прив'язка друків до проекту

### Під час друку

1. Відкрийте дашборд під час активного друку
2. Натисніть **Прив'язати до проекту** на бічній панелі
3. Виберіть існуючий проект або створіть новий
4. Друк автоматично прив'язується до проекту після завершення

### З історії

1. Перейдіть до **Історії**
2. Знайдіть відповідний друк
3. Натисніть на друк → **Прив'язати до проекту**
4. Виберіть проект зі списку, що розкривається

### Масова прив'язка

1. Виберіть кілька друків в історії за допомогою прапорців
2. Натисніть **Дії → Прив'язати до проекту**
3. Виберіть проект — всі вибрані друки будуть прив'язані

## Огляд витрат

Кожен проект обчислює загальні витрати на основі:

| Тип витрат | Джерело |
|-----------|--------|
| Витрата філаменту | Грами × ціна за грам за матеріалом |
| Електроенергія | кВт·год × ціна на електроенергію (від Tibber/Nordpool якщо налаштовано) |
| Знос машини | Розраховано з [Прогнозу зносу](../monitoring/wearprediction) |
| Ручна вартість | Довільні статті, які ви додаєте вручну |

Огляд витрат відображається як таблиця та кругова діаграма за друком та загалом.

:::tip Погодинні ціни
Активуйте інтеграцію з Tibber або Nordpool для точних витрат на електроенергію на кожен друк. Дивіться [Ціна на електроенергію](../integrations/energy).
:::

## Виставлення рахунків

1. Відкрийте проект і натисніть **Генерувати рахунок-фактуру**
2. Заповніть:
   - **Дата рахунку** та **дата оплати**
   - **Ставка ПДВ** (0 %, 15 %, 25 %)
   - **Надбавка** (%)
   - **Примітка для клієнта**
3. Перегляньте рахунок у форматі PDF
4. Натисніть **Завантажити PDF** або **Надіслати клієнту** (через електронну пошту)

Рахунки зберігаються в проекті та можуть бути знову відкриті та відредаговані до відправки.

:::info Дані клієнта
Дані клієнта (ім'я, адреса, код організації) беруться з облікового запису клієнта, прив'язаного до проекту. Дивіться [E-commerce](../integrations/ecommerce) для керування клієнтами.
:::

## Статус проекту

| Статус | Опис |
|--------|------|
| Активний | Проект у роботі |
| Завершено | Всі друки готові, рахунок надіслано |
| Архівовано | Приховано із стандартного перегляду, але доступно для пошуку |
| На паузі | Тимчасово призупинено |

Змініть статус, натиснувши на індикатор статусу вгорі проекту.

## Поширення проекту

Генеруйте посилання для поширення для показу огляду проекту клієнтам:

1. Натисніть **Поширити проект** в меню проекту
2. Виберіть, що показувати:
   - ✅ Друки та зображення
   - ✅ Загальна витрата філаменту
   - ❌ Витрати та ціни (приховані за замовчуванням)
3. Встановіть час закінчення дії посилання
4. Скопіюйте та поширте посилання

Клієнт бачить сторінку лише для читання без необхідності входу.
