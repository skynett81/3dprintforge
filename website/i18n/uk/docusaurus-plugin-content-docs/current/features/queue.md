# Черга друку

Черга друку дозволяє планувати друки заздалегідь та автоматично відправляти їх до доступних принтерів, коли вони вільні.

Перейдіть до: **https://localhost:3443/#queue**

## Створення черги

1. Перейдіть до **Черга друку** в навігаційному меню
2. Натисніть **Нове завдання** (іконка +)
3. Заповніть:
   - **Ім'я файлу** — завантажте `.3mf` або `.gcode`
   - **Цільовий принтер** — виберіть конкретний принтер або **Автоматично**
   - **Пріоритет** — Низький / Нормальний / Високий / Критичний
   - **Запланований початок** — зараз або конкретна дата/час
4. Натисніть **Додати до черги**

:::tip Перетягування
Ви можете перетягувати файли безпосередньо з провідника файлів на сторінку черги для швидкого додавання.
:::

## Додавання файлів

### Завантажити файл

1. Натисніть **Завантажити** або перетягніть файл до поля завантаження
2. Підтримувані формати: `.3mf`, `.gcode`, `.bgcode`
3. Файл зберігається в бібліотеці файлів та прив'язується до завдання черги

### З бібліотеки файлів

1. Перейдіть до **Бібліотека файлів** та знайдіть файл
2. Натисніть **Додати до черги** для файлу
3. Завдання створюється зі стандартними налаштуваннями — редагуйте за потреби

### З історії

1. Відкрийте попередній друк в **Історії**
2. Натисніть **Друкувати знову**
3. Завдання додається з тими ж налаштуваннями, що й минулого разу

## Пріоритет

Черга обробляється в порядку пріоритету:

| Пріоритет | Колір | Опис |
|-----------|-------|------|
| Критичний | Червоний | Відправляється до першого вільного принтера незалежно від інших завдань |
| Високий | Помаранчевий | Перед нормальними та низькими завданнями |
| Нормальний | Синій | Стандартний порядок (FIFO) |
| Низький | Сірий | Відправляється лише коли немає завдань з вищим пріоритетом |

Перетягуйте завдання в черзі для зміни порядку вручну в межах одного рівня пріоритету.

## Автоматичне відправлення

Коли **Автоматичне відправлення** активоване, 3DPrintForge відстежує всі принтери та автоматично відправляє наступне завдання:

1. Перейдіть до **Налаштування → Черга**
2. Увімкніть **Автоматичне відправлення**
3. Виберіть **Стратегію відправлення**:
   - **Перший вільний** — відправляє до першого принтера, який звільниться
   - **Найменш використовуваний** — пріоритет принтеру з найменшою кількістю друків сьогодні
   - **Round-robin** — рівномірно чергує між усіма принтерами

:::warning Підтвердження
Активуйте **Вимагати підтвердження** в налаштуваннях, якщо хочете вручну схвалювати кожне відправлення перед надсиланням файлу.
:::

## Розподілений старт (Staggered start)

Розподілений старт корисний для уникнення одночасного запуску та завершення всіх принтерів:

1. У діалозі **Нове завдання** розгорніть **Розширені налаштування**
2. Активуйте **Розподілений старт**
3. Встановіть **Затримку між принтерами** (наприклад, 30 хвилин)
4. Система автоматично розподілить час початку

**Приклад:** 4 однакових завдання із 30-хвилинною затримкою починаються о 08:00, 08:30, 09:00 та 09:30.

## Статус черги та відстеження

Огляд черги показує всі завдання зі статусом:

| Статус | Опис |
|--------|------|
| Очікує | Завдання в черзі та очікує принтера |
| Заплановано | Має запланований час початку в майбутньому |
| Відправляється | Передається до принтера |
| Друкується | Виконується на вибраному принтері |
| Завершено | Готово — прив'язано до історії |
| Помилка | Помилка при відправці або під час друку |
| Скасовано | Скасовано вручну |

:::info Сповіщення
Активуйте сповіщення для подій черги в розділі **Налаштування → Сповіщення → Черга**, щоб отримувати повідомлення, коли завдання починається, завершується або не вдається. Дивіться [Сповіщення](./notifications).
:::
