# Планувальник

Планувальник дозволяє організовувати та автоматизувати завдання друку з календарним виглядом та інтелектуальною чергою друку.

## Календарний вигляд

Календарний вигляд надає огляд усіх запланованих та виконаних друків:

- **Вигляд за місяць, тиждень та день** — виберіть рівень деталізації
- **Кодування кольором** — різні кольори за принтером та статусом
- **Натисніть на подію** — перегляньте деталі друку

Завершені друки відображаються автоматично на основі історії друку.

## Черга друку

Черга друку дозволяє ставити завдання в чергу для відправки до принтера по черзі:

### Додавання завдання до черги

1. Натисніть **+ Додати завдання**
2. Виберіть файл (з SD-карти принтера, локального завантаження або FTP)
3. Вкажіть пріоритет (високий, нормальний, низький)
4. Виберіть цільовий принтер (або "автоматично")
5. Натисніть **Додати**

### Керування чергою

| Дія | Опис |
|-----|------|
| Перетягнути | Змінити порядок |
| Призупинити чергу | Тимчасово зупинити відправку |
| Пропустити | Відправити наступне завдання без очікування |
| Видалити | Вилучити завдання з черги |

:::tip Розподіл на кілька принтерів
За наявності кількох принтерів черга може автоматично розподіляти завдання до вільних принтерів. Активуйте **Автоматичне відправлення** в розділі **Планувальник → Налаштування**.
:::

## Заплановані друки

Налаштуйте друки, які мають розпочатися у певний час:

1. Натисніть **+ Запланувати друк**
2. Виберіть файл та принтер
3. Вкажіть час початку
4. Налаштуйте сповіщення (необов'язково)
5. Збережіть

:::warning Принтер повинен бути вільним
Заплановані друки починаються лише якщо принтер знаходиться в режимі очікування у вказаний час. Якщо принтер зайнятий, початок переноситься до наступного доступного часу (налаштовується).
:::

## Балансування навантаження

З автоматичним балансуванням навантаження завдання розумно розподіляються між принтерами:

- **Round-robin** — рівномірний розподіл між усіма принтерами
- **Найменш завантажений** — відправити до принтера з найкоротшим розрахунковим часом завершення
- **Вручну** — ви самостійно вибираєте принтер для кожного завдання

Налаштування в розділі **Планувальник → Балансування навантаження**.

## Сповіщення

Планувальник інтегрується з каналами сповіщень:

- Сповіщення при запуску завдання
- Сповіщення при завершенні завдання
- Сповіщення при помилці або затримці

Дивіться [Огляд функцій](./overview#varsler) для налаштування каналів сповіщень.
