# Таймлапс

3DPrintForge може автоматично робити знімки під час друку та збирати їх у відео таймлапс. Відео зберігаються локально та можуть відтворюватися безпосередньо в дашборді.

Перейдіть до: **https://localhost:3443/#timelapse**

## Активація

1. Перейдіть до **Налаштування → Таймлапс**
2. Увімкніть **Активувати запис таймлапсу**
3. Виберіть **Режим запису**:
   - **По шарах** — один знімок на шар (рекомендується для високої якості)
   - **За часом** — один знімок кожні N секунд (наприклад, кожні 30 секунд)
4. Виберіть принтери, для яких активувати таймлапс
5. Натисніть **Зберегти**

:::tip Інтервал знімків
«По шарах» дає рівномірнішу анімацію, оскільки рух постійний. «За часом» використовує менше місця.
:::

## Налаштування запису

| Налаштування | Стандартне значення | Опис |
|-------------|--------------------|----|
| Роздільна здатність | 1280×720 | Розмір зображення (640×480 / 1280×720 / 1920×1080) |
| Якість зображення | 85 % | Якість стиснення JPEG |
| FPS у відео | 30 | Кадрів на секунду у готовому відео |
| Формат відео | MP4 (H.264) | Вихідний формат |
| Повернути зображення | Вимкнено | Повернути на 90°/180°/270° для напрямку монтажу |

:::warning Місце на диску
Таймлапс з 500 знімками у 1080p займає приблизно 200–400 МБ до об'єднання. Готове MP4-відео зазвичай 20–80 МБ.
:::

## Зберігання

Знімки та відео таймлапсу зберігаються в `data/timelapse/` в папці проекту. Структура організована за принтером та друком:

```
data/timelapse/
├── <printer-id>/                     ← Унікальний ID принтера
│   ├── 2026-03-22_назвамоделі/        ← Сесія друку (дата_назвамоделі)
│   │   ├── frame_0001.jpg
│   │   ├── frame_0002.jpg
│   │   ├── frame_0003.jpg
│   │   └── ...                       ← Вихідні знімки (видаляються після об'єднання)
│   ├── 2026-03-22_назвамоделі.mp4     ← Готове відео таймлапсу
│   ├── 2026-03-20_3dbenchy.mp4
│   └── 2026-03-15_підставкателефон.mp4
├── <printer-id-2>/                   ← Кілька принтерів
│   └── ...
```

:::tip Зовнішнє сховище
Для економії місця на системному диску можна зробити символічне посилання папки таймлапсу на зовнішній диск:
```bash
# Приклад: перемістити на зовнішній диск, підмонтований на /mnt/storage
mv data/timelapse /mnt/storage/timelapse

# Створити символічне посилання назад
ln -s /mnt/storage/timelapse data/timelapse
```
Дашборд автоматично слідує за символічним посиланням. Можна використовувати будь-який диск або мережевий ресурс.
:::

## Автоматичне об'єднання

Коли друк завершується, знімки автоматично збираються у відео за допомогою ffmpeg:

1. 3DPrintForge отримує подію «print complete» від MQTT
2. ffmpeg викликається зі зібраними знімками
3. Відео зберігається в папці зберігання
4. Сторінка таймлапсу оновлюється новим відео

Прогрес можна переглянути на вкладці **Таймлапс → Обробляється**.

## Відтворення

1. Перейдіть до **https://localhost:3443/#timelapse**
2. Виберіть принтер зі списку, що розкривається
3. Натисніть на відео в списку для відтворення
4. Використовуйте елементи керування відтворенням:
   - ▶ / ⏸ — Відтворити / Пауза
   - ⏪ / ⏩ — Перемотати назад / вперед
   - Кнопки швидкості: 0.5× / 1× / 2× / 4×
5. Натисніть **Повний екран** для відкриття на весь екран
6. Натисніть **Завантажити** для завантаження MP4-файлу

## Видалення таймлапсу

1. Виберіть відео в списку
2. Натисніть **Видалити** (іконка кошика)
3. Підтвердіть у діалоговому вікні

:::danger Постійне видалення
Видалені відео таймлапсу та вихідні знімки не підлягають відновленню. Спочатку завантажте відео, якщо хочете його зберегти.
:::

## Поширення таймлапсу

Відео таймлапсу можна поширювати через посилання з часовим обмеженням:

1. Виберіть відео та натисніть **Поширити**
2. Встановіть час закінчення дії (1 година / 24 години / 7 днів / без обмеження)
3. Скопіюйте згенероване посилання та поширте його
4. Отримувач не потребує входу для перегляду відео
