# Інтеграція з Bambu Cloud

3DPrintForge може підключатися до **Bambu Lab Cloud** для отримання зображень моделей, історії друку та даних про філамент. Дашборд чудово працює без хмарного підключення, але інтеграція з хмарою дає додаткові переваги.

## Переваги хмарної інтеграції

| Функція | Без хмари | З хмарою |
|---------|----------|---------|
| Статус принтера в реальному часі | Так | Так |
| Локальна історія друку | Так | Так |
| Зображення моделей з MakerWorld | Ні | Так |
| Профілі філаменту від Bambu | Ні | Так |
| Синхронізація історії друку | Ні | Так |
| Філамент AMS з хмари | Ні | Так |

## Підключення до Bambu Cloud

1. Перейдіть до **Налаштування → Bambu Cloud**
2. Введіть свою електронну адресу та пароль Bambu Lab
3. Натисніть **Увійти**
4. Виберіть, які дані синхронізувати

:::warning Конфіденційність
Ім'я користувача та пароль не зберігаються у відкритому вигляді. Дашборд використовує API Bambu Labs для отримання OAuth-токена, який зберігається локально. Ваші дані ніколи не покидають ваш сервер.
:::

## Синхронізація

### Зображення моделей

Коли хмара підключена, зображення моделей автоматично завантажуються з **MakerWorld** і відображаються в:
- Історії друку
- Дашборді (під час активного друку)
- Переглядачі 3D-моделей

### Історія друку

Хмарна синхронізація імпортує історію друку з програми Bambu Lab. Дублікати фільтруються автоматично на основі мітки часу та серійного номера.

### Профілі філаменту

Офіційні профілі філаменту Bambu Labs синхронізуються та відображаються у сховищі філаментів. Ви можете використовувати їх як основу для власних профілів.

## Що працює без хмари?

Усі основні функції працюють без хмарного підключення:

- Пряме MQTT-підключення до принтера через LAN
- Статус у реальному часі, температура, камера
- Локальна історія та статистика друку
- Сховище філаменту (керується вручну)
- Сповіщення та планувальник

:::tip Режим тільки LAN
Хочете використовувати дашборд без підключення до інтернету? Він чудово працює в ізольованій мережі — просто підключіться до принтера через IP і залиште хмарну інтеграцію вимкненою.
:::

## Усунення несправностей

**Помилка входу:**
- Перевірте правильність електронної адреси та пароля для програми Bambu Lab
- Перевірте, чи обліковий запис використовує двофакторну автентифікацію (ще не підтримується)
- Спробуйте вийти та увійти знову

**Синхронізація зупиняється:**
- Токен міг застаріти — вийдіть і увійдіть знову в Налаштуваннях
- Перевірте підключення до інтернету на вашому сервері
