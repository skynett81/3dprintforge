# Усунення несправностей при невдалому друку

Щось пішло не так? Не панікуйте — більшість помилок друку мають прості рішення. Bambu Dashboard допоможе вам швидко знайти причину.

## Крок 1 — Перевірте коди помилок HMS

HMS (Handling, Monitoring, Sensing) — це система помилок Bambu Labs. Усі помилки автоматично записуються в панелі управління.

1. Перейдіть до **Моніторинг → Помилки**
2. Знайдіть невдалий друк
3. Натисніть на код помилки для детального опису та запропонованого рішення

Типові коди HMS:

| Код | Опис | Швидке рішення |
|-----|------|----------------|
| 0700 1xxx | Помилка AMS (застрягання, проблема з двигуном) | Перевірте шлях філаменту в AMS |
| 0300 0xxx | Помилка екструзії (недо/надекструзія) | Очистіть сопло, перевірте філамент |
| 0500 xxxx | Помилка калібрування | Виконайте повторне калібрування |
| 1200 xxxx | Відхилення температури | Перевірте кабельні з'єднання |
| 0C00 xxxx | Помилка камери | Перезапустіть принтер |

:::tip Коди помилок в історії
У **Історія → [Друк] → Журнал HMS** ви можете побачити всі коди помилок, що виникли під час друку — навіть якщо друк "завершився".
:::

## Поширені помилки та рішення

### Погана адгезія (перший шар не прилипає)

**Симптоми:** Виріб відривається від столу, скручується, перший шар відсутній

**Причини та рішення:**

| Причина | Рішення |
|---------|---------|
| Брудна пластина | Протріть спиртом IPA |
| Неправильна температура пластини | Збільшіть на 5°C |
| Неправильний Z-зсув | Повторно виконайте Auto Bed Leveling |
| Відсутній клей (PETG/ABS) | Нанесіть тонкий шар клею |
| Занадто висока швидкість першого шару | Знизьте до 20–30 мм/с для першого шару |

**Швидкий контрольний список:**
1. Пластина чиста? (IPA + безворсовий папір)
2. Ви використовуєте правильну пластину для типу філаменту? (дивіться [Вибір правильної пластини](./velge-rett-plate))
3. Калібрування Z виконано після останньої заміни пластини?

---

### Варпінг (кути підіймаються)

**Симптоми:** Кути вигинаються від пластини, особливо на великих плоских моделях

**Причини та рішення:**

| Причина | Рішення |
|---------|---------|
| Різниця температур | Закрийте передні дверцята принтера |
| Відсутність країв (brim) | Увімкніть brim у Bambu Studio (3–5 мм) |
| Пластина занадто холодна | Збільшіть температуру пластини на 5–10°C |
| Філамент із високою усадкою (ABS) | Використовуйте Engineering Plate + камера >40°C |

**ABS і ASA особливо чутливі.** Завжди забезпечуйте:
- Передні дверцята закриті
- Мінімальна вентиляція
- Engineering Plate + клей
- Температура камери 40°C+

---

### Стрінгінг (нитки між деталями)

**Симптоми:** Тонкі пластикові нитки між окремими частинами моделі

**Причини та рішення:**

| Причина | Рішення |
|---------|---------|
| Вологий філамент | Сушіть філамент 6–8 годин (60–70°C) |
| Занадто висока температура сопла | Знизьте на 5°C |
| Недостатня ретракція | Збільшіть довжину ретракції в Bambu Studio |
| Занадто низька швидкість переміщення | Збільшіть швидкість переміщення до 200+ мм/с |

**Тест на вологість:** Прислухайтеся до тріскання або спостерігайте за бульбашками в екструзії — це вказує на вологий філамент. Bambu AMS має вбудований вимірювач вологості; перевірте вологість у **Статус AMS**.

:::tip Сушарка для філаменту
Інвестуйте в сушарку для філаменту (наприклад, Bambu Filament Dryer), якщо ви працюєте з нейлоном або TPU — вони вбирають вологу менш ніж за 12 годин.
:::

---

### Спагеті (виріб перетворюється на клубок)

**Симптоми:** Філамент висить у вільних нитках у повітрі, виріб не впізнаваний

**Причини та рішення:**

| Причина | Рішення |
|---------|---------|
| Рання погана адгезія → відірвався → обвалився | Дивіться розділ про адгезію вище |
| Занадто висока швидкість | Знизьте швидкість на 20–30% |
| Неправильна конфігурація підтримок | Увімкніть підтримки в Bambu Studio |
| Навіс занадто крутий | Розділіть модель або поверніть на 45° |

**Використовуйте Print Guard для автоматичного зупинення спагеті** — дивіться наступний розділ.

---

### Недоекструзія (тонкі, слабкі шари)

**Симптоми:** Шари не суцільні, дірки в стінках, слабка модель

**Причини та рішення:**

| Причина | Рішення |
|---------|---------|
| Частково засмічене сопло | Виконайте Cold Pull (дивіться обслуговування) |
| Філамент занадто вологий | Сушіть філамент |
| Занадто низька температура | Збільшіть температуру сопла на 5–10°C |
| Занадто висока швидкість | Знизьте на 20–30% |
| Пошкоджена трубка PTFE | Перевірте і замініть трубку PTFE |

## Використання Print Guard для автоматичного захисту

Print Guard відстежує знімки камери за допомогою розпізнавання зображень і автоматично зупиняє друк при виявленні спагеті.

**Увімкнення Print Guard:**
1. Перейдіть до **Моніторинг → Print Guard**
2. Увімкніть **Автоматичне виявлення**
3. Виберіть дію: **Призупинити** (рекомендовано) або **Скасувати**
4. Встановіть чутливість (починайте з **Середньої**)

**Коли Print Guard втручається:**
1. Ви отримуєте сповіщення зі знімком камери того, що було виявлено
2. Друк призупиняється
3. Ви можете вибрати: **Продовжити** (якщо хибнопозитивне) або **Скасувати друк**

:::info Хибнопозитивні результати
Print Guard іноді може реагувати на моделі з багатьма тонкими стовпцями. Зменшіть чутливість або тимчасово вимкніть для складних моделей.
:::

## Діагностичні інструменти в панелі управління

### Журнал температур
У **Історія → [Друк] → Температури** ви можете переглянути криву температур упродовж усього друку. Зверніть увагу на:
- Раптові падіння температури (проблема сопла або столу)
- Нерівні температури (потреба в калібруванні)

### Статистика філаменту
Перевірте, чи витрачений філамент відповідає оцінці. Велике відхилення може вказувати на недоекструзію або розрив філаменту.

## Коли звертатися до підтримки?

Зверніться до підтримки Bambu Labs, якщо:
- Код HMS повторюється після виконання всіх запропонованих рішень
- Ви бачите механічні пошкодження принтера (зігнуті стержні, зламані шестерні)
- Значення температури є неможливими (наприклад, сопло показує -40°C)
- Оновлення прошивки не вирішує проблему

**Корисно мати готовим для підтримки:**
- Коди помилок HMS з журналу помилок панелі управління
- Знімок камери помилки
- Який філамент і налаштування використовувались (можна експортувати з історії)
- Модель принтера і версія прошивки (відображається у **Налаштування → Принтер → Інформація**)
