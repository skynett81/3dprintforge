# Налаштування сховища філаменту

Сховище філаменту в Bambu Dashboard дає вам повний огляд усіх ваших котушок — що залишилося, що ви використали і які котушки зараз знаходяться в AMS.

## Автоматичне створення з AMS

Коли ви підключаєте принтер із AMS, панель управління автоматично зчитує інформацію з RFID-чіпів на котушках Bambu:

- Тип філаменту (PLA, PETG, ABS, TPU тощо)
- Колір (із шістнадцятковим кодом)
- Марка (Bambu Lab)
- Вага котушки і залишкова кількість

**Ці котушки автоматично створюються в сховищі** — вам нічого не потрібно робити. Перегляньте їх у розділі **Філамент → Сховище**.

:::info Тільки котушки Bambu мають RFID
Сторонні котушки (наприклад, eSUN, Polymaker, наповнювачі Bambu без чіпа) не розпізнаються автоматично. Їх потрібно додавати вручну.
:::

## Додавання котушок вручну

Для котушок без RFID або для котушок, які не знаходяться в AMS:

1. Перейдіть до **Філамент → Сховище**
2. Натисніть **+ Нова котушка** у верхньому правому куті
3. Заповніть поля:

| Поле | Приклад | Обов'язкове |
|------|---------|-------------|
| Марка | eSUN, Polymaker, Bambu | Так |
| Тип | PLA, PETG, ABS, TPU | Так |
| Колір | #FF5500 або виберіть із колірного кола | Так |
| Початкова вага | 1000 г | Рекомендовано |
| Залишок | 850 г | Рекомендовано |
| Діаметр | 1,75 мм | Так |
| Нотатка | "Куплено 2025-01, добре працює" | Необов'язково |

4. Натисніть **Зберегти**

## Налаштування кольорів і марок

Ви можете редагувати котушку в будь-який час, натиснувши на неї в огляді сховища:

- **Колір** — Виберіть із колірного кола або введіть шістнадцяткове значення. Колір використовується як візуальний маркер у огляді AMS
- **Марка** — Відображається в статистиці та фільтрації. Створюйте власні марки у розділі **Філамент → Марки**
- **Профіль температури** — Введіть рекомендовану температуру сопла і столу від виробника філаменту. Панель управління може попереджати, якщо ви вибираєте неправильну температуру

## Розуміння синхронізації AMS

Панель управління синхронізує статус AMS у реальному часі:

```
Слот AMS 1 → Котушка: Bambu PLA Білий  [███████░░░] 72% залишилось
Слот AMS 2 → Котушка: eSUN PETG Сірий  [████░░░░░░] 41% залишилось
Слот AMS 3 → (порожній)
Слот AMS 4 → Котушка: Bambu PLA Червоний [██████████] 98% залишилось
```

Синхронізація оновлюється:
- **Під час друку** — витрата віднімається в реальному часі
- **Після завершення друку** — остаточна витрата записується в історію
- **Вручну** — натисніть на значок синхронізації на котушці, щоб отримати оновлені дані від AMS

:::tip Коригування оцінки AMS
Оцінка AMS від RFID не завжди точна на 100% після першого використання. Зважте котушку та вручну оновіть вагу для найкращої точності.
:::

## Перевірка витрати та залишку

### По котушці
Натисніть на котушку в сховищі, щоб побачити:
- Загальна використана кількість (грами, усі друки)
- Розрахункова залишкова кількість
- Список усіх друків, у яких використовувалася ця котушка

### Зведена статистика
У розділі **Аналіз → Аналіз філаменту** ви бачите:
- Витрату за типом філаменту з часом
- Які марки ви використовуєте найбільше
- Розрахункову вартість на основі ціни покупки за кг

### Сповіщення про низький рівень
Налаштуйте сповіщення, коли котушка наближається до кінця:

1. Перейдіть до **Філамент → Налаштування**
2. Увімкніть **Сповіщення про низький рівень запасів**
3. Встановіть поріг (наприклад, 100 г залишилося)
4. Виберіть канал сповіщень (Telegram, Discord, електронна пошта)

## Порада: Зважуйте котушки для точності

Оцінки від AMS і статистики друку ніколи не бувають абсолютно точними. Найточніший метод — зважити саму котушку:

**Як це зробити:**

1. Знайдіть тарну вагу (порожня котушка) — зазвичай 200–250 г, перевірте сайт виробника або нижню частину котушки
2. Зважте котушку з філаментом на кухонних вагах
3. Відніміть тарну вагу
4. Оновіть **Залишок** у профілі котушки

**Приклад:**
```
Зважена вага:   743 г
Тара (порожня): - 230 г
Залишок філаменту: 513 г
```

:::tip Генератор етикеток для котушок
У розділі **Інструменти → Етикетки** ви можете друкувати етикетки з QR-кодом для своїх котушок. Відскануйте код телефоном, щоб швидко відкрити профіль котушки.
:::
