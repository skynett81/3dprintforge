# Кілька принтерів

Маєте більше одного принтера? Bambu Dashboard створений для управління парком — ви можете відстежувати, керувати та координувати всі принтери з одного місця.

## Додавання нового принтера

1. Перейдіть до **Налаштування → Принтери**
2. Натисніть **+ Додати принтер**
3. Заповніть:

| Поле | Приклад | Пояснення |
|------|---------|----------|
| Серійний номер (SN) | 01P... | Знаходиться в Bambu Handy або на екрані принтера |
| IP-адреса | 192.168.1.101 | Для LAN-режиму (рекомендовано) |
| Код доступу | 12345678 | 8-значний код на екрані принтера |
| Назва | "Bambu #2 - P1S" | Відображається в панелі |
| Модель | P1P, P1S, X1C, A1 | Виберіть правильну модель для правильних значків і функцій |

4. Натисніть **Перевірити з'єднання** — ви повинні побачити зелений статус
5. Натисніть **Зберегти**

:::tip Давайте принтерам описові назви
"Bambu 1" і "Bambu 2" — заплутано. Використовуйте назви на кшталт "X1C - Виробництво" і "P1S - Прототипи", щоб зберегти порядок.
:::

## Огляд парку

Після додавання всіх принтерів вони відображаються разом на панелі **Парк**. Тут ви бачите:

```
┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
│ X1C - Виробн.   │  │ P1S - Прототипи │  │ A1 - Хобі-кімн. │
│ ████████░░ 82%  │  │ Неактивний      │  │ ████░░░░░░ 38%  │
│ 1г 24хв залиш. │  │ Готовий до друку│  │ 3г 12хв залиш. │
│ Темп: 220/60°C  │  │ AMS: 4 котушки  │  │ Темп: 235/80°C  │
└─────────────────┘  └─────────────────┘  └─────────────────┘
```

Ви можете:
- Натиснути на принтер для повного детального перегляду
- Бачити всі температури, статус AMS та активні помилки одразу
- Фільтрувати за статусом (активні друки, неактивні, помилки)

## Черга друку — розподіл роботи

Черга друку дозволяє планувати друки для всіх принтерів з одного місця.

**Як це працює:**
1. Перейдіть до **Черга**
2. Натисніть **+ Додати завдання**
3. Виберіть файл і налаштування
4. Виберіть принтер або виберіть **Автоматичне призначення**

### Автоматичне призначення
При автоматичному призначенні панель управління вибирає принтер на основі:
- Доступної потужності
- Філаменту, доступного в AMS
- Запланованих вікон обслуговування

Увімкніть у **Налаштування → Черга → Автоматичне призначення**.

### Пріоритизація
Перетягуйте завдання в черзі для зміни порядку. Завдання з **Високим пріоритетом** переходить перед звичайними завданнями.

## Поступовий запуск — уникнення стрибків споживання

Якщо ви запускаєте багато принтерів одночасно, фаза нагрівання може призвести до різкого стрибка споживання. Поступовий запуск розподіляє запуск:

**Як увімкнути:**
1. Перейдіть до **Налаштування → Парк → Поступовий запуск**
2. Увімкніть **Розподілений запуск**
3. Встановіть затримку між принтерами (рекомендовано: 2–5 хвилин)

**Приклад із 3 принтерами і затримкою 3 хвилини:**
```
08:00 — Принтер 1 починає нагрів
08:03 — Принтер 2 починає нагрів
08:06 — Принтер 3 починає нагрів
```

:::tip Важливо для автомату захисту
X1C споживає приблизно 1000 Вт під час нагрівання. Три принтери одночасно = 3000 Вт, що може вибити автомат 16А. Поступовий запуск усуває проблему.
:::

## Групи принтерів

Групи принтерів дозволяють логічно організовувати принтери та надсилати команди всій групі:

**Створення групи:**
1. Перейдіть до **Налаштування → Групи принтерів**
2. Натисніть **+ Нова група**
3. Дайте групі назву (наприклад, "Виробничий цех", "Хобі-кімната")
4. Додайте принтери до групи

**Функції групи:**
- Перегляд зведеної статистики групи
- Надсилання команди паузи всій групі одночасно
- Встановлення вікна обслуговування для групи

## Моніторинг усіх принтерів

### Перегляд кількох камер
Перейдіть до **Парк → Перегляд камер**, щоб бачити всі потоки камер поруч:

```
┌──────────────┐  ┌──────────────┐
│  X1C Потік   │  │  P1S Потік   │
│  [Живий]     │  │  [Неакт.]    │
└──────────────┘  └──────────────┘
┌──────────────┐  ┌──────────────┐
│  A1 Потік    │  │  + Додати    │
│  [Живий]     │  │              │
└──────────────┘  └──────────────┘
```

### Сповіщення для кожного принтера
Ви можете налаштувати різні правила сповіщень для різних принтерів:
- Виробничий принтер: сповіщати завжди, включаючи ніч
- Хобі-принтер: сповіщати лише вдень

Дивіться [Сповіщення](./notification-setup) для налаштування.

## Поради щодо управління парком

- **Стандартизуйте слоти філаменту**: Тримайте PLA білий у слоті 1, PLA чорний у слоті 2 на всіх принтерах — так розподіл роботи простіший
- **Перевіряйте рівні AMS щодня**: Дивіться [Щоденне використання](./daily-use) для ранкової рутини
- **Обслуговування по черзі**: Не обслуговуйте всі принтери одночасно — завжди тримайте хоча б один активним
- **Чітко називайте файли**: Назви файлів на кшталт `logo_x1c_pla_0.2mm.3mf` полегшують вибір правильного принтера
