# Стрімінг 3D-друку через OBS

Bambu Dashboard має вбудоване накладання OBS, яке показує статус принтера, прогрес, температури та потік камери безпосередньо у вашому стрімі.

## Передумови

- Встановлено OBS Studio ([obsproject.com](https://obsproject.com))
- Bambu Dashboard запущено і підключено до принтера
- (Необов'язково) Камера Bambu увімкнена для живого потоку

## Крок 1 — Джерело браузера OBS

OBS має вбудоване **Джерело браузера**, яке показує веб-сторінку безпосередньо у вашій сцені.

**Додавання накладання в OBS:**

1. Відкрийте OBS Studio
2. У розділі **Джерела** (Sources) натисніть **+**
3. Виберіть **Браузер** (Browser)
4. Дайте джерелу ім'я, наприклад "Bambu Overlay"
5. Заповніть:

| Налаштування | Значення |
|-------------|---------|
| URL | `http://localhost:3000/obs/overlay` |
| Ширина | `1920` |
| Висота | `1080` |
| FPS | `30` |
| Власний CSS | Дивіться нижче |

6. Поставте позначку **Керувати аудіо через OBS**
7. Натисніть **OK**

:::info Налаштуйте URL для вашого сервера
Панель управління запущена на іншій машині, ніж OBS? Замініть `localhost` на IP-адресу сервера, наприклад `http://192.168.1.50:3000/obs/overlay`
:::

## Крок 2 — Прозорий фон

Щоб накладання вписалося в зображення, фон повинен бути прозорим:

**У налаштуваннях Джерела браузера OBS:**
- Поставте позначку **Видалити фон** (Shutdown source when not visible / Remove background)

**Власний CSS для примусової прозорості:**
```css
body {
  background-color: rgba(0, 0, 0, 0) !important;
  margin: 0;
  overflow: hidden;
}
```

Вставте це в поле **Власний CSS** у налаштуваннях Джерела браузера.

Накладання тепер показує лише сам віджет — без білого або чорного фону.

## Крок 3 — Налаштування накладання

У Bambu Dashboard ви можете налаштувати, що показує накладання:

1. Перейдіть до **Функції → Накладання OBS**
2. Налаштуйте:

| Налаштування | Варіанти |
|-------------|---------|
| Позиція | Верх ліво, право, низ ліво, право |
| Розмір | Малий, середній, великий |
| Тема | Темна, світла, прозора |
| Колір акценту | Виберіть колір, що відповідає стилю стріму |
| Елементи | Виберіть, що відображати (дивіться нижче) |

**Доступні елементи накладання:**

- Назва принтера та статус (онлайн/друкує/помилка)
- Індикатор прогресу з відсотком і залишком часу
- Філамент і колір
- Температура сопла і столу
- Використаний філамент (грами)
- Огляд AMS (компактний)
- Статус Print Guard

3. Натисніть **Попередній перегляд**, щоб побачити результат без переходу в OBS
4. Натисніть **Зберегти**

:::tip URL для кожного принтера
Маєте кілька принтерів? Використовуйте окремі URL накладань:
```
/obs/overlay?printer=1
/obs/overlay?printer=2
```
:::

## Потік камери в OBS (окреме джерело)

Камеру Bambu можна додати як окреме джерело в OBS — незалежно від накладання:

**Варіант 1: Через проксі камери панелі**

1. Перейдіть до **Система → Камера**
2. Скопіюйте **URL потоку RTSP або MJPEG**
3. В OBS: Натисніть **+** → **Медіаджерело** (Media Source)
4. Вставте URL
5. Поставте позначку **Повтор** (Loop) і вимкніть локальні файли

**Варіант 2: Джерело браузера з переглядом камери**

1. В OBS: Додайте **Джерело браузера**
2. URL: `http://localhost:3000/obs/camera?printer=1`
3. Ширина/висота: відповідна роздільній здатності камери (1080p або 720p)

Тепер ви можете вільно розміщувати потік камери в сцені і накладати поверх накладання.

## Поради для хорошого стріму

### Налаштування сцени для стрімінгу

Типова сцена для стрімінгу 3D-друку:

```
┌─────────────────────────────────────────┐
│                                         │
│      [Потік камери з принтера]          │
│                                         │
│  ┌──────────────────┐                  │
│  │ Bambu Overlay    │  ← Низ ліво      │
│  │ Друк: Logo.3mf   │                  │
│  │ ████████░░ 82%   │                  │
│  │ 1г 24хв залиш.  │                  │
│  │ 220°C / 60°C     │                  │
│  └──────────────────┘                  │
└─────────────────────────────────────────┘
```

### Рекомендовані налаштування

| Параметр | Рекомендоване значення |
|----------|----------------------|
| Розмір накладання | Середній (не надто домінуючий) |
| Частота оновлення | 30 FPS (відповідає OBS) |
| Позиція накладання | Низ ліво (уникає обличчя/чату) |
| Кольорова тема | Темна із синім акцентом |

### Сцени та перемикання сцен

Створіть власні сцени OBS:

- **"Друк виконується"** — перегляд камери + накладання
- **"Пауза / очікування"** — статичне зображення + накладання
- **"Готово"** — зображення результату + накладання з "Завершено"

Перемикайтеся між сценами за допомогою клавіатурного скорочення в OBS або через Scene Collection.

### Стабілізація зображення камери

Камера Bambu іноді може зависати. У панелі управління в **Система → Камера**:
- Увімкніть **Автоматичне перепідключення** — панель управління автоматично перепідключиться
- Встановіть **Інтервал перепідключення** на 10 секунд

### Звук

3D-принтери видають звук — особливо AMS і охолодження. Розгляньте:
- Розмістіть мікрофон подалі від принтера
- Додайте фільтр шуму на мікрофон в OBS (Noise Suppression)
- Або використовуйте фонову музику / звук чату замість

:::tip Автоматичне перемикання сцен
OBS має вбудовану підтримку перемикання сцен на основі назв. Поєднайте з плагіном (наприклад, obs-websocket) та API Bambu Dashboard для автоматичного перемикання сцен, коли друк починається і закінчується.
:::
