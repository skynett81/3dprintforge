# Навігація в панелі управління

Це керівництво дає вам швидке введення в організацію панелі управління та ефективну навігацію.

## Бічна панель

Бічна панель ліворуч — ваш центр навігації. Вона організована в секції:

```
┌────────────────────┐
│ 🖨  Статуси принт. │  ← Один рядок на принтер
├────────────────────┤
│ Огляд              │
│ Парк               │
│ Активний друк      │
├────────────────────┤
│ Філамент           │
│ Історія            │
│ Проєкти            │
│ Черга              │
│ Планувальник       │
├────────────────────┤
│ Моніторинг         │
│  └ Print Guard     │
│  └ Помилки         │
│  └ Діагностика     │
│  └ Обслуговування  │
├────────────────────┤
│ Аналіз             │
│ Інструменти        │
│ Інтеграції         │
│ Система            │
├────────────────────┤
│ ⚙ Налаштування    │
└────────────────────┘
```

**Бічну панель можна приховати**, натиснувши на значок гамбургера (☰) у верхньому лівому куті. Корисно на менших екранах або в режимі кіоску.

## Основна панель

Коли ви натискаєте на елемент у бічній панелі, вміст відображається в основній панелі праворуч. Макет змінюється:

| Панель | Макет |
|--------|-------|
| Огляд | Сітка карток із усіма принтерами |
| Активний друк | Велика картка деталей + криві температур |
| Історія | Таблиця з фільтрами |
| Філамент | Перегляд карток із котушками |
| Аналіз | Графіки та діаграми |

## Натискання на статус принтера для деталей

Картка принтера на панелі огляду є клікабельною:

**Одне натискання** → Відкриває панель деталей для цього принтера:
- Температури в реальному часі
- Активний друк (якщо виконується)
- Статус AMS з усіма слотами
- Останні помилки та події
- Швидкі кнопки: Пауза, Зупинити, Освітлення вкл/викл

**Натискання на значок камери** → Відкриває перегляд камери в реальному часі

**Натискання на значок ⚙** → Налаштування принтера

## Клавіатурне скорочення — палітра команд

Палітра команд забезпечує швидкий доступ до всіх функцій без навігації:

| Скорочення | Дія |
|-----------|-----|
| `Ctrl + K` (Linux/Windows) | Відкрити палітру команд |
| `Cmd + K` (macOS) | Відкрити палітру команд |
| `Esc` | Закрити палітру |

У палітрі команд ви можете:
- Шукати сторінки та функції
- Запустити друк безпосередньо
- Призупинити / відновити активні друки
- Змінити тему (світла/темна)
- Перейти на будь-яку сторінку

**Приклад:** Натисніть `Ctrl+K`, введіть "пауза" → виберіть "Призупинити всі активні друки"

## Налаштування віджетів

Панель огляду може бути налаштована з вибраними вами віджетами:

**Як редагувати панель:**
1. Натисніть **Редагувати макет** (значок олівця) у верхньому правому куті панелі огляду
2. Перетягніть віджети на бажану позицію
3. Натисніть і перетягніть кут віджета для зміни розміру
4. Натисніть **+ Додати віджет** для додавання нових:

Доступні віджети:

| Віджет | Показує |
|--------|---------|
| Статус принтерів | Картки для всіх принтерів |
| Активний друк (великий) | Детальний перегляд поточного друку |
| Огляд AMS | Усі слоти та рівні філаменту |
| Крива температури | Графік у реальному часі |
| Тариф на електроенергію | Графік цін на наступні 24 години |
| Лічильник філаменту | Загальне споживання за останні 30 днів |
| Ярлик історії | Останні 5 друків |
| Потік камери | Живе зображення камери |

5. Натисніть **Зберегти макет**

:::tip Збережіть кілька макетів
Ви можете мати різні макети для різних цілей — компактний для щоденного використання, великий для розміщення на великому екрані. Перемикайтеся між ними за допомогою вибирача макетів.
:::

## Тема — перемикання між світлою та темною

**Швидке перемикання:**
- Натисніть на значок сонця/місяця у верхньому правому куті навігації
- Або: `Ctrl+K` → введіть "тема"

**Постійне налаштування:**
1. Перейдіть до **Система → Теми**
2. Виберіть між:
   - **Світла** — білий фон
   - **Темна** — темний фон (рекомендовано вночі)
   - **Автоматична** — відповідає системному налаштуванню вашого пристрою
3. Виберіть колір акценту (синій, зелений, фіолетовий тощо)
4. Натисніть **Зберегти**

## Навігація за допомогою клавіатури

Для ефективної навігації без миші:

| Скорочення | Дія |
|-----------|-----|
| `Tab` | Наступний інтерактивний елемент |
| `Shift+Tab` | Попередній елемент |
| `Enter` / `Пробіл` | Активувати кнопку/посилання |
| `Esc` | Закрити модальне вікно/випадаючий список |
| `Ctrl+K` | Палітра команд |
| `Alt+1` – `Alt+9` | Перейти безпосередньо до перших 9 сторінок |

## PWA — встановлення як програми

3DPrintForge можна встановити як прогресивну веб-програму (PWA) і запускати як самостійну програму без меню браузера:

1. Перейдіть до панелі в Chrome, Edge або Safari
2. Натисніть на значок **Встановити програму** в адресному рядку
3. Підтвердіть встановлення

Дивіться [Документацію PWA](../system/pwa) для більш детальної інформації.

## Режим кіоску

Режим кіоску приховує всю навігацію і показує тільки панель — ідеально для виділеного екрана в майстерні:

1. Перейдіть до **Система → Кіоск**
2. Увімкніть **Режим кіоску**
3. Виберіть, які віджети відображатимуться
4. Встановіть інтервал оновлення

Дивіться [Документацію кіоску](../system/kiosk) для повного налаштування.
