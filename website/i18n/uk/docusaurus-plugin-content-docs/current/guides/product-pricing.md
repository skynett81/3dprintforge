# Ціноутворення продуктів — розрахунок ціни продажу

Цей посібник пояснює, як використовувати калькулятор витрат для визначення правильної ціни продажу 3D-друків, які ви продаєте.

## Огляд витрат

Вартість 3D-друку складається з таких компонентів:

| Компонент | Опис | Приклад |
|-----------|-------------|---------|
| **Філамент** | Вартість матеріалу на основі ваги та ціни котушки | 100г × 0,25 kr/г = 25 kr |
| **Відходи** | Втрати матеріалу (очищення, невдалі друки, підтримки) | 10% додатково = 2,50 kr |
| **Електрика** | Споживання електроенергії під час друку | 3,5год × 150W × 1,50 kr/kWh = 0,79 kr |
| **Знос** | Сопло + вартість машини протягом терміну служби | 3,5год × 0,15 kr/год = 0,53 kr |
| **Праця** | Ваш час на налаштування, постобробку, пакування | 10 хв × 200 kr/год = 33,33 kr |
| **Надбавка** | Маржа прибутку | 20% = 12,43 kr |

**Загальна собівартість виробництва** = сума всіх компонентів

## Налаштування параметрів

### Базові налаштування

Перейдіть до **Filament → ⚙ Налаштування** та заповніть:

1. **Ціна електрики (kr/kWh)** — ваша ціна електроенергії. Перевірте рахунок за електрику або використовуйте інтеграцію Nordpool
2. **Потужність принтера (W)** — зазвичай 150W для принтерів Bambu Lab
3. **Вартість машини (kr)** — скільки ви заплатили за принтер
4. **Термін служби машини (години)** — очікуваний термін служби (3000-8000 годин)
5. **Вартість праці (kr/година)** — ваша погодинна ставка
6. **Час підготовки (хв)** — середній час на заміну філаменту, перевірку платформи, пакування
7. **Надбавка (%)** — бажана маржа прибутку
8. **Вартість сопла (kr/година)** — знос сопла (HS01 ≈ 0,05 kr/год)
9. **Фактор відходів** — втрати матеріалу (1,1 = 10% додатково, 1,15 = 15%)

:::tip Типові значення для Bambu Lab
| Налаштування | Хобіст | Напівпрофесіонал | Професіонал |
|---|---|---|---|
| Ціна електрики | 1,50 kr/kWh | 1,50 kr/kWh | 1,00 kr/kWh |
| Потужність принтера | 150W | 150W | 150W |
| Вартість машини | 5 000 kr | 12 000 kr | 25 000 kr |
| Термін служби машини | 3 000год | 5 000год | 8 000год |
| Вартість праці | 0 kr/год | 150 kr/год | 250 kr/год |
| Час підготовки | 5 хв | 10 хв | 15 хв |
| Надбавка | 0% | 30% | 50% |
| Фактор відходів | 1,05 | 1,10 | 1,15 |
:::

## Розрахунок вартості

1. Перейдіть до **Калькулятора витрат** (`https://localhost:3443/#costestimator`)
2. **Перетягніть** файл `.3mf` або `.gcode`
3. Система автоматично зчитує: вагу філаменту, орієнтовний час, кольори
4. **Прив'яжіть котушки** — виберіть, які котушки зі складу використовуються
5. Натисніть **Розрахувати вартість**

### Результат показує:

- **Філамент** — вартість матеріалу за кольором
- **Відходи/втрати** — на основі фактора відходів
- **Електрика** — використовує спотову ціну в реальному часі з Nordpool, якщо доступна
- **Знос** — сопло + вартість машини
- **Праця** — погодинна ставка + час підготовки
- **Собівартість виробництва** — сума всього вищезазначеного
- **Надбавка** — ваша маржа прибутку
- **Загальна вартість** — мінімум, який ви повинні брати
- **Рекомендовані ціни продажу** — маржа 2×, 2,5×, 3×

## Стратегії ціноутворення

### Маржа 2× (рекомендований мінімум)
Покриває собівартість виробництва + непередбачені витрати. Використовуйте для друзів/родини та простої геометрії.

### Маржа 2,5× (стандарт)
Добрий баланс між ціною та цінністю. Працює для більшості продуктів.

### Маржа 3× (преміум)
Для складних моделей, багатокольорових, високої якості або нішевих ринків.

:::warning Не забувайте про приховані витрати
- Невдалі друки (5-15% усіх друків не вдаються)
- Філамент, який не використовується повністю (останні 50г часто проблемні)
- Час, витрачений на обслуговування клієнтів
- Упаковка та доставка
- Обслуговування принтера
:::

## Приклад: Ціноутворення тримача для телефону

| Параметр | Значення |
|-----------|-------|
| Вага філаменту | 45г PLA |
| Час друку | 2 години |
| Спотова ціна | 1,20 kr/kWh |

**Розрахунок:**
- Філамент: 45г × 0,25 kr/г = 11,25 kr
- Відходи (10%): 1,13 kr
- Електрика: 2год × 0,15kW × 1,20 = 0,36 kr
- Знос: 2год × 0,15 = 0,30 kr
- Праця: (2год + 10хв) × 200 kr/год = 433 kr (або 0 для хобі)
- **Собівартість виробництва (хобі)**: ~13 kr
- **Ціна продажу 2,5×**: ~33 kr

## Зберегти кошторис

Натисніть **Зберегти кошторис**, щоб заархівувати розрахунок. Збережені кошториси знаходяться на вкладці **Збережені** в калькуляторі витрат.
