# Спільнотні філаменти

Спільнотні філаменти — це спільна база даних профілів філаменту, наданих користувачами 3DPrintForge з усього світу. Знаходьте найкращі налаштування для вашого філаменту та діліться власним досвідом.

Перейдіть до: **https://localhost:3443/#community**

## Перегляд профілів спільноти

### Пошук

1. Перейдіть до **Спільнотні філаменти**
2. Використовуйте поле пошуку для пошуку за:
   - Матеріалом (PLA, PETG, ABS, PA, TPU тощо)
   - Постачальником (PolyMaker, Bambu Lab, Prusament, Sunlu тощо)
   - Назвою кольору або hex-кодом кольору
   - Властивістю (прозорий, матовий, шовковий, глітер тощо)
3. Фільтруйте за **Рейтингом** (тільки найвищі)
4. Фільтруйте за **Моделлю принтера** для профілів, оптимізованих для вашого принтера

### Картки профілів

Кожна картка профілю показує:

| Поле | Приклад |
|---|---|
| Назва філаменту | PolyTerra PLA – Matte Forest Green |
| Матеріал | PLA |
| Постачальник | PolyMaker |
| Автор | ім'я користувача |
| Рейтинг | ⭐⭐⭐⭐½ (4.6 / 5) |
| Кількість тестів | 23 |
| Рекомендовані температури | Сопло: 215°C / Стіл: 55°C |
| Рекомендована швидкість | Стандартна |

Натисніть на картку для перегляду повного профілю з усіма параметрами та коментарями користувачів.

## Імпорт профілю

1. Відкрийте профіль зі спільноти
2. Натисніть **Імпортувати до сховища**
3. Виберіть:
   - **Прив'язати до існуючої котушки** — профіль прив'язується до зареєстрованої котушки
   - **Створити нову котушку** — створює новий запис котушки з даними профілю
4. За потреби відрегулюйте температуру або інші налаштування
5. Натисніть **Імпортувати**

Профіль тепер доступний у вашому сховищі філаменту.

## Внесок власних профілів

### Експорт та поширення

1. Перейдіть до **Сховище філаменту**
2. Знайдіть котушку, якою хочете поділитися
3. Натисніть **Поділитися зі спільнотою**
4. Заповніть:
   - **Назва постачальника** — точна назва постачальника
   - **Назва продукту** — точна назва продукту з упаковки
   - **Назва кольору** — офіційна назва кольору та hex-код
   - **Коментар** — ваш досвід, поради та хитрощі
   - **Модель принтера** — на якому принтері ви тестували
5. Натисніть **Надіслати**

Профіль перебуває на розгляді та зазвичай публікується протягом 24 годин.

### Вимоги до внеску

:::warning Вимоги до якості
- Вказуйте лише реальні, протестовані налаштування — не спекулятивні значення
- Використовуйте точні назви постачальника та продукту (перевірте упаковку)
- Один профіль на конкретний продукт та модель принтера
- Уникайте дублікатів — перевірте, чи профіль вже існує
:::

## Оцінювання профілів

Допоможіть спільноті, оцінивши профілі, які ви тестували:

1. Відкрийте профіль спільноти
2. Натисніть **Поставити оцінку**
3. Вкажіть кількість зірок (1–5) та короткий коментар
4. Натисніть **Надіслати**

Профілі з високим рейтингом відображаються вгорі в результатах пошуку.

## Офлайн-використання

База спільноти синхронізується локально автоматично (оновлюється щодня). Всі профілі доступні без підключення до інтернету після першої синхронізації.

Завантажте базу вручну: **Спільнотні філаменти → Налаштування → Синхронізувати зараз**
