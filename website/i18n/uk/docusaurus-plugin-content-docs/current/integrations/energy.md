# Ціна на електроенергію

Інтеграція ціни на електроенергію отримує актуальні ціни від Tibber або Nordpool для точного розрахунку вартості електроенергії на кожний друк та сповіщень про вигідні або невигідні часи для друку.

Перейдіть до: **https://localhost:3443/#settings** → **Інтеграції → Ціна електроенергії**

## Інтеграція Tibber

Tibber — це постачальник електроенергії з відкритим API для спот-цін.

### Налаштування

1. Увійдіть на [developer.tibber.com](https://developer.tibber.com)
2. Згенеруйте **Personal Access Token**
3. У Bambu Dashboard: вставте токен у поле **Tibber API Token**
4. Виберіть **Будинок** (звідки отримувати ціни, якщо у вас кілька будинків)
5. Натисніть **Тест з'єднання**
6. Натисніть **Зберегти**

### Доступні дані від Tibber

- **Споткціна зараз** — поточна ціна з урахуванням податків (грн/кВт·год)
- **Ціни на наступні 24 години** — Tibber надає завтрашні ціни приблизно о 13:00
- **Історія цін** — до 30 днів назад
- **Вартість на друк** — розрахована на основі фактичного часу друку × погодинних цін

## Інтеграція Nordpool

Nordpool — це енергетична біржа, що надає сирі споткціни для Північної Європи.

### Налаштування

1. Перейдіть до **Інтеграції → Nordpool**
2. Виберіть **Цінову зону**: NO1 (Осло) / NO2 (Крістіансанн) / NO3 (Тронгейм) / NO4 (Тромсе) / NO5 (Берген)
3. Виберіть **Валюту**: NOK / EUR
4. Виберіть **Податки та збори**:
   - Увімкніть **Включити ПДВ** (25 %)
   - Заповніть **Тариф мережі** (грн/кВт·год) — дивіться рахунок від постачальника мережі
   - Заповніть **Податок на споживання** (Elavgift, грн/кВт·год)
5. Натисніть **Зберегти**

:::info Тариф мережі
Тариф мережі варіюється залежно від постачальника та моделі ціноутворення. Перевірте свій останній рахунок за електроенергію для правильної ставки.
:::

## Погодинні ціни

Погодинні ціни відображаються у вигляді стовпчастої діаграми на наступні 24–48 годин:

- **Зелений** — дешеві години (нижче середнього)
- **Жовтий** — середня ціна
- **Червоний** — дорогі години (вище середнього)
- **Сірий** — години без доступного прогнозу цін

Наведіть курсор на годину для перегляду точної ціни (грн/кВт·год).

## Історія цін

Перейдіть до **Ціна електроенергії → Історія** для перегляду:

- Середньої добової ціни за останні 30 днів
- Найдорожчої та найдешевшої години на день
- Загальної вартості електроенергії для друків на день

## Цінові сповіщення

Налаштуйте автоматичні сповіщення на основі ціни електроенергії:

1. Перейдіть до **Ціна електроенергії → Цінові сповіщення**
2. Натисніть **Нове сповіщення**
3. Виберіть тип сповіщення:
   - **Ціна нижче порогу** — сповістити, коли ціна впаде нижче X грн/кВт·год
   - **Ціна вище порогу** — сповістити, коли ціна підніметься вище X грн/кВт·год
   - **Найдешевша година сьогодні** — сповістити, коли починається найдешевша година дня
4. Виберіть канал сповіщення
5. Натисніть **Зберегти**

:::tip Розумне планування
Поєднайте цінові сповіщення з чергою друку: налаштуйте автоматизацію, яка автоматично надсилає завдання черги, коли ціна на електроенергію низька (потрібна інтеграція вебхуку або Home Assistant).
:::

## Ціна електроенергії в калькуляторі витрат

Активована інтеграція ціни електроенергії забезпечує точні витрати на електроенергію в [Калькуляторі витрат](../analytics/costestimator). Виберіть **Актуальна ціна** замість фіксованої для використання поточної ціни Tibber/Nordpool.
