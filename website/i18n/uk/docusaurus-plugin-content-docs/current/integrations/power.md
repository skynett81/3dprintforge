# Вимірювання електроенергії

Підключіть розумну розетку з вимірюванням енергії до принтера для реєстрації фактичного споживання електроенергії на кожний друк — не лише оцінок.

Перейдіть до: **https://localhost:3443/#settings** → **Інтеграції → Вимірювання електроенергії**

## Підтримувані пристрої

| Пристрій | Протокол | Рекомендація |
|---|---|---|
| **Shelly Plug S / Plus Plug S** | HTTP REST / MQTT | Рекомендовано — просте налаштування |
| **Shelly 1PM / 2PM** | HTTP REST / MQTT | Для стаціонарної установки |
| **Shelly Gen2 / Gen3** | HTTP REST / MQTT | Новіші моделі з розширеним API |
| **Пристрої Tasmota** | MQTT | Гнучко для власних конфігурацій |

:::tip Рекомендований пристрій
Shelly Plug S Plus з прошивкою 1.0+ протестований та рекомендований. Підтримує Wi-Fi, MQTT та HTTP REST без залежності від хмари.
:::

## Налаштування з Shelly

### Передумови

- Розетка Shelly підключена до тієї ж мережі, що й Bambu Dashboard
- Shelly налаштований зі статичним IP або резервуванням DHCP

### Конфігурація

1. Перейдіть до **Налаштування → Вимірювання електроенергії**
2. Натисніть **Додати лічильник електроенергії**
3. Виберіть **Тип**: Shelly
4. Заповніть:
   - **IP-адреса**: наприклад `192.168.1.150`
   - **Канал**: 0 (для одинарних розеток)
   - **Автентифікація**: ім'я користувача та пароль, якщо налаштовано
5. Натисніть **Тест з'єднання**
6. Прив'яжіть розетку до **Принтера**: виберіть зі списку
7. Натисніть **Зберегти**

### Інтервал опитування

Стандартний інтервал опитування — 10 секунд. Зменшіть до 5 для більш точних вимірювань, збільшіть до 30 для зниження навантаження на мережу.

## Налаштування з Tasmota

1. Налаштуйте пристрій Tasmota з MQTT (дивіться документацію Tasmota)
2. У Bambu Dashboard: виберіть **Тип**: Tasmota
3. Заповніть MQTT-топік для пристрою: наприклад `tasmota/power-plug-1`
4. Прив'яжіть до принтера та натисніть **Зберегти**

Bambu Dashboard автоматично підписується на `{topic}/SENSOR` для вимірювань потужності.

## Що вимірюється

Коли вимірювання електроенергії увімкнено, наступні дані реєструються для кожного друку:

| Показник | Опис |
|---|---|
| **Миттєва потужність** | Ватти під час друку (в реальному часі) |
| **Загальне споживання енергії** | кВт·год за весь друк |
| **Середня потужність** | кВт·год / час друку |
| **Вартість електроенергії** | кВт·год × ціна (від Tibber/Nordpool) |

Дані зберігаються в історії друку та доступні для аналізу.

## Вигляд в реальному часі

Миттєве споживання потужності відображається в:

- **Дашборді** — як додатковий віджет (увімкніть у налаштуваннях віджетів)
- **Огляді парку** — як невеликий індикатор на картці принтера

## Порівняння з оцінкою

Після друку відображається порівняння:

| | Оцінка | Факт |
|---|---|---|
| Споживання енергії | 1.17 кВт·год | 1.09 кВт·год |
| Вартість електроенергії | 2.16 грн | 2.02 грн |
| Відхилення | — | -6.8 % |

Постійне відхилення можна використати для калібрування оцінок у [Калькуляторі витрат](../analytics/costestimator).

## Автоматичне вимкнення принтера

Shelly/Tasmota може автоматично вимикати принтер після завершення друку:

1. Перейдіть до **Вимірювання електроенергії → [Принтер] → Автоматичне вимкнення**
2. Увімкніть **Вимкнути через X хвилин після завершення друку**
3. Встановіть затримку (наприклад, 10 хвилин)

:::danger Охолодження
Дайте принтеру охолонути щонайменше 5–10 хвилин після завершення друку перед відключенням живлення. Сопло повинно охолонути нижче 50°C, щоб уникнути теплового повзання в hotend.
:::
