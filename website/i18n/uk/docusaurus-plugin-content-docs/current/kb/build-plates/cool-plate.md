# Cool Plate (Smooth PEI)

Cool Plate — це гладка PEI-платформа, яка дає блискучу рівну нижню поверхню на друках. Вона включена як стандартна платформа з більшістю принтерів Bambu Lab.

## Характеристики

- **Поверхня:** Гладка, блискуча нижня поверхня на друку
- **Матеріал:** PEI (Polyetherimide) покриття на пружній сталевій пластині
- **Макс. температура столу:** 90 °C

## Найкраще підходить для

- PLA (всі варіанти) — без клейового олівця
- PLA+ — без клейового олівця
- PLA Silk — без клейового олівця
- PLA-CF — без клейового олівця

## Використовуйте з обережністю

:::danger PETG — завжди використовуйте клейовий олівець
PETG прилипає **надзвичайно сильно** до гладкої PEI. Без клейового олівця ви ризикуєте відірвати PEI-покриття від платформи при видаленні друку. Завжди використовуйте клейовий олівець як бар'єр або виберіть Engineering Plate.
:::

## Обслуговування

### Після кожного друку
- Дайте платформі охолонути до кімнатної температури (40 °C або нижче) перед видаленням друку
- PLA відпускає платформу сам по собі при 25–30 °C
- Злегка зігніть платформу для звільнення впертих друків

### Щотижневе очищення
Протріть платформу ізопропанолом (IPA, 70–99%):
```
1. Нанесіть IPA на чистий безворсовий папір (не розпилюйте безпосередньо на платформу)
2. Протріть поверхню в одному напрямку
3. Дайте повністю висохнути (30 секунд) перед наступним друком
```

### При поганій адгезії
1. Вимийте теплою водою та м'яким миючим засобом
2. Добре прополощіть
3. Витріть безворсовим папером
4. Наприкінці протріть IPA

:::warning Не використовуйте ацетон
Ацетон розчиняє PEI-покриття та руйнує платформу назавжди. Використовуйте лише IPA або воду з милом.
:::

## Заміна платформи

Платформа зношена, коли:
- PEI-покриття помітно пошкоджене (подряпини, рубці, лущення)
- Друки більше не прилипають навіть після ретельного очищення
- Поверхня нерівна і дає поганий перший шар

Cool Plate зазвичай служить 200–500 друків при нормальному використанні та гарному обслуговуванні.
