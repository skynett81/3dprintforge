# Engineering Plate (Textured PEI)

Engineering Plate — це текстурована PEI-платформа, розроблена для технічних матеріалів, таких як PETG, ABS, ASA та PA. Вона дає гарну текстуровану нижню поверхню та є найбільш універсальною платформою в асортименті Bambu Lab.

## Характеристики

- **Поверхня:** Текстурована PEI — дає текстуровану/матову нижню поверхню на друку
- **Адгезія:** Краща механічна адгезія, ніж у гладкої PEI
- **Макс. температура столу:** 110 °C
- **Видалення:** Друки легко відпускаються після охолодження до 40–50 °C

## Найкраще підходить для

- PETG та PETG-CF (без клейового олівця, або з ним для додаткової надійності)
- ABS та ASA (з клейовим олівцем)
- PA6 / PA12 / PA-CF (з клейовим олівцем — обов'язково)
- TPU (без клейового олівця)
- PLA (працює добре, дає текстуровану нижню поверхню)

## Не підходить для

- Друків, що потребують гладкої нижньої поверхні (використовуйте Cool Plate)
- PC при високих температурах (використовуйте High Temp Plate)

## Обслуговування

### Очищення
```bash
# Щотижнево (або після використання клейового олівця):
1. Вимийте теплою водою та м'яким миючим засобом
2. Добре прополощіть
3. Дайте висохнути, або витріть безворсовим папером
4. Завершальне протирання IPA

# Між друками без клейового олівця:
1. Протріть IPA та безворсовим папером
```

### Після клейового олівця
Завжди мийте платформу після використання клейового олівця. Залишки клею можуть давати нерівну адгезію для наступного друку.

:::tip Напрямок текстури
Engineering Plate має напрямок текстури. Завжди монтуйте платформу з кліпсами у правильній орієнтації, щоб уникнути застрягання друку у заглибинах текстури.
:::

## Заміна платформи

Engineering Plate зношена, коли:
- Текстура витерта/гладка в центральній зоні (найбільш використовуваній)
- Друки не прилипають навіть після очищення та клейового олівця
- Платформа має глибокі подряпини або деформації

Нормальний строк служби: 300–700 друків при гарному обслуговуванні.
