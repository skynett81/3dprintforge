# High Temp Plate

High Temp Plate розроблена для матеріалів, що потребують високої температури столу. Вона витримує до 120 °C та необхідна для полікарбонату (PC) та високотемпературних варіантів PA.

## Характеристики

- **Поверхня:** Smooth (гладка) з високотемпературним покриттям
- **Макс. температура столу:** 120 °C
- **Нижня поверхня друку:** Блискуча/гладка
- **Адгезія:** Хороша при високій температурі — відпускає при охолодженні

## Найкраще підходить для

| Матеріал | Температура столу | Клейовий олівець |
|-----------|---------------|---------|
| ABS | 90–110 °C | Так |
| ASA | 90–110 °C | Так |
| PC | 100–120 °C | Так (обов'язково) |
| PA6 | 80–90 °C | Так |
| PA12 | 70–85 °C | Так |
| PA-CF | 90–100 °C | Так |

## Не підходить для

- PLA — платформа занадто гаряча, PLA прилипає занадто сильно
- PETG — ризик пошкодження платформи без клейового олівця
- TPU — Engineering Plate підходить краще

## Клейовий олівець

Клейовий олівець настійно рекомендується (і для PC обов'язковий) для:
1. Покращення адгезії першого шару
2. Захисту покриття платформи
3. Полегшення видалення друку після охолодження

Нанесіть тонкий рівний шар. Для PC та PA: використовуйте трохи більше, ніж для ABS.

:::warning PC потребує особливої обережності
Полікарбонат (PC) друкується при температурі сопла 250–280 °C та столу 100–120 °C. Без закритого корпусу (камери) та без High Temp Plate відсоток успіху дуже низький. Тільки X1C та P1S рекомендуються для друку PC.
:::

## Обслуговування

```bash
# Після PC/ABS з клейовим олівцем:
1. Дайте платформі охолонути до 50 °C
2. Обережно видаліть друк (зігніть платформу)
3. Вимийте теплою водою та миючим засобом
4. Добре висушіть — вода на гарячій платформі може пошкодити покриття

# Між друками:
1. Протріть IPA
2. Перевірте покриття на знос
```

:::danger Уникайте теплового удару
Ніколи не кладіть холодну платформу на гарячий стіл і не розпилюйте холодний IPA на гарячу платформу. Різниця температур може пошкодити покриття або спричинити деформацію сталевої пластини.
:::

## Строк служби

High Temp Plate служить менше, ніж Engineering Plate через вищі теплові навантаження:
- **Нормальне використання (ABS/ASA):** 200–400 друків
- **Інтенсивне використання (PC):** 100–200 друків

Замінюйте при видимому зносі покриття або постійних проблемах з адгезією.
