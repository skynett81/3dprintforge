# Textured PEI Plate

Textured PEI Plate (не плутати з Engineering Plate) має грубішу текстуру та особливо підходить для TPU та інших гнучких матеріалів, а також для широкого спектру стандартних матеріалів.

## Характеристики

- **Поверхня:** Груба PEI-текстура — дає виразну текстуровану нижню поверхню
- **Адгезія:** Відмінне механічне зчеплення, особливо для TPU
- **Видалення:** Дуже легке після охолодження — зігніть платформу і друки відпустяться
- **Макс. температура столу:** 100 °C

## Найкраще підходить для

- **TPU** — це найкраща платформа для гнучких матеріалів
- PLA та PLA+ — хороша адгезія, гарна текстурована нижня поверхня
- PETG — з невеликою кількістю клейового олівця
- PLA-CF — без клейового олівця

## Сумісність

| Матеріал | Придатність | Клейовий олівець |
|-----------|---------|---------|
| TPU | ★★★★★ | Ні |
| PLA | ★★★★★ | Ні |
| PLA+ | ★★★★★ | Ні |
| PETG | ★★★★ | Небагато (рекомендовано) |
| PLA-CF | ★★★★ | Ні |
| ABS | ★★★ | Так |
| PA12 | ★★★ | Так |

## TPU на Textured PEI

TPU ідеально прилипає до Textured PEI без клейового олівця:
1. Встановіть температуру столу 30–45 °C
2. Ретельно відкалібруйте Z-зміщення — трохи вище, ніж для PLA
3. Дайте столу охолонути до нижче 30 °C після друку
4. Злегка зігніть платформу — TPU відпускається без зусиль

:::tip Холодне видалення
TPU відмінно відпускає Textured PEI при охолодженні. Якщо у вас виникають проблеми з видаленням друку, почекайте, поки платформа повністю охолоне (20–25 °C) та зігніть сильніше.
:::

## Обслуговування

Textured PEI міцна та проста в обслуговуванні:

```bash
# Між друками:
- Протріть IPA та безворсовим паперовим рушником

# Щомісяця (або при поганій адгезії):
- Вимийте теплою водою та м'яким миючим засобом
- Прополощіть та добре висушіть
- Завершіть обробкою IPA
```

## Поверхня готового друку

Груба текстура дає виразну текстуровану нижню сторону на всіх друках. Це:
- Бажано для прототипів та функціональних деталей (краще зчеплення)
- Небажано для декоративних/візуальних друків (використовуйте Cool Plate для гладкої поверхні)
