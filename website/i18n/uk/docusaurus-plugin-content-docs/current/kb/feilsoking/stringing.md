# Stringing

Stringing (або "oozing") — це тонкі пластикові нитки, що утворюються між окремими частинами об'єкта, коли сопло переміщується без екструзії. Це надає деталі вигляд "павутини".

## Причини stringing

1. **Занадто висока температура сопла** — гарячий пластик рідкий і капає
2. **Погані налаштування ретракції** — філамент не відтягується достатньо швидко
3. **Вологий філамент** — вологість спричиняє пару та додаткову течію
4. **Занадто низька швидкість** — сопло довго перебуває в транзитних позиціях

## Діагностика

**Вологий філамент?** Чи чуєте ви тріск або хлопання під час друку? Тоді філамент вологий — спочатку висушіть його, перш ніж налаштовувати інші параметри.

**Занадто висока температура?** Чи бачите ви краплі з сопла в "паузні" моменти? Знизьте температуру на 5–10 °C.

## Рішення

### 1. Висушіть філамент

Вологий філамент — найпоширеніша причина stringing, яку неможливо усунути лише налаштуваннями:

| Матеріал | Температура сушіння | Час |
|----------|---------------------|-----|
| PLA | 45–55 °C | 4–6 годин |
| PETG | 60–65 °C | 6–8 годин |
| TPU | 55–60 °C | 6–8 годин |
| PA | 75–85 °C | 8–12 годин |

### 2. Знизьте температуру сопла

Починайте зі зниження по 5 °C за раз:
- PLA: спробуйте 210–215 °C (від 220 °C)
- PETG: спробуйте 235–240 °C (від 245 °C)

:::warning Занадто низька температура погіршує злиття шарів
Знижуйте температуру обережно. Занадто низька температура дає погане злиття шарів, слабкий друк та проблеми з екструзією.
:::

### 3. Налаштуйте ретракцію

Ретракція відтягує філамент назад у сопло під час "transit"-руху для запобігання краплям:

```
Bambu Studio → Філамент → Ретракція:
- Відстань ретракції: 0.4–1.0 мм (direct drive)
- Швидкість ретракції: 30–45 мм/с
```

:::tip Принтери Bambu Lab мають direct drive
Всі принтери Bambu Lab (X1C, P1S, A1) використовують екструдер з прямою подачею. Direct drive потребує **коротшої** відстані ретракції, ніж системи Bowden (типово 0.5–1.5 мм проти 3–7 мм).
:::

### 4. Збільшіть швидкість переміщення

Швидкий рух між точками дає соплу менше часу для краплів:
- Збільшіть "travel speed" до 200–300 мм/с
- Принтери Bambu Lab добре справляються з цим

### 5. Активуйте "Avoid Crossing Perimeters"

Налаштування слайсера, яке змушує сопло уникати переміщень через відкриті ділянки, де stringing буде помітний:
```
Bambu Studio → Якість → Avoid crossing perimeters
```

### 6. Знизьте швидкість (для TPU)

Для TPU рішення протилежне, ніж для інших матеріалів:
- Знизьте швидкість друку до 20–35 мм/с
- TPU еластичний і стискається при занадто великій швидкості — це спричиняє "витікання"

## Після налаштувань

Перевірте за допомогою стандартної тестової моделі для stringing (наприклад, "torture tower" з MakerWorld). Змінюйте по одній змінній за раз та спостерігайте за результатом.

:::note Ідеальний результат рідко можливий
Певний stringing є нормальним для більшості матеріалів. Зосередьтеся на зменшенні до прийнятного рівня, а не на повному усуненні. PETG завжди матиме трохи більше stringing, ніж PLA.
:::
