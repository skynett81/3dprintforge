# ABS

ABS (Acrylonitrile Butadiene Styrene) — термопласт з хорошою термостабільністю та ударостійкістю. Потребує закритого корпусу та більш вибагливий, ніж PLA/PETG, але дає міцні функціональні деталі.

## Налаштування

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 240–260 °C |
| Температура столу | 90–110 °C |
| Температура камери | 45–55 °C (X1C/P1S) |
| Охолодження деталей | 0–20% |
| Aux вентилятор | 0% |
| Швидкість | 80–100% |
| Сушіння | Рекомендовано (4–6 год при 70 °C) |

## Рекомендовані платформи друку

| Платформа | Придатність | Клейовий олівець? |
|-------|---------|----------|
| Engineering Plate (Textured PEI) | Відмінно | Так (рекомендовано) |
| High Temp Plate | Відмінно | Так |
| Cool Plate (Smooth PEI) | Уникайте | — |
| Textured PEI | Добре | Так |

:::tip Клейовий олівець для ABS
Завжди використовуйте клейовий олівець на Engineering Plate для ABS. Він покращує адгезію та полегшує відокремлення друку без пошкодження платформи.
:::

## Закритий корпус (камера)

ABS **потребує** закритої камери для запобігання warping:

- **X1C та P1S:** Вбудована камера з активним керуванням температурою — ідеально для ABS
- **P1P:** Частково відкритий — додайте верхні кришки для кращих результатів
- **A1 / A1 Mini:** Відкритий CoreXY — **не рекомендується** для ABS без власноруч зробленого корпусу

Тримайте камеру закритою протягом усього друку. Не відкривайте для перевірки — якщо почекаєте до охолодження, уникнете warping при вийманні.

## Warping

ABS дуже схильний до warping (кути піднімаються):

- **Підвищіть температуру столу** — спробуйте 105–110 °C
- **Використовуйте brim** — 5–10 мм brim в Bambu Studio
- **Уникайте протягів** — закрийте всі потоки повітря навколо принтера
- **Знизьте охолодження деталей до 0%** — охолодження викликає скручування

:::warning Пари
ABS виділяє парни стиролу під час друку. Забезпечте хорошу вентиляцію в кімнаті або використовуйте HEPA/фільтр з активованим вугіллям. Bambu P1S має вбудований фільтр.
:::

## Постобробка

ABS можна шліфувати, фарбувати та склеювати простіше, ніж PETG та PLA. Його також можна згладжувати ацетонними парами для гладкої поверхні — але будьте дуже обережні з впливом ацетону.

## Зберігання

Сушіть при **70 °C протягом 4–6 годин** перед друком. Зберігайте в запечатаній коробці — ABS поглинає вологу, що дає тріскаючі звуки та слабкі шари.
