# Матриця сумісності

Ця сторінка надає повний огляд того, які матеріали працюють з якими робочими пластинами, принтерами та типами сопел. Використовуйте таблиці як довідник при плануванні друку з новими матеріалами.

---

| PVA | Відмінно | Добре | Не рекомендовано | Добре | Ні |
| HIPS | Не рекомендовано | Добре | Добре | Прийнятно | Ні |
| PVB | Добре | Добре | Не рекомендовано | Добре | Ні |

**Пояснення:**
- **Відмінно** — працює оптимально, рекомендована комбінація
- **Добре** — працює добре, прийнятна альтернатива
- **Прийнятно** — працює, але не ідеально — потрібні додаткові заходи
- **Погано** — може працювати з модифікаціями, але не рекомендовано
- **Не рекомендовано** — погані результати або ризик пошкодження пластини

:::tip PETG та Cool Plate
PETG **занадто сильно** прилипає до Cool Plate (Smooth PEI) і може зірвати PEI-покриття при знятті деталі. Завжди використовуйте клей як розділову плівку або обирайте Engineering Plate.
:::

:::warning PC та вибір пластини
PC потребує High Temp Plate через високі температури столу (100–120 °C). Інші пластини можуть бути постійно деформовані при цих температурах.
:::

**Пояснення:**
- **Так** — повна підтримка, рекомендовано
- **Так*** — потрібне загартоване сопло (HS01 або аналог)
- **Можливо**** — може працювати з обмеженнями, офіційно не рекомендовано
- **Ні** — не підходить (немає камери, недостатня температура тощо)

:::danger Вимоги до камери
Матеріали, що потребують камери (ABS, ASA, PA, PC):
- **A1 та A1 Mini** мають відкриту раму — не підходять
- **P1P** має відкриту раму — потрібен аксесуар камери
- **P1S** має камеру, але без активного нагріву
- **X1C та X1E** мають повну камеру з активним нагрівом — рекомендуються для вимогливих матеріалів
:::

:::danger Вуглецеве та скловолокно потребують загартованого сопла
Усі матеріали з **-CF** (вуглецеве волокно) або **-GF** (скловолокно) **потребують сопла з загартованої сталі**. Латунь зношується за години-дні з цими матеріалами. Рекомендується Bambu Lab HS01.

Матеріали, що потребують загартованого сопла:
- PLA-CF
- PETG-CF
- PA-CF / PA-GF
- PC-CF / PC-GF
:::

:::tip Латунь vs загартована сталь для звичайних матеріалів
Латунне сопло забезпечує **кращу теплопровідність** і, відповідно, рівнішу екструзію для звичайних матеріалів (PLA, PETG, ABS). Загартована сталь працює добре, але може потребувати на 5–10 °C вищу температуру. Використовуйте латунь для щоденного використання та переходьте на загартовану сталь для CF/GF-матеріалів.
:::

## Поради щодо зміни матеріалу

При зміні матеріалів в AMS або вручну правильне прочищення важливе для уникнення забруднення.

### Рекомендований об'єм прочищення

| Перехід | Об'єм прочищення | Примітка |
|-----------|-------------|---------|
| PLA → PLA (інший колір) | 100–150 мм³ | Стандартна зміна кольору |
| PLA → PETG | 200–300 мм³ | Підвищення температури, інша текучість |
| PETG → PLA | 200–300 мм³ | Зниження температури |
| ABS → PLA | 300–400 мм³ | Велика різниця температур |
| PLA → ABS | 300–400 мм³ | Велика різниця температур |
| PA → PLA | 400–500 мм³ | Нейлон залишається в хотенді |
| PC → PLA | 400–500 мм³ | PC потребує ретельного прочищення |
| Темний → Світлий колір | 200–300 мм³ | Темний пігмент важко вимити |
| Світлий → Темний колір | 100–150 мм³ | Легший перехід |

### Зміна температури при зміні матеріалу

| Перехід | Рекомендація |
|----------|-----------|
| Холодний → Гарячий (напр. PLA → ABS) | Нагрійте до нового матеріалу, ретельно прочистіть |
| Гарячий → Холодний (напр. ABS → PLA) | Спочатку прочистіть при високій температурі, потім знизьте |
| Подібні температури (напр. PLA → PLA) | Стандартне прочищення |
| Велика різниця (напр. PLA → PC) | Проміжний крок з PETG може допомогти |

:::warning Нейлон та PC залишають залишки
PA (Нейлон) та PC особливо важко прочищувати. Після використання цих матеріалів:
1. Прочистіть **PETG** або **ABS** при високій температурі (260–280 °C)
2. Використайте мінімум **500 мм³** матеріалу для прочищення
3. Візуально перевірте екструзію — вона має бути повністю чистою без зміни кольору
:::

## Швидка довідка — вибір матеріалу

Не впевнені, який матеріал потрібен? Використовуйте цей посібник:

| Потреба | Рекомендований матеріал |
|-------|-------------------|
| Прототипування / щоденне використання | PLA |
| Механічна міцність | PETG, PLA Tough |
| Зовнішнє використання | ASA |
| Термостійкість | ABS, ASA, PC |
| Гнучкі деталі | TPU |
| Максимальна міцність | PA-CF, PC-CF |
| Прозорий | PETG (натуральний), PC (натуральний) |
| Естетика / декор | PLA Silk, PLA Sparkle |
| Защіпки / живі петлі | PETG, PA |
| Контакт з їжею | PLA (з застереженнями) |
