# ASA

ASA (Acrylonitrile Styrene Acrylate) — це UV-стійкий варіант ABS, спеціально розроблений для зовнішнього використання. Матеріал поєднує міцність і жорсткість ABS зі значно кращою стійкістю до UV-випромінювання, старіння та впливу погодних умов.

## Налаштування

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 240–260 °C |
| Температура столу | 90–110 °C |
| Температура камери | 40–50 °C (рекомендовано) |
| Охолодження деталі | 30–50% |
| Швидкість | 80–100% |
| Потрібне сушіння | Так |

## Рекомендовані робочі пластини

| Пластина | Придатність | Клей? |
|-------|---------|----------|
| Engineering Plate | Відмінно | Ні |
| High Temp Plate | Добре | Так |
| Textured PEI | Прийнятно | Так |
| Cool Plate (Smooth PEI) | Не рекомендовано | — |

:::tip Engineering Plate найкраща для ASA
Engineering Plate забезпечує найнадійніше зчеплення для ASA без клею. Пластина витримує високі температури столу та забезпечує хорошу адгезію без постійного прилипання деталі.
:::

## Вимоги до принтера

ASA потребує **закритої камери (enclosure)** для найкращих результатів. Без закритої камери виникають:

- **Деформація (warping)** — кути піднімаються від робочої пластини
- **Розшарування** — погане зчеплення між шарами
- **Тріщини на поверхні** — видимі тріщини вздовж друку

| Принтер | Підходить для ASA? | Примітка |
|---------|---------------|---------|
| X1C | Відмінно | Повністю закритий, активний нагрів |
| X1E | Відмінно | Повністю закритий, активний нагрів |
| P1S | Добре | Закритий, пасивний нагрів |
| P1P | Можливо з додатком | Потрібен аксесуар камери |
| A1 | Не рекомендовано | Відкрита рама |
| A1 Mini | Не рекомендовано | Відкрита рама |

## ASA vs ABS — порівняння

| Властивість | ASA | ABS |
|----------|-----|-----|
| UV-стійкість | Відмінна | Погана |
| Зовнішнє використання | Так | Ні (жовтіє та стає крихким) |
| Деформація | Помірна | Висока |
| Поверхня | Матова, рівна | Матова, рівна |
| Хімічна стійкість | Добра | Добра |
| Ціна | Трохи вища | Нижча |
| Запах під час друку | Помірний | Сильний |
| Ударна міцність | Добра | Добра |
| Температуростійкість | ~95–105 °C | ~95–105 °C |

:::warning Вентиляція
ASA виділяє гази під час друку, які можуть бути подразнюючими. Друкуйте у добре провітрюваному приміщенні або з системою фільтрації повітря. Не друкуйте ASA у приміщенні, де ви перебуваєте тривалий час без вентиляції.
:::

## Сушіння

ASA має **середню гігроскопічність** та поглинає вологу з повітря з часом.

| Параметр | Значення |
|-----------|-------|
| Температура сушіння | 65 °C |
| Час сушіння | 4–6 годин |
| Рівень гігроскопічності | Середній |
| Ознаки вологи | Потріскування, бульбашки, погана поверхня |

- Зберігайте у герметичному пакеті з силікагелем після відкриття
- AMS з осушувачем достатній для короткострокового зберігання
- Для тривалого зберігання: використовуйте вакуумний пакет або сушильну коробку для філаменту

## Сфери застосування

ASA — переважний матеріал для всього, що використовується **на вулиці**:

- **Автомобільні компоненти** — корпуси дзеркал, деталі панелі приладів, ковпачки клапанів
- **Садові інструменти** — кліпси, затискачі, деталі садових меблів
- **Зовнішні вивіски** — таблички, літери, логотипи
- **Деталі дронів** — шасі, кріплення камер
- **Кріплення сонячних панелей** — кронштейни та кутники
- **Деталі поштової скриньки** — механізми та декорації

## Поради для успішного друку ASA

### Брім та адгезія

- **Використовуйте брім** для великих деталей та деталей з малою площею контакту
- Брім 5–8 мм ефективно запобігає деформації
- Для менших деталей можна спробувати без бріму, але тримайте напоготові як запасний варіант

### Уникайте протягів

- **Закрийте всі двері та вікна** у кімнаті під час друку
- Протяги та холодне повітря — найгірші вороги ASA
- Не відкривайте дверцята камери під час друку

### Стабільність температури

- Дайте камері прогрітися **10–15 хвилин** перед початком друку
- Стабільна температура камери дає рівніші результати
- Уникайте розміщення принтера біля вікон або вентиляційних отворів

### Охолодження

- ASA потребує **обмеженого охолодження деталі** — 30–50% є типовим
- Для навісів і мостів можна збільшити до 60–70%, але очікуйте деяке розшарування
- Для механічних деталей: зменшіть охолодження, надаючи перевагу зчепленню шарів над деталізацією

:::tip Вперше працюєте з ASA?
Почніть з маленької тестової деталі (наприклад, куб 30 мм), щоб відкалібрувати налаштування. ASA поводиться дуже схоже на ABS, але з трохи меншою схильністю до деформації. Якщо у вас є досвід з ABS, ASA здасться покращенням.
:::

## Усадка

ASA усідає більше, ніж PLA та PETG, але зазвичай трохи менше, ніж ABS:

| Матеріал | Усадка |
|-----------|----------|
| PLA | ~0.3–0.5% |
| PETG | ~0.3–0.6% |
| ASA | ~0.5–0.7% |
| ABS | ~0.7–0.8% |

Для деталей з жорсткими допусками: компенсуйте 0.5–0.7% у слайсері або спочатку протестуйте з пробними деталями.

## Постобробка

- **Згладжування ацетоном** — ASA можна згладити парами ацетону, як і ABS
- **Шліфування** — добре шліфується наждачним папером 200–400
- **Склеювання** — ціаноакрилатний клей або ацетонове склеювання працює відмінно
- **Фарбування** — добре приймає фарбу після легкого шліфування

:::danger Поводження з ацетоном
Ацетон є легкозаймистим і виділяє токсичні гази. Завжди використовуйте у добре провітрюваному приміщенні, уникайте відкритого вогню та використовуйте засоби захисту (рукавички та окуляри).
:::
