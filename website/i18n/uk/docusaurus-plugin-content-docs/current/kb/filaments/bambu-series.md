# Філаменти Bambu Lab

Bambu Lab виробляє широкий асортимент філаментів, спеціально оптимізованих для своїх принтерів. Усі філаменти Bambu Lab постачаються з **RFID-міткою**, яку принтер автоматично розпізнає та встановлює правильні налаштування.

## RFID та AMS

Усі філаменти Bambu Lab мають вбудований **RFID-чіп** у котушці. Це забезпечує:

- **Автоматичне розпізнавання** — принтер зчитує тип матеріалу, колір та налаштування
- **Залишкова кількість** — приблизна кількість філаменту на котушці
- **Правильні налаштування** — температура, швидкість та охолодження встановлюються автоматично
- **Сумісність з AMS** — безшовна зміна матеріалу в AMS

:::tip Сторонні філаменти в AMS
AMS працює також зі сторонніми філаментами, але налаштування потрібно встановити вручну в Bambu Studio. RFID-автовизначення є ексклюзивним для філаментів Bambu Lab.
:::

## Серія PLA

Серія PLA від Bambu Lab є найширшою та охоплює все від базових продуктів до спеціальних ефектів.

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 220 °C |
| Температура столу | 35–45 °C |
| Охолодження | 100% |
| RFID | Так |
| Сумісний з AMS | Так |

Стандартний філамент для повсякденного друку. Доступний у широкому виборі кольорів.

### PLA Matte

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 220 °C |
| Температура столу | 35–45 °C |
| Охолодження | 100% |

Забезпечує рівну матову поверхню, яка краще приховує лінії шарів, ніж стандартний PLA. Популярний вибір для естетичного друку.

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 230 °C |
| Температура столу | 45–55 °C |
| Охолодження | 80% |
| Поверхня | Глянцева, металевий відтінок |

Забезпечує блискучу, шовковисту поверхню з металевим ефектом. Потребує дещо нижчого охолодження та швидкості, ніж стандартний PLA.

### PLA Sparkle

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 220–230 °C |
| Температура столу | 35–45 °C |
| Охолодження | 100% |

Містить гліттерні частинки, що надають мерехтливий ефект. Друкується на стандартних налаштуваннях PLA.

### PLA Marble

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 220 °C |
| Температура столу | 35–45 °C |

Забезпечує унікальний мармуровий ефект з варіаціями кольору по всьому друку. Кожен друк трохи відрізняється.

|-----------|-------|
| Температура сопла | 220–230 °C |

Зміцнений PLA зі збільшеною ударною міцністю. Підходить для механічних деталей, яким потрібна більша міцність, ніж стандартний PLA.

| Параметр | Значення |

Поєднує гліттерний ефект з колірними градієнтами для унікального візуального ефекту. Друкується на стандартних налаштуваннях PLA.

| Параметр | Значення |
|-----------|-------|

| Температура столу | 70–80 °C |
| Охолодження | 50–70% |

Високошвидкісна версія PETG для швидшої екструзії без втрати якості. Ідеально для великих деталей та серійного виробництва.

### PETG-CF

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 250–270 °C |
| Температура столу | 70–80 °C |
| Охолодження | 40–60% |
| Сопло | Загартована сталь обов'язково |

PETG, армований вуглецевим волокном, зі збільшеною жорсткістю та розмірною стабільністю. Потрібне загартоване сопло (HS01 або аналог).

Усі філаменти, армовані вуглецевим волокном (PLA-CF, PETG-CF, PA-CF, PC-CF), потребують сопла з загартованої сталі. Латунь зношується за години-дні з CF-матеріалами.
:::

## Контроль якості та консистентність кольору

Bambu Lab дотримується суворого контролю якості своїх філаментів:

- **Допуск діаметра** — ±0.02 мм (лідер галузі)
- **Консистентність кольору** — контроль партій забезпечує однаковий колір між котушками
- **Якість котушки** — рівне намотування без вузлів і накладань
- **Вакуумне запечатування** — кожна котушка запакована з осушувачем у вакуумі
- **Тестування температурного профілю** — кожна партія тестується на оптимальну температуру

:::tip Номери кольорів для консистентності
Bambu Lab використовує номери кольорів (наприклад, "Bambu PLA Matte Charcoal") з контролем партій. Якщо вам потрібен ідентичний колір на кількох котушках для великого проєкту, замовляйте з однієї партії або зверніться до підтримки для підбору партій.
:::
