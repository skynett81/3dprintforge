## Поради за матеріалами

### Стандартні (PLA, PETG)
- PLA — найпростіший матеріал — ідеальний для прототипів та декору
- PETG забезпечує кращу міцність та термостійкість, але може більше тягнути нитки
- Обидва працюють без корпусу та зі стандартним латунним соплом

### Інженерні (ABS, ASA, PC)
- Потрібен закритий принтер для запобігання деформації
- ASA стійкий до UV — використовуйте для зовнішніх деталей
- PC забезпечує найвищу міцність та термостійкість, але найскладніший у друці

### Композитні (CF/GF)
- **Завжди** використовуйте загартоване сталеве сопло — вуглецеве волокно швидко зношує латунні сопла
- CF-варіанти дають жорсткіші та легші деталі з матовою поверхнею
- GF-варіанти дешевші, але дають грубішу поверхню

### Гнучкі (TPU)
- Друкуйте повільно (50 мм/с або менше) для найкращих результатів
- Зменшіть ретракцію для запобігання засмічення
- Екструдери з прямим приводом працюють набагато краще за Bowden

### Підтримка (PVA, HIPS)
- PVA розчиняється у воді — ідеальний для підтримки PLA
- HIPS розчиняється в лімонені — використовується разом з ABS
- Обидва дуже чутливі до вологи — зберігайте сухими

:::warning Нейлонові матеріали
PA, PA-CF та PA-GF є **надзвичайно** гігроскопічними. Вони поглинають вологу з повітря за лічені хвилини. Завжди сушіть 12+ годин перед використанням та друкуйте безпосередньо з сушильної коробки, якщо можливо.
:::
