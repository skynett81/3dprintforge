# PVA та матеріали підтримки

Матеріали підтримки використовуються для друку складних геометрій з навісами, мостами та внутрішніми порожнинами, які неможливо надрукувати без тимчасової підтримки. Після друку матеріал підтримки видаляється — механічно або шляхом розчинення в розчиннику.

## Огляд

| Матеріал | Розчинник | Комбінувати з | Час розчинення | Складність |
|-----------|-----------|-------------|----------------|-------------------|
| PVA | Вода | PLA, PETG | 12–24 години | Складно |
| HIPS | d-Лімонен | ABS, ASA | 12–24 години | Помірно |
| PVB | Ізопропанол (IPA) | PLA, PETG | 6–12 годин | Помірно |
| BVOH | Вода | PLA, PETG, PA | 4–8 годин | Складно |

---

## PVA (Полівініловий спирт)

PVA — це водорозчинний матеріал підтримки, найбільш поширений вибір для PLA-друку зі складними структурами підтримки.

### Налаштування

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 190–210 °C |
| Температура столу | 45–60 °C |
| Охолодження деталі | 100% |
| Швидкість | 60–80% |
| Ретракція | Збільшена (6–8 мм) |

### Рекомендовані робочі пластини

| Пластина | Придатність | Клей? |
|-------|---------|----------|
| Cool Plate (Smooth PEI) | Відмінно | Ні |
| Textured PEI | Добре | Ні |
| Engineering Plate | Добре | Ні |
| High Temp Plate | Уникати | — |

### Сумісність

PVA найкраще працює з матеріалами, що друкуються при **подібних температурах**:

| Основний матеріал | Сумісність | Примітка |
|---------------|---------------|---------|
| PLA | Відмінна | Ідеальна комбінація |
| PETG | Добра | Температура столу може бути трохи високою для PVA |
| ABS/ASA | Погана | Занадто висока температура камери — PVA деградує |
| PA (Нейлон) | Погана | Занадто високі температури |

### Розчинення

- Покладіть готовий друк у **теплу воду** (прибл. 40 °C)
- PVA розчиняється протягом **12–24 годин** залежно від товщини
- Періодично перемішуйте воду для прискорення процесу
- Міняйте воду кожні 6–8 годин для швидшого розчинення
- Ультразвуковий очищувач дає значно швидший результат (2–6 годин)

:::danger PVA надзвичайно гігроскопічний
PVA поглинає вологу з повітря **дуже швидко** — навіть кілька годин на відкритому повітрі можуть зіпсувати результат друку. PVA, що поглинув вологу, спричиняє:

- Сильне бурхливе кипіння та потріскування
- Погану адгезію до основного матеріалу
- Нитки та липку поверхню
- Засмічення сопла

**Завжди сушіть PVA безпосередньо перед використанням** та друкуйте з сухого середовища (сушильна коробка).
:::

### Сушіння PVA

| Параметр | Значення |
|-----------|-------|
| Температура сушіння | 45–55 °C |
| Час сушіння | 6–10 годин |
| Рівень гігроскопічності | Надзвичайно високий |
| Метод зберігання | Герметична коробка з осушувачем, завжди |

## HIPS (Ударостійкий полістирол)

HIPS — матеріал підтримки, що розчиняється у d-лімонені (цитрусовий розчинник). Це переважний матеріал підтримки для ABS та ASA.

### Налаштування

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 220–240 °C |
| Температура столу | 90–100 °C |
| Температура камери | 40–50 °C (рекомендовано) |
| Охолодження деталі | 20–40% |
| Швидкість | 70–90% |

### Сумісність

| Основний матеріал | Сумісність | Примітка |
|---------------|---------------|---------|
| ABS | Відмінна | Ідеальна комбінація — подібні температури |
| ASA | Відмінна | Дуже добра адгезія |
| PLA | Погана | Занадто велика різниця температур |
| PETG | Погана | Різна термічна поведінка |

### Розчинення у d-лімонені

- Покладіть друк у **d-лімонен** (цитрусовий розчинник)
- Час розчинення: **12–24 години** при кімнатній температурі
- Нагрівання до 35–40 °C прискорює процес
- d-Лімонен можна використовувати повторно 2–3 рази
- Після розчинення промийте деталь водою та висушіть

### Переваги над PVA

- **Набагато менш чутливий до вологи** — простіше зберігати та обробляти
- **Міцніший як матеріал підтримки** — витримує більше без руйнування
- **Краща термічна сумісність** з ABS/ASA
- **Легше друкувати** — менше засмічень та проблем

:::warning d-Лімонен є розчинником
Використовуйте рукавички та працюйте у провітрюваному приміщенні. d-Лімонен може подразнювати шкіру та слизові оболонки. Зберігайте в недоступному для дітей місці.
:::

## PVB (Полівінілбутираль)

PVB — унікальний матеріал підтримки, що розчиняється в ізопропанолі (IPA) і може використовуватися для згладжування поверхонь парами IPA.

### Налаштування

| Параметр | Значення |
|-----------|-------|
| Температура сопла | 200–220 °C |
| Температура столу | 55–75 °C |
| Охолодження деталі | 80–100% |
| Швидкість | 70–80% |

### Сумісність

| Основний матеріал | Сумісність | Примітка |
|---------------|---------------|---------|
| PLA | Добра | Прийнятна адгезія |
| PETG | Помірна | Температура столу може варіюватися |
| ABS/ASA | Погана | Занадто високі температури |

### Згладжування поверхні парами IPA

Унікальна властивість PVB — поверхню можна згладити парами IPA:

1. Помістіть деталь у закриту ємність
2. Покладіть зволожену IPA тканину на дно (без прямого контакту з деталлю)
3. Дайте парі діяти **30–60 хвилин**
4. Вийміть та дайте висохнути 24 години
5. Результат — гладка, напівглянцева поверхня

:::tip PVB як покриття поверхні
Хоча PVB переважно є матеріалом підтримки, його можна друкувати як зовнішній шар на деталях з PLA для отримання поверхні, яку можна згладити парами IPA. Це дає обробку, схожу на ABS, згладжений ацетоном.
:::

## Поради для двоекструзійного та багатокольорового друку

### Загальні рекомендації

- **Обсяг прочищення** — матеріали підтримки потребують доброго прочищення при зміні матеріалу (мінімум 150–200 мм³)
- **Інтерфейсні шари** — використовуйте 2–3 інтерфейсних шари між підтримкою та основною деталлю для чистої поверхні
- **Відстань** — встановіть відстань підтримки 0.1–0.15 мм для легкого видалення після розчинення
- **Шаблон підтримки** — використовуйте трикутний шаблон для PVA/BVOH, сітку для HIPS

### Налаштування AMS

- Розмістіть матеріал підтримки в **слоті AMS з осушувачем**
- Для PVA: розгляньте зовнішню сушильну коробку з боуден-з'єднанням
- Налаштуйте правильний профіль матеріалу в Bambu Studio
- Протестуйте з простою моделлю з навісом перед друком складних деталей

### Поширені проблеми та рішення

| Проблема | Причина | Рішення |
|---------|-------|---------|
| Підтримка не тримається | Занадто велика відстань | Зменшіть відстань інтерфейсу до 0.05 мм |
| Підтримка тримається занадто міцно | Занадто мала відстань | Збільшіть відстань інтерфейсу до 0.2 мм |
| Бульбашки у матеріалі підтримки | Вологість | Ретельно висушіть філамент |
| Нитки між матеріалами | Недостатня ретракція | Збільшіть ретракцію на 1–2 мм |
| Погана поверхня біля підтримки | Замало інтерфейсних шарів | Збільшіть до 3–4 інтерфейсних шарів |

:::tip Починайте просто
Для першого друку з матеріалом підтримки: використовуйте PLA + PVA, просту модель з чітким навісом (45°+) та стандартні налаштування в Bambu Studio. Оптимізуйте в міру набуття досвіду.
:::
