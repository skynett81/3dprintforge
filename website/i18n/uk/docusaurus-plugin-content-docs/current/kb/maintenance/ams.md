# Обслуговування AMS

AMS (Automatic Material System) — це точна система, яка потребує регулярного обслуговування для надійної роботи. Найпоширеніші проблеми — забруднений шлях філаменту та вологість усередині корпусу.

## PTFE-трубки

PTFE-трубки транспортують філамент від AMS до принтера. Вони є одними з перших деталей, що зношуються.

### Огляд
Перевіряйте PTFE-трубки на наявність:
- **Зламів або перегинів** — перешкоджають подачі філаменту
- **Зносу біля з'єднань** — біла пудра навколо входів
- **Деформації форми** — особливо при використанні CF-матеріалів

### Заміна PTFE-трубок
1. Звільніть філамент з AMS (запустіть цикл розвантаження)
2. Натисніть блакитне стопорне кільце навколо трубки біля з'єднання
3. Витягніть трубку (потрібне тверде захоплення)
4. Відріжте нову трубку до потрібної довжини (не коротшу за оригінал)
5. Вставте до упору та зафіксуйте

:::tip AMS Lite проти AMS
AMS Lite (A1/A1 Mini) має простішу конфігурацію PTFE порівняно з повноцінним AMS (P1S/X1C). Трубки коротші та простіші в заміні.
:::

## Шлях філаменту

### Очищення шляху філаменту
Філаменти залишають пил і залишки в шляху подачі, особливо CF-матеріали:

1. Виконайте розвантаження всіх слотів
2. Використовуйте стиснене повітря або м'яку кисть для видалення пилу
3. Пропустіть чистий шматок нейлону або очищувального PTFE-філаменту через канал

### Датчики
AMS використовує датчики для визначення положення філаменту та його обриву. Тримайте вікна датчиків чистими:
- Обережно протирайте лінзи датчиків чистою кисточкою
- Не наносьте IPA безпосередньо на датчики

## Вологість

AMS не захищає філамент від вологи. Для гігроскопічних матеріалів (PA, PETG, TPU) рекомендується:

### Сухі альтернативи AMS
- **Запечатана коробка:** Розмістіть котушки в щільній коробці з силікагелем
- **Bambu Dry Box:** Офіційний аксесуар-сушильна коробка
- **Зовнішній подавач:** Використовуйте подавач філаменту поза AMS для чутливих матеріалів

### Індикатори вологості
Розмістіть карти-індикатори вологості (гігрометри) всередині корпусу AMS. Замінюйте пакети силікагелю при відносній вологості понад 30%.

## Приводні колеса та механізм затиску

### Огляд
Перевіряйте приводні колеса (колеса екструдера в AMS) на:
- Залишки філаменту між зубцями
- Знос зубчастого вінця
- Нерівномірне зусилля при ручному протягуванні

### Очищення
1. Використовуйте зубну щітку або кисть для видалення залишків між зубцями приводного колеса
2. Продуйте стисненим повітрям
3. Уникайте мастила — рівень тяги відкалібровано для сухої роботи

## Інтервали обслуговування

| Дія | Інтервал |
|-----|----------|
| Візуальний огляд PTFE-трубок | Щомісяця |
| Очищення шляху філаменту | Кожні 100 годин |
| Перевірка датчиків | Щомісяця |
| Заміна силікагелю (при сухому налаштуванні) | За потреби (при 30%+ відносній вологості) |
| Заміна PTFE-трубок | При видимому зносі |
