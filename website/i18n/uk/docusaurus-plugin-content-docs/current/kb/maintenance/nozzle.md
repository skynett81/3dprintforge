# Обслуговування сопла

Сопло є одним з найважливіших компонентів принтера. Правильне обслуговування продовжує термін служби та забезпечує якісні результати друку.

## Типи сопел

| Тип сопла | Матеріали | Ресурс (орієнтовно) | Макс. темп |
|-----------|-----------|---------------------|------------|
| Латунь (стандарт) | PLA, PETG, ABS, TPU | 200–500 годин | 300 °C |
| Загартована сталь | Всі, включно CF/GF | 300–600 годин | 300 °C |
| HS01 (Bambu) | Всі, включно CF/GF | 500–1000 годин | 300 °C |

:::danger Ніколи не використовуйте латунне сопло з CF/GF
Філаменти з вуглецевим та скляним волокном зношують латунні сопла за години. Замініть на загартовану сталь перед друком CF/GF-матеріалів.
:::

## Очищення

### Просте очищення (між котушками)
1. Розігрійте сопло до 200–220 °C
2. Вручну протисніть філамент, доки він не вийде чистим
3. Швидко витягніть філамент ("холодне протягування" — дивіться нижче)

### Очищення IPA
Для стійких забруднень:
1. Розігрійте сопло до 200 °C
2. Нанесіть 1–2 краплі IPA на кінець сопла (обережно!)
3. Дайте пару розчинити залишки
4. Протягніть свіжий філамент

:::warning Будьте обережні з IPA на гарячому соплі
IPA кипить при 83 °C і активно випаровується на гарячому соплі. Використовуйте мінімальну кількість та уникайте вдихання парів.
:::

## Холодне протягування (Cold Pull)

Холодне протягування — найефективніший метод видалення забруднень і вуглецевих залишків із сопла.

**Покрокова інструкція:**
1. Розігрійте сопло до 200–220 °C
2. Вручну вставте нейлоновий філамент (або той, що в соплі)
3. Дайте нейлону просочитися в соплі 1–2 хвилини
4. Знизьте температуру до 80–90 °C (для нейлону)
5. Зачекайте, поки сопло охолоне до цільового значення
6. Різко і впевнено витягніть філамент одним рухом
7. Огляньте кінець: він повинен мати форму внутрішньої поверхні сопла — чистий і без залишків
8. Повторіть 3–5 разів до отримання чистого білого протягування

:::tip Нейлон для холодного протягування
Нейлон дає найкращий результат для холодного протягування, оскільки добре захоплює забруднення. Білий нейлон дозволяє легко побачити, чи протягування чисте.
:::

## Заміна сопла

### Ознаки того, що сопло треба замінити
- Нерівні нижні поверхні та погана розмірна точність
- Постійні проблеми з екструзією після очищення
- Видимий знос або деформація отвору сопла
- Сопло вичерпало розрахунковий ресурс

### Процедура (P1S/X1C)
1. Розігрійте сопло до 200 °C
2. Загальмуйте двигун екструдера (звільніть філамент)
3. Використайте ключ для послаблення сопла (проти годинникової стрілки)
4. Замінюйте сопло, поки воно гаряче — **не залишайте інструмент на соплі при охолодженні**
5. Затягніть до потрібного моменту (не перетягуйте)
6. Запустіть калібрування після заміни

:::warning Завжди замінюйте в гарячому стані
Момент затяжки холодного сопла може розламати деталь при нагріванні. Завжди замінюйте та затягуйте сопло в гарячому стані (200 °C).
:::
