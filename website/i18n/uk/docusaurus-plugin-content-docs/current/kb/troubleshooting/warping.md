# Warping

Warping виникає, коли кути або краї деталі піднімаються від платформи під час або після друку. Причина — термічне скорочення матеріалу.

## Що таке warping?

Коли пластик охолоджується, він стискається. Верхні шари гарячіші за нижні — це створює напругу, яка тягне краї вгору та деформує деталь. Чим більша різниця температур, тим сильніший warping.

## Матеріали, найбільш схильні до warping

| Матеріал | Ризик warping | Потребує закритого корпусу |
|----------|---------------|---------------------------|
| PLA | Низький | Ні |
| PETG | Низький–Помірний | Ні |
| ABS | Високий | Так |
| ASA | Високий | Так |
| PA/Nylon | Дуже високий | Так |
| PC | Дуже високий | Так |
| TPU | Низький | Ні |

## Рішення

### 1. Використовуйте закритий корпус (камеру)

Найважливіший захід для ABS, ASA, PA та PC:
- Тримайте температуру камери 40–55 °C для найкращого результату
- X1C та P1S: активуйте вентилятори камери в "закритому" режимі
- A1/P1P: використовуйте кришку-накладку для утримання тепла

### 2. Використовуйте brim

Brim — це одношаровий периметр з широкими краями, що утримує деталь на платформі:

```
Bambu Studio:
1. Виберіть деталь у слайсері
2. Перейдіть до Support → Brim
3. Встановіть ширину 5–10 мм (чим більший warping, тим ширше)
4. Тип: Outer Brim Only (рекомендовано)
```

:::tip Посібник з ширини brim
- PLA (рідко потрібно): 3–5 мм
- PETG: 4–6 мм
- ABS/ASA: 6–10 мм
- PA/Nylon: 8–15 мм
:::

### 3. Підвищіть температуру столу

Вища температура столу зменшує різницю температур між шарами:
- ABS: спробуйте 105–110 °C
- PA: 85–95 °C
- PETG: 80–85 °C

### 4. Зменшіть охолодження деталі

Для матеріалів зі схильністю до warping — знижуйте або вимикайте охолодження деталі:
- ABS/ASA: 0–20% охолодження деталі
- PA: 0–30% охолодження деталі

### 5. Уникайте протягів і холодного повітря

Тримайте принтер подалі від:
- Вікон і зовнішніх дверей
- Кондиціонерів і вентиляторів
- Протягів у приміщенні

Для P1P та A1: прикрийте отвори картоном під час критичного друку.

### 6. Draft Shield

Draft Shield — це тонка стінка навколо об'єкта, що утримує тепло всередині:

```
Bambu Studio:
1. Перейдіть до Support → Draft Shield
2. Активуйте та встановіть відстань (3–5 мм)
```

Особливо корисно для високих, стрункіх об'єктів.

### 7. Заходи при проектуванні моделі

При проектуванні власних моделей:
- Уникайте великих пласких основ (додайте фаску/заокруглення в кутах)
- Розділіть великі пласкі деталі на менші секції
- Використовуйте "mouse ears" — маленькі кола в кутах — у слайсері або CAD

## Warping після охолодження

Іноді деталь виглядає добре, але warping виникає після зняття з платформи:
- Завжди чекайте, поки платформа та деталь **повністю охолонуть** (нижче 40 °C) перед зніманням
- Для ABS: дайте охолонути всередині закритої камери для повільнішого охолодження
- Уникайте класти гарячу деталь на холодну поверхню
