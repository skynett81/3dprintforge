# Мащення

Правильне мащення рухомих частин зменшує знос, знижує рівень шуму та забезпечує точний рух. Принтери Bambu Lab використовують лінійні системи переміщення, які потребують періодичного мащення.

## Типи мастил

| Компонент | Тип мастила | Продукт |
|-----------|-------------|---------|
| Лінійні стержні (XY) | Легка машинна олія або PTFE-спрей | 3-в-1, Super Lube |
| Гвинтовий шпиндель Z-осі | Густе мастило | Super Lube fett |
| Лінійні підшипники | Легке літієве мастило | Bambu Lab grease |
| З'єднання кабельних ланцюгів | Не потребує (суха робота) | — |

## Лінійні стержні

### Осі X та Y
Стержні — це загартовані стальні прути, що ковзають крізь лінійні підшипники:

```
Інтервал: Кожні 200–300 годин або при скрипі
Кількість: Дуже мало — одна крапля на точку стержня достатня
Метод:
1. Вимкніть принтер
2. Вручну перемістіть каретку до кінця
3. Нанесіть 1 краплю легкої олії в середину стержня
4. Повільно переміщуйте каретку вперед-назад 10 разів
5. Витріть надлишок олії безворсовим папером
```

:::warning Не перемащуйте
Занадто багато олії притягує пил і утворює абразивну пасту. Використовуйте мінімальні кількості і завжди витирайте надлишок.
:::

### Z-вісь (вертикальна)
Z-вісь використовує гвинтовий шпиндель (leadscrew), який потребує мастила (не олії):

```
Інтервал: Кожні 200 годин
Метод:
1. Вимкніть принтер
2. Нанесіть тонкий шар мастила вздовж шпинделя
3. Вручну рухайте Z-вісь вгору-вниз (або через меню обслуговування)
4. Мастило розподіляється автоматично
```

## Лінійні підшипники

Bambu Lab P1S та X1C використовують лінійні підшипники (MGN12) на Y-осі:

```
Інтервал: Кожні 300–500 годин
Метод:
1. Видаліть трохи старого мастила голкою або зубочисткою з отвору для подачі
2. Введіть нове мастило шприцом з тонкою канюлею
3. Рухайте вісь вперед-назад для рівномірного розподілу мастила
```

Bambu Lab продає офіційне мастило (Bambu Lubricant), відкаліброване для цієї системи.

## Обслуговування мащення для різних моделей

### X1C / P1S
- Y-вісь: Лінійні підшипники — мастило Bambu
- X-вісь: Вуглецеві стержні — легка олія
- Z-вісь: Подвійний гвинтовий шпиндель — мастило Bambu

### A1 / A1 Mini
- Всі осі: Стальні стержні — легка олія
- Z-вісь: Одинарний гвинтовий шпиндель — мастило Bambu

## Ознаки необхідності мащення

- **Скрип або скрегіт** при переміщенні
- **Виброс-візерунки** на вертикальних стінках (VFA)
- **Неточні розміри** без інших видимих причин
- **Підвищений рівень шуму** від системи переміщення

## Інтервали мащення

| Дія | Інтервал |
|-----|----------|
| Олія XY-стержнів | Кожні 200–300 годин |
| Мастило Z-шпинделя | Кожні 200 годин |
| Мастило лінійних підшипників (X1C/P1S) | Кожні 300–500 годин |
| Повний цикл обслуговування | Раз на пів року (або 500 годин) |

Використовуйте модуль обслуговування в дашборді для автоматичного відстеження інтервалів.
