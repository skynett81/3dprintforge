# Початкове налаштування

Коли дашборд запускається вперше, майстер налаштування відкривається автоматично.

## Майстер налаштування

Майстер доступний за адресою `https://ваш-сервер:3443/setup`. Він проведе вас через:

1. Створення облікового запису адміністратора
2. Додавання принтера
3. Перевірка підключення
4. Налаштування сповіщень (необов'язково)

## Додавання принтера

Для підключення принтера вам знадобляться три речі:

| Поле | Опис | Приклад |
|------|------|---------|
| IP-адреса | Локальна IP принтера | `192.168.1.100` |
| Серійний номер | 15 символів, вказано під принтером | `01P09C123456789` |
| Access Code | 8 символів, є в мережевих налаштуваннях принтера | `12345678` |

### Знайдіть Access Code на принтері

**X1C / P1S / P1P:**
1. Перейдіть до **Налаштувань** на екрані
2. Виберіть **WLAN** або **LAN**
3. Знайдіть **Access Code**

**A1 / A1 Mini:**
1. Натисніть екран і виберіть **Налаштування**
2. Перейдіть до **WLAN**
3. Знайдіть **Access Code**

:::tip Статична IP-адреса
Призначте статичну IP-адресу принтеру у вашому роутері (резервування DHCP). Тоді вам не доведеться оновлювати дашборд щоразу, коли принтер отримає нову IP.
:::

## Налаштування AMS

Після підключення принтера статус AMS оновлюється автоматично. Ви можете:

- Дати назву та колір кожному слоту
- Прив'язати котушки до вашого складу філаменту
- Переглядати витрату філаменту на котушку

Перейдіть до **Налаштування → Принтер → AMS** для ручного налаштування.

## HTTPS-сертифікати {#https-sertifikater}

### Самопідписаний сертифікат (за замовчуванням)

Дашборд автоматично генерує самопідписаний сертифікат при запуску. Щоб довіряти йому у браузері:

- **Chrome/Edge:** Натисніть "Додатково" → "Перейти до сайту"
- **Firefox:** Натисніть "Додатково" → "Прийняти ризик і продовжити"

### Власний сертифікат

Помістіть файли сертифіката до папки та налаштуйте в `config.json`:

```json
{
  "ssl": {
    "cert": "/шлях/до/cert.pem",
    "key": "/шлях/до/key.pem"
  }
}
```

:::info Let's Encrypt
Використовуєте доменне ім'я? Отримайте безкоштовний сертифікат за допомогою Let's Encrypt і Certbot, і вкажіть `cert` та `key` на файли в `/etc/letsencrypt/live/ваш-домен/`.
:::

## Змінні середовища

Усі налаштування можна перевизначити через змінні середовища:

| Змінна | За замовчуванням | Опис |
|--------|-----------------|------|
| `PORT` | `3000` | HTTP-порт |
| `HTTPS_PORT` | `3443` | HTTPS-порт |
| `NODE_ENV` | `production` | Середовище |
| `AUTH_SECRET` | (авто) | JWT-секрет |

## Налаштування кількох принтерів

Ви можете додавати принтери в розділі **Налаштування → Принтери → Додати принтер**. Використовуйте вибір принтера вгорі дашборду для перемикання між ними.
