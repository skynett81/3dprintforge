# Технічне обслуговування

Модуль технічного обслуговування допомагає планувати та відстежувати все обслуговування принтерів Bambu Lab — від заміни сопла до мащення напрямних.

Перейдіть до: **https://localhost:3443/#maintenance**

## План обслуговування

3DPrintForge постачається з попередньо налаштованими інтервалами обслуговування для всіх моделей принтерів Bambu Lab:

| Завдання | Інтервал (стандарт) | Модель |
|----------|--------------------|----|
| Очистити сопло | Кожні 200 годин | Всі |
| Замінити сопло (латунне) | Кожні 500 годин | Всі |
| Замінити сопло (загартоване) | Кожні 2000 годин | Всі |
| Змастити вісь X | Кожні 300 годин | X1C, P1S |
| Змастити вісь Z | Кожні 300 годин | Всі |
| Очистити шестерні AMS | Кожні 200 годин | AMS |
| Очистити камеру | Кожні 500 годин | X1C |
| Замінити трубку PTFE | За потреби / 1000 годин | Всі |
| Калібрування (повне) | Щомісяця | Всі |

Всі інтервали можна налаштувати для кожного принтера.

## Журнал заміни сопла

1. Перейдіть до **Обслуговування → Сопла**
2. Натисніть **Записати заміну сопла**
3. Заповніть:
   - **Дата** — автоматично встановлюється на сьогодні
   - **Матеріал сопла** — Латунь / Загартована сталь / Мідь / Рубінова вставка
   - **Діаметр сопла** — 0.2 / 0.4 / 0.6 / 0.8 мм
   - **Бренд/модель** — необов'язково
   - **Ціна** — для журналу витрат
   - **Годин при заміні** — автоматично отримується з лічильника часу друку
4. Натисніть **Зберегти**

Журнал показує всю історію сопел, відсортовану за датою.

:::tip Попереднє нагадування
Встановіть **Сповіщати за X годин** (наприклад, 50 годин) для отримання сповіщення задовго до наступної рекомендованої заміни.
:::

## Створення завдань обслуговування

1. Натисніть **Нове завдання** (іконка +)
2. Заповніть:
   - **Назва завдання** — наприклад, «Змастити вісь Y»
   - **Принтер** — виберіть відповідний принтер(и)
   - **Тип інтервалу** — Години / Дні / Кількість друків
   - **Інтервал** — наприклад, 300 годин
   - **Останнє виконання** — вкажіть, коли виконувалось востаннє (встановіть задню дату)
3. Натисніть **Створити**

## Інтервали та нагадування

Для активних завдань відображається:
- **Зелений** — до наступного обслуговування > 50% інтервалу залишилось
- **Жовтий** — до наступного обслуговування < 50% залишилось
- **Помаранчевий** — до наступного обслуговування < 20% залишилось
- **Червоний** — обслуговування прострочено

### Налаштування нагадувань

1. Натисніть на завдання → **Редагувати**
2. Активуйте **Нагадування**
3. Встановіть **Сповіщати при** наприклад 10% залишку до строку
4. Виберіть канал сповіщення (дивіться [Сповіщення](../features/notifications))

## Позначення як виконане

1. Знайдіть завдання в списку
2. Натисніть **Виконано** (іконка галочки)
3. Інтервал скидається з сьогоднішньої дати/годин
4. Запис журналу створюється автоматично

## Журнал витрат

Всі завдання обслуговування можуть мати пов'язані витрати:

- **Запчастини** — сопла, трубки PTFE, мастила
- **Час** — витрачені години × погодинна ставка
- **Зовнішнє обслуговування** — оплачений ремонт

Витрати підсумовуються за принтером та відображаються в огляді статистики.

## Історія обслуговування

Перейдіть до **Обслуговування → Історія** для перегляду:
- Всіх виконаних завдань обслуговування
- Дата, годин та вартість
- Хто виконував (у системах з кількома користувачами)
- Коментарі та нотатки

Експортуйте історію у CSV для бухгалтерських цілей.
