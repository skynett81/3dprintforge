# Print Guard

Print Guard — це система моніторингу в реальному часі 3DPrintForge. Вона безперервно відстежує камеру, датчики та дані принтера та виконує налаштовувані дії, коли щось йде не так.

Перейдіть до: **https://localhost:3443/#protection**

## Виявлення подій XCam

Принтери Bambu Lab надсилають події XCam через MQTT, коли камера AI виявляє проблеми:

| Подія | Код | Серйозність |
|-------|-----|------------|
| Виявлено спагеті | `xcam_spaghetti` | Критична |
| Відлипання від пластини | `xcam_detach` | Висока |
| Несправність першого шару | `xcam_first_layer` | Висока |
| Стрінгінг | `xcam_stringing` | Середня |
| Помилка екструзії | `xcam_extrusion` | Висока |

Для кожного типу події можна налаштувати одну або кілька дій:

- **Сповістити** — надіслати сповіщення через активні канали
- **Пауза** — поставити друк на паузу для ручної перевірки
- **Стоп** — негайно скасувати друк
- **Нічого** — ігнорувати подію (все одно журналювати)

:::danger Стандартна поведінка
За замовчуванням події XCam налаштовані на **Сповістити** та **Пауза**. Змініть на **Стоп**, якщо ви повністю довіряєте виявленню AI.
:::

## Моніторинг датчиків

Print Guard безперервно відстежує дані датчиків та подає сигнал тривоги при відхиленнях:

### Відхилення температури

1. Перейдіть до **Print Guard → Температура**
2. Встановіть **Макс. відхилення від цільової температури** (рекомендовано: ±5°C для сопла, ±3°C для столу)
3. Виберіть **Дію при відхиленні**: Сповістити / Пауза / Стоп
4. Встановіть **Затримку** (секунди) перед виконанням дії — дає температурі час стабілізуватися

### Малий залишок філаменту

Система розраховує залишок філаменту на котушках:

1. Перейдіть до **Print Guard → Філамент**
2. Встановіть **Мінімальний поріг** у грамах (наприклад, 50 г)
3. Виберіть дію: **Пауза та сповістити** (рекомендується) для ручної заміни котушки

### Виявлення зупинки друку

Виявляє несподівану зупинку друку (тайм-аут MQTT, обрив філаменту тощо):

1. Активуйте **Виявлення зупинки**
2. Встановіть **Тайм-аут** (рекомендовано: 120 секунд без даних = зупинка)
3. Дія: Завжди сповіщати — друк міг вже зупинитися

## Конфігурація

### Активація Print Guard

1. Перейдіть до **Налаштування → Print Guard**
2. Увімкніть **Активувати Print Guard**
3. Виберіть принтери для моніторингу
4. Натисніть **Зберегти**

### Правила для кожного принтера

Різні принтери можуть мати різні правила:

1. Натисніть на принтер в огляді Print Guard
2. Вимкніть **Успадковувати глобальні правила**
3. Налаштуйте власні правила для цього принтера

## Журнал та історія подій

Всі події Print Guard журналюються:

- Перейдіть до **Print Guard → Журнал**
- Фільтруйте за принтером, типом події, датою та серйозністю
- Натисніть на подію для перегляду детальної інформації та вжитих заходів
- Експортуйте журнал у CSV

:::tip Хибні спрацьовування
Якщо Print Guard ініціює непотрібні паузи, відрегулюйте чутливість в розділі **Print Guard → Налаштування → Чутливість**. Починайте з «Низька» та поступово збільшуйте.
:::
