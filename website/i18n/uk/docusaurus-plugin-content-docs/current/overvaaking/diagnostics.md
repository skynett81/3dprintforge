# Діагностика

Сторінка діагностики надає поглиблений огляд стану здоров'я, продуктивності та стану принтера з часом.

Перейдіть до: **https://localhost:3443/#diagnostics**

## Оцінка здоров'я

Для кожного принтера розраховується **оцінка здоров'я** від 0 до 100 на основі:

| Фактор | Вага | Опис |
|--------|------|------|
| Відсоток успіху (30д) | 30 % | Частка успішних друків за останні 30 днів |
| Знос компонентів | 25 % | Середній знос критичних частин |
| HMS-помилки (30д) | 20 % | Кількість та серйозність помилок |
| Статус калібрування | 15 % | Час з останнього калібрування |
| Стабільність температури | 10 % | Відхилення від цільової температури під час друку |

**Інтерпретація оцінки:**
- 🟢 80–100 — Відмінний стан
- 🟡 60–79 — Добре, але щось слід перевірити
- 🟠 40–59 — Знижена продуктивність, рекомендується обслуговування
- 🔴 0–39 — Критичний, обслуговування необхідне

:::tip Історія
Натисніть на графік здоров'я для перегляду динаміки оцінки з часом. Різкі падіння можуть вказувати на конкретну подію.
:::

## Графіки телеметрії

Сторінка телеметрії показує інтерактивні графіки для всіх значень датчиків:

### Доступні набори даних

- **Температура сопла** — фактична проти цільової
- **Температура столу** — фактична проти цільової
- **Температура камери** — температура навколишнього середовища всередині машини
- **Двигун екструдера** — споживання струму та температура
- **Швидкість вентиляторів** — інструментальна голівка, камера, AMS
- **Тиск** (X1C) — тиск у камері для AMS
- **Прискорення** — дані вібрації (ADXL345)

### Навігація по графіках

1. Виберіть **Часовий період**: Остання година / 24 години / 7 днів / 30 днів / Власний
2. Виберіть **Принтер** зі списку, що розкривається
3. Виберіть **Набори даних** для відображення (підтримується множинний вибір)
4. Прокручуйте для масштабування по часовій шкалі
5. Натисніть та перетягуйте для панорамування
6. Двічі натисніть для скидання масштабу

### Експорт телеметричних даних

1. Натисніть **Експортувати** на графіку
2. Виберіть формат: **CSV**, **JSON** або **PNG** (зображення)
3. Вибраний часовий період та набори даних експортуються

## Bed Mesh

Візуалізація bed mesh показує калібрування рівності платформи друку:

1. Перейдіть до **Діагностика → Bed Mesh**
2. Виберіть принтер
3. Остання сітка відображається як 3D-поверхня та теплова карта:
   - **Синій** — нижче центру (увігнутий)
   - **Зелений** — приблизно рівний
   - **Червоний** — вище центру (опуклий)
4. Наведіть на точку для перегляду точного відхилення в мм

### Сканування bed mesh з інтерфейсу

1. Натисніть **Сканувати зараз** (потребує, щоб принтер був вільним)
2. Підтвердіть у діалозі — принтер автоматично починає калібрування
3. Зачекайте завершення сканування (приблизно 3–5 хвилин)
4. Нова сітка відображається автоматично

:::warning Спочатку нагрійте
Bed mesh слід сканувати з нагрітим столом (50–60°C для PLA) для точного калібрування.
:::
