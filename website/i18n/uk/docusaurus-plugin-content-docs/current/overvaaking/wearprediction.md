# Прогноз зносу

Прогноз зносу розраховує очікуваний строк служби критичних компонентів на основі фактичного використання, типу філаменту та поведінки принтера — щоб ви могли планувати обслуговування проактивно, а не реактивно.

Перейдіть до: **https://localhost:3443/#wear**

## Компоненти, що відстежуються

Bambu Dashboard відстежує знос 8 компонентів для кожного принтера:

| Компонент | Основний фактор зносу | Типовий строк служби |
|----------|----------------------|---------------------|
| **Сопло (латунне)** | Тип філаменту + години | 300–800 годин |
| **Сопло (загартоване)** | Години + абразивний матеріал | 1500–3000 годин |
| **Трубка PTFE** | Години + висока температура | 500–1500 годин |
| **Шестерня екструдера** | Години + абразивний матеріал | 1000–2000 годин |
| **Стержень осі X (CNC)** | Кількість друків + швидкість | 2000–5000 годин |
| **Поверхня платформи друку** | Кількість друків + температура | 200–500 друків |
| **Шестерні AMS** | Кількість замін філаменту | 5000–15000 замін |
| **Вентилятори камери** | Години в роботі | 3000–8000 годин |

## Розрахунок зносу

Знос розраховується як сукупний відсоток (0–100% зношено):

```
Знос % = (фактичне використання / очікуваний строк служби) × 100
        × множник матеріалу
        × множник швидкості
```

**Множники матеріалу:**
- PLA, PETG: 1.0× (нормальний знос)
- ABS, ASA: 1.1× (трохи більш агресивний)
- PA, PC: 1.2× (жорсткий для PTFE та сопла)
- CF/GF-композити: 2.0–3.0× (дуже абразивний)

:::warning Вуглецеве волокно
Філаменти з армуванням вуглецевим волокном (CF-PLA, CF-PA тощо) надзвичайно швидко зношують латунні сопла. Використовуйте загартоване сталеве сопло та очікуйте 2–3× швидший знос.
:::

## Розрахунок строку служби

Для кожного компонента відображається:

- **Поточний знос** — використаний відсоток
- **Розрахунковий залишок строку служби** — години або друки
- **Розрахункова дата закінчення** — на основі середнього використання за останні 30 днів
- **Довірчий інтервал** — межа невизначеності прогнозу

Натисніть на компонент для перегляду детального графіку накопичення зносу з часом.

## Сповіщення

Налаштуйте автоматичні сповіщення для кожного компонента:

1. Перейдіть до **Знос → Налаштування**
2. Для кожного компонента встановіть **Поріг сповіщення** (рекомендовано: 75% та 90%)
3. Виберіть канал сповіщення (дивіться [Сповіщення](../funksjoner/notifications))

**Приклад повідомлення:**
> ⚠️ Сопло (латунне) на Мій X1C зношено на 78%. Розрахунковий строк служби: ~45 годин. Рекомендується: Запланувати заміну сопла.

## Вартість обслуговування

Модуль зносу інтегрується з журналом витрат:

- **Вартість компонента** — ціна запасної частини
- **Загальна вартість заміни** — сума для всіх компонентів, що наближаються до ліміту
- **Прогноз на наступні 6 місяців** — розрахункові витрати на обслуговування вперед

Вкажіть ціни компонентів в розділі **Знос → Ціни**:

1. Натисніть **Встановити ціни**
2. Заповніть ціну за одиницю для кожного компонента
3. Ціна використовується в прогнозах витрат і може варіюватися залежно від моделі принтера

## Скидання лічильника зносу

Після обслуговування скиньте лічильник для відповідного компонента:

1. Перейдіть до **Знос → [Назва компонента]**
2. Натисніть **Позначити як замінений**
3. Заповніть:
   - Дату заміни
   - Вартість (необов'язково)
   - Нотатку (необов'язково)
4. Лічильник зносу скидається та перераховується

Скидання відображаються в історії обслуговування.

:::tip Калібрування
Порівняйте прогноз зносу з фактичними даними досвіду та відрегулюйте параметри строку служби в розділі **Знос → Налаштувати строк служби** для адаптації розрахунків до вашого фактичного використання.
:::
