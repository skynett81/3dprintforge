# Режим кіоску

Режим кіоску призначений для настінних екранів, телевізорів або спеціалізованих моніторів, що безперервно відображають статус принтерів — без клавіатури, взаємодії з мишею або браузерного інтерфейсу.

Перейдіть до: **https://localhost:3443/#settings** → **Система → Кіоск**

## Що таке режим кіоску

У режимі кіоску:
- Меню навігації приховане
- Жодних інтерактивних елементів управління не видно
- Дашборд оновлюється автоматично
- Екран ротує між принтерами (якщо налаштовано)
- Тайм-аут неактивності вимкнено

## Увімкнення режиму кіоску через URL

Додайте `?kiosk=true` до URL для увімкнення режиму кіоску без зміни налаштувань:

```
https://localhost:3443/?kiosk=true
https://localhost:3443/#fleet?kiosk=true
```

Режим кіоску вимикається видаленням параметра або додаванням `?kiosk=false`.

## Налаштування кіоску

1. Перейдіть до **Налаштування → Система → Кіоск**
2. Налаштуйте:

| Налаштування | Стандартне значення | Опис |
|---|---|---|
| Стандартний вигляд | Огляд парку | Яка сторінка відображається |
| Інтервал ротації | 30 секунд | Час на принтер в ротації |
| Режим ротації | Лише активні | Ротувати тільки між активними принтерами |
| Тема | Темна | Рекомендовано для екранів |
| Розмір шрифту | Великий | Читабельно з відстані |
| Годинник | Вимк | Показати годинник у куті |

## Огляд парку для кіоску

Огляд парку оптимізований для кіоску:

```
https://localhost:3443/#fleet?kiosk=true&cols=3&size=large
```

Параметри для огляду парку:
- `cols=N` — кількість стовпців (1–6)
- `size=small|medium|large` — розмір картки

## Ротація одного принтера

Для ротації між окремими принтерами (один принтер одночасно):

```
https://localhost:3443/?kiosk=true&rotate=true&interval=20
```

- `rotate=true` — увімкнути ротацію
- `interval=N` — секунди на принтер

## Налаштування на Raspberry Pi / NUC

Для спеціалізованого обладнання кіоску:

### Chromium в режимі кіоску (Linux)

```bash
chromium-browser \
  --kiosk \
  --noerrdialogs \
  --disable-infobars \
  --no-first-run \
  --app="https://localhost:3443/?kiosk=true"
```

Додайте команду до автозапуску (`~/.config/autostart/bambu-kiosk.desktop`).

### Автоматичний вхід та запуск

1. Налаштуйте автоматичний вхід у операційній системі
2. Створіть запис автозапуску для Chromium
3. Вимкніть заставку та енергозбереження:
   ```bash
   xset s off
   xset -dpms
   xset s noblank
   ```

:::tip Спеціалізований обліковий запис
Створіть спеціалізований обліковий запис Bambu Dashboard з роллю **Гість** для пристрою кіоску. Тоді пристрій матиме лише доступ для читання та не зможе змінювати налаштування, навіть якщо хтось отримає доступ до екрана.
:::

## Налаштування концентратора

Режим концентратора відображає сторінку огляду з усіма принтерами та ключовою статистикою — призначений для великих телевізорів:

```
https://localhost:3443/#hub?kiosk=true
```

Вигляд концентратора включає:
- Сітку принтерів зі статусом
- Агреговані ключові показники (активні друки, загальний прогрес)
- Годинник та дату
- Останні HMS-сповіщення
