# Звіти

Bambu Dashboard може надсилати автоматичні звіти електронною поштою зі статистикою та зведенням активності — щотижнево, щомісяця або обидва варіанти.

Перейдіть до: **https://localhost:3443/#settings** → **Система → Звіти**

## Передумови

Звіти вимагають налаштування сповіщень електронною поштою. Налаштуйте SMTP в **Налаштування → Сповіщення → Електронна пошта** перед активацією звітів. Дивіться [Сповіщення](../funksjoner/notifications).

## Активація автоматичних звітів

1. Перейдіть до **Налаштування → Звіти**
2. Активуйте **Щотижневий звіт** та/або **Щомісячний звіт**
3. Виберіть **Час відправлення**:
   - Щотижнево: день тижня та час
   - Щомісяця: день місяця (наприклад, 1-й понеділок / останній п'ятниця)
4. Заповніть **Електронна пошта отримувача** (через кому для кількох)
5. Натисніть **Зберегти**

Надішліть тестовий звіт для перевірки форматування: натисніть **Надіслати тестовий звіт зараз**.

## Вміст щотижневого звіту

Щотижневий звіт охоплює останні 7 днів:

### Зведення
- Загальна кількість друків
- Кількість успішних / невдалих / скасованих
- Відсоток успіху та зміна від попереднього тижня
- Найактивніший принтер

### Активність
- Друки на день (мінідіаграма)
- Загальна кількість годин друку
- Загальна витрата філаменту (грами та вартість)

### Філамент
- Споживання за матеріалом та постачальником
- Орієнтовний залишок за котушкою (котушки нижче 20 % виділені)

### Обслуговування
- Завдання обслуговування, виконані цього тижня
- Прострочені завдання обслуговування (червоне попередження)
- Завдання, що мають бути виконані наступного тижня

### HMS-помилки
- Кількість HMS-помилок цього тижня за принтером
- Невизнані помилки (потребують уваги)

## Вміст щомісячного звіту

Щомісячний звіт охоплює останні 30 днів та містить все з щотижневого звіту, плюс:

### Тренд
- Порівняння з попереднім місяцем (%)
- Карта активності (мініатюра теплової карти за місяць)
- Динаміка місячного відсотку успіху

### Витрати
- Загальна вартість філаменту
- Загальна вартість електроенергії (якщо налаштовано вимірювання)
- Загальна вартість зносу
- Сукупні витрати на обслуговування

### Знос та стан
- Оцінка стану за принтером (зі зміною від попереднього місяця)
- Компоненти, що наближаються до терміну заміни

### Статистичні highlights
- Найдовший успішний друк
- Найбільш використовуваний тип філаменту
- Принтер з найвищою активністю

## Налаштування звіту

1. Перейдіть до **Налаштування → Звіти → Налаштування**
2. Відмітьте / зніміть відмітку з розділів, які хочете включити
3. Виберіть **Фільтр принтерів**: всі принтери або вибір
4. Виберіть **Відображення логотипу**: показати логотип Bambu Dashboard у заголовку або вимкнути
5. Натисніть **Зберегти**

## Архів звітів

Всі надіслані звіти зберігаються та можуть бути відкриті повторно:

1. Перейдіть до **Налаштування → Звіти → Архів**
2. Виберіть звіт зі списку (сортується за датою)
3. Натисніть **Відкрити** для перегляду HTML-версії
4. Натисніть **Завантажити PDF** для завантаження звіту

Звіти автоматично видаляються через **90 днів** (налаштовується).
