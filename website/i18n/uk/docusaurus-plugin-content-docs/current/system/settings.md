# Налаштування

Всі налаштування 3DPrintForge зібрані на одній сторінці з чіткими категоріями. Ось огляд того, що знаходиться в кожній категорії.

Перейдіть до: **https://localhost:3443/#settings**

## Принтери

Управляйте зареєстрованими принтерами:

| Налаштування | Опис |
|---|---|
| Додати принтер | Зареєструйте новий принтер з серійним номером та кодом доступу |
| Назва принтера | Власна відображувана назва |
| Модель принтера | X1C / X1C Combo / X1E / P1S / P1S Combo / P1P / P2S / P2S Combo / A1 / A1 Combo / A1 mini / H2S / H2D / H2C |
| MQTT-підключення | Bambu Cloud MQTT або локальний MQTT |
| Код доступу | LAN Access Code з додатку Bambu Lab |
| IP-адреса | Для локального (LAN) режиму |
| Налаштування камери | Увімкнути/вимкнути, роздільна здатність |

Дивіться [Початкове налаштування](../getting-started/setup) для покрокового налаштування першого принтера.

## Сповіщення

Дивіться повну документацію в [Сповіщення](../features/notifications).

Короткий огляд:
- Увімкнути/вимкнути канали сповіщень (Telegram, Discord, електронна пошта тощо)
- Фільтр подій для кожного каналу
- Тихі години (часовий проміжок без сповіщень)
- Кнопка тесту для кожного каналу

## Тема

Дивіться повну документацію в [Тема](./themes).

- Світлий / Темний / Авто режим
- 6 кольорових палітр
- Власний акцентний колір
- Округлення та щільність

## OBS-оверлей

Конфігурація для OBS-оверлею:

| Налаштування | Опис |
|---|---|
| Стандартна тема | dark / light / minimal |
| Стандартна позиція | Кут для оверлею |
| Стандартний масштаб | Масштабування (0.5–2.0) |
| Показати QR-код | Показати QR-код дашборду в оверлеї |

Дивіться [OBS-оверлей](../features/obs-overlay) для повного синтаксису URL та налаштування.

## Енергія та електроенергія

| Налаштування | Опис |
|---|---|
| Tibber API Token | Доступ до споткцін Tibber |
| Цінова зона Nordpool | Виберіть норвезький ціновий регіон |
| Тариф мережі (грн/кВт·год) | Ваша ставка тарифу мережі |
| Потужність принтера (Вт) | Налаштуйте споживання потужності за моделлю принтера |

## Home Assistant

| Налаштування | Опис |
|---|---|
| MQTT-брокер | IP, порт, ім'я користувача, пароль |
| Префікс Discovery | Стандарт: `homeassistant` |
| Увімкнути discovery | Публікувати пристрої до HA |

## Вебхуки

Глобальні налаштування вебхуків:

| Налаштування | Опис |
|---|---|
| URL вебхуку | URL отримувача для подій |
| Секретний ключ | Підпис HMAC-SHA256 |
| Фільтр подій | Які події надсилаються |
| Спроби повтору | Кількість спроб при помилці (стандарт: 3) |
| Тайм-аут | Секунди до відмови запиту (стандарт: 10) |

## Налаштування черги

| Налаштування | Опис |
|---|---|
| Автоматичний dispatch | Увімкнути/вимкнути |
| Стратегія dispatch | Перший вільний / Найменш використовуваний / Кругова |
| Вимагати підтвердження | Ручне затвердження перед відправкою |
| Розподілений старт | Затримка між принтерами в черзі |

| Налаштування | Опис |

| Налаштування | Опис |
|---|---|
| Порт сервера | Стандарт: 3443 |

| Очищення бази даних | Автоматичне видалення старої історії |
