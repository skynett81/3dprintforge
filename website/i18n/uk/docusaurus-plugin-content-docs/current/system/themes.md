# Тема

Bambu Dashboard має гнучку систему тем, що дозволяє налаштовувати зовнішній вигляд під ваш смак та ситуацію використання.

Перейдіть до: **https://localhost:3443/#settings** → **Тема**

## Кольоровий режим

Вибирайте між трьома режимами:

| Режим | Опис |
|---|---|
| **Світлий** | Світлий фон, темний текст — добре в добре освітлених приміщеннях |
| **Темний** | Темний фон, світлий текст — стандарт та рекомендовано для моніторингу |
| **Авто** | Слідує налаштуванням операційної системи (темний/світлий ОС) |

Змінюйте режим у верхній частині налаштувань теми або через гарячу клавішу в панелі навігації (значок місяця/сонця).

## Кольорові палітри

Доступно шість попередньо налаштованих кольорових палітр:

| Палітра | Основний колір | Стиль |
|---|---|---|
| **Bambu** | Зелений (#00C853) | Стандарт, натхненний Bambu Lab |
| **Синя ніч** | Синій (#2196F3) | Спокійний та професійний |
| **Захід сонця** | Помаранчевий (#FF6D00) | Теплий та енергійний |
| **Фіолетовий** | Фіолетовий (#9C27B0) | Творчий та виразний |
| **Червоний** | Червоний (#F44336) | Висока контрастність, привертає увагу |
| **Монохромний** | Сірий (#607D8B) | Нейтральний та мінімалістичний |

Натисніть на палітру для негайного попереднього перегляду та активації.

## Власний акцентний колір

Використовуйте свій власний колір як акцентний:

1. Натисніть **Власний колір** під вибором палітри
2. Використовуйте вибір кольору або введіть hex-код (наприклад `#FF5722`)
3. Попередній перегляд оновлюється в реальному часі
4. Натисніть **Застосувати** для активації

:::tip Контрастність
Переконайтеся, що акцентний колір має хороший контраст з фоном. Система попереджає, якщо колір може викликати проблеми з читабельністю (стандарт WCAG AA).
:::

## Округлення

Відрегулюйте округлення кнопок, карток та елементів:

| Налаштування | Опис |
|---|---|
| **Гострий** | Без округлення (прямокутний стиль) |
| **Малий** | Ледь помітне округлення (4 пікселі) |
| **Середній** | Стандартне округлення (8 пікселів) |
| **Великий** | Виразне округлення (16 пікселів) |
| **Pill** | Максимальне округлення (50 пікселів) |

Переміщуйте повзунок для ручного налаштування між 0–50 пікселями.

## Щільність

Налаштовуйте щільність інтерфейсу:

| Налаштування | Опис |
|---|---|
| **Просторий** | Більше повітря між елементами |
| **Стандартний** | Збалансований, стандартне налаштування |
| **Компактний** | Щільніше пакування — більше інформації на екрані |

Компактний режим рекомендований для екранів нижче 1080p або вигляду кіоску.

## Типографія

Виберіть шрифт:

- **System** — використовує стандартний шрифт операційної системи (швидко завантажується)
- **Inter** — чіткий та сучасний (стандартний вибір)
- **JetBrains Mono** — моноширинний, підходить для числових значень
- **Nunito** — м'якший та більш округлений стиль

## Анімації

Вимкніть або налаштуйте анімації:

- **Повні** — всі переходи та анімації активні (стандарт)
- **Зменшені** — лише необхідні анімації (враховує налаштування ОС)
- **Вимкнені** — без анімацій для максимальної продуктивності

:::tip Режим кіоску
Для вигляду кіоску увімкніть **Компактний** + **Темний** + **Зменшені анімації** для оптимальної продуктивності та читабельності з відстані. Дивіться [Режим кіоску](./kiosk).
:::

## Експорт та імпорт налаштувань теми

Діліться темою з іншими:

1. Натисніть **Експортувати тему** — завантажується `.json`-файл
2. Поділіться файлом з іншими користувачами Bambu Dashboard
3. Вони імпортують через **Імпортувати тему** → виберіть файл
