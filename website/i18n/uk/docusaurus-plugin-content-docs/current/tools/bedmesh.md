# Bed Mesh

Інструмент Bed mesh надає візуальне представлення рівності платформи друку — що є ключовим для хорошої адгезії та рівного першого шару.

Перейдіть до: **https://localhost:3443/#bedmesh**

## Що таке bed mesh?

Принтери Bambu Lab сканують поверхню платформи друку за допомогою зонда та створюють карту (mesh) відхилень висоти. Прошивка принтера автоматично компенсує відхилення під час друку. 3DPrintForge візуалізує цю карту для вас.

## Візуалізація

### 3D-поверхня

Карта bed mesh відображається як інтерактивна 3D-поверхня:

- Використовуйте мишу для обертання вигляду
- Прокрутка для масштабування
- Натисніть **Вигляд зверху** для огляду з пташиного польоту
- Натисніть **Бічний вигляд** для перегляду профілю

Кольорова шкала показує відхилення від середньої висоти:
- **Синій** — нижче центру (вгнутий)
- **Зелений** — приблизно рівний (відхилення < 0.1 мм)
- **Жовтий** — помірне відхилення (0.1–0.2 мм)
- **Червоний** — велике відхилення (> 0.2 мм)

### Теплова карта

Натисніть **Теплова карта** для плоского 2D-вигляду карти mesh — простіше читати для більшості людей.

Теплова карта показує:
- Точні значення відхилень (мм) для кожної точки вимірювання
- Помічені проблемні точки (відхилення > 0.3 мм)
- Розміри сітки (кількість рядків × стовпців)

## Сканування bed mesh з інтерфейсу

:::warning Вимоги
Для сканування принтер повинен бути вільний, а температура столу — стабілізована. Прогрійте стіл до потрібної температури ПЕРЕД скануванням.
:::

1. Перейдіть до **Bed Mesh**
2. Виберіть принтер зі списку
3. Натисніть **Сканувати зараз**
4. Виберіть температуру столу для сканування:
   - **Холодний** (кімнатна температура) — швидко, але менш точно
   - **Гарячий** (50–60°C PLA, 70–90°C PETG) — рекомендовано
5. Підтвердіть у діалозі — принтер автоматично починає послідовність зондування
6. Зачекайте завершення сканування (3–8 хвилин залежно від розміру mesh)
7. Нова карта mesh відображається автоматично

## Посібник з калібрування

Після сканування система надає конкретні рекомендації:

| Знахідка | Рекомендація |
|---|---|
| Відхилення < 0.1 мм всюди | Відмінно — жодних дій не потрібно |
| Відхилення 0.1–0.2 мм | Добре — компенсація виконується прошивкою |
| Відхилення > 0.2 мм у кутах | Відрегулюйте пружини столу вручну (якщо можливо) |
| Відхилення > 0.3 мм | Стіл може бути пошкоджений або неправильно встановлений |
| Центр вищий за кути | Теплове розширення — нормально для гарячих столів |

:::tip Порівняння з попереднім
Натисніть **Порівняти з попереднім** для перевірки зміни карти mesh з часом — корисно для виявлення поступового деформування платформи.
:::

## Історія mesh

Всі сканування mesh зберігаються з позначкою часу:

1. Натисніть **Історія** на бічній панелі bed mesh
2. Виберіть два сканування для порівняння (відображається карта різниць)
3. Видаліть старі сканування, які вам більше не потрібні

## Експорт

Експортуйте дані mesh у вигляді:
- **PNG** — зображення теплової карти (для документації)
- **CSV** — необроблені дані з X, Y та відхиленням висоти для кожної точки
