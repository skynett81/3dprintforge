# Етикетки

Інструмент етикеток генерує професійні етикетки для котушок філаменту — QR-коди, етикетки котушок для термічних принтерів та картки кольорів для візуальної ідентифікації.

Перейдіть до: **https://localhost:3443/#labels**

## QR-коди

Генеруйте QR-коди, що посилаються на інформацію про філамент у дашборді:

1. Перейдіть до **Етикетки → QR-коди**
2. Виберіть котушку, для якої хочете згенерувати QR-код
3. QR-код генерується автоматично та відображається у попередньому перегляді
4. Натисніть **Завантажити PNG** або **Друкувати**

QR-код містить URL до профілю філаменту у дашборді. Скануйте телефоном для швидкого доступу до інформації про котушку.

### Пакетна генерація

1. Натисніть **Вибрати все** або відмітьте окремі котушки
2. Натисніть **Згенерувати всі QR-коди**
3. Завантажте як ZIP з одним PNG на котушку, або роздрукуйте всі одразу

## Етикетки котушок

Професійні етикетки для термічних принтерів з повною інформацією про котушку:

### Вміст етикетки (стандарт)

- Колір котушки (заповнений кольоровий блок)
- Назва матеріалу (великий шрифт)
- Постачальник
- Hex-код кольору
- Температурні рекомендації (сопло та стіл)
- QR-код
- Штрих-код (необов'язково)

### ZPL для термічних принтерів

Генеруйте ZPL-код (Zebra Programming Language) для принтерів Zebra, Brother та Dymo:

1. Перейдіть до **Етикетки → Термічний друк**
2. Виберіть розмір етикетки: **25×54 мм** / **36×89 мм** / **62×100 мм**
3. Виберіть котушку(и)
4. Натисніть **Згенерувати ZPL**
5. Надішліть ZPL-код на принтер через:
   - **Друкувати напряму** (USB-підключення)
   - **Копіювати ZPL** та надіслати через команду терміналу
   - **Завантажити .zpl-файл**

:::tip Налаштування принтера
Для автоматичного друку налаштуйте принтер-станцію в **Налаштування → Принтер етикеток** з IP-адресою та портом (стандарт: 9100 для RAW TCP).
:::

### PDF-етикетки

Для звичайних принтерів генеруйте PDF з правильними розмірами:

1. Виберіть розмір етикетки з шаблону
2. Натисніть **Згенерувати PDF**
3. Роздрукуйте на самоклеючому папері (Avery або аналоги)

## Картки кольорів

Картки кольорів — це компактна сітка, що візуально показує всі котушки:

1. Перейдіть до **Етикетки → Картки кольорів**
2. Виберіть котушки для включення (всі активні, або виберіть вручну)
3. Виберіть формат картки: **A4** (4×8), **A3** (6×10), **Letter**
4. Натисніть **Згенерувати PDF**

Кожне поле показує:
- Кольоровий блок з фактичним кольором
- Назву матеріалу та hex-код кольору
- Номер матеріалу (для швидкого посилання)

Ідеально заламінувати та повісити біля принтера.

## Спільні кольорові палітри

Експортуйте вибір кольорів як спільну палітру:

1. Перейдіть до **Етикетки → Кольорові палітри**
2. Виберіть котушки для включення в палітру
3. Натисніть **Поділитися палітрою**
4. Скопіюйте посилання — інші можуть імпортувати палітру до свого дашборду
5. Палітра відображається з hex-кодами та може бути експортована до **Adobe Swatch** (`.ase`) або **Procreate** (`.swatches`)
