# Досягнення

Досягнення (achievements) — це елемент гейміфікації, який нагороджує за важливі моменти та цікаві події у вашому шляху до 3D-друку. Збирайте досягнення та спостерігайте прогрес до наступного розблокування.

Перейдіть до: **https://localhost:3443/#achievements**

## Ступені рідкості

Досягнення класифікуються за чотирма ступенями рідкості:

| Ступінь | Колір | Опис |
|---|---|---|
| **Звичайний** | Сірий | Прості віхи, легко досягти |
| **Незвичайний** | Зелений | Потребує певних зусиль або часу |
| **Рідкісний** | Синій | Потребує цілеспрямованих зусиль з часом |
| **Легендарний** | Золотий | Виняткові досягнення |

## Приклади досягнень

### Друкарські віхи (Звичайний / Незвичайний)
| Досягнення | Вимога |
|---|---|
| Перший друк | Завершіть свій перший друк |
| Цілий день | Друкуйте більше 24 годин загалом |
| Висока успішність | 10 успішних друків поспіль |
| Колекціонер філаменту | Зареєструйте 10 різних типів філаменту |
| Різнокольоровий | Завершіть багатокольоровий друк |

### Об'ємні досягнення (Незвичайний / Рідкісний)
| Досягнення | Вимога |
|---|---|
| Кілограм | Використайте 1 кг філаменту загалом |
| 10 кг | Використайте 10 кг філаменту загалом |
| 100 друків | 100 успішних друків |
| 500 годин | 500 накопичених годин друку |
| Нічна зміна | Завершіть друк тривалістю понад 20 годин |

### Обслуговування та турбота (Незвичайний / Рідкісний)
| Досягнення | Вимога |
|---|---|
| Сумлінний | Зафіксуйте завдання технічного обслуговування |
| Доглядач принтера | 10 зафіксованих завдань обслуговування |
| Без відходів | Створіть друк з ефективністю матеріалу > 90 % |
| Майстер сопел | Замініть сопло 5 разів (задокументовано) |

### Легендарні досягнення
| Досягнення | Вимога |
|---|---|
| Невтомний | 1000 успішних друків |
| Титан філаменту | 50 кг загального споживання філаменту |
| Тиждень без помилок | 7 днів без жодного невдалого друку |
| Бібліотекар друку | 100 різних моделей у бібліотеці файлів |

## Перегляд досягнень

Сторінка досягнень показує:

- **Розблоковані** — досягнення, яких ви досягли (з датою)
- **Близькі** — досягнення, до яких ви близькі (індикатор прогресу)
- **Заблоковані** — всі досягнення, яких ви ще не досягли

Фільтруйте за **Ступенем**, **Категорією** або **Статусом** (розблоковано / в процесі / заблоковано).

## Індикатор прогресу

Для досягнень з підрахунком відображається індикатор прогресу:

```
Кілограм — 1 кг філаменту
[████████░░] 847 г / 1000 г (84.7 %)
```

## Сповіщення

Ви автоматично отримуєте сповіщення при досягненні нового досягнення:
- **Спливаюче вікно браузера** з назвою досягнення та зображенням
- Необов'язково: сповіщення через Telegram / Discord (налаштуйте в **Налаштування → Сповіщення → Досягнення**)

## Підтримка кількох користувачів

У системах з кількома користувачами кожен користувач має власний профіль досягнень. **Таблиця лідерів** (leaderboard) показує рейтинг за:

- Загальна кількість розблокованих досягнень
- Загальна кількість друків
- Загальна кількість годин друку

:::tip Приватний режим
Вимкніть таблицю лідерів в **Налаштування → Досягнення → Приховати з таблиці лідерів** для приватності профілю.
:::
