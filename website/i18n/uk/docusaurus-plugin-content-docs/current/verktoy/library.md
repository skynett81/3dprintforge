# Бібліотека файлів

Бібліотека файлів — це центральне місце для зберігання та управління всіма вашими 3D-моделями та файлами G-коду — з автоматичним аналізом G-коду та інтеграцією з MakerWorld і Printables.

Перейдіть до: **https://localhost:3443/#library**

## Завантаження моделей

### Одиночне завантаження

1. Перейдіть до **Бібліотека файлів**
2. Натисніть **Завантажити** або перетягніть файли в область завантаження
3. Підтримувані формати: `.3mf`, `.gcode`, `.bgcode`, `.stl`, `.obj`
4. Файл автоматично аналізується після завантаження

:::info Папка зберігання
Файли зберігаються в папці, налаштованій в **Налаштування → Бібліотека файлів → Папка зберігання**. Стандарт: `./data/library/`
:::

### Пакетне завантаження

Перетягніть цілу папку для завантаження всіх підтримуваних файлів одразу. Файли обробляються у фоновому режимі, і ви отримаєте сповіщення, коли все буде готово.

## Аналіз G-коду

Після завантаження файли `.gcode` та `.bgcode` аналізуються автоматично:

| Показник | Опис |
|---|---|
| Орієнтовний час друку | Час, розрахований з команд G-коду |
| Витрата філаменту | Грами та метри за матеріалом/кольором |
| Кількість шарів | Загальна кількість шарів |
| Висота шару | Зареєстрована висота шару |
| Матеріали | Виявлені матеріали (PLA, PETG тощо) |
| Відсоток заповнення | Якщо доступно в метаданих |
| Матеріал підтримок | Орієнтовна вага підтримок |
| Модель принтера | Цільовий принтер з метаданих |

Дані аналізу відображаються на картці файлу та використовуються [Калькулятором витрат](../analyse/costestimator).

## Картки файлів та метадані

Кожна картка файлу показує:
- **Назву файлу** та формат
- **Дату завантаження**
- **Мініатюру** (з `.3mf` або згенеровану)
- **Аналізований час друку** та витрату філаменту
- **Теги** та категорію
- **Пов'язані друки** — кількість разів, коли друкувалось

Натисніть на картку для відкриття детального вигляду з повними метаданими та історією.

## Організація

### Теги

Додавайте теги для зручного пошуку:
1. Натисніть на файл → **Редагувати метадані**
2. Введіть теги (через кому): `benchy, тест, PLA, калібрування`
3. Шукайте в бібліотеці за фільтром тегів

### Категорії

Організовуйте файли за категоріями:
- Натисніть **Нова категорія** на бічній панелі
- Перетягніть файли до категорії
- Категорії можуть бути вкладеними (підкатегорії підтримуються)

## Зв'язок з MakerWorld

1. Перейдіть до **Налаштування → Інтеграції → MakerWorld**
2. Увійдіть за допомогою акаунту Bambu Lab
3. Поверніться до бібліотеки: натисніть на файл → **Зв'язати з MakerWorld**
4. Знайдіть модель на MakerWorld та виберіть правильний результат
5. Метадані (дизайнер, ліцензія, рейтинг) імпортуються з MakerWorld

Зв'язок показує ім'я дизайнера та оригінальний URL на картці файлу.

## Зв'язок з Printables

1. Перейдіть до **Налаштування → Інтеграції → Printables**
2. Вставте ваш API-ключ Printables
3. Зв'яжіть файли з моделями Printables так само, як з MakerWorld

## Надсилання на принтер

З бібліотеки файлів ви можете надсилати безпосередньо на принтер:

1. Натисніть на файл → **Надіслати на принтер**
2. Виберіть цільовий принтер
3. Виберіть слот AMS (для багатокольорових друків)
4. Натисніть **Почати друк** або **Додати до черги**

:::warning Пряме надсилання
Пряме надсилання негайно починає друк без підтвердження в Bambu Studio. Переконайтеся, що принтер готовий.
:::
