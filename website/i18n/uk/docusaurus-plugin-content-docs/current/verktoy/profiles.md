# Профілі друку

Профілі друку — це збережені набори налаштувань друку, які ви можете використовувати повторно для різних друків та принтерів. Економте час і забезпечуйте стабільну якість, визначивши профілі для різних цілей.

Перейдіть до: **https://localhost:3443/#profiles**

## Створення профілю

1. Перейдіть до **Інструменти → Профілі друку**
2. Натисніть **Новий профіль** (значок +)
3. Заповніть:
   - **Назва профілю** — описова назва, наприклад «PLA - Швидке виробництво»
   - **Матеріал** — виберіть зі списку (PLA / PETG / ABS / PA / PC / TPU тощо)
   - **Модель принтера** — X1C / X1C Combo / X1E / P1S / P1S Combo / P1P / P2S / P2S Combo / A1 / A1 Combo / A1 mini / H2S / H2D / H2C / Всі
   - **Опис** — необов'язковий текст

4. Заповніть налаштування (дивіться розділи нижче)
5. Натисніть **Зберегти профіль**

## Налаштування в профілі

### Температура
| Поле | Приклад |
|---|---|
| Температура сопла | 220°C |
| Температура столу | 60°C |
| Температура камери (X1C) | 35°C |

### Швидкість
| Поле | Приклад |
|---|---|
| Параметр швидкості | Стандартний |
| Макс. швидкість (мм/с) | 200 |
| Прискорення | 5000 мм/с² |

### Якість
| Поле | Приклад |
|---|---|
| Висота шару | 0.2 мм |
| Відсоток заповнення | 15 % |
| Шаблон заповнення | Grid |
| Матеріал підтримок | Авто |

### AMS та кольори
| Поле | Опис |
|---|---|
| Об'єм очищення | Кількість промивки при зміні кольору |
| Бажані слоти | Які слоти AMS надаються перевага |

### Додатково
| Поле | Опис |
|---|---|
| Режим сушіння | Увімкнути сушіння AMS для вологих матеріалів |
| Час охолодження | Пауза між шарами для охолодження |
| Швидкість вентилятора | Швидкість вентилятора охолодження у відсотках |

## Редагування профілю

1. Натисніть на профіль у списку
2. Натисніть **Редагувати** (значок олівця)
3. Внесіть зміни
4. Натисніть **Зберегти** (перезаписати) або **Зберегти як новий** (створює копію)

:::tip Версіонування
Використовуйте «Зберегти як новий» для збереження робочого профілю під час експериментів зі змінами.
:::

## Використання профілю

### З бібліотеки файлів

1. Виберіть файл у бібліотеці
2. Натисніть **Надіслати на принтер**
3. Виберіть **Профіль** зі списку
4. Використовуються налаштування профілю

### З черги друку

1. Створіть нове завдання черги
2. Виберіть **Профіль** у налаштуваннях
3. Профіль прив'язується до завдання черги

## Імпорт та експорт профілів

### Експорт
1. Виберіть один або кілька профілів
2. Натисніть **Експортувати**
3. Виберіть формат: **JSON** (для імпорту в інші дашборди) або **PDF** (для друку/документації)

### Імпорт
1. Натисніть **Імпортувати профілі**
2. Виберіть `.json`-файл, експортований з іншого Bambu Dashboard
3. Існуючі профілі з однаковою назвою можна перезаписати або зберегти обидва

## Поділитися профілями

Діліться профілями з іншими через модуль спільнотних філаментів (дивіться [Спільнотні філаменти](../integrasjoner/community)) або через прямий JSON-експорт.

## Профіль за замовчуванням

Встановіть профіль за замовчуванням для кожного матеріалу:

1. Виберіть профіль
2. Натисніть **Встановити як стандартний для [матеріал]**
3. Профіль за замовчуванням вибирається автоматично при надсиланні файлу з цим матеріалом
