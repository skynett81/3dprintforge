# 活动日历

活动日历以可视化方式展示全年的打印机活动情况，灵感来自 GitHub 的贡献概览。

访问地址：**https://localhost:3443/#calendar**

## 热力图概览

日历以 365 天（52 周）的彩色格子网格方式显示：

- **灰色** — 当天无打印记录
- **浅绿色** — 1–2 次打印
- **绿色** — 3–5 次打印
- **深绿色** — 6–10 次打印
- **墨绿色** — 11 次及以上打印

格子按星期纵向排列（周一至周日），从左（一月）到右（十二月）横向排列。

:::tip 颜色编码
可通过日历上方的选择器将热力图指标从**打印次数**切换为**小时数**或**消耗耗材克数**。
:::

## 年份选择

点击 **< 年份 >** 在不同年份之间切换：

- 所有有打印记录的年份均可访问
- 默认显示当前年份
- 未来日期显示为灰色（无数据）

## 单日详情

点击某个格子查看该日的详细信息：

- **日期**和星期
- **打印次数** — 成功和失败的数量
- **消耗耗材总量**（克）
- **总打印时间**
- **打印列表** — 点击可在历史记录中打开

## 月度概览

热力图下方显示月度概览，包含：
- 每月打印总数的柱状图
- 本月最活跃的一天
- 与去年同月的对比（百分比）

## 打印机筛选

从顶部下拉列表中选择打印机，仅显示该台打印机的活动记录；或选择**全部**查看汇总视图。

多打印机视图可通过点击显示选择器中的**堆叠**来以堆叠方式显示颜色。

## 连续记录与历史最佳

日历下方显示：

| 统计项 | 说明 |
|---|---|
| **最长连续** | 至少有一次打印的最长连续天数 |
| **当前连续** | 正在进行的活跃天数连续记录 |
| **最活跃的一天** | 打印次数最多的一天 |
| **最活跃的一周** | 打印次数最多的一周 |
| **最活跃的一月** | 打印次数最多的月份 |

## 导出

点击**导出**下载日历数据：

- **PNG** — 热力图图片（用于分享）
- **CSV** — 原始数据，每行一天（日期、次数、克数、小时数）

PNG 导出已针对社交媒体分享进行优化，以打印机名称和年份作为副标题。
