# 耗材分析

耗材分析为您提供完整的耗材消耗洞察——您使用了什么、花了多少钱，以及可以节省多少。

访问地址：**https://localhost:3443/#filament-analytics**

## 消耗概览

顶部显示所选时间段的摘要：

- **总消耗量** — 所有材料的克数和米数
- **估算费用** — 基于每卷耗材的注册价格
- **最常用材料** — 类型和供应商
- **再利用率** — 实际模型用料与支撑/清洗用料的比例

### 按材料分类的消耗量

饼图和表格展示各材料的分布情况：

| 列 | 说明 |
|---|---|
| 材料 | PLA、PETG、ABS、PA 等 |
| 供应商 | 拓竹、PolyMaker、Prusament 等 |
| 使用克数 | 总重量 |
| 米数 | 估算长度 |
| 费用 | 克数 × 每克价格 |
| 打印次数 | 使用该材料的打印次数 |

点击某行可深入查看单卷耗材级别的详情。

## 消耗率

消耗率显示每单位时间的平均耗材消耗量：

- **每小时克数** — 活跃打印期间
- **每周克数** — 含打印机停机时间
- **每次打印克数** — 每次打印的平均值

这些数据用于预测未来的需求。

:::tip 采购计划
使用消耗率规划耗材库存。当估计库存将在 14 天内耗尽时，系统会自动发出警报（可配置）。
:::

## 费用预测

基于历史消耗率计算：

- **未来 30 天的预计消耗量**（每种材料的克数）
- **未来 30 天的预计费用**
- **建议库存量**（足够 30 / 60 / 90 天使用）

若有至少一年的数据，预测还会考虑季节性变化。

## 废料与效率

完整文档请参见[废料追踪](./waste)。耗材分析显示摘要：

- **AMS 清洗** — 克数及占总消耗量的比例
- **支撑材料** — 克数及比例
- **实际模型用料** — 剩余比例（效率%）
- **废料估算费用** — 废料的实际花费

## 耗材卷日志

所有耗材卷（活跃和已用完的）均有记录：

| 字段 | 说明 |
|---|---|
| 耗材卷名称 | 材料名称和颜色 |
| 初始重量 | 开始使用时记录的重量 |
| 剩余重量 | 计算得出的剩余量 |
| 已使用 | 已使用的总克数 |
| 最后使用时间 | 最后一次打印的日期 |
| 状态 | 活跃 / 已用完 / 已存储 |

## 价格记录

为了进行准确的费用分析，请记录每卷耗材的价格：

1. 进入**耗材库**
2. 点击某卷耗材 → **编辑**
3. 填写**购买价格**和**购买时重量**
4. 系统将自动计算每克价格

未记录价格的耗材卷将使用**默认每克价格**（在**设置 → 耗材 → 默认价格**中设置）。

## 导出

1. 点击**导出耗材数据**
2. 选择时间段和格式（CSV / PDF）
3. CSV 包含每次打印的一行数据，含克数、费用和材料信息
