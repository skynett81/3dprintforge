# 打印对比

打印对比功能允许您并排分析两次打印——便于比较设置、材料、打印机或同一模型的不同版本。

访问地址：**https://localhost:3443/#comparison**

## 选择要对比的打印

1. 进入**打印对比**
2. 点击**选择打印 A** 并在历史记录中搜索
3. 点击**选择打印 B** 并在历史记录中搜索
4. 点击**对比**以加载对比视图

:::tip 快速访问
在**历史记录**中，您可以右键点击某次打印，选择**设为打印 A** 或**与…对比**，直接跳转到对比模式。
:::

## 指标对比

指标以两列（A 和 B）显示，并标记出哪一项更优：

| 指标 | 说明 |
|---|---|
| 成功状态 | 已完成 / 已中断 / 失败 |
| 时长 | 总打印时间 |
| 耗材消耗 | 总克数及每种颜色的用量 |
| 耗材效率 | 模型用料占总消耗的百分比 |
| 最高喷嘴温度 | 记录到的最高喷嘴温度 |
| 最高热床温度 | 记录到的最高热床温度 |
| 速度设置 | 静音 / 标准 / 运动 / 涡轮 |
| AMS 换色次数 | 颜色切换次数 |
| HMS 错误 | 打印过程中的错误记录 |
| 打印机 | 使用的打印机型号 |

最优值的单元格以绿色背景显示。

## 温度图表

两个温度图表并排（或叠加）显示：

- **分离显示** — A 图表在左，B 图表在右
- **叠加显示** — 两者在同一图表中以不同颜色显示

使用叠加显示可直接比较温度稳定性和升温速度。

## 相册图片

若两次打印均有里程碑截图，则以网格形式显示：

| 打印 A | 打印 B |
|---|---|
| 25% 图片 A | 25% 图片 B |
| 50% 图片 A | 50% 图片 B |
| 75% 图片 A | 75% 图片 B |
| 100% 图片 A | 100% 图片 B |

点击图片可以带有幻灯片动画的全屏预览方式打开。

## 延时摄影对比

若两次打印均有延时摄影，则并排显示视频：

- 同步播放 — 两者同时开始和暂停
- 独立播放 — 分别控制每个视频

## 设置差异

系统自动突出显示打印设置的差异（从 G-code 元数据中提取）：

- 不同的层高
- 不同的填充图案或百分比
- 不同的支撑设置
- 不同的速度配置

差异在设置表格中以橙色标记显示。

## 保存对比

1. 点击**保存对比**
2. 为对比结果命名（例如："PLA vs PETG - Benchy"）
3. 对比结果将被保存，可通过**历史记录 → 对比**访问

## 导出

1. 点击**导出**
2. 选择 **PDF** 以生成包含所有指标和图片的报告
3. 该报告可关联到项目，用于记录材料选择决策
