# 费用估算器

费用估算器允许您在将任务发送到打印机之前，根据耗材消耗、电价和机器损耗来估算一次打印的总费用。

访问地址：**https://localhost:3443/#cost-estimator**

## 上传文件

1. 进入**费用估算器**
2. 将文件拖放到上传区域，或点击**选择文件**
3. 支持的格式：`.3mf`、`.gcode`、`.bgcode`
4. 点击**分析**

:::info 分析
系统将分析 G-code 以提取耗材消耗、预计打印时间和材料配置文件。通常需要 2–10 秒。
:::

## 耗材费用计算

分析完成后显示：

| 字段 | 示例值 |
|---|---|
| 预计耗材用量 | 47.3 g |
| 材料（来自文件） | PLA |
| 每克价格 | 0.025 元（来自耗材库） |
| **耗材费用** | **1.18 元** |

在下拉列表中切换材料，可比较不同耗材类型或供应商的费用。

:::tip 材料覆盖
如果 G-code 不包含材料信息，请从列表中手动选择材料。价格将自动从耗材库中获取。
:::

## 电费计算

电费基于以下内容计算：

- **预计打印时间** — 来自 G-code 分析
- **打印机功率** — 按打印机型号配置（瓦特）
- **电价** — 固定价格（元/千瓦时）或来自 Tibber/Nordpool 的实时价格

| 字段 | 示例值 |
|---|---|
| 预计打印时间 | 3 小时 22 分钟 |
| 打印机功率 | 350 W（X1C） |
| 预计用电量 | 1.17 千瓦时 |
| 电价 | 1.85 元/千瓦时 |
| **电费** | **2.16 元** |

激活 Tibber 或 Nordpool 集成，可根据所需开始时间使用预计的按小时电价。

## 机器损耗

损耗费用基于以下内容估算：

- 打印时间 × 每台打印机型号的每小时费用
- 研磨性材料（CF、GF 等）的额外损耗

| 字段 | 示例值 |
|---|---|
| 打印时间 | 3 小时 22 分钟 |
| 每小时损耗费用 | 0.80 元/小时 |
| **损耗费用** | **2.69 元** |

每小时费用根据零部件价格和预期使用寿命计算（参见[磨损预测](../monitoring/wearprediction)）。

## 总费用

| 费用项目 | 金额 |
|---|---|
| 耗材 | 1.18 元 |
| 电费 | 2.16 元 |
| 机器损耗 | 2.69 元 |
| **合计** | **6.03 元** |
| + 利润加成（30%） | 1.81 元 |
| **建议售价** | **7.84 元** |

在百分比字段中调整加成率，以计算建议的客户售价。

## 保存估算

点击**保存估算**将分析结果关联到项目：

1. 选择现有项目或创建新项目
2. 估算结果将被保存，可作为发票的依据
3. 实际费用（打印完成后）将自动与估算结果进行对比

## 批量计算

同时上传多个文件，以计算一套完整作品的总费用：

1. 点击**批量模式**
2. 上传所有 `.3mf`/`.gcode` 文件
3. 系统将分别计算各项费用并汇总
4. 将摘要导出为 PDF 或 CSV
