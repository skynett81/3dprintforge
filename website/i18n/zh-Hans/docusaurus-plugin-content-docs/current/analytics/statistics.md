# 统计数据

统计页面为您提供打印机活动的完整概览，包含关键指标、趋势和可自定义时间段的耗材消耗数据。

访问地址：**https://localhost:3443/#statistics**

## 关键指标

页面顶部显示四个 KPI 卡片：

| 关键指标 | 说明 |
|---|---|
| **成功率** | 成功打印次数占总打印次数的比例 |
| **耗材总量** | 所选时间段内使用的克数 |
| **总打印时间** | 累计打印时间 |
| **平均打印时间** | 每次打印的中位时长 |

每个关键指标都显示与上一时间段的变化（↑ 上升 / ↓ 下降），以百分比偏差表示。

## 成功率

成功率按打印机和总体分别计算：

- **成功** — 打印完成，无中断
- **已中断** — 用户手动停止
- **失败** — 被打印守护、HMS 错误或硬件故障停止

点击成功率图表，查看哪些打印失败及其原因。

:::tip 提高成功率
使用[错误模式分析](../monitoring/erroranalysis)识别并纠正打印失败的原因。
:::

## 趋势

趋势视图以折线图显示随时间的发展变化：

1. 选择**时间段**：最近 7 / 30 / 90 / 365 天
2. 选择**分组方式**：日 / 周 / 月
3. 选择**指标**：打印次数 / 小时数 / 克数 / 成功率
4. 点击**对比**以叠加两个指标

图表支持缩放（滚动）和平移（点击并拖动）。

## 耗材消耗

耗材消耗以以下方式显示：

- **柱状图** — 每日/周/月的消耗量
- **饼图** — 各材料（PLA、PETG、ABS 等）的分布
- **表格** — 详细列表，含每种材料的总克数、米数和费用

### 按打印机分类的消耗量

使用顶部的多选筛选器：
- 仅显示一台打印机
- 并排比较两台打印机
- 查看所有打印机的汇总数据

## 活动日历

在统计页面直接查看紧凑型 GitHub 风格热力图（简化视图），或前往完整的[活动日历](./calendar)查看更详细的视图。

## 导出

1. 点击**导出统计数据**
2. 选择日期范围和要包含的指标
3. 选择格式：**CSV**（原始数据）、**PDF**（报告）或 **JSON**
4. 文件将下载到本地

CSV 导出与 Excel 和 Google 表格兼容，可进行进一步分析。

## 与上一时间段对比

激活**显示上一时间段**以将图表与相应的上一时间段叠加：

- 最近 30 天 vs. 之前的 30 天
- 当月 vs. 上月
- 当年 vs. 去年

这样可以轻松判断您的打印量是否比以前增加或减少，以及成功率是否有所提升。
