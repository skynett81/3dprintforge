# 废料追踪

废料追踪为您提供完整的洞察，了解在打印过程中有多少耗材被浪费——AMS 清洗、材料切换时的冲洗以及支撑材料——以及这些废料的实际花费。

访问地址：**https://localhost:3443/#waste**

## 废料分类

3DPrintForge 区分三类废料：

| 类别 | 来源 | 典型用量 |
|---|---|---|
| **AMS 清洗** | 多色打印中 AMS 换色时产生 | 每次切换 5–30 克 |
| **材料切换冲洗** | 在不同材料之间切换时的清洗 | 每次切换 10–50 克 |
| **支撑材料** | 打印后需要拆除的支撑结构 | 因情况而异 |

## AMS 清洗追踪

AMS 清洗数据直接来自 MQTT 遥测和 G-code 分析：

- **每次换色的克数** — 从 G-code 清洗块中计算
- **换色次数** — 从打印日志中统计
- **清洗总消耗量** — 所选时间段内的总量

:::tip 减少清洗量
Bambu Studio 提供按颜色组合设置清洗量的功能。对于颜色差异较小的颜色对（例如白色→浅灰色），减少清洗量可以节省耗材。
:::

## 效率计算

效率计算公式：

```
效率 % = (模型用料 / 总消耗量) × 100

总消耗量 = 模型用料 + 清洗用料 + 支撑材料
```

**示例：**
- 模型：45 克
- 清洗：12 克
- 支撑：8 克
- 总计：65 克
- **效率：69%**

效率以趋势图的形式随时间显示，帮助您了解是否有所改善。

## 废料费用

基于已记录的耗材价格计算：

| 项目 | 计算方式 |
|---|---|
| 清洗费用 | 清洗克数 × 每克价格（按颜色） |
| 支撑费用 | 支撑克数 × 每克价格 |
| **废料总费用** | 以上各项之和 |
| **每次成功打印的费用** | 废料总费用 / 打印次数 |

## 按打印机和材料分类的废料

按以下条件筛选视图：

- **打印机** — 查看哪台打印机产生的废料最多
- **材料** — 查看每种耗材类型的废料
- **时间段** — 日、周、月、年

表格视图按废料量从高到低排列，包含估算费用。

## 优化建议

系统会自动生成减少废料的建议：

- **翻转换色顺序** — 如果颜色 A→B 的清洗量多于 B→A，系统会建议反转顺序
- **合并换色层** — 将具有相同颜色的层分组，以减少切换次数
- **支撑结构优化** — 通过改变模型方向来估算支撑材料的减少量

:::info 精确度
清洗计算是从 G-code 估算的。由于打印机行为差异，实际废料可能相差 10–20%。
:::

## 导出和报告

1. 点击**导出废料数据**
2. 选择时间段和格式（CSV / PDF）
3. 废料数据可作为费用项目包含在项目报告和发票中

另请参见[耗材分析](./filamentanalytics)获取完整的消耗概览。
