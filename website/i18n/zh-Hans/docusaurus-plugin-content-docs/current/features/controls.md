# 打印机控制

控制面板让您可以直接从仪表板对打印机进行完全的手动控制。

## 温度控制

### 喷嘴
- 设置 0–350 °C 之间的目标温度
- 点击**设置**发送命令
- 实时读数以动态环形仪表显示

### 热床
- 设置 0–120 °C 之间的目标温度
- 打印完成后自动关闭（可配置）

### 腔体
- 查看腔体温度（实时读数）
- **X1E、H2S、H2D、H2C**：通过 M141 进行主动腔体温度控制（可控制目标温度）
- **X1C**：被动腔体保温 — 显示腔体温度，但无法直接控制
- **P1S**：被动腔体保温 — 显示温度，无主动腔体温度控制
- **P1P、A1、A1 mini 及不带 chamberHeat 的 H 系列**：无腔体传感器

:::warning 最高温度
请勿超过喷嘴和热床的建议温度。硬化钢喷嘴（HF 类型）：最高 300 °C。黄铜喷嘴：最高 260 °C。请参阅打印机手册。
:::

## 速度配置

速度控制提供四种预设配置：

| 配置 | 速度 | 适用场景 |
|--------|----------|-------------|
| 静音 | 50% | 降噪，夜间打印 |
| 标准 | 100% | 正常使用 |
| 运动 | 124% | 加快打印速度 |
| 涡轮 | 166% | 最高速度（质量下降） |

滑块允许您在 50–200% 之间设置自定义百分比。

## 风扇控制

手动控制风扇转速：

| 风扇 | 说明 | 范围 |
|-------|-------------|--------|
| 部件冷却风扇 | 冷却打印对象 | 0–100% |
| 辅助风扇 | 腔体循环 | 0–100% |
| 腔体风扇 | 主动腔体冷却 | 0–100% |

:::tip 推荐设置
- **PLA/PETG：** 部件冷却 100%，辅助风扇 30%
- **ABS/ASA：** 部件冷却 0–20%，腔体风扇关闭
- **TPU：** 部件冷却 50%，低速
:::

## G-code 控制台

直接向打印机发送 G-code 命令：

```gcode
; 示例：移动打印头位置
G28 ; 所有轴回原点
G1 X150 Y150 Z10 F3000 ; 移动到中心
M104 S220 ; 设置喷嘴温度
M140 S60  ; 设置热床温度
```

:::danger 谨慎使用 G-code
错误的 G-code 可能损坏打印机。请只发送您了解的命令。避免在打印过程中使用 `M600`（换料）。
:::

## 耗材操作

在控制面板中，您可以：

- **加载耗材** — 加热喷嘴并送入耗材
- **卸载耗材** — 加热并抽出耗材
- **清洁喷嘴** — 运行清洁周期

## 宏命令

将 G-code 命令序列保存为宏，一键执行：

1. 点击**新建宏**
2. 为宏命名
3. 编写 G-code 序列
4. 保存并一键运行

热床校准示例宏：
```gcode
G28
M84
M500
```

## 打印控制

在打印过程中，您可以：

- **暂停** — 在当前层完成后暂停打印
- **继续** — 继续已暂停的打印
- **停止** — 中止打印（不可逆）
- **紧急停止** — 立即停止所有电机
