# 主控制台

主控制台是 3DPrintForge 的核心控制中心。它显示所选打印机的实时状态，并允许您根据需要监控、控制和自定义视图。

访问地址：**https://localhost:3443/**

## 实时概览

当打印机处于活跃状态时，所有数值通过 MQTT 持续更新：

- **喷嘴温度** — 带目标温度的动态 SVG 环形仪表
- **热床温度** — 同款热床平台环形仪表
- **进度百分比** — 大型进度指示器及剩余时间
- **层计数器** — 当前层 / 总层数
- **速度** — 静音 / 标准 / 运动 / 涡轮，附带滑块

:::tip 实时更新
所有数值直接从打印机通过 MQTT 实时更新，无需刷新页面。延迟通常低于 1 秒。
:::

## 3D 模型视图

如果打印机发送了包含模型的 `.3mf` 文件，将显示交互式 3D 预览：

1. 打印开始时模型自动加载
2. 用鼠标拖动旋转模型
3. 滚动鼠标进行缩放
4. 点击**重置**返回默认视图

:::info 支持说明
3D 视图需要打印机发送模型数据。并非所有打印任务都包含此数据。
:::

## AMS 状态

AMS 面板显示所有已安装的 AMS 设备及其槽位和耗材：

- **槽位颜色** — 来自拓竹元数据的可视化颜色表示
- **耗材名称** — 材料和品牌
- **活跃槽位** — 打印时以脉冲动画标记
- **错误** — AMS 出错时（堵塞、空槽、受潮）显示红色指示灯

点击某个槽位可查看完整的耗材信息，并将其关联到耗材库存。

## 摄像头实时画面

实时摄像头画面通过 ffmpeg（RTSPS → MPEG1）转换：

1. 打开仪表板时摄像头自动启动
2. 点击摄像头画面进入全屏
3. 使用**快照**按钮拍摄静态图片
4. 点击**隐藏摄像头**释放空间

:::warning 性能提示
摄像头流约使用 2–5 Mbit/s 带宽。在网络连接较慢时请禁用摄像头。
:::

## 温度趋势图

在 AMS 面板下方，显示最近 30 分钟的迷你图表（趋势图）：

- 喷嘴温度随时间变化
- 热床温度随时间变化
- 腔体温度（如可用）

点击某个趋势图可打开完整的遥测图表视图。

## 小组件自定义

仪表板使用拖放式网格布局：

1. 点击**自定义布局**（右上角的铅笔图标）
2. 将小组件拖到所需位置
3. 拖动角落调整大小
4. 点击**锁定布局**冻结位置
5. 点击**保存**以保留布局

可用的小组件：
| 小组件 | 说明 |
|---|---|
| 摄像头 | 实时摄像头画面 |
| AMS | 耗材卷和耗材状态 |
| 温度 | 喷嘴和热床的环形仪表 |
| 进度 | 进度指示器和时间估计 |
| 遥测 | 风扇、压力、速度 |
| 3D 模型 | 交互式模型视图 |
| 趋势图 | 迷你温度图表 |

:::tip 布局保存
布局以每用户形式保存在浏览器的 localStorage 中。不同用户可以有不同的布局。
:::
