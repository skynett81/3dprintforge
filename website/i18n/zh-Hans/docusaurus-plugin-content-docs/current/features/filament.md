# 耗材库存

耗材库存为您提供所有耗材卷的完整概览，与 AMS 和打印历史记录集成。

## 概览

库存显示所有已注册的耗材卷，包括：

- **颜色** — 可视化颜色卡片
- **材料** — PLA、PETG、ABS、TPU、PA 等
- **供应商** — 拓竹、Polymaker、eSUN 等
- **重量** — 剩余克数（估算或称重）
- **AMS 槽位** — 耗材卷所在的槽位
- **状态** — 活跃、已用完、干燥中、已存储

## 添加耗材卷

1. 点击 **+ 新耗材卷**
2. 填写材料、颜色、供应商和重量
3. 扫描 NFC 标签（如有），或手动输入
4. 保存

:::tip 拓竹耗材卷
拓竹官方耗材卷可通过拓竹云集成自动导入。请参见[拓竹云](../getting-started/bambu-cloud)。
:::

## AMS 同步

仪表板连接到打印机后，AMS 状态将自动同步：

- 槽位显示来自 AMS 的正确颜色和材料
- 每次打印后更新消耗量
- 空耗材卷自动标记

要将本地耗材卷关联到 AMS 槽位：
1. 进入**耗材 → AMS**
2. 点击要关联的槽位
3. 从库存中选择耗材卷

## 干燥记录

记录干燥周期，以追踪受潮情况：

| 字段 | 说明 |
|------|-------------|
| 干燥日期 | 耗材卷干燥的时间 |
| 温度 | 干燥温度（°C） |
| 时长 | 小时数 |
| 方式 | 烤箱、干燥箱、耗材干燥器 |

:::info 推荐干燥温度
有关特定材料的干燥时间和温度，请参见[知识库](../kb/intro)。
:::

## 颜色卡片

颜色卡片视图以颜色方式直观呈现耗材卷。便于快速找到正确的颜色。可按材料、供应商或状态筛选。

## NFC 标签

Bambu Dashboard 支持 NFC 标签，实现耗材卷的快速识别：

1. 在库存中为耗材卷写入 NFC 标签 ID
2. 用手机扫描标签
3. 仪表板中将直接打开该耗材卷

## 导入和导出

### 导出
将整个库存导出为 CSV：**耗材 → 导出 → CSV**

### 导入
从 CSV 导入耗材卷：**耗材 → 导入 → 选择文件**

CSV 格式：
```
名称,材料,颜色十六进制,供应商,重量克,NFC_ID
PLA 白色,PLA,#FFFFFF,拓竹,1000,
PETG 黑色,PETG,#000000,Polymaker,850,ABC123
```

## 统计数据

在**耗材 → 统计数据**中，您可以找到：

- 各材料的总消耗量（最近 30/90/365 天）
- 各打印机的消耗量
- 各耗材卷的预计剩余使用寿命
- 最常用的颜色和供应商
