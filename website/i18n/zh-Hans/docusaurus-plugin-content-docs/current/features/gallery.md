# 相册

相册汇集了每次打印过程中自动拍摄的截图。图片在固定里程碑处拍摄，为您提供打印过程的可视化记录。

访问地址：**https://localhost:3443/#gallery**

## 里程碑截图

3DPrintForge 会在以下里程碑处自动从摄像头拍摄截图：

| 里程碑 | 时间点 |
|---|---|
| **25%** | 打印四分之一处 |
| **50%** | 打印中途 |
| **75%** | 打印四分之三处 |
| **100%** | 打印完成 |

截图将关联到相应的打印历史记录条目并在相册中显示。

:::info 要求
里程碑截图需要摄像头已连接并处于活跃状态。已停用的摄像头不会生成图片。
:::

## 启用截图功能

1. 进入**设置 → 相册**
2. 开启**自动里程碑截图**
3. 选择要启用的里程碑（默认四个全部开启）
4. 选择**图片质量**：低（640×360）/ 中（1280×720）/ 高（1920×1080）
5. 点击**保存**

## 图片浏览

相册按打印分组：

1. 使用顶部**筛选器**选择打印机、日期或文件名
2. 点击某个打印行展开并查看全部四张图片
3. 点击图片打开预览

### 预览

预览显示：
- 全尺寸图片
- 里程碑和时间戳
- 打印名称和打印机
- **←** / **→** 在同一次打印的图片之间切换

## 全屏视图

在预览中点击**全屏**（或按 `F`）填满整个屏幕。使用方向键在图片之间切换。

## 下载图片

- **单张图片**：在预览中点击**下载**
- **某次打印的所有图片**：点击打印行的**下载全部** — 获得 `.zip` 文件
- **多选下载**：勾选复选框，然后点击**下载所选**

## 删除图片

:::warning 存储空间
相册图片会随时间占用大量空间。请设置自动删除旧图片。
:::

### 手动删除

1. 选择一张或多张图片（勾选）
2. 点击**删除所选**
3. 在对话框中确认

### 自动清理

1. 进入**设置 → 相册 → 自动清理**
2. 启用**删除超过 X 天的图片**
3. 设置天数（例如 90 天）
4. 清理每晚凌晨 3:00 自动运行

## 与打印历史的关联

每张图片都关联到历史记录中的一个打印条目：

- 在相册中点击某次打印的**在历史中查看**，跳转到历史记录条目
- 如果存在 100% 图片，历史记录将显示该图片的缩略图

## 分享

通过限时链接分享相册图片：

1. 在预览中打开图片
2. 点击**分享**
3. 选择过期时间并复制链接
