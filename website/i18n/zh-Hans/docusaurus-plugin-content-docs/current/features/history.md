# 打印历史

打印历史提供了通过仪表板进行的所有打印的完整记录，包括统计数据、耗材消耗和模型来源链接。

## 历史记录表

该表格显示所有打印，包含以下信息：

| 列 | 说明 |
|---------|-------------|
| 日期/时间 | 开始时间 |
| 模型名称 | 文件名或 MakerWorld 标题 |
| 打印机 | 使用的打印机 |
| 时长 | 总打印时间 |
| 耗材 | 材料和使用克数 |
| 平台 | 层数和重量（克） |
| 状态 | 已完成、已中断、已失败 |
| 图片 | 缩略图（通过云集成） |

## 搜索和筛选

使用搜索框和筛选器查找打印记录：

- 按模型名称进行全文搜索
- 按打印机、材料、状态、日期筛选
- 按所有列排序

## 模型来源链接

如果打印是从 MakerWorld 开始的，将显示直接指向该模型页面的链接。点击模型名称在新标签页中打开 MakerWorld。

:::info 拓竹云
模型链接和缩略图需要拓竹云集成。请参见[拓竹云](../getting-started/bambu-cloud)。
:::

## 耗材追踪

每次打印记录：

- **材料** — PLA、PETG、ABS 等
- **使用克数** — 估算的消耗量
- **耗材卷** — 使用的耗材卷（如在库存中已注册）
- **颜色** — 颜色的十六进制代码

这样可以准确了解随时间推移的耗材消耗情况，帮助您规划采购。

## 统计数据

在**历史 → 统计数据**中，您可以找到汇总数据：

- **总打印次数** — 及成功率
- **总打印时间** — 小时和天数
- **耗材消耗** — 各材料的克数和千米数
- **每日打印次数** — 滚动图表
- **最常用材料** — 饼图
- **打印时长分布** — 直方图

可按时间段筛选统计数据（7 天、30 天、90 天、1 年、全部）。

## 导出

### CSV 导出
导出完整历史记录或筛选后的结果：
**历史 → 导出 → 下载 CSV**

CSV 文件包含所有列，可在 Excel、LibreOffice Calc 中打开或导入其他工具。

### 自动备份
历史记录是 SQLite 数据库的一部分，在更新时自动备份。在**设置 → 备份**中手动备份。

## 编辑

您可以在事后编辑打印日志条目：

- 更正模型名称
- 添加备注
- 更正耗材消耗量
- 删除错误记录的打印

右键点击某行选择**编辑**，或点击铅笔图标。
