# 项目

项目允许您将相关打印分组，追踪材料费用，为客户开具发票，并分享您的工作概览。

访问地址：**https://localhost:3443/#projects**

## 创建项目

1. 点击**新建项目**（+ 图标）
2. 填写：
   - **项目名称** — 描述性名称（最多 100 个字符）
   - **客户** — 可选的客户账户（参见[电商](../integrations/ecommerce)）
   - **描述** — 简短文字说明
   - **颜色** — 选择用于视觉识别的颜色
   - **标签** — 逗号分隔的关键词
3. 点击**创建项目**

## 将打印关联到项目

### 打印期间

1. 打印进行时打开仪表板
2. 点击侧边栏中的**关联到项目**
3. 选择现有项目或创建新项目
4. 打印完成时将自动关联到项目

### 从历史记录

1. 进入**历史记录**
2. 找到相应的打印
3. 点击打印 → **关联到项目**
4. 从下拉列表中选择项目

### 批量关联

1. 在历史记录中用复选框选择多次打印
2. 点击**操作 → 关联到项目**
3. 选择项目 — 所有选中的打印将被关联

## 费用概览

每个项目根据以下内容计算总费用：

| 费用类型 | 来源 |
|---|---|
| 耗材消耗 | 克数 × 每种材料的每克价格 |
| 电费 | 千瓦时 × 电价（如已配置 Tibber/Nordpool） |
| 机器损耗 | 根据[磨损预测](../monitoring/wearprediction)计算 |
| 手动费用 | 您手动添加的自由文本项目 |

费用概览以表格和饼图的形式按打印和总计显示。

:::tip 按小时电价
激活 Tibber 或 Nordpool 集成，获取每次打印的准确电费。请参见[电价](../integrations/energy)。
:::

## 发票

1. 打开项目并点击**生成发票**
2. 填写：
   - **发票日期**和**到期日**
   - **税率**（0%、15%、25%）
   - **利润加成**（%）
   - **客户备注**
3. 以 PDF 格式预览发票
4. 点击**下载 PDF** 或**发送给客户**（通过邮件）

发票保存在项目下，在发送之前可以重新打开和编辑。

:::info 客户数据
客户数据（姓名、地址、公司注册号）从关联到项目的客户账户中获取。请参见[电商](../integrations/ecommerce)管理客户。
:::

## 项目状态

| 状态 | 说明 |
|---|---|
| 活跃 | 项目正在进行中 |
| 已完成 | 所有打印完成，发票已发送 |
| 已归档 | 在标准视图中隐藏，但可搜索 |
| 搁置中 | 暂时停止 |

点击项目顶部的状态指示器更改状态。

## 分享项目

生成一个可分享的链接，向客户展示项目概览：

1. 在项目菜单中点击**分享项目**
2. 选择要显示的内容：
   - ✅ 打印和图片
   - ✅ 总耗材消耗量
   - ❌ 费用和价格（默认隐藏）
3. 设置链接的过期时间
4. 复制并分享链接

客户无需登录即可查看只读页面。
