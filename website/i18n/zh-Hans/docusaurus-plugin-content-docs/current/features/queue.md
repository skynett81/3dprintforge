# 打印队列

打印队列允许您提前计划打印任务，并在打印机空闲时自动发送给它们。

访问地址：**https://localhost:3443/#queue**

## 创建队列

1. 在导航菜单中进入**打印队列**
2. 点击**新建任务**（+ 图标）
3. 填写：
   - **文件名** — 上传 `.3mf` 或 `.gcode` 文件
   - **目标打印机** — 选择特定打印机或**自动**
   - **优先级** — 低 / 普通 / 高 / 紧急
   - **计划开始时间** — 现在或特定日期/时间
4. 点击**加入队列**

:::tip 拖放
您可以直接将文件从文件浏览器拖放到队列页面快速添加。
:::

## 添加文件

### 上传文件

1. 点击**上传**或将文件拖到上传区域
2. 支持的格式：`.3mf`、`.gcode`、`.bgcode`
3. 文件保存在文件库中并关联到队列任务

### 从文件库

1. 进入**文件库**并找到文件
2. 点击文件上的**加入队列**
3. 任务以默认设置创建 — 如有需要可编辑

### 从历史记录

1. 在**历史记录**中打开之前的打印
2. 点击**再次打印**
3. 任务以与上次相同的设置添加

## 优先级

队列按优先级顺序处理：

| 优先级 | 颜色 | 说明 |
|---|---|---|
| 紧急 | 红色 | 无论其他任务如何，发送给第一台空闲打印机 |
| 高 | 橙色 | 排在普通和低优先级任务之前 |
| 普通 | 蓝色 | 默认顺序（先进先出） |
| 低 | 灰色 | 仅在没有更高优先级任务等待时发送 |

在队列中拖放任务，可在同一优先级内手动调整顺序。

## 自动派送

**自动派送**启用后，3DPrintForge 将监控所有打印机并自动发送下一个任务：

1. 进入**设置 → 队列**
2. 开启**自动派送**
3. 选择**派送策略**：
   - **最先空闲** — 发送给第一台变为空闲的打印机
   - **使用最少** — 优先选择今天打印次数最少的打印机
   - **循环轮换** — 在所有打印机之间均匀轮换

:::warning 需要确认
如果您希望在文件发送前手动批准每次派送，请在设置中启用**需要确认**。
:::

## 错峰启动

错峰启动有助于避免所有打印机同时开始和结束：

1. 在**新建任务**对话框中，展开**高级设置**
2. 启用**错峰启动**
3. 设置**打印机之间的延迟**（例如 30 分钟）
4. 系统将自动分配开始时间

**示例：** 4 个相同任务，每台间隔 30 分钟，分别在 08:00、08:30、09:00 和 09:30 启动。

## 队列状态和跟踪

队列概览显示所有任务的状态：

| 状态 | 说明 |
|---|---|
| 等待中 | 任务在队列中等待打印机 |
| 已计划 | 未来有计划的开始时间 |
| 发送中 | 正在传输到打印机 |
| 打印中 | 正在所选打印机上进行 |
| 已完成 | 完成 — 已关联到历史记录 |
| 失败 | 发送或打印过程中出错 |
| 已中断 | 手动中断 |

:::info 通知
在**设置 → 通知 → 队列**中启用队列事件通知，当任务开始、完成或失败时接收消息。请参见[通知](./notifications)。
:::
