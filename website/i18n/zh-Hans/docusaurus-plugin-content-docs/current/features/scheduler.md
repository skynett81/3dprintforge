# 计划器

计划器允许您通过日历视图和智能打印队列组织和自动化打印任务。

## 日历视图

日历视图提供所有计划和已完成打印的概览：

- **月视图、周视图和日视图** — 选择详细程度
- **颜色编码** — 按打印机和状态使用不同颜色
- **点击事件** — 查看打印详情

已完成的打印根据打印历史自动显示。

## 打印队列

打印队列允许您将任务排队，按顺序发送到打印机：

### 向队列添加任务

1. 点击 **+ 添加任务**
2. 选择文件（来自打印机 SD 卡、本地上传或 FTP）
3. 设置优先级（高、普通、低）
4. 选择目标打印机（或"自动"）
5. 点击**添加**

### 队列管理

| 操作 | 说明 |
|----------|-------------|
| 拖放 | 重新排列顺序 |
| 暂停队列 | 临时停止派送 |
| 跳过 | 不等待直接发送下一个任务 |
| 删除 | 从队列中删除任务 |

:::tip 多打印机派送
有多台打印机时，队列可以自动将任务分配给空闲的打印机。在**计划器 → 设置**下启用**自动派送**。
:::

## 计划打印

设置在特定时间开始的打印：

1. 点击 **+ 计划打印**
2. 选择文件和打印机
3. 设置开始时间
4. 配置通知（可选）
5. 保存

:::warning 打印机必须空闲
计划打印仅在指定时间打印机处于待机模式时才会开始。如果打印机正忙，开始时间将推迟到下一个可用时间（可配置）。
:::

## 负载均衡

通过自动负载均衡，可在打印机之间智能分配任务：

- **循环轮换** — 在所有打印机之间均匀分配
- **最不繁忙** — 发送给预计完成时间最短的打印机
- **手动** — 您为每个任务自己选择打印机

在**计划器 → 负载均衡**中配置。

## 通知

计划器与通知渠道集成：

- 任务开始时通知
- 任务完成时通知
- 出错或延迟时通知

请参见[功能概览](./overview#通知)配置通知渠道。
