# 机群概览

机群概览在一个页面上为您提供所有已连接打印机的紧凑视图。非常适合工作室、教室或拥有多台打印机的用户。

访问地址：**https://localhost:3443/#fleet**

## 多打印机网格

所有已注册的打印机以自适应网格显示：

- **卡片大小** — 小型（紧凑）、中型（标准）、大型（详细）
- **列数** — 根据屏幕宽度自动调整，或手动设置
- **更新** — 每张卡片通过 MQTT 独立更新

每张打印机卡片显示：
| 字段 | 说明 |
|---|---|
| 打印机名称 | 带型号图标的配置名称 |
| 状态 | 空闲 / 打印中 / 暂停 / 错误 / 已断开 |
| 进度 | 带剩余时间的进度条 |
| 温度 | 喷嘴和热床（紧凑显示） |
| 当前耗材 | 来自 AMS 的颜色和材料 |
| 摄像头缩略图 | 每 30 秒更新一次的静态图片 |

## 每台打印机的状态指示

状态颜色让您一眼判断远处的状态：

- **绿色脉冲** — 正在打印
- **蓝色** — 空闲就绪
- **黄色** — 已暂停（手动或由打印守护触发）
- **红色** — 检测到错误
- **灰色** — 已断开或不可访问

:::tip 信息亭模式
在壁挂式屏幕上以信息亭模式使用机群概览。请参见[信息亭模式](../system/kiosk)进行设置。
:::

## 排序

点击**排序**选择顺序：

1. **名称** — 按字母顺序 A–Z
2. **状态** — 活跃打印机排在前面
3. **进度** — 完成度最高的排在前面
4. **最近活跃** — 最近使用的排在前面
5. **型号** — 按打印机型号分组

排序设置在下次访问时记忆。

## 筛选

使用顶部的筛选字段限制显示范围：

- 输入打印机名称或部分名称
- 从下拉列表选择**状态**（全部 / 打印中 / 空闲 / 错误）
- 选择**型号**只显示某种打印机类型（X1C、P1S、A1 等）
- 点击**重置筛选**显示全部

:::info 搜索
搜索实时筛选，无需刷新页面。
:::

## 在机群概览中执行操作

右键点击卡片（或点击三个点图标）进行快捷操作：

- **打开仪表板** — 直接跳转到该打印机的主控制台
- **暂停打印** — 将打印机设为暂停状态
- **停止打印** — 中止正在进行的打印（需要确认）
- **查看摄像头** — 在弹出窗口中打开摄像头视图
- **进入设置** — 打开打印机设置

:::danger 停止打印
停止打印是不可逆的。请始终在显示的对话框中确认。
:::

## 汇总统计数据

机群概览顶部显示一个摘要行：

- 打印机总数
- 正在打印的台数
- 今日总耗材消耗量
- 最长进行中打印的预计完成时间
