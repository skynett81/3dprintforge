# 功能概览

Bambu Dashboard 将监控和管理拓竹打印机所需的一切功能集中在一处。

## 仪表板

主仪表板显示活跃打印机的实时状态：

- **温度** — 喷嘴和热床的动态 SVG 环形仪表
- **进度** — 带预计完成时间的进度百分比
- **摄像头** — 实时摄像头画面（RTSPS → MPEG1，通过 ffmpeg）
- **AMS 面板** — 所有 AMS 槽位的可视化展示，含耗材颜色
- **速度控制** — 调整速度的滑块（静音、标准、运动、涡轮）
- **统计面板** — Grafana 风格的滚动图表面板
- **遥测** — 风扇、温度、压力的实时数值

面板支持拖放以自定义布局。使用锁定按钮固定布局。

## 耗材库存

完整文档请参见[耗材](./filament)。

- 追踪所有耗材卷的名称、颜色、重量和供应商
- AMS 同步 — 查看 AMS 中的耗材卷
- 干燥日志和干燥计划
- 颜色卡片和 NFC 标签支持
- 导入/导出（CSV）

## 打印历史

完整文档请参见[历史](./historikk)。

- 完整的打印记录
- 每次打印的耗材追踪
- MakerWorld 模型链接
- 统计数据和 CSV 导出

## 计划安排

完整文档请参见[计划器](./scheduler)。

- 打印的日历视图
- 带优先级的打印队列
- 多打印机派送

## 打印机控制

完整文档请参见[控制](./controls)。

- 温度控制（喷嘴、热床、腔体）
- 速度配置控制
- 风扇控制
- G-code 控制台
- 耗材加载/卸载

## 通知

Bambu Dashboard 支持 7 个通知渠道：

| 渠道 | 事件 |
|-------|----------|
| Telegram | 打印完成、错误、暂停 |
| Discord | 打印完成、错误、暂停 |
| 邮件 | 打印完成、错误 |
| ntfy | 所有事件 |
| Pushover | 所有事件 |
| SMS（Twilio） | 严重错误 |
| Webhook | 自定义负载 |

在**设置 → 通知**中配置。

## 打印守护

打印守护通过摄像头（xcam）和传感器监控活跃打印：

- 检测到意面状打印时自动暂停
- 可配置的灵敏度级别
- 检测到事件的日志记录

## 维护

维护部分追踪：

- 每个组件（喷嘴、平台、AMS）的下次建议维护时间
- 基于打印历史的磨损追踪
- 手动记录维护任务

## 多打印机

通过多打印机支持，您可以：

- 从一个仪表板管理多台打印机
- 使用打印机选择器在打印机之间切换
- 同时查看所有打印机的状态概览
- 通过打印队列分配打印任务

## OBS 覆盖层

专用的 `obs.html` 页面为 OBS Studio 集成提供干净的覆盖层，用于打印直播。

## 更新

通过 GitHub Releases 内置自动更新。可在**设置 → 更新**中直接从仪表板接收通知和更新。
