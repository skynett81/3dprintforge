# 延时摄影

Bambu Dashboard 可以在打印过程中自动拍照，并将其合成为延时摄影视频。视频本地存储，可直接在仪表板中播放。

访问地址：**https://localhost:3443/#timelapse**

## 启用

1. 进入**设置 → 延时摄影**
2. 开启**启用延时摄影录制**
3. 选择**录制模式**：
   - **按层** — 每层一张图片（推荐高质量）
   - **按时间** — 每 N 秒一张图片（例如每 30 秒）
4. 选择要启用延时摄影的打印机
5. 点击**保存**

:::tip 图片间隔
"按层"可获得最流畅的动画，因为运动是一致的。"按时间"使用更少的存储空间。
:::

## 录制设置

| 设置 | 默认值 | 说明 |
|---|---|---|
| 分辨率 | 1280×720 | 图片大小（640×480 / 1280×720 / 1920×1080） |
| 图片质量 | 85% | JPEG 压缩质量 |
| 视频帧率 | 30 | 成品视频的每秒帧数 |
| 视频格式 | MP4（H.264） | 输出格式 |
| 旋转图片 | 关闭 | 根据安装方向旋转 90°/180°/270° |

:::warning 存储空间
1080p 下 500 张图片的延时摄影在合成前约使用 200–400 MB。成品 MP4 视频通常为 20–80 MB。
:::

## 存储

延时摄影图片和视频存储在项目文件夹的 `data/timelapse/` 中。结构按打印机和打印任务组织：

```
data/timelapse/
├── <printer-id>/                     ← 唯一打印机 ID
│   ├── 2026-03-22_modellnavn/        ← 打印会话（日期_模型名称）
│   │   ├── frame_0001.jpg
│   │   ├── frame_0002.jpg
│   │   ├── frame_0003.jpg
│   │   └── ...                       ← 原始图片（合成后删除）
│   ├── 2026-03-22_modellnavn.mp4     ← 成品延时视频
│   ├── 2026-03-20_3dbenchy.mp4
│   └── 2026-03-15_telefonstativ.mp4
├── <printer-id-2>/                   ← 多台打印机
│   └── ...
```

:::tip 外部存储
要节省系统盘空间，可以将延时摄影文件夹符号链接到外部磁盘：
```bash
# 示例：移动到挂载在 /mnt/storage 的外部磁盘
mv data/timelapse /mnt/storage/timelapse

# 创建指向原位置的符号链接
ln -s /mnt/storage/timelapse data/timelapse
```
仪表板将自动跟随符号链接。您可以使用任何磁盘或网络共享。
:::

## 自动合成

打印完成后，图片将通过 ffmpeg 自动合成为视频：

1. Bambu Dashboard 从 MQTT 接收"打印完成"事件
2. 调用 ffmpeg 处理收集的图片
3. 视频保存到存储文件夹
4. 延时摄影页面自动更新，显示新视频

您可以在**延时摄影 → 处理中**标签页下查看进度。

## 播放

1. 访问 **https://localhost:3443/#timelapse**
2. 从下拉列表中选择打印机
3. 点击列表中的视频进行播放
4. 使用播放控件：
   - ▶ / ⏸ — 播放 / 暂停
   - ⏪ / ⏩ — 倒退 / 快进
   - 速度按钮：0.5× / 1× / 2× / 4×
5. 点击**全屏**以全屏模式打开
6. 点击**下载**以下载 MP4 文件

## 删除延时摄影

1. 在列表中选择视频
2. 点击**删除**（垃圾桶图标）
3. 在对话框中确认

:::danger 永久删除
已删除的延时摄影视频和原始图片无法恢复。如果您希望保留，请先下载视频。
:::

## 分享延时摄影

延时摄影视频可通过限时链接分享：

1. 选择视频并点击**分享**
2. 设置过期时间（1 小时 / 24 小时 / 7 天 / 永不过期）
3. 复制生成的链接并分享
4. 接收者无需登录即可观看视频
