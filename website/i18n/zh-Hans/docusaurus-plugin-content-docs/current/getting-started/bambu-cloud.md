# 拓竹云集成

3DPrintForge 可连接到 **拓竹云（Bambu Lab Cloud）**，以获取模型图片、打印历史和耗材数据。仪表板在没有云连接的情况下也能正常运行，但云集成可提供额外功能。

## 云集成的优势

| 功能 | 无云连接 | 有云连接 |
|---------|-----------|----------|
| 打印机实时状态 | 是 | 是 |
| 打印历史（本地） | 是 | 是 |
| MakerWorld 模型图片 | 否 | 是 |
| 拓竹耗材配置文件 | 否 | 是 |
| 打印历史同步 | 否 | 是 |
| 云端 AMS 耗材数据 | 否 | 是 |

## 连接拓竹云

1. 进入**设置 → 拓竹云**
2. 输入您的拓竹账号邮箱和密码
3. 点击**登录**
4. 选择需要同步的数据类型

:::warning 隐私说明
用户名和密码不会以明文存储。仪表板使用拓竹官方 API 获取 OAuth 令牌，令牌保存在本地。您的数据不会离开您自己的服务器。
:::

## 数据同步

### 模型图片

连接云端后，模型图片将自动从 **MakerWorld** 获取，显示在：
- 打印历史
- 仪表板（打印进行时）
- 3D 模型查看器

### 打印历史

云同步从拓竹 App 导入打印历史。系统根据时间戳和序列号自动过滤重复记录。

### 耗材配置文件

拓竹官方耗材配置文件将同步并显示在耗材库存中。您可以将这些配置文件作为创建自定义配置的起点。

## 无云连接时的功能

所有核心功能在无云连接时均可正常使用：

- 通过局域网直接 MQTT 连接打印机
- 实时状态、温度、摄像头画面
- 本地打印历史和统计数据
- 耗材库存（手动管理）
- 通知和计划器

:::tip 纯本地模式
希望在完全断网的环境中使用仪表板？在隔离网络中同样运行良好——只需通过 IP 连接打印机，关闭云集成即可。
:::

## 故障排查

**登录失败：**
- 检查邮箱和密码是否与拓竹 App 的账号一致
- 检查账号是否启用了双因素验证（暂不支持）
- 尝试退出后重新登录

**同步中断：**
- 令牌可能已过期——在设置中退出后重新登录
- 检查服务器的网络连接
