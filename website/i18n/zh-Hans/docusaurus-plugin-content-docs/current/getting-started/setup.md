# 首次设置

仪表板首次运行时，将自动打开设置向导。

## 设置向导

向导可在 `https://您的服务器:3443/setup` 访问。它将引导您完成：

1. 创建管理员账户
2. 添加打印机
3. 测试连接
4. 配置通知（可选）

## 添加打印机

连接打印机需要以下三项信息：

| 字段 | 说明 | 示例 |
|------|-------------|---------|
| IP 地址 | 打印机的本地 IP | `192.168.1.100` |
| 序列号 | 15 位字符，印在打印机底部 | `01P09C123456789` |
| 访问码 | 8 位字符，在打印机网络设置中查看 | `12345678` |

### 在打印机上查找访问码

**X1C / P1S / P1P：**
1. 在屏幕上进入**设置**
2. 选择 **WLAN** 或 **LAN**
3. 查找**访问码**

**A1 / A1 Mini：**
1. 点击屏幕，选择**设置**
2. 进入 **WLAN**
3. 查找**访问码**

:::tip 固定 IP 地址
在路由器中为打印机设置固定 IP（DHCP 地址预留），这样就无需在每次打印机获取新 IP 时更新仪表板。
:::

## AMS 配置

打印机连接后，AMS 状态将自动更新。您可以：

- 为每个托架命名并设置颜色
- 将线卷关联到您的耗材库存
- 查看每个线卷的耗材消耗量

进入**设置 → 打印机 → AMS** 进行手动配置。

## HTTPS 证书 {#https-sertifikater}

### 自签名证书（默认）

仪表板在启动时自动生成自签名证书。若要在浏览器中信任该证书：

- **Chrome/Edge：** 点击「高级」→「继续访问该网站」
- **Firefox：** 点击「高级」→「接受风险并继续」

### 自定义证书

将证书文件放置到指定目录，并在 `config.json` 中配置：

```json
{
  "ssl": {
    "cert": "/路径/到/cert.pem",
    "key": "/路径/到/key.pem"
  }
}
```

:::info Let's Encrypt
使用域名？可通过 Let's Encrypt 和 Certbot 免费生成证书，将 `cert` 和 `key` 指向 `/etc/letsencrypt/live/您的域名/` 中的文件。
:::

## 环境变量

所有设置均可通过环境变量覆盖：

| 变量 | 默认值 | 说明 |
|----------|---------|-------------|
| `PORT` | `3000` | HTTP 端口 |
| `HTTPS_PORT` | `3443` | HTTPS 端口 |
| `NODE_ENV` | `production` | 运行环境 |
| `AUTH_SECRET` | （自动生成） | JWT 密钥 |

## 多打印机设置

您可以在**设置 → 打印机 → 添加打印机**中添加多台打印机。使用仪表板顶部的打印机选择器在不同打印机之间切换。
