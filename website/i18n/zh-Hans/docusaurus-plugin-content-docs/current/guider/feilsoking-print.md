# 打印失败排查

出了问题？不要担心——大多数打印问题都有简单的解决方案。Bambu Dashboard帮助你快速找到原因。

## 第1步 — 检查HMS错误代码

HMS（处理、监控、感测）是Bambu Labs的错误系统。所有错误都会自动记录在仪表板中。

1. 进入**监控 → 错误**
2. 找到失败的打印
3. 点击错误代码查看详细说明和建议解决方案

常见HMS代码：

| 代码 | 描述 | 快速解决方案 |
|------|------|------------|
| 0700 1xxx | AMS错误（卡料，电机问题） | 检查AMS中的耗材路径 |
| 0300 0xxx | 挤出错误（欠挤/过挤） | 清洁喷嘴，检查耗材 |
| 0500 xxxx | 校准错误 | 重新运行校准 |
| 1200 xxxx | 温度偏差 | 检查电缆连接 |
| 0C00 xxxx | 摄像头错误 | 重启打印机 |

:::tip 历史记录中的错误代码
在**历史记录 → [打印] → HMS日志**下，你可以查看打印过程中出现的所有错误代码——即使打印"已完成"。
:::

## 常见问题和解决方案

### 粘附不良（第一层不粘）

**症状：** 打印件从平台脱落、卷曲、缺失第一层

**原因和解决方案：**

| 原因 | 解决方案 |
|------|---------|
| 脏平台 | 用IPA酒精擦拭 |
| 平台温度不正确 | 提高5°C |
| Z偏移错误 | 重新运行自动调平 |
| 缺少固体胶棒（PETG/ABS） | 涂一薄层固体胶棒 |
| 第一层速度过快 | 第一层降低到20〜30 mm/s |

**快速检查清单：**
1. 平台干净吗？（IPA + 无绒纸巾）
2. 你使用的是正确的耗材类型对应的平台吗？（参见[选择正确的平台](./velge-rett-plate)）
3. 上次换平台后是否已进行Z校准？

---

### 翘曲（边角抬起）

**症状：** 边角从平台向上弯曲，尤其是大型平面模型

**原因和解决方案：**

| 原因 | 解决方案 |
|------|---------|
| 温度差异 | 关闭打印机前盖 |
| 缺少裙边 | 在Bambu Studio中启用裙边（3〜5 mm） |
| 平台太凉 | 提高平台温度5〜10°C |
| 收缩率高的耗材（ABS） | 使用工程平台 + 腔体>40°C |

**ABS和ASA尤其容易翘曲。** 务必确保：
- 前盖关闭
- 尽量减少通风
- 工程平台 + 固体胶棒
- 腔体温度40°C以上

---

### 拉丝（零件之间的丝线）

**症状：** 模型独立部件之间出现细塑料丝

**原因和解决方案：**

| 原因 | 解决方案 |
|------|---------|
| 潮湿耗材 | 干燥耗材6〜8小时（60〜70°C） |
| 喷嘴温度过高 | 降低5°C |
| 回抽不足 | 在Bambu Studio中增加回抽长度 |
| 移动速度太低 | 将移动速度增加到200+ mm/s |

**湿度测试：** 听是否有爆裂声或查看挤出中的气泡——表明耗材受潮。Bambu AMS有内置湿度计；在**AMS状态**下检查湿度。

:::tip 耗材干燥器
如果使用尼龙或TPU，可以购置耗材干燥器（例如Bambu Filament Dryer）——这些材料在不到12小时内就会吸收水分。
:::

---

### 意大利面（打印件崩塌成一团）

**症状：** 耗材在空中松散悬挂，打印件无法辨认

**原因和解决方案：**

| 原因 | 解决方案 |
|------|---------|
| 初始粘附不良 → 脱落 → 崩塌 | 参见上方的粘附部分 |
| 速度过快 | 将速度降低20〜30% |
| 支撑配置错误 | 在Bambu Studio中启用支撑 |
| 悬空过陡 | 分割模型或旋转45° |

**使用Print Guard自动阻止意大利面** — 参见下一节。

---

### 欠挤出（层薄弱）

**症状：** 层不结实，壁上有孔，模型脆弱

**原因和解决方案：**

| 原因 | 解决方案 |
|------|---------|
| 喷嘴部分堵塞 | 执行冷拔（参见维护） |
| 耗材过潮 | 干燥耗材 |
| 温度太低 | 提高喷嘴温度5〜10°C |
| 速度过快 | 降低20〜30% |
| PTFE管损坏 | 检查并更换PTFE管 |

## 使用Print Guard进行自动保护

Print Guard通过图像识别监控摄像头图像，并在检测到意大利面时自动停止打印。

**启用Print Guard：**
1. 进入**监控 → Print Guard**
2. 启用**自动检测**
3. 选择操作：**暂停**（推荐）或**取消**
4. 设置灵敏度（从**中等**开始）

**Print Guard介入时：**
1. 你会收到一条包含检测到的内容的摄像头图片的通知
2. 打印暂停
3. 你可以选择：**继续**（如果是误报）或**取消打印**

:::info 误报
Print Guard有时会对有很多细柱子的模型有反应。对于复杂模型，降低灵敏度或暂时停用。
:::

## 仪表板中的诊断工具

### 温度日志
在**历史记录 → [打印] → 温度**下，你可以查看整个打印过程的温度曲线。注意：
- 突然的温度下降（喷嘴或平台问题）
- 不均匀的温度（需要校准）

### 耗材统计
检查消耗的耗材是否与估算相符。大的偏差可能表明欠挤出或耗材断裂。

## 何时联系支持？

以下情况请联系Bambu Labs支持：
- 按照所有建议解决方案操作后HMS代码仍然重复出现
- 看到打印机有机械损伤（弯曲的导轨，损坏的齿轮）
- 温度值不可能（例如喷嘴读数-40°C）
- 固件更新无法解决问题

**联系支持时有用的信息：**
- 仪表板错误日志中的HMS错误代码
- 错误的摄像头图片
- 使用的耗材和设置（可从历史记录导出）
- 打印机型号和固件版本（显示在**设置 → 打印机 → 信息**下）
