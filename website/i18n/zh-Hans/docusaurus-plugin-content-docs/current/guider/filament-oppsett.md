# 设置耗材库存

Bambu Dashboard中的耗材库存让你全面掌握所有线轴的情况——剩余多少、已使用多少，以及哪些线轴正在AMS中。

## 从AMS自动创建

连接带有AMS的打印机时，仪表板会自动从Bambu线轴的RFID芯片读取信息：

- 耗材类型（PLA、PETG、ABS、TPU等）
- 颜色（带十六进制色码）
- 品牌（Bambu Lab）
- 线轴重量和剩余量

**这些线轴会自动在库存中创建** — 你无需做任何操作。在**耗材 → 库存**下查看它们。

:::info 只有Bambu线轴有RFID
第三方线轴（例如eSUN、Polymaker、没有芯片的Bambu补充装）不会自动识别。这些需要手动输入。
:::

## 手动添加线轴

对于没有RFID的线轴，或不在AMS中的线轴：

1. 进入**耗材 → 库存**
2. 点击右上角的**+ 新建线轴**
3. 填写字段：

| 字段 | 示例 | 必填 |
|------|------|------|
| 品牌 | eSUN、Polymaker、Bambu | 是 |
| 类型 | PLA、PETG、ABS、TPU | 是 |
| 颜色 | #FF5500 或从色轮选择 | 是 |
| 起始重量 | 1000 g | 建议 |
| 剩余量 | 850 g | 建议 |
| 直径 | 1.75 mm | 是 |
| 备注 | "2025年1月购买，效果良好" | 可选 |

4. 点击**保存**

## 配置颜色和品牌

你可以随时点击库存中的线轴进行编辑：

- **颜色** — 从色轮中选择或输入十六进制值。颜色在AMS概览中用作视觉标记
- **品牌** — 显示在统计和筛选中。在**耗材 → 品牌**下创建自定义品牌
- **温度配置** — 输入耗材生产商推荐的喷嘴和平台温度。如果选择错误温度，仪表板可以发出警告

## 了解AMS同步

仪表板实时同步AMS状态：

```
AMS插槽1 → 线轴：Bambu PLA 白色  [███████░░░] 剩余72%
AMS插槽2 → 线轴：eSUN PETG 灰色  [████░░░░░░] 剩余41%
AMS插槽3 → （空）
AMS插槽4 → 线轴：Bambu PLA 红色  [██████████] 剩余98%
```

同步更新时机：
- **打印期间** — 消耗量实时扣除
- **打印结束时** — 最终消耗量记录在历史记录中
- **手动** — 点击线轴上的同步图标从AMS获取最新数据

:::tip 修正AMS估算
首次使用后，来自RFID的AMS估算并不总是100%准确。手动称量线轴并更新重量以获得更好的精度。
:::

## 查看消耗量和剩余量

### 按线轴
点击库存中的线轴查看：
- 总使用量（克，所有打印）
- 预计剩余量
- 使用该线轴的所有打印列表

### 汇总统计
在**分析 → 耗材分析**下查看：
- 按时间的耗材类型消耗量
- 你最常使用的品牌
- 基于每公斤购买价格的预计成本

### 低库存警报
设置线轴接近用尽时的警报：

1. 进入**耗材 → 设置**
2. 启用**低库存提醒**
3. 设置阈值（例如剩余100 g）
4. 选择通知渠道（Telegram、Discord、邮件）

## 提示：称量线轴以获得更高精度

AMS和打印统计中的估算从来都不是完全精确的。最准确的方法是称量线轴本身：

**操作步骤：**

1. 找到皮重（空线轴）— 通常200〜250 g，查看生产商网站或线轴底部
2. 用厨房秤称量带耗材的线轴
3. 减去皮重
4. 在线轴配置文件中更新**剩余量**

**示例：**
```
称量重量：    743 g
皮重（空）：- 230 g
剩余耗材：   513 g
```

:::tip 线轴标签生成器
在**工具 → 标签**下，你可以为线轴打印带有QR码的标签。用手机扫描代码即可快速打开线轴配置文件。
:::
