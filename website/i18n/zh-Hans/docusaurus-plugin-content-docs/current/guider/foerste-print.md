# 你的第一次打印

本指南将带你完成整个流程——从连接打印机到完成打印——以Bambu Dashboard作为控制中心。

## 第1步 — 检查打印机是否已连接

打开仪表板时，你会在侧边栏顶部或概览面板上看到打印机的状态卡片。

**绿色状态**表示打印机在线且准备就绪。

| 状态 | 颜色 | 含义 |
|------|------|------|
| 在线 | 绿色 | 准备打印 |
| 空闲 | 灰色 | 已连接，但未激活 |
| 打印中 | 蓝色 | 打印进行中 |
| 错误 | 红色 | 需要处理 |

如果打印机显示红色状态：
1. 检查打印机是否已开机
2. 确认它已连接到与仪表板相同的网络
3. 进入**设置 → 打印机**，确认IP地址和访问码

:::tip 使用LAN模式以获得更快的响应
LAN模式比云模式延迟更低。如果打印机和仪表板在同一网络上，请在打印机设置中启用它。
:::

## 第2步 — 上传你的模型

Bambu Dashboard不会直接启动打印——这是Bambu Studio或MakerWorld的工作。打印开始后仪表板才接管。

**通过Bambu Studio：**
1. 在PC上打开Bambu Studio
2. 导入或打开你的`.stl`或`.3mf`文件
3. 切片模型（选择耗材、支撑、填充等）
4. 点击右上角的**打印**

**通过MakerWorld：**
1. 在[makerworld.com](https://makerworld.com)找到模型
2. 直接从网站点击**打印**
3. Bambu Studio自动打开并加载模型

## 第3步 — 开始打印

在Bambu Studio中选择发送方式：

| 方式 | 要求 | 优点 |
|------|------|------|
| **云端** | Bambu账号 + 网络 | 随处可用 |
| **LAN** | 同一网络 | 更快，无需云端 |
| **SD卡** | 物理访问 | 无网络要求 |

点击**发送**——打印机接收任务并自动开始预热阶段。

:::info 打印将出现在仪表板中
Bambu Studio发送任务几秒钟内，正在进行的打印会显示在仪表板**当前打印**下方。
:::

## 第4步 — 在仪表板中监控

打印进行中时，仪表板为你提供完整概览：

### 进度
- 完成百分比和预计剩余时间显示在打印机卡片上
- 点击卡片查看包含层信息的详细视图

### 温度
详情面板显示实时温度：
- **喷嘴** — 当前温度和目标温度
- **热床** — 当前温度和目标温度
- **腔体** — 打印机内部温度（对ABS/ASA很重要）

### 摄像头
点击打印机卡片上的摄像头图标，即可在仪表板中查看实时画面。你可以在做其他事情的同时将摄像头在独立窗口中保持打开。

:::warning 检查最初几层
最初3〜5层至关重要。现在粘附不良意味着之后打印失败。观察摄像头，确认耗材整齐均匀地铺设。
:::

### Print Guard
Bambu Dashboard具有AI驱动的**Print Guard**，可自动检测意大利面错误并可暂停打印。在**监控 → Print Guard**下启用它。

## 第5步 — 打印完成后

打印完成后，仪表板会显示完成消息（如果你设置了[通知](./varsler-oppsett)，还会发送通知）。

### 查看历史记录
进入侧边栏中的**历史记录**查看已完成的打印：
- 总打印时间
- 耗材消耗量（使用的克数、预计费用）
- 打印过程中的错误或HMS事件
- 完成时的摄像头图像（如果已启用）

### 记录备注
点击历史记录中的打印并添加备注——例如"需要多一点裙边"或"完美结果"。当你再次打印同一模型时很有用。

### 检查耗材消耗
在**耗材**下，你可以看到线轴重量已根据使用量更新。仪表板会自动扣除。

## 新手建议

:::tip 不要离开第一次打印
注意观察最初的10〜15分钟。当你确认打印粘附良好后，可以让仪表板监控剩余部分。
:::

- **称量空线轴** — 输入线轴起始重量以精确计算剩余量（参见[耗材库存](./filament-oppsett)）
- **设置Telegram通知** — 无需等待即可在打印完成时收到通知（参见[通知](./varsler-oppsett)）
- **检查打印平台** — 干净的平台 = 更好的粘附性。在打印之间用IPA（异丙醇）擦拭
- **使用正确的平台** — 参见[选择正确的打印平台](./velge-rett-plate)了解适合你耗材的平台
