# 多台打印机

你有不止一台打印机吗？Bambu Dashboard专为机群管理而设计——你可以从一个地方监控、控制和协调所有打印机。

## 添加新打印机

1. 进入**设置 → 打印机**
2. 点击**+ 添加打印机**
3. 填写：

| 字段 | 示例 | 说明 |
|------|------|------|
| 序列号 (SN) | 01P... | 在Bambu Handy或打印机屏幕上找到 |
| IP地址 | 192.168.1.101 | LAN模式（推荐） |
| 访问码 | 12345678 | 打印机屏幕上的8位代码 |
| 名称 | "Bambu #2 - P1S" | 显示在仪表板中 |
| 型号 | P1P, P1S, X1C, A1 | 选择正确型号以获取正确的图标和功能 |

4. 点击**测试连接** — 你应该看到绿色状态
5. 点击**保存**

:::tip 给打印机起有描述性的名称
"Bambu 1"和"Bambu 2"容易混淆。使用"X1C - 生产"和"P1S - 原型"这样的名称来保持清晰。
:::

## 机群概览

添加所有打印机后，它们会集中显示在**机群**面板中。在这里你可以看到：

```
┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
│ X1C - 生产      │  │ P1S - 原型      │  │ A1 - 爱好室     │
│ ████████░░ 82%  │  │ 空闲            │  │ ████░░░░░░ 38%  │
│ 剩余1小时24分   │  │ 准备打印        │  │ 剩余3小时12分   │
│ 温度: 220/60°C  │  │ AMS: 4个线轴   │  │ 温度: 235/80°C  │
└─────────────────┘  └─────────────────┘  └─────────────────┘
```

你可以：
- 点击打印机查看完整详情
- 一次查看所有温度、AMS状态和当前错误
- 按状态筛选（活动打印、空闲、错误）

## 打印队列 — 分配工作

打印队列让你可以从一个地方为所有打印机计划打印任务。

**工作原理：**
1. 进入**队列**
2. 点击**+ 添加任务**
3. 选择文件和设置
4. 选择打印机，或选择**自动分配**

### 自动分配
自动分配时，仪表板根据以下条件选择打印机：
- 可用容量
- AMS中可用耗材
- 计划的维护窗口

在**设置 → 队列 → 自动分配**中启用。

### 优先级
在队列中拖放任务以更改顺序。**高优先级**任务会排到普通任务前面。

## 交错启动 — 避免电流峰值

如果你同时启动多台打印机，预热阶段可能会产生强烈的电流峰值。交错启动分散启动时间：

**启用方法：**
1. 进入**设置 → 机群 → 交错启动**
2. 启用**分散启动**
3. 设置打印机之间的延迟（建议：2〜5分钟）

**3台打印机延迟3分钟的示例：**
```
08:00 — 打印机1开始预热
08:03 — 打印机2开始预热
08:06 — 打印机3开始预热
```

:::tip 与保险丝大小相关
X1C预热时消耗约1000W。三台同时 = 3000W，可能触发16A保险丝。交错启动可消除此问题。
:::

## 打印机组

打印机组让你可以逻辑化地组织打印机并向整个组发送命令：

**创建组：**
1. 进入**设置 → 打印机组**
2. 点击**+ 新建组**
3. 给组命名（例如"生产车间"、"爱好室"）
4. 将打印机添加到组中

**组功能：**
- 查看组的汇总统计
- 同时向整个组发送暂停命令
- 为组设置维护窗口

## 监控所有打印机

### 多摄像头视图
进入**机群 → 摄像头视图**，并排查看所有摄像头画面：

```
┌──────────────┐  ┌──────────────┐
│  X1C 画面    │  │  P1S 画面    │
│  [直播]      │  │  [空闲]      │
└──────────────┘  └──────────────┘
┌──────────────┐  ┌──────────────┐
│  A1 画面     │  │  + 添加      │
│  [直播]      │  │              │
└──────────────┘  └──────────────┘
```

### 按打印机配置通知
你可以为不同打印机配置不同的通知规则：
- 生产打印机：始终通知，包括夜间
- 爱好打印机：仅白天通知

通知设置请参见[通知](./varsler-oppsett)。

## 机群运营提示

- **标准化耗材槽位**：在所有打印机上保持槽位1为PLA白色，槽位2为PLA黑色 — 任务分配更简单
- **每天检查AMS料量**：早晨例行检查请参见[日常使用](./daglig-bruk)
- **轮流维护**：不要同时维护所有打印机 — 始终保持至少一台活跃
- **清晰命名文件**：`logo_x1c_pla_0.2mm.3mf`这样的文件名便于选择正确的打印机
