# 选择正确的打印平台

正确的打印平台对于良好的粘附性和轻松取下打印件至关重要。错误的组合会导致粘附不良，或打印件粘太牢而损坏平台。

## 概览表

| 耗材 | 推荐平台 | 固体胶棒 | 平台温度 |
|------|---------|---------|---------|
| PLA | 冷平台 / 纹理PEI | 否 / 是 | 35–45°C |
| PETG | 纹理PEI | **是（必须）** | 70°C |
| ABS | 工程平台 / 高温平台 | 是 | 90–110°C |
| ASA | 工程平台 / 高温平台 | 是 | 90–110°C |
| TPU | 纹理PEI | 否 | 35–45°C |
| PA（尼龙） | 工程平台 | 是 | 90°C |
| PC | 高温平台 | 是 | 100–120°C |
| PLA-CF / PETG-CF | 工程平台 | 是 | 45–90°C |
| PVA | 冷平台 | 否 | 35°C |

## 平台描述

### 冷平台（光滑PEI）
**最适合：** PLA、PVA
**表面：** 光滑，给打印件底部光滑表面
**取下：** 轻轻弯曲平台或等其冷却 — 打印件自然脱落

不要将冷平台与PETG一起使用 — 它粘附**太牢**，可能撕裂平台的涂层。

### 纹理PEI（有纹理）
**最适合：** PETG、TPU、PLA（底面有纹理效果）
**表面：** 有纹理，打印件底部有粗糙美观的效果
**取下：** 等到室温 — 自然脱落

:::warning PETG在纹理PEI上需要固体胶棒
没有固体胶棒，PETG会极其牢固地粘附在纹理PEI上，取下时可能剥离涂层。始终在整个表面涂一薄层固体胶棒（Bambu固体胶棒或Elmer's消失紫色胶水）。
:::

### 工程平台
**最适合：** ABS、ASA、PA、PLA-CF、PETG-CF
**表面：** 具有比纹理PEI更低粘附力的亚光PEI表面
**取下：** 冷却后易于取下。ABS/ASA使用固体胶棒

### 高温平台
**最适合：** PC、PA-CF、高温下的ABS
**表面：** 无变形地承受高达120°C的平台温度
**取下：** 冷却至室温

## 常见错误

### 光滑冷平台上的PETG（不用固体胶棒）
**问题：** PETG粘结过强，无法无损取下
**解决方案：** 始终使用带固体胶棒的纹理PEI或工程平台

### 冷平台上的ABS
**问题：** 翘曲 — 打印过程中边角抬起
**解决方案：** 工程平台 + 固体胶棒 + 提高腔体温度（关闭前盖）

### 高温平台上的PLA
**问题：** 平台温度过高导致粘附过强，难以取下
**解决方案：** PLA使用冷平台或纹理PEI

### 固体胶棒涂太多
**问题：** 厚固体胶棒导致大象脚（第一层外扩）
**解决方案：** 只涂一薄层 — 固体胶棒几乎看不见才行

## 更换平台

1. **让平台冷却**至室温（或戴手套 — 平台可能很烫）
2. 从前面抬起平台取出
3. 放入新平台 — 磁铁会将其固定到位
4. 在Bambu Studio或通过仪表板的**控制 → 校准**进行平台更换后的**自动校准**（流量率和床面调平）

:::info 记住更换后要校准
平台厚度略有不同。不进行校准，第一层可能离平台太远或撞入平台。
:::

## 平台维护

### 清洁（每2〜5次打印后）
- 用IPA（异丙醇70〜99%）和无绒纸巾擦拭
- 避免用裸手触摸表面 — 皮肤油脂会降低粘附力
- 对于纹理PEI：多次打印后用温水和温和洗洁精清洗

### 清除固体胶棒残留
- 将平台加热至60°C
- 用湿布擦拭
- 最后用IPA擦拭

### 更换
出现以下情况时更换平台：
- 取下打印件后出现明显凹坑或痕迹
- 清洁后仍持续粘附不良
- 涂层出现气泡或斑点

根据耗材类型和护理情况，Bambu平台通常可用200〜500次打印。

:::tip 正确存储平台
将未使用的平台放在原包装中或竖立存放 — 不要叠放重物在上面。变形的平台会导致第一层不均匀。
:::
