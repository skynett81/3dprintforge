# 日常使用

本指南介绍如何在日常生活中高效使用Bambu Dashboard — 从开始一天到关灯为止。

## 早晨例行检查

打开仪表板，快速检查以下几点：

### 1. 检查打印机状态
概览面板显示所有打印机的状态。注意查看：
- **红色图标** — 需要处理的错误
- **待处理消息** — 夜间的HMS警告
- **未完成的打印** — 如果有夜间打印，是否已完成？

### 2. 检查AMS料量
进入**耗材**或查看仪表板上的AMS小部件：
- 是否有低于100g的线轴？更换或订购新的
- 今天的打印任务是否使用了正确槽位中的耗材？

### 3. 检查警报和事件
在**通知日志**（铃铛图标）中查看：
- 夜间发生的事件
- 自动记录的错误
- 触发警报的HMS代码

## 开始打印

### 从文件（Bambu Studio）
1. 打开Bambu Studio
2. 加载并切片模型
3. 发送到打印机 — 仪表板自动更新

### 从队列
如果你提前计划了打印任务：
1. 进入**队列**
2. 点击**开始下一个**或将任务拖到顶部
3. 用**发送到打印机**确认

有关队列管理的完整信息，请参见[打印队列文档](../features/queue)。

### 计划打印（调度器）
要在特定时间开始打印：
1. 进入**调度器**
2. 点击**+ 新任务**
3. 选择文件、打印机和时间
4. 启用**电价优化**自动选择最便宜的时段

详情请参见[调度器](../features/scheduler)。

## 监控正在进行的打印

### 摄像头视图
点击打印机卡片上的摄像头图标。你可以：
- 在仪表板中查看实时画面
- 在单独的标签页中打开以进行后台监控
- 手动截图

### 进度信息
当前打印卡片显示：
- 完成百分比
- 预计剩余时间
- 当前层/总层数
- 活动耗材和颜色

### 温度
实时温度曲线显示在详情面板中：
- 喷嘴温度 — 应在±2°C内保持稳定
- 平台温度 — 对良好粘附很重要
- 腔体温度 — 逐渐升高，对ABS/ASA尤其重要

### Print Guard
如果**Print Guard**已激活，仪表板会自动监控意大利面和体积偏差。如果检测到问题：
1. 打印暂停
2. 你会收到通知
3. 摄像头图像被保存供后续检查

## 打印后 — 检查程序

### 检查质量
1. 打开摄像头，在结果还在平台上时查看
2. 进入**历史记录 → 最近打印**查看统计数据
3. 记录备注：哪些方面好，哪些方面可以改进

### 归档
历史记录中的打印永远不会自动归档 — 它们会一直保留。如果想整理：
- 点击打印 → **归档**将其移至存档
- 使用**项目**将相关打印分组

### 更新耗材重量
如果你为了精度而称量线轴（推荐）：
1. 称量线轴
2. 进入**耗材 → [线轴]**
3. 更新**剩余重量**

## 维护提醒

仪表板自动跟踪维护间隔。在**维护**下查看：

| 任务 | 间隔 | 状态 |
|------|------|------|
| 清洁喷嘴 | 每50小时 | 自动检查 |
| 润滑导轨 | 每200小时 | 仪表板跟踪 |
| 校准平台 | 更换平台后 | 手动提醒 |
| 清洁AMS | 每月 | 日历通知 |

在**监控 → 维护 → 通知**下激活维护警报。

:::tip 设置每周维护日
每周固定的维护日（例如周日晚上）可以避免不必要的停机时间。使用仪表板中的提醒功能。
:::

## 电价 — 最佳打印时机

如果你已连接电价集成（Nordpool / Home Assistant）：

1. 进入**分析 → 电价**
2. 查看未来24小时的价格图表
3. 最便宜的时段用绿色标记

使用启用了**电价优化**的**调度器** — 仪表板将自动在最便宜的可用时间窗口内开始任务。

:::info 典型最便宜时段
在挪威，夜间（01:00〜06:00）通常最便宜。前一天晚上将8小时打印任务放入队列可节省30〜50%的电费。
:::
