# 将3D打印直播到OBS

3DPrintForge内置OBS叠加层，可以直接在你的直播中显示打印机状态、进度、温度和摄像头画面。

## 前提条件

- 安装OBS Studio（[obsproject.com](https://obsproject.com)）
- 3DPrintForge运行中并已连接打印机
- （可选）Bambu摄像头已启用以获取实时画面

## 第1步 — OBS浏览器源

OBS内置**浏览器源**，可以直接在场景中显示网页。

**在OBS中添加叠加层：**

1. 打开OBS Studio
2. 在**来源**（Sources）下点击**+**
3. 选择**浏览器**（Browser）
4. 给来源命名，例如"Bambu Overlay"
5. 填写：

| 设置 | 值 |
|------|-----|
| URL | `http://localhost:3000/obs/overlay` |
| 宽度 | `1920` |
| 高度 | `1080` |
| FPS | `30` |
| 自定义CSS | 见下方 |

6. 勾选**通过OBS控制音频**
7. 点击**确定**

:::info 根据你的服务器调整URL
OBS在与仪表板不同的机器上运行？将`localhost`替换为服务器的IP地址，例如`http://192.168.1.50:3000/obs/overlay`
:::

## 第2步 — 透明背景

要使叠加层融入画面，背景必须透明：

**在OBS浏览器源设置中：**
- 勾选**删除背景**（Shutdown source when not visible / Remove background）

**强制透明的自定义CSS：**
```css
body {
  background-color: rgba(0, 0, 0, 0) !important;
  margin: 0;
  overflow: hidden;
}
```

将此粘贴到浏览器源设置中的**自定义CSS**字段。

叠加层现在只显示小部件本身——没有白色或黑色背景。

## 第3步 — 自定义叠加层

在3DPrintForge中你可以配置叠加层显示什么：

1. 进入**功能 → OBS叠加层**
2. 配置：

| 设置 | 选项 |
|------|------|
| 位置 | 左上、右上、左下、右下 |
| 大小 | 小、中、大 |
| 主题 | 暗色、亮色、透明 |
| 强调色 | 选择适合直播风格的颜色 |
| 元素 | 选择显示内容（见下方） |

**可用叠加层元素：**

- 打印机名称和状态（在线/打印中/错误）
- 带百分比和剩余时间的进度条
- 耗材和颜色
- 喷嘴温度和平台温度
- 已用耗材（克）
- AMS概览（紧凑型）
- Print Guard状态

3. 点击**预览**，无需切换到OBS即可查看结果
4. 点击**保存**

:::tip 每台打印机的URL
有多台打印机？使用单独的叠加层URL：
```
/obs/overlay?printer=1
/obs/overlay?printer=2
```
:::

## OBS中的摄像头画面（单独来源）

Bambu摄像头可以作为OBS中的单独来源添加——独立于叠加层：

**方式1：通过仪表板的摄像头代理**

1. 进入**系统 → 摄像头**
2. 复制**RTSP或MJPEG流的URL**
3. 在OBS中：点击**+** → **媒体源**（Media Source）
4. 粘贴URL
5. 勾选**循环**，禁用本地文件

**方式2：带摄像头视图的浏览器源**

1. 在OBS中：添加**浏览器源**
2. URL：`http://localhost:3000/obs/camera?printer=1`
3. 宽度/高度：匹配摄像头分辨率（1080p或720p）

现在你可以在场景中自由放置摄像头画面并将叠加层叠加在上面。

## 良好直播的技巧

### 直播场景设置

3D打印直播的典型场景：

```
┌─────────────────────────────────────────┐
│                                         │
│      [来自打印机的摄像头画面]           │
│                                         │
│  ┌──────────────────┐                  │
│  │ Bambu Overlay    │  ← 左下角        │
│  │ 打印: Logo.3mf   │                  │
│  │ ████████░░ 82%   │                  │
│  │ 剩余1小时24分    │                  │
│  │ 220°C / 60°C     │                  │
│  └──────────────────┘                  │
└─────────────────────────────────────────┘
```

### 推荐设置

| 参数 | 推荐值 |
|------|--------|
| 叠加层大小 | 中（不要太显眼） |
| 更新频率 | 30 FPS（与OBS匹配） |
| 叠加层位置 | 左下（避免遮挡脸部/聊天） |
| 颜色主题 | 带蓝色强调的暗色 |

### 场景和切换场景

创建自定义OBS场景：

- **"打印中"** — 摄像头视图 + 叠加层
- **"暂停/等待"** — 静态图像 + 叠加层
- **"完成"** — 结果图像 + 显示"已完成"的叠加层

用OBS中的键盘快捷键或通过场景集合切换场景。

### 摄像头画面稳定化

Bambu摄像头有时可能会冻结。在仪表板**系统 → 摄像头**下：
- 启用**自动重连** — 仪表板会自动重新连接
- 将**重连间隔**设置为10秒

### 音频

3D打印机会发出声音——尤其是AMS和冷却风扇。考虑：
- 将麦克风放在远离打印机的地方
- 在OBS中为麦克风添加噪音过滤器（Noise Suppression）
- 或使用背景音乐/聊天声音

:::tip 自动切换场景
OBS内置基于标题的场景切换支持。结合插件（例如obs-websocket）和3DPrintForge API，在打印开始和停止时自动切换场景。
:::
