# 在仪表板中导航

本指南简要介绍仪表板的组织方式以及如何高效导航。

## 侧面板

左侧的侧面板是你的导航中心。它按部分组织：

```
┌────────────────────┐
│ 🖨  打印机状态      │  ← 每台打印机一行
├────────────────────┤
│ 概览               │
│ 机群               │
│ 当前打印           │
├────────────────────┤
│ 耗材               │
│ 历史记录           │
│ 项目               │
│ 队列               │
│ 调度器             │
├────────────────────┤
│ 监控               │
│  └ Print Guard     │
│  └ 错误            │
│  └ 诊断            │
│  └ 维护            │
├────────────────────┤
│ 分析               │
│ 工具               │
│ 集成               │
│ 系统               │
├────────────────────┤
│ ⚙ 设置            │
└────────────────────┘
```

点击左上角的汉堡图标（☰）可以**隐藏侧面板**。在小屏幕或信息亭模式下很有用。

## 主面板

点击侧面板中的项目时，内容会显示在右侧的主面板中。布局因面板而异：

| 面板 | 布局 |
|------|------|
| 概览 | 所有打印机的卡片网格 |
| 当前打印 | 大型详情卡片 + 温度曲线 |
| 历史记录 | 可筛选的表格 |
| 耗材 | 线轴的卡片视图 |
| 分析 | 图表和图形 |

## 点击打印机状态查看详情

概览面板上的打印机卡片可以点击：

**单击** → 打开该打印机的详情面板：
- 实时温度
- 当前打印（如果正在进行）
- 所有插槽的AMS状态
- 最新错误和事件
- 快捷按钮：暂停、停止、灯光开/关

**点击摄像头图标** → 打开实时摄像头视图

**点击⚙图标** → 打印机设置

## 键盘快捷键 — 命令面板

命令面板无需导航即可快速访问所有功能：

| 快捷键 | 操作 |
|--------|------|
| `Ctrl + K` (Linux/Windows) | 打开命令面板 |
| `Cmd + K` (macOS) | 打开命令面板 |
| `Esc` | 关闭面板 |

在命令面板中你可以：
- 搜索页面和功能
- 直接开始打印
- 暂停/恢复当前打印
- 切换主题（亮色/暗色）
- 导航到任何页面

**示例：** 按`Ctrl+K`，输入"暂停" → 选择"暂停所有当前打印"

## 小部件自定义

概览面板可以使用你选择的小部件进行自定义：

**如何编辑仪表板：**
1. 点击概览面板右上角的**编辑布局**（铅笔图标）
2. 将小部件拖到所需位置
3. 点击并拖动小部件的角落来调整大小
4. 点击**+ 添加小部件**添加新的：

可用小部件：

| 小部件 | 显示内容 |
|--------|---------|
| 打印机状态 | 所有打印机的卡片 |
| 当前打印（大） | 正在进行的打印的详细视图 |
| AMS概览 | 所有插槽和耗材量 |
| 温度曲线 | 实时图表 |
| 电价 | 未来24小时的价格图表 |
| 耗材计量器 | 过去30天的总消耗量 |
| 历史记录快捷方式 | 最近5次打印 |
| 摄像头画面 | 实时摄像头图像 |

5. 点击**保存布局**

:::tip 保存多个布局
你可以有不同用途的不同布局——日常使用的紧凑布局，悬挂在大屏上的大布局。使用布局选择器切换。
:::

## 主题 — 在亮色和暗色之间切换

**快速切换：**
- 点击导航右上角的太阳/月亮图标
- 或：`Ctrl+K` → 输入"主题"

**永久设置：**
1. 进入**系统 → 主题**
2. 在以下选项中选择：
   - **亮色** — 白色背景
   - **暗色** — 深色背景（夜间推荐）
   - **自动** — 跟随设备系统设置
3. 选择主题色（蓝色、绿色、紫色等）
4. 点击**保存**

## 键盘导航

无鼠标高效导航：

| 快捷键 | 操作 |
|--------|------|
| `Tab` | 下一个交互元素 |
| `Shift+Tab` | 上一个元素 |
| `Enter` / `Space` | 激活按钮/链接 |
| `Esc` | 关闭模态/下拉 |
| `Ctrl+K` | 命令面板 |
| `Alt+1` – `Alt+9` | 直接导航到前9个页面 |

## PWA — 作为应用安装

3DPrintForge可以作为渐进式网络应用（PWA）安装，作为没有浏览器菜单的独立应用运行：

1. 在Chrome、Edge或Safari中访问仪表板
2. 点击地址栏中的**安装应用**图标
3. 确认安装

更多详情请参见[PWA文档](../system/pwa)。

## 信息亭模式

信息亭模式隐藏所有导航，只显示仪表板——非常适合打印工作室的专用屏幕：

1. 进入**系统 → 信息亭**
2. 启用**信息亭模式**
3. 选择要显示的小部件
4. 设置更新间隔

完整设置请参见[信息亭文档](../system/kiosk)。
