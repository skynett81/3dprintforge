# 产品定价 — 计算销售价格

本指南说明如何使用成本计算器为您销售的3D打印品找到合适的销售价格。

## 成本概览

3D打印的成本由以下部分组成：

| 组成部分 | 说明 | 示例 |
|-----------|-------------|---------|
| **耗材** | 基于重量和线轴价格的材料成本 | 100g × 0.25 kr/g = 25 kr |
| **废料** | 材料浪费（清洗、失败打印、支撑） | 10%额外 = 2.50 kr |
| **电力** | 打印期间的电力消耗 | 3.5小时 × 150W × 1.50 kr/kWh = 0.79 kr |
| **磨损** | 喷嘴 + 机器在使用寿命内的价值 | 3.5小时 × 0.15 kr/小时 = 0.53 kr |
| **人工** | 您用于设置、后处理、包装的时间 | 10分钟 × 200 kr/小时 = 33.33 kr |
| **加价** | 利润率 | 20% = 12.43 kr |

**总生产成本** = 所有组成部分的总和

## 配置设置

### 基本设置

前往 **Filament → ⚙ 设置** 并填写：

1. **电价（kr/kWh）** — 您的电价。查看电费账单或使用Nordpool集成
2. **打印机功率（W）** — Bambu Lab打印机通常为150W
3. **机器成本（kr）** — 您为打印机支付的金额
4. **机器寿命（小时）** — 预期使用寿命（3000-8000小时）
5. **人工成本（kr/小时）** — 您的时薪
6. **准备时间（分钟）** — 更换耗材、检查打印板、包装的平均时间
7. **加价（%）** — 您期望的利润率
8. **喷嘴成本（kr/小时）** — 喷嘴磨损（HS01 ≈ 0.05 kr/小时）
9. **废料系数** — 材料浪费（1.1 = 10%额外，1.15 = 15%）

:::tip Bambu Lab的典型值
| 设置 | 爱好者 | 半专业 | 专业 |
|---|---|---|---|
| 电价 | 1.50 kr/kWh | 1.50 kr/kWh | 1.00 kr/kWh |
| 打印机功率 | 150W | 150W | 150W |
| 机器成本 | 5 000 kr | 12 000 kr | 25 000 kr |
| 机器寿命 | 3 000小时 | 5 000小时 | 8 000小时 |
| 人工成本 | 0 kr/小时 | 150 kr/小时 | 250 kr/小时 |
| 准备时间 | 5分钟 | 10分钟 | 15分钟 |
| 加价 | 0% | 30% | 50% |
| 废料系数 | 1.05 | 1.10 | 1.15 |
:::

## 计算成本

1. 前往**成本计算器**（`https://localhost:3443/#costestimator`）
2. **拖放**一个`.3mf`或`.gcode`文件
3. 系统自动读取：耗材重量、预估时间、颜色
4. **关联线轴** — 选择库存中使用的线轴
5. 点击**计算成本**

### 结果显示：

- **耗材** — 每种颜色的材料成本
- **废料/损耗** — 基于废料系数
- **电力** — 如果可用，使用Nordpool的实时现货价格
- **磨损** — 喷嘴 + 机器价值
- **人工** — 时薪 + 准备时间
- **生产成本** — 以上所有项目的总和
- **加价** — 您的利润率
- **总成本** — 您应收取的最低金额
- **建议销售价格** — 2×、2.5×、3×利润率

## 定价策略

### 2×利润率（推荐最低值）
覆盖生产成本 + 意外支出。用于朋友/家人和简单几何形状。

### 2.5×利润率（标准）
价格与价值之间的良好平衡。适用于大多数产品。

### 3×利润率（高端）
适用于复杂模型、多色、高质量或利基市场。

:::warning 别忘了隐藏成本
- 失败打印（所有打印的5-15%会失败）
- 未用完的耗材（最后50g通常很难使用）
- 花在客户服务上的时间
- 包装和运输
- 打印机维护
:::

## 示例：为手机支架定价

| 参数 | 值 |
|-----------|-------|
| 耗材重量 | 45g PLA |
| 打印时间 | 2小时 |
| 现货价格 | 1.20 kr/kWh |

**计算：**
- 耗材：45g × 0.25 kr/g = 11.25 kr
- 废料（10%）：1.13 kr
- 电力：2小时 × 0.15kW × 1.20 = 0.36 kr
- 磨损：2小时 × 0.15 = 0.30 kr
- 人工：（2小时 + 10分钟）× 200 kr/小时 = 433 kr（爱好者为0）
- **生产成本（爱好者）**：约13 kr
- **销售价格 2.5×**：约33 kr

## 保存估算

点击**保存估算**以存档计算结果。保存的估算可在成本计算器的**已保存**标签页中找到。

## 电子商务

如果您使用[电子商务模块](../integrations/ecommerce)，可以将成本估算直接关联到订单，实现自动价格计算。
