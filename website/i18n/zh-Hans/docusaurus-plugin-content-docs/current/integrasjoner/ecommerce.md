# 电商

电商模块为您提供完整的客户、订单和开票管理系统——非常适合专业或半专业销售 3D 打印品的用户。

访问地址：**https://localhost:3443/#orders**

:::danger 需要电商许可证
电商模块需要有效的许可证。许可证**只能通过 [geektech.no](https://geektech.no) 购买**。没有有效许可证，该模块将被锁定，无法使用。
:::

## 许可证 — 购买与激活

### 购买许可证

1. 访问 **[geektech.no](https://geektech.no)** 并创建账户
2. 选择 **Bambu Dashboard — 电商许可证**
3. 选择许可证类型：

| 许可证类型 | 说明 | 打印机数量 |
|---|---|---|
| **爱好者** | 单台打印机，个人使用和小额销售 | 1 |
| **专业版** | 最多 5 台打印机，商业使用 | 1–5 |
| **企业版** | 无限台打印机，全面支持 | 无限 |

4. 完成付款
5. 您将通过邮件收到一个**许可证密钥**

### 激活许可证

1. 在仪表板中进入**设置 → 电商**
2. 在字段中粘贴**许可证密钥**
3. 点击**激活许可证**
4. 仪表板将通过 geektech.no 服务器验证密钥
5. 成功激活后将显示许可证类型、到期日期和打印机数量

:::warning 许可证密钥与您的安装绑定
密钥针对一个 Bambu Dashboard 安装激活。如果您需要将许可证迁移到新服务器，请联系 [geektech.no](https://geektech.no)。
:::

### 许可证验证

- 许可证在启动时**在线验证**，此后每 24 小时验证一次
- 网络中断时，许可证可离线使用最多 **7 天**
- 许可证过期 → 模块锁定，但现有数据保留
- 续期通过 **[geektech.no](https://geektech.no)** → 我的许可证 → 续期

### 检查许可证状态

进入**设置 → 电商**或调用 API：

```bash
curl -sk https://localhost:3443/api/ecom-license/status
```

响应包含：
```json
{
  "active": true,
  "type": "professional",
  "expires": "2027-03-22",
  "printers": 5,
  "licensee": "公司名称",
  "provider": "geektech.no"
}
```

## 客户

### 创建客户

1. 进入**电商 → 客户**
2. 点击**新客户**
3. 填写：
   - **姓名 / 公司名称**
   - **联系人**（对于企业客户）
   - **电子邮件地址**
   - **电话**
   - **地址**（账单地址）
   - **公司注册号**（可选，用于增值税注册企业）
   - **备注** — 内部备注
4. 点击**创建**

### 客户列表

客户列表显示：
- 姓名和联系信息
- 总订单数
- 总销售额
- 最近订单日期
- 状态（活跃 / 非活跃）

点击客户查看所有订单和开票历史。

## 订单管理

### 创建订单

1. 进入**电商 → 订单**
2. 点击**新订单**
3. 从列表中选择**客户**
4. 添加订单项：
   - 从文件库选择文件/模型，或添加自由文本项目
   - 设置数量和单价
   - 如果关联到项目，系统自动计算费用
5. 设置**预计交货日期**
6. 点击**创建订单**

### 订单状态

| 状态 | 说明 |
|---|---|
| 询价中 | 已收到询价，尚未确认 |
| 已确认 | 客户已确认 |
| 生产中 | 打印进行中 |
| 待交付 | 已完成，等待取货/发货 |
| 已交付 | 订单完成 |
| 已取消 | 被客户或您取消 |

点击订单 → **更改状态**以更新状态。

### 将打印关联到订单

1. 打开订单
2. 点击**关联打印**
3. 从历史记录中选择打印（支持多选）
4. 费用数据自动从打印历史中获取

## 开票

完整的开票文档请参见[项目 → 开票](../funksjoner/projects#fakturering)。

可直接从订单生成发票：

1. 打开订单
2. 点击**生成发票**
3. 检查金额和税率
4. 下载 PDF 或发送到客户邮箱

### 发票编号系列

在**设置 → 电商**中设置发票编号系列：
- **前缀**：例如 `2026-`
- **起始编号**：例如 `1001`
- 发票编号按升序自动分配

## 报告和税务

### 费用报告

系统追踪所有交易费用：
- 在**电商 → 费用**下查看费用
- 将费用标记为已报告以用于会计目的
- 按周期导出费用摘要

### 统计数据

在**电商 → 统计数据**中：
- 月销售额（柱状图）
- 按销售额排名的顶级客户
- 最畅销的模型/材料
- 平均订单规模

导出为 CSV 以用于会计系统。
