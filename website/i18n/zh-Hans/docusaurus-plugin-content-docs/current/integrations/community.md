# 社区耗材

社区耗材是由全球 Bambu Dashboard 用户贡献的共享耗材配置文件数据库。找到适合您耗材的最佳设置，并分享您的使用经验。

访问地址：**https://localhost:3443/#community**

## 浏览社区配置文件

### 搜索

1. 进入**社区耗材**
2. 使用搜索框按以下条件查找：
   - 材料（PLA、PETG、ABS、PA、TPU 等）
   - 供应商（PolyMaker、拓竹、Prusament、Sunlu 等）
   - 颜色名称或颜色十六进制代码
   - 特性（透明、哑光、丝绸、亮片等）
3. 按**评分**筛选（仅显示评分最高的）
4. 按**打印机型号**筛选，查看针对您打印机优化的配置文件

### 配置文件卡片

每个配置文件卡片显示：

| 字段 | 示例 |
|---|---|
| 耗材名称 | PolyTerra PLA – Matte Forest Green |
| 材料 | PLA |
| 供应商 | PolyMaker |
| 贡献者 | 用户名 |
| 评分 | ⭐⭐⭐⭐½（4.6 / 5） |
| 测试次数 | 23 |
| 推荐温度 | 喷嘴：215°C / 热床：55°C |
| 推荐速度 | 标准 |

点击卡片查看包含所有参数和用户评论的完整配置文件。

## 导入配置文件

1. 打开社区配置文件
2. 点击**导入到库存**
3. 选择：
   - **关联到现有耗材卷** — 配置文件关联到已注册的耗材卷
   - **创建新耗材卷** — 使用配置文件数据创建新的耗材卷条目
4. 根据需要调整温度或其他设置
5. 点击**导入**

配置文件现已在您的耗材库存中可用。

## 贡献您的配置文件

### 导出和分享

1. 进入**耗材库存**
2. 找到要分享的耗材卷
3. 点击**分享到社区**
4. 填写：
   - **供应商名称** — 准确的供应商名称
   - **产品名称** — 包装上的确切产品名称
   - **颜色名称** — 官方颜色名称和十六进制代码
   - **评论** — 您的使用经验、技巧和窍门
   - **打印机型号** — 您测试所用的打印机
5. 点击**提交**

配置文件正在审核中，通常在 24 小时内发布。

### 贡献指南

:::warning 质量要求
- 只提供经过实际测试的设置，不提供推测性数值
- 使用确切的供应商和产品名称（查看包装）
- 每个特定产品和打印机型号一个配置文件
- 避免重复配置文件 — 先检查配置文件是否已存在
:::

## 评价配置文件

帮助社区对您测试过的配置文件进行评价：

1. 打开社区配置文件
2. 点击**给予评价**
3. 选择星级（1–5）并写简短评论
4. 点击**提交**

评分高的配置文件将排在搜索结果顶部。

## 离线使用

社区数据库自动在本地同步（每天更新）。首次同步后，所有配置文件无需网络连接即可访问。

手动下载数据库：**社区耗材 → 设置 → 立即同步**
