# 电价

电价集成通过 Tibber 或 Nordpool 获取实时电价，为每次打印提供准确的电费计算，并在打印的好时机或不好时机发出提醒。

访问地址：**https://localhost:3443/#settings** → **集成 → 电价**

## Tibber 集成

Tibber 是一家提供现货价格开放 API 的电力供应商。

### 设置

1. 登录 [developer.tibber.com](https://developer.tibber.com)
2. 生成**个人访问令牌**
3. 在 Bambu Dashboard 中：在 **Tibber API Token** 下粘贴令牌
4. 选择**住所**（如果您有多个住所，请选择要获取价格的住所）
5. 点击**测试连接**
6. 点击**保存**

### 来自 Tibber 的可用数据

- **当前现货价格** — 含税费的即时价格（元/千瓦时）
- **未来 24 小时价格** — Tibber 大约在 13:00 提供次日价格
- **历史价格** — 最多 30 天前
- **每次打印的费用** — 根据实际打印时间 × 按小时价格计算

## Nordpool 集成

Nordpool 是提供北欧地区原始现货价格的能源交易所。

### 设置

1. 进入**集成 → Nordpool**
2. 选择**价格区域**：NO1（奥斯陆）/ NO2（克里斯蒂安桑）/ NO3（特隆赫姆）/ NO4（特罗姆斯）/ NO5（卑尔根）
3. 选择**货币**：NOK / EUR
4. 选择**税费**：
   - 勾选**含增值税**（25%）
   - 填写**电网费**（元/千瓦时）— 请查看电网公司的账单
   - 填写**能源税**（元/千瓦时）
5. 点击**保存**

:::info 电网费
电网费因电网公司和定价模式而异。请检查最近的电费账单以获取正确的费率。
:::

## 按小时电价

按小时电价以未来 24–48 小时的柱状图显示：

- **绿色** — 低价时段（低于平均值）
- **黄色** — 平均价格
- **红色** — 高价时段（高于平均值）
- **灰色** — 没有可用价格预测的时段

悬停在某个时段上查看精确价格（元/千瓦时）。

## 历史价格

进入**电价 → 历史**查看：

- 最近 30 天的每日平均价格
- 每天最贵和最便宜的时段
- 每天打印的总电费

## 价格提醒

设置基于电价的自动提醒：

1. 进入**电价 → 价格提醒**
2. 点击**新建提醒**
3. 选择提醒类型：
   - **价格低于阈值** — 当电价低于 X 元/千瓦时时提醒
   - **价格高于阈值** — 当价格超过 X 元/千瓦时时提醒
   - **今日最低价时段** — 当今天最低价时段开始时提醒
4. 选择通知渠道
5. 点击**保存**

:::tip 智能规划
将价格提醒与打印队列结合：设置一个自动化，当电价低时自动发送队列任务（需要 webhook 集成或 Home Assistant）。
:::

## 费用估算器中的电价

激活电价集成后，[费用估算器](../analytics/costestimator)可提供准确的电费计算。选择**实时价格**而非固定价格，以使用当前的 Tibber/Nordpool 价格。
