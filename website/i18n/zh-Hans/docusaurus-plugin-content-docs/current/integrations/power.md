# 功率监测

将带有能耗测量功能的智能插座连接到打印机，记录每次打印的实际用电量——而非仅凭估算。

访问地址：**https://localhost:3443/#settings** → **集成 → 功率监测**

## 支持的设备

| 设备 | 协议 | 推荐度 |
|---|---|---|
| **Shelly Plug S / Plus Plug S** | HTTP REST / MQTT | 推荐 — 设置简单 |
| **Shelly 1PM / 2PM** | HTTP REST / MQTT | 适用于固定安装场景 |
| **Shelly Gen2 / Gen3** | HTTP REST / MQTT | 较新型号，提供更多 API 功能 |
| **Tasmota 设备** | MQTT | 灵活，适用于自定义构建 |

:::tip 推荐设备
Shelly Plug S Plus（固件 1.0+）经过测试和推荐。支持 Wi-Fi、MQTT 和 HTTP REST，无需依赖云服务。
:::

## Shelly 设置

### 前提条件

- Shelly 插座已连接到与 Bambu Dashboard 相同的网络
- Shelly 已配置静态 IP 或 DHCP 地址预留

### 配置步骤

1. 进入**设置 → 功率监测**
2. 点击**添加功率计**
3. 选择**类型**：Shelly
4. 填写：
   - **IP 地址**：例如 `192.168.1.150`
   - **通道**：0（单插口插座）
   - **身份验证**：如已配置，填写用户名和密码
5. 点击**测试连接**
6. 将插座关联到**打印机**：从下拉列表中选择
7. 点击**保存**

### 轮询间隔

默认轮询间隔为 10 秒。减少至 5 秒可获得更精确的测量，增加至 30 秒可降低网络负载。

## Tasmota 设置

1. 为 Tasmota 设备配置 MQTT（请参阅 Tasmota 文档）
2. 在 Bambu Dashboard 中：选择**类型**：Tasmota
3. 填写设备的 MQTT 主题：例如 `tasmota/power-plug-1`
4. 关联到打印机并点击**保存**

Bambu Dashboard 将自动订阅 `{topic}/SENSOR` 获取功率数据。

## 测量内容

启用功率监测后，每次打印将记录以下数据：

| 指标 | 说明 |
|---|---|
| **瞬时功率** | 打印过程中的实时瓦特数 |
| **总用电量** | 整次打印的 kWh |
| **平均功率** | kWh / 打印时长 |
| **用电费用** | kWh × 电价（来自 Tibber/Nordpool） |

数据保存在打印历史中，可用于分析。

## 实时显示

瞬时功耗显示位置：

- **仪表板** — 作为额外小部件（在小部件设置中启用）
- **机群概览** — 作为打印机卡片上的小型指示器

## 与估算值对比

打印完成后将显示对比结果：

| | 估算值 | 实际值 |
|---|---|---|
| 用电量 | 1.17 kWh | 1.09 kWh |
| 用电费用 | 2.16 kr | 2.02 kr |
| 偏差 | — | -6.8 % |

持续的偏差可用于校准[费用估算器](../analytics/costestimator)中的估算参数。

## 打印完成后自动断电

Shelly/Tasmota 可在打印完成后自动关闭打印机电源：

1. 进入**功率监测 → [打印机] → 自动关闭**
2. 启用**打印完成后 X 分钟关闭电源**
3. 设置延时时间（例如 10 分钟）

:::danger 冷却注意
断电前请让打印机至少冷却 5–10 分钟。喷嘴应冷却至 50°C 以下，以防止热端热蠕变。
:::
