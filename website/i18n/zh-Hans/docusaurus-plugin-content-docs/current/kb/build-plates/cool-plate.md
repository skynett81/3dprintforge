# 光滑 PEI 板（Cool Plate）

Cool Plate 是一款光滑 PEI 平台，为打印件提供光滑亮面底部。大多数拓竹打印机附带此平台作为标准配置。

## 特性

- **表面：** 光滑，打印件底面亮面光滑
- **材质：** 弹簧钢板上的 PEI（聚醚酰亚胺）涂层
- **最高热床温度：** 90°C

## 最适合

- PLA（所有变体）——无需固体胶
- PLA+——无需固体胶
- PLA Silk——无需固体胶
- PLA-CF——无需固体胶

## 谨慎使用

:::danger PETG——必须使用固体胶
PETG 在光滑 PEI 上的粘附力**极强**。不涂固体胶时，取出打印件可能会撕毁平台上的 PEI 涂层。请务必使用固体胶作为隔离层，或改用工程板。
:::

## 维护

### 每次打印后
- 等平台冷却至室温（40°C 或以下）后再取出打印件
- PLA 在 25–30°C 时会自动从平台脱落
- 轻轻弯曲平台以取下粘附较紧的打印件

### 每周清洁
用异丙醇（IPA，70–99%）擦拭平台：
```
1. 将 IPA 倒在干净的无绒擦纸上（不要直接喷在平台上）
2. 沿一个方向擦拭表面
3. 彻底晾干（30 秒）后再开始下一次打印
```

### 粘附不良时
1. 用温水和温和洗洁精清洗
2. 彻底冲洗
3. 用无绒纸擦干
4. 最后用 IPA 擦拭一遍

:::warning 禁止使用丙酮
丙酮会溶解 PEI 涂层，造成永久性损坏。只能使用 IPA 或水/洗洁精。
:::

## 更换平台

出现以下情况时需要更换平台：
- PEI 涂层出现明显损坏（划痕、疤痕、脱落）
- 彻底清洁后仍然粘附不良
- 表面不平整，导致首层质量差

正常使用和良好维护下，Cool Plate 通常可使用 200–500 次打印。
