# 工程板（Engineering Plate）

Engineering Plate 是一款纹理 PEI 平台，专为 PETG、ABS、ASA 和 PA 等技术材料设计。它能为打印件提供精美的纹理底面，是拓竹产品系列中最通用的平台。

## 特性

- **表面：** 纹理 PEI——为打印件提供纹理/哑光底面
- **粘附性：** 比光滑 PEI 具有更好的机械附着力
- **最高热床温度：** 110°C
- **脱离：** 冷却至 40–50°C 后打印件易于取出

## 最适合

- PETG 和 PETG-CF（无需固体胶，或涂少量增强安全性）
- ABS 和 ASA（需涂固体胶）
- PA6 / PA12 / PA-CF（需涂固体胶——必须）
- TPU（无需固体胶）
- PLA（效果良好，呈现纹理底面）

## 不适合

- 需要光滑底面的打印件（请使用 Cool Plate）
- 高温 PC 打印（请使用 High Temp Plate）

## 维护

### 清洁
```bash
# 每周（或使用固体胶后）：
1. 用温水和温和洗洁精清洗
2. 彻底冲洗
3. 晾干，或用无绒纸擦干
4. 最后用 IPA 擦拭

# 未使用固体胶的打印之间：
1. 用 IPA 和无绒纸擦拭
```

### 使用固体胶后
使用固体胶后必须清洗平台。残留固体胶会影响下次打印的粘附均匀性。

:::tip 纹理方向
Engineering Plate 的纹理有方向性。请始终按正确朝向安装平台（使用夹片固定），避免打印件嵌入纹理凹槽中难以取出。
:::

## 更换平台

出现以下情况时需要更换平台：
- 中间区域的纹理已磨平/光滑（使用最频繁的部分）
- 清洁和涂固体胶后仍然粘附不良
- 平台出现深划痕或变形

良好维护下正常使用寿命：300–700 次打印。
