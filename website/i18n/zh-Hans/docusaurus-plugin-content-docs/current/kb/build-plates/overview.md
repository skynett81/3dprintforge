# 打印平台概览

拓竹提供四种不同的打印平台，每种针对不同材料和使用场景进行了优化。选择正确的平台可避免打印失败和平台损坏。

## 平台介绍

### 光滑 PEI 板（Cool Plate）
光滑 PEI 涂层平台，PLA 和 PLA 变体的最佳选择。**使用 PETG 必须涂固体胶**——不涂固体胶时 PETG 粘附力过强，可能撕毁涂层。

[了解更多 →](./cool-plate)

### 工程板（Engineering Plate）
纹理 PEI 具有更好的技术材料粘附力，赋予打印件漂亮的纹理底面。最佳全能平台。

[了解更多 →](./engineering-plate)

### 高温板（High Temp Plate）
适用于高温度要求材料：ABS、ASA、PC、PA。可承受高达 120°C 的热床温度。

[了解更多 →](./high-temp-plate)

### 纹理 PEI 板（Textured PEI Plate）
提供粗糙纹理。TPU 和需要良好机械附着力的材料的最佳选择。冷却后打印件易于脱离。

[了解更多 →](./textured-pei)

## 使用固体胶

固体胶有两个用途：
1. **隔离层**（光滑 PEI 板上的 PETG）——防止粘附过强导致平台损坏
2. **增强粘附**（ABS、PA 在所有平台上）——改善材料与平台之间的接触

**推荐固体胶：** 拓竹官方固体胶，或普通 Pritt 胶棒。涂抹薄薄一层，等待 30 秒晾干。

:::tip 用完固体胶后的清洗
使用固体胶后，用温水和温和洗洁精清洗平台。彻底晾干后再进行下一次打印。
:::
