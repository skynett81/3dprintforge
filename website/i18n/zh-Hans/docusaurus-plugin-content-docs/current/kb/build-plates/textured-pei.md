# 纹理 PEI 板（Textured PEI Plate）

Textured PEI Plate（不要与工程板混淆）具有更粗糙的纹理，特别适合 TPU 和其他柔性材料，同时也适用于多种常见材料。

## 特性

- **表面：** 粗糙 PEI 纹理——为打印件提供明显的纹理底面
- **粘附性：** 出色的机械附着力，尤其适合 TPU
- **脱离：** 冷却后极易脱离——弯曲平台即可轻松取出
- **最高热床温度：** 100°C

## 最适合

- **TPU** ——这是柔性材料的最佳平台
- PLA 和 PLA+——粘附良好，呈现精美纹理底面
- PETG——涂少量固体胶
- PLA-CF——无需固体胶

## 兼容性

| 材料 | 适用性 | 固体胶 |
|-----------|---------|---------|
| TPU | ★★★★★ | 否 |
| PLA | ★★★★★ | 否 |
| PLA+ | ★★★★★ | 否 |
| PETG | ★★★★ | 少量（推荐） |
| PLA-CF | ★★★★ | 否 |
| ABS | ★★★ | 是 |
| PA12 | ★★★ | 是 |

## TPU 在纹理 PEI 上的打印

TPU 无需固体胶即可完美粘附在纹理 PEI 上：
1. 设置热床温度 30–45°C
2. 仔细校准 Z 偏移——略高于 PLA 的设置
3. 打印完成后等热床冷却至 30°C 以下
4. 轻轻弯曲平台——TPU 毫不费力地脱落

:::tip 冷却取件
TPU 冷却后可从纹理 PEI 轻松脱离。如果取出打印件有困难，等平台完全冷却（20–25°C）后再用力弯曲。
:::

## 维护

纹理 PEI 坚固耐用，维护简单：

```bash
# 打印之间：
- 用 IPA 和无绒擦纸擦拭

# 每月（或粘附不良时）：
- 用温水和温和洗洁精清洗
- 冲洗并彻底擦干
- 最后用 IPA 处理
```

## 成品打印件的底面效果

粗糙纹理会在所有打印件底面留下明显的纹理图案。这对以下情况：
- 原型和功能性部件有益（更好的抓握力）
- 装饰性/视觉性打印件则不理想（请使用 Cool Plate 获得光滑底面）
