# 高温板（High Temp Plate）

High Temp Plate 专为需要高热床温度的材料设计。可承受高达 120°C 的温度，是打印聚碳酸酯（PC）和高温 PA 变体的必备平台。

## 特性

- **表面：** 带高温涂层的光滑表面
- **最高热床温度：** 120°C
- **打印件底面：** 亮面/光滑
- **粘附性：** 高温下粘附好——冷却后脱离

## 最适合

| 材料 | 热床温度 | 固体胶 |
|-----------|---------------|---------|
| ABS | 90–110°C | 是 |
| ASA | 90–110°C | 是 |
| PC | 100–120°C | 是（必须） |
| PA6 | 80–90°C | 是 |
| PA12 | 70–85°C | 是 |
| PA-CF | 90–100°C | 是 |

## 不适合

- PLA——平台温度过高，PLA 粘附过强
- PETG——不涂固体胶有损坏平台的风险
- TPU——工程板效果更好

## 固体胶

强烈推荐使用固体胶（PC 必须使用），原因如下：
1. 改善首层粘附
2. 保护平台涂层
3. 冷却后便于取出打印件

涂抹薄薄一层均匀的固体胶。PC 和 PA 用量略多于 ABS。

:::warning PC 需要特别注意
聚碳酸酯（PC）需要喷嘴温度 250–280°C、热床温度 100–120°C。没有密闭腔室和 High Temp Plate，成功率极低。打印 PC 建议仅使用 X1C 和 P1S。
:::

## 维护

```bash
# 使用固体胶打印 PC/ABS 后：
1. 等平台冷却至 50°C
2. 轻轻取出打印件（弯曲平台）
3. 用温水和洗洁精清洗
4. 彻底擦干——水分残留在热平台上可能损坏涂层

# 打印之间：
1. 用 IPA 擦拭
2. 检查涂层磨损情况
```

:::danger 避免热冲击
切勿将冷平台放在热热床上，也不要将冷 IPA 喷在热平台上。温度剧变可能损坏涂层或导致钢板变形。
:::

## 使用寿命

High Temp Plate 因承受更高热负荷，使用寿命比工程板短：
- **正常使用（ABS/ASA）：** 200–400 次打印
- **高强度使用（PC）：** 100–200 次打印

出现涂层明显磨损或持续粘附问题时需要更换。
