# 拉丝（Stringing）

拉丝（也称"渗料"）是在喷嘴移动时不挤料的情况下，部件不同区域之间形成的细塑料丝。这会使打印件呈现"蜘蛛网"状外观。

## 拉丝的原因

1. **喷嘴温度过高** — 高温塑料流动性强，容易滴落
2. **回抽设置不当** — 耗材不能足够快地缩回
3. **耗材受潮** — 水分导致蒸汽和额外流出
4. **速度过低** — 喷嘴在移动位置停留时间过长

## 诊断

**耗材受潮？** 打印时听到噼啪/爆裂声？说明耗材受潮——先烘料，再调整其他设置。

**温度过高？** 喷嘴在"暂停"时滴料？将温度降低 5–10°C。

## 解决方案

### 1. 烘干耗材

受潮耗材是无法通过调参解决的拉丝最常见原因：

| 材料 | 烘干温度 | 时间 |
|-----------|----------------|-----|
| PLA | 45–55°C | 4–6 小时 |
| PETG | 60–65°C | 6–8 小时 |
| TPU | 55–60°C | 6–8 小时 |
| PA | 75–85°C | 8–12 小时 |

### 2. 降低喷嘴温度

每次降低 5°C 逐步尝试：
- PLA：尝试 210–215°C（从 220°C 降低）
- PETG：尝试 235–240°C（从 245°C 降低）

:::warning 温度过低会导致层间粘合不良
小心降低温度。温度过低会导致层间粘合差、打印件强度低和挤出问题。
:::

### 3. 调整回抽设置

回抽在"空行程"移动时将耗材从喷嘴中抽回，防止滴料：

```
Bambu Studio → 耗材 → 回抽：
- 回抽距离：0.4–1.0 mm（直驱）
- 回抽速度：30–45 mm/s
```

:::tip 拓竹打印机均使用直驱
所有拓竹打印机（X1C、P1S、A1）均使用直驱挤出机。直驱挤出机需要比鲍登管系统**更短**的回抽距离（通常 0.5–1.5 mm，而鲍登管为 3–7 mm）。
:::

### 4. 提高空行程速度

快速移动可减少喷嘴滴料的时间：
- 将"空行程速度"提高至 200–300 mm/s
- 拓竹打印机可以很好地应对这一速度

### 5. 启用"避免穿越外轮廓"

切片软件设置，使喷嘴绕开拉丝会被看到的开放区域：
```
Bambu Studio → 质量 → 避免穿越外轮廓
```

### 6. 降低速度（适用于 TPU）

TPU 的解决方案与其他材料相反：
- 将打印速度降低至 20–35 mm/s
- TPU 具有弹性，速度过高时会被压缩——从而产生"延迟渗料"

## 调整后的测试

使用标准拉丝测试模型（例如 MakerWorld 上的"torture tower"）进行测试。每次只调整一个变量，观察变化。

:::note 完美几乎不可能
大多数材料都会有一定程度的拉丝，这是正常现象。目标是将其减少到可接受的水平，而非完全消除。PETG 的拉丝通常比 PLA 稍多，这很正常。
:::
