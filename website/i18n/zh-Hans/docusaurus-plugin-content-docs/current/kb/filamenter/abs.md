# ABS

ABS（丙烯腈-丁二烯-苯乙烯）是一种具有良好热稳定性和抗冲击性的热塑性材料。它需要密闭腔室，比 PLA/PETG 更难打印，但能生产坚固耐用的功能性部件。

## 打印参数

| 参数 | 数值 |
|-----------|-------|
| 喷嘴温度 | 240–260°C |
| 热床温度 | 90–110°C |
| 腔室温度 | 45–55°C（X1C/P1S） |
| 零件冷却 | 0–20% |
| 辅助风扇 | 0% |
| 速度 | 80–100% |
| 干燥 | 推荐（70°C / 4–6 小时） |

## 推荐打印平台

| 平台 | 适用性 | 固体胶？ |
|-------|---------|----------|
| 工程板（纹理 PEI） | 极佳 | 是（推荐） |
| 高温板 | 极佳 | 是 |
| 光滑 PEI 板（Cool Plate） | 避免使用 | — |
| 纹理 PEI 板 | 良好 | 是 |

:::tip ABS 使用固体胶
打印 ABS 时工程板上始终使用固体胶。可以改善粘附性，并使打印件更易取出而不损坏平台。
:::

## 密闭腔室

ABS **必须**使用密闭腔室以防止翘边：

- **X1C 和 P1S：** 内置腔室，主动温控——ABS 的理想选择
- **P1P：** 半开放——加装顶盖可获得更好效果
- **A1 / A1 Mini：** 开放式 CoreXY——**不建议**在没有自制外壳的情况下打印 ABS

整个打印过程中保持腔室密闭。不要开门检查打印——等到冷却后再打开，同时也能避免取件时翘边。

## 翘边

ABS 极容易翘边（角落翘起）：

- **提高热床温度** — 尝试 105–110°C
- **使用边缘（Brim）** — 在 Bambu Studio 中设置 5–10 mm 边缘
- **避免气流** — 关闭打印机周围所有气流
- **零件冷却降至 0%** — 冷却会引起变形

:::warning 有害气体
ABS 打印时会释放苯乙烯蒸汽。请确保房间通风良好，或使用 HEPA/活性炭过滤器。拓竹 P1S 内置过滤器。
:::

## 后处理

ABS 比 PETG 和 PLA 更容易打磨、喷漆和粘接。也可以用丙酮蒸汽抛光获得光滑表面——但接触丙酮时请务必格外小心。

## 存储

打印前在 **70°C 烘干 4–6 小时**。存储在密封盒中——ABS 会吸收水分，导致打印时有噼啪声和分层问题。
