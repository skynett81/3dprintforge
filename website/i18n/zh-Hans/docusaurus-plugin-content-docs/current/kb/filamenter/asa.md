# ASA

ASA（丙烯腈-苯乙烯-丙烯酸酯）是ABS的抗紫外线变体，专门为户外使用而开发。该材料结合了ABS的强度和刚性，同时具有显著更好的抗紫外线辐射、抗老化和抗风化能力。

## 设置

| 参数 | 值 |
|-----------|-------|
| 喷嘴温度 | 240–260 °C |
| 热床温度 | 90–110 °C |
| 腔室温度 | 40–50 °C（推荐） |
| 零件冷却 | 30–50% |
| 速度 | 80–100% |
| 需要烘干 | 是 |

## 推荐打印平台

| 平台 | 适用性 | 固体胶? |
|-------|---------|----------|
| Engineering Plate | 优秀 | 否 |
| High Temp Plate | 良好 | 是 |
| Textured PEI | 可接受 | 是 |
| Cool Plate (Smooth PEI) | 不推荐 | — |

:::tip Engineering Plate最适合ASA
Engineering Plate无需固体胶即可为ASA提供最可靠的附着力。该平台可承受高热床温度，并在不使零件永久粘附的情况下提供良好的粘附力。
:::

## 打印机要求

ASA需要**密封腔室（箱体）**才能获得最佳效果。没有箱体会出现：

- **翘曲** — 角落从打印平台翘起
- **分层** — 层间粘结不良
- **表面裂纹** — 打印件上可见的裂纹

| 打印机 | 是否适合ASA？ | 备注 |
|---------|---------------|---------|
| X1C | 优秀 | 全封闭，主动加热 |
| X1E | 优秀 | 全封闭，主动加热 |
| P1S | 良好 | 封闭，被动加热 |
| P1P | 需配件 | 需要箱体配件 |
| A1 | 不推荐 | 开放式框架 |
| A1 Mini | 不推荐 | 开放式框架 |

## ASA vs ABS — 对比

| 特性 | ASA | ABS |
|----------|-----|-----|
| 抗紫外线 | 优秀 | 差 |
| 户外使用 | 是 | 否（发黄变脆） |
| 翘曲 | 中等 | 高 |
| 表面 | 哑光、均匀 | 哑光、均匀 |
| 耐化学性 | 良好 | 良好 |
| 价格 | 稍高 | 较低 |
| 打印时气味 | 中等 | 强烈 |
| 冲击强度 | 良好 | 良好 |
| 耐热温度 | ~95–105 °C | ~95–105 °C |

:::warning 通风
ASA在打印过程中会释放可能具有刺激性的气体。请在通风良好的房间或有空气过滤系统的地方打印。不要在没有通风的房间内长时间停留时打印ASA。
:::

## 烘干

ASA具有**中等吸湿性**，会随时间吸收空气中的水分。

| 参数 | 值 |
|-----------|-------|
| 烘干温度 | 65 °C |
| 烘干时间 | 4–6小时 |
| 吸湿等级 | 中等 |
| 受潮迹象 | 爆裂声、气泡、表面质量差 |

- 开封后存放在含硅胶的密封袋中
- 带干燥剂的AMS足够短期存储
- 长期存储：使用真空袋或耗材烘干箱

## 应用领域

ASA是所有需要在**户外**使用的零件的首选材料：

- **汽车部件** — 后视镜外壳、仪表板细节、气门帽
- **园艺工具** — 夹子、卡扣、花园家具零件
- **户外标牌** — 标志、字母、标识
- **无人机部件** — 起落架、相机支架
- **太阳能板安装件** — 支架和角件
- **邮箱部件** — 机构和装饰件

## 成功打印ASA的技巧

### 边缘和附着

- **大型零件**和接触面积小的零件使用边缘（brim）
- 5–8 mm的边缘可有效防止翘曲
- 较小的零件可以尝试不使用边缘，但要做好备用准备

### 避免气流

- 打印期间**关闭房间所有门窗**
- 气流和冷空气是ASA的最大敌人
- 打印期间不要打开腔室门

### 温度稳定性

- 在打印开始前让腔室预热**10–15分钟**
- 稳定的腔室温度可获得更均匀的结果
- 避免将打印机放在窗户或通风口附近

### 冷却

- ASA需要**有限的零件冷却** — 30–50%是典型值
- 对于悬垂和桥接，可提高到60–70%，但预期会有一些分层
- 机械零件：降低冷却，优先考虑层间粘结而非细节

:::tip 第一次使用ASA？
从一个小型测试件开始（例如30 mm的立方体）来校准您的设置。ASA的表现与ABS非常相似，但翘曲倾向略低。如果您有ABS经验，ASA会感觉是一次升级。
:::

## 收缩

ASA比PLA和PETG收缩更多，但通常比ABS略少：

| 材料 | 收缩率 |
|-----------|----------|
| PLA | ~0.3–0.5% |
| PETG | ~0.3–0.6% |
| ASA | ~0.5–0.7% |
| ABS | ~0.7–0.8% |

对于公差严格的零件：在切片器中补偿0.5–0.7%，或先用测试件验证。

## 后处理

- **丙酮蒸汽抛光** — ASA可以像ABS一样用丙酮蒸汽抛光
- **打磨** — 使用200–400目砂纸可良好打磨
- **粘接** — CA胶或丙酮粘接效果优秀
- **喷漆** — 轻微打磨后可良好上漆

:::danger 丙酮使用注意
丙酮易燃且会释放有毒气体。始终在通风良好的房间使用，避免明火，并佩戴防护装备（手套和护目镜）。
:::
