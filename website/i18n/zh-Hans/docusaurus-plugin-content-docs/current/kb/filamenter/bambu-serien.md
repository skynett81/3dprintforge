# Bambu Lab耗材

Bambu Lab生产各种专门为其打印机优化的耗材。所有Bambu Lab耗材均配备**RFID标签**，打印机可自动检测并设置正确参数。

## RFID和AMS

所有Bambu Lab耗材在线轴中内置**RFID芯片**。这提供：

- **自动识别** — 打印机读取材料类型、颜色和设置
- **剩余量** — 线轴上的估计剩余耗材
- **正确设置** — 温度、速度和冷却自动设置
- **AMS兼容性** — AMS中的无缝材料切换

:::tip 第三方耗材在AMS中使用
AMS也可以使用第三方耗材，但您需要在Bambu Studio中手动设置参数。RFID自动检测是Bambu Lab耗材的专属功能。
:::

## 质量控制和颜色一致性

Bambu Lab对其耗材实施严格的质量控制：

- **直径公差** — ±0.02 mm（行业领先）
- **颜色一致性** — 批次控制确保卷轴间颜色一致
- **卷轴质量** — 均匀缠绕，无打结或重叠
- **真空密封** — 每个卷轴均带干燥剂真空包装
- **温度曲线测试** — 每批次均在最佳温度下测试

:::tip 颜色编号确保一致性
Bambu Lab使用带批次控制的颜色编号（如"Bambu PLA Matte Charcoal"）。如果大型项目需要多个卷轴的相同颜色，请从同一批次订购或联系客服进行批次匹配。
:::
