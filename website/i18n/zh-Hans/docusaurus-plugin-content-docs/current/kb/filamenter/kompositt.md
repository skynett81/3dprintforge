# 复合材料（CF/GF）

复合耗材含有短碳纤维（CF）或玻璃纤维（GF），混合在 PLA、PETG、PA 或 ABS 等基础塑料中。它们具有更高的刚性、更轻的重量和更好的尺寸稳定性。

## 可用类型

| 耗材 | 基础材料 | 刚性 | 减重效果 | 难度 |
|----------|-------|---------|--------------|------------------|
| PLA-CF | PLA | 高 | 中等 | 简单 |
| PETG-CF | PETG | 高 | 中等 | 中等 |
| PA6-CF | 尼龙 6 | 极高 | 良好 | 较难 |
| PA12-CF | 尼龙 12 | 极高 | 良好 | 中等 |
| ABS-CF | ABS | 高 | 中等 | 中等 |
| PLA-GF | PLA | 高 | 中等 | 简单 |

## 必须使用硬化钢喷嘴

:::danger 切勿将黄铜喷嘴用于 CF/GF 材料
碳纤维和玻璃纤维具有极强的磨蚀性。它们会在数小时到数天内磨损标准黄铜喷嘴。所有 CF 和 GF 材料必须使用**硬化钢喷嘴**或 **HS01 喷嘴**。

- 拓竹硬化钢喷嘴（0.4 mm）
- 拓竹 HS01 喷嘴（特殊涂层，寿命更长）
:::

## 打印参数（PA-CF 示例）

| 参数 | 数值 |
|-----------|-------|
| 喷嘴温度 | 270–290°C |
| 热床温度 | 80–100°C |
| 零件冷却 | 0–20% |
| 速度 | 80% |
| 干燥 | 80°C / 12 小时 |

PLA-CF：喷嘴 220–230°C，热床 35–50°C——比 PA-CF 容易得多。

## 打印平台

| 平台 | 适用性 | 固体胶？ |
|-------|---------|----------|
| 工程板（纹理 PEI） | 极佳 | 是（PA 基材） |
| 高温板 | 良好 | 是 |
| 光滑 PEI 板 | 避免（CF 会划伤） | — |
| 纹理 PEI 板 | 良好 | 是 |

:::warning 平台可能被划伤
CF 材料在取件时可能划伤光滑平台。始终使用工程板或纹理 PEI 板。不要拽下打印件——轻轻弯曲平台。
:::

## 表面处理

CF 耗材呈现哑面碳纤维质感，无需喷漆。表面略有孔隙，可用环氧树脂浸渍以获得更光滑的效果。

## 磨损与喷嘴寿命

| 喷嘴类型 | CF 材料下的寿命 | 成本 |
|----------|---------------|---------|
| 黄铜（标准） | 数小时至数天 | 低 |
| 硬化钢 | 200–500 小时 | 中等 |
| HS01（拓竹） | 500–1000 小时 | 较高 |

喷嘴出现明显磨损时更换：喷嘴孔径扩大、壁厚变薄、尺寸精度下降。

## 干燥

CF 变体的 PA 和 PETG 与其基础材料一样需要干燥：
- **PLA-CF：** 推荐干燥，但不关键
- **PETG-CF：** 65°C / 6–8 小时
- **PA-CF：** 80°C / 12 小时——至关重要
