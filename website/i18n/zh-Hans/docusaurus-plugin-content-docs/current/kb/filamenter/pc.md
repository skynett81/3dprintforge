# PC（聚碳酸酯）

聚碳酸酯是可用于FDM打印的最强热塑性材料之一。它结合了极高的冲击强度、110–130 °C的耐热性以及天然透明性。PC是一种打印要求较高的材料，但能提供接近注塑件品质的结果。

## 设置

| 参数 | 纯PC | PC-ABS混合 | PC-CF |
|-----------|--------|-------------|-------|
| 喷嘴温度 | 260–280 °C | 250–270 °C | 270–290 °C |
| 热床温度 | 100–120 °C | 90–110 °C | 100–120 °C |
| 腔室温度 | 50–60 °C（必需） | 45–55 °C | 50–60 °C |
| 零件冷却 | 0–20% | 20–30% | 0–20% |
| 速度 | 60–80% | 70–90% | 50–70% |
| 需要烘干 | 是（关键） | 是 | 是（关键） |

## 推荐打印平台

| 平台 | 适用性 | 固体胶? |
|-------|---------|----------|
| High Temp Plate | 优秀（必需） | 否 |
| Engineering Plate | 可接受 | 是 |
| Textured PEI | 不推荐 | — |
| Cool Plate (Smooth PEI) | 不可使用 | — |

:::danger High Temp Plate为必需
PC需要100–120 °C的热床温度。Cool Plate和Textured PEI无法承受这些温度，会被损坏。纯PC**必须**使用High Temp Plate。
:::

## 打印机和设备要求

### 密封箱体（必需）

PC需要温度稳定在50–60 °C的**完全密封腔室**。没有它，会出现严重翘曲、分层和脱层。

### 硬化喷嘴（强烈推荐）

纯PC没有磨蚀性，但PC-CF和PC-GF**需要硬化钢喷嘴**（如Bambu Lab HS01）。纯PC由于高温也推荐使用硬化喷嘴。

### 打印机兼容性

| 打印机 | 是否适合PC？ | 备注 |
|---------|--------------|---------|
| X1C | 优秀 | 全封闭，HS01可用 |
| X1E | 优秀 | 专为工程材料设计 |
| P1S | 有限 | 封闭但无主动腔室加热 |
| P1P | 不推荐 | 无箱体 |
| A1 / A1 Mini | 不可使用 | 开放式框架，温度不足 |

:::warning 仅推荐X1C和X1E
PC需要主动腔室加热以获得一致的结果。P1S可能在小零件上给出可接受的结果，但较大零件预计会出现翘曲和分层。
:::

## 烘干

PC**非常容易吸湿**，会快速吸收水分。潮湿的PC会导致灾难性的打印结果。

| 参数 | 值 |
|-----------|-------|
| 烘干温度 | 70–80 °C |
| 烘干时间 | 6–8小时 |
| 吸湿等级 | 高 |
| 最大推荐湿度 | < 0.02% |

- 打印前**务必**烘干PC — 即使新开封的卷轴也可能已吸收水分
- 如有可能，从烘干箱直接打印
- AMS**不足以**存储PC — 湿度过高
- 使用带主动加热的专用耗材烘干机

:::danger 水分会毁坏PC打印件
潮湿PC的迹象：强烈的爆裂声、表面气泡、极差的层间粘合、拉丝。潮湿的PC无法通过调整设置来补偿 — **必须**先烘干。
:::

## PC混合物

### PC-ABS

聚碳酸酯与ABS的混合物，结合了两种材料的强度：

- **比纯PC更容易打印** — 更低的温度和更少的翘曲
- **冲击强度**介于ABS和PC之间
- **工业中流行** — 用于汽车内饰和电子设备外壳
- 喷嘴250–270 °C，热床90–110 °C打印

### PC-CF（碳纤维）

碳纤维增强PC，用于最大刚度和强度：

- **极其刚硬** — 非常适合结构零件
- **轻量** — 碳纤维减轻重量
- **需要硬化喷嘴** — 黄铜在数小时内磨损
- 喷嘴270–290 °C，热床100–120 °C打印
- 比纯PC更贵，但机械性能接近铝

### PC-GF（玻璃纤维）

玻璃纤维增强PC：

- **比PC-CF便宜**，刚度良好
- 比PC-CF**更白的表面**
- **需要硬化喷嘴** — 玻璃纤维磨蚀性非常强
- 刚度比PC-CF略低，但冲击强度更好

## 应用领域

PC用于需要**最大强度和/或耐热性**的场合：

- **机械零件** — 齿轮、紧固件、承载联轴器
- **光学零件** — 透镜、导光体、透明盖板（透明PC）
- **耐热零件** — 发动机舱、加热元件附近
- **电子设备外壳** — 具有良好冲击强度的保护壳
- **工具和夹具** — 精密装配工具

## 成功打印PC的技巧

### 第一层

- 将第一层速度降至**30–40%**
- 前3–5层将热床温度提高至标准以上5 °C
- **大多数PC零件必须使用边缘（brim）** — 使用8–10 mm

### 腔室温度

- 在打印开始前让腔室达到**50 °C+**
- **打印期间不要打开腔室门** — 温度下降会立即导致翘曲
- 打印后：让零件在腔室中**缓慢**冷却（1–2小时）

### 冷却

- 使用**最少的零件冷却**（0–20%）以获得最佳层间粘合
- 对于桥接和悬垂：临时增加到30–40%
- PC打印中优先考虑层间强度而非美观

### 设计注意事项

- **避免锐角** — 以最小1 mm半径倒圆
- **均匀壁厚** — 不均匀的厚度产生内应力
- **大面积平面**较困难 — 分割或添加加强筋

:::tip PC新手？从PC-ABS开始
如果您从未打印过PC，请从PC-ABS混合物开始。它比纯PC宽容得多，让您在没有极端要求的情况下获得材料经验。掌握PC-ABS后，再进阶到纯PC。
:::

## 后处理

- **打磨** — PC可以很好地打磨，但透明PC请使用湿磨
- **抛光** — 透明PC可以抛光到接近光学品质
- **粘接** — 二氯甲烷粘接可形成不可见的接缝（使用防护装备！）
- **喷漆** — 需要底漆以获得良好附着力
- **退火** — 120 °C处理1–2小时可减少内应力

:::warning 二氯甲烷粘接
二氯甲烷有毒，需要通风设备、耐化学品手套和护目镜。始终在通风良好的房间或通风柜中操作。
:::
