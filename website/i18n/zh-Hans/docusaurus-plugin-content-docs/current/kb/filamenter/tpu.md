# TPU

TPU（热塑性聚氨酯）是一种柔性材料，用于保护壳、密封圈、轮子和其他需要弹性的部件。

## 打印参数

| 参数 | 数值 |
|-----------|-------|
| 喷嘴温度 | 220–240°C |
| 热床温度 | 30–45°C |
| 零件冷却 | 50–80% |
| 速度 | 30–50%（重要） |
| 回抽 | 最小或禁用 |
| 干燥 | 推荐（60°C / 6–8 小时） |

:::danger 低速度至关重要
TPU 必须低速打印。速度过高会导致材料在挤出机中压缩并堵塞。从 30% 速度开始，谨慎地逐步提高。
:::

## 推荐打印平台

| 平台 | 适用性 | 固体胶？ |
|-------|---------|----------|
| 纹理 PEI 板 | 极佳 | 否 |
| 光滑 PEI 板（Cool Plate） | 良好 | 否 |
| 工程板 | 良好 | 否 |

## 回抽设置

TPU 具有弹性，对激进的回抽反应不佳：

- **直驱（X1C/P1S/A1）：** 回抽 0.5–1.0 mm，25 mm/s
- **鲍登管（避免用于 TPU）：** 极为困难，不推荐

对于非常软的 TPU（Shore A 85 或以下）：完全禁用回抽，依靠温度和速度控制。

## 技巧

- **干燥耗材** — 受潮的 TPU 极难打印
- **使用直驱挤出机** — 拓竹 P1S/X1C/A1 全部使用直驱
- **避免过高温度** — 超过 250°C 时 TPU 会降解，导致打印件变色
- **拉丝倾向** — TPU 容易拉丝；降低 5°C 温度或增加冷却

:::tip Shore 硬度
TPU 有多种 Shore 硬度（A85、A95、A98）。Shore A 越低，材料越软，打印难度越高。拓竹 TPU 为 Shore A 95——适合入门的好选择。
:::

## 存储

TPU 极易吸湿。受潮的 TPU 会导致：
- 气泡和嘶嘶声
- 打印件软弱易脆（对柔性材料来说很矛盾）
- 拉丝

打印前**始终在 60°C 烘干 6–8 小时**。存放在带硅胶干燥剂的密封盒中。
