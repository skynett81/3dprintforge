:::danger 碳纤维和玻璃纤维需要硬化喷嘴
所有含**-CF**（碳纤维）或**-GF**（玻璃纤维）的材料**需要硬化钢喷嘴**。黄铜在这些材料下数小时至数天内磨损。推荐使用Bambu Lab HS01。

需要硬化喷嘴的材料：
- PLA-CF
- PETG-CF
- PA-CF / PA-GF
- PC-CF / PC-GF
:::

:::tip 普通材料的黄铜与硬化钢对比
黄铜喷嘴为普通材料（PLA、PETG、ABS）提供**更好的热传导性**，因此挤出更均匀。硬化钢也能良好工作，但可能需要高5–10 °C的温度。日常使用黄铜，CF/GF材料时换用硬化钢。
:::

## 换料技巧

在AMS中或手动切换材料时，正确的清洗对于避免污染很重要。

### 推荐清洗量

| 切换 | 清洗量 | 备注 |
|-----------|-------------|---------|
| PLA → PLA (换色) | 100–150 mm³ | 标准换色 |
| PLA → PETG | 200–300 mm³ | 温度升高，流量不同 |
| PETG → PLA | 200–300 mm³ | 温度降低 |
| ABS → PLA | 300–400 mm³ | 温差大 |
| PLA → ABS | 300–400 mm³ | 温差大 |
| PA → PLA | 400–500 mm³ | 尼龙残留在热端 |
| PC → PLA | 400–500 mm³ | PC需要彻底清洗 |
| 深色 → 浅色 | 200–300 mm³ | 深色颜料难以清除 |
| 浅色 → 深色 | 100–150 mm³ | 较容易过渡 |

### 换料时的温度变化

| 过渡 | 建议 |
|----------|-----------|
| 低温 → 高温（如PLA → ABS） | 加热至新材料温度，彻底清洗 |
| 高温 → 低温（如ABS → PLA） | 先在高温下清洗，然后降温 |
| 相似温度（如PLA → PLA） | 标准清洗 |
| 大温差（如PLA → PC） | 用PETG作为中间过渡可能有帮助 |

:::warning 尼龙和PC会留下残留物
PA（尼龙）和PC特别难以清洗。使用这些材料后：
1. 用**PETG**或**ABS**在高温（260–280 °C）下清洗
2. 至少使用**500 mm³**清洗材料
3. 目视检查挤出 — 应完全干净无变色
:::

不确定需要哪种材料？使用此指南：

| 需求 | 推荐材料 |
