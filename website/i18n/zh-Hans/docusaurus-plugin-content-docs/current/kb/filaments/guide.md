## 各材料小贴士

### 标准 (PLA, PETG)
- PLA是最简单的材料 — 非常适合原型和装饰
- PETG提供更好的强度和耐热性，但拉丝可能更多
- 两者都无需密封仓，可使用标准黄铜喷嘴

### 工程 (ABS, ASA, PC)
- 需要密封打印机以避免翘曲
- ASA耐紫外线 — 用于户外部件
- PC提供最高强度和耐热性，但最难打印

### 复合材料 (CF/GF)
- **始终**使用硬化钢喷嘴 — 碳纤维会快速磨损黄铜喷嘴
- CF变体生产更硬更轻的哑光表面零件
- GF变体更便宜但表面更粗糙

### 柔性 (TPU)
- 缓慢打印（50 mm/s或以下）以获得最佳效果
- 减少回抽以避免堵塞
- 直驱挤出机比鲍登式效果好得多

### 支撑 (PVA, HIPS)
- PVA溶于水 — 非常适合PLA支撑
- HIPS溶于柠檬烯 — 与ABS配合使用
- 两者都对湿度非常敏感 — 保持干燥存放

:::warning 尼龙材料
PA、PA-CF和PA-GF**极度**吸湿。它们在几分钟内就会吸收空气中的水分。使用前务必干燥12小时以上，如有可能直接从干燥箱中打印。
:::
