# PETG

PETG（聚对苯二甲酸乙二醇酯-改性）是功能性部件的热门材料。比 PLA 更强韧、耐热性更好，且能承受轻度化学品接触。

## 打印参数

| 参数 | 数值 |
|-----------|-------|
| 喷嘴温度 | 230–250°C |
| 热床温度 | 70–85°C |
| 零件冷却 | 30–60% |
| 速度 | 标准 |
| 干燥 | 推荐（65°C / 6–8 小时） |

## 推荐打印平台

| 平台 | 适用性 | 固体胶？ |
|-------|---------|----------|
| 工程板（纹理 PEI） | 极佳 | 否/是* |
| 纹理 PEI 板 | 良好 | 是** |
| 光滑 PEI 板（Cool Plate） | 见警告 | 见警告 |
| 高温板 | 良好 | 是 |

:::danger 重要：光滑 PEI 板上的 PETG 必须使用固体胶
PETG 在光滑 PEI（Cool Plate）上的粘附力**极强**。不涂固体胶时，取出打印件可能会**撕毁平台涂层**。在光滑 PEI 上打印 PETG 时，务必涂薄薄一层固体胶作为隔离层。

**替代方案：** 使用工程板或纹理 PEI 板——这些平台提供良好粘附性而不会损坏平台。
:::

## 成功打印技巧

- **降低零件冷却** — 冷却过度会导致层间分离和打印件脆裂
- **调整喷嘴温度** — 拉丝严重时降低 5–10°C；层间粘合差时升高
- **首层热床温度** — 80–85°C 以获得良好粘附，首层后降至 70°C
- **降低速度** — PETG 比 PLA 更难打印，从 80% 速度开始

:::warning 拉丝问题
PETG 容易拉丝。增加回抽距离（直驱尝试 0.8–1.5 mm），提高回抽速度，每次降低 5°C 喷嘴温度。
:::

## 干燥

PETG 比 PLA 吸湿更快。受潮的 PETG 会导致：
- 打印时冒泡和嘶嘶声
- 层间较弱，表面有孔隙
- 拉丝加重

线卷开封时间较长时，打印前在 **65°C 烘干 6–8 小时**。

## 存储

始终存放在带硅胶干燥剂的密封袋或干燥盒中。PETG 在潮湿环境中不应暴露超过几天。
