# PLA

PLA（聚乳酸）是最适合新手的耗材。它易于打印，表面效果好，无需密闭腔室或特殊预热处理。

## 打印参数

| 参数 | 标准 PLA | PLA+ | PLA Silk |
|-----------|-------------|------|---------|
| 喷嘴温度 | 220°C | 230°C | 230°C |
| 热床温度 | 35–45°C | 45–55°C | 45–55°C |
| 腔室温度 | — | — | — |
| 零件冷却 | 100% | 100% | 80% |
| 速度 | 标准 | 标准 | 80% |
| 是否需要干燥 | 否 | 否 | 否 |

## 推荐打印平台

| 平台 | 适用性 | 固体胶？ |
|-------|---------|----------|
| 光滑 PEI 板（Cool Plate） | 极佳 | 否 |
| 纹理 PEI 板 | 良好 | 否 |
| 工程板 | 良好 | 否 |
| 高温板 | 避免使用 | — |

## 成功打印技巧

- **无需固体胶** — PLA 在大多数平台上无需固体胶即可良好粘附
- **等待平台冷却** — 冷却到室温后 PLA 更容易脱离
- **首层速度** — 设为 50–70% 以获得更好粘附
- **零件冷却** — 保持在 100% 以获得更清晰的细节和更好的桥接效果

:::tip Z 偏移
仔细校准首层 Z 偏移。PLA 在 Cool Plate 上：实时调整到首层略显透明、贴合良好，但不要被压扁。
:::

## 变体

### PLA+
比标准 PLA 更强韧、更耐热。打印温度略高（225–235°C）。更灵活，更易后处理。

### PLA Silk
呈现亮面金属质感。需要较低冷却和略低速度以获得最佳效果。桥接难度更高。

### PLA-CF（碳纤维）
碳纤维增强 PLA 刚性更高、重量更轻。必须使用**硬化钢喷嘴**——切勿将黄铜喷嘴用于任何 CF 材料。

### PLA 哑光
无光泽哑面效果。打印参数与标准 PLA 相同。

## 存储

PLA 吸湿速度不如 PETG 和 PA 快，但仍应保持干燥存储：

- **推荐：** 密封袋加硅胶干燥剂
- **受潮迹象：** 打印时噼啪声、表面气泡、打印件强度差

如有必要，在 **45–55°C 烘干 4–6 小时**。
