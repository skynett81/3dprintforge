# PVA和支撑材料

支撑材料用于打印具有悬垂、桥接和内部空腔的复杂几何形状，这些形状没有临时支撑就无法打印。打印后，支撑材料通过机械方式或溶剂溶解来移除。

## 概览

| 材料 | 溶剂 | 搭配使用 | 溶解时间 | 难度 |
|-----------|-----------|-------------|----------------|-------------------|
| PVA | 水 | PLA, PETG | 12–24小时 | 困难 |
| HIPS | d-柠檬烯 | ABS, ASA | 12–24小时 | 中等 |
| PVB | 异丙醇 (IPA) | PLA, PETG | 6–12小时 | 中等 |
| BVOH | 水 | PLA, PETG, PA | 4–8小时 | 困难 |

---

## PVA（聚乙烯醇）

PVA是一种水溶性支撑材料，是具有复杂支撑结构的PLA打印中最常用的选择。

### 设置

| 参数 | 值 |
|-----------|-------|
| 喷嘴温度 | 190–210 °C |
| 热床温度 | 45–60 °C |
| 零件冷却 | 100% |
| 速度 | 60–80% |
| 回抽 | 增加（6–8 mm） |

### 推荐打印平台

| 平台 | 适用性 | 固体胶? |
|-------|---------|----------|
| Cool Plate (Smooth PEI) | 优秀 | 否 |
| Textured PEI | 良好 | 否 |
| Engineering Plate | 良好 | 否 |
| High Temp Plate | 避免 | — |

### 兼容性

PVA与在**相似温度**下打印的材料配合最佳：

| 主体材料 | 兼容性 | 备注 |
|---------------|---------------|---------|
| PLA | 优秀 | 理想组合 |
| PETG | 良好 | 热床温度对PVA可能稍高 |
| ABS/ASA | 差 | 腔室温度过高 — PVA降解 |
| PA（尼龙） | 差 | 温度过高 |

### 溶解

- 将完成的打印件放入**温水**（约40 °C）中
- PVA根据厚度在**12–24小时**内溶解
- 定期搅拌水以加速过程
- 每6–8小时换水以加快溶解
- 超声波清洗器可显著加快速度（2–6小时）

:::danger PVA极易吸湿
PVA从空气中吸收水分**非常快** — 即使几小时的暴露也可能毁坏打印结果。吸收了水分的PVA会导致：

- 严重的冒泡和爆裂声
- 与主体材料的粘附不良
- 拉丝和粘腻表面
- 喷嘴堵塞

**使用前务必立即烘干PVA**，并从干燥环境（烘干箱）中打印。
:::

### PVA烘干

| 参数 | 值 |
|-----------|-------|
| 烘干温度 | 45–55 °C |
| 烘干时间 | 6–10小时 |
| 吸湿等级 | 极高 |
| 存储方法 | 始终使用带干燥剂的密封盒 |

## HIPS（高抗冲聚苯乙烯）

HIPS是一种溶解于d-柠檬烯（柑橘类溶剂）的支撑材料。它是ABS和ASA的首选支撑材料。

### 设置

| 参数 | 值 |
|-----------|-------|
| 喷嘴温度 | 220–240 °C |
| 热床温度 | 90–100 °C |
| 腔室温度 | 40–50 °C（推荐） |
| 零件冷却 | 20–40% |
| 速度 | 70–90% |

### 兼容性

| 主体材料 | 兼容性 | 备注 |
|---------------|---------------|---------|
| ABS | 优秀 | 理想组合 — 相似温度 |
| ASA | 优秀 | 粘附力非常好 |
| PLA | 差 | 温差过大 |
| PETG | 差 | 热学行为不同 |

### d-柠檬烯溶解

- 将打印件放入**d-柠檬烯**（柑橘类溶剂）中
- 溶解时间：室温下**12–24小时**
- 加热至35–40 °C可加速过程
- d-柠檬烯可重复使用2–3次
- 溶解后用水冲洗零件并晾干

### 相比PVA的优势

- **对湿度的敏感度低得多** — 更容易存储和处理
- **作为支撑材料更强** — 承受更多而不分解
- **与ABS/ASA更好的热兼容性**
- **更容易打印** — 更少堵塞和问题

:::warning d-柠檬烯是溶剂
请戴手套并在通风良好的房间工作。d-柠檬烯可能刺激皮肤和粘膜。请存放在儿童接触不到的地方。
:::

## PVB（聚乙烯醇缩丁醛）

PVB是一种独特的支撑材料，可溶于异丙醇（IPA），还可用于通过IPA蒸汽平滑表面。

### 设置

| 参数 | 值 |
|-----------|-------|
| 喷嘴温度 | 200–220 °C |
| 热床温度 | 55–75 °C |
| 零件冷却 | 80–100% |
| 速度 | 70–80% |

### 兼容性

| 主体材料 | 兼容性 | 备注 |
|---------------|---------------|---------|
| PLA | 良好 | 可接受的粘附力 |
| PETG | 中等 | 热床温度可能有变化 |
| ABS/ASA | 差 | 温度过高 |

### IPA蒸汽表面平滑

PVB的独特特性是可以用IPA蒸汽平滑表面：

1. 将零件放入密封容器中
2. 在底部放置浸有IPA的布（不要与零件直接接触）
3. 让蒸汽作用**30–60分钟**
4. 取出，晾干24小时
5. 结果是光滑的半光泽表面

:::tip PVB作为表面处理
虽然PVB主要是支撑材料，但也可以作为PLA零件的最外层打印，以获得可用IPA蒸汽平滑的表面。这提供了类似于丙酮平滑ABS的效果。
:::

## 支撑材料对比

| 特性 | PVA | HIPS | PVB | BVOH |
|----------|-----|------|-----|------|
| 溶剂 | 水 | d-柠檬烯 | IPA | 水 |
| 溶解时间 | 12–24小时 | 12–24小时 | 6–12小时 | 4–8小时 |
| 湿度敏感性 | 极高 | 低 | 中等 | 极高 |
| 难度 | 困难 | 中等 | 中等 | 困难 |
| 价格 | 高 | 中等 | 高 | 非常高 |
| 最佳搭配 | PLA, PETG | ABS, ASA | PLA | PLA, PETG, PA |
| 可获取性 | 良好 | 良好 | 有限 | 有限 |
| AMS兼容 | 是（带干燥剂） | 是 | 是 | 有问题 |

## 双挤出和多色打印技巧

### 一般准则

- **清洗量** — 支撑材料在换料时需要充分清洗（至少150–200 mm³）
- **界面层** — 在支撑和主体零件之间使用2–3个界面层以获得清洁表面
- **距离** — 将支撑距离设置为0.1–0.15 mm，以便溶解后轻松移除
- **支撑图案** — PVA/BVOH使用三角形图案，HIPS使用网格

### AMS设置

- 将支撑材料放入**带干燥剂的AMS槽位**
- 对于PVA：考虑使用带鲍登连接的外部烘干箱
- 在Bambu Studio中配置正确的材料配置文件
- 在打印复杂零件前用简单的悬垂模型测试

### 常见问题和解决方案

| 问题 | 原因 | 解决方案 |
|---------|-------|---------|
| 支撑不粘附 | 距离过大 | 将界面距离减小到0.05 mm |
| 支撑粘附过牢 | 距离过小 | 将界面距离增加到0.2 mm |
| 支撑材料中有气泡 | 湿气 | 彻底烘干耗材 |
| 材料间拉丝 | 回抽不足 | 增加1–2 mm回抽 |
| 支撑面表面质量差 | 界面层不足 | 增加到3–4个界面层 |

:::tip 从简单开始
首次使用支撑材料打印：使用PLA + PVA、一个有明显悬垂（45°以上）的简单模型，以及Bambu Studio的默认设置。随着经验积累逐步优化。
:::
