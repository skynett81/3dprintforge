# AMS 维护

AMS（自动换料系统）是一个精密系统，需要定期维护才能可靠运行。最常见的问题是耗材通道脏污和机箱内湿度过高。

## PTFE 管

PTFE 管将耗材从 AMS 输送到打印机，是最先磨损的部件之一。

### 检查
检查 PTFE 管是否有：
- **折痕或弯折** — 阻碍耗材流动
- **接头处磨损** — 入口处有白色粉末
- **形变** — 尤其是使用 CF 材料后

### 更换 PTFE 管
1. 从 AMS 卸下耗材（运行卸料循环）
2. 按下管接头处蓝色锁紧环
3. 拔出管子（需要用力）
4. 将新管剪至适当长度（不短于原管）
5. 推入直到到底，锁紧

:::tip AMS Lite 与 AMS
AMS Lite（A1/A1 Mini）的 PTFE 配置比完整 AMS（P1S/X1C）简单。管子更短，更换更方便。
:::

## 耗材通道

### 清洁耗材路径
耗材会在通道中留下粉末和残留物，CF 材料尤为严重：

1. 卸载所有槽位的耗材
2. 用气吹或软刷清除松散粉末
3. 通过通道运行一段干净的尼龙或 PTFE 清洁耗材

### 传感器
AMS 使用传感器检测耗材位置和断料情况。保持传感器窗口清洁：
- 用干净的软刷轻轻擦拭传感器镜头
- 避免将 IPA 直接滴在传感器上

## 防潮

AMS 不能保护耗材免受湿度影响。对于吸湿性材料（PA、PETG、TPU）建议：

### 干燥 AMS 替代方案
- **密封盒：** 将线卷放在装有硅胶干燥剂的密封盒中
- **拓竹干燥盒：** 官方干燥盒配件
- **外部送料器：** 敏感材料绕过 AMS 直接从外部送料

### 湿度监测
在 AMS 机箱内放置湿度指示卡（湿度计）。相对湿度超过 30% 时更换硅胶干燥剂包。

## 驱动齿轮和夹持机构

### 检查
检查驱动齿轮（AMS 内的挤出轮）是否有：
- 齿间有耗材残留
- 齿轮磨损
- 手动拉扯时摩擦力不均匀

### 清洁
1. 用牙刷或刷子清除驱动齿轮齿间的残留物
2. 用气吹
3. 避免使用油脂和润滑剂——摩擦力是针对干燥运行校准的

## 维护周期

| 工作内容 | 间隔 |
|-----------|---------|
| 目视检查 PTFE 管 | 每月 |
| 清洁耗材通道 | 每 100 小时 |
| 检查传感器 | 每月 |
| 更换干燥剂（干燥配置）| 按需（湿度 30%+ 时） |
| 更换 PTFE 管 | 出现明显磨损时 |
