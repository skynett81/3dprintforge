# 粘附不良

粘附不良是 3D 打印中最常见的问题之一。首层不粘，或打印中途脱落。

## 症状

- 首层不粘——打印件移动或翘起
- 边缘和角落翘起（翘边）
- 打印件在打印过程中脱落
- 首层不均匀，有空洞或松散细丝

## 排查清单——按顺序尝试

### 1. 清洁平台
粘附不良最常见的原因是平台上有油脂或污垢。

```
1. 用 IPA（异丙醇）擦拭平台
2. 避免用裸手触摸打印表面
3. 持续有问题时：用温水和温和洗洁精清洗
```

### 2. 校准 Z 偏移

Z 偏移是首层打印时喷嘴与平台之间的高度。偏高 = 细丝悬空。偏低 = 喷嘴刮平台。

**正确的 Z 偏移：**
- 首层应略显透明
- 细丝应轻轻压入平台，有一点"压扁"效果
- 相邻细丝应轻微融合在一起

在打印过程中通过**控制 → 实时调整 Z**来调整 Z 偏移。

:::tip 打印时实时调整
3DPrintForge 在打印过程中显示 Z 偏移调整按钮。查看首层的同时，以 ±0.02 mm 为步长进行调整。
:::

### 3. 检查热床温度

| 材料 | 温度过低 | 推荐温度 |
|-----------|-------------|---------|
| PLA | 低于 30°C | 35–45°C |
| PETG | 低于 60°C | 70–85°C |
| ABS | 低于 80°C | 90–110°C |
| TPU | 低于 25°C | 30–45°C |

每次尝试升高 5°C。

### 4. 使用固体胶

固体胶可改善大多数材料在大多数平台上的粘附性：
- 涂薄薄一层均匀的固体胶
- 等待 30 秒晾干后再开始打印
- 对以下材料尤为重要：ABS、PA、PC、PETG（在光滑 PEI 上）

### 5. 降低首层速度

首层速度越低，耗材与平台的接触越好：
- 标准速度：首层 50 mm/s
- 尝试：30–40 mm/s
- Bambu Studio：**质量 → 首层速度** 中设置

### 6. 检查平台状态

即使设置完美，磨损的平台也会粘附不良。出现以下情况时更换平台：
- PEI 涂层有明显损坏
- 清洁后仍无改善

### 7. 使用边缘（Brim）

对于容易翘边的材料（ABS、PA、大型平底物体）：
- 在切片软件中添加边缘（Brim）：宽度 5–10 mm
- 增大接触面积，防止边缘翘起

## 特殊情况

### 大型平底物体
大型平底物体最容易脱落。解决措施：
- 边缘宽度 8–10 mm
- 提高热床温度
- 关闭腔室（ABS/PA）
- 降低零件冷却

### 平台表面上釉
长期使用过多固体胶可能导致平台表面上釉。用水彻底清洗，重新开始。

### 更换材料后
不同材料需要不同设置。确认热床温度和平台类型已针对新材料进行配置。
