# 翘边（Warping）

翘边是指打印件的角落或边缘在打印过程中或打印后从平台上翘起。这是由材料热收缩引起的。

## 什么是翘边？

塑料冷却时会收缩。上层比下层温度高——这会产生将边缘向上拉的应力，使打印件弯曲。温差越大，翘边越严重。

## 最容易翘边的材料

| 材料 | 翘边风险 | 需要密闭腔室 |
|-----------|-------------|-----------------|
| PLA | 低 | 否 |
| PETG | 低–中 | 否 |
| ABS | 高 | 是 |
| ASA | 高 | 是 |
| PA/尼龙 | 非常高 | 是 |
| PC | 非常高 | 是 |
| TPU | 低 | 否 |

## 解决方案

### 1. 使用密闭腔室

对于 ABS、ASA、PA 和 PC，这是最重要的措施：
- 腔室温度保持 40–55°C 效果最佳
- X1C 和 P1S：在"密闭"模式下启用腔室风扇
- A1/P1P：使用遮盖罩保持热量

### 2. 使用边缘（Brim）

边缘是额外的单层宽边，将打印件固定在平台上：

```
Bambu Studio：
1. 在切片软件中选择打印件
2. 进入"支撑" → 边缘（Brim）
3. 宽度设为 5–10 mm（翘边越严重，边缘越宽）
4. 类型：仅外边缘（推荐）
```

:::tip 边缘宽度参考
- PLA（很少需要）：3–5 mm
- PETG：4–6 mm
- ABS/ASA：6–10 mm
- PA/尼龙：8–15 mm
:::

### 3. 提高热床温度

更高的热床温度可减少层间温差：
- ABS：尝试 105–110°C
- PA：85–95°C
- PETG：80–85°C

### 4. 降低零件冷却

对于容易翘边的材料——降低或关闭零件冷却：
- ABS/ASA：0–20% 零件冷却
- PA：0–30% 零件冷却

### 5. 避免气流和冷空气

让打印机远离：
- 窗户和外门
- 空调和风扇
- 室内气流

对于 P1P 和 A1：在关键打印时用硬纸板遮挡开口。

### 6. 防风罩（Draft Shield）

防风罩是环绕物体的薄壁，用于保持内部热量：

```
Bambu Studio：
1. 进入"支撑" → 防风罩
2. 启用并设置间距（3–5 mm）
```

对于高而细的物体特别有效。

### 7. 模型设计优化

设计自己的模型时：
- 避免大型平底（在角落添加倒角/圆角）
- 将大型平面部件分割成较小部分
- 在切片软件或 CAD 中使用"鼠耳"——在角落添加小圆形辅助粘附点

## 冷却后翘边

有时打印件看起来正常，但取下后才发生翘边：
- 取出前始终等到平台和打印件**完全冷却**（低于 40°C）
- 对于 ABS：在密闭腔室内慢慢冷却
- 避免将热的打印件放在冷的表面上
