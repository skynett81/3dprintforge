# 喷嘴维护

喷嘴是打印机最关键的部件之一。正确维护可延长使用寿命，确保良好的打印效果。

## 喷嘴类型

| 喷嘴类型 | 适用材料 | 预计寿命 | 最高温度 |
|----------|-----------|-------------------|----------|
| 黄铜（标准） | PLA、PETG、ABS、TPU | 200–500 小时 | 300°C |
| 硬化钢 | 所有材料，含 CF/GF | 300–600 小时 | 300°C |
| HS01（拓竹） | 所有材料，含 CF/GF | 500–1000 小时 | 300°C |

:::danger 切勿将黄铜喷嘴用于 CF/GF 材料
碳纤维和玻璃纤维填充耗材会在数小时内磨损黄铜喷嘴。打印 CF/GF 材料前请更换为硬化钢喷嘴。
:::

## 清洁

### 简单清洁（换料之间）
1. 将喷嘴加热到 200–220°C
2. 手动推入耗材直到挤出干净
3. 快速抽出耗材（"冷拔"——见下文）

### IPA 清洁
用于顽固残留物：
1. 将喷嘴加热到 200°C
2. 在喷嘴尖端滴 1–2 滴 IPA（小心操作！）
3. 让蒸汽溶解残留物
4. 推入新鲜耗材

:::warning 热喷嘴上使用 IPA 需谨慎
IPA 在 83°C 沸腾，在热喷嘴上会剧烈蒸发。用量要少，避免吸入蒸汽。
:::

## 冷拔（Cold Pull）

冷拔是从喷嘴中去除污染物和碳化残留物最有效的方法。

**操作步骤：**
1. 将喷嘴加热到 200–220°C
2. 手动将尼龙耗材（或当前耗材）推入喷嘴
3. 让尼龙在喷嘴中浸泡 1–2 分钟
4. 将温度降至 80–90°C（尼龙）
5. 等待喷嘴冷却至目标温度
6. 一次性快速有力地抽出耗材
7. 检查末端：应呈喷嘴内腔形状——干净无残留
8. 重复 3–5 次，直到抽出的耗材干净发白

:::tip 冷拔用尼龙
尼龙是冷拔的最佳材料，因为它能很好地粘附污染物。白色尼龙便于判断抽出是否干净。
:::

## 喷嘴更换

### 需要更换的迹象
- 底面粗糙，尺寸精度差
- 清洁后挤出问题仍然存在
- 喷嘴孔出现明显磨损或变形
- 喷嘴已超过预计使用寿命

### 操作步骤（P1S/X1C）
1. 将喷嘴加热到 200°C
2. 松开挤出机电机（释放耗材）
3. 用扳手松开喷嘴（逆时针）
4. 趁热更换——**不要让喷嘴带着工具冷却**
5. 拧紧至适当力矩（不要过紧）
6. 更换后运行校准

:::warning 始终趁热更换
冷喷嘴的紧固力矩在加热时可能撑裂部件。始终在喷嘴热时（200°C）进行安装和紧固。
:::

## 维护周期

| 工作内容 | 间隔 |
|-----------|---------|
| 清洁（冷拔） | 每 50 小时，或换料时 |
| 目视检查 | 每周 |
| 更换喷嘴（黄铜） | 200–500 小时 |
| 更换喷嘴（硬化钢） | 300–600 小时 |
