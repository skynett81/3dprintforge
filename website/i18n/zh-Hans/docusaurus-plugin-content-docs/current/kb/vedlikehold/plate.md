# 打印平台维护

打印平台对良好的粘附性和成功打印至关重要。正确的清洁和使用方法可以大大延长其使用寿命。

## 清洁

### 打印之间（IPA）
IPA（异丙醇）是清除油脂残留和指纹的标准方法：

```
1. 等平台冷却到 40°C 以下
2. 将 IPA 倒在无绒纸巾上（不要直接喷在平台上）
3. 用画圆的方式擦拭表面
4. 晾干 30–60 秒
5. 之后不要触摸表面（手指上的油脂会转移到平台上）
```

:::tip 避免指纹
手指油脂是粘附不良最常见的原因。处理平台时始终拿边缘，或戴一次性手套。
:::

### 深度清洁（水和洗洁精）
每月进行一次，或使用固体胶后：

1. 温水（30–50°C）加少量温和洗洁精
2. 用软布或海绵轻轻擦洗（不要用磨砂面）
3. 在流水下彻底冲洗
4. 用无绒纸擦干
5. 完全晾干后，用 IPA 做最后清洁

### 固体胶后的清洁
打印后**必须**清洗固体胶残留：
1. 将平台在温水中浸泡 2–3 分钟
2. 固体胶在水中容易溶解
3. 如上所述冲洗并擦干

## 防止损坏

### 正确取出打印件
- **等待冷却** — 大多数材料在 30–40°C 时会自然脱落
- **弯曲平台** — 弹簧钢板易于弯曲；向下压边缘以松动打印件
- **切勿使用尖锐工具** — 刀子和刮刀会划伤并损坏 PEI 涂层

:::danger 禁止使用金属刮刀
金属刮刀会划伤 PEI 涂层，大幅缩短平台使用寿命。使用弯曲技巧，或在必要时使用塑料刮刀。
:::

### 温度注意
- **不要将冷平台放在热热床上** — 避免热冲击
- **不要将冷 IPA 喷在热平台上** — 同样的原因

## 何时更换平台？

出现以下情况时更换平台：
- PEI 涂层明显损坏（撕裂、脱落、深划痕）
- 彻底清洁和深度清洁后仍无法粘附
- 表面变形或不平整
- 中间区域已磨平光滑（工程板）

### 典型使用寿命

| 平台 | 正常使用 | 高强度使用 |
|-------|------------|--------------|
| 光滑 PEI 板 | 200–500 次打印 | 100–200 次打印 |
| 工程板 | 300–700 次打印 | 200–400 次打印 |
| 高温板 | 200–400 次打印 | 100–200 次打印 |
| 纹理 PEI 板 | 300–600 次打印 | 200–300 次打印 |

## 存储

未使用的平台存储建议：
- 放在原始保护套或平整文件夹中
- 避免直射阳光（紫外线会降解 PEI）
- 避免多尘环境——灰尘会嵌入纹理，降低粘附性
