# 润滑

正确润滑运动部件可以减少磨损、降低噪音并保证精准运动。拓竹打印机使用需要定期润滑的线性运动系统。

## 润滑类型

| 部件 | 润滑类型 | 推荐产品 |
|-----------|-------------|---------|
| 线性导杆（XY） | 轻质机油或 PTFE 喷雾 | 3-合-1、Super Lube |
| Z 轴丝杆 | 浓稠润滑脂 | Super Lube 润滑脂 |
| 线性滑块 | 轻质锂基脂 | 拓竹润滑脂 |
| 拖链接头 | 无（干燥） | — |

## 线性导杆

### X 轴和 Y 轴
导杆是经过抛光的钢杆，通过线性滑块滑动：

```
周期：每 200–300 小时，或出现吱呀声时
用量：极少——每根导杆一滴即可
操作步骤：
1. 关闭打印机
2. 手动将滑车移至端部
3. 在导杆中间涂抹 1 滴轻质机油
4. 缓慢来回移动滑车 10 次
5. 用无绒纸擦去多余的油
```

:::warning 不要过度润滑
过多的油会吸附灰尘，形成磨蚀性膏状物。用量要少，并始终擦去多余部分。
:::

### Z 轴（垂直）
Z 轴使用丝杆（leadscrew），需要润滑脂（而非机油）：

```
周期：每 200 小时
操作步骤：
1. 关闭打印机
2. 沿丝杆涂抹薄薄一层润滑脂
3. 手动（或通过维护菜单）上下移动 Z 轴
4. 润滑脂会自动分布均匀
```

## 线性滑块

拓竹 P1S 和 X1C 在 Y 轴上使用线性滑块（MGN12）：

```
周期：每 300–500 小时
操作步骤：
1. 用针或牙签从注油口取出少量旧脂
2. 用注射器和细针管注入新润滑脂
3. 来回移动轴以使润滑脂均匀分布
```

拓竹官方润滑脂（Bambu Lubricant）是针对该系统校准的推荐产品。

## 各型号润滑维护

### X1C / P1S
- Y 轴：线性滑块——使用拓竹润滑脂
- X 轴：碳纤维杆——使用轻质机油
- Z 轴：双丝杆——使用拓竹润滑脂

### A1 / A1 Mini
- 所有轴：钢导杆——使用轻质机油
- Z 轴：单丝杆——使用拓竹润滑脂

## 需要润滑的迹象

- 运动时出现**吱呀声或刮擦声**
- 垂直壁面上出现**振纹（VFA）**
- 无其他原因的**尺寸不精准**
- 运动系统**噪音增大**

## 润滑周期

| 工作内容 | 间隔 |
|-----------|---------|
| 润滑 XY 导杆 | 每 200–300 小时 |
| 润滑 Z 轴丝杆 | 每 200 小时 |
| 润滑线性滑块（X1C/P1S） | 每 300–500 小时 |
| 完整维护周期 | 每半年（或 500 小时） |

使用仪表板中的维护模块自动追踪各项间隔。
