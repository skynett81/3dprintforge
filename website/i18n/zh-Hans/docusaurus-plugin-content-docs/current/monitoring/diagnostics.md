# 诊断

诊断页面为您提供打印机健康状况、性能和历史状态的深度概览。

访问地址：**https://localhost:3443/#diagnostics**

## 健康评分

每台打印机根据以下因素计算 0–100 的**健康评分**：

| 因素 | 权重 | 说明 |
|---|---|---|
| 成功率（30天） | 30% | 近 30 天成功打印的比例 |
| 组件磨损 | 25% | 关键部件的平均磨损程度 |
| HMS 错误（30天） | 20% | 错误数量和严重程度 |
| 校准状态 | 15% | 上次校准距今的时间 |
| 温度稳定性 | 10% | 打印过程中偏离目标温度的程度 |

**评分解读：**
- 80–100 — 状态极佳
- 60–79 — 状态良好，但有些方面需要关注
- 40–59 — 性能下降，建议维护
- 0–39 — 状态危急，必须进行维护

:::tip 历史记录
点击健康图表查看评分随时间的变化趋势。评分的大幅下降可能指示某一具体事件。
:::

## 遥测图表

遥测页面显示所有传感器数值的交互式图表：

### 可用数据集

- **喷嘴温度** — 实际值与目标值
- **热床温度** — 实际值与目标值
- **腔室温度** — 机器内部环境温度
- **挤出机电机** — 电流消耗和温度
- **风扇转速** — 打印头风扇、腔室风扇、AMS 风扇
- **气压**（X1C）— AMS 腔室气压
- **加速度** — 振动数据（ADXL345）

### 图表操作

1. 选择**时间范围**：最近一小时 / 24 小时 / 7 天 / 30 天 / 自定义
2. 从下拉列表中选择**打印机**
3. 选择要显示的**数据集**（支持多选）
4. 滚动鼠标缩放时间轴
5. 点击并拖动进行平移
6. 双击重置缩放

### 导出遥测数据

1. 点击图表上的**导出**
2. 选择格式：**CSV**、**JSON** 或 **PNG**（图片）
3. 将导出所选时间范围和数据集

## 热床网格

热床网格可视化显示打印平台的平整度校准情况：

1. 进入**诊断 → 热床网格**
2. 选择打印机
3. 最新网格将以 3D 曲面和热图形式显示：
   - **蓝色** — 低于中心（凹形）
   - **绿色** — 接近平整
   - **红色** — 高于中心（凸形）
4. 悬停在某点查看精确偏差（mm）

### 从 UI 扫描热床网格

1. 点击**立即扫描**（需要打印机处于空闲状态）
2. 在对话框中确认——打印机将自动开始校准
3. 等待扫描完成（约 3–5 分钟）
4. 新网格将自动显示

:::warning 请先预热
应在热床预热后（PLA 约 50–60°C）扫描热床网格，以获得准确的校准结果。
:::
