# 错误模式分析

错误模式分析利用打印和错误的历史数据来识别规律、原因和关联——并提供具体的改进建议。

访问地址：**https://localhost:3443/#error-analysis**

## 分析内容

系统分析以下数据点：

- HMS 错误代码和发生时间
- 出错时使用的耗材类型和品牌
- 出错时的温度（喷嘴、热床、腔室）
- 打印速度和打印配置文件
- 一天中的时段和星期几
- 上次维护距今的时间
- 打印机型号和固件版本

## 关联分析

系统寻找错误与各因素之间的统计关联：

**可检测到的关联示例：**
- 「78% 的 AMS 堵丝错误发生在使用 X 品牌耗材时」
- 「连续打印 6 小时以上后，喷嘴堵塞的发生率是平时的 3 倍」
- 「腔室温度低于 18°C 时，粘附失败率升高」
- 「拉丝问题与湿度超过 60% 存在关联（需连接湿度计）」

具有统计显著性（p < 0.05）的关联排列在最前面。

:::info 数据要求
分析结果在历史记录达到至少 50 次打印时最为准确。打印次数较少时，结果将显示低置信度估算。
:::

## 改进建议

基于分析结果，系统生成具体建议：

| 建议类型 | 示例 |
|---|---|
| 耗材 | 「换用其他品牌的 PA-CF——4 次错误中有 3 次使用了 X 品牌」 |
| 温度 | 「PETG 热床温度提高 5°C——预计可减少 60% 的粘附失败」 |
| 速度 | 「4 小时后将速度降至 80%——预计可减少 45% 的喷嘴堵塞」 |
| 维护 | 「清洁挤出机齿轮——磨损与 40% 的挤出错误存在关联」 |
| 校准 | 「运行热床调平——上周 15 次粘附失败中有 12 次与校准偏差相关」 |

每条建议显示：
- 预计效果（错误减少百分比）
- 置信度（低 / 中 / 高）
- 逐步实施指南
- 相关文档链接

## 对健康评分的影响

分析与健康评分相关联（参见[诊断](./diagnostics)）：

- 显示最影响评分的因素
- 估算实施每条建议后的评分提升
- 按潜在评分提升对建议进行优先级排序

## 时间线视图

进入**错误分析 → 时间线**查看按时间顺序排列的概览：

1. 选择打印机和时间范围
2. 错误以时间线上的点显示，按类型用颜色区分
3. 水平线标记维护任务
4. 错误集群（短时间内多次错误）以红色突出显示

点击某个集群可打开针对该特定时间段的分析。

## 报告

生成错误分析的 PDF 报告：

1. 点击**生成报告**
2. 选择时间范围（例如最近 90 天）
3. 选择内容：关联分析、建议、时间线、健康评分
4. 下载 PDF 或发送到邮箱

如果打印机关联到项目，报告将保存在该项目下。

:::tip 每周回顾
在**设置 → 报告**中设置每周自动发送邮件报告，无需手动访问仪表板即可保持更新。请参阅[报告](../system/reports)。
:::
