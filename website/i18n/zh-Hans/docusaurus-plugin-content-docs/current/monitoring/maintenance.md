# 维护

维护模块帮助您规划和追踪拓竹打印机的所有维护工作——从更换喷嘴到润滑导轨。

访问地址：**https://localhost:3443/#maintenance**

## 维护计划

3DPrintForge 为所有拓竹打印机型号预配置了维护间隔：

| 任务 | 间隔（默认） | 适用型号 |
|---|---|---|
| 清洁喷嘴 | 每 200 小时 | 全部 |
| 更换喷嘴（黄铜） | 每 500 小时 | 全部 |
| 更换喷嘴（硬化钢） | 每 2000 小时 | 全部 |
| 润滑 X 轴 | 每 300 小时 | X1C、P1S |
| 润滑 Z 轴 | 每 300 小时 | 全部 |
| 清洁 AMS 齿轮 | 每 200 小时 | AMS |
| 清洁腔室 | 每 500 小时 | X1C |
| 更换 PTFE 管 | 按需 / 1000 小时 | 全部 |
| 全面校准 | 每月 | 全部 |

所有间隔均可按打印机单独调整。

## 喷嘴更换记录

1. 进入**维护 → 喷嘴**
2. 点击**记录喷嘴更换**
3. 填写：
   - **日期** — 自动设为今天
   - **喷嘴材质** — 黄铜 / 硬化钢 / 铜 / 红宝石
   - **喷嘴直径** — 0.2 / 0.4 / 0.6 / 0.8 mm
   - **品牌/型号** — 可选
   - **价格** — 用于费用记录
   - **更换时的打印时长** — 自动从打印计时器获取
4. 点击**保存**

记录按日期排序显示所有喷嘴更换历史。

:::tip 提前提醒
设置**提前 X 小时提醒**（例如 50 小时），以便在下次建议更换前充分预警。
:::

## 创建维护任务

1. 点击**新任务**（+ 图标）
2. 填写：
   - **任务名称** — 例如「润滑 Y 轴」
   - **打印机** — 选择适用的打印机
   - **间隔类型** — 小时 / 天 / 打印次数
   - **间隔** — 例如 300 小时
   - **上次执行时间** — 填写上次执行日期（可回填历史日期）
3. 点击**创建**

## 间隔与提醒

活动任务以颜色显示状态：
- **绿色** — 距下次维护时间 > 剩余间隔的 50%
- **黄色** — 距下次维护时间 < 50%
- **橙色** — 距下次维护时间 < 20%
- **红色** — 维护已到期

### 配置提醒

1. 点击某个任务 → **编辑**
2. 启用**提醒**
3. 设置**在**剩余 10% 时提醒
4. 选择通知渠道（参见[通知](../features/notifications)）

## 标记为已完成

1. 在列表中找到该任务
2. 点击**完成**（勾选图标）
3. 间隔从今天的日期/打印时长重新计算
4. 系统自动创建日志条目

## 费用记录

所有维护任务均可关联费用：

- **零件** — 喷嘴、PTFE 管、润滑剂
- **工时** — 耗时 × 时薪
- **外部服务** — 付费维修

费用按打印机汇总，显示在统计概览中。

## 维护历史

进入**维护 → 历史**查看：
- 所有已完成的维护任务
- 日期、打印时长和费用
- 执行人（多用户系统）
- 备注和说明

将历史记录导出为 CSV 用于会计目的。
