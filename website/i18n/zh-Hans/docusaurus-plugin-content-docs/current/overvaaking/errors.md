# 错误日志

错误日志汇总了您打印机的所有错误和 HMS 警报（健康、维护、安全）。Bambu Dashboard 内置了涵盖 269+ 条拓竹打印机 HMS 代码的数据库。

访问地址：**https://localhost:3443/#errors**

## HMS 代码

拓竹打印机在出现问题时通过 MQTT 发送 HMS 代码。Bambu Dashboard 自动将这些代码翻译为可读的错误消息：

| 代码 | 示例 | 类别 |
|---|---|---|
| `0700 0100 0001 0001` | Nozzle heatbreak clogged | 喷嘴/挤出机 |
| `0700 0200 0002 0001` | AMS filament stuck | AMS |
| `0700 0300 0003 0001` | Bed leveling failed | 打印平台 |
| `0700 0500 0001 0001` | MC disconnect | 电子元件 |

完整列表涵盖 X1C、X1C Combo、X1E、P1S、P1S Combo、P1P、P2S、P2S Combo、A1、A1 Combo、A1 mini、H2S、H2D 和 H2C 的所有 269+ 条已知代码。

## 严重程度分级

错误分为四个级别：

| 级别 | 颜色 | 说明 |
|---|---|---|
| **危急** | 红色 | 需要立即处理——打印已停止 |
| **高** | 橙色 | 应尽快处理——打印可继续 |
| **中** | 黄色 | 需要排查——无即时风险 |
| **信息** | 蓝色 | 提示信息，无需操作 |

## 搜索与筛选

使用错误日志顶部的工具栏：

1. **全文搜索** — 按错误消息、HMS 代码或打印机描述搜索
2. **打印机** — 仅显示某台打印机的错误
3. **类别** — AMS / 喷嘴 / 打印平台 / 电子元件 / 校准 / 其他
4. **严重程度** — 全部 / 危急 / 高 / 中 / 信息
5. **日期** — 按日期范围筛选
6. **未确认** — 仅显示尚未确认的错误

点击**重置筛选**查看所有错误。

## Wiki 链接

每条 HMS 代码都附有拓竹 Wiki 链接，内容包括：

- 完整错误描述
- 可能的原因
- 逐步故障排查指南
- 拓竹官方建议

点击错误条目上的**打开 Wiki** 在新标签页中打开相关 Wiki 页面。

:::tip 本地缓存
Bambu Dashboard 在本地缓存 Wiki 内容以供离线使用。内容每周自动更新。
:::

## 确认错误

确认操作将错误标记为已处理，但不会删除它：

1. 点击列表中的某条错误
2. 点击**确认**（勾选图标）
3. 输入可选的处理说明
4. 错误将标记为已确认并移至「已确认」列表

### 批量确认

1. 使用复选框选择多条错误
2. 点击**确认所选**
3. 所有选中的错误将同时被确认

## 统计数据

错误日志顶部显示：

- 近 30 天的错误总数
- 未确认错误数量
- 最常出现的 HMS 代码
- 错误最多的打印机

## 导出

1. 点击**导出**（下载图标）
2. 选择格式：**CSV** 或 **JSON**
3. 导出将应用当前筛选条件——请先设置所需筛选
4. 文件将自动下载

## 新错误通知

启用新 HMS 错误的通知：

1. 进入**设置 → 通知**
2. 勾选**新 HMS 错误**
3. 选择通知的最低严重程度（建议：**高**及以上）
4. 选择通知渠道

请参阅[通知](../funksjoner/notifications)了解渠道设置。
