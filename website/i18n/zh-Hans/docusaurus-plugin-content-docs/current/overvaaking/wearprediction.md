# 磨损预测

磨损预测根据实际使用情况、耗材类型和打印机行为计算关键部件的预计使用寿命——让您能够主动规划维护，而非被动响应。

访问地址：**https://localhost:3443/#wear**

## 监控的组件

Bambu Dashboard 追踪每台打印机的 8 个组件磨损情况：

| 组件 | 主要磨损因素 | 典型寿命 |
|---|---|---|
| **喷嘴（黄铜）** | 耗材类型 + 使用时长 | 300–800 小时 |
| **喷嘴（硬化钢）** | 使用时长 + 磨蚀性材料 | 1500–3000 小时 |
| **PTFE 管** | 使用时长 + 高温 | 500–1500 小时 |
| **挤出机齿轮** | 使用时长 + 磨蚀性材料 | 1000–2000 小时 |
| **X 轴碳管** | 打印次数 + 速度 | 2000–5000 小时 |
| **打印平台表面** | 打印次数 + 温度 | 200–500 次 |
| **AMS 齿轮** | 换料次数 | 5000–15000 次 |
| **腔室风扇** | 运行时长 | 3000–8000 小时 |

## 磨损计算

磨损以综合百分比（0–100% 已磨损）表示：

```
磨损% = （实际使用量 / 预期寿命）× 100
       × 材料系数
       × 速度系数
```

**材料系数：**
- PLA、PETG：1.0×（正常磨损）
- ABS、ASA：1.1×（略微偏高）
- PA、PC：1.2×（对 PTFE 和喷嘴损耗较大）
- CF/GF 复合材料：2.0–3.0×（极强磨蚀性）

:::warning 碳纤维耗材
碳纤维增强耗材（CF-PLA、CF-PA 等）会极快磨损黄铜喷嘴。请使用硬化钢喷嘴，并预期磨损速度加快 2–3 倍。
:::

## 寿命计算

每个组件显示：

- **当前磨损** — 已使用百分比
- **预计剩余寿命** — 小时或打印次数
- **预计到期日期** — 基于最近 30 天的平均使用量
- **置信区间** — 预测的不确定性范围

点击某个组件查看磨损随时间累积的详细图表。

## 提醒

为每个组件配置自动提醒：

1. 进入**磨损 → 设置**
2. 为每个组件设置**提醒阈值**（建议：75% 和 90%）
3. 选择通知渠道（参见[通知](../funksjoner/notifications)）

**示例提醒消息：**
> ⚠️ 我的 X1C 上的喷嘴（黄铜）已磨损 78%。预计剩余寿命：约 45 小时。建议：安排喷嘴更换。

## 维护费用

磨损模块与费用记录集成：

- **每个组件的费用** — 备件价格
- **更换总费用** — 所有接近上限组件的费用总和
- **未来 6 个月预测** — 预估的维护费用

在**磨损 → 价格**中设置组件价格：

1. 点击**设置价格**
2. 填写每个组件的单价
3. 该价格用于费用预测，可按打印机型号区分

## 重置磨损计数器

维护后，重置相关组件的计数器：

1. 进入**磨损 → [组件名称]**
2. 点击**标记为已更换**
3. 填写：
   - 更换日期
   - 费用（可选）
   - 备注（可选）
4. 磨损计数器重置并重新计算

重置记录将显示在维护历史中。

:::tip 校准
将磨损预测与实际经验数据对比，在**磨损 → 配置寿命**中调整寿命参数，使计算结果更符合您的实际使用情况。
:::
