# 备份

3DPrintForge 可备份所有配置、历史记录和数据，便于在系统故障、服务器迁移或更新问题时轻松恢复。

访问地址：**https://localhost:3443/#settings** → **系统 → 备份**

## 备份内容

| 数据类型 | 是否包含 | 说明 |
|---|---|---|
| 打印机设置和配置 | ✅ | |
| 打印历史 | ✅ | |
| 耗材库存 | ✅ | |
| 用户和角色 | ✅ | 密码以哈希值存储 |
| 设置 | ✅ | 含通知配置 |
| 维护日志 | ✅ | |
| 项目和发票 | ✅ | |
| 文件库（元数据） | ✅ | |
| 文件库（文件） | 可选 | 体积可能较大 |
| 延时摄影视频 | 可选 | 体积可能非常大 |
| 相册图片 | 可选 | |

## 创建手动备份

1. 进入**设置 → 备份**
2. 选择要包含的内容（见上表）
3. 点击**立即创建备份**
4. 备份创建过程中显示进度指示
5. 完成后点击**下载**

备份保存为带时间戳文件名的 `.zip` 文件：
```
3dprintforge-backup-2026-03-22T14-30-00.zip
```

## 下载备份

备份文件保存在服务器的备份文件夹中（可配置）。您也可以直接下载：

1. 进入**备份 → 现有备份**
2. 在列表中找到备份（按日期排序）
3. 点击**下载**（下载图标）

:::info 存储位置
默认存储文件夹：`./data/backups/`。在**设置 → 备份 → 存储文件夹**中更改。
:::

## 计划自动备份

1. 在**备份 → 计划**中启用**自动备份**
2. 选择间隔：
   - **每日** — 在 03:00 执行（可配置）
   - **每周** — 指定星期和时间
   - **每月** — 每月第一天
3. 选择**保留备份数量**（例如 7——自动删除较旧的备份）
4. 点击**保存**

:::tip 外部存储
对于重要数据：将外部硬盘或网络存储挂载为备份存储文件夹，这样即使系统盘故障，备份也能保全。
:::

## 从备份恢复

:::warning 恢复将覆盖现有数据
恢复操作会用备份文件中的内容替换所有现有数据。请确保在操作前先备份当前数据。
:::

### 从服务器上的现有备份恢复

1. 进入**备份 → 现有备份**
2. 在列表中找到备份
3. 点击**恢复**
4. 在对话框中确认
5. 恢复完成后系统将自动重启

### 从已下载的备份文件恢复

1. 点击**上传备份**
2. 从您的计算机中选择 `.zip` 文件
3. 文件验证后将显示包含的内容
4. 点击**从文件恢复**
5. 在对话框中确认

## 备份验证

3DPrintForge 在恢复前验证所有备份文件：

- 检查 ZIP 格式是否有效
- 验证数据库架构与当前版本是否兼容
- 如果备份来自旧版本则显示警告（将自动执行迁移）
