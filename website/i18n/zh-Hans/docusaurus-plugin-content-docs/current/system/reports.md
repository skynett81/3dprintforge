# 报告

3DPrintForge 可自动发送包含统计数据和活动摘要的邮件报告——按周、按月或两者都发送。

访问地址：**https://localhost:3443/#settings** → **系统 → 报告**

## 前提条件

报告功能需要已配置邮件通知。在启用报告前，请先在**设置 → 通知 → 邮件**中设置 SMTP。请参阅[通知](../features/notifications)。

## 启用自动报告

1. 进入**设置 → 报告**
2. 启用**每周报告**和/或**每月报告**
3. 选择**发送时间**：
   - 每周：星期几和具体时间
   - 每月：当月第几天（例如每月第一个工作日 / 最后一个周五）
4. 填写**收件人邮箱**（多个收件人用逗号分隔）
5. 点击**保存**

发送测试报告以预览格式：点击**立即发送测试报告**。

## 每周报告内容

每周报告涵盖最近 7 天：

### 摘要
- 打印总次数
- 成功 / 失败 / 中止次数
- 成功率及与上周相比的变化
- 最活跃的打印机

### 活动
- 每日打印次数（迷你图表）
- 总打印时长
- 总耗材消耗量（克和费用）

### 耗材
- 按材料和品牌分类的消耗量
- 各线卷预计剩余量（低于 20% 的线卷高亮显示）

### 维护
- 本周已完成的维护任务
- 逾期维护任务（红色警告）
- 下周即将到期的任务

### HMS 错误
- 本周每台打印机的 HMS 错误数量
- 未确认的错误（需要关注）

## 每月报告内容

每月报告涵盖最近 30 天，包含每周报告的所有内容，另外还有：

### 趋势
- 与上月对比（%）
- 活动热力图（当月缩略热力图）
- 月度成功率趋势

### 费用
- 耗材总费用
- 电费总计（如已配置功率监测）
- 磨损总费用
- 维护总费用

### 磨损与健康
- 每台打印机的健康评分（含与上月相比的变化）
- 即将需要更换的组件

### 统计亮点
- 最长的成功打印
- 最常用的耗材类型
- 活跃度最高的打印机

## 自定义报告

1. 进入**设置 → 报告 → 自定义**
2. 勾选/取消勾选要包含的章节
3. 选择**打印机筛选**：所有打印机或特定打印机
4. 选择**标志显示**：在页眉显示 3DPrintForge 标志或关闭
5. 点击**保存**

## 报告归档

所有已发送的报告均被存档，可随时重新查看：

1. 进入**设置 → 报告 → 归档**
2. 从列表中选择报告（按日期排序）
3. 点击**打开**查看 HTML 版本
4. 点击**下载 PDF** 下载报告

报告在 **90 天**后自动删除（可配置）。
