# 设置

Bambu Dashboard 的所有设置集中在一个页面，按清晰的类别组织。以下是各类别的内容概览。

访问地址：**https://localhost:3443/#settings**

## 打印机

管理已注册的打印机：

| 设置 | 说明 |
|---|---|
| 添加打印机 | 使用序列号和访问码注册新打印机 |
| 打印机名称 | 自定义显示名称 |
| 打印机型号 | X1C / X1C Combo / X1E / P1S / P1S Combo / P1P / P2S / P2S Combo / A1 / A1 Combo / A1 mini / H2S / H2D / H2C |
| MQTT 连接 | 拓竹云 MQTT 或本地 MQTT |
| 访问码 | 拓竹 App 中的局域网访问码 |
| IP 地址 | 用于本地（LAN）模式 |
| 摄像头设置 | 启用/禁用、分辨率 |

参见[快速开始](../kom-i-gang/oppsett)了解首台打印机的逐步设置。

## 通知

完整文档请参阅[通知](../funksjoner/notifications)。

快速概览：
- 启用/禁用通知渠道（Telegram、Discord、邮件等）
- 按渠道的事件筛选
- 免打扰时段（指定时间段内不发送通知）
- 每个渠道的测试按钮

## 主题

完整文档请参阅[主题](./themes)。

- 浅色 / 深色 / 自动模式
- 6 种色彩方案
- 自定义强调色
- 圆角和紧凑度

## OBS 覆盖层

OBS 覆盖层配置：

| 设置 | 说明 |
|---|---|
| 默认主题 | dark / light / minimal |
| 默认位置 | 覆盖层角落位置 |
| 默认缩放 | 缩放比例（0.5–2.0） |
| 显示二维码 | 在覆盖层中显示仪表板二维码 |

参见 [OBS 覆盖层](../funksjoner/obs-overlay)了解完整 URL 语法和设置。

## 电力与电价

| 设置 | 说明 |
|---|---|
| Tibber API Token | 获取 Tibber 现货电价 |
| Nordpool 价格区域 | 选择电价区域 |
| 电网费（元/kWh） | 您的电网费率 |
| 打印机功率（W） | 按打印机型号配置功耗 |

## Home Assistant

| 设置 | 说明 |
|---|---|
| MQTT 代理 | IP、端口、用户名、密码 |
| 发现前缀 | 默认：`homeassistant` |
| 启用发现 | 向 HA 发布设备 |

## Webhook

全局 Webhook 设置：

| 设置 | 说明 |
|---|---|
| Webhook URL | 事件接收地址 |
| 密钥 | HMAC-SHA256 签名 |
| 事件筛选 | 发送哪些事件 |
| 重试次数 | 失败时的重试次数（默认：3） |
| 超时时间 | 请求放弃前的等待秒数（默认：10） |

## 队列设置

| 设置 | 说明 |
|---|---|
| 自动调度 | 启用/禁用 |
| 调度策略 | 第一台空闲 / 使用最少 / 轮询 |
| 需要确认 | 发送前手动批准 |
| 错峰启动 | 队列中打印机之间的启动延迟 |

## 安全

| 设置 | 说明 |
|---|---|
| 会话时长 | 自动退出前的小时/天数 |
| 强制 2FA | 要求所有用户启用 2FA |
| IP 白名单 | 限制访问到指定 IP 地址 |
| HTTPS 证书 | 上传自定义证书 |

## 系统

| 设置 | 说明 |
|---|---|
| 服务器端口 | 默认：3443 |
| 日志格式 | JSON / 文本 |
| 日志级别 | Error / Warn / Info / Debug |
| 数据库清理 | 自动删除旧历史记录 |
| 更新 | 检查新版本 |
