# 主题

3DPrintForge 拥有灵活的主题系统，让您可以根据个人喜好和使用场景自定义外观。

访问地址：**https://localhost:3443/#settings** → **主题**

## 颜色模式

在三种模式中选择：

| 模式 | 说明 |
|---|---|
| **浅色** | 浅色背景，深色文字——适合光线充足的环境 |
| **深色** | 深色背景，浅色文字——默认，推荐用于监控 |
| **自动** | 跟随操作系统设置（系统深色/浅色模式） |

在主题设置顶部更改模式，或通过导航栏的快捷按钮切换（月亮/太阳图标）。

## 色彩方案

提供六种预设色彩方案：

| 方案 | 主色 | 风格 |
|---|---|---|
| **Bambu** | 绿色（#00C853） | 默认，灵感来自拓竹 |
| **深蓝夜色** | 蓝色（#2196F3） | 沉稳专业 |
| **日落** | 橙色（#FF6D00） | 温暖活力 |
| **紫色** | 紫色（#9C27B0） | 创意独特 |
| **红色** | 红色（#F44336） | 高对比度，醒目 |
| **单色** | 灰色（#607D8B） | 中性极简 |

点击某个方案即可立即预览并激活。

## 自定义强调色

使用完全自定义的颜色作为强调色：

1. 点击色彩方案选择器下方的**自定义颜色**
2. 使用颜色选择器或输入十六进制代码（例如 `#FF5722`）
3. 预览实时更新
4. 点击**应用**以激活

:::tip 对比度
请确保强调色与背景有足够的对比度。如果颜色可能造成可读性问题，系统会发出警告（WCAG AA 标准）。
:::

## 圆角

调整按钮、卡片和元素的圆角：

| 设置 | 说明 |
|---|---|
| **直角** | 无圆角（矩形风格） |
| **小** | 微妙圆角（4 px） |
| **中** | 标准圆角（8 px） |
| **大** | 明显圆角（16 px） |
| **胶囊** | 最大圆角（50 px） |

拖动滑块在 0–50 px 之间手动调整。

## 紧凑度

调整界面元素的密度：

| 设置 | 说明 |
|---|---|
| **宽松** | 元素间距更大 |
| **标准** | 平衡，默认设置 |
| **紧凑** | 更密集的排列——屏幕显示更多信息 |

紧凑模式推荐用于 1080p 以下的显示器或信息亭视图。

## 字体

选择字体：

- **系统** — 使用操作系统默认字体（加载速度快）
- **Inter** — 清晰现代（默认选择）
- **JetBrains Mono** — 等宽字体，适合数据值显示
- **Nunito** — 更柔和圆润的风格

## 动画

关闭或调整动画效果：

- **完整** — 所有过渡和动画效果（默认）
- **减弱** — 仅必要的动画（遵循操作系统偏好）
- **关闭** — 无动画效果，最大性能

:::tip 信息亭模式
信息亭视图推荐使用**紧凑** + **深色** + **减弱动画**，以获得最佳性能和远距离可读性。请参阅[信息亭模式](./kiosk)。
:::

## 导出和导入主题设置

与他人分享您的主题：

1. 点击**导出主题** — 下载一个 `.json` 文件
2. 将文件分享给其他 3DPrintForge 用户
3. 对方通过**导入主题** → 选择文件来导入
