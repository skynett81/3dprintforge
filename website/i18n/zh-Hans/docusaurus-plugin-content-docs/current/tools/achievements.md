# 成就

成就系统是一个游戏化功能，奖励您在打印历程中达成的里程碑和精彩时刻。收集成就，查看解锁下一个成就的进度。

访问地址：**https://localhost:3443/#achievements**

## 稀有度等级

成就分为四个稀有度等级：

| 等级 | 颜色 | 说明 |
|---|---|---|
| **普通** | 灰色 | 简单里程碑，容易达成 |
| **稀有** | 绿色 | 需要一定时间或努力 |
| **珍贵** | 蓝色 | 需要长期持续努力 |
| **传说** | 金色 | 非凡成就 |

## 成就示例

### 打印里程碑（普通 / 稀有）
| 成就 | 条件 |
|---|---|
| 第一次打印 | 完成您的第一次打印 |
| 整整一天 | 累计打印超过 24 小时 |
| 高成功率 | 连续 10 次成功打印 |
| 耗材收藏家 | 注册 10 种不同的耗材类型 |
| 多色打印 | 完成一次多色打印 |

### 数量成就（稀有 / 珍贵）
| 成就 | 条件 |
|---|---|
| 一公斤 | 累计使用 1 kg 耗材 |
| 十公斤 | 累计使用 10 kg 耗材 |
| 百次打印 | 100 次成功打印 |
| 五百小时 | 累计 500 小时打印时长 |
| 夜班工人 | 完成一次超过 20 小时的打印 |

### 维护与保养（稀有 / 珍贵）
| 成就 | 条件 |
|---|---|
| 尽职尽责 | 记录一次维护任务 |
| 打印机护理师 | 记录 10 次维护任务 |
| 零浪费 | 完成材料利用率 > 90% 的打印 |
| 喷嘴大师 | 更换喷嘴 5 次（已记录） |

### 传说成就
| 成就 | 条件 |
|---|---|
| 永不疲倦 | 1000 次成功打印 |
| 耗材泰斗 | 累计 50 kg 耗材消耗 |
| 完美一周 | 连续 7 天无任何失败打印 |
| 打印图书馆员 | 文件库中有 100 种不同模型 |

## 查看成就

成就页面显示：

- **已解锁** — 您已获得的成就（含解锁日期）
- **即将完成** — 接近达成的成就（含进度条）
- **未解锁** — 所有尚未获得的成就

按**等级**、**类别**或**状态**（已解锁 / 进行中 / 未解锁）筛选。

## 进度条

对于有计数的成就，显示进度条：

```
一公斤 — 1 kg 耗材
[████████░░] 847 g / 1000 g (84.7%)
```

## 通知

当您获得新成就时，系统自动发送通知：
- **浏览器弹窗**，显示成就名称和图标
- 可选：通过 Telegram / Discord 发送通知（在**设置 → 通知 → 成就**中配置）

## 多用户支持

在多用户系统中，每个用户有独立的成就档案。**排行榜**按以下条件显示排名：

- 已解锁成就总数
- 打印总次数
- 总打印时长

:::tip 隐私模式
在**设置 → 成就 → 从排行榜隐藏**中关闭排行榜，以保持档案私密。
:::
