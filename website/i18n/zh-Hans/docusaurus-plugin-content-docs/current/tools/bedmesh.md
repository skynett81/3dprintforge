# 热床网格

热床网格工具提供打印平台平整度的可视化表示——对良好的粘附性和均匀的首层至关重要。

访问地址：**https://localhost:3443/#bedmesh**

## 什么是热床网格

拓竹打印机使用探针扫描打印平台表面，生成高度偏差图（网格）。打印机固件在打印过程中自动补偿这些偏差。3DPrintForge 为您可视化该网格。

## 可视化

### 3D 曲面

热床网格以交互式 3D 曲面显示：

- 用鼠标旋转视图
- 滚动缩放
- 点击**俯视图**切换到鸟瞰视角
- 点击**侧视图**查看截面轮廓

颜色表示相对于平均高度的偏差：
- **蓝色** — 低于中心（凹形）
- **绿色** — 接近平整（偏差 < 0.1 mm）
- **黄色** — 中等偏差（0.1–0.2 mm）
- **红色** — 较大偏差（> 0.2 mm）

### 热力图

点击**热力图**切换到更直观的 2D 平面视图。

热力图显示：
- 每个测量点的精确偏差值（mm）
- 标记的问题点（偏差 > 0.3 mm）
- 测量网格的尺寸（行数 × 列数）

## 从 UI 扫描热床网格

:::warning 前提条件
扫描前必须打印机处于空闲状态，且热床温度已稳定。在扫描之前，请将热床预热到所需温度。
:::

1. 进入**热床网格**
2. 从下拉列表中选择打印机
3. 点击**立即扫描**
4. 选择扫描时的热床温度：
   - **冷床**（室温）— 速度快，但准确性较低
   - **热床**（PLA：50–60°C，PETG：70–90°C）— 推荐
5. 在对话框中确认——打印机将自动开始探针程序
6. 等待扫描完成（3–8 分钟，取决于网格大小）
7. 新网格图自动显示

## 校准指导

扫描后，系统提供具体建议：

| 发现 | 建议 |
|---|---|
| 各处偏差 < 0.1 mm | 极佳——无需操作 |
| 偏差 0.1–0.2 mm | 良好——固件补偿可处理 |
| 角落偏差 > 0.2 mm | 手动调整热床弹簧（如可调节） |
| 偏差 > 0.3 mm | 热床可能损坏或安装不当 |
| 中心高于角落 | 热膨胀——热床预热时属正常现象 |

:::tip 历史对比
点击**与上次对比**查看网格图是否随时间变化——有助于发现平台逐渐弯曲的问题。
:::

## 网格历史

所有网格扫描均带时间戳保存：

1. 点击热床网格侧面板中的**历史**
2. 选择两次扫描进行对比（显示差异图）
3. 删除不再需要的旧扫描

## 导出

将网格数据导出为：
- **PNG** — 热力图图片（用于记录文档）
- **CSV** — 每个点的 X、Y 和高度偏差原始数据
