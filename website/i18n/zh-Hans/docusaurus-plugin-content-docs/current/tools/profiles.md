# 打印配置文件

打印配置文件是保存的打印参数集，可在不同打印和打印机之间复用。通过为不同目的定义配置文件，节省时间并确保质量一致。

访问地址：**https://localhost:3443/#profiles**

## 创建配置文件

1. 进入**工具 → 打印配置文件**
2. 点击**新建配置文件**（+ 图标）
3. 填写：
   - **配置文件名称** — 描述性名称，例如「PLA - 快速生产」
   - **材料** — 从列表中选择（PLA / PETG / ABS / PA / PC / TPU 等）
   - **打印机型号** — X1C / X1C Combo / X1E / P1S / P1S Combo / P1P / P2S / P2S Combo / A1 / A1 Combo / A1 mini / H2S / H2D / H2C / 全部
   - **说明** — 可选文本

4. 填写设置（见下方各节）
5. 点击**保存配置文件**

## 配置文件中的设置

### 温度
| 字段 | 示例 |
|---|---|
| 喷嘴温度 | 220°C |
| 热床温度 | 60°C |
| 腔室温度（X1C） | 35°C |

### 速度
| 字段 | 示例 |
|---|---|
| 速度设置 | 标准 |
| 最大速度（mm/s） | 200 |
| 加速度 | 5000 mm/s² |

### 质量
| 字段 | 示例 |
|---|---|
| 层厚 | 0.2 mm |
| 填充密度 | 15% |
| 填充图案 | 网格 |
| 支撑材料 | 自动 |

### AMS 与颜色
| 字段 | 说明 |
|---|---|
| 清料量 | 换色时的清料量 |
| 优先槽位 | 优先使用哪个 AMS 槽位 |

### 高级
| 字段 | 说明 |
|---|---|
| 干燥模式 | 为潮湿材料启用 AMS 烘料 |
| 冷却时间 | 层间冷却暂停时间 |
| 风扇速度 | 冷却风扇速度百分比 |

## 编辑配置文件

1. 点击列表中的配置文件
2. 点击**编辑**（铅笔图标）
3. 进行修改
4. 点击**保存**（覆盖）或**另存为新配置文件**（创建副本）

:::tip 版本管理
在试验新参数时，使用「另存为新配置文件」保留已有效的配置文件。
:::

## 使用配置文件

### 从文件库

1. 在文件库中选择文件
2. 点击**发送到打印机**
3. 从下拉列表中选择**配置文件**
4. 应用配置文件中的设置

### 从打印队列

1. 创建新队列任务
2. 在设置中选择**配置文件**
3. 配置文件与队列任务关联

## 导入和导出配置文件

### 导出
1. 选择一个或多个配置文件
2. 点击**导出**
3. 选择格式：**JSON**（用于导入其他仪表板）或 **PDF**（用于打印/归档）

### 导入
1. 点击**导入配置文件**
2. 选择从其他 Bambu Dashboard 导出的 `.json` 文件
3. 同名的现有配置文件可以覆盖或两者都保留

## 分享配置文件

通过社区耗材模块分享配置文件（参见[社区耗材](../integrations/community)），或直接通过 JSON 导出分享。

## 默认配置文件

为每种材料设置默认配置文件：

1. 选择配置文件
2. 点击**设为 [材料] 的默认值**
3. 发送该材料的文件时，默认配置文件将自动选择
