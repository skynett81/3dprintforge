# 标签

标签工具为您的耗材线卷生成专业标签——二维码、适用于热敏打印机的线卷标签以及用于视觉识别的色卡。

访问地址：**https://localhost:3443/#labels**

## 二维码

生成链接到仪表板中耗材信息的二维码：

1. 进入**标签 → 二维码**
2. 选择要生成二维码的线卷
3. 二维码自动生成并显示在预览中
4. 点击**下载 PNG** 或**打印**

二维码包含仪表板中耗材配置文件的 URL。用手机扫描即可快速查看线卷信息。

### 批量生成

1. 点击**全选**或勾选单个线卷
2. 点击**生成所有二维码**
3. 下载为 ZIP 文件（每个线卷一个 PNG），或一次性打印所有

## 线卷标签

适用于热敏打印机的专业标签，包含完整线卷信息：

### 标签内容（默认）

- 线卷颜色（填充色块）
- 材料名称（大字体）
- 品牌商
- 颜色十六进制代码
- 温度推荐（喷嘴和热床）
- 二维码
- 条形码（可选）

### 热敏打印机 ZPL

为 Zebra、Brother 和 Dymo 打印机生成 ZPL 代码（Zebra 编程语言）：

1. 进入**标签 → 热敏打印**
2. 选择标签尺寸：**25×54 mm** / **36×89 mm** / **62×100 mm**
3. 选择线卷
4. 点击**生成 ZPL**
5. 通过以下方式将 ZPL 代码发送到打印机：
   - **直接打印**（USB 连接）
   - **复制 ZPL** 并通过终端命令发送
   - **下载 .zpl 文件**

:::tip 打印机设置
如需自动打印，在**设置 → 标签打印机**中配置打印站的 IP 地址和端口（RAW TCP 默认端口：9100）。
:::

### PDF 标签

适用于普通打印机，生成具有正确尺寸的 PDF：

1. 从模板中选择标签尺寸
2. 点击**生成 PDF**
3. 在不干胶纸上打印（Avery 或同类产品）

## 色卡

色卡是一个紧凑的网格，以视觉方式显示所有线卷：

1. 进入**标签 → 色卡**
2. 选择要包含的线卷（所有活跃线卷，或手动选择）
3. 选择卡片格式：**A4**（4×8）、**A3**（6×10）、**Letter**
4. 点击**生成 PDF**

每个格子显示：
- 实际颜色的色块
- 材料名称和颜色十六进制代码
- 材料编号（便于快速参考）

建议过塑后张贴在打印机工作站旁。

## 共享色彩方案

将选定颜色导出为共享色彩方案：

1. 进入**标签 → 色彩方案**
2. 选择要包含在方案中的线卷
3. 点击**分享方案**
4. 复制链接——其他人可以将方案导入到他们的仪表板
5. 方案显示十六进制代码，可导出为 **Adobe Swatch**（`.ase`）或 **Procreate**（`.swatches`）格式
